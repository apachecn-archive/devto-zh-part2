# 成为一个粗暴的女人(在科技领域)

> 原文：<https://dev.to/jennifer/on-being-an-abrasive-woman-in-tech-44jk>

*最初发布于[jmkoni.com](https://www.jenniferkonikowski.com/blog/2018/4/3/on-being-an-abrasive-woman-in-tech)T3】*

各位，故事时间到了！所有的名字都被改了。

我早期的一份工作是在一家有家庭氛围的公司工作。大家一起出去玩，我们都是“朋友”该公司也非常重视“美好”我把这句话用引号括起来，因为 80-90%的时候，因为不友好而被批评的人都是女性。见鬼，有一个家伙经常告诉人们他们“他妈的愚蠢”，我们会一笑置之。在这家公司，我的团队包括:

*   我的经理(彼得)
*   一个在队里呆了一年的家伙(基南)
*   一个在公司工作了很长时间但刚刚加入团队的人(Erlich)
*   还有我

有一段时间，我真的很喜欢我的经理和我的团队。一个警告是 Erlich，就像他的名字一样，是一个完全的讨厌鬼。他可能很有趣，但他会分散我的注意力，破坏我的工作，而且通常会把我吓跑。大约有六个月的时间，基南、厄尔利希和我共用一间办公室。后来一个新的开发人员加入了团队(Richard)，我和他一起搬进了一间办公室。这里安静多了，我做了很多事情。又过了六个月，理查德被解雇了。我去找彼得，要求我和一个相关团队的另一个人呆在一间办公室里，而不是和埃尔利希一起搬回办公室。我告诉彼得，埃尔利希让我分心，他吓了我一跳，每天 6-8 个小时不和他共用一间办公室，我会更舒服。

几天后我去度假，在我度假的时候，厄尔利希被放了。我有点担心时机，但他也是个差劲的员工！所以我觉得和我说的没关系。然而，在我回来后，Keenan 开始对待我不同。我正在做一个演示，解释我们应该做些什么来支持，我应该从他那里得到反馈。我回到基南至少五次，每次他都告诉我做一些其他的大改变。每次，我都做出改变，但每次，都不够。

然后基南向彼得举报我粗鲁。我不记得彼得给我发的那封电子邮件的确切内容了(抄送给了人力资源部)，但这听起来好像是我骂了我们团队的另一个人(尼尔森)，我不得不绞尽脑汁回想我说过的话。我一直在想我是不是昏过去了，忘记说了什么可怕的话。当我终于记起来的时候，我意识到我说的都是相当直接的“不，你错了。”我因为告诉同事他们错了而被人力资源部举报。我和彼得谈了这件事，他向我道歉，说他应该先和我谈谈，但在那个时候，我意识到我可能只能换工作了。

我在很多地方都见过这种双重标准，但这是最令人震惊的。大约一个月前，当基南决定提出“唐氏综合症患者不是人”的论点时，我正在吃午饭，然而我告诉一个人他们错了，这被认为是无礼的。

我没有这个故事的摘要。这只是我需要说的一件事，希望它能让其他人更仔细地考虑他们是如何对待女同事的。
