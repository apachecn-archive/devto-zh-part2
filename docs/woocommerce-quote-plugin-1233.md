# WooCommerce 报价插件

> 原文：<https://dev.to/fmeaddons/woocommerce-quote-plugin-1233>

[![](../Images/33f00c025721b47f6dedfe097c85d805.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Sr1hlx8A--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vflo1q341hxplcvsr800.png)

WooCommerce 报价插件启用产品页面上的添加到报价按钮，允许客户在您的电子商店上询问报价。如果他们需要不同规格的产品，他们可以通过产品页面上的报价表进行报价。WooCommerce 报价插件为你提供了一个简单的后端来管理报价，你可以很容易地回答它们。它有助于店主增加销售，因为你可以销售具有特定属性的产品，而不是一般的产品。这个插件只花了你 19 美元。
