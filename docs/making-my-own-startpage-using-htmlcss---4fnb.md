# 使用 HTML/CSS 制作我自己的起始页

> 原文：<https://dev.to/gauravchaddha1996/making-my-own-startpage-using-htmlcss---4fnb>

本周我从 Github 学生包中的 Thinkful 前端开发课程中学习了 HTML 和 CSS。我一直想拥有一个很酷的浏览器起始页，所以我继续前进，用我学到的 HTML/CSS 自己编写了一个。我花了大约 5-6 个小时来创建我的第一个起始页。这是它的样子。

[![](../Images/77d2d996cbb865031aa4dceb9f70c277.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CVmvyvKD--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/f4in3yj6pt9i1qw3lqjz.png)

我还必须部署一个小型服务器来为 Firefox 提供起始页，因为我希望链接看起来比“file:///”更好。所以一些快速的谷歌搜索会导致一个小的启动服务器脚本。然后，我使用该服务器的地址和端口作为我在 Firefox 中的主页。
