# 使用 Bael(NetlifyCMS 博客主题)的十大理由

> 原文：<https://dev.to/jasperketone/top-10-reasons-to-use-bael-a-netlifycms-blog-theme-4i6>

[![](../Images/1d0c2246ab83f6cf7675262e400fb2c7.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--EcX_Qtbp--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ybm05inq9i0uipok125x.jpg)

Bael 是一个免费的模板，给你一个简单的方法来开始一个使用现代技术的博客，比如静态站点 JAMstack 架构、CSS 网格布局、响应式设计、使用 Sendgrid 处理电子邮件时事通讯注册的无服务器功能，以及模糊搜索——所有这些都包裹在一种野兽派美学之中。

Bael 使用 [Nuxt.js](https://nuxtjs.org) 、 [Vue.js](https://vuejs.org) 、 [Netlify CMS](https://netlifycms.org) 运行，由 [Netlify](https://netlify.com) 托管。Bael 需要 Netlify 和 Github/Gitlab 的帐户才能部署。需要一个免费的 Sendgrid 帐户和 API 密钥来设置电子邮件简讯注册。

[![](../Images/2615398149205e6fb55f5e9ad4653e82.png)T2】](https://i.giphy.com/media/1pmgKY7Wu7te0/giphy-downsized.gif)

**使用 Bael 的十大理由:**

1.  **它很快** — Bael 作为静态生成的单页应用程序运行。在第一页加载后，后面的页面不需要整页加载。

2.  **渐进式网络应用** ( [PWA](https://developers.google.com/web/progressive-web-apps/) ) — Bael 保存其 CSS/JS/HTML 和图像资产，以实现更快的加载、更可靠的离线体验，以及安装在 Android 主屏幕上的能力。贝尔在谷歌灯塔 PWA 审计中获得了[完美 100 分](https://pwa-directory.appspot.com/pwas/5710428913532928)。

3.  **响应式** —该设计使用 CSS 媒体查询和网格布局将自身调整到任何设备显示尺寸。

4.  **内置搜索** —轻松找到任何博客文章。live search 会查找拼写错误的单词，并在您键入时立即更新搜索结果。

5.  **电子邮件注册表单** —使用无服务器[注册表单](https://bael-theme.jake101.com/blog/2018-07-08-email-newsletter-setup-instructions)，为你的时事通讯收集电子邮件。由[发送电网](https://sendgrid.com)和[供电网络功能](https://www.netlify.com/docs/functions/)。

6.  **JAMstack** —使用 [JAMstack](https://jamstack.org/) (一种基于客户端 JavaScript、可重用 API 和预建标记的现代 web 开发架构)构建网站的三大好处之一就是更好的性能、更便宜的托管和更高的安全性。

7.  [**一键安装 Bael**](https://app.netlify.com/start/deploy?repository=https://github.com/jake-101/bael-template)**——在 [Netlify 的托管平台](https://www.netlify.com/features/)上轻松安装自己的博客。你需要一个 Netlify 的账户和一个像 Github 或 Bitbucket 这样的代码库服务。**

8.  可定制的 —在你博客的管理部分设置你自己的网站图标、页脚菜单、链接和其他选项。作为开源软件，你可以自由地修改代码以供你自己使用或者为了每个人的利益。

9.  **野兽派设计** —功能性和简约性，Bael 是对当今网页设计过度友好和过度趋势的回应。

10.  **它是免费的** — Bael 是 100%免费的。主题是免费的， [CMS](https://www.netlifycms.org/) 是免费的，Netlify 的虚拟主机是免费的，Sendgrid 对 2000 名用户免费，Github/Bitbucket 提供慷慨的免费计划。[今天就试试](https://app.netlify.com/start/deploy?repository=https://github.com/jake-101/bael-template)。
