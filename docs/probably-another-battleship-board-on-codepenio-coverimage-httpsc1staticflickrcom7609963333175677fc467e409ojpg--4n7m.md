继续我的 [2018 年决议](https://dev.to/nickytonline/2018-resolutions-1deo)今年做代码笔，这是我最新的。

如前所述，我想开始制作代码笔的原因之一是想看看没有 JS 的帮助我能做多少事情。在这个例子中，Javascript 仅用于构建白板的标记，因为我懒得全部手工完成。还有一个小功能来模拟一个游戏移动作为一个错过/击中。

我一直在做 [@wesbos](https://dev.to/wesbos) 的牛逼 [CSS 网格课程](https://cssgrid.io)，所以我觉得用这些新技能再造一个经典的桌游《战舰》会很有趣。我确信 Code Pen 上可能有成吨的这种东西，但老实说，我不在乎。做这个很有趣。仍然有一些调整要做(当我在手机上看它时，游戏棋子的中心是关闭的)，但是到目前为止，我对它很满意。

也许在某个时候，我会制作完整的游戏，但现在，一个有一些模拟命中和未命中的棋盘。
