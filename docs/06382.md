# 用于 Oracle 的 dbForge 数据库工具得到了彻底的改造

> 原文:[https://dev . to/devart software/db forge-database-tools-for-Oracle-received-an-extreme-make-3 pke](https://dev.to/devartsoftware/dbforge-database-tools-for-oracle-received-an-extreme-makeover-3pke)

Devart 引入了针对 Oracle 数据库的 dbForge 工具的重大更新:模式比较、数据比较、比较包和数据生成器。升级后的工具在功能和视觉上都有了很大的改进。

数据库工具和数据连接解决方案的捷克软件供应商 Devart 发布了针对 Oracle 数据库的 dbForge 产品的显著改进版本:模式比较、数据比较、比较包和数据生成器。以下是更新后的工具中实现的最突出的改进:

dbForge Schema Compare for Oracle 的新版本现在提供:
-25+个新的模式比较选项。
-全新的模式同步选项，包括对表、视图、过程、函数、触发器、包和 Java 源代码使用 DROP 和 CREATE 而不是 ALTER 的能力。

更新后的 dbForge Data Compare for Oracle 包括以下特性:

-数据比较过程得到了改进，增加了一系列新功能，允许指定正则表达式以通过掩码包含/排除表或视图，并通过掩码忽略列；忽略空白、排除新添加的对象、公差间隔等。
-列映射对话框现在具有比较和同步复选框，可以很容易地从比较和同步中排除/包括列对。
-数据同步过程得到了增强，能够检查并发更新并同步序列的当前值。

升级后的 dbForge Compare Bundle for Oracle 通过以下新功能使处理数据库更改变得非常简单:
-除了 HTML 和 Excel 文件格式之外，比较报告现在还包括 XML 和 XML for Excel 文件格式。
-支持 PowerShell
-能够设置和保存命令行执行文件(。bat)来运行例行的数据和模式比较任务

借助升级后的 dbForge Data Generator for Oracle，用户现在可以:
-保存自定义生成器
-键入或选择要在数据填充之前和/或之后执行的脚本。
-使用新的行分布模式
微调生成数据的方式(例如，按指定的时间)–创建一个批处理文件(。bat)并优化日常数据生成任务
