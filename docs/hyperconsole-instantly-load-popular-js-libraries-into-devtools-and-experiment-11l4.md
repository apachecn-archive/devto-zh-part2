# hyperConsole——立即将流行的 JS 库加载到 devTools 和 experiment 中。

> 原文：<https://dev.to/itsarnavb/hyperconsole-instantly-load-popular-js-libraries-into-devtools-and-experiment-11l4>

我觉得尝试不同的库太费时间了。

所以我做了一个工具，可以让你把任何库(在 cdnjs 上)加载到你的浏览器中，所以你可以用你的控制台来摆弄它。

需要几秒钟。[看看吧](https://hyperconsole.itsarnavb.me/)
