# 热情

> 原文:[https://dev.to/lucpattyn/passion-361j](https://dev.to/lucpattyn/passion-361j)

我经常走在街上，思考我真正热爱的是什么。今天，我撞见这辆停在巴纳尼巴扎尔(位于孟加拉国达卡市)附近街道上的 1970 年代的丰田公共汽车，意识到激情不能只针对一件事。我们都对许多事情充满激情，有时“追求你的激情”可能是一个真正的误导。有些激情是有待发现而不是追求的。
