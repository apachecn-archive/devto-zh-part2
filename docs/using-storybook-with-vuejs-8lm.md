# 使用故事书和 VueJS

> 原文：<https://dev.to/auth0/using-storybook-with-vuejs-8lm>

Storybook 让我们可以交互式地开发和测试用户界面组件，而不必运行应用程序。因为 Storybook 作为一个组件库，有自己的 Webpack 配置，我们可以独立开发，而不用担心项目依赖性和需求。

在这篇文章中，您将学习如何通过使用 GitHub 上流行的看板渐进 Web 应用程序(PWA ),将 Storybook 与现有的 Vue.js 项目相集成，该应用程序由我的队友 Steve Hobbs 创建。新的 Vue 项目也可以遵循这个过程。
