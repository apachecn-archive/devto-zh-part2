        好像要讨论解决排序问题的插入算法，以及使用对算法设计的分割统治方法进行合并排序的开发和执行时间的分析。

        *   2.1 插入排序插入排序是希望解决排序问题( sorting problem )进行排序的数值...关键概念上是对列进行排序，但实际输入是带有 n 个元素的数组插入排序...在扑克牌中插入时左侧变小
            1.  初始条件:在循环即将开始执行之前，循环不变式为真
            2.  循环条件:在循环的第几次重复之前，如果循环不变式为真，则在下一次重复之前也为真
            3.  结束条件:循环停止时，可以从不变式中得到证明算法正当性的有力性质
