# 无需编写任何代码即可获得#Hacktoberfest 衬衫

> 原文：<https://dev.to/jerodsanto/get-your-hacktoberfest-shirt-without-writing-a-lick-of-code-1kcb>

Hacktoberfest 又来了，我们开发者会做*任何事情*来获得一件免费衬衫。

然而，我们内心有一场战争。*获得免费赠品*的倾向与*超级懒*的倾向相左，留下许多疑惑...**“去那个甜蜜的地方最简单的路线是什么？!"**

[![Good news, everyone!](../Images/467333c36126c2e22a55ab599cdd9dfa.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--GlYxNdfg--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/reh4m9wd9ptr51rne52r.jpg)

我们有*两个*回购，您的贡献在功能上相当于点击 Hacktoberfest 简单按钮:
