# 做开发人员需要计算机科学学位吗？

> 原文：<https://dev.to/_patrickgod/do-you-need-a-computer-science-degree-to-be-a-developer--247n>

这是永恒的争论。如今学位真的有必要吗？

简短的回答是:没有。

长的那个:看情况…

但是让我解释一下。

第一，你想成为什么样的开发者？一个软件开发者，游戏开发者，网页，应用，还有别的吗？

有很多可能性，但无论哪种方式，你的旅程都可能非常相似。

我确实学过计算机科学。例如，我获得了学士和硕士学位，一直想成为一名游戏开发者。

然后，事实上，我在一个游戏开发工作室找到了一份工作，在尝试了其他行业后，我现在是一名后端 web 开发人员。我喜欢它，但是我仍然喜欢在业余时间自己制作游戏。

如果我今天必须决定，如果我应该再去一次大学，我不确定我是否真的会这样做。

毕业后的那段时间教会了我一件事:如今有太多的可能性。

首先，你可以试着找一份实习工作来了解某份工作。或者如果你已经确定，你可以申请学徒。

除此之外，还有很多很多的训练营，在那里你可以在几个月内学会软件开发的技巧，而不是几年。

但是让我们看看学习计算机科学的优势。

你会有一个总体的印象。你可能从汇编开始，然后学习 C、C++、Java，甚至可能是 C#。你也会学到很多理论和数学。您还可以参加其他有趣的课程，如 IT 安全性、设计模式等，以获得更深入的理解。

但是你可以用另一种方式学习。

对我来说，计算机科学学位主要是进入科技行业的入场券。

你有硕士学位？太好了！你应该已经知道这个和那个，但最重要的事实是，你能够在几年内完成一个项目！这一点至关重要。

几年来，你能够专注于一件事，一个大项目，甚至完成它。在那个行业里每个人都需要的技能。

这表明你有耐心和毅力专注于一个项目。

例如，像谷歌这样的大公司似乎绝对希望你有一个学位。

不仅仅是任何学位。你需要完美的成绩，而且你最好是在一所著名的大学学习过。但即使这样，他们也会用几次编码面试来测试你。

毕业后，都是你的经历，你的推荐人。

对我个人来说，这是一个很大的优势，我出版了一个独立游戏，并制作了一些在线课程。人们不需要和我说话就可以看到我的技术。

好玩的事实，一个邀请我去面试的公司，居然买了我的一门课，甚至还修完了。这种感觉太棒了。

所以拥有这些公开的参考资料会有很大的帮助。我接触过的所有招聘人员都喜欢它，因为，当然，这样他们更容易说服潜在雇主参加面试！

而且，拜托，作为一个雄心勃勃的开发人员，你总是有一些未完成的附带项目在你的硬盘上积灰。完成他们，并向世界展示他们！

想象一下:你在学校或在家实习时做的一个副业项目获得了巨大的成功。看到这一点的正确雇主不会在乎你的学位。他们已经知道你有这个能力！

所以自己决定吧。计算机科学学位是这张入场券，如果时局不佳，也许它还能为学位提供一些保障。

但不幸的是，这要花费你几年的时间，也许还要前期投资。但话说回来，这也可能是对你自己和你未来的某种投资。

走另一条路可能不那么安全，但你有机会更早进入这个行业，发展你想要的技能，而不需要其他东西。

这真的取决于你。不好意思！
