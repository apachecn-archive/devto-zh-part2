# 地理囚犯评论

> 原文：<https://dev.to/horia141/prisoners-of-geography-review-1nf7>

刚刚看完[《地理的囚徒——告诉你全球政治所有你需要知道的十张地图】](https://www.amazon.com/Prisoners-Geography-Explain-Everything-About/dp/1501121472/ref=sr_1_1?ie=UTF8&qid=1518083088&sr=8-1&keywords=prisoners+of+geography+by+tim+marshall)。标题有点长，但这本书并不长，而且很容易上手。这是《T2》对地缘政治的介绍，或者说世界地理如何影响政治和国家发展。与[《枪炮、病菌与钢铁》](https://www.amazon.com/Guns-Germs-Steel-Fates-Societies/dp/0393354326/ref=sr_1_1?s=books&ie=UTF8&qid=1518083244&sr=1-1&keywords=gun+germs+and+steel)或[《民族为何失败》](https://www.amazon.com/Why-Nations-Fail-Origins-Prosperity/dp/0307719227/ref=sr_1_1?s=books&ie=UTF8&qid=1518083258&sr=1-1&keywords=why+nations+fail) ( [我的评论](http://horia141.com/why-nations-fail-review.html))一脉相承。但是这两位试图强加一个统一的理论来解释为什么世界是现在这个样子，这一位有着更温和的目标，并且把地理看作是塑造当今国家的一个因素。这很好，因为不需要扭曲数据来适应模型。

这本书有十章，每一章都涵盖了世界上一个相当大的地理区域。依次是俄罗斯、中国、美国、西欧、非洲、中东、印度和巴基斯坦、韩国和日本、拉丁美洲和北极。每一章都以该地区的历史和地理概况开始。从那里，各种扣除。例如，如果某处有一个狭窄的直道，那么控制它并能够巡逻它就成为了一个战略目标。对东南亚或美国来说，马拉卡海峡或佛罗里达海峡就是这样。另一方面，如果有山脉或沙漠，那么这就是一个国家的自然边界。喜马拉雅山是中国和印度之间的缓冲带，索诺兰沙漠是美国和墨西哥之间的缓冲带。

我发现最有趣的是地理对国家发展的巨大影响。这是对《枪炮、病菌与钢铁》的呼应，但我感觉在这里强调要好得多。例如，欧洲有一个庞大的通航河流网络，这使得贸易变得更加容易，尤其是在公共交通和大型公路网络出现之前。另一方面，撒哈拉以南非洲有许多河流，但它们不适合航行——它们有许多瀑布和很大的水位差。交易发生的地点也非常重要。阿根廷和智利等南美国家有一些麻烦，因为它们距离北美和西欧的大型经济体太远。同上，可耕地和淡水资源的问题是非常普遍的，即使在过去两个世纪的技术进步中也是如此。可能需要一个国家 2%的人口来种植食物来养活每个人，但是对土地的需求仍然很大。

一章的内容有一半是献给过去的。但另一半是献给未来的，这也是我觉得这本书最出彩的地方。因为地理上的限制仍然会迫使国家以某种方式行事(同时还有其他限制)。一些预测的事情已经开始成形——对北极更激烈的争夺(这是由全球变暖造成的),东南亚各岛屿的主权问题，中东边界的重新配置从后殖民时代的边界到更多的社会地理边界等。它并不完美，像英国退出欧盟这样的事情会给整个系统带来麻烦(它们就是“为什么国家会失败”所说的偶然因素——具有巨大长期影响的随机因素)，但它是一个人武器库中的好工具。当然，过去十年的一些事件变得更容易理解了。

总的来说，这是一本很好的、内容丰富的读物，对于寻求建立一个世界运行方式的模型来说，无疑是一个福音。
