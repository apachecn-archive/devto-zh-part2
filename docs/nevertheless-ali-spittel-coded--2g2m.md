# 然而，阿里·斯皮特尔编码了

> 原文：<https://dev.to/aspittel/nevertheless-ali-spittel-coded--2g2m>

## 我开始/继续编码是因为...

我完全是靠运气开始编程的——在大学的一个学期，我有一门额外的课程，并且我参加了一门使用 Python 的计算机科学导论课。我爱上了它。最后我在班上表现很好，下学期我被要求去辅助教学。我决定辅修计算机科学，甚至尝试双修。

下学期，我又上了一门计算机科学课:C++中的数据结构和算法。这是一场灾难。我几乎每周都熬夜做项目，而我在班上的表现仍然很差。回过头来看，不仅仅是我-班上一半的人都放弃了，这不是一个非常令人鼓舞的环境。还是那时候，我觉得自己不擅长编码。我不干了。

那个夏天，我在找一份实习工作。那时，我正在为政治活动做一些简单的统计工作。一家公司问我以前有没有做过代码。我说我有一点经验。他们提出给我报酬，而大多数政治实习是没有报酬的。他们用 Python 对我进行了数据科学库的培训，我重新爱上了写代码。我从那里学到了 JavaScript、Ruby on Rails 和 Django。现在，我已经是我工作过的编程语言的两位数了！从那以后，我在同一家公司全职工作了两年。我现在教人们如何编码，这也真的很有趣！

我一直在编码，因为构建东西实在是太有趣了！对我来说，编码已经成为我工作之外的一个艺术出口，我真的很喜欢技术社区的大部分。

引用桑迪·梅斯的话:

> 我们这些工作是写软件的人非常幸运。构建软件是一种无罪恶感的快乐，因为我们可以用我们的创造力来完成工作。我们安排了我们的生活，两者兼得；我们可以享受编写代码的纯粹行为，确信我们编写的代码是有用的。我们生产重要的东西。我们是现代手工艺者，建造构成当今现实的结构，不亚于砌砖工或造桥工，我们有理由为自己的成就感到自豪。
> 
> 这是所有程序员都有的，从最热情的新手到明显疲惫的长者，无论是在最轻量级的互联网创业公司工作还是在最古板、根深蒂固的企业工作。我们想尽最大努力。我们希望我们的工作有意义。我们想一路玩得开心。

## 我是这方面的专家...

没什么-我其实无所谓！我擅长很多东西——教学、Python、CSS、设计等。但是，我在所有这些事情上都是新手！我是 2014 年才开始专业做编码的，很会捡新技能。我不认为我能称自己为任何方面的专家。事实上，我可以说我是一个多面手。我做过数据科学、前端、后端、教学和设计工作，但我不是这些方面的专家！我还只有 23 岁！

我真的希望科技行业发展到这样一个地步，人们可以承认自己不是专家、天才、忍者或摇滚明星，并把这变成积极的而不是消极的。我希望人们能够认识到还有更多东西需要学习，并喜欢这一点，而不是试图相互超越或假装比他们更专业。我擅长很多事情，我喜欢这样——我不需要成为任何事情的专家，至少现在不需要！

## 我想吹嘘一下...

我的博客和演讲工作！我给自己设定了一个相当艰巨的挑战:每周学习一项新技术，开发一个应用程序，然后根据我的经验写一篇文章。我在去年年底才开始这样做，但是我一直坚持这样做，并且从中获得了很多乐趣。

去年我也开始在会议上发言，我得到了如此好的回应，以至于我不得不开始拒绝机会(这有点苦乐参半)。我非常喜欢教别人，喜欢上台谈论我觉得有趣的事情。

此外，我的[作品集网站](https://www.alispit.tel/)重写非常有趣，它真的让我看到了我的创作潜力。

## 我仰望...

整个科技行业有如此多的女性——甚至有如此多的女性在职业生涯中处于与我相似的位置，或者比我更早接触编程。尤其是那些有工作经历、大学毕业后进入科技行业的女性。我知道在大多数程序员中，我知道我是最老的 19 岁就开始编程的人之一！

我也非常钦佩那些成就了 DC 科技界女性的女性。保持女性在科技领域的第一城市令人惊叹，许多人投入了大量(无偿)工作，使社区变得如此包容和广泛！

## 我对支持女性和非二进制代码的盟友的建议是...

两个快速提示(了不起的盟友为我做的事情！):

在科技活动中留意你的女性朋友。仍然有很多人在聚会和会议上变得很奇怪，科技仍然是一个真正由男性主导的领域(除非他们关注多元化)。如果你注意到谈话变得有点奇怪，也许试着加入谈话。

确保你不会意外淹没了女同事的声音或成就。如果你看到别人这样做，试着用你的声音来反驳。尤其是当你处于领导地位的时候。

## 我对其他女性和非二进制程序员的建议是...

要知道，人也可能是混蛋——不仅仅是你。那个发表奇怪评论的家伙可能也对其他女人发表过奇怪的评论。那个对你的员工评估做出性别歧视评论的老板可能也对其他员工做出了同样的评论。

在申请之前看看公司员工的照片——确保公司里还有其他女性，如果可能的话——在你将要加入的团队里。另外，试着和那些女人交朋友，对新来的女人格外友好。我(为数不多的)女同事对我的职业生涯和理智有着不可估量的价值！

## 离别的思念...

我知道我在这里对女性在科技行业的地位有点负面，但我真的看到这个行业变得越来越好，越来越包容。我喜欢编程和教人们如何编码，对我来说，这超越了技术的许多缺点。
