很长一段时间以来，我发现我周围的开发者讨厌 PHP，我仍然不明白为什么会这样。

我不喜欢 PHP 的一些东西，但这并不意味着它是一种糟糕的编程语言...我认为每种编程语言都有自己的目的，所以与其开始抱怨*“为什么 PHP 不适合我”*我认为我们应该花更多的时间来比较不同的编程语言，并选择实现您项目目标的语言。
