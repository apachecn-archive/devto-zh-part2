# 然而，雅克·施拉格编码了

## 我开始编码是因为...

当我上小学的时候，我爸爸回到了学校，这样他就可以成为一名网站开发人员。那年圣诞节，当我开始询问他如何制作网站时，他给了我他的 HTML4 参考书。我开始阅读，从不回头。虽然编程一直是我的一个热情爱好，但直到我在研究生院认识到政策领域对 web 开发有着严重的需求时，它才成为我考虑从事的职业。我很幸运能够为自己创造一个职位，在那里我可以研究我关心的主题，并构建很酷的 web 项目。

## 我最近克服了...

对分享我的工作和参与开发社区感到害羞。冒名顶替综合症是难以置信的真实存在，也是进入这个职业令人沮丧的一面。我希望成为我所在的社区中更积极的一员，而不仅仅是潜伏在一旁。

## 我想吹嘘一下...

我所在的团队。我们是智囊团中一个屡获殊荣的创意机构，正在制作一些非常酷的东西。我们也(几乎)是一个完全由女性组成的团队，这在科技和智库领域都是罕见的。我喜欢来上班，每天都被同事们的工作所激励，这给了我继续工作的勇气，即使当冒名顶替综合症的感觉变得无法抗拒时。

## 我给盟友的建议是支持女性主义者的守则....

大声说出你的支持，对你挣扎的事情表现得脆弱。当你怀疑自己是否有能力在这个领域取得成功时，知道还有其他人在看到 JavaScript 生态系统的状态时也同样感到失落(不过说真的——我*终于*掌握了 Webpack 3)意味着一切。
