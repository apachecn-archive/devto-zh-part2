# Firebase -表格和报告

> 原文:[https://dev.to/scottlepp/firebase-表格-报告-4ibh](https://dev.to/scottlepp/firebase---tables--reports-4ibh)

使用 FireDrill 以表格形式查看和报告 Firebase 数据。

在 [FireDrill](https://firedrill.co/) 中，我们可以从默认的卡片视图切换到表格视图。

[![](../Images/35eefdcf298c196c960e766931584464.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--MtV05trT--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/992/0%2A5f9F-v_N8OhzbdmD)

现在，我们可以看到带有排序和过滤的数据的表格视图。不错！

[![](../Images/d486c0460ab11a59a152d3f0712b8c44.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--oCeEGaUb--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/0%2AeDj1LAetCKUtB9Xq)

我们还可以调整我们看到的列，并显示一个总计页脚。

[![](../Images/cac6d08d42583d40dbaf48a598665355.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--GZ6wJW76--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/430/0%2A--bi41WwTW4cWcKl)

在取消选中**类别**并选中上面的**总计**后，我们不再看到**类别**列，我们现在看到的是**价格**字段的总和。

[![](../Images/90ba4558e1ee49cab32d7ed4227c7e10.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--qH_cTuLr--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/0%2AMUW41S1t8ghidF-L)

一份很好的鞋类总库存报告！让我们把它打印出来，这样我们就可以把它作为 PDF 文档分享了。

[![](../Images/3cc85909591f99f3e803f13894242455.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--VhIlmhDC--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/309/0%2AZTg1DUhBIhYO3Lnd)

我在要打印的报告的右侧看到了预览。我用的是 CutePDF 驱动程序，它允许打印成 PDF 格式，或者如果你想要一份硬拷贝，只需选择你的打印机。

[![](../Images/e4a4d4051f80bec7875872a88aaa1cb2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--er0cxJCi--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1019/0%2AkmsxdIZOKYrN_-aM)

这是我们的 pdf 报告！

[![](../Images/a1a7411db92ea4a766e6fc40923ff82d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--9jmR4jzb--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/0%2A03m2m3tRWfObCSIV)
