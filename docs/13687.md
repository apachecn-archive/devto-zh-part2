# Skiplist 的第一天

> 原文:[https://dev.to/dev3l/first-day-at-skip-list-41d3](https://dev.to/dev3l/first-day-at-skip-list-41d3)

有没有想过做自己的事？你内心深处一个未说出口的召唤，然而一股暗物质的力量阻止了你？追随它是可怕的，忽视它又是痛苦的。

两周前，我吃了红色药丸。
[Skiplist](https://www.skiplist.com/) 是我的新现实。

* * *

今天是我创业旅程的第一个*正式*日。我们讨论了当前的商业书籍、期望和产品理念。做了一些客户承诺。我遇到了一个新的优秀团队。我需要棒球卡。

* * *

我们生活在一个我们站在巨人肩膀上的时代。制作有思想的软件的能力比我们之前的任何时代都要容易。托管服务的组合方式使得现代产品的总拥有成本比传统的人工操作成本低得多。

自动化消除了成本。我们满怀信心地将*每个*测量的生产承诺交付到零停机、动态扩展的生态系统中。它是我们的宠物黑天鹅。我希望你能看到它！

我们不会成为蛮力雇佣军，相反 Skiplist 定位于培育下一代编码忍者。我有伟大的想法。我觉得自己疯了。我认为这很好。几天前的晚上，我睡觉时差点惊恐发作。

# 合作伙伴

安德鲁·沃尔夫，他的同类中的第一个，已经独自穿越黑暗一年多了。他是一匹无情的纯种种马。我所知道的唯一一个消耗知识达到(甚至超过)我的量的人。他是部分机器。我们自己的，凯洛·伦。他与源头合一，源头与他同在。

从一个完全不同的角度来看，我们的合作伙伴 JT·谭是一位当之无愧的产品大师。这个人看待事物的方式与众不同，他是一个深海探险家，将功能直接移植到用户的存在感中。他是缺失的一环。他是顾客的巫毒罗盘。我们的狙击步枪。

最后还有我自己。我谦卑地与传奇人物一起走过这段旅程。我渴望成为敏捷社区的思想领袖。我走的是技术工匠的道路。我想制造能与古代武士挥舞的灵魂承载武士刀相媲美的产品。我渴望让人们变得了不起。我想分享软件工程的秘密。
