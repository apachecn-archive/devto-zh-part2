# 用户体验 em Westworld

自从出道[西域](http://www.imdb.com/title/tt0475784/)以来，我对系列以及它是如何建成的很感兴趣。从一开始，照片就引起了我很大的注意，尤其是数据的显示方式，故事的发展方式，以及用来作为故事背景的所有分配。这里的提议是将[用户体验](//www.eusoudev.com.br/iceberg-de-ux/) ( UX)与我们如何看到它应用于西方世界的体验中进行比较。

上周，我完成了西方世界的第二季，HBO 系列，被认为是取代王位游戏的赌注。它由 Jonathan Nolan 和 Lisa Joy 开发，于 2016 年上映。在第一季取得巨大成功后，数月前我们获得了第二季首演的殊荣，这也在世界各国取得了很大成功。

就像第一季一样，这一季继续以非常引人入胜的方式展现出观众喜爱的样子，让观众从集的开头到集的结尾都保持警惕，在集的结尾也有这样的反应。![👇](../Images/aeba3976027c224cc841c5cd84b5ab92.png)[![👇](../Images/aeba3976027c224cc841c5cd84b5ab92.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--6k1xZB52--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.org/images/core/emoji/2.3/72x72/1f447.png)[![👇](../Images/aeba3976027c224cc841c5cd84b5ab92.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--6k1xZB52--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.org/images/core/emoji/2.3/72x72/1f447.png)![👇](../Images/aeba3976027c224cc841c5cd84b5ab92.png)

[![ux em westworld](../Images/34b8db6e6addfa98f0280ed9eb1ae688.png)T2】](https://i.giphy.com/media/xT0xeJpnrWC4XWblEk/giphy.gif)

该系列以 1973 年的同名电影为基础，由美国作家迈克尔·克赖顿创作和导演，同时也以其续集《未来世界》(Futureworld，1976)为基础。西方世界不仅仅是一个关于机器人的系列，而是一个关于人类行为及其缺陷的系列，关于我们如何对待我们的愿望以及我们的行为。当然，这是我最喜欢的系列之一，对于今后的故事发展，我的期望很高，尤其是在最后一季之后。

如果你不知道系列，我用几句话告诉你——我发誓不会给扰流板。Westworld 是 Delos 的主题公园之一，是一家拥有许多雄心勃勃计划的巨大公司，在其中提供了完全创新的、超出标准的体验，赋予客人在里面随心所欲的力量；它坐落在古老的美国西部，由人工智能机器人‘人工智能’居住，看起来完全像人类，但存在着作为客人的娱乐体验。

客人们去西方世界享受假期，体验沉浸其中的体验，这样他们就能体验到不同的情感，每个人在访问的背后都有不同的动机，故事由此展开。

从我开始意识到系列背景下客人的体验是多么重要的那一刻起，我就在第一季回到了那里，注意到了一些在用户体验的宇宙中有意义的点。

## delos 的经验

除了是人气电视剧之外，还有收录在< per | | | >轻蔑的场面和行为中的 UX 元素问题是--通过简单地将机器人称为宿主，给每个机器人添加独特的服装、语言、外观和情感等细节，使机器人人性化。

一个例子就是一个人物不知对方是否是□时问:□，因为经验是如此接近人与人之间的互动，以至于看起来不像人与机器之间的互动，而她反过来回答‘你不知道’，这

```
“Você é real?” “Se você não sabe, isso importa?” John P. Johnson - HBO
```

客人们反复出现的主题把‘作为人’混为一谈，说明了西方世界为了确保作为客人，你完全沉浸在自己的经历中而走了多远。奇怪，但作为一个 UX 领域的专家，他们做了我们每天都要做的事情:那就是尽可能让我们的用户体验更加真实。

## 的行程

当一位新客人来到西方世界时，他会被护送到一个房间，在那里他们会选择要带去公园的物品。下面的 gif 显示了威廉，一位新客人被带到这个房间里，他可以从中选择他想参加哪种类型的叙述。

在决定戴哪顶帽子时决定了他在公园里的行程他决定了威廉在西方世界时想成为什么样的人。[西方世界的白帽子和黑帽子](http://variety.com/2016/tv/news/westworld-recap-chestnut-1201883126/)远不止是美学上的选择。

虽然有点重，但选择背后的比喻很清楚。如果威廉选择白帽子，那么他就是英雄。如果他挑一个黑人，那他就是个坏蛋。根据西方世界的网站，一位客人选择的帽子□你打算怎么选

```

                                 Sala dos chapéus
```

在这一选择之后，逗留准备开始。摘下帽子，∞乘火车到公园中央，从此便开始体验独特的生活。

```
 Chegada em Westworld
```

从那时起，客人沉浸在自己的故事里，在那里他将被带去体验他走进这个公园时提出的体验。
