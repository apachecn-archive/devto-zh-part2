### 自我介绍！你是谁？你在哪里工作？

嗨！我是克里斯蒂娜·巴兰。我是 Shopify 的应用安全工程师，在我们的多伦多办公室工作。

### 是谁或什么让你进入编程和科技行业的？

我从很小的时候就对计算机感兴趣。电子游戏是我的“入门毒品”。谢天谢地，我的父母非常鼓励我对电子游戏的热爱。我两岁的时候，他们让我开始玩世嘉 Master 系统，不久之后，他们又让我开始玩电脑游戏。作为一个每周跳舞 30 个小时的内向孩子，电脑是治愈我“内向宿醉”的良药。我喜欢 90 年代后期的一个叫尼奥宠物的游戏。这有点像电子鸡遇到了某种儿童社交网络。你可以定制你的宠物的“主页”，所以我开始学习 HTML 和 CSS 来做到这一点。我很快对 Neopets 失去了兴趣，爱上了写代码，开始自学 PHP 和 Javascript。我在网上有一些非常棒的导师。特别是一个叫克莱尔的女人，几年前就有了自己的托管网站。我从她的工作中受到了启发，并找到了其他一些女性，她们为任何对建立自己的网站感兴趣的人提供托管空间。

### 你是怎么过渡到保安的？那个空间让你兴奋的是什么？

我对安全感兴趣已经有很长一段时间了，可能可以追溯到高中时代。我有一个独立的博客，有人在上面留下了非常不恰当的评论。我设法追踪他们的 IP 地址到我学校的一台电脑上。一点社会工程学帮助我找出了谁应该对此负责，我能够和他交谈并阻止这种行为。不幸的是，我的 CompSci 项目并没有真正提供真正的计算机安全课程；许多基础知识被融入到其他课程中。我一直认为，为了专注于安全，我需要注册一个硕士项目。那成了我的终极目标:追求硕士兼职，最终转行。然而，我了解到，实际上有一些学院和大学提供的计算机安全课程，有些甚至可以在网上找到。我参加了几个研究生证书课程:一个是斯坦福大学的高级计算机安全课程，另一个是瑞尔森大学的数字取证和密码学课程。在完成这些工作的同时，我很高兴能够转到应用安全团队中的一个职位，这样我的日常工作就更符合我的总体职业目标。

这个空间有很多让我兴奋的地方，但我认为我最感兴趣的是对个人的保护。我们生活在一个联系越来越紧密的世界中，我们听到了婴儿监视器被黑客攻击和陌生人对蹒跚学步的孩子说不恰当的话的故事。这种权力和对个人生活的访问是不可接受的，我想成为解决方案的一部分。

### 你会如何建议某人“闯入”计算机安全行业？

不要因为没有某种头衔或学位而踌躇不前。开始尽可能多的学习，黑进应用程序(合法的)，参与一个 bug 赏金项目，确保你能够证明你理解你正在学习的东西！大多数安全团队人手严重不足，因此，如果你能展示出热情、毅力和能力，你应该能够找到一个职位，这也将有助于你继续成长！

### instagram 上一个著名的(~10k)粉丝对你的职业生涯有什么帮助？

我的 Instagram following 在过去的 6 个月里才真正发展起来，但它在建立网络和促进与该领域其他人的介绍方面非常棒！

### 迄今为止，在你的软件生涯中，你学到的最难的一课是什么？

这绝对是为了防止不安全感和自我怀疑阻碍你。我与“冒名顶替综合征”做了很多斗争，并且非常接近拒绝机会，因为我认为我根本没有成功的希望(Shopify 几乎就是其中之一！).我仍然在与它斗争，但我发现这几乎是一个普遍的斗争，与他人讨论它真的是一种宣泄。

### 对于成功的软件职业生涯，你的第一条建议是什么？

永远不要停止学习。我们的行业变化如此之快，虽然不可能成为每个领域的专家，但了解相关问题和技术非常重要。我真的不相信每个人都需要回到学校或者以某种结构化、制度化的方式继续学习；这恰好是我学得最好的方式。我们有如此多令人难以置信的可用资源——免费在线课程、该领域专家的博客文章、聚会、黑客马拉松、会议等等。如果你的公司不支持你的职业发展，他们就违背了他们的最大利益。如果他们确实为职业发展提供支持，确保你(负责任地哈哈)利用它来避免阻碍你的成长！

### 当你对某人做哑剧编程时，你是用霸王龙的手臂还是摆动的手指？

总是霸王龙手臂，但最好是作为打字猫 gif。

### 工作之外，你有什么爱好吗？你认为它们对你的科技事业有任何帮助吗？

我弹了 17 年的钢琴，虽然我现在已经生疏了，但我仍然努力练习。我在大学结束了我的舞蹈生涯，但我用跑步、骑自行车和瑜伽取代了它。我也参与了多伦多的素食主义者社区。我目前正在学习普通话，我认为这可能是对我的职业生涯最有益的爱好。

### 你会推荐什么书/资源？

我打算无耻地推销我的同事彼得·雅沃斯基的书《网络黑客 101 》。这是一个真正神奇的资源，转发是由 bug bounty 网站的创始人 Hacker0x01 撰写的。我还推荐[网络安全简易捆绑包](https://www.humblebundle.com/books/)。它们只是偶尔可用，但数量很大。这些书以 pdf 格式呈现，非常便宜，一部分收入捐给了慈善机构。我也觉得动手实践真的很重要。大多数大城市都有夺旗比赛，网上也有很多。有机会真正尝试入侵一个易受攻击的 web、移动或桌面应用程序是一个巨大的挑战(并且不会导致逮捕！).在谷歌上快速搜索“在线 CTF”或“你所在城市<>的 CTFs”会返回许多不错的选项！
