## 为什么 Python 适合数据科学？

Python 是成熟的，从书籍到在线课程都有大量的可用资源。它有一套重要的数据科学库可供使用。它是一种现成的编程语言，具有不同的包，用于加载和处理数据、可视化数据、将输入转换为数字矩阵或实际的机器学习和评估。
