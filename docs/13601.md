# 借助更新的 dbForge Studio for MySQL，实现对 MySQL 数据库更快、更可控的管理

> 原文:[https://dev . to/devart software/achieve-faster-and-more-controlled-management-of-MySQL-databases-with-updated-dbforge-studio-for-MySQL-4pk 7](https://dev.to/devartsoftware/achieve-faster-and-more-controlled-management-of-mysql-databases-with-updated-dbforge-studio-for-mysql-4pk7)

数据库工具和数据连接解决方案的捷克软件供应商 Devart 发布了其通用 MySQL 和 MariaDB GUI 工具的显著改进版本，即 dbForge Studio for MySQL v . 7.4。dbForge Studio for MySQL 中最突出和值得注意的更新包括:

1)支持 Google Sheets，便于在 MySQL 表和 Google 电子表格之间迁移表数据。这个特性被集成到了便捷的数据导出/导入向导中，用户现在唯一需要做的就是指定 Google 电子表格的 URL/ID。

2)通过执行通知和事务提醒以及突出显示当前代码行，改进了 SQL 文档管理。

3)增强的代码完成，支持几十个新变量和函数(Percona 服务器变量、MySQL 服务器变量、MariaDB 服务器变量等)。)

4)模式/数据比较和同步向导都有一个新的进度窗口，其中列出了正在进行的进程和经过的时间。还实现了一个全新的模式比较和同步引擎。

5)现在，dbForge Studio for MySQL 允许通过命令行界面无缝复制数据库，并提醒用户数据库复制的结果。

dbForge Studio for MySQL 是一个用于 MySQL 和 MariaDB 数据库管理、开发和管理的通用 GUI 工具。IDE 允许您创建和执行查询、开发和调试存储例程、自动化数据库对象管理、通过直观的界面分析表格数据等等。

要查看 dbForge Studio for MySQL v.7.4 中实现的更新的完整列表，请访问:[https://www.devart.com/dbforge/mysql/studio/](https://www.devart.com/dbforge/mysql/studio/)。
