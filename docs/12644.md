# 继续教育——我们行业的一个独特问题

> 原文:[https://dev.to/martyhimmel/continuing-education——我们行业的一个独特问题——2bmg](https://dev.to/martyhimmel/continuing-education---a-unique-problem-with-our-industry-2bmg)

我最近的一些想法。

考虑这些面试问题(同一个问题的变体):

你在做什么来继续你的学业？

或者

如何掌握最新技术？

从表面上看，它们是相当标准的问题。在任何行业中，人们通常都希望保持消息灵通，继续学习，并在他们所做的事情上做得更好。

似乎很合理。除了科技行业之外，从来没人问过我这个问题。

让我倒回去一点。我做了 15 年的牙科技师——制作假牙、牙冠、牙桥、金属框架等。我在工作中学到了这个行业的一切。这是从比我更远的人那里学习，做我所学的事情，以及在别人没有答案的时候想出答案的结合。但这一切都发生在工作时间——从来没有在我的私人/空闲时间。

在科技行业，我们经常被期望在下班时间跟上潮流和不断变化的技术。然而在其他行业，你要么在工作中学习，要么在工作时间接受培训。

行业的专业人士不应该期望个人付费的培训(如在线课程)、个人项目或下班后见面成为我们培训的标准。如果一个人*想要并选择*去做那些活动，那就不一样了，但这不应该是一种职业期望。

回到上面的问题——它们甚至不应该是面试问题。就像我说的，在科技行业之外，从来没有人问过我这个问题。人们通常认为你是在工作中学习的。

任何在工作时间不给员工提供培训/学习机会的雇主，都是在伤害他们的员工，进而伤害他们的产品和客户。如果我们的行业(作为一个整体)明白了这一点，那么这个问题可能会被遗忘，这是理所应当的。
