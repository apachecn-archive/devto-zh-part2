# 装满水桶

> 原文：<https://dev.to/devno/fill-the-bucket-29c>

我的第一个团结游戏。游戏的目标是把球放进桶里。它使用物理学，你可以在地图上放置物体来改变球的路径。我试图保持简单，并尽快发布，因为我们都知道你可以在一个简单的游戏上花费任何时间。

我发布了 iOS 和 Android 版。

[下载装满水桶](https://pecumo.com/fill-the-bucket.php)
