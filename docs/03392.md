# 如何在一个小镇远程工作而不死于尝试的 4 个技巧

### 原本在[培养基上](https://medium.com/@nicolasjengler/4-tips-on-how-to-work-remotely-from-a-small-town-and-not-die-trying-b5e4ad0a2c72)

在一个小城镇远程工作有时甚至比在一个大城市更难，这就是为什么一个好的支持系统和一些在线朋友可以带来很大的不同。

我已经远程工作了大约 8 年。当我住在父母家时，我的第一份工作是在我的卧室里用 HTML 和 CSS 从一个. psd 文件得到一个新的网页设计(顺便说一下，我非常不适合这份工作)。

如果你从未做过内部员工或自由职业者，远程工作通常在最初的几个月或几年更容易，因为你从未体验过与同事并肩工作的动力，或有社会规范指导的预定义的例行公事，这就是为什么你会发现自己不知道办公室里共存的习惯。例如，如果你缺乏找到动力和灵感的能力，或者缺乏设定工作时间的自主性，远程工作有时会很困难。但如果你住在大城市，如果在一个共享空间工作是你前进的动力，或者如果你需要一些外界压力，让你穿上除睡衣以外的衣服，那么你有共同工作的空间会很棒。遗憾的是，小城镇通常没有共同工作的空间(至少在第三世界国家是这样)，住在一个小城镇里完成你的工作和成为一个经过几天或几周的数字流浪者是完全不同的，这就是为什么我决定列出这个清单。

虽然我在成为远程前端开发人员/产品设计师的旅程中有 6 个月的间隙(这值得另发一篇文章)，但我工作生涯中大约 96%的时间是在一个人口约为 45，000 人的城镇中完成的。以下是一些如何在偏远的小城市生活时尽可能高效和积极的方法:

1.  共用一间办公室或习惯咖啡店:
    如果远程工作一段时间后，你开始怀念办公室里的人际关系和闲聊，我建议你找一间愿意租你一张桌子的办公室，或者去一家小酒吧，把自己变成常客。
    我个人租用了办公室的一部分，几个朋友要经营他们的打印店。我们分担一些费用，可以说，我在家里有了一席之地。和人在一起绝对有助于我避免独自在家的恐惧，它也迫使我走出去走几个街区，呼吸一些新鲜空气，甚至可能在这个过程中跑跑腿。

2.  **如果你找不到当地的社区，就在网上搜索:**
    正如你可能想象的那样，一个欠发达国家的 45000 人的城市并不是一个设计师和开发商云集的地方，这就是为什么在这里找到一个社区非常困难。因此，我发现自己与一些以前的同事建立了一个在线社区，并且还使用了一些空间，如 [Toptal](https://www.toptal.com/) 或 [Designlab](https://trydesignlab.com/) Slack teams。有时我甚至和一些朋友用 Skype 聊天，我们每个人都在做自己的事情，同时聊天并寻求反馈，这是一种很好的感觉，有人陪着你一起做类似的事情，同时建立联系。

3.  **参加网络研讨会:**
    在你所在的城市，聚会和集会可能会发生几次，或者完全不存在，这就是为什么你应该习惯网络研讨会，以便实时获得最前沿的培训和教育。我还建议组织聚会，让镇上的人知道，你可能是唯一一个参加聚会的人，但有时其他不知名的同事或希望进入这个行业的人可能会出现，并意识到他们并不孤单！

4.  确保你的工作空间或家庭办公室状况良好，界限分明:
    我认为，如果你在一个偏远的地方工作，你可能也想利用在自己的地方工作的优势，因此你需要有一个合适的工作空间(或一间办公室，如果可能的话，在你的家里)，充分装饰，并与你所在地方的其他房间分开，配备完成工作所需的所有工具。这有助于记住这是工作发生的地方，也有助于保持专注而不被你重要的其他人或家人分散注意力。

就是这样！这就是我在远离雇主总部的地方工作时，一直坚持到今天的原因。虽然这些并不是揭示生活的小贴士，但它们看起来很明显，我并没有意识到它们中的大多数会提高我的工作质量，改善工作和生活的平衡，以避免精疲力竭，所以我把这些留在这里，让其他可能处于我位置的人来发现。

我真诚地希望这能让你在工作中找到一个更好的位置。直到下一个帖子！
