## [](#what-do-you-do)你是干什么的？

我是一家科技公司的全栈开发人员，该公司为抵押贷款行业开发基于云的软件。

在此之前，我在一家电子商务公司工作。我在 15 家以上的 Magento 商店工作。当时我是做灯栈的。从那以后，我的注意力转移到了更新的前端 JavaScript 技术上。

## [](#why-did-you-start-coding)你为什么开始编码？

写代码是我的艺术形式。我一直是有创造力的类型。我从来不会画画，但我总是有想法。大学一上网络编程课，我就迷上了。我终于可以在脑海中创造出我所想象的东西，并让它看起来和功能完全符合我的想象。

人们可能不认为开发人员很有艺术细胞，但编程实际上是一种极具创造性的职业。这是基于逻辑的创造力。

我喜欢每天和这么多聪明的人在一起。我们是创造者。我们可以把想法变成现实。我们可以改变未来。这个领域的可能性是无穷的，令人振奋。

## [](#what-are-you-currently-learning)你目前在学什么？

我目前正在使用带有 React/Redux 的 TypeScript。TypeScript 在短时间内取得了很大的进步，我真的很喜欢到目前为止我所学到的东西。

我也开始使用艾娃进行单元测试。我希望很快能更多地了解 React Native 和 PWA。

## [](#have-you-ever-felt-challenged-as-a-female-in-this-field)作为一名女性，你在这个领域有没有感受到挑战？

诚实的答案是肯定的。直到我进入职业生涯后，我才体验到这一点。一开始，人们很高兴能给我一个机会，因为我是一个新鲜有趣的人。

当我开始申请更高级的职位时，事情发生了一些变化。人们对信任我更加紧张。我与众不同，充满风险。有些人喜欢他们觉得舒服的东西...但是聪明的人都有天赋。

如果你觉得因为你的性别，你没有得到公平的对待，那就继续前进。如果他们看不到你，别人会看到的。我向你保证。在这个行业里，有太多的机会等着你被低估自己的价值。

## [](#any-advice-for-other-women-or-nonbinary-folks-in-stem)对 STEM 中的其他女性或非二进制乡亲有什么建议？

不要害怕说出来，做你自己！我怎么强调这一点都不为过。当你向世界展示你的实力时，大门会为你敞开。

给所有性别的开发人员一些建议——分享你的想法并与其他人合作。不要害怕未知。挑战自我。和比你聪明的人在一起，向他们学习。保持你的激情！
