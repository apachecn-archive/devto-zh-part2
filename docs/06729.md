# 如何在 Android 上开始 Web 开发

网络开发是一个趋势性的职业道路，有大量的就业机会。大多数人认为用电脑是不可能进行网络开发的。今天我们将了解如何在您的 Android 手机上开始网络开发。

> Android 是基于 Linux 内核的，所以它绝对有可能做我们在 Linux 电脑上能做的任何事情。

如果你没有台式机/笔记本电脑，但想开始网络开发，你并不孤单，几年前我也在同样的地方。那时候我有一部老式 J2ME 手机，但现在大多数人都有智能手机。如果你手里有一部 android 手机，脑子里有创造力，那就是我们用我们的 Android 手机开始网页开发之旅所需要的一切。

[![Coding](../Images/6aea99ed58f386ef2e3539acc9774e7c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--N4QhSSrG--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://media1.tenor.com/images/505ddb5e0b0e8c3e96b66e1469ef47c1/tenor.gif%3Fitemid%3D4903969)

我把这个指南分成了几个部分，所以对新手来说很容易。

话不多说，我们开始建吧。

## 第 0 部分:学习 Web 开发

> 如果你已经学习了 Web 开发，你可以跳过这一部分，进入第 1 部分

如果你查看 Play Store，你会发现很多学习 Web 开发的解决方案，但是很少有真正有用的。

所以今天我想出了一个 Android 上 Web 开发的有用资源列表。

### ·solo learn:

这是我最喜欢的学习几乎所有广泛使用的编程语言和网络技术的方法之一，比如 HTML(它不是编程语言)，JavaScript，CSS 等等。

它有更多有用的功能，比如如果你有问题可以提问，向世界展示你的代码等等。

[**从游戏商店**安装](https://play.google.com/store/apps/details?id=com.sololearn)

### 【蚂蚱:

如果你想学习网络开发，你必须学习由谷歌开发的 T2 JavaScript，这是学习 JavaScript 的最佳资源之一。

您可以学习基础和高级 JavaScript，还可以了解有用的 JavaScript 库，如 Grasshopper 的 D3Js。你还可以通过使用画布来学习动画等等

[**从游戏商店**安装](https://play.google.com/store/apps/details?id=com.area120.grasshopper)

### 【Py

类似于 SoloLearn，它有很多学习 HTML、CSS 和 JavaScript 的资源。

你也可以用 Py 学习其他语言

[**从游戏商店**安装](https://play.google.com/store/apps/details?id=com.py.learn)

## 第 1 部分(A):编码

如果你已经知道或刚刚学会了网页编程语言，如 JavaScript 和标记语言，如 HTML，你可以开始编写惊人的网站，网页应用程序。

这里有一些最好的 Android 代码编辑器，

### ·艾德网

AIDE Web 是最古老的 Android Web 开发解决方案之一。这是一个成熟的 IDE，有很多有用的功能。

它不仅是一个 IDE，也是学习 Web 开发的一个解决方案

*我给你推荐这个*

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.aide.web)

### ·快速编辑

这是目前最好的代码编辑器之一，你可以在里面编写 HTML，CSS，JavaScript 并预览编译后的 HTML

它支持大量的语言，非常轻便，只有 3.3 兆字节

这是我最喜欢的 android 编辑器

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.rhmsoft.edit)

### 【droide dit

这是一个轻量级的强大的 Android 代码编辑器，它支持许多编程语言，也支持直接从 Dropbox 打开文件。

您也可以在浏览器中预览 HTML。

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.aor.droidedit)

## 第一部分(B):非编码

> 如果你不是程序员，但是仍然想做 web 开发或者喜欢非编码的 WYSIWYG 解决方案，这部分是为你准备的，否则你可以跳过它

### 【sim diff

它是 play store 上最受欢迎的非编码网站构建器。数百万人使用

起初它可能看起来很混乱，但是如果你花足够时间，它会变得容易理解。

我不喜欢的一点是，它的用户界面看起来又旧又乱，如果使用平板电脑或大屏幕手机，这还可以忍受，但如果使用小屏幕手机，有时会令人头疼。

但是你应该试一试

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.simple_different.android)

### 【mobi rise

我经常用它来创建网站原型。它默认使用引导 CSS 框架。

如果你有一个大屏幕或平板电脑，它很容易使用，否则它可能是一种痛苦

它有利于创建投资组合，创业网站，产品页面等。

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.mobirise.mobirise)

> 如果你已经完成了你的网页建设，你可能想向全世界展示它，只要看看下面

## 第 2 部分:电话服务和电话测试

> 如果你没有自己的主机服务或服务器，或者只是想测试你的网站，不要担心，你可以从你的 android 手机上服务和测试你的网站

### 【PAW 服务器

如果你想从你的 android 设备提供网站服务，它是首选的服务器，但从 Android 提供网站服务会损害你的设备。

这是目前广泛使用的 Android 网络服务器之一。它有很多有用的功能。

它非常轻便，易于使用

[从 Play Storr 安装](https://play.google.com/store/apps/details?id=de.fun2code.android.pawserver)

*设置 Paw 服务器有问题，请遵循此* [*指南*](https://www.makeuseof.com/tag/turn-android-device-web-server/)

### 【Esminis.com】T2

这不是一个应用程序，而是一个由 Tautvydas Andrikys aka 开发的 Android 服务器应用程序列表。埃斯米尼。

该列表包括:

*   Apache 服务器
*   PHP 服务器
*   MySQL (MariaDB)服务器
*   Mongo 数据库服务器

它们是在 Android 上测试你的网站的首选和强大的方法。

[点击这里查看](https://esminis.com)

## 第三部分:上传到服务器

如果你有一个主机服务，你需要上传你的网站给全世界，上传文件给主机提供商的最好方式是 FTP(除非他们不支持 FTP)

### 【和 TP

这是最古老和广泛使用的 Android 客户端之一。

它简单、轻巧且功能强大

[从游戏商店安装](https://play.google.com/store/apps/details?id=lysesoft.andftp)

### 【ES 文件浏览器

这是一个适用于文件管理的解决方案。

它也可以用作 FTP 客户端。这是一个强大的解决方案

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.estrongs.android.pop)

### 【turbo FTP

这是一个快速和功能齐全的 Android FTP 客户端。它有现代的材料设计

[从游戏商店安装](https://play.google.com/store/apps/details?id=turbo.client)

### 连接 Bot

如果你想使用 SSH 从你的云上传或同步文件，使用它的 Android 开源 SSH 客户端

[从游戏商店安装](https://play.google.com/store/apps/details?id=org.connectbot)
