## 简介

过去，当我教 11 - 15 岁的孩子或年轻人编程时，总是很痛苦，因为没有多少可用的工具可以轻松使用 python。

## 在线编辑的崛起

首先，当你在教 python 的时候，你需要安装 python，这对开发者来说很简单，但是安装起来却很困难。

根据操作系统的不同，无论是 Mac 电脑、T2 电脑、Windows 或 T4 Linux，在安装到不同版本的 python 时总会出现奇怪的错误。这导致仅仅在学生计算机上安装 python 就浪费了大量时间。

幸运的是，随着像 **Repl** 、 **CodeSandbox** 、 **Jupyter Notebook** 这样的在线代码编辑器的兴起。它有助于跳过上述问题。

## 问题与编辑

另一个问题在于编辑的选择。当你的教学初学者第一次学习编程或者已经从可视化编程语言毕业，像 **Scratch** 或者 **Blockly** 。

大多数代码编辑器有两种变体，第一种是极简方法，它是纯基于文本的，就像与 Python 捆绑在一起的 IDLE。

这不是真正的初学者友好使用，缺乏某些关键功能，如放大和缩小按钮。这在向一班学生展示代码时非常有用。

另一种方法是专业的，就像 **PyCharm** 或 **Jupyter notebook** 。

这些编辑器或者我们称之为集成开发环境或简称为 IDE，对于初学者来说并不真正有用，因为大量的视觉混乱和功能可能会让初学者在刚开始时感到厌烦。

## 什么是 Mu？

为了解决以上两个问题，Mu 在 Raspberry Pi 基金会的支持下，用一个星期天的时间开发了 BBC Microbit editor，并开发了一个新的 python 编辑器，用于向初学者教授 python。

他们根据树莓派基金会的学生和老师的反馈，创造了 Mu。

我是在听一集《和我说 Python》采访 Mu 的创作者时偶然发现的。

## Mu 如何帮助初学者教编程？

除了上面提到的问题，Mu 有一个对初学者有用的极简用户界面。

Mu 有许多有用的特性，可以帮助教学 python 编程。就像 checker 按钮给初学者一个更快的反馈循环，不需要老师的帮助。

通过有用的建议和易于理解的错误消息来指出他们的代码有什么问题。

可视化调试器对于大多数专业 ide 中的调试目的非常有用，它允许学生在一行一行的代码执行中查看他们的代码有什么问题。

最后，绘图仪绘制了一个折线图，这在与传感器交互或处理数值以在图表上显示数据时非常有用。

## 视觉游戏与电子游戏开发

Mu 支持各种微控制器，如用于电子或机器人项目的 BBC Microbit、Raspberry Pi 和 Adafruit 板。

除此之外，游戏开发也可以通过 Mu 的模式之一进行。一个游戏开发库，没有样板代码来创建 python 游戏。

## 结论

在使用 Mu 之后，我的最初印象让我意识到这是一个向初学者教授 python 编程的极好的编辑器。

因为它减少了对教师和助手的过度依赖，通过使用 checker 和可视调试器来帮助学生首次克服编程的绊脚石。

其易用性和与微控制器的集成可与 Arduino 相媲美。

对于教师和学习者来说，将 python 代码烧入微控制器而无需多个步骤是天赐之物。

最后，Mu 可以从拇指驱动器上运行，这对于想要自己练习的学生或者在学校环境中工作的教师在计算机上安装程序是非常有用的。
