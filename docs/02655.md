# 什么是数据绑定？

> 原文:[https://dev.to/flippedcoding/what-is-data-binding-ghn](https://dev.to/flippedcoding/what-is-data-binding-ghn)

是时候学习另一个概念了，这个概念其实很简单，但是人们把它变得更复杂了！数据绑定是任何 web 开发项目的重要组成部分。事实上，它是如此重要，以至于它负责从你在屏幕上看到的内容到你数据库中的更新。这就是为什么是时候学习什么是数据绑定了。

数据绑定将用户和数据源(通常是数据库)联系起来。当您在网站上提交表单时，数据绑定将您的信息发送到数据库。例如，当您需要在某个网站上重置密码时，您可以提交一个包含您的电子邮件地址的表单。它会对数据库进行某种调用以获取您的用户信息，然后您会被带到另一个页面，在那里您需要输入新密码，这将更新数据库。

它看起来像这样:

这就是数据绑定的全部内容。它将网站的前端连接到后端服务器。使用数据绑定有一些巨大的优势。基于用户与页面的交互方式实时更新页面是可能的。数据绑定的另一个优点是，它减少了您必须编写的代码量，因为您不再需要查找元素并手动获取它们的值。

您可能还会看到性能的提升。这是因为您的代码被简化为使用数据库中的信息。你不需要做那么多的数据库调用，这样可以节省你大量的时间。

我想提到的最后一点是，数据绑定不一定要涉及数据库。您的数据源也可以是数组或集合，这取决于您所使用的框架。
