# 黑客马拉松的项目想法(第一次参加)

> 原文：<https://dev.to/anku/project-ideas-for-a-hackathon-going-for-the-first-time-g9n>

我是印度的一名 CSE 大二学生。下个月有一个由 AngelHack 组织的黑客马拉松，我想参加。

他们要求我们建造一些东西来解决社会或环境问题，并对我们的当地社区产生积极影响。

我将和我的两个朋友去参加黑客马拉松。我有在 MERN 堆栈(1 年)的经验，一个家伙知道 ML(没有很多经验)，另一个做了前端工作。

如果我们的项目需要，我们准备好学习新的东西(这是我们的目标)。因为这是我们第一次举办黑客马拉松，所以我们不确定要建造什么。

如果你们能给我们一些建议，这将是一个很大的帮助。谢了。
