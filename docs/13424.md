我想做一些简短的视频教程或者讲解。你想了解什么？以下是我认为可能适合一个简短视频的一些主题:

*   如何阅读技术书籍
*   如何评价你在网上找到的技术资料，或者，为什么栈溢出上传最多的答案可能不是你需要的答案
*   觉得 VS 代码很棒？有一个 Javascript IDE 至少和代码一样强大，并且支持一个杀手级的特性
*   如何训练你的猫不在你的键盘上走
*   保持一个编码笔记本
