# [](#dev-showcase-page-with-bulma-css)用布尔玛 CSS 开发展示页面

一个[开发者展示页面](https://bulmadev.chkrishnatej.com/)展示他们的项目和博客。

使用[布尔玛 CSS](https://bulma.io) 展示页面被设计成鲜艳的颜色。瓷砖和卡片展示你的工作和博客文章。

## [](#how-is-it-built)是怎么建成的？

*   整个项目是用布尔玛 CSS 和一个小 javascript 为响应式导航菜单栏构建的。

*   少数几个功能的小代码片段来自不同的开发人员，他们都得到了应有的认可。

## [](#how-to-use-it)怎么用？

*   使用此链接
    `https://github.com/chkrishnatej/bulma-dev-page.git`克隆回购

*   适当地更新名称和链接。

*   在您的本地机器上测试它。

*   在服务器上提供服务。

## [](#github-pages)Github 页

Github pages 允许你免费托管你的静态站点。要设置 Github 页面，请点击[链接](https://pages.github.com/)。

你甚至可以为你的 Github 页面设置一个自定义域。遵循 github 页面中的[自定义域](https://help.github.com/articles/using-a-custom-domain-with-github-pages/)
