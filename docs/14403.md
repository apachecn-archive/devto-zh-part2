# 南希编码，编码，再编码

> 原文:[https://dev . to/Nancy d/Nancy-deschenes-coded-and-coded-and-coded-some-more-7hn](https://dev.to/nancyd/nancy-deschenes-coded-and-coded-and-coded-some-more--7hn)

我有一段时间一直想发布我的介绍，还有什么比今天更好的时间呢，2018 年 3 月 8 日。

# 从前...

这一切都开始于很久以前。我一如既往地好奇。我喜欢用我的积木和砖块建造东西。我喜欢破解密码，阅读神秘书籍(*著名的五本*有人知道吗？).我安静害羞，从不远离书本。我第一次接触电脑是在我去我爸爸工作的地方，那里的终端连接着一台功能强大的电脑。他向我展示了如何引导一只乌龟在屏幕上画出形状。嗯，与其说它是一只乌龟，不如说它是一只箭，但是当然，让我们假装它是乌龟。后来，我爸爸给家里买了一台电脑，我开始尝试更多的东西。有时，他会建议我做一个特别的练习(“写一个程序，可以计算出正确数量的 25 美分、10 美分、5 美分和 1 美分来表示一个数量”)，有时，我只是想知道，“如果我……”

那些年，我大部分时间都和女生在一起。我有姐妹，去了所有的女子学校，玩花样滑冰，参加女童子军。我不记得有人告诉我数学是男生的。我可以亲眼看到我在这个话题上做得很好，所以如果有人提到它，我没有注意。我看书，玩电脑，不停地学习我手头的任何东西。

# 在离家很远的大学里...

起初，我并不认为我会以编程为生。我想上的大学只有计算机工程学位，我没有考上工科学校的成绩。我被 BSc 项目录取了，但是我不知道我对哪个专业感兴趣。物理？化学？生物化学？我从数学开始。当我有更好的主意时，我可以随时改变。甚至可能转到工程。"你为什么不攻读计算机科学的学士学位？"我的顾问建议道。我怎么会错过这个选择呢？这是该计划的第一年，总的来说，有点混乱；有些课程是由理学院开设的，有些是由工程学院开设的。那时我发现了互联网、聊天程序、新闻组、电子邮件、调制解调器！他们甚至有一个装满 NeXT 机器的实验室！那些是如此光滑的机器，编程如此有趣！界面生成器允许你在应用程序窗口上放置 ui 组件，通过在两者之间拖动一条线来将事件与控件挂钩，这太有趣了！每当我设计一个 UI 时，甚至是今天，我都希望 NeXTSTEP 界面生成器简单易用。好吧，我可能会有点怀旧，我可能会通过粉红色的镜片回忆起那些岁月，但谁能说得清，那是很久很久以前的事了！我不能说我注意到了性别歧视。当然，我们寡不敌众，但我没怎么注意。

有一次，另一个学生，一个我以前没怎么注意过的人，让我把作业给他，让他抄下来。我拒绝了，他很生气。这个回答不适合他，他说，“如果你再对我不尊重，我会打断你身上的每一根骨头。”附近的其他学生毫不客气地把他送出了学生休息室。那是我最后一次见到他，但其他女学生也报告了类似的遭遇。我有点动摇，但与此同时，我看到了别人是如何介入帮助的。他显然有问题...

# 我找到了一份工作

出了校门，该找工作了。我很幸运，我知道另一所大学有一个职位空缺。我被雇佣了。为了跟上速度，我很快学会了 COBOL 和 SQL。在业余时间，我做了一个聊天程序，所以我不会忘记如何用 c 写。然后，为了好玩，我学习了 HTML。我发现了 PERL 和 C 语言的 CGI 编程，并开始忙于制作东西。和一个朋友一起，我们用 PERL 和 db (Berkeley DB，一个键值数据库)编写了一个最早的内容管理系统。我当了一年左右的系统管理员。现在我知道那不适合我；我大部分时间都在编程！

就在那时，我开始从事全职咨询工作。我在家工作，自由支配时间。我可以每周五下午去滑冰，但我也会工作到深夜。我和这位朋友一起改进了内容管理系统，开发了各种 Java 小程序游戏，开发了一个可以判断用户何时登录网站的小程序，编写了一个浏览器内聊天系统，开发了大型数据库、小型数据库，先用 PERL，然后用 Java。

我不断地学习和编码。开发和构建系统，找到新的方法来做旧的事情，只是更好。

# 和现在...

我还在学习。我是 myTurn.com 的技术联合创始人，这是一个出借图书馆、出租和资产跟踪的平台。我仍然在家工作。每天早上，我让我的三个孩子起床上学，做午饭，推着他们，这样两个可以乘公交车的孩子就不会错过了，哦，等等，太晚了，开车送他们，第三个去学校，然后开始工作！或者也许我会去我的另一个办公室，在星巴克，或者我的另一个办公室，早餐餐厅不介意我在那里呆几个小时，或者...

我喜欢灵活性，喜欢挑战，喜欢不断地为我们的客户和客户的客户做得更好。我喜欢玩新技术，在新概念上拓展我的思维，然后发现如何应用这些概念来提高我的工作质量。我和以前一样好奇，我尽可能多的阅读，小说，非小说，技术类的...我都吃光了。当我有时间的时候。

我还在滑冰。我现在是童子军领队(男孩和女孩)。我想有些事情是不会改变的。
