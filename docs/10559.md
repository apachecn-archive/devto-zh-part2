# Ticketfly.com 数据泄露

我收到了[的通知，我被告知](https://haveibeenpwned.com)我的名字、电子邮件地址、邮寄地址和电话号码被公开在网上，作为涉及 2600 万账户记录的[ticketfly.com](http://ticketfly.com)数据赎金的一部分。[主板报道](https://www.billboard.com/articles/business/8458918/ticketfly-warned-hacker-vulnerability-website-down)攻击者要求 1BTC(约 7500 美元)的赎金。不清楚这一请求是被有意忽略还是被忽视。从理论上讲，保护我的客户记录需要 ticketfly.com 花费 0.000288 美元。现在黑暗网知道我总是买票去看他们也许是伟人。

# [](#would-i-pay-the-1btc-ransom)我会支付 1BTC 赎金吗？

众所周知，勒索者永远不会停止索要更多的钱。同样可以肯定的是，仅仅因为黑客没有立即公开转储数据，她/他仍然会在黑市上出售这些数据。

然而，这是一个低成本的赎金要求。支付赎金可能会赢得修复漏洞的时间，或许还能让 ticketfly 保持在线。我被告知，场馆不能购买，出售，或验证本周末音乐会的门票。这可能会给每一个售票场所造成超过 7500 美元的损失。

我会付赎金的。
