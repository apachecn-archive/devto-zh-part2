# 你最初是如何开始学习编程的？

> 原文:[https://dev . to/Andy/how-you-first-start-out-learning-programming-4o 26](https://dev.to/andy/how-did-you-first-start-out-learning-programming--4o26)

我的朋友开始自学 Python，作为一名开发人员，看到有人对这个领域感兴趣总是很棒。当然，我仍然有点怀疑他是否会继续学习，但目前为止他自己做得很好。他从 Codecademy 开始，但发现它并没有真正深入掌握。然后他尝试了 YouTube 课程，还不错，但不知什么原因也不太喜欢。在那之后，他开始玩 PyGame，这似乎是他最喜欢的游戏，但他也尝试了 Pluralsight，因为他获得了三个月的免费试用(我认为还不错)。现在他在跟踪奥丁计划，或者类似的计划。

无论如何，这让我想知道其他人是如何开始学习的，特别是因为我不能真正给他提供关于尝试什么的最佳建议。我学习编程的经历和他的不太一样，而且我有机会接触到(例如，编写 bootcamp 代码。)那他没有。

你进入编程的第一步是什么，你现在知道了什么，你希望你做了什么？
