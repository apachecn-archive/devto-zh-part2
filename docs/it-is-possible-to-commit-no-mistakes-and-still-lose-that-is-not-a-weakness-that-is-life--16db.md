# 不犯错误仍然失败是可能的。那不是弱点，那是生活。

> 原文：<https://dev.to/rommik/it-is-possible-to-commit-no-mistakes-and-still-lose-that-is-not-a-weakness-that-is-life--16db>

一年前，我在找一份新工作，每周做一次面试。我并不急于换工作，所以我想找到一家完美的公司，在那里我可以接受挑战，发展新技能，同时获得丰厚的报酬。即使在蒙特利尔这样的大型 IT 市场，这也不是一个容易的组合。

一家公司对我的简历感兴趣，邀请我担任高级网站开发人员。他们在一个很棒的位置！蒙特利尔市中心，地铁(我们称之为 metro)出口就在他们的大楼里。初步面试是与人力资源，我们涵盖了文化，产品，组织。结构、福利和薪水。有一个相互的和专业的火花。我被邀请参加该团队的第一次面试。

我遇到了两位资深开发人员，我们愉快地交谈了一个小时。我们谈论了我过去的经历、公司和他们的产品、他们使用的技术堆栈、单元测试的重要性，当然还有一个关于设计模式的问题。一切都很顺利！最后，我们握了握手，寒暄了几句，然后我就回家了。

我对面试并不陌生。我真的很喜欢它们，并且在过去的几年里做了很多！这有助于我保持敏锐的技能，让我很好地了解目前的市场状况，有哪些机会，以及潜在雇主的要求。当然，有一种更简单的方法可以做到这一切...堆积如山的年度调查、市场统计以及关于如何通过面试的书籍或教程。不过，要想在面试中表现出色，最重要的是在令人畏惧的薪资谈判中表现出色，唯一的方法就是练习、练习、再练习！

在回家的路上，我在脑海中回放过去的一个半小时，分析已经说过或没有说过的话，如果我忘记提到一些事情或过度分享，我就能够以不同的方式呈现自己。总的来说，我对这次会议很有信心。我一回到家，就给人力资源部门发了一封感谢信，让他们发给面试官。我甚至还附上了一个链接，链接到了我们之前讨论过的一篇关于微服务架构的文章。第二天，我接到了一个跟进的邀请，这次是和部门经理。

哦，真幸运！原来，这位经理就是我目前在公司任职的那个人。我甚至坐过他的旧椅子(当时他还是个开发者)。我有很多故事要告诉他，他也很乐意分享一些内部笑话。我们笑着互相点头。任何有助于与面试者建立融洽关系的事情都会对你有利。正如许多采访专家所说。尽管如此，我还是决定把那些关于不得不处理代码库和灾难性架构的沮丧留给自己，这些都是在这个家伙离开公司后留给我的团队的。这不完全是他的错，但他是高级开发人员...但是我跑题了。

所以，我在复试。这将是一个技术问题，我会写代码，三个面试官会看，我们会讨论它。这个练习是阅读一个文件中每行的单词，并找出是回文的单词，这个单词可以向前或向后拼写相同，就像 madam 或 kayak 一样。这需要使用测试驱动程序开发来完成——编写一个测试，让它失败，编写代码，然后让测试通过。简单，最重要的是，我前一天刚刚完成了一个回文教程，它仍然历历在目。我告诉他们我的意外过度准备，并提出将练习改为其他方式。他们拒绝了。编码部分非常简单。我一边写一边聊天，通过了测试。我做过的最简单的编码会议。接下来的几个问题，我的手开始颤抖，我带着非常自信的感觉回家了。这是我做过的最好的采访。它在我的脑海里完美地出现了。又一封感谢信，我在等着那份工作...

两天后，我收到一封回信，上面写着“...很遗憾，您没有通过我们高级解决方案开发人员职位的测试。你做得很好，但并不伟大，他们正在寻找一个强大的编码能力的开发人员。再次感谢您的宝贵时间。”哇！真的吗？我惊呆了！他们给合适的候选人发邮件了吗？我仔细检查了邮件，是的！的确，是写给我的！没有错...我被拒绝了。哎哟！

我以前被拒绝过，虽然很受伤，但拒绝是最大的教训，我真的相信这一点。我总是会问一个礼貌而尖锐的问题“为什么？”。大多数招聘人员都乐于建设性地批评他人，而其他人只是不予理睬，不予回复。不过，这次我破例了。我甚至没有用一句“谢谢”来确认收到了邮件。这是废话！好但不伟大？他们挑了个回文题！这是他们在编程 101 里教的东西！我气了几天。让我难过的不是拒绝，而是“好但不伟大”的评论。你如何从中成长？奥运选手很好但不伟大(那些没拿金牌的)。他们我理解，但是程序员呢？

一年后，我想起这家公司，自嘲。他们拒绝的原因是什么并不重要！也许我的编码真的很糟糕，或者他们找到了更好的候选人。我的反应是错误和愚蠢的。我想从这次经历中吸取的教训很少。

1)不要让情绪和骄傲妨碍你。

2)在你签署录用通知之前，你不会被录用。不管面试有多棒。我见过有人在真正拿到下一份工作之前就大张旗鼓地辞职了！

3)你对一个人/一种情况的看法可能是错误的，你正在错过一些东西。如果你意识到了这一点，那就进行调查，了解为什么你对最初的印象是错误的。我应该继续问我的标准问题“为什么？”尤其是在这种情况下。

4)失败的面试是了解自己新情况的机会。他们定义了我的职业生涯，帮我找到了我真正喜欢为之工作的公司。

5)不犯错误仍然失败是可能的。在这种情况下，你振作起来，继续前进，到别处去试试运气。没关系！

求职愉快！
