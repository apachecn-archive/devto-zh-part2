# 你是如何跟踪优秀的资源的？

> 原文:[https://dev . to/Laura gift 21/how-do-you-keep-track-of-awesome-resources-2e8d](https://dev.to/lauragift21/how-do-you-keep-track-of-awesome-resources-2e8d)

很多时候，我在网上找到很棒的资源，但大多数我当时真的不能阅读或浏览。所以我想到的是，要么把它存到口袋里，要么把链接加入书签。

老实说，由于一些我不知道的不可预见的原因，我很少回那些链接。所以我想到了一个主意，为什么不在 GitHub 上创建一个 repo，并将其开源，这样你就可以保存你找到的那些链接，并把它们放在几个类别中。酷的是其他人也可以为这个列表做贡献，这对我来说是一个双赢。

这让我创建了这个名为[牛逼学习资源](https://github.com/lauragift21/awesome-learning-resources)的项目。我在 10 个月前开始将我在网上找到的很棒的资源添加到这个回购中，到目前为止，我的列表上已经有超过 200 个资源，我倾向于现在检查这些资源，然后感谢其他贡献者为这个回购贡献其他资源。

那么我为什么要写这个呢？T3】

我的意思是，只要能帮助你实现目标，所有的想法都是有价值的。这个很棒的列表是一个我可以随时查看的列表，在任何时候都可以找到很酷的资源。我甚至有一个名为“病态选择”的类别，在那里我记录了我在网上发现的很棒的东西。网络上充满了令人兴奋的东西，我不可能一目了然地全部消化，所以我创建这个列表的原因之一是作为所有这些伟大资源的避风港。
