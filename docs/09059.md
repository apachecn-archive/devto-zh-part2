# 结论

我认为每个演讲者都有自己的过程，你会发现你的过程。对我来说，我知道我的工作过程中的一些重要的事情。例如，我第一次演讲时，感觉糟透了。任何演讲都有有限的内容，我需要在面对至关重要的听众之前提取出来。此外，当我有听众时，我倾向于复习时间，所以我故意把我的演讲写得比时间段短 5-10 分钟。

当你走下去的时候，你会发现你自己的过程，但是记住，当你走上舞台的时候，感觉自信和准备好的最简单的方法就是准备好你自己的标准。
