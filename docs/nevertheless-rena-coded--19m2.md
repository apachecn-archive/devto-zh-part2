# 然而，瑞纳编码了

> 原文：<https://dev.to/fbkrena/nevertheless-rena-coded--19m2>

### 科技还是艺术？=S

我一直对技术感兴趣，在高中时我选择了一所大学，在那里我将有一个电信和计算机科学的专业，但我并没有完全找到自己的位置，有些东西并没有让我感到完全舒服，我觉得我需要一个艺术补充。

回顾过去，我不太确定是否仅仅是我的这个想法让我去寻找另一条路，或者是那些不停地问你科学是否是一个女孩合适的职业的声音...

因此，在一个复杂的重新开始和高中毕业后，我报名参加了美术学院纺织专业的一所学校，如果我说我不喜欢我的职业生涯，我会撒谎，但我一直在寻找一种更有效的方式来做技术支持的事情，有有趣的结果和对电子纺织品实验的关注。毕业后，我被选中参加了一个名为“专注于艺术的技术挪用”的研讨会，这让我觉得自己处在一个完美的地方，创造力、艺术、技术、逻辑思维，我所热爱的一切的理想交汇点，“技术虫”比以往任何时候都更加强烈地觉醒了，不仅仅是用户的想法深深地植入了我的脑海，我开始寻找更多的课程和研讨会，于是 [Laboratoria](http://www.laboratoria.la/en/impact/) 出现了。

### 什么是实验室？

代码学院，可能在其他国家更常见，但在墨西哥不太常见，更没有像他们这样的项目。我让自己做好了前端开发人员的准备，参加了为期 6 个月的强化训练营，不仅包括技术准备，还强化了软技能和敏捷纪律；从一开始，他们就打破了我对开发人员的许多不自觉的刻板印象，始终作为团队(小队)的一部分工作，寻求加强我自己和我同事的能力，这是这个实验室只培训女性的重要一点。想知道我为什么邀请你去他们的网站看看。

### 姐妹情谊

因此，在这六个月里，我成长了，我哭了，我笑了，我在 50 位非凡、才华横溢、美丽动人的女性的陪伴下学习了代码和其他对我的生活至关重要的因素；我们从坐在同一张桌子上的陌生人转变为加入更大的姐妹会，因为我们是墨西哥城的第四代，但该计划也在秘鲁和智利运行:)

### 现在...

我是一个技巧，一个创造者，有时用线或用代码线在其他人身上绘画，我喜欢两种方式，现在我与一个不可思议的团队合作，为一个寻求帮助世界各地妇女的组织，我有一个惊人的机会帮助和影响他人的生活，最好的事情之一是有时我在我最喜欢的睡衣上这样做，因为我完全远程工作，好吧，这不是最好的部分，但它是一个大优势。

永远不要想象这些代码会如何改变我的生活，一年前，我第一次在一个小型的黑客马拉松中出现在谷歌的办公室里，那是我第一次面对一个黑客马拉松，我们没有考虑体验，仅仅是单词...即使在今天，仍有让我害怕的挑战，还有很多事情要做，要学习，但我很幸运能在一个总是愿意指导我的团队中，一个支持我的姐妹会，是我支持、鼓励和陪伴每一个有需要的女孩和妇女的努力的一部分。

加强女生联谊会，让越来越多的女性增强自身能力，这是代码赋予你的超能力之一，当我们让代码按照我们想要的方式工作时，我认识的所有开发人员都达到了超能力的程度，在这一点之前可能会有一千次失败，但这种感觉无法用足够的语言来描述。

### 致所有支持女性和非二进制编码者的盟友....

我邀请你们成为榜样，让我们向其他妇女和女孩展示我们如何做出改变，以及多样性对更好的结果有多么重要，但这只有通过让我们被看到才能实现。

如果你没有一个例子，相信你能实现它会更复杂，所以让我们加入我们的声音和代码行。因为女孩们喜欢代码，我们像摇滚明星一样做。
