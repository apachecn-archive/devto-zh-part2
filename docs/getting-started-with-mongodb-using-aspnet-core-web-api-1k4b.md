# 使用 ASP.NET 核心 Web API 开始使用 MongoDB

> 原文：<https://dev.to/matheusrodrigues/getting-started-with-mongodb-using-aspnet-core-web-api-1k4b>

> [你可以在这里下载代码](https://github.com/mrodrigues1/MongoDB-WebApi)

最近，我试图进入 NoSQL 数据库，更具体地说，进入文档数据库。

经过一些研究，MongoDB 是我的首选。MongoDB 是一个众所周知和流行的文档数据库，它存储类似 JSON 的文档。它非常容易访问，有支持主要语言和框架的驱动程序。拥有大量非官方驱动程序的强大社区支持。

目标是展示如何设置一个 MongoDB 实例，以及如何在 ASP.NET 核心中使用 web API 来使用它。

## **所需软件:**

*   [MongoDB](https://www.mongodb.com/download-center#community)
*   [Visual Studio 2017 社区版](https://www.visualstudio.com/downloads/)
*   [邮递员](https://www.getpostman.com/)

*我用 windows 10 创建了这个教程。
