# 成就解锁！获得所有 AWS 助理+从业者认证。

> 原文:[https://dev . to/David _ j _ eddy/achievement-unlocked-all-AWS-associate-practice-certificates-obtained-o36](https://dev.to/david_j_eddy/achievement-unlocked-all-aws-associate--practitioner-certifications-obtained-o36)

从我的博客上传的十字。

人生有目标是好事。努力的目标，标志着努力回报的里程碑，最重要的是个人努力成果的价值。作为一个目标导向型的人，它帮助我实现了我想要的生活目标。高中毕业，独立生活，完成大学学业(并毕业)，获得一份“真正的”职业，[在一个我的个人贡献可以产生重大影响的地方工作](https://www.sourcetoad.com/)，获得前 3 名提供商的所有云认证。

最后一个还没有发生，但今天我到达了路上的第一个里程碑:获得了所有四个 associate +从业者 AWS 认证。[云从业者](https://aws.amazon.com/certification/certified-cloud-practitioner/)、[解决方案助理](https://aws.amazon.com/certification/certified-solutions-architect-associate/)、[开发者助理](https://aws.amazon.com/certification/certified-developer-associate/)，最后是[系统运营助理](https://aws.amazon.com/certification/certified-sysops-admin-associate/)。我想说这很容易，但那是谎言。每天 2 到 3 个小时，每周 3 到 5 天，持续将近一年。加上培训材料的费用(感谢云专家[和 Linux 学院](https://acloud.guru)[提供的惊人资源)。](https://linuxacademy.com/)

浆糊年也不容易。离开了一个稳定的职位，开始了一个小规模的创业，我是一个团队的领导者，加入了一个不断增长的中档安全公司`dream job`，搬到了离所述工作更近的地方，不得不买一辆新的(呃)(二手)汽车，这花去了我很大一部分积蓄。然后三周后被理想的工作解雇了。这让我失业了近两个月，老实说，真的很沮丧。最终，我在埃森哲获得了一个新职位，协助一家排名前三的电信公司采用开发(秒)运营构建渠道。考虑到移居荷兰生活和工作的可能性非常真实，但却在最后一刻被否决，这令人心碎。

像任何体面和合理的投资一样，获得认证需要预先付款，一路上不断努力，并有耐心达到最终目标。许多晚上的社交错过了保持正轨；许多“有趣的时光”都是为了以后的回报而错过的。

在过去的 6 个月里，投资开始有回报了。比预期的要快得多。每年超过 20%的薪水增长，更多的享受我每天所做的事情，成为一个需求越来越高的群体中的一员。如果你了解市场，随着需求上升，价格也会上涨。在这种情况下，这意味着工资。更高质量的机会向我走来，最后，我完全真正意识到，我可以去任何地方，有体面的薪水和工作保障。不再是“我需要这份工作”，而不是“你希望我在这里”。

我知道这是最近的一个里程碑，是一个小小的成就，但是要想到达星星，你首先要建造一个发射台。
