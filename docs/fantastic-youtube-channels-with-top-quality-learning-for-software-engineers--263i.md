# 奇妙的 youtube 频道，为软件工程师提供高质量的学习

> 原文：<https://dev.to/perigk/fantastic-youtube-channels-with-top-quality-learning-for-software-engineers--263i>

不可否认，视频培训是最受欢迎的方法之一，对于自学成才的程序员来说，而且根据福布斯的调查，在千禧一代中最受欢迎。

> 任何停止学习的人都老了，无论是二十岁还是八十岁。任何坚持学习的人都会保持年轻。生活中最重要的事情是保持你的思想年轻。(亨利·福特)

据我所知，许多人依赖于一个或多个最受欢迎的视频培训平台，如 [Pluralsight](https://www.pluralsight.com/) 、 [Udemy](https://www.udemy.com/) 或 [Coursera](https://www.coursera.org/) ，但他们使用的账户是由他们工作的公司资助的。

但是对于那些没有那么多好处，也不能为此每年花费 300 美元左右预算的人来说呢？谢天谢地， [youtube](http://youtube.com) 正在托管相当多的频道，为软件工程师提供高质量的材料。以下是我的最爱，排名不分先后。

[![](../Images/baf4bb442424dd7cca5748eb5dff720e.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--C_-r8AhY--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/500/1%2AJtzEheXwoYY3vP0nKbR00A.jpeg)

1.  德里克正在提供一系列多样化的技术课程。从[组装](https://www.youtube.com/playlist?list=PLGLfVvz_LVvQu9IwUcpn8KOZsOvoHx8sU)和[设计模式](https://www.youtube.com/playlist?list=PLF206E906175C7E07)到 [Django](https://www.youtube.com/playlist?list=PLGLfVvz_LVvSMqZiTTsAC7C8Ypp81Jt6D) 、 [Rails](https://www.youtube.com/playlist?list=PLGLfVvz_LVvSngZQwrhYXlPnJf1zYqghI) 和 [UML](https://www.youtube.com/playlist?list=PLGLfVvz_LVvQ5G-LdJ8RLqe-ndo7QITYc) 。他极其清晰的口音让字幕毫无用处。我喜欢这个频道，因为它不是纯粹的技术，而是对开发人员生活中的软技能有很好的把握，比如[谈判](https://www.youtube.com/playlist?list=PL9028E98DC5D03FBD)和[销售](https://www.youtube.com/watch?v=6bNwBJ4dky8&list=PL61952EC6E585D073)技巧。
2.  拥有 4350 个视频的 thenewboston 在写这篇文章的时候，这个家伙在睡觉的时候真的在录像。像德里克·巴纳斯一样，thenewboston(或现实世界中的巴基·罗伯茨)也提供各种主题的教程。编程课程占据了该频道图书馆的大部分，但也有一些数学、生物和游戏课程。
3.  FreeCodeCamp talksFreeCodeCamp 是一个伟大的——也是相对较新的——活动，在这里，各种充满激情的人就现代技术话题发表演讲。你甚至可以把你的演讲送到那里发表。没有理由在这里浪费你的时间，因为有一篇很棒的[文章](https://medium.freecodecamp.org/introducing-freecodecamp-talks-awesome-tech-talks-you-can-watch-for-free-55e5d96e2fa5)回答了所有的问题。
4.  这个频道的所有者 Ramon Lopez 是一位伟大的软件工程师，我已经关注他一段时间了。最初，我被他的[系统设计](https://www.youtube.com/playlist?list=PLA8lYuzFlBqAy6dkZHj5VxUAaqr4vwrka)文章所吸引，但后来我也喜欢上了他的[职业建议](https://www.youtube.com/playlist?list=PLA8lYuzFlBqCnvkSNeM7AYIy9AkTVjEJs)播放列表。现在看起来暂停了一段时间，希望很快能刷新。
5.  Traversy Media 一个其权力与网络编程密切相关的频道。快速浏览他的播放列表会让你毫不怀疑。:)他主要是关注 Node。Js 和普通的 Javascript，但是也有一些 PHP、Python 和 CSS。这里我最喜欢的部分，是他创作的长视频，在那里你可以在大约一个小时内很好地掌握他所说的技术，就像[这个](https://www.youtube.com/watch?v=-zW1zHqsdyc)一样。
