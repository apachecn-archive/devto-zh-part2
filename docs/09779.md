# 比特币是什么？比特币的价格为什么这么高？

> 原文:[https://dev . to/wenso _ Smith/what-is bit coin-why-is bit coins-price-so-high-21md](https://dev.to/wenso_smith/what-is-bitcoin-why-is-bitcoins-price-so-high-21md)

最近，在绝大多数人当中，关于比特币有很多争论和问题。大约五年前，它首次被引入虚拟经济世界。从那以后，人们对比特币一直感到好奇和困惑。

在我们进入比特币及其飙升价格的细节之前，让我们先了解一下比特币的意义。

比特币只不过是加密货币，意味着虚拟现金，通常存储在网上钱包中。在线钱包，如 Paytm、Ola Money 等，您可以随时使用它们购买任何产品或服务。

比特币是最古老和最受欢迎的加密货币形式，其行为与经济中存在的任何货币一样，只是它不是有形的，并且是分散的。它不仅可以用于任何种类的购买，而且可以用作投资工具或用于交易目的。简而言之，它也被称为 BTD，类似于印度卢比的 INR。

它是如何获得如此多的流行仍然是未知的，或者就此而言，任何加密货币仍然是匿名的。尽管所有的交易数据仍然公开，但参与此类交易的人永远不会被披露。这就是为什么它吸引了很多试图逃税的人。

最近一段时间，比特币的价格大幅上涨，许多小玩家因此获得了巨额收益，而大玩家几乎成了百万富翁。然而，没有人知道这些收益是真实的，还是只是一个随时可能破裂的泡沫。

比特币的币值与其他货币的币值一样。这也是基于同样的规则，即人们愿意购买多少来交换它。为了通过比特币进行任何交易，要遵循的第一步是“挖掘”。这发生在一个计算机系统中，在该计算机中，一个具有 64 位数解的复杂数学问题被求解。每解决一个问题，就会处理一块比特币，作为奖励，矿工会获得一个新的比特币。

为了在虚拟经济中创造平衡，并应对计算机芯片不断膨胀的能力，谜题的复杂性得到了调整，这确保了每天产生的新比特币的稳定流入。目前，大约有 1600 万比特币存在。

有一些特定的比特币协议需要遵守，这使得比特币市场得以运转。从技术上讲，矿工最多只能创造 2100 万个比特币。

比特币的价格推动了这一主要理论，该理论认为，比特币将很快受益于大型投资机构的巨额资金注入，因为未来的 BTC 产品将允许投资者做多和做空比特币。

比特币价格上涨的另一个原因是，很少有投资者使用比特币作为存储价值，并且不以任何当前价格出售比特币。
