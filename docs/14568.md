## [](#i-begancontinue-to-code-because)我开始/继续编码是因为...

我小时候对电脑游戏很着迷。在我的游戏生涯中，我做了很多事情，从用基本的 HTML 和 CSS 建立我的枫叶故事公会网站，到编写一些可能被称为“不道德”的应用程序来帮助游戏中的用户。

## [](#ive-always-wanted-to-try)我一直想试试...

开博客！我知道这些日子它们一毛钱一打，但这是我很久以来一直希望做的事情。我对女性和少数种族/文化群体在科技领域的交集特别感兴趣。

## 我对自己编码的决定质疑最多的一次是...

当一个朋友告诉我，我可能已经找到了一份工作，因为我是一个象征性的少数族裔:女性和亚洲人。当时，我相信了我的朋友。当我开始工作时，我的冒名顶替综合症比以往任何时候都更加普遍；我质疑为什么我每天都被雇佣。不过，我很幸运，因为我公司的首席运营官是一位非常坚强、赋权的女性——她曾经说过一句话，我想我永远也不会忘记:“生活是艰难的，所以要尽可能利用一切。”虽然这可能会引起一些人的争议，但我把这句话记在了心里。我是否被雇佣为象征性的少数族裔无关紧要。我手里握着这个惊人的机会，我可以选择全力以赴，或者生闷气，质疑自己。我选择了前者。

## [](#my-advice-for-allies-to-support-women-and-nonbinary-folks-who-code-is)我对支持女性和非二进制代码的盟友的建议是....

当别人无礼时，不要害怕叫他们出来。有时很难为自己辩护，所以即使是像一个知道的、不赞成的眼神这样的小事也会被欣赏。
