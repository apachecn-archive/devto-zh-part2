呃，我被解雇了。是的，我在一英里外就看到了。事实是，在你的职业生涯中，你可能会被解雇几次，所以你应该做好准备，学会察觉它的到来。

第一次被解雇可能有点令人震惊。突然你每天去的那个地方已经不在你的生活中了。你醒来时无处可去，不知道下一步要去哪里。不仅仅是一个结束，也可以是你职业生涯下一阶段的开始。

过一段时间，当公司的情况变得糟糕时，你会有一种第六感。你可能会注意到其他部门的员工跳槽，或者你的经理突然不敢直视你的眼睛。突然，你在周计划会议上的 12 个要点在一段时间内停留在一个要点上，没有新的工作被分配。有很多事情可以告诉你末日即将来临。

如果之前有过一轮裁员，而你幸存了下来，那确实是开始寻找工作的好时机。注意公司是否为了试图自救而改弦易辙。如果没有，你可能要开始寻找或者至少做好准备。不是老生常谈，但是做同样的事情却期待不同的结果是疯狂的定义。

如果你知道你很有可能会被解雇，更新你的简历并立即开始存钱。把多余的钱存起来，尽量提前至少两个月的开销。如果你不介意跳槽，那就在你还在目前工作的时候开始找另一份工作。

无论你采取哪种策略，看看目前需要什么技能。利用你所拥有的时间来堵塞任何知识漏洞，并在 GitHub 上获取一些例子。例如，当我觉得我的工作岌岌可危时，我就加入了这一潮流。亚特兰大的每个人似乎都想改变他们现有的。NET 网站到 Angular。Udemy 是一个很好的学习资源，而且很便宜。

被解雇后，第一件要做的事就是告诉招聘人员你正在找新工作。如果你已经和招聘人员建立了关系，那么立即通知他们，并给他们发送一份更新版的简历。不要打开“我在找工作！”除非你真的需要一个新的招聘人员。你会被电话和邮件淹没。他们会坚持把你的简历寄到两个小时路程之外的地方，而且不需要在家工作。使用你过去用过的招聘人员也更好，因为你和他们有良好的历史，他们可以为你担保。他们大概已经知道*不是*要送你什么了。

如果你住在美国，尽快在第一周登记你的失业保险。即使你拿到了遣散费。在你的遣散费之外有一点额外的收入会是一笔不错的奖金。如果你被迫去你的劳工部，停下来犒劳一下自己，比如在星巴克喝杯咖啡，或者在某个地方吃个汉堡，然后休息一下午。

到这个周末，你可能已经安排了一些面试或者一两次技术筛选。你甚至可能会找到一些听起来像是加薪的“独角兽工作”。这种可能性加上几个备份，可以让你感觉更有动力，可以帮助你在周末放松。你正在做一些事情，你正在得到结果，现在你只需要让事情发生。

在不参加面试或不与招聘人员交谈的时候，练习你的技能，参加在线课程。做一个开放的项目，让自己专注于你的技能。晚上休息。确保你在工作时遵循同样的常规。不要熬夜和睡懒觉。如果你喜欢锻炼，坚持你的锻炼养生法。在你通常吃午餐的时候吃午餐。需要的时候，暂停学习/找工作。

在每个星期，除非你有一个面试，试着周五休息，给自己一个三天的周末。奖励自己努力找工作。

最棒的是当你得到那份工作的时候。现在你可以放松了。更好的是，当你的第一天是一天或一周之后，你在重新投入工作之前有一点休假时间。

当你像我一样在这一行干了 21 年以上，被解雇是开发人员职业生涯的一部分。公司来来去去，职位也是。所以，确保你做好了准备，以应对万一发生的事情。
