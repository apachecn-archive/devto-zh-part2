# 30 多年的计算经验

> 原文：<https://dev.to/lefebvre/30-years-of-computing--56c2>

为了好玩，这里列出了我这些年来用过的电脑:

*   雅达利 400(1983):我的第一台 16K 内存的电脑。我在上面学习了[的基础](https://en.wikipedia.org/wiki/Atari_BASIC)，有好几个月我都无法保存我写的程序，因为我买不起[卡带驱动器](https://en.wikipedia.org/wiki/Atari_8-bit_computer_peripherals#1010)！
*   Atari 800XL (1984):巨大的 64K 内存和内置 BASIC。我继续用 BASIC 编程，并添加了 [Pascal](https://www.atarimagazines.com/v4n7/kyanpascal.html) 和 [C](https://www.atarimagazines.com/v7n7/LightspeedC.html) 。加上我得到了一个 [180K 的磁盘驱动器](https://en.wikipedia.org/wiki/Atari_8-bit_computer_peripherals#1050)！
*   [Atari 1040ST](https://en.wikipedia.org/wiki/Atari_ST#STF_and_STFM_models) (1988):我上大学的时候买的这个。它有令人难以置信的 1MB 内存，超清晰的 640x480 黑白 CRT 显示器和 720K 磁盘驱动器。我主要是用 [GFA BASIC](https://en.wikipedia.org/wiki/GFA_BASIC) 、[个人 Pascal](http://www.atarimania.com/utility-atari-st-personal-pascal_29326.html) 和各种 [C 编译器](https://sourceforge.net/projects/sozobon/)编写的。
*   Atari Mega STe(1992):4MB 内存，16Mhz CPU 和 105MB 硬盘，这是一个精心设计的 Atari！这是一个伟大的系统，我希望我一直保留着，因为它在今天会很值钱。在此期间，我制作并销售了一些共享软件，分布在 [Delphi](https://en.wikipedia.org/wiki/Delphi_(online_service)) 和 [GEnie](https://en.wikipedia.org/wiki/GEnie) 上。
*   通用美国电话电报公司个人电脑(1995):唉，雅达利倒闭了，所以我最终转向了个人电脑。当时我无法让自己使用 Windows 3.1(或 Windows 95)，所以我专门购买了一台兼容 [OS/2 Warp](https://en.wikipedia.org/wiki/OS/2#1994.E2.80.931996:_The_.22Warp.22_years) 的机器，我从大约 20 张软盘安装了这台机器。这个坏男孩有第一个奔腾 CPU，它运行在 60Mhz 左右(并且有[臭名昭著的浮点错误](https://en.wikipedia.org/wiki/Pentium_FDIV_bug))。此时我开始了我的职业生涯，在[高级启示](https://www.revelation.com/index.php/products/advanced-revelation)编程。
*   (1996 年至 2001 年)当 OS/2 没有得到重视时，我最终构建了自己的个人电脑来运行 Windows NT 4。我从未使用过任何基于 DOS 的 Windows 版本。在此期间，我主要使用 [PowerBuilder](https://en.wikipedia.org/wiki/PowerBuilder) 进行编程。
*   随着 OS X 的发布，我购买了我的第一台麦金塔电脑。我马上把它切换到 Mac OS X 启动，从来没有使用 Mac OS 9。这对我来说是一个很大的改变，因为我作为 PowerBuilder 开发人员使用 Windows 已经很多年了，现在开始作为. NET 开发人员。但是我很快发现我也想为 Mac 写程序。不想使用 [CodeWarrior](https://en.wikipedia.org/wiki/CodeWarrior) ，我开始使用 [Xojo](http://www.xojo.com) (当时是 REALbasic)。
*   PowerMac G4 Dual 1Ghz (2002 年):这是我的第一台双处理器机器。有许多小而吵的风扇，吵得令人难以置信。最终，苹果实际上做了某种“召回”，用更安静的风扇取代了电源。
*   [iMac Core Duo 2.0Ghz](https://everymac.com/systems/apple/imac/specs/imac_cd_2.0_20.html) (2006):我在 Parallels desktop 发布的当天购买了最初的英特尔 iMac，这样我就可以轻松运行 Windows XP (for。NET 开发)在我的 Mac 上。这是一个拒绝，并附带一个比我订购的更好的显卡，一个很好的惊喜。因为它是 32 位英特尔 CPU，它对我来说使用寿命有限，因为它最多只能支持 2GB 内存。
*   [MacBook Pro 15 " Core 2 Duo 2.4 GHz](https://everymac.com/systems/apple/macbook_pro/specs/macbook-pro-core-2-duo-2.4-15-santa-rosa-specs.html)(2008):我开始做全职咨询的时候买了第一台笔记本电脑。也是一个拒绝，它配备了一个比我订购的更大的硬盘。我主要用它来与 [Xojo](http://www.xojo.com) )和. NET 进行编码。最终，它被升级到最大 6GB 的内存，7200rpm 的内部硬盘和 ExpressCard 插槽中的 SSD。它的 Nvidia 卡已经不在保修期内，但作为其 Nvidia 更换计划的一部分，它属于苹果公司的[。](https://www.macworld.com/article/1154456/laptop-computers/nvidiagpu-macbookpro.html)
*   [Mac Pro 四核 2.93 Ghz](https://everymac.com/systems/apple/mac_pro/specs/mac-pro-quad-core-2.93-early-2009-nehalem-specs.html) (2010):这是我目前的设置，已经对固态硬盘、32GB 内存、3.2Ghz CPU 和用于 4K 视频的 Nvidia GeForce 750ti 进行了多次升级。它仍然非常快速和强大。我用它来做 [Xojo 软件开发](http://developer.xojo.com)，视频编辑，虚拟机和任何我能做的事情。我渴望看到苹果最终发布的下一代 Mac Pro，希望是在 2018 年。
*   [MacBook Air 11 " 1.4 GHz](https://everymac.com/systems/apple/macbook-air/specs/macbook-air-core-2-duo-1.4-11-late-2010-specs.html)(2010 年):这是第一款至今仍在使用的“新型号”Air 外形。它非常慢，只有 2GB 内存和缓慢的英特尔 CPU。
*   [MacBook Air 11 英寸 1.3Ghz](https://everymac.com/systems/apple/macbook-air/specs/macbook-air-core-i5-1.3-11-mid-2013-specs.html) (2014 年):我购买这款产品作为替代 2010 年 Air 的退换货，结果它配备了 8GB 内存，而不是我订购的 4GB。由于我不经常远程操作，这对于我的目的来说还是不错的，但是屏幕很糟糕。

我在 1983 年买的 Atari 400 有 16K 内存和 2Mhz CPU。我的 Mac Pro 有 32GB 的内存(33，554，432K)和 3.2Ghz 的 CPU (3200Mhz)。这是 35 年来惊人的进步！未来 30+年会带来什么？

我想听听其他人的经历。在评论中分享你的计算历史。
