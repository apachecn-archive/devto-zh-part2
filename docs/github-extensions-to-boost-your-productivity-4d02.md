# GitHub 扩展提高您的生产力

> 原文：<https://dev.to/theoutlander/github-extensions-to-boost-your-productivity-4d02>

以下是我使用的 GitHub 扩展。它们将帮助您提高在 GitHub 上的工作效率。请在评论中分享你最喜欢的扩展。

* * *

**[八叉树](https://github.com/buunguyen/octotree)通过左侧面板**
[![Octotree](img/b87876baf1a2d98487bb1fa78fa0b3e4.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ewQF0qZZ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/buunguyen/octotree/master/docs/chrome-github.jpg) ，可以在浏览器内轻松浏览源代码

* * *

**[【github-dashboard】](https://github.com/muan/github-dashboard)提供了过滤 Github.com 活动仪表板上事件的能力**
[![github-dashboard](img/c96dfce79a7f60d0a573dbfdb28ec187.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--238Xsa8G--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://user-images.githubusercontent.com/1153134/43299617-d3e58f7c-9128-11e8-91ec-a1df1eb048a1.png)

* * *

**[Sourcegraph](https://about.sourcegraph.com/) 允许在代码**
[![Sourcegrap](img/2673b63940a2b728a134821cf8d25133.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--hs3Wpwzp--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d33wubrfki0l68.cloudfront.net/59b2ae831cac6955e985730fa0820420839cb9fd/6cf99/products/code-intelligence/githubcodeintelligence.png) 内进行定义和查找引用

* * *

**[Octolinker](https://github.com/OctoLinker/OctoLinker) 通过将模块导入转换成可点击的链接，可以更有效地浏览代码，这样你就可以很容易地查看它们的源代码**
[![Octolinker](img/dfc590ae1f7a55d1136939987fda43cd.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--lEDg0kQ0--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://user-images.githubusercontent.com/1393946/34275053-dca87148-e69b-11e7-9d8f-e43ce361755b.gif)

**[github-repo-size](https://github.com/harshjv/github-repo-size) 在 GitHub**
[![Github-repo-size](img/dae02b9650002571ed88858088ccb745.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--y_Brtn71--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/harshjv/github-repo-size/master/screenshot.png) 上显示存储库大小和单个文件大小
