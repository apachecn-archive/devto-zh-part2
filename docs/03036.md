# 个人网站的单一网页模板

> 原文:[https://dev . to/ssbozy/single-page-template-for-personal-website-d1f](https://dev.to/ssbozy/single-webpage-template-for-personal-website-d1f)

亲爱的黑客们，我需要你们的帮助。我正在寻找一个单页的个人网站模板。此时你一定在想，“桑迪，你为什么不去谷歌一下？”。为了回答这个问题和你的好奇心，“兄弟，我已经做过了！”。问题是，大多数模板基本上不是为开发人员、工程师或黑客设计的。它们是为有作品集的设计师、艺术家或摄影师准备的。另一类人是投简历的人。我不属于他们任何一个。

因此，随着我的基本问题的解释，我不会浪费时间解释为什么，并定义我在寻找什么。也许你们很多人都在寻找这样的东西，我也想帮助解决其他人的问题。

1.  一个简单的极简网站，几乎没有动画。
2.  可定制的
3.  简单的配色方案

展示以下部分:
4a。关于我
4b。我的工作和项目
4c。能够链接到我的 Github/LinkedIn/Twitter
4d。联系页面

(可选)动态生成此页面的能力。但是一个简单的 HTML 总是可以的。

也许我要求的有点多，但是，我想我也打算涵盖其他人正在寻找的用例。请原谅我，如果我错过了一些明显的来源。
