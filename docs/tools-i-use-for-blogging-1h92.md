# 我写博客用的工具

> 原文：<https://dev.to/bauripalash/tools-i-use-for-blogging-1h92>

这篇文章是专门为庆祝 dev.to 上的 200 多名追随者而写的。

[![.](../Images/0ea48b5b91c0069b7f828113cadb4023.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--hk64ebyd--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/j0bsgal330ecicqxvfzb.jpg)

我在 13 岁的时候就开始写关于技术、自然和科学的博客，2014 年我开始了一个名为 **AlphaPedia** 的博客博客，每月仅有 1000 名独立访客，每天有 5 到 10 名访客。那时候我对博客一无所知，所以几个星期后我厌倦了，开始专注于编程，所以我离开了博客！

但同样，在 2018 年，Palash Bauri 带着丰富的知识和技能回来了。。。我开始了一个新的博客 **[帕拉什](https://palash.tk)** 的冒险，这对我来说是一种成功。

现在它有超过 7000 个独立访问者，并且还在增加。

> 我在考虑写一篇文章，如何在没有钱的情况下创建和维护一个好看的博客。

我主要使用开源技术来运行和维护我的博客，今天我将与你们分享我写博客使用的工具。

所以我们开始吧！

### github 页

如果你来自开源社区，你可能听过平台的名字 **[GitHub](https://github.com)**

是的，你猜对了，我使用 [GitHub 页面](https://pages.github.com)来托管我的博客。

### 去吧

像往常一样，使用 GitHub 和推送更改肯定需要版本控制。

为了易于使用和默认于 GitHub，我使用了 [Git](https://git-scm.com) ，

### 《杰基尔》&碧玉 2(主题)

Jekyll 是 GitHub 页面的首选静态站点生成器，所以像往常一样，我使用 Jekyll 在 [Travis-CI](https://travis-ci.com) 的帮助下构建我的站点(我不得不使用 [Travis-CI](https://travis-ci.com) ，因为出于安全原因 GitHub 不支持某些插件)

我一直想用 **[Ghost](https://github.com/TryGhost/Ghost)** 博客平台，但是找不到便宜的使用方法。但幸运的是，我找到了一个名为 [Jasper2](https://github.com/jekyller/jasper2) 的 Jekyll 主题，它帮助我的博客获得了一个类似 **[Ghost](https://github.com/TryGhost/Ghost)** 的外观。

### 【非凡

在电脑上写文章时，我用[显著的](https://github.com/jamiemcg/remarkable)来写。

好用又快！

### 马克尔

Android 的开源 Markdown 编辑器，我目前用它来写这篇文章。

我过去常常写日记，记下一些想法等等

Markor 是一个很棒的工具！

### 【MrHyde

当我在手机上想发表一篇新文章或做一些修改时，我通常会使用维护 Jekyll 博客的 Android 应用程序 [MrHyde](https://play.google.com/store/apps/details?id=org.faudroids.mrhyde) 。

这是一个非常有用的应用程序，它甚至有一个编辑器来进行一些即时更改。

但坏消息是，它不再由开发者维护了。😟

> 由于 GDPR 的原因，MrHyde 可能无法在您所在国家的 Play Store 上购买，所以您可以从这里下载[https://m.apkpure.com/mrhyde/org.faudroids.mrhyde](https://m.apkpure.com/mrhyde/org.faudroids.mrhyde)

### ·Canva、Adobe Spark Post、GIMP

为了制作博客横幅和一些图形，我经常使用安卓系统时使用 Canva 或 T2 Adobe Spark Post，在电脑上我使用 T4 GIMP。

它们是非常棒的工具。

这些是我用来写博客的工具。你可以试试，我希望你会喜欢。
