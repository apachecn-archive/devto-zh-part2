# 尽管如此，弗拉特卡·帕维希奇还是编码了

> 原文：<https://dev.to/vlatkapavisic/nevertheless-vlatka-pavii-coded--42p2>

## 我开始编码了...

16 岁的时候。为了有更好的学校，我搬到了克罗地亚的首都。这所学校专注于数学和科学。我的同学已经学了两年编程，所以我必须在一个学期内赶上他们。语言是帕斯卡语。我从一开始就着了迷，并决定继续朝这个方向学习。几年后，我获得了计算机科学硕士学位，并用汇编语言、C、C++、C#、Python、Ruby 编写了许多行代码...

## 我继续编码...

在工作。我在法国巴黎的一个学生招聘平台做软件工程师。我对职业的选择使我有可能搬到我梦想中的城市:)

在家。我有一个个人项目——我从零开始开发的美食博客，名为“美食黑客”([http://cuisinehacker.herokuapp.com](http://cuisinehacker.herokuapp.com))。此外，我目前正在 Coursera 学习机器学习，所以我也为此做了一些编程作业。

在铁路妇女研讨会上。我指导那些想学习 Ruby on Rails 的女性。

## 我对想学编程的女性的建议...

乐观、自信、有耐心。你不会在一夜之间达到目标。这需要花很多时间在你的笔记本电脑上，大量的思考和训练...我建议你有一个系统的学习方法。从书籍、教程、在线课程中学习...做笔记。当你学习了理论和语法之后，就可以开始练习了。做一个个人项目，把你的兴趣和编程联系起来(就像我在博客上做的那样)。参加当地聚会，与社区保持联系。

科技是一个很好的行业。无论我是在解决问题、寻找错误的原因，还是在学习新技术，我都不会对工作感到厌倦...此外，程序员工资很高，:D，我想告诉女孩和妇女所有的好处，有一天有一个性别更加平衡的技术世界。
