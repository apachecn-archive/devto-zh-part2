# 跟随谁在 JS 和前端世界中保持最新

> 原文：<https://dev.to/aibolik/whom-to-follow-to-stay-up-to-date-in-js-and-frontend-world-3k19>

嗨，戴夫们。在这篇文章中，我想分享我在科技界追随和崇拜的人，以了解他们分享的所有趋势和其他很酷的东西。由于我主要是 JavaScript/前端开发人员，这个列表将包含主要来自这部分的人。

这里有一个列表:

*   丹·阿布拉莫夫([@丹 _ 阿布拉莫夫](https://twitter.com/dan_abramov))——我不认为我真的需要介绍丹·阿布拉莫夫，但无论如何。他是 [Redux](https://redux.js.org) 的主要创作者和 [create-react-app](https://github.com/facebook/create-react-app) 的合著者，ReactJS 团队的一员，只是一个好人；).他在 egghead.io 上也有令人惊叹的课程

*   feross([@ feross](https://twitter.com/feross))——一个伟大的人，我在看了他关于林挺的演讲后就开始关注他，在那里他介绍了[标准](https://standardjs.com)，是的，他是那个工具的创造者。他也是许多其他酷项目的创造者，他更像是科技企业家，但他也有在大公司工作的丰富经验。我喜欢有创造力的人。顺便说一下，他还有[很棒的博客](https://feross.org/)(他演讲后我读了很多东西)。

*   阿里·斯皮特尔([@阿斯皮特尔](https://twitter.com/ASpittel))——阿里是一名开发者、教师和博客作者。我只是喜欢她写的和分享的。她也是一个活跃的博客写手，有时事通讯，她在网上分享一些很棒的文章。她也是创作者——看一看 https://learncodefrom.us/

*   迈克尔杰克逊([@ mjackson](https://twitter.com/mjackson))——不是那个:)。流行`react-router`的合著者和[再培训](https://reacttraining.com)的所有者。科技界的伟大人物，尤其是他专注于(据我所知)React。作为 react-router 的共同创造者，他的指南写得非常好。他还提供了很好的培训和研习班。此外，他真的很关心网络的未来。就看这个帖子吧。

*   Kyle Simpson([@ getify](https://twitter.com/getify))——当我们谈论 JavaScript 时，不包括 Kyle Simpson 是不公平的，也就是 GETIFY。我的 JavaScript 基础是在看了他的[你不知道的 JS](https://github.com/getify/You-Dont-Know-JS) 系列丛书后加强的。我强烈推荐给所有新的(不仅仅是初学者)JS 开发者。你最好去他的推特账户，了解他，读他简介里提到的两本书。

*   托德格言([@ toddmotto](https://twitter.com/toddmotto))——如果我喜欢 Angular，并经常与它打交道，托德格言可能是我应该认识的第一人，而[他的博客](https://toddmotto.com/)将是(几乎)任何与 Angular 相关问题的“首选”资源。他在上写了很多很棒的东西，他也是谷歌开发专家和创造者[终极角度](https://ultimateangular.com/)课程系列。

*   Chris Coyier([@ Chris Coyier](https://twitter.com/chriscoyier))-codepen . io 和 CSS-tricks 的联合创始人。他比任何人都知道更多的 CSS 技巧。他在自己的博客/网站上也很活跃。一定要跟上。

*   Wes Bos([@ Wes Bos](https://twitter.com/wesbos))——Wes Bos，我称这个人为“连环导师”，他有几十门 JavaScript 相关的课程和大量其他有用的资料。只需前往他在 wesbos.comT2 的网站。顺便说一下，最近他推出了 [AdvancedReact 和 GraphQL 课程](https://advancedreact.com)——我在新闻简报中提到过。这个课程已经有很多正反馈了。也许你也应该看看。

*   rising stack([@ rising stack](https://twitter.com/RisingStack))——我知道我要说的是人。但是这家公司真的很棒。我没有调查他们在做什么，但他们对开发者社区非常有用，因为他们分享了来自真实经验的非常有用的文章。你一定要看看[他们的科技博客](https://blog.risingstack.com)，并关注他们以了解最新动态。

这个列表只是整个事情的一小部分。事实上，有更多的人是我所追随和钦佩的。欢迎在评论中分享你的“hereos”。

谢谢你邀请我，再见！
