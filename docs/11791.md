## [](#readrr)[Readrr](https://readrr.surge.sh/)

所以我让我姐姐在 Gravit 上做了一个网站的快速模型，我实现了它。看起来像这样是因为我姐姐对设计的品味。它是免费和开源的，所以你可以在这里查看，也许还可以查看 **[回购](https://github.com/johnpaulada/readrr)** 。

## [](#stdlibio)[Stdlib.io](https://stdlib.io/)

尽管 JavaScript 是一种非常流行的语言，但它没有像 C++和 Python 那样的标准库。Stdlib.io 是一个旨在成为 JavaScript 标准库的项目。它有数学、数据结构甚至绘图的功能！缺点是这个库有相当糟糕的文档，而且它相当大——大到我不能用 Jsdelivr 作为 CDN，因为它不能处理它。除此之外，还挺牛逼的。

## [](#plover)[千鸟](https://plover.io/)

如果你曾经尝试过苹果的隔空投送，它可能是你快速发送文件到附近设备的选择。Plover 是没有苹果设备的人的一个选择。前往 plover.io 网站，找到你想发送文件的人，然后就可以了！不过你必须连接到同一个网络。
