> 原文:[https://dev.to/bharris/phoenix-303](https://dev.to/bharris/phoenix--303)

受 [oodsnet](http://oods.net) (以及我的 pull 请求添加 [darkmode](https://github.com/exezin/oodsnet/pull/5) )的启发，我开始创建自己的 tilde.team fork(现在的 [forum.tilde.team](https://forum.tilde.team/) )。

第一步是将 css 切换到 [tilde.team 标准](https://tilde.team/css/)并更新 bootstrap 的类。一旦我让它运行起来并与 tilde.team linux auth 服务集成，我就向其他 tildeans 征求意见和建议。

[~micaiah](https://dev.to/~micaiah/) 很想帮忙，但也想学习一门新的语言和/或框架，所以我们决定重新开始，用 [elixir/phoenix](http://phoenixframework.org) 重新创建整个论坛。我们以前讨论过长生不老药，但是从来没有一个令人信服的用例来强迫我们学习它。

该项目已经上线，源代码在 [github](https://github.com/tilde-team/forum) 上。
