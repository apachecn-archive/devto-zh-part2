理解 Javascript 中 For、For…In 和 ForEach 循环的区别

For 循环
当你预先知道脚本应该运行多少次时，就使用 for 循环。让我们看看下面的 for 循环语法的例子，它将迭代 n 次。(根据循环次数，指定第 n 个值。)

语法:
for(I = 0；I<n；i++) {
//做点什么
}

例
const array = ['skptricks '，' php '，' Java ']；
for(I = 0；我<列阵；i++){
console . log(array[I])
}

## 输出:

> " skptricks"
> "php"
> "java "
