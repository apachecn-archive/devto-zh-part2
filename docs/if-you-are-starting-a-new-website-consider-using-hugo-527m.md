如果你正在创建一个新网站，可以考虑使用 Hugo。除了它是“世界上最快的网站建设框架”之外，它还具有强大的功能，使网站开发过程更加令人愉快。

## CLI

一旦你在电脑上安装了 Hugo，你就可以使用 Hugo 的命令行界面了。您可以在一毫秒内启动一个本地 web 服务器。更不用说直播重装是默认工作的。

## 主题

如果你不想或者不知道如何建立一个自定义主题，你可以使用 Hugo 提供的数百个主题中的一个。然而，如果你想与众不同，如果你想挑战你的创造性，你可以为你的新项目创建一个自定义的风格。您甚至可以使用内置管道来处理 Sass 或 Scss 文件，或者缩小最终的 css 文件。

## 模板

开始使用 Hugo 时，您应该了解的一个关键特性是查找顺序。Hugo 搜索正确模板的机制是经过深思熟虑的，旨在满足每一个项目结构，甚至是那些不平凡和简单的项目结构。

在模板内部，您可以使用变量、函数、逻辑、管道和其他有用的特性。如果您有一个想要在不同的模板中重用的代码片段，那么您应该将这个可重用的代码提取到一个称为 partial 的单独文件中。

## 上下文

你应该理解的另一个关键特征是*上下文*。上下文或“点”取决于环境。如果您在循环内部，那么上下文就是当前项。如果您在基本模板中，那么上下文就是当前页面及其所有参数。

Hugo 允许你定义不依赖于上下文的变量。例如，在比较循环内部的当前项目参数时，该变量可能很有用。

还有全球背景。您可以随时从每个模板文件或部分模板文件访问全局上下文。
