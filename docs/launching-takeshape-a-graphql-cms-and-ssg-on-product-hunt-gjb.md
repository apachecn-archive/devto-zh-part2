我们花了数年时间，使用各种可用的内容管理系统来构建内容体验。我们一直想要一个像 TakeShape 这样的工具，所以我们决定建造它。

TakeShape 是唯一一个在一个产品中提供 JAMStack 所有元素的服务。灵活的内容建模、GraphQL API 和静态站点生成，为开发人员提供了简单易用的 web UI 和 CLI。我们看到网络的未来将建立在静态和动态内容的平衡上，所以我们建立了 TakeShape 来支持这一愿景。
