# 和我一起写(第六天)

> 原文:[https://dev.to/matt24ray/write-with-me-day-6-2g8a](https://dev.to/matt24ray/write-with-me-day-6-2g8a)

所以，你可能已经注意到我已经错过了几天的写作挑战。然而，我不会停下来，而是继续前进。这对我来说是一个积极的动力。其他几个人也联系了我，说他们也喜欢激励。所以，只要我和其他人受到鼓舞，我就要继续写下去:)

今天我见到了杰斯·陈。她是脸书公司的产品设计师，她给了我一些急需的灵感，让我追随自己对设计的热情。

首先，Jess 让我对不同类型的设计有了一些了解。特别是工业设计(硬件领域的产品设计)和服务设计之间的区别。我不会详细讨论它们的区别，因为在这一点上我不能给出一个合适的介绍。

杰斯鼓励我和其他设计师在一起。即使我去参加一个聚会，问“我为什么要去”，我也可能会听到一些对话，让我对设计社区有所了解。她说的一句话让我印象深刻，“接触社区是件好事。”

在过去的几个月里，我开始对设计类的东西产生了极大的热情。感谢 Jess 今天给了我一些灵感，让我在这个领域继续成长。

如果你也想要一些灵感，可以去看看杰斯的网站。
