# 周五节日部署！！！#deployFridayFest

> 原文：<https://dev.to/acanimal/deploy-on-friday-festival--deployfridayfest-8gh>

谈论在周五部署到生产是一个有争议的问题。一些公司对此有严格的政策，而另一些公司则更加灵活，对此没有任何问题。

就我个人而言，我认为这取决于许多因素:变化的类型，它可能产生的影响，你的过程有多好和自动化(我指的是应用变化、测试和部署等)，公司如何组织成团队，等等。

我想发起一项“倡议”😅要知道有多少开发人员在周五进行生产部署，正因为如此，每次你在周五进行生产部署时，发布一条带有`#deployFridayFest`标签的 tweet。

干杯！
