# 然而，克莱尔·波拉德编码

## 我继续编码是因为...

编码使我能够解决我所有激情交汇的问题。我用它来做我的日常工作，解决工程中的技术问题，举办山地车比赛，为自己和他人主持网站，用我的十字绣、编织和手工制作...一切和一切！

在过去的 12 个月里，我对自己的多重身份生活方式已经适应得多了，我很高兴能在小轮车赛道上颠簸，完成我最新的十字绣，并在同一天涉猎一门新语言。这主要归功于艾米莉·瓦普尼克(Emilie Wapnick)的精彩著作《如何成为一切》(How to be Everything)，这本书极大地帮助我接受了生活中没有“一个真正的使命”,而是有多个我热爱并能努力去做的不同事情。

编码是一种工具，我可以用多种方式来做好事。

## 我最近克服了...

害怕我的事业朝我当时不太适应的方向发展。我最初认为，从编码转向营销会降低我在编程和跟上我们产品进展方面的效率，但如果有什么不同的话，那就是当我回到编码时，我更加兴奋，因为我的效率更高了。

## 我想吹嘘一下...

放弃我的业余时间来帮助人们实现他们的努力，无论多么微不足道:要么是我的未婚妻和他的公司 [PiBorg](https://www.piborg.org) ，在 [Royston Rockets](http://roystonrockets.co.uk/) 担任 BMX 教练，作为[制造商协会](http://www.guildofmakers.org/) #MakersHour 主持人，并借我的社交媒体技能帮助 Peter Jukes 报道英国臭名昭著的[电话窃听审判](http://www.pressgazette.co.uk/beyond-contempt-how-peter-jukes-invented-new-way-funding-court-reporting-and-found-himself/)(这导致我在一本书中被提及，并在 Hugh Grant 出现时成为 100%的隐士)。

## 我对支持女性和非二进制代码的盟友的建议是....

大声说出那些了不起的女人，非二元的，流动的，跨性别的，男人的，或者他们在你生活中的身份。正如精彩的蒂姆·米金在他的 UWA 演讲中所说:

“在赞美你所钦佩的人时，要热情大方。”

这让他们对自己感到惊讶，有时如果事情对那个人来说很艰难，一个信息可以改变事情并激励他们。

## 我对其他女性和非二进制程序员的建议是...

大胆一点。尝试你平常舒适区之外的新事物。预计有时会失败。求助。尽可能利用你的空闲时间去帮助别人。所有这些事情都会把你带到有趣和令人兴奋的新地方。
