# 成为首席技术官的 5 个简单步骤

成为首席技术官是当代技术职业生涯的巅峰。每年都必须快速适应新技术发展的组织依靠熟练的首席技术官来规划他们的路线。当不规划技术升级策略时，CTO 保持最大的系统可用性并实施软件改进，同时抵御数据泄露和网络攻击。难怪首席技术官的平均底薪和最高法院法官差不多，也难怪企业会争夺最好的首席技术官。

英国咨询公司 Agil8 调查了杰出首席技术官的职业道路，结果显示，他们平均需要 24 年时间，在四家公司的八个职位上才能晋升到最高职位。当然，不是每个人都符合这个模式，但是 [似乎有一条共同的道路](https://www.agil8.com/path-to-cto/#) 。让我们把这条路分成几个步骤。

1.  **获取凭证**

公司更喜欢拥有认证大学学位的首席技术官，通常是软件工程、计算机科学或信息技术等专业的首席技术官。通常，科技学位持有者被鼓励去读 MBA，而商科专业的学生则被敦促去读科技领域的硕士学位。原因很简单——首席技术官必须有足够的信誉来管理一个技术专业人员团队，但必须有足够的商业头脑来进行战略投资、管理大量预算和分配稀缺资源，所有这些都要与组织的战略目标保持一致。

2.  **实践经验**

一旦离开学校，一个有抱负的首席技术官应该发展和展示技术能力，最好包括组织关注的关键领域之一，如电子商务、物联网、人工智能、大数据或网络安全；我们的目标是在一个有着复杂现实问题的非结构化环境中展示从学术能力到工作能力的成功转变。工作场所还提供了一个增加新技能的机会，鉴于技术变化的速度，这一点至关重要。毕竟，一个知识过时 24 年(被任命为首席技术官的平均时间长度)的新首席技术官将面临挑战。

3.  **行业经验**

随着向现实世界技术问题的过渡完成，有抱负的首席技术官应该决定致力于发展哪个行业的专业知识，因为在担任首席技术官之前，一般的首席技术官至少要在同一个行业的负责任的技术领导岗位上工作 5 到 7 年。自然，电子商务公司最有可能在其专业范围内寻找 CTO，而不是来自重型设备制造业，了解企业级 ERP 系统和一般信息技术，但不了解前端开发或网站优化的人。

4.  **声誉和网络**

在一个行业的职业阶梯上向上爬，未来的首席技术官应该培养好奇心和良好的判断力以及专业技能。在会议上发言并与其他专业人士建立关系至关重要。成为一个招聘人员知道在寻找职位空缺时可以联系的人也是有帮助的。可能是推荐其他人，但是建立关系经常证明是有益的。然而，有一点需要考虑——福布斯称 54%的 CTO 招聘来自内部候选人。

5.  **沟通技巧**

几乎所有的首席技术官都认为沟通是最重要的技能之一。CTO 必须向同事解释技术，阐明策略，向组织解释技术政策，并激励员工。发展强大的书面和口头沟通技巧是在组织阶梯的最高层成功表现的必要条件，这也许是区分两个技能相当的候选人的关键属性。

了解组织面临的挑战，利用专业网络获取背景信息，在专业人员的面试辅导中寻求专业人士的帮助[](https://employmentboost.com/executive-career-interview-coaching/)，安排推荐人，让未来的同事相信他们会得到很好的帮助——这些是旅程中的最后一步。
