## 用户和邮箱

**Git** 将使用这些数据来*标记*您创建的任何提交:

```
git config --global user.name "Your name goes here"
git config --global user.email "your@email.goes.here" 
```

## 有用的别名

将此添加到您的`~/.gitconfig` :

```
[alias]
    l = log --oneline --decorate --graph

    ll = log --graph --abbrev-commit --decorate --date=relative \
    --format=format:'%C(bold blue)%h%C(reset) - %C(bold green)(%ar)%C(reset) \
    %C(white)%s%C(reset) %C(dim white)- %an%C(reset)%C(bold yellow)%d%C(reset)'

    lll = log --graph --abbrev-commit --decorate --date=relative \
    --format=format:'%C(bold blue)%h%C(reset) - %C(bold green)(%ar)%C(reset) \
    %C(white)%s%C(reset) %C(dim white)- %an%C(reset)%C(bold yellow)%d%C(reset)' \
    --branches

    co = checkout
    ci = commit
    man = help h = help a = add
    f = fetch
    d = diff
    dc = diff --cached
    dt = difftool
    dtc = difftool --cached
    ds = diff --stat
    dsc = diff --stat --cached
    s = status --short --branch
    b = branch

[credential]
    helper = cache

[diff]
    algorithm = patience 
```

## 全球。吉蒂尔

让我们定义一个全局**。gitignore** 文件:

```

## 处理行尾

### Linux / Mac

如果你是 **Linux / Mac** 用户:

```
git config --global core.autocrlf input
git config --global core.safecrlf true 
```

### 窗口

如果你是 **Windows** 用户:

```
