# 高性能 FreshBooks 和 Zoho CRM 连接，带有升级的 Devart Excel 插件

> 原文：<https://dev.to/devartsoftware/high-performance-freshbooks-and-zoho-crm-connectivity-with-upgraded-devart-excel-add-ins-e49>

数据连接解决方案和数据库工具的捷克软件供应商 Devart 发布了一个大大改进的 Devart Excel 插件版本，这些工具允许像处理普通 Excel 电子表格一样处理数据库和云数据。Devart Excel 插件现在完全支持新的 FreshBooks API (alpha)和带有 OAuth 身份验证的 Zoho CRM API v2。此外，用户现在可以从 FreshBooks 对象中查询已删除的记录，如客户、员工、费用、发票、项目、付款。

在[https://www.devart.com/excel-addins/](https://www.devart.com/excel-addins/)仔细看看 Devart Excel 插件中的一些当前和以前的特性。
