## 1。[埃米特](https://emmet.io/download/)

[![img](../Images/9eae38c0da335c0dcbd04469457071d6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--v91PIVEU--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://boostlog-public.s3.us-west-1.amazonaws.com/articles/5b31ef9244deba00540475a1/%25E3%2582%25B9%25E3%2582%25AF%25E3%2583%25AA%25E3%2583%25BC%25E3%2583%25B3%25E3%2582%25B7%25E3%2583%25A7%25E3%2583%2583%25E3%2583%2588%25202018-06-26%252016.25.18-1529997927721.png)

让你的 HTML 和 CSS 代码比以往任何时候都更容易与 Emmet 包，它允许您使用缩写在您的设计代码。它帮助你将复杂的结构写成简短的文本字符串。

## 2。[棉绒](https://atom.io/packages/linter)

[![img](../Images/2f1b578f13a57008a9a8735dc130441e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--607V8--I--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.github.com/dotcypress/coffee-lint/master/screenshot.png)

atomlinter.github.io 上有完整的 linter 列表，每种语言都有不同的 linter。你需要为你的语言安装一个特定的 linter。基本上，它是一个用来分析源代码的工具。

## 3。[原子美化](https://atom.io/packages/atom-beautify)

[![img](../Images/095253b8df9ecdbd8b99ace9a0374742.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--nWJ02tsj--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.github-camo.com/8f053415f4dfba4849d9bbe8327425d54511d94b/68747470733a2f2f636c6f75642e67697468756275736572636f6e74656e742e636f6d2f6173736574732f313838353333332f31363534323732382f64636163333730302d343038612d313165362d386533352d3963386663343433326564632e706e67)

Atom 美化包是为了让你的代码更加清晰易读。你可以用这个软件包美化你选择的文本部分或整个文件。毫无疑问，一个干净、可读的代码库使开发变得更容易，这正是这个包所做的。这个包用于各种开发语言，如:HTML，CSS，JavaScript，PHP，Python，Ruby，C#，Java，C，C++，Objective-C，CoffeeScript 等。

## 4。[电传打字机](https://atom.io/packages/teletype)

[![img](../Images/9f90f3e8f8212ff471c0cdf9b58e2d12.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--L3M09-h6--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://www.publickey1.jp/2017/teletype4atom01.gif)

让我们与团队成员共享您的工作空间，并使用 Teletype 实时协作您的工作。它通过与另一个程序员分享你的代码来帮助你增长知识。了解你的上司的技巧的最好方法是他们如何处理不同的情况。

## 5。 [Git-Plus](https://atom.io/packages/git-plus)

[![img](../Images/75ab3cef4182496b33813f2dda0bf6c6.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--WIwvlD4k--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.github-camo.com/78e2bafa5f9b3afdf47d7e02e3f949fea4801fc0/68747470733a2f2f7261772e67697468756275736572636f6e74656e742e636f6d2f616b6f6e77692f6769742d706c75732f6d61737465722f636f6d6d69742e676966) 

现在不需要切换到终端就可以借助短码进行各种 Git 动作。只需键入 cmd + shift + h 或 ctrl + shift + h。现在，您可以通过使用 Git Plus 包运行每一个 Git 命令，而无需使用终端。

## 6。[高亮选中](https://atom.io/packages/highlight-selected)

[![img](../Images/40e7fcfbf68246a220c0773ddc4e7b51.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--I6SVGotE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.github-camo.com/fb3c3e8f4170fc20047810e53cdfa1041f302a28/687474703a2f2f692e696d6775722e636f6d2f4335466e7a7a512e676966)

现在，通过使用高亮显示选定的 g it 包来选择一个单词，可以很容易地跟踪该单词在当前文件中的所有出现。双击选择一个单词，看看它还用在什么地方。

## 7。[文件图标](https://atom.io/packages/file-icons)

[![img](../Images/4ddd1ceaf020cf9e81b72ec072043a0f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--KFbpTIEP--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://boostlog-public.s3.us-west-1.amazonaws.com/articles/5b31ef9244deba00540475a1/%25E3%2582%25B9%25E3%2582%25AF%25E3%2583%25AA%25E3%2583%25BC%25E3%2583%25B3%25E3%2582%25B7%25E3%2583%25A7%25E3%2583%2583%25E3%2583%2588%25202018-06-26%252016.23.44-1529997846880.png)

这个软件包通过给你添加彩色文件图标的扩展来帮助你以一种更系统的方式组织你的文件。它将帮助您在树中找到文件并保持概览。你也可以根据自己的需要调整图标的大小。

## 8。[颜料](https://atom.io/packages/pigments)

[![img](../Images/d22089c271baed24e85ef8d6454e08e4.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--NutDvXw1--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.github-camo.com/802d8b759d01e70861f95f99495731f19b145b03/687474703a2f2f61626533332e6769746875622e696f2f61746f6d2d7069676d656e74732f7069676d656e74732e6769663f7261773d74727565)

现在用颜料包处理 CSS 文件中的颜色代码不再困难。它扫描你的 CSS 文件并检测所有颜色代码。

## 9。[打开最近的文件/文件夹](https://atom.io/packages/open-recent)

[![img](../Images/8af84cc4e163bf0ec5dcffb543543b2d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--69xW6hKY--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://i.imgur.com/d9y4iAi.png) 

打开最近关闭的文件或文件夹用这个包让轻松开始工作快速没有任何延迟。

## 10。[自动关闭 HTML](https://atom.io/packages/autoclose-html)

[![img](../Images/17050107f89b5677626111fc0a287fbb.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--B9832Apz--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://hswolff.com/images/posts/2015/02/atom/autoclose-html.gif)

自动关闭 HTML 标签帮助您关闭打开的 HTML 标签，以防您忘记关闭。

它让你的标记创建速度翻倍！这个包允许你定义哪些标签应该像`<p></p>`一样内联完成。
