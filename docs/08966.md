# **REST API 只是一个 UI**

这可能是一个有点愚蠢或大胆的说法，但如果你仔细想想，我认为这是有道理的。当我意识到这一点时，技术栈发生了变化:

```
MongoDB <-> Core (Node.js) <-> REST API (Node.js) <-> React Native (Android & iOS) 
```
