# 尽管如此，利兹·罗德里格斯编码

## [](#i-began-to-code-because)我开始编码是因为

我喜欢计算机、数学、科学和艺术。一位(女)导师建议我主修计算机科学，因为它结合了我所有的兴趣，而且我在数学分班中考得很好。四年后，经历了许多起伏和几乎放弃，我坚持下来，并获得了计算机科学学士学位！走过毕业舞台是一个史诗般的时刻。

## [](#i-recently-overcame)我最近克服了

自从我进入 IT 领域并已经很久没有真正接触代码后，我克服了恐惧回到了软件工程领域。我喜欢 web 开发的发展方向，以及我们如何从开源社区的软件中创造出如此多的东西，除了想法、浏览器和文本编辑器，什么都没有。十多年后，我决定离开我的 IT 工作，从硬件/系统转向软件。我仍然并将永远在那段旅程中！

## [](#i-want-to-brag-about)我想吹嘘一下

我一生都在支持科技界的女性。大学毕业后，我开始了我的第一个博客，它应该是一个支持科技界女性的论坛(当然，我不知道如何让它成长，但我确实尝试过😀).我参与了 linux、开源和各种在线社区。当我遇到对 STEM 感兴趣的女性时，我会与她们分享我所知道的所有资源以鼓励她们，并告诉她们我在这方面的经历。

## [](#my-advice-for-allies)我对盟友的忠告

支持做代码的女人就是坚持做下去，永远！现在是一个比以往任何时候都更好的时机，让你的声音被听到，并支持你的女性同胞/非二进制#代码姐妹。出现在聚会上，在他们遇到困难时帮助他们，表现出同情，鼓舞他们。我们和我们想象的一样好，如果不是更好的话，同样的目标，我们可以用我们独特的想法和观点带来多样性，创造出令人惊叹的产品，做令人惊叹的事情来帮助我们的社区！
