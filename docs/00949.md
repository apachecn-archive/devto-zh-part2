# 不同种类的循环和条件的基本介绍

> 原文:[https://dev . to/seandershancevic/a-basic-intro-to-different-species-of-loops-and-conditionals-2nd](https://dev.to/seandershancevic/a-basic-intro-to-different-kinds-of-loops-and-conditionals-2lnd)

在 javascript 中循环信息是几乎每个程序或功能网站都会发生的事情。虽然形成这些循环并在 javascript 中测试它们的基础真的很简单！首先，循环是一段代码，根据其编写方式，循环通过该特定部分，直到满足某些条件。这就是条件句出现的地方。它们决定这些循环运行的方式和时间，因此您可以控制它们的功能。这里有一些你在编程生涯早期会遇到的更常见的循环类型的例子！

第一个是“for”循环。“for”循环遍历数据，直到满足括号中的条件。只要“I”小于或等于“< =”10，循环就会运行。

另一种类型的循环是“while”循环。这类似于“for”循环，因为当您传递的条件为真时，它也会遍历数据。当“I”小于或等于 10 时，循环将继续运行。

条件对循环很重要，因为否则循环将永远运行下去。“if”和“else”是两个密切相关的重要条件句。它基本上意味着“如果”这些条件不满足，运行这段代码。否则或“否则”，运行此代码。

这些只是演示这些循环和条件如何工作的基本计数示例，但是您可以开始想象这些工具在使用大量数据编码时会有多么有用。循环的应用几乎是无止境的，所以尽早掌握它们是一件好事！
