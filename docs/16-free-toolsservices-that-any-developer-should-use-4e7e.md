# 面向任何开发者的免费工具&服务

我要感谢所有允许他们的产品免费使用的服务提供商和开发者。报答他们的最好方式是什么？广告。

我将分享我在我的工作&家庭环境中使用的最有用的免费开源工具。**这些工具可以用在任何 env/语言/框架/项目中。**如果你是大公司的开发人员或自由开发人员，这篇文章应该是一个好的开始。

> 也许你已经知道了大部分的工具，希望今天你能听到一个新的。

我会跳过 Slack 之类的聊天客户端，Stack Overflow 之类的服务， [twitter](https://twitter.com/B3aT) ， **YouTube** 以及其他 **Google** 的产品。我使用它们中的大多数，我生活中的一切都始于一个电子表格。**收件箱** &谷歌**文档**对于原型、想法、公式、项目文档，单独工作或在团队中共享都是极好的。

## A 级—开发人员、运营人员、质量保证人员和 IT 人员的工作效率

Visual Studio 代码 — **代码**编辑器，比其他人更接近于 IDE。我只用了几个星期，但它很快就成为了第一，对我来说，它取代了 Atom/Sublime/Notepad。 [Visual Studio](https://dev.toundefined) 适用于任何语言和**配置文件** (json、xml、ini……)。它代替了 Unity3D 的 MonoDevelop，我用 Go 做 TDD，用 Java，C#等做代码。

作为微软的产品，我一开始非常怀疑，但我放弃了我的一切，它处理得很完美。它运行速度很快，可以跨平台运行。虽然(目前)我仍然会在大型项目中使用我的 PHPStorm (webstorm)。它有一个非常好的学习曲线，[它的特点之一是键盘映射](https://code.visualstudio.com/docs/getstarted/keybindings#_keymap-extensions)，我在 VSCode 中使用 Intellij 键盘快捷键，这多酷啊？

[**特雷罗**](https://trello.com/adriangeorgescu/recommend) —任务**清单**，待办事项清单，购物清单，参观地点清单，与孩子一起做的事情，想法记事本，小项目/团队的任务管理，bug 追踪器，功能规划，我该继续吗？Trello 运行在所有平台上，免费层已经足够好了。它也有许多集成插件。

[![](../Images/fa00a44573bf3e8c3b0f94583b0ff9fd.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--InBkULed--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/2284/1%2AYGnnb4DPrOzxi0zxVmky6g.png)

[**GetShareX**](https://getsharex.com/) — **截图**并对您的桌面/窗口/区域进行 Screecast。我用它来写错误报告、截图、例子、记录“如何…”等等。和 Snagit/Jing 差不多但是更好(是开源的)。记录视频或 gif，有脚本动作，如截图后自动上传到 imgur 和许多其他功能。对于 Linux ，我混合使用了快门[和 Ubuntu 截图内置工具。](http://shutter-project.org/)

[![](../Images/80bb93c52730922440c179ecac577fc0.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--2VKUaiP3--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/2486/1%2ASX7IeD6Sc545R-fm16fN9w.png)

**—记录你工作所花的**时间**。时间是一种有价值的货币，所以我一直记录我是如何度过的。 [Toggl](https://toggl.com/) 有强大的空闲检测(测出来的时间会更准确)，浏览器，桌面，移动客户端。支持多个项目，甚至生成报告(适合自由职业者/远程工作者)。这都是关于计算投入项目的**资源与收益**。**

 **[![](../Images/47b2fa4535a9f71e8724e1c690d46f9e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--4bcegj6X--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/2012/1%2Apl1fXb_yiW2f2Izy1FjQFw.png)

[**代码大战**。com](http://www.codewars.com/r/vWPS5A)—算法。如果 [TopCoder](https://www.topcoder.com/) 对你来说太严肃，而[code wars](https://codecombat.com/)太幼稚，你会爱上 [Codewars](https://www.codewars.com/) 。这一切都是为了保持你的大脑健康。如果你被困在工作中，一遍又一遍地做同样的算法，或者整天修复遗留代码，你需要一些好的问题，这些问题可以在几分钟内解决。支持多种难度和大多数编程语言。**这都是为了提高你的编程技能**。

[**源码树**](https://www.sourcetreeapp.com/) — git 客户端。对于 git 命令，我通常使用 bash，但是当我需要 UI 时，我使用 SourceTree。它是跨平台的，容易理解，我们用它来教非技术团队成员如何使用 git。如果你正在寻找一些更简单&更友好的东西，检查一下 [GitHub for desktop](https://desktop.github.com/) 。

**存储**——我使用 [Bitbucket](https://bitbucket.org/) 用于免费无限的私有 git 库，使用[T5】github](https://github.com/)用于开源库，使用 gist*(代码片段)*。Dropbox/Google drive 用于其他资源，而 [backblaze](http://backblaze.com) 用于托管图像/视频。

这个工具将提高你的浏览速度，搜索历史/书签，关闭标签页，打开你想得到的链接。它遵循 Vim 咒语，一切都用键盘完成。在 Chrome 和 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/vimium-ff/) 上工作的[。](https://chrome.google.com/webstore/detail/vimium/dbepggeogbaibhgnhhndojpepiihcmeb)

*   我将很快转移到 GitLab，很可能它将取代 BitBucket(对我来说)，因为它们具有丰富的可互换性和 devops 特性。

*   [InterviewBit.com](https://www.interviewbit.com/)如果你想练习编码技术面试和系统设计，我强烈推荐你。它有最好的用户界面，UX，问题，解决方案和测试质量。

*   [**UTime**](https://chrome.google.com/webstore/detail/utime/kpcibgnngaaabebmcabmkocdokepdaki?hl=en) —转换 UTime 时间戳/日期的 chrome 扩展

*   [**邮差**](https://www.getpostman.com/) — Web API 测试器，简单牛逼。可以保存&分享模板，付费层支持 JS 自动测试！

*   **Bash**for Windows——来自 [Git for Windows](https://git-scm.com/downloads) 、 [MobaXterm](http://mobaxterm.mobatek.net/) (如果你经常连接到远程服务器)或者 [Native Win10 Bash](https://www.howtogeek.com/249966/how-to-install-and-use-the-linux-bash-shell-on-windows-10/)

*   **(windows)一个非常丰富的 web 开发代理。我用它来调试 web 应用程序(它们的请求)。 [CharlesProxy](https://www.charlesproxy.com/) 更好，也适用于 iOS，但它**不是**免费的。**

## M 类—介质

[**gimp 2**](https://www.gimp.org/)——对于我的图形需求来说绰绰有余。我用它来编辑照片，为原型创建简单的 **GFX** ，调整图像大小/变换图像，创建拼贴画(不同大小，透明度等)。我花了很长时间来适应它(来自烟花)，但这是值得的。提示:激活单窗口模式。

[![](../Images/9ebfda30e9277d47deea24d272781d57.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CFYTdH0u--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/2400/1%2ATPleTzkFSL5z3vp5KpYY9g.jpeg)

[**Canva**](https://www.canva.com/) —当你需要**社交媒体**海报、横幅、拼贴画、页眉或标志，而你又缺少设计师时，Canva 团队会帮你。它为小项目提供了一个很好的免费层，非常棒。它有数以千计的预制模板和照片，您可以在您的商业项目中自由使用。

[![](../Images/e2b48af1cfb09d42724606be87952c6e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--UUqF3vQk--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/2198/1%2AsrzI8epDtN-W7QBYcZUUAQ.png)

[**Piktochart**](https://piktochart.com/) —一个词，信息图。皮克托哈特对他们了如指掌。它没有这么好的免费层，但这是一个可怕的开始。
