我今天学到了许多新的技术和工具，但它们可能不是与我的工作最相关的，但它们确实很有趣。我今天刚刚制作了我的第一个 slack 表情符号，在这个过程中我学到了很多。如果你也想成为 slack 频道的摇滚明星，系好安全带，我会教你如何做你自己的表情符号。首先，你需要一个图片编辑器，因为我不想买任何东西，而且这是一个很棒的软件，所以我选择了 GIMP。

对于 slack 来说，最好的表情符号大小是 128×128 像素。所以打开一个 128×128 的新图片，接下来你可以添加你喜欢的背景图片或者文字。我创建了一个有滚动文本的表情符号，所以我用白色背景滚动洋红色文本。如你所见，这是我公司自定义表情符号的一个巨大缺口，需要尽快弥补。

[![](../Images/adeb8a60bf95f1a92050c3cc78e1a5c2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--2PjVQxRG--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://christine-seeman.com/wp-content/uploads/2018/06/Screen-Shot-2018-06-07-at-3.22.02-PM-1024x616.png)

从那里决定你想要它做什么。我选择滚动，因为这看起来很简单。你的每一层都是动画的不同部分。所以通过滚动，每一层都稍微向左移动。你可以做五彩纸屑或颜色，然后每一层都是运动。要检查它看起来像什么，去过滤器>动画>播放…这将显示你的动画会是什么。一旦你对外观满意，然后使用滤镜>动画>优化(GIF ),然后导出为 GIF 图像。更多详细说明，请查看 GIMP.org 的这个[精彩教程。现在你已经准备好上传它并在 slack 上使用，如果你在那个部门需要一些帮助，请查看](https://www.gimp.org/tutorials/Simple_Animations/) [Slack 的帮助指南](https://get.slack.help/hc/en-us/articles/206870177-Create-custom-emoji)。
