# 俄罗斯方块 JavaScript 俄罗斯方块游戏(2006 年 3 月)

> 原文：<https://dev.to/jalbam/tetrssimus--tetris-game-in-javascript-march-2006-hc3>

“俄罗斯方块”是一个完全用 DHTML (HTML，CSS 和 JavaScript)编写的开源俄罗斯方块克隆。

[![Screenshot](../Images/bdd106469f63c7354f1e07379b99c765.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--WtZ-MOcs--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/http://dhtmlgames.com/tetrissimus/tetrissimus.gif)

这是我的第三个 DHTML 游戏，制作于 2006 年初。

编辑游戏代码以便添加不同形状的新块或者改变当前的块或者甚至删除它们是非常容易的。你也可以通过编辑脚本开头的配置变量来改变游戏行为。要改变背景图像，你只需要用另一个来替换它。

这个游戏只使用了一张图片作为背景，这是不必要的，所以你能看到的所有其他图片都是用纯 HTML 和 CSS 制作的。

官方语言是西班牙语和英语。

你可以在互联网上找到这个游戏的其他修改和改编版本，包括一个 Opera 小部件，甚至一个 Drupal 模块。

这款游戏已经在 BeOS、Linux、NetBSD、OpenBSD、FreeBSD、Windows、Mac OS X 等平台下测试过。
