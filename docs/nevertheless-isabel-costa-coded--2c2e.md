## 我开始编码是因为...

我进入大学后就开始写代码了。我对计算机知之甚少。在第一堂计算机科学课上，我有点迷路了。一个学生的导师花时间听我讲述编程的挫折，并在一个周五的晚上好心地帮助我，只是为了让我了解 Python 语言(我学习的第一门语言)的基础知识。那天我终于进入了编码领域，并开始喜欢上了它。

## 我目前正在入侵...

我目前是一名 Android 开发人员，我的硕士学位论文专注于安全性。

## 我最近克服了...

在花了这么多时间熟悉 Java 编码之后，我通过做一个示例项目来学习 Kotlin。首先，我尝试使用官方网站上的练习进行培训，但我发现从头开始编写项目代码时学习它更有帮助。

## 我对其他女性和非二进制程序员的建议是...

让你周围的人激励你，相信你。

如果没有亲自找到导师，也不要气馁。互联网上有很多鼓舞人心的人，即使你不认识他们，他们也能激励你。

如果你已经在这个行业了，帮助和支持那些刚进入这个行业并且可能要转行的人。

对初学者来说:编码和其他需要时间的技能一样。如果别人似乎比你学得快，不要气馁。每个人都有自己的学习节奏。
