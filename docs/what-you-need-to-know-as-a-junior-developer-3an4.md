# 作为初级开发人员需要知道的事情。

每个新手都想做一些很酷的东西，但是想要快速完成的冲动有时会导致跳过一些可能导致漏洞的程序。作为一个新手开发者，下面是需要做和避免的 5 个要点。

## 1。接受你是初级开发人员。

接受成为初级开发人员会让你走上一条程序的轨道，这实际上会导致适当的学习和一致性。

## 2。理解，不要模仿。

模仿是创造力和创新的敌人。有时我们觉得需要模仿某人，但理解应该放在第一位。

## 3。给自己时间去掌握。

大脑只不过是一座砖上垒砖的房子；给自己成长的时间是发展技能的一项基本技能。记住这不是一场比赛。

## 4。不要追随社交媒体趋势。

不是每个人都同意我的观点，但是摆脱这种行为是发生在我身上最好的事情之一。我知道我们都想加入那些使用一种趋势语言或框架的酷开发人员的团队，但是缺乏对基础的正确理解将导致那些工具的糟糕使用。先给自己时间了解基础就行了。

## 5。永不停息！

没有人生来就有代码大脑或某种魔力。每一个大师和严肃的开发者都是通过不断的实践和学习达到顶峰的。编码并不难，但你对它的耐心和热情将受到考验。
