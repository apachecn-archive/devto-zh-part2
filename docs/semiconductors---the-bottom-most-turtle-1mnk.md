# 半导体——最底层的海龟

> 原文：<https://dev.to/jacoby/semiconductors---the-bottom-most-turtle-1mnk>

我**现在应该**思考和写关于数据库的东西，但是我的大脑不会去那里。所以，我要去别的技术上的地方。

我写代码。或者，正如布莱恩·威斯特所说:

> “我是一个用闪电和神秘指令赋予摇滚生命的邪教成员，让每一天都变得更快、更可怕。”

[丘奇-图灵命题](https://en.wikipedia.org/wiki/Church%E2%80%93Turing_thesis)使得晦涩难懂的指令变得更加简单，该命题是“每一个有效可计算的函数都是可计算的函数”。这可以并且已经被带到了计算和哲学相遇的点，但是在实际意义上，如果你能为它制作一个电路，你就能为它制作一个函数，反之亦然。而且，如果你能为它创建一个函数，你也可以用一种你更容易理解的语言为它创建一个等价的函数，所以 Python 转换成或包装了 C 语言，C 语言转换成机器语言，这是机器接受的一组神秘指令。

但是我们怎么说服 rock 把灯光变成猫图呢？通过改造岩石。通过选择哪块石头。

有导电的东西；它们被称为导体，我们用它们制作电线。

有些东西不导电；它们被称为绝缘体，我们用它们来处理用电的事情，而不用电击自己。

但是还有 [*半导体*](https://en.wikipedia.org/wiki/Semiconductor) ，它们是什么？它们是[类金属](https://en.wikipedia.org/wiki/Metalloid)，具有金属和非金属的属性，并且不擅长导电。但是通过添加其他元素，它们可以在非常小的水平上被制成非常差的导体或绝缘体，这就是为什么我们用硅而不是铜或金来制造芯片。

吉他手可能听说过在早期(因此更好)的模糊踏板中使用的其他材料，特别是锗。他们的音调可能高度依赖于环境因素，以至于人们被告知将踏板放在冰箱里以获得更好的声音。此外，因为计算爱好者不是主观的“音调怪胎”，所以硅是目前更普遍的选择。([这并不是说我们不再尝试锗了。)](https://www.purdue.edu/newsroom/releases/2014/Q4/germanium-comes-home-to-purdue-for-semiconductor-milestone.html)

在这上面是形成电路的门，和一系列逻辑[【海龟】](https://en.wikipedia.org/wiki/Turtles_all_the_way_down)，堆积到你正在阅读的网页浏览器，但显然，最底部的海龟是我们处理实际形成芯片的原子的地方。
