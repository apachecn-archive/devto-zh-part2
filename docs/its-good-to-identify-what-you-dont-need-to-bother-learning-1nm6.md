你单子上的东西我也有。

不久前，我想学习一切，并认为这是可能的。一些职位描述让我觉得我需要知道所有的事情。现在我知道得更多了——有太多的事情，我需要专注于从我的工作中获得任何显著的结果和满足感。

基本上，我现在专注于前端，而且主要是逻辑和架构，而不是外观。我会做 CSS 但是我不喜欢。我可以用 javascript 做后端，但是前端有更多有趣的东西。现在，我想在更深的层次上学习 React 和它周围的一切。其他的事情目前都不在我的“学习清单”上。

这个帖子里还有很多我没有强调的答案。关键是管理你自己的期望，接受你永远学不到所有东西的事实，这很重要。

没有人告诉你以后不能改变你的想法，但是在软件开发的某些领域落后，而在其他领域更深入是可以的。管理这种相互作用对你的职业发展很重要。
