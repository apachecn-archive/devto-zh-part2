# 学习 Python 的资源

> 原文:[https://dev . to/Jessica garson/resources-for-learning-python-hd6](https://dev.to/jessicagarson/resources-for-learning-python-hd6)

大约一周一次，一些刚开始写代码的人联系我，问我是否知道任何学习 Python 的资源。由于我目前正在 NYU 大学教授 Python 入门课程，这个问题已经成为每天都会遇到的事情，所以我想我应该列出我最喜欢的问题。

下面是一个学习 Python 的推荐资源列表，最初发布在我的 GitHub 资源库中，是为我现在教的班级准备的。

## [](#just-getting-started)刚刚入门

这里有一些资源可以帮助你开始学习如何编码。

*   [潜入 Python3](http://www.diveintopython3.net/)

*   [艰难地学习 Python](https://learnpythonthehardway.org/python3/)

*   [用 Python 自动化枯燥的东西](https://automatetheboringstuff.com/)
