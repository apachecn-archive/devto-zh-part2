到目前为止，指导是一个人一生中最奇妙的学习经历！

指导是一个特殊的时刻，在这里你真正地、深刻地与他人联系在一起。这是你可以分享你的激情、希望、目标、梦想和疑虑的时刻。这是一个你可以指导或被指导的时刻，在这个过程中，唯一的目标是，最终，无论是从专业角度还是个人角度，导师和学员都发现自己在生活中高出一个层次。

在我的职业生涯中，我有很多机会与优秀的年轻专业人士一起工作，有幸成为他们的导师，帮助和指导他们通过“原力之道”。成为一名导师对我来说总是一次奇妙的经历，让我意识到我们可以对一个人的职业生涯和生活产生巨大的积极影响。直到今天，做这种类型的工作对我来说是一种瘾和一种乐趣，是我无论去哪里，只要有可能，都非常热衷于做的事情。

但这还不是故事的全部！

与此同时，我有机会与聪明的头脑、经验丰富的专业人士一起工作，他们赢得了我的钦佩和尊重，我从他们身上学到了很多东西。我很自豪地说我是学员。直到今天，我仍然接近那些我的榜样，那些我钦佩和尊敬的专业人士，我请他们做我的导师。我请他们指导我，与我分享他们的知识，与我分享一条建议。我请求他们接受我作为他们谦卑的学生。

我参加的每个项目，我所在的每个团队，我所在的每个社区，我参加的每个会议和聚会，我都努力帮助那些来征求我的意见、寻求我的指导的人。我参与其中，并在这个过程中学到了很多东西。我也寻找那些了解我感兴趣的主题和话题的人。我遵循他们的指导和建议，并向他们回报我的承诺。

他们所有人，包括导师和学员，都赢得了我深深的尊重和认可。

你可以放心，现在，你周围有很多人拥有丰富的知识，并且愿意分享这些知识。非常乐意分享激情的人们。与此同时，你周围的很多人都在寻求指导，寻求帮助。向前看，给他们一个机会，询问他们知道什么，寻求一些指导，提供帮助，不要羞于寻求帮助。

上周，我加入了 [Meet a Mentor](http://meetamentor.co.uk/) ，这是由 [Barry Cranford](https://twitter.com/bcrecworks) 领导的一个令人惊叹的倡议，它始于[伦敦 Java 社区组](https://www.meetup.com/Londonjavacommunity/)邮件列表上的一个讨论。这个社区的主要目标是提供一个连接导师和学员的平台，所有这些都是免费的！

我加入 [Meet a Mentor](http://meetamentor.co.uk/) 的主要目的是为自己找到一位导师。与此同时，这也是我帮助其他专业人士的绝佳机会，这些专业人士希望分享他们的激情和梦想，寻求帮助和指导。

我邀请你们所有人今天参加[会见导师](http://meetamentor.co.uk/)，我要求你们准备好帮助和被帮助。不管你的梦想和目标看起来有多疯狂，多棒，多困难。那里有人可以帮助你，或者那里有人需要你的帮助。

当你在那里的时候，我会非常高兴地说“嗨！”敬你。
