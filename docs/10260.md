# 我们设计了一个软件产品，但从未被制造出来😕

> 原文:[https://dev . to/jaymeedwards/that-time-I-designed-a-software-product-that-never-get-build-42gi](https://dev.to/jaymeedwards/that-time-i-designed-a-software-product-that-never-got-built--42gi)

我参与过的最令人沮丧的软件项目之一，是当我们设计了一个产品却从未被生产出来！

这个项目中发生了一些不同的事情，使得它变得很困难...

首先，项目被销售为使用敏捷过程，但是*在最后一分钟*突然被转换为固定成本。

其次，管理岗位上的人太多了。

这造成了混乱，因为*团队成员得到了相互矛盾的指令*。

客户告诉每个经理不同的事情！

最后，客户以前从未设计过软件产品。

我们试图警告他们设计一个比他们的整个愿景更简单的产品，但他们不听。

因为他们试图预先设计产品，*他们设计了一个对于第一次发布来说太复杂的产品*。

尽管我们尽了最大努力让项目保持在正轨上，但最终对他们来说建造 T1 太贵了。
