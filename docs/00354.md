### 最佳命令行实用程序

作为开发人员，您可能最常使用命令行。这里有一些命令行实用程序，可以让您的生活稍微轻松一些。

### [【http ie】](https://github.com/jakubroztocil/httpie)

HTTPie 是一个命令行 HTTP 客户端。它的目标是使 CLI 与 web 服务的交互尽可能人性化。它提供了一个简单的命令，允许使用简单自然的语法发送任意 HTTP 请求，并显示彩色输出。HTTPie 可用于测试、调试，以及通常与 HTTP 服务器的交互。

[![httpie](../Images/7cbe41795a25ac2c5673fea83b777f9f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--MkOh_rwe--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/nyf2vtf5u3s08qoc717y.png)

### [cheat.sh](https://github.com/chubin/cheat.sh)

顾名思义，cheat.sh 是世界上最好的社区驱动的小抄库。它有一个简单的 curl/浏览器界面，涵盖了 56 种编程语言、几个 DBMSes 和 1000 多个最重要的 UNIX/Linux 命令。它随处可用，无需安装。

[![cheat.sh](../Images/8a502736a319ae8c6511f7e444be05d4.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--_3cIJlfv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vvfn0t551y3wpn664xn9.png)

### [lazygit](https://github.com/jesseduffield/lazygit)

Lazy 是一个简单的 git 命令终端 UI。它允许轻松地添加文件，解决合并冲突，检查最近的分支，滚动日志/分支差异/提交/隐藏，快速推/拉，压缩和重命名提交。

[![lazygit](../Images/b23f3a9b450bf1b7c45223132af1413f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--FOjSapmH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/c211jp1k9abbhlq973pw.png)

### [任务簿](https://github.com/klauscfhq/taskbook)

taskbook 利用简单、最少的使用语法，这需要一个平坦的学习
曲线，使您能够在终端内跨
多个电路板有效地管理您的任务和笔记。所有数据都被自动写入
存储器，以防止损坏，并且永远不会与任何人或任何东西共享。删除的项目会自动存档，可以随时检查或恢复
。

### [终止器](https://github.com/faressoft/terminalizer)

Terminalizer 记录您的终端并生成动画 gif 图像。由于开发人员有时需要记录一些终端交互，这个工具最适合这项工作。
