# 嘿，我还在编码。

> 原文：<https://dev.to/jenc/well-hey-im-still-coding--4f79>

## 我开始/继续编码是因为...

艺术虚荣心。我在尼奥宠物和我的空间:D 长大

最终，我想在浏览器中制作动画，而不是等待导出视频。

几乎每天我都会看到一些让我们的生活变得更容易的东西，然后想，“我希望我也能做到”。记录警察暴行的 ACLU 视频[app](https://www.aclu.org/issues/criminal-law-reform/reforming-police-practices/aclu-apps-record-police-conduct)，Daniel Shiffman 的[编码火车](http://thecodingtrain.com/)真的让我印象深刻。

所以我想成为一名出色的开发者，这样我就可以为社会事业开发软件。

我在 dev 变得更好，因为我想挣一份可以维持生活的工资。

## 我最近克服了...

我对数据库的恐惧
我对部署的恐惧
我对永远做一个糟糕的开发者的恐惧

## 我想吹嘘一下...

我如何处理失败和冲突。我将如何一遍又一遍地尝试，直到有所成就。

## 我给盟友的建议是支持女性主义者的守则....

倾听与你不同的人的心声，并和他们一起出去玩。为你认识的人制作一个维基百科页面，并试着维护它。只有在他们要求的时候才提供建议。
