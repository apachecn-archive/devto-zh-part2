# 寻找动力

> 原文:[https://dev.to/zellwk/finding-motivation-3dhf](https://dev.to/zellwk/finding-motivation-3dhf)

[https://www.youtube.com/embed/tlEB85OXuUU](https://www.youtube.com/embed/tlEB85OXuUU)

许多人宁愿放松也不愿工作。

他们说他们没有工作的动力。他们需要有动力。

听起来很熟悉？

## [](#you-dont-need-motivation)你不需要动力

你已经有动力了。这就是你一开始想工作的原因。所以不工作才会有负罪感。

你不缺乏动力。

问题不在于动机。

问题是你没有按照你的动机行动。

你让你的阻力阻止了你。

## [](#you-need-a-system)你需要一个系统

在《转向专业》一书中，史蒂文·普雷斯菲尔德讲述了专业人士是如何出现并每天完成工作的。

他们不喜欢的时候就会出现。他们累了还是会出现。

他们每天在同一时间开始工作。他们不会停止，直到他们完成了他们设定的目标。

你需要一个这样的系统。

你需要创造一个你可以工作的环境。设置开始时间。设置结束时间。不要让任何事情分散你的注意力。

坚持做下去，你会养成习惯的。当你养成了这个习惯，维护这个系统就变得容易多了。

## [](#decide-what-youre-going-to-do-in-advance)提前决定你要做什么

想学 JavaScript，不把“我想学 JavaScript”作为任务，因为 JavaScript 太宽泛了。

你可以缩小范围，说“我今天要关注函数”。第二天，你可以说“我要学循环”，等等。

在前一天晚上(或前一周)把大任务分解成小任务。

## [](#recharge-before-your-session)会话前充电

工作需要能量。

你应该为自己充电。

不同的人充值不同。我通常冥想，小睡一会儿，或者淋浴。

找出适合你的方法，并在你的疗程开始前做。
