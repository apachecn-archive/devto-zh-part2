# ★今天我们的 PHP 包下载量达到了 1000 万

> 原文:[https://dev . to/freekmurze/today-we-hit-1000 万-php-package-downloads-ige](https://dev.to/freekmurze/today-we-hit-10-million-php-package-downloads-ige)

在过去的几年里，我工作的公司 Spatie 发布了许多 T2 的 Laravel 和 T4 的 PHP 包。这些包主要是为了在我们自己的项目中使用而构建的。我们不是在真空中运作。我们周围有一个社区。他们利用我们的工作。他们通过提交问题和打开拉取请求来帮助改进我们的包。

今天我们到达了一个显著的里程碑。据 Packagist 称，我们的软件包的总下载量已经突破了 1000 万大关。我们要庆祝这一成就，并感谢我们的贡献者。我们已经推出了一个网站，每个贡献者可以查找他或她做出了哪些贡献。你可以在这里找到它:[https://10million.spatie.be/](https://10million.spatie.be/)

[![](../Images/d9b82f18216ace11fb0b548899d53b30.png)T2】](https://10million.spatie.be)

更神奇的是下载速度。两周前 Packagist 报告说我们的软件包每个月被下载一百万次。这个数字正在迅速上升。按照这个速度，我们将在今年年底或明年年初达到 2000 万次下载。

有人告诉我们，我们公司很特别。不是的。我们是一家 6 人公司。我们只是热爱我们的工作。我们每天都在开源上做一些工作。如果你想知道更多关于我们的开源故事是如何开始的，以及我们是如何管理事情的，[阅读这篇博文](https://freek.dev/the-story-behind-our-open-source-efforts)。

在我们公司网站的开源页面上，你会找到我们已经发布的每个 PHP、Laravel 和 JS 包的列表。我们有一些非常酷的东西在我们的管道中，我们迫不及待地向你展示。致我们所有的贡献者:非常感谢，你们都很棒！
