## [](#getting-started)入门

首先初始化项目。我们的解决方案使用 [Create React App](https://github.com/facebook/create-react-app) ，但是您可以随意使用任何东西，甚至是静态 HTML。

查看[黑客新 API](https://github.com/HackerNews/API) 文档。您将需要/topstories 端点来获取故事 id 列表，并需要/item 端点来分别获取每个故事的数据。

最后，将您的项目投入生产。我们选择使用 Chrome 扩展，但是你可以用任何你想要的方式来管理它。GitHub pages 是一个很好的选择。最终的解决方案应该显示黑客新闻的头条新闻列表。

加入 [Slack 频道](https://community.gitconnected.com)进行协作并获得帮助。这个项目可能会很难，所以一定要和一些朋友一起做！
