# 然而，斯蒂芬妮·韦恩编码

> 原文：<https://dev.to/stephhappens/nevertheless-stephanie-winn-coded--38fe>

#### 我开始编码是因为我怀念从零开始构建东西的创造力以及随之而来的成就感。当我还是一个少年的时候，我就在制作 Myspace 的布局时摆弄过 HTML 和内联 CSS。当时我不知道我在编码。对我来说只是好玩！在我们进行了职业能力测试后，我的测试结果显示我会成为一名出色的系统工程师或 IT 专业人士。我在农村长大，班里的大多数女孩都嘲笑我，这让我很尴尬。尽管我热爱计算机，但我最终还是攻读了公共关系学位。作为一名公关专业人士，我所做的一切都感觉很糟糕，从来没有真正成功过。终于找到了铁码(RIP)，意识到了自己的职业一直缺失的东西！

#### 在我学习的时候，我会处理那些阻碍我实现目标的评论。关于学习缓慢、了解不够的评论我甚至曾经在一次 Javascript 会议上被问及是否是招聘人员。尽管我还只是个学生，但我一直希望这些是孤立的事件。作为一名网络开发人员，我现在意识到，要让女性在科技领域工作变得安全，我们还有很长的路要走。但是我不会放弃。

#### 我知道我属于这里，我将继续打破计算机程序员“看起来”的模式。我感谢所有让技术发展到今天的女性，我很高兴能留下自己的印记，让女性和非二进制人群更容易看到这是一个安全和包容的环境。
