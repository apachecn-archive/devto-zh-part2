# 微软 CDAs 谈软件开发人员软技能的重要性

> 原文:[https://dev . to/_ Patrick god/Microsoft-cdas-on-the-importance of-soft-skills-for-soft-developers-16o 5](https://dev.to/_patrickgod/microsoft-cdas-on-the-importance-of-soft-skills-for-software-developers-16o5)

大多数开发人员对他们的职业和生活都有一定的目标。

对我来说，在很长一段时间里，就是发布我自己的独立游戏。我达到了那个目标。

但是我想，要实现任何目标，不仅仅是技术技能的问题。交流、个人成长、走出你的舒适区和建立这些软技能也非常重要。

在法兰克福的微软技术峰会上，我遇到了以下微软 CDA(云开发者倡导者),他们肯定拥有许多开发者渴望的职位，我询问(并记录)了他们对软技能对每个开发者的重要性的看法。

我问的第一个人是 Shayne Boyer。他在 11 岁时编写了他的第一个基于文本的角色扮演游戏，现在是 CDA for。NET/。他说，这都是关于交流的，你绝对应该**加入当地的团体，接触新朋友，交流你的想法**——在工作中也是如此。

以下是(短)视频:

[https://www.youtube.com/embed/eSz_5wOGd8E](https://www.youtube.com/embed/eSz_5wOGd8E)

接下来是扎卡里·德普塔瓦和 T2。Zach 的主要专业背景是 Linux，因此他也是 Linux 的 CDA(是的，是微软的 Linux CDA。欢迎来到 2018 年！).他认为你应该说出你的故事，变得有亲和力，克服你的恐惧。如果你这样做了，**很棒的事情正在发生。**

Bernd 最初来自德国。他在旧金山学习了六年，然后为谷歌工作，然后为 Twitter 工作，甚至为政府工作，现在是微软 Linux、容器(Kubernetes)和 Python 的 CDA。在他看来，**导师很重要**。两者都是，得到指导，你自己也是导师。
