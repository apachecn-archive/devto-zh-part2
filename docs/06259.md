## 非编码方案

不是每个人都是程序员，但这个世界充满了可能性，没有人能阻止你创建自己的创新应用程序

### ·Sketchware

> 如果你完全是新手，我强烈推荐 Sketchware 给你

通过一些块编码，创造性的想法和一点点耐心，你可以用它创建下一个趋势应用。

Sketchware 非常容易理解，有巨大的潜力，如果你停留在任何地方，他们有文档，论坛，slack 频道以及子编辑。

也适合创作游戏。

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.besome.sketch)

### 【AppyPie 应用构建器】

如果您的工作需要专业的商业应用程序，这是您的最佳选择

AppyPie App Builder 中有大量可用的模板，其中一定有一个符合您的特定需求。

只需选择一个模板，定制它，给它一个名称和 id...您的应用程序即将就绪。

这是我发现的最简单的解决方法之一

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.letsappbuilder)

## 编码方案

### 援助

这是通过编码构建应用程序的最佳解决方案。这就像 Android 上的 Android Studio

这是迄今为止最古老、最常用的 Android IDE 之一，它支持 Java 和 C++

它有强大的编辑器，语法高亮，自动完成等等

这是一个用于构建和学习应用程序开发的整体解决方案。如果你是一名程序员或者正在学习编程，我强烈建议你这么做。

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.aide.ui)

### ·代码错误

这是一个有巨大潜力的应用程序，但现在什么都没有了。

它有一个强大的编辑器，语法高亮以及许多其他强大的功能。

它有一个缺点，除非你购买完整版，否则你必须等待几分钟才能重新启用它。

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.majosoft.anacode)
