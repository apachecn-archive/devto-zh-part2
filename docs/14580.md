# Devs，为什么在 tmux windows 上使用终端标签？

> 原文:[https://dev . to/therma tix/devs-why-do-you-use-terminal-tabs-over-tmux-windows-51hb](https://dev.to/thermatix/devs-why-do-you-use-terminal-tabs-over-tmux-windows--51hb)

快速思考一下，我注意到很多开发者似乎使用终端标签，甚至在使用有窗口的 tmux 时，所以我很好奇，为什么在 tmux 窗口上使用终端标签？

需要指出的一点是，当你使用标签时，它会占用一些屏幕空间，而这些空间可以用来显示更多的终端内容。通过使用 tmux 窗口，它会在 tmux 状态行中显示打开的窗口，而不是单独的一行。
