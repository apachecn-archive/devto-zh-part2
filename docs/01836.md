# 系统安装人员

都在`utils/`文件夹里。在您的系统上安装 Sparky 时很有用:

*   `$ sparrowdo --local_mode --sparrowfile=utils/install-sparky-web-systemd.pl6`

*   `$ sparrowdo --local_mode --sparrowfile=utils/install-sparkyd-systemd.pl6`
