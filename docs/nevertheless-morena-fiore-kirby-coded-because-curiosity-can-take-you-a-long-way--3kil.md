# 尽管如此，莫雷纳·菲奥雷·科尔比还是进行了编码，因为好奇心可以带你走很长一段路。

> 原文：<https://dev.to/morena/nevertheless-morena-fiore-kirby-coded-because-curiosity-can-take-you-a-long-way--3kil>

我是一个十几岁的独生子，在意大利的一个小镇上，呆在家里很无聊，在 1 点到 4 点之间你不能出去，因为每个人都在吃午饭或睡觉。我总是在寻找新的刺激，所以在那些不虔诚的时间里，那意味着探索 Windows 98 和 XP 的每一个功能；或者摆弄 Poser 的人体模型，或者尝试安装的每一个剪贴画、音频或字体。

然后我爸妈给我弄了互联网！我忍不住好奇，开始想网站是怎么做的？我用图像地图创建了我的第一个(对不起！)和表格进行布局(太抱歉了！)，尽管它们只是崇拜我最喜欢的足球运动员尼古拉·文托拉和电视节目《道森小溪》的网站。

我开始提出的下一个问题是，我如何向其中添加数据？于是开始学习 PHP 和 MySQL。我应该补充一点，我从未真正学过计算机科学，我的父母供不起我上大学，我的人生目标是搬到伦敦。在这里，我试图去上大学，但经济上搞砸了(这是我一生中最悲伤的事情之一)，我不得不在三个月后辍学。

所以学习计算机科学的基础知识从来没有真正发生过。我开始挑选我需要学习的东西，并在途中挑选我需要的东西。

今天，我知道我无法在白板面试中幸存下来，也无法告诉你什么算法可以做什么，或者无法给你我日常使用的大多数东西一个合适的定义，但我会跳入任何摆在我面前的新技术的海洋中，不管有多高。

最重要的是，我每天都想学。

盟友们，多亏了你们，像我这样非常愿意学习，但并不总是有办法学习的人才能增长知识和信心。感谢你们中的每一个人，他们向我解释了一些事情，并且相信我的技能。
