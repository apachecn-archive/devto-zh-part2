# 给我父亲的一首情诗

> 原文:[https://dev.to/amandasopkin/a-love-poem-for-my-ide-3197](https://dev.to/amandasopkin/a-love-poem-for-my-ide--3197)

我能否把你比做夏季的一天？
你更可爱也更公式化:
可怕的 bug 确实动摇了我所有代码的基础，
有时候我真想把我的电脑扔掉:

有时我怀疑你的进口系统，我经常诅咒你的名字；每一个异常有时都会中断；
是我的错，还是你的错，还是大自然的变化

但你的调试套件不会褪色，
也不会不揭露一些可憎的事
也不会 vim 取代你的魅力，
当我的抱怨被写在永恒的诗句里:
只要发展者能搞砸，或眼睛能看见，
只要我的挫折存在，这些给你生命
