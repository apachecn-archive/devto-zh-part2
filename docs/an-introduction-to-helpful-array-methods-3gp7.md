# 有用的数组方法介绍

> 原文：<https://dev.to/seandershancevic/an-introduction-to-helpful-array-methods-3gp7>

无论如何，我都不是编码方面的专家，事实上，我目前正在参加一个全栈开发课程，而我才刚刚上了一半。也就是说，我学到了一些关于数组以及如何处理数组中存储的数据的有用的东西。然而，这篇文章只是关于数组方法。我学到的关于数组方法的第一件事是，这些方法将为您遍历数组，根据您使用的方法搜索数据。这意味着我们不需要使用“for”循环来遍历整个数组，方法会为我们做这件事。

第一个是 forEach()数组方法。这个方法遍历整个数组，基本上是抓取数组中的每个元素，然后你可以对这些数据执行不同的操作。

[![](../Images/ca0b01ae9533b2291db3b20bca7426f9.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--5NI-4Qhv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/6ud52wojtiiclvo3qhsb.png)

下一个是 filter()方法。这允许我们根据给定的规格从阵列中过滤出数据。

[![](../Images/a9eaad7e303765c5c6e3875e2eaf7774.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--o3bGdtll--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/isx7n0snsaounznqr2re.png)

另一个有用的方法是 includes()方法。这将遍历数组并查找任何具有指定参数的元素，类似于 filter()方法。

[![](../Images/4a63ada3c06f8d508d7de91d227ce505.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--_x2Cy4-q--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ceyhjgaylwbkmmmcyiuq.png)

push()允许你在数组末尾添加新元素。这对于制作任何类型的列表都很有用。

[![](../Images/1dd5a9789714407f4cc8e82d0e1419d9.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--DLVByfW4--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/84uq338fh8lah3alrgun.png)

最后，splice()是一个非常有用的方法，因为它允许将元素“拼接”到数组中，或者添加到数组中，但是它也可以以相同的方式删除或“拼接”出项目。

[![](../Images/06a653576276cfdbd5e80b90fc7a55b6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--MjNHwgA6--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/jvp1yc7zfui4y1lpbz8h.png)

这些只是一些不同类型的数组方法，但是我发现其中一些特别有用！
