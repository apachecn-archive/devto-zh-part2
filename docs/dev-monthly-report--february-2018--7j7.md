# 开发月报—2018 年 2 月

> 原文：<https://dev.to/peter/dev-monthly-report--february-2018--7j7>

三月快乐！感觉二月过得特别快，但我想这是最短的一个月...

让我们来看一些亮点——

研讨会:我们组织了第一批现场活动！ [@ben](https://dev.to/ben) 主持了一场关于 web performance 的讲座， [@cassidoo](https://dev.to/cassidoo) 分享了她对机械键盘的热爱和知识， [@annarankin](https://dev.to/annarankin) 用 React 和 Jest 教授测试， [@aspittel](https://dev.to/aspittel) 展示了如何使用 CSS 创作艺术。我们的日程表上还有更多[重大活动](https://dev.to/events)，所以一定要去看看。

**开源之路**:我们刚刚邀请了第一位外部贡献者🎊🎊开始查看核心开发代码库。这是未来几个月开源开发平台的重要一步。如果您有兴趣尽早获得帮助以提供反馈、影响我们的方向或以任何其他方式提供帮助，请填写此[表格](https://thepracticaldev.typeform.com/to/HkjWky)。

**赞助**:这是我们与社区赞助商合作的第二个月。本在这里有[的完整公告](https://dev.to/ben/introducing-our-march-2018-sponsors--3bj2)，但是我想再次真诚地感谢 Algolia、Triplebyte 和 Manifold 在这个月对我们的支持。他们的支持对我们平台的财务可持续性至关重要。

**DEV Badge** :虽然我们在准备开源的过程中一直都很低调，但是这个月我们还是整合了一些不错的特性。我特别兴奋的一件事是新的[开发者徽章](https://dev.to/p/badges)。考虑在你的个人主页上展示徽章，展示一些开发人员的骄傲🙌🏽

强劲的数字:尽管是最短的一个月，我们在二月份显示了稳定的增长。我们提供了 125 万次服务，在我们的社区中注册会员已经超过 6 万人。有机搜索流量也是📈这意味着更多的人在搜索相关关键词后发现了你的精彩文章。

一如既往，非常感谢你参与这次狂野之旅。本、杰斯和我很乐意回答下面评论区的任何问题。
