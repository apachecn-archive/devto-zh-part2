# Flutter 只是谷歌操作系统总体规划的第一阶段

> 原文：<https://dev.to/jonstodle/flutter-is-only-phase-one-in-googles-os-master-plan-5dgc>

我最近一直在玩谷歌新的跨平台努力，它看起来进展得相当不错。我也一直在思考 Flutter 的未来会是怎样。

在它的第一阶段，它“只是”一种跨平台的方式来编写高性能的可视化应用程序，但我认为这只是谷歌计划使他们的新操作系统 Fuchsia 相关的第一步。

在这一点上，谷歌正在开发新的操作系统已经不是什么秘密了。目前还没有太多的细节，但有一点已经被发现，那就是 Flutter 将成为这个新操作系统的 SDK 或应用开发工具包。在最好的情况下，开发人员将看到他们的应用程序在谷歌的新操作系统上运行时几乎没有变化。

我认为，当试图推出一个新的操作系统时，一个主要的困难是让用户使用这个平台。没有应用程序，用户不会有飞跃，但是没有用户，开发者也不会有飞跃。

然而，谷歌试图创造的局面是，许多应用程序将能够立即在他们的新操作系统上运行。“想要支持我们新的花式平台吗？做这 5 个小小的改变，你就可以开始了！”。或者更好:“运行这个工具，它会告诉你什么必须被修复，以及它是否可以被工具自动修复。”

三星也在尝试推出他们的新操作系统 Tizen。虽然我仍然非常怀疑他们是否会成功取代 Android，但他们也在进行类似的赌注，但与。网芯。。NET Core 将成为 Tizen 的主要 SDK，这使得三星能够通过这种方式进入一个相当大的开发者群体。

这一切如何实现只有在未来才能知道，但我认为谷歌可能有力量和影响力迫使(另一个)新的操作系统进入市场。
