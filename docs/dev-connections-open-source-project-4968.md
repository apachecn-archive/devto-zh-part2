# 开发连接

这是一个简单的开发配置文件，以帮助我们所有人进行沟通，所以开发连接的关键要素是建立联系。

## 如何为这个项目做贡献

*   分叉此存储库(分叉按钮)

*   进行更改:

    将你的名字、github、linkedin 等添加到 CONTRIBUTORS.md 文件中

```
 ## Name: Your Name
      * Place: Country
      * Job: Your Job Title
      * GitHub: [Github Name](Github Link)
      * LinkedIn: [LinkedIn Name](LinkedIn Link)
      * Twitter: [Handle Name](Twitter Link)
      * Portfolio: [Portfolio Name](Portfolio Link) 
```

*   提交和推送
*   去把它给我。
*   git commit -m '第一次提交'
*   git 推送源分支-名称
*   从分叉的存储库中创建一个新的拉请求
*   等待合并批准
