### **我的背景**

首先，让我自我介绍一下。我叫朱利安·克里斯蒂安·安德森，是一名来自印度尼西亚的网站开发人员。我没有任何技术背景，事实上我有工业工程学位。今年 6 月，我完成了我的工业工程学位，在完成学位的前一年，我发现我的热情实际上是在科技行业。所以回到 2017 年，我通过 Youtube 和 Udemy 视频自学了网络开发。几个月后，我决定在最后一个学期做一名自由职业者。当谈到基本的东西时，事情对我来说可能不会变得那么难，但我面临着一个障碍。

### **我的感觉**

几个月后，我遇到了一个障碍，因为我的目标是成为一名全栈 javascript 开发人员。在 Javascript 世界里，有很多你应该知道的框架和事情。我从一个框架跳到另一个框架，因为我被其中一个框架卡住了。从一个框架跳到另一个框架根本没有解决我的问题。最糟糕的是，它不断给我的开发者之旅增加更多的麻烦。那时我甚至不知道我下一步该做什么，或者我该如何解决我的问题。因为当我在 Udemy 注册一门课程时，我不能和那个人面对面，有时很难用文字的形式解释一个问题。我试图在我的城市找到一个社区，但我找不到，所以我把这个问题留给自己。

### 接下来我做了什么？

我试着在我的城市附近找一些工作间。我发现的第一件事是一个位于我所在城市的移动开发者训练营，但我对此毫无兴趣。幸运的是，我在印度尼西亚的雅加达遇到了一个完整的 Javascript Bootcamp。我读了他们的推荐，研究了他们的课程，然后我被他们吸引住了。他们将在七月开始训练营，而我的期末项目的截止日期是六月。我和我的父母谈过这件事，我很感激他们同意了。他们给了我全力支持，然后我试图尽快完成我的最终项目，这样我就可以参加训练营了。

### 什么是编码训练营？

根据谷歌对 bootcamp 的定义是:

> "新兵训练营，纪律严明."

而根据[http://thefirehoseproject.com/developer-guide/1](http://thefirehoseproject.com/developer-guide/1)编码 Bootcamp 的定义是:

> “一个技术培训项目，教授雇主需要的编程技能。编码训练营使编码水平不高的学生能够专注于编码的最重要方面，并立即应用他们的新编码技能来解决现实世界的问题。”

所以基本上这是一个严格的编程训练。但真正强大的是，它只教会我们某种“技能”或“语言”。有很多编程语言，学习它们真让人不知所措。所以解决这个问题的方法是建立一个关注一般编程基础和特定编程语言的编码训练营。

### 你在编码训练营做什么？

所以我的编码训练营分为 4 个阶段。在阶段 0，我只在周一、周三和周五来校园。每天都有一个两小时的讲座，那一周会有作业，我们应该在家里完成。在这个阶段，我觉得它不是那么忙，你仍然可以在讲座后做其他事情。你还会有一次现场编码会议，这基本上是每周测试你的知识的一次考试。此阶段 0 持续 6 周。阶段 0 是关于基本的 Javascript 和编程。我们学习数组、循环、对象等等。

在以一个不错的分数完成了第 0 阶段后，现在我进入了第一阶段。我可以毫无困难地度过 0 阶段，因为我学习 Javascript 已经有一段时间了。没有 javascript 和编程经验，我做不好。阶段 1 到 3 称为“沉浸阶段”。

在第一阶段，我每天从早上 9 点到下午 6 点去训练营。有时我会因为训练营的未完成任务而在深夜回家。因此，在沉浸式阶段的典型一天中，我们有两次讲座，每次 1-1.5 小时。剩下的时间我们都花在了课堂上学到的实用的东西上。训练营通常会在周末布置任务，让我们理解前一周所学的一切。

### 我对 Bootcamp 编码的看法💻

**优点:**

1.  对我来说，对于那些想在短时间内深入了解编程或某种编程语言的人来说，编码训练营是一个很好的解决方案。因为它通常在时间上更短到一个 CS 学位。
2.  编码训练营也是一个很好的注册项目，因为我们在里面学到的一切都集中在编程和语言的重要部分。他们也给我们的作业一个真实世界的范例问题，这样我们甚至在毕业前就能对它有所了解。
3.  在训练营中，我最喜欢的是他们的导师系统。导师在生活的各个方面都是非常关键的东西。在我进入训练营之前，我没有任何网络开发方面的指导，当我遇到问题时，我不知道该怎么办。现在，因为讲师成为我们的导师，我真的可以通过他们在 web 开发领域的指导提高很多。
4.  我也喜欢因为社区而参加训练营。在加入训练营之前，我有少量编码的朋友。我不能和他们谈论科技行业，我也不知道去哪里谈。现在，我有很多有着同样热情的朋友，所以能够在训练营中与很多开发者联系是一件有趣的事情。
5.  最后一件事是在训练营，他们有一个非常结构化的课程，并随着时间的推移得到改善。因此，它将始终是最新的，并在许多方面帮助我们。

**缺点:**

1.  因为训练营的时间很短，我觉得节奏太快了，如果你每天都不集中注意力，你可能会落后。每一天甚至每一次讲座都可以是不同的学习内容。
2.  编码训练营也占用了我们一天中的大部分时间，所以我觉得除了编码训练营的任务之外，我什么也做不了。有时候我真的想探索一些别的东西，但是回家后我觉得很累，就直接上床睡觉了。
3.  你必须全力以赴。不要误解我，我喜欢全力以赴做我做的事情。我以前是一名自由职业者，现在我甚至没有时间做客户工作，因为训练营的任务太多了。所以唯一的办法就是在训练营结束后一直忙到午夜。它会完全减少你的睡眠时间，这完全是一件坏事，因为你第二天早上必须这么早醒来。

#### **那么，我推荐加入一个编码训练营吗？**

如果你有时间和金钱，我强烈建议你加入编码训练营。这有点贵，但对未来来说是一项非常大的投资。

我也建议你在进入训练营之前先学习课程，看看它是否是你需要的东西，否则就是浪费钱。准备好你需要的一切，永远不要停止学习，即使在困难的情况下也不要放弃！
