# 你知道甲骨文拥有 Javascript 的商标吗？

> 原文:[https://dev . to/the outlander/Oracle-owns-the-trade mark-to-JavaScript-1 fil](https://dev.to/theoutlander/oracle-owns-the-trademark-to-javascript-1fil)

我使用 Javascript 已经超过 20 年了，但我不知道甲骨文拥有 Javascript 的商标。如果有的话，我会猜到是 Mozilla 拥有它。

甲骨文在他们收购太阳微系统公司后继承了这个商标(我甚至不知道太阳已经注册了 Javascript！).

我偶然看到 Reddit 上的一个帖子，其中[苹果撤下了一个应用](https://www.reddit.com/r/javascript/comments/8d0bg2/oracle_owns_javascript_so_apple_is_taking_down_my/)，因为 AppStore 在没有得到甲骨文授权的情况下显示了 JAVASCRIPT 这个词。

鉴于这是一个如此普遍的术语，我想知道他们是否会在未来继续执行它。许多人建议，我们应该称之为 ES 前进，到此为止。我不确定我对此有什么感觉。

你知道这件事吗？你有什么想法？
