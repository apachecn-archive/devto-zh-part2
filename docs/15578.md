# 当你全职工作的时候创业是个好主意吗？

> 原文:[https://dev . to/jj Santos/is-a-good-idea-to-start-a-startup-while you-work-full-time-41p 4](https://dev.to/jjsantos/is-a-good-idea-to-start-a-startup-while-you-work-full-time-41p4)

这件事我想了很久。我知道创建一家初创公司本身就很困难，我也知道如果你是一名全职员工，那就更困难了(*我目前正处于这个阶段*)。当你试图在你是全职员工的情况下创建一个新的企业，这是一个复杂的情况，因为某些原因，比如时间；如果你不把时间花在你的项目上，你将无法实现你的目标，或者要花很长时间才能实现。

因此...我的问题是:我目前是一名全职员工，但我也有一个正在进行的个人项目。我读过许多人写的文章，他们说为什么不好，为什么不好，但是我想从一个更人性化的角度来看你对此的看法，以及你是否处于同样的情况。
