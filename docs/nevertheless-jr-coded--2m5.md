# 其一使命

### 向前交

我继续写代码，因为我热爱我的工作。我着迷于我能创造的东西。当我在一个帮助他人的项目中工作并在某些方面有所作为时，这给了我满足感。这就是为什么我在高等教育方面做了很多工作。这也是我在主流项目中贡献易访问特性的原因。

### 有主见不咄咄逼人

我最近参与了一个项目，我的工作归功于另一个团队成员。他剽窃了我的代码，败坏了我的名声。我知道这种情况不会有好结果，所以我利用这个机会以一种自信的方式面对这个人。我陈述了问题，并明确表示我知道他做了什么和不同意什么。然后我开始了更好的生活。

### 帮助其他女人

> 这是所有女性开发者(和我们的盟友)的使命，当我们工作时，通过提供资源和支持来相互指导和帮助。
> 
> 我的理念是专注于创造一个提升其他女性程序员的姐妹情谊的环境。我不关注如何改变一些科技环境中存在的重男轻女态度，而是如何改变我所做的事情来帮助其他女性取得成功。这是我们所有人的使命。

### 我对女性代码盟友的建议

指导、提供资源和沟通。
