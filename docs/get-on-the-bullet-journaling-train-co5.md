# 登上子弹日记列车

> 原文：<https://dev.to/rpalo/get-on-the-bullet-journaling-train-co5>

二月初，我了解了[子弹日志](http://bulletjournal.com/get-started/)。在试用了几个月后，我爱上了它，我想分享一个介绍和我个人的修改和技巧。

## 子弹日记:是什么？

从根本上说，子弹日记只是一种有效记录你的想法的结构化方法，不会给你的生活带来额外的不必要的负担。子弹日记理解你很忙，你并不总是有时间坐下来倾诉你最深处的想法和感受。“亲爱的日记……”相反，你把你的想法浓缩成小块小块的短语，其中最基本的是“子弹”

[![A dot](../Images/fef758b9cab08a4d45dbf889f9beba0c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--yScBf7hI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-dot.jpg)

这个不起眼的点表示一项任务或带有行动项目的事情。我们一会儿将讨论更多的细节。我想先完成我的高级概述。

我们将介绍几个主要策略。

1.  你需要什么
2.  索引
3.  长远观点
4.  月视图
5.  每日日志
6.  列表/注释页面
7.  穿线
8.  不同类型的子弹
9.  迁移和计划

在这些话题之后，我将总结一些心态技巧和我学到的东西。

## 子弹日记:怎么做

### 1。你需要什么

你真的可以使用任何可以记录信息的媒介，纸质的或数字的。然而，我更喜欢用一本精美的精装笔记本，上面至少要有一个书签。我也喜欢用相当厚的纸，上面有图形线或点阵。如果它略带黄色，那就更好了。我买了这个 [![](../Images/280624c638e167aa0266e131a3a3a373.png)](//ir-na.amazon-adsystem.com/e/ir?t=ryanpalo-20&l=am2&o=1&a=1441323716) ，我很喜欢它。我也强烈建议你买一支非常棒的笔。比如，至少五美元的笔。我发现，如果我的工具让我感觉自己像一个学者，这增加了子弹日记的冥想效果。😀由于我的笔记本后面有一个小口袋盖，我喜欢在里面放一把小尺子，以防我想画出任何真正的直线。

现在你有了工具，让我们开始吧。

### 2。索引

索引是把你的整个日记联系在一起的东西，让你可以在任何需要的时候做笔记，而不用担心给自己留多少页。打开你的笔记本，翻到第一页。在这两页和后面两页标上“索引”字样

[![The blank Index section](../Images/6576a6686b08d3198fb786aa2996de6c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--yGZfwAcJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-index-blank.jpg)

此外，在这四页中的每一页的底部外侧角落，对页面进行编号(1、2、3 和 4)。这个索引会告诉你日志中不同部分的开始和结束。因为我们的日志中还没有任何内容，所以我们的索引是空白的。我们将在添加接下来的部分时填充一些内容。最重要的是要知道，当我们添加部分时，它们最终会是这样的:

[![An index item](../Images/e15d6ab00845d3270967b1822728296a.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--OiSdZ8TY--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-index-item.jpg)

因为我们的日志是灵活的，它允许我们用其他部分打断部分。我们用逗号分隔的页面区域来跟踪这些跳转。

### 3。长远观点

长远观点是对未来六个月的两页浏览。打开笔记本，翻到索引页后的第一页空白页。如果你有一把尺子，现在是使用它的时候了。在每页上画两条水平线，将这两页分成六个相等的区域。在左上方用未来六个月标记这些部分。

[![The blank Long View section](../Images/a8f075cf55a59fdab3343f0b645d5eb0.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s---XcZoTpH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-long-blank.jpg)

此部分用于保存长期目标或不能立即执行的任务和信息。如果有人邀请你参加三个月后的婚礼，这将是最完美的部分。使用项目符号在此部分添加一些目标。当你完成这些任务后，你可以在要点上画一个小 X。稍后我们将更多地讨论不同项目的不同项目符号样式。

您将使用这一部分。在你接下来几个月填更短期的部分时提醒你一些事情。而且，因为我们已经在我们的杂志上增加了一个新的部分，不要忘记在每一页的底部加上页码，并且把这个部分加到你的索引上。

[![Adding the Long View to the Index](../Images/d812c93891f5cddd428ad9094e5c4fa2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--3MJLa2ev--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-index-long-view.jpg)

Bullet Journaling 是一个持续的过程，它记录当下和短期的事情，同时频繁地放大图片。当我们继续接下来的几个部分，逐渐放大你的生活时，你会看到这一点。

### 4。月视图

太好了！既然我们已经掌握了六个月的目标，下一个主页面类型就是月份概览页面。在长视图之后，打开你的日志到下一个空白跨页。用当前月份的名称作为两页的标题(这应该是你的长视图中的第一个月)。左侧页面将成为您的日历页面。把一个月中的每一天填入垂直栏。对于 3 月，这将是 1 到 31，每个人都在自己的行上。就在这些数字旁边，放一两个字母来表示这个日期是星期几。我也喜欢在每个星期之间画一条水平线，帮助我更容易直观地看到我在哪里。

[![Calendar month view](../Images/7f9533e751c9a060f52399a1347efd8e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--OzJZzI5h--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-month-calendar.jpg)

此跨页的右页将是项目符号页。写下你能想到的这个月将要做的事情、任务、目标或笔记。我喜欢把任何重要的日期/节日/生日写在数字和字母旁边的左边。一定要回到你的长远观点，看看是否有什么可以导入到你的第一个月部分。这将是一种常用的技术。

[![Task Month View](../Images/91d1f29e5de4cdccf8074433e2abfd77.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--BR7bAhIm--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-month-task.jpg)

完成后，不要忘记添加一些页码，并将此部分添加到索引中。

### 5。每日日志

这种条目类型将是您在项目符号日志中所做的大部分工作。再一次，打开你的日志，在你第一个月的视图后，打开第一张空白的表格，在上面标上“(月份名称)每日日志”。然后，在页面上放置一个带有今天日期的标题。把你今天想完成的任务或待办事项放进去。就像之前一样，不要忘记翻回到月视图甚至长视图，看看是否有需要导入到今天任务中的项目。

试着现实一点，但是也不要害怕设定一些挑战性的目标。如果你今天没有完成你计划的事情，不要紧张。一次只做一天。在一天结束的时候，我喜欢记下我那天做的其他事情或者发生的其他事情。我发现，随着时间的推移，这帮助我更好地记住了过去的日子和星期，帮助我把每一天都变得更加特别。我倾向于在一周结束时记得我完成的事情，避免事情变得单调。

[![Filled-in Daily Log](../Images/b7b069e55e04f7808a4cde61173e3dc3.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--AxhZjd9L--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-daily-log.jpg)

添加一些页码，并将你的每日日志添加到你的索引中。因为你不确定这将占用多少页，只需输入起始页码和破折号。如果发生了其他事情，打断了您的每日日志，您可以添加结束页码和一个逗号，并在再次开始备份时将每日日志的下一部分放入索引中。我们将在线程部分详细讨论这一点。

### 6。列表/注释页面

可能会发生这样的情况，你正在浏览你的每日日志，你决定看一个如何做的视频或参加一个在线课程——你想在那里做笔记。我在阅读编程书籍和教程时经常这样做，我发现它真的帮助我更好地记住了一些东西，并且给了我一个比不断地搜索相同的东西更好的信息来源。简单地在页面顶部放一个标题，给你自己几个空白页来记笔记，然后开始记笔记。到达每一页时添加页码，并将此部分添加到索引中，以便以后可以再次找到它。

[![List Page](../Images/788dad5525136fc61c402bca671093de.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--uKwGv8In--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-list-page.jpg)

### 7。穿线

这是子弹日志的关键技术之一，也是它如此灵活的原因。这就是你如何安全地用一张购物清单或一页笔记打断你的日常日志，而不用担心中间会留下一大堆空白页。您的笔记分区可以相互中断，并且可以随意停止和开始多次。它只是让你的索引看起来像这样:

[![A skipping Index entry](../Images/e15d6ab00845d3270967b1822728296a.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--OiSdZ8TY--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-index-item.jpg)

我还发现，在一节的页码上方给自己记下下一节的开始或结束位置是很有用的，这样，如果不需要的话，我就不必一直翻索引了。

[![Threading note above a page number](../Images/b7410e0c5e55d762db714334726c2e41.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--nD-aCl9_--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-threading-note.jpg)

### 8。不同类型的子弹

正如我们已经讨论过的，项目符号日志的基本任务单元是一个点。

[![A dot](../Images/fef758b9cab08a4d45dbf889f9beba0c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--yScBf7hI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-dot.jpg)

当你完成这个任务时，你应该在它上面画一个小 X，像这样。

[![Different bullet types](../Images/cbe52a129ee66bb0a2a52cf896781039.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--tu_OQtzE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-types.jpg)

子弹日记的主要目的之一是帮助你剔除那些不值得你花时间的事情。如果你决定一项任务不能完成或者你不想做，就在上面划一条线。

如果你想记下一些事情，但这并不是一项真正需要完成的任务，那就用破折号。

如果你有一项重要的任务，或者你想引起别人的注意，在它旁边标上一个小星号。

如果你有一些令人兴奋或整洁的东西(通常搭配一个信息/注释破折号)，在它旁边放一个小感叹号。

如果某样东西需要更仔细的观察，在它旁边放一只小眼睛(或者类似眼睛的东西)。

### 9。迁移和计划

在每个月底，你应该检查前一个月的日志和前一个月的每日日志。找出所有未完成的任务。现在是一个很好的时机，把那些最终无关紧要或没有完成的事情划一条线。如果还有一些事情需要完成，你有几个选择。你可以放一个向前的箭头，推进你下个月的月日志或者你已经创建的收藏/列表页面。这被称为“迁移”任务。

[![A migrated task](../Images/47a23f5df4c03959ce6d2df726c04b95.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--H8wENj7s--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-migration.jpg)

或者，您可以将其发送回长期视图，以便在将来的某个时间完成。在它前面放一个向后的箭头。然后一定要把它写在远景中合适的地方。这被称为“安排”任务。(我喜欢称之为给未来的我制造问题)。

[![A scheduled task](../Images/e3e1dea6203aae73169873e08167ea64.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--1ECBuyeo--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://assertnotmagic.com/img/bullet-scheduling.jpg)

## 锁定并加载

如果你已经做到了这一步，你正在成为一名子弹日记大师的路上。以下是要带走的主要东西。子弹应该很快，所以它们不会成为你时间表上的负担，但它们也不必如此。如果你有额外的时间坐下来做更详细的笔记，那对你有好处。

这种方法被设计成低应力的。如果你忘记做每日日志，不要惊慌。第二天补上，做一些信息破折号来提醒你那天做了什么。如果你让它成为一件有压力的事情，那么当你错过一天的时候，你会因为不得不做两天有压力的事情而感到难过，你会更想跳过它。保持一致的最好方法是让你的子弹日记成为一个快乐的地方，不要因为做错了而自责。放自己一马。

最后的提示:确保把你的名字和联系信息放在你的子弹日志的最前面。你不想丢失它，当他们找到它的时候，没有任何人联系你的机会，是吗？

欲了解更多信息，请访问官方[子弹杂志网站](http://bulletjournal.com/)。他们有很多很棒的资源，你甚至可以从他们那里买到真正的“子弹期刊”。

每个人写日记的方式都有所不同。我妈妈喜欢在她的每月日志中加入真正的艺术和色彩。我很少在日记里画画。我喜欢项目符号、直线和一致的格式(因此我的日志内置了标尺袋)。它安抚了我。我想听听你的意见。如果你有另外一种我没有提到的你想放在你的日志里的页面，在评论里给我看看，或者发一张图片给我，我会和我的粉丝们分享(两位数，宝贝，哇哦耶！).
