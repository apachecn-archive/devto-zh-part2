# 基于 CookieCutter 的网络项目生成器

> 原文：<https://dev.to/kostjapalovic/cookiecutter-based-project-generator-on-the-web--5d67>

我最近开始了几个 Django 项目，主要是尝试一些新技术和自动化我的一些工作流程。

然后我想，有一个 [cookiecutter](https://github.com/audreyr/cookiecutter) 项目可以自动完成项目创建的大部分工作。

虽然这是一个很棒很有用的工具，但我不喜欢在终端中一行一行地回答问题来生成项目。

我想要一个网络界面。

几个小时后，我想出了一个解决方案，它驻留在这里:[发电机。KPavlovsky.pro](https://generator.kpavlovsky.pro/)

Python 和 Python-Django 项目包含在可用的模板中。在感受带有问题的表单(取自 templates' cookiecutter.json)并点击`Generate`后，您将获得新项目的 ZIP 存档。

希望它至少对某些人有用。
