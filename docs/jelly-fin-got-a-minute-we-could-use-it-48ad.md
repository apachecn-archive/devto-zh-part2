我想建立一个好的框架，这样我们就可以在前进的过程中添加一些东西，但是我一直在和 Jest 作斗争，并且很好地反应了 Native playing。如果你有一些时间，叉/克隆我们的项目，并帮助我们得到一个简单的冒烟测试 Travis/Jest，以确保我们的应用程序可以渲染。
