# 为什么编码应该成为发展中国家小学的必修课？

> 原文：<https://dev.to/hopperyelena17/why-coding-should-be-compulsory-in-primary-schools-of-developing-countries--1fko>

可以肯定地说，全世界的技术进步都是由信息技术的发展推动的。软件开发人员和程序员不断尝试新的软件应用程序和算法，为利益相关者创造价值，并彻底改变用户体验。由于全球范围内几乎所有商业活动的数字化，对更好的软件应用程序的需求不断增加。此外，智能手机和智能手机应用的大规模普及也要求为用户开发新的软件。然而，只有在早期和全世界范围内提供这种培训，才能满足对编码知识日益增长的需求。编码应该成为必修课，应该成为所有学校课程的一部分。当局有必要了解编码的潜在重要性，以及它如何对就业市场产生良好影响。对程序员日益增长的需求表明，任何具有编码知识的人都将有很好的机会找到好工作。
即使编码已经成为许多国家课程中不可或缺的一部分，但为了丰富全球人才储备，更多的新兴经济体应该主动培养更多的程序员和软件开发人员。这只有在学生从很小的时候就有机会学习编码，从而探索新的职业可能性的情况下才有可能。此外，编码知识还将允许他们开发自己的应用程序，从而也可以作为自由程序员工作。目前，编码只在碎片化的群体和社区中相当流行。许多学生通过从 PenMyPaper 的[廉价写作服务购买更多的时间来腾出额外的时间编写代码。这让他们可以追求自己的爱好和激情。
在不久的将来，希望更多的发展中国家将编码引入学校课程，以使学生能够成为程序员和软件开发人员。这不仅有助于发展国民经济，同时对学生也有好处。因此，可以强调的是，从长远来看，在学生的早期阶段引入编码是非常必要的。](https://penmypaper.com/)
