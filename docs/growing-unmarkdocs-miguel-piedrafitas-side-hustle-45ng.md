# 成长中的非马克多克斯，米格尔·彼德拉菲塔的副业

> 原文：<https://dev.to/m1guelpf/growing-unmarkdocs-miguel-piedrafitas-side-hustle-45ng>

#### 问，问，问

今天，欢迎米格尔·彼德拉菲塔。

**大家好，欢迎来到零成长。请为我们的观众介绍一下你自己。**

嗨！我是 Miguel Piedrafita，16 岁，热爱编程。你可能(但可能不会)从 [OrgManager](https://orgmanager.miguelpiedrafita.com) 、 [DependenCI](https://dependenci.miguelpiedrafita.com) 、 [Webslides](https://slides.zone) 、 [CoderYouth](https://coderyouth.club) 、 [UnMarkDocs](https://unmarkdocs.co) 或我的任何其他项目中认识我。

什么是 UnMarkDocs，你是如何想出这个主意的？

UnMarkDocs 是一个 SaaS 解决方案，它允许你从 GitHub 托管的文档中创建漂亮的页面。它使用了 [UnMarkDocs 风格的降价](http://spec.unmarkdocs.co)，这是一种降价风格，提供了嵌入或提醒等扩展。我的灵感来自于像[Laravel Documentation](https://laravel.com/docs)或[Spatie’s Documentation](https://docs.spatie.be)这样的网站，它们从 GitHub 仓库中呈现它们的内容。

最初，我并没有开发一个 SaaS 应用，我只是通过设计一个文档网站来测试我的设计技能。然后，我发现了一个有趣的库，它允许您将 Markdown 解析为 HTML，并提供了一种扩展它的方法。在我得到这个工作之后，我决定我想在我自己的项目中使用它，所以我让它与 GitHub 同步，然后我决定如果其他人也能使用它会很好。

运行 UnMarkDocs，你最喜欢的方面是什么？

我认为运行 UnMarkDocs 最喜欢的事情就是使用它。我在那里存放我的学校笔记、家庭作业、个人笔记和项目文档。还有，最近我在 GitHub 上发现了一个有趣的项目，他们有一个 UnMarkDocs 页面！感觉棒极了。

让你坚持下去的动力是什么？

学习新事物。构建 UnMarkDocs 一直是(现在仍然是)一种令人敬畏的学习体验。我已经了解了为什么流行的文档托管服务会这样工作，了解了如何与真实用户一起在应用程序中工作，何时添加新东西，如何营销它，等等。

还有，这是我搭建的第一个真正吸引人的项目。到目前为止，我的大多数项目只被一小部分用户使用，我直接和他们分享链接，但是对于 UnMarkDocs，情况就不同了。事实上，我们仍处于私人测试阶段，拥有的用户比我的任何其他项目都多！

你能分享一下你如何获得第一个非朋友/家庭用户的故事吗？

我在 Twitter 上给一些人发了短信，问他们是否想尝试我正在做的一件新东西。我得到的第一个回复来自[丹尼尔·琼斯](https://twitter.com/dannydjones)，他不仅通过建议新功能和与我讨论用例来帮助我，还分享了一个巨大的文档，其中有很多对 UX、UI、用户流等的反馈。谢谢你，丹尼尔！

迄今为止，哪种渠道或增长技巧对你最有效？

发微博。说真的，**请人们与他们的朋友**分享你的新东西，你会对结果感到惊讶！

**你能分享一个完全失败的成长实验吗，为什么？**

我原以为发布到 Betalist 会让很多用户出现在早期访问列表上，但是效果并不像预期的那样。我认为这是由于糟糕的时机(这家初创公司晚了一个月才被报道)和糟糕的落地文案。在更改了登录页面中的一些文本后，我们开始收到电子邮件，但为时已晚。

你打算如何扩大你的用户群？

计划是在发布日在 ProductHunt 和类似的网站上发布，并且给流行的 OSS 回购的维护者发邮件。如果我可以让一些项目注册，他们的用户可能会跟随。

你认为自己在 3 个月(短期目标)和 5 年(长期目标，不切实际的希望)后会怎样？

在三个月内，我希望最终推出 UnMarkDocs，并可能创建一个具有一些附加功能的高级层。从长远来看，我想继续经营 UnMarkDocs，也许可以从中获得收入。

对我自己来说，事情有点不同。正如我所说，我想继续致力于 UnMarkDocs，并不时推出新功能，但当然，我也计划[继续为开源](https://github.com/m1guelpf)做贡献，在其他事情上做[的工作，我也必须专注于学校。](https://miguelpiedrafita.com/projects)
