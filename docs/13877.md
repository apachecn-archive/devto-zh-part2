# 在更新的 Devart Excel 外接程序 1.7 中大大改进了编辑模式和数据刷新

> 原文:[https://dev . to/devart software/大幅改进-编辑模式-数据刷新-更新-devart-excel-add-ins-17-3241](https://dev.to/devartsoftware/greatly-improved-edit-mode-and-data-refresh-in-updated-devart-excel-add-ins-17-3241)

Devart team 是一家公认的为开发人员和 DBA 提供专业数据库管理软件的供应商，它已经宣布发布 Devart Excel Add-ins 1.7。该更新允许更加方便和灵活地编辑和刷新云和数据库数据，包括:

1)高级编辑模式，消除了许多以前的限制。现在用户可以自由地:

```
• Copy and paste data in the Edit Mode, including pasting multiple rows, and copying data in the same workbook and worksheet with the imported table
• Use many Excel editing features, like formulas, scripts, Autofill, etc. 
• Sort and filter of the table with imported data, hiding and unhiding columns, allows editing data outside of the imported table on the same worksheet, etc.
• ...and many more. 
```

2)刷新改进，保留用户在表外输入的数据、同一工作表上的导入数据、该表中的数据排序设置和表样式。此外，新的刷新选项允许用户:

```
• See the refresh operation progress and cancel it if necessary
• Disable confirmation dialog boxes when starting refresh 
```

```

• Excel Add-in for Magento offers a more convenient connection editor for connecting to Magento 2 servers
• Excel Add-in for Salesforce now allows connecting to Salesforce Sandbox 
```
