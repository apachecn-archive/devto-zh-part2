# 如何成为一名自由职业的网络开发者

> 原文:[https://dev . to/flipped coding/how-to-be-a-freelancing-web-developer-o74](https://dev.to/flippedcoding/how-to-become-a-freelancing-web-developer-o74)

每个人都想兼职赚点小钱。这就是自由职业者如此受网络开发者欢迎的原因。它给了你一个机会去做一些快速的小项目来赚几百美元，并且练习你的技能。你可能听说过零工经济是如何改变我们所有人的工作方式的。

幸运的是，对网络开发人员的需求如此之大，以至于没有足够的人来满足需求。没有多少企业不想要某种形式的网站，这就是你的切入点。

要成为一名成功的自由职业网站开发者，你可以做以下几件事:

*   **登录自由职业网站**

    *   这是最显而易见的事情，尤其是如果你是网络开发或自由职业的新手。从这些网站开始:Fiverr、UpWork 或 PeoplePerHour。这些是最受自由职业者欢迎的网站，所以我要坦率地告诉你。
    *   你在这些网站上赢得项目的几率很低。有成千上万的其他开发人员有更好的评论，更有向客户推销的经验，以及丰富的投资组合。老实说，这就是你应该报名的原因。
    *   查看更有经验的开发人员的工作是尽可能快地达到速度的最好方法。这也会给你一个机会去学习如何写出好的广告词，期待什么样的项目，以及实际上你能期望得到多少。
*   **谷歌你所在地区的小企业**

    *   这里有一个超级秘密。当你作为一名自由职业者试图建立客户群时，没有什么比向当地企业推销更好或更容易的了。你可以查看他们当前的网站，并给他们打电话，建议如何改变他们的网站以吸引更多的顾客。如果你根本找不到网站，你可以说服他们让你给他们建一个网站。
    *   作为一个即将到来的自由职业者和网络开发人员，你能学到的最好的技能是销售。不管你是想做自由职业者还是为公司工作，你都需要知道如何向人们展示你是最适合他们的人。所以，把你在网上的练习用在现实生活中。
    *   这是我作为自由职业者最成功的地方。你会惊讶于你周围有多少人想要网站，并且不介意付钱给你来制作它。此外，当你在你所在的城市建立关系时，仅仅因为人们的交谈就能带来更多的顾客。
*   **随时准备好更新的投资组合**

    *   有没有客户并不重要。如果你想以自由职业者的身份赚钱，你需要一个自己维护的投资组合。这是你向人们展示你能做他们需要的事情的方式。拥有一个 GitHub 账户，里面有你的一些代码也是一个好主意，但这不是你应该强调的事情。
    *   如果你之前没有客户，你可以使用你做的任何项目。你也可以借此机会学习网页设计并开发自己的设计。作为一名自由职业者，能够很好地设计和开发一个网站会让你处于一个不可思议的位置。
    *   另一方面，如果你所有的作品都是专有代码(你不能给别人看的东西)，那么你的作品集至少需要有这些网站的截图。在这种情况下，你还应该至少有一个人可以作为证明人，证明某人对你的工作感到满意。

这些是我作为自由职业者取得成功的一些方法。我不得不承认这不适合我。一旦你建立了足够多的客户，这是相当一致的工作。老实说，我更喜欢让别人来和客户交流，而我只做代码。这是一次很棒的经历，很高兴知道如果你的工作出了问题，你还有另外一种赚钱的方式。
