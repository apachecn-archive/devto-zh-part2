# 我宁愿你没有...

> 原文：<https://dev.to/mdabek/id-rather-you-did-not--o1>

这里有一个小亮点:[c++中的安全编程实践](https://vimeo.com/283693861)

主要的要点是列出“我宁愿你没有”:

1.  使用 C
2.  用新的分配
3.  多做数学
4.  相信你的外部输入
5.  大量使用指针(因为它们没有语义！)
6.  编写“聪明”的代码
7.  经常使用 shared_ptr
8.  经常使用共享状态

尽情享受吧！

好奇的学生们请注意: [C++核心指南](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md)
