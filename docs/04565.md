## 安装

首先，通过 git 克隆回购:

```
$ git clone https://github.com/mariorodeghiero/vue-theme-iterm2.git 
```

> 安装项目主题

1.  发射 iTerm 2。在[iterm2.com](http://www.iterm2.com)获得最新版本
2.  点击 **iTerm2** 菜单标题
3.  选择**首选项...**选项
4.  选择**配置文件**
5.  导航至**颜色**选项卡
6.  点击**颜色预设**
7.  点击**导入**
8.  选择 **vue-theme.itermcolors** 文件
9.  返回**颜色预设**并选择 Vue 主题
10.  重启你的终端。

> 安装天啊主题

1.  将`vue-theme.zsh-theme`文件放到`~/.oh-my-zsh/themes/`目录中。
2.  打开你的 ZSH 偏好设置

```
 $ open ~/.zshrc 
```

1.  将主题变量改为`ZSH_THEME=vue-theme`。
2.  重启你的终端。
