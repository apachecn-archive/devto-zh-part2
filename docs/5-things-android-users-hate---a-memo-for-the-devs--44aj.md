# Android 用户讨厌的 5 件事——给开发者的备忘录

> 原文：<https://dev.to/dddddannnnn/5-things-android-users-hate---a-memo-for-the-devs--44aj>

Android 用户对他们的设备的忠诚程度不亚于 iOS 用户对他们设备的忠诚程度。当然，这两种操作系统都有改进的空间，不管其采用程度有多狂热。

以下是 Android 用户希望在未来几代设备中看到改进的 5 件事。开发人员在开发新应用时可能会发现这很有用。

### 1.电池寿命

每个手机用户都希望电池能够永远使用。Android 设备的电池寿命往往比苹果手机短，这是 Android 忠实用户的常见抱怨。

作为回应，谷歌最近在 Android 8.0 Oreo 更新中引入了一个新功能，警告用户正在消耗电池的特定应用程序。遵循提示中的建议有助于延长电池寿命。

所以记住这一点:如果你的应用程序很耗电，人们会很快卸载它。

### 2.消失的 WiFi 连接

这个问题源于 Android 手机的内置编程，以节省电池寿命。不断搜索 WiFi 信号会耗尽手机电池，因此当设备处于“睡眠”模式时，WiFi 连接会关闭。一旦手机被唤醒，信号需要一段时间才能重新连接。

此外，如果您的手机处于“省电”模式，您将需要关闭，以恢复您的连接。

### 3.自动更正

本来是为了方便，让我们听起来像更好的作家和语法正确的演讲者，却变成了我们生存的祸根。你每天在社交媒体上看到多少“当自动更正变坏”的迷因？写文本或电子邮件时，让自动更正功能不断改变你真正想说的话，会让人非常沮丧。

在你的 Android 设备上解决自动纠正问题的最好和最简单的方法之一就是简单地安装一个不同的键盘。

### 4.内存不足

从发短信、发电子邮件、保存语音邮件、连接所有社交媒体应用，到拍摄大量照片和视频，你都在使用大量的存储空间。当然，在你的设备上不断弹出“存储空间不足”的信息是很烦人的。

在你的 Android 手机上释放更多存储空间的一个简单方法是[利用 Google Photos](https://www.pcmag.com/feature/345998/25-tricks-to-master-google-photos) 或 Dropbox 作为你所有照片和视频文件的备份/主文件夹。删除旧邮件、语音邮件和短信也会有所帮助。

### 5.无法回滚

我们有多经常会后悔新版本的应用程序做了一些不寻常的事情？如果有一个简单的选项来回滚整个操作系统或一些应用程序就好了！选择性更新安装也是一个好主意。安全更新是一回事，但是向用户习惯的 UI 添加额外的步骤是另一回事

Android 用户仍然专注于他们的设备，大多数人永远不会考虑穿越到“黑暗面”并成为 iOS 用户。然而， [Android 开发者](https://www.itechart.com/development/mobile/android/)应该继续注意这些最常见的抱怨，并在每个新版本中继续改进。
