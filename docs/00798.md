# 作为一名自由网络开发人员，我能成功吗？

> 原文:[https://dev . to/uy Binh/am-I-able-pull-it-off-as-a-freelance-web-developer-47n](https://dev.to/uybinh/am-i-able-to-pull-it-off-as-a-freelance-web-developer-47n)

大家好，我不知道在这里分享我的故事是否合适，但我真的需要和某人谈谈。

我是一名来自越南的自学程序员。众所周知，成为一名自学成才的程序员并不容易，尤其是在我生活的地方。我最初是一名银行家，在那里做了 5 年全职工作后，我在 5 个月前辞去了工作，去追求我的梦想，成为一名自由网络开发员。

[![Web Developer](../Images/931cd074e8affb43a2bdf0e7e4df2e64.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--l_WsU-it--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/6czgu1r1bh6ldahpjrgg.jpg)

说实话，一开始我想的不多，只想用电脑编码和工作。那时，我跟随 Odin 项目课程和 freeCodeCamp，以及 Udemy 课程和一些编程书籍。现在，我已经完成了 Odin 项目，并在 FreeCodeCamp 获得了 4 个认证。现在，我可以编写我的[个人网站](https://uybinh.com)来分享我的[作品集](https://uybinh.com/porfolio)，用 Ruby 编写一些我自己需要的工具，用 Node.js 编写 JavaScript

但是最近我通常会有一种没有动力的感觉，很失落。我感到孤独，因为我没有任何从事技术工作的朋友，也没有当地开发人员的网络，还因为我是一个内向的人。我怀疑我的能力，担心我是否能完成它，作为一名开发者，我是否能过上我想要的生活。不知道下一步该怎么办？
