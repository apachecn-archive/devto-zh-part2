# 为此开始使用 ngrx/effects

> 原文：<https://dev.to/angular/start-using-ngrxeffects-for-this-1o24>

您可能只使用 ngrx/effects 通过用 ngrx 动作触发效果来处理与外部源的通信。但是你知道 ngrx/effects 的用途不止这些吗？

在本帖中，我们将介绍 ngrx/effects 的基础知识。
我们还将看看一些有用但不常见的用例:

*   外部源，如间隔事件和调整大小事件；
*   处理对话流；
*   向用户显示通知；
*   如何在你的影响范围内进入商店；
*   路由器重定向；
*   发送数据进行分析/监控；
