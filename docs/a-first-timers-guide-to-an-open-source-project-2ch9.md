# 一个开源项目的新手指南

> 原文：<https://dev.to/auth0/a-first-timers-guide-to-an-open-source-project-2ch9>

在 Auth0，我们都是关于开源的。作为一家企业，我们积极为开源做出贡献，并从中受益，在多个项目中拥有超过 25，000 颗星。我们的社区支持这一倡议，为可能参与项目的开发人员提供直接的建议和支持，无论他们是否开源。

总的来说，开源鼓励新手和代码新手做出贡献，因为这是开源的本质:为了每个人的利益，一起学习和协作。第一次可能会有点害怕和不知所措，但有一些项目和活动是专门为帮助你参与进来并在做出第一次贡献时获得信心而设计的。
