# 介绍轻松——跨平台终端主题

> 原文:[https://dev.to/mkuehnel/introducing-relaxed-a-跨平台-终端-主题-3p5c](https://dev.to/mkuehnel/introducing-relaxed--a-cross-platform-terminal-theme-3p5c)

我在优化我的开发环境和工作空间上投入了大量的时间——字体和配色方案是其中很大的一部分。对我来说，不同的配色方案(以及字体)都为我的开发环境设置了基调，并为我解析文本的方式提供了实用性。

我对 iTerm 主题进行了相当长时间的微调，最近发布了跨平台主题，可用于 Hyper、iTerm、macOS 终端、Konsole、PuTTY、Terminator、白蚁、Xfce 终端和 Xresources 🕶️

检查[https://github.com/mischah/Relaxed](https://github.com/mischah/Relaxed)得到它。
