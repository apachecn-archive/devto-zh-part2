## 调整、重构和/或修复你的代码

今天是关于你和你的设计/代码完美主义的。你想改变你的组件 API 使之更加灵活吗？你是否打算在第五天调整间距，但没有时间？也许你需要更多的时间来研究如何在第六天获得满分之前解决更多的问题。我们都必须睡觉，花时间与朋友和家人在一起，并有其他事情可能会影响你的工作流程；今天是你强势收官的一天，让一切都符合你认为合适的规格。
