我特别感兴趣的是:

*   Shazron Abdullah:带节点的无服务器应用。Openwhisk 中的 JS 和 Adobe I/O Runtime——我发现功能即服务(FaaS)特别有趣，任何不把我和某个特定厂商捆绑在一起的解决方案都是如此
*   Arun Michael Dsouza: Houdini -前方的路在何方 -能够对 CSS 进行更改并将其插入到流程中的想法很重要，就像我们现在可以对 Shadow DOM 和自定义元素进行的那样，以扩展 HTML 的功能
*   任何与可访问性相关的话题——这是我目前的一大弱点，也是我需要解决的问题
