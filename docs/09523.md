# Entity Framework Core 2.1.1 的所有额外功能现在都存在于 Entity Developer 和 dotConnect 提供程序中

> 原文:[https://dev . to/devart software/all-perks-of-entity-framework-core-211-now-in-entity-developer-and-dot connect-providers-2 AMB](https://dev.to/devartsoftware/all-perks-of-entity-framework-core-211-now-in-entity-developer-and-dotconnect-providers-2amb)

数据库工具和数据连接解决方案的捷克软件供应商 Devart 推出了对 ADO.NET 数据提供者和实体开发者(ORM 模型的可视化设计者)的更新。Devart 产品的新版本得到了改造，现在支持实体框架核心 2.1.1。此外，更新后的 PostgreSQL dot connect 通过 SSH 协议提供了更高的连接安全性。

以下是更新后的 Devart 产品最显著的改进:

1)支持最新的 Entity Framework Core 2.1.1
借助增强的 Devart 产品，用户现在可以受益于 Entity Framework Core 2.1.1 中的所有新功能:延迟加载、查询类型、值转换、数据播种、参数化构造函数等等。
2)更安全的 dotConnect for PostgreSQL 支持 SSH 协议
新版本的 dotConnect for PostgreSQL 为用户提供了通过安全 SSH 协议创建连接的能力。
