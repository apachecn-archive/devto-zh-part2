## [](#usage)用法。

*   在项目根目录下克隆或下载该存储库。
*   初始化包. json
    *   -> `yarn init`
*   安装所有软件包。
    *   -> `yarn install`
*   创建静态网站时。
    *   -> `gulp`或`yarn gulp`
*   创建 Web 应用程序时，使用 JavaScript 框架或库。
    *   发展-> `yarn dev`
    *   构建-> `yarn build`

ー

*   检查 npm 包更新版本。(ncu)
    *   -> `yarn check-pkg`
*   更新 package.json. ( ncu)上的 npm 包
    *   -> `yarn update-pkg`
*   删除 yarn.lock & node_modules。
    *   -> `rm yarn.lock` & `rm -rf node_modules`
*   回复:安装所有软件包。
    *   -> `yarn install`
