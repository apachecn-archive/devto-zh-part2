# 为什么 HTML 对 web 开发很重要

> 原文:[https://dev . to/Yoon sl/why-html-is-important-for-web-developing-k80](https://dev.to/yoonsl/why-html-is-important-for-web-developing-k80)

我学会的第一种网页显示语言是 HTML。许多人会认为 HTML 是一种非常容易学习和理解的语言，我也这么认为，但是 HTML 就像我们在幼儿园学的加减法。如果不知道如何使用数学的基本运算，我们将无法解决任何其他类型的高等数学问题。就像数学的基础一样，HTML 是 web 开发的基础，并且总是需要的。

为什么总是需要 HTML？如果我用 JavaScript 和一个空的 HTML 页面创建一个网站。我将能够在不写任何东西到 HTML(除了，

and ),but even if we are using JavaScript, we would still need to know the HTML syntax to create HTML displays.
