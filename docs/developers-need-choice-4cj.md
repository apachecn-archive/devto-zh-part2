# 开发者需要选择

> 原文：<https://dev.to/manifold/developers-need-choice-4cj>

闭上眼睛，为我想象这个场景。你被一家新餐馆聘为副主厨。你走进厨房，他们说“我们刚刚与 DepotMart 签署了一份采购订单，这是你的 DepotMart 刀具，接下来你的所有原料都将来自 DepotMart，还有一些 DepotMart 品牌的鞋子、衬衫和裤子。”你说“但是我习惯了我的刀，我很喜欢我的裤子，我不知道 DepotMart 甚至卖杂货。”他们回复“太糟糕了，采购部门签了合同，这就是你们正在处理的事情..接下来的五年。”

现在，用微软、亚马逊、谷歌或任何其他主要的单云取代 DepotMart。那些因销售操作系统、零售和搜索服务而欠股东人情的公司，现在有责任为你提供构建软件的工具和基础设施。

不要误解我，大盒子云有一些非常好的工具和稳定、可扩展的基础设施，这些基础设施具有相互交织的组件，可以很好地协同工作。(我们中的许多人非常高兴完全处于杰夫·贝索斯温暖的怀抱中，满足我们所有的需求)但从现在起 2 年、5 年或 10 年后，这将把我们带到哪里呢？在没有竞争激励的情况下，我们真的相信他们会主动创新吗？*咳*铁路和电话公司*咳*

另一方面，像 [Mailgun](https://www.manifold.co/services/mailgun) 这样的公司的生死取决于它们是否为交易电子邮件提供了令人惊叹的体验。 [LogDNA](https://www.manifold.co/services/logdna) 只通过在云中提供恒星日志来维持业务。有数百家这样的开发者服务公司，他们把一件事做得非常好，把开发者体验放在首位。

这就是我们开始[歧管](https://www.manifold.co/)的原因。

我们为开发者提供便利，让他们使用一张信用卡和一个账户购买 [Postgres](https://www.manifold.co/services/jawsdb-postgres) 、 [memcache](https://www.manifold.co/services/memcachier-cache) 、 [monitoring](https://www.manifold.co/services/scoutapp) 等。您可以集成到 [Terraform、Kubernetes](https://blog.manifold.co/manifold-%EF%B8%8F-kubernetes-terraform-9dcf09703e5a) 、 [Laravel](https://blog.manifold.co/announcing-manifolds-laravel-integration-12b9b0389579) 和[中，您可以邀请团队成员](https://www.manifold.co/teams)并基于每个项目进行管理。但这些是最好的，它们是维生素。我们真正追求的是给开发者选择的自由，在开发者为开发者创造的工具和服务中选择..一个蓬勃发展的公司生态系统，致力于让您的生活更轻松，让您的代码更有效，不断创新，推动软件的发展。

如果你提供一个令人惊叹的开发者服务，[让我们来谈谈](//mailto:peter@manifold.co)！我们一直在寻找与我们有着共同目标的公司，让开发人员的生活变得更轻松。无论你是后起之秀还是已经在游戏中呆了一段时间，我们今年都将推出大量的服务，希望你能加入我们。
