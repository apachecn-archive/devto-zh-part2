# Vulkan 内存分配器 2.0.0

> 原文：<https://dev.to/reg__/vulkan-memory-allocator-200-4n2j>

在上周的游戏开发者大会(GDC)上，我发布了 [**Vulkan 内存分配器**](https://github.com/GPUOpen-LibrariesAndSDKs/VulkanMemoryAllocator/) 库的最终版本 2.0.0。现在它已经有了很好的文档，多亏了开源社区的贡献，它可以在 Windows、Linux、Android 和 MacOS 上编译和工作。与此同时，我发布了 [VMA 转储 Vis](https://github.com/GPUOpen-LibrariesAndSDKs/VulkanMemoryAllocator/tree/master/tools/VmaDumpVis)——一个在图片上可视化 Vulkan 内存的 Python 脚本。从现在开始，我将继续在“开发”分支上进行增量开发，偶尔合并到“主”分支。如果您有任何反馈、建议或发现错误，请随时联系我。
