# 没有教育系统的学习

> 原文:[https://dev.to/bn_geek/learning-without-educational-system-107](https://dev.to/bn_geek/learning-without-educational-system--107)

当我学习的时候，我仍然经常注意到我们的老师保持着旧的教学方法，除了一些优秀的老师..之后的理论课程是一些问题的解释，最后是所有你已经学过的东西的实践..

在互联网技术这样的领域，也许这种方法会一直奏效，也许不会..但就我个人而言，我相信没有过程或预先计划的学习，你呢？

来，我来解释一下我简单的观点..！

*   学习学术规范

我不是学习领域的专家，但我也是一名开发人员，非常了解谷歌和谷歌搜索，所以在谷歌中使用关键词简单搜索，如“学习”“学术”“规范”“系统”，我发现了非常清楚的结果，因为教育系统是一个允许学生学习任何东西的过程，从基础到专业，并使用特定的方法促进学习。

学术进步:由老师监控，但由学生个人推动。学生从一个水平到下一个水平的进步是基于学术准备，而不是实际年龄或“座位时间”高年级学生可能在三年、四年或五年内完成成绩单工作。Counterpane 拒绝让在所有大学预备课程中期末成绩不低于“75”的学生升级或毕业。

*   没有学术过程(计划)

从我的经历开始，我学到了很多东西，在不同的技术中，没有任何预处理或预先计划，我也学到了很多没有理论的技术..也许看起来很奇怪，但这是事实，在我过去的 2 到 3 年里，我作为 IT 开发人员和自由职业者为许多人和初创公司工作，我还作为高级网站开发人员在一家网站代理公司的“SEO”团队工作..有些事情比其他人更让我难以忘怀，比如当你需要的时候，“你学习，发现，被迫去了解，尝试很多，了解新技术”!！

让我们告诉你我从谁那里学到了很多新技术、技巧和技能，而没有计划或过程。我将使用这个正方形列表来做这件事，因为我的英语不太好:“伙计们，:D:

*   与团队合作总是让你“不经过过程”地学习新东西，因为如果一个朋友或同事向你解释一个技巧，他会更简短和有用

*   在你从事实时项目的过程中遇到技术问题会让你“在没有过程的情况下”学习新的技能或技巧

*   当项目经理要求你使用特定的技术来完成这个项目时，你不会说“不”,所以即使你从来不知道它，你也会去查阅文档，你甚至很难使用它，所以你正在学习“没有过程”

*   当你在做一个兼职项目时，你试图使用新事物，所以你只需要一个文档和一些博客帖子或指南，而不需要保持有序，所以你打算“没有过程”地做它

*   当你打算搜索一个教程来了解一项新技术时，你会在网上找到很多，你选择一个..你读了吗..你从中学习..记住，不是老师或其他人用学术过程来教你这些..他只是一个与你分享他已经知道的东西的人，所以你甚至可以“没有过程地”学习！！。

*   请记住，你可以教我们你所知道的，这样我们就可以不经过过程而学习:D

我给你们分享了我的观点..让我们知道你的:)
