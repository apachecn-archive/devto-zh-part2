# 快速终端提示

> 原文:[https://dev.to/voter101/quick-terminal-tip-1can](https://dev.to/voter101/quick-terminal-tip-1can)

最初发布于:[https://mociun.xyz/quick-terminal-tip/](https://mociun.xyz/quick-terminal-tip/)

有一件小事让我作为软件开发人员的生活变得更好。似乎没有多少人知道这件事。

诀窍是热键终端窗口。它基本上是一个始终可见的附加终端窗口，您可以使用键盘快捷键来显示/隐藏它。

[![Additional window!](../Images/1d0cc0ebe78eb784e17192b5fe698b67.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--TEDay1yn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://cdn-images-1.medium.com/max/1600/1%2A6Pq051EVu6ORgiZ323XYeg.gif)

这个额外的窗口作为代码便签簿或专用的 man 窗口似乎非常方便。它非常方便，我有时把它当作主要的终端。

我相信这个功能应该在大多数平台上都有。我个人使用 iTerm2(仅限 Mac，抱歉)，它内置了这个功能。在我的例子中，新窗口在我按下 CTRL 键两次后显示/隐藏起来。
