# 在 Ubuntu 中设置代理

> 原文:[https://dev.to/ech0server/setting-a-proxy-in-ubuntu-efe](https://dev.to/ech0server/setting-a-proxy-in-ubuntu-efe)

要在 Ubuntu 中配置代理，您可以在不同的地方设置它:

## 贴切

为了将 apt 与代理一起使用，您必须用以下代码编辑`/etc/apt/apt.conf`文件:

```
Acquire::http::proxy "http://username:password@host:port/";
Acquire::ftp::proxy "ftp://username:password@host:port/";
Acquire::https::proxy "https://username:password@host:port/"; 
```

## 环境变量

要将代理作为环境变量添加，请使用以下代码编辑`/etc/environment`文件:

```
http_proxy=http://username:password@host:port/
ftp_proxy=ftp://username:password@host:port/
https_proxy=https://username:password@host:port/ 
```
