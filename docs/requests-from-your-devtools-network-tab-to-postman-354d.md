# 从您的 DevTools 网络选项卡到 Postman 的请求

> 原文：<https://dev.to/osuka42/requests-from-your-devtools-network-tab-to-postman-354d>

从 Chrome DevTools 的网络选项卡中，选择 XHR 调用，并选择点击->复制->复制为 cURL。
[![Network Tab](../Images/eb1ddc265f8e1c9f51c80c5d83097161.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--jOL_prcv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/u2xzn0hrvudir2ribqxb.png)

现在在博文中转到导入->粘贴原始文本->粘贴
[![Postman](../Images/d763b2ac23d31df273625a6579095b33.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--YuyAO-bP--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/v79f3cca7iomit0fa66u.png)

这将复制确切的请求，包括用户代理、头、cookies、方法负载等..并根据您的需要进行修改。希望这能让你调试/开发得更开心！
