# 在 Vim 中将 snake_case 转换为 CamelCase

> 原文:[https://dev . to/acro 5 piano/convert-snake case-to-camel case-in-vim-47lf](https://dev.to/acro5piano/convert-snakecase-to-camelcase-in-vim-47lf)

这是一个小片段。

在您的 shell 中创建这个函数:

```
# .bashrc / .zshrc

camelcase() {
    perl -pe 's#(_|^)(.)#\u$2#g'
} 
```

然后，您可以将标准输入转换为 CamelCase:

```
~> echo array_map | camelcase
ArrayMap 
```

这些功能本身就很有用。此外，您可以在 Vim 中使用它们。

选择要转换的 snake_case 行，然后键入`!camelcase`并按回车键。

# [](#appendix)附录

您可以使用此函数将 CamelCase 转换为 snake _ case:

```
function snakecase
    perl -pe 's#([A-Z])#_\L$1#g' | perl -pe 's#^_##'
end 
```
