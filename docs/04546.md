使用这个简单的 bash 别名可以很容易地返回到您在终端中访问的前一个目录。

🔥实际上，这是一天中最好的开发技巧，我一直在使用它。

⌨️键入`-`返回上一个目录

🤔怎么会？
✅将此别名添加到您的。bashrc/。zshrc 文件
♻️重装你的终端

```
alias -- -="cd -" 
```
