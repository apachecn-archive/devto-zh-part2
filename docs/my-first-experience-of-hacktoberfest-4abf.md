### 学习 Github

我以前用过 Github，并且已经知道它的基本命令，因为我在 Bitbucket 上维护过一些项目。但是正如你在第一张图片中看到的，我以前没有把它作为开源平台。只有在 10 月份，我一直在不断地贡献，并将在安拉的带领下更进一步。在这一个月里，我学到了很多值得分享的东西。

#### 问题，请购单，合并，关闭和恢复请购单

了解了参与开源项目所需的最重要的组成部分，即问题和 PRs，以及标签如何支持问题跟踪。

#### github pages-github 页面

了解 Github 页面。在 Github 页面上制作了我自己的[在线简介](https://ows-ali.github.io)，还将 Jekyll 主题添加到了[库](https://ows-ali.github.io/Hacktoberfest)。

#### Github 机器人

在 Github 上了解奇妙的机器人。我还使用了机器人(当然还有其他人的贡献来写消息),这些机器人在贡献者创建问题、在我的回购上制作他们的第一次公关或在我的回购上合并他们的第一次公关时欢迎并感谢他们。

#### PR 模板、问题模板、投稿、CodeOfConduct 等文件

在参加 Hacktoberfest 之前，我并不清楚这些文件背后的需求和概念，但现在我意识到这些存储库应该拥有这些文件，并且我将尝试在我未来的存储库中也制作这些文件。

### **曝光**

在参加 Hacktoberfest 期间，我也开始更多地使用 Twitter，我在那里发现了一些很好的趋势。

#### 趋势

1.  #100DaysOfCode(有趣的是，我也开始了这个挑战，但因为我不知道我是否能够完成它，所以我没有提到它，你也不知道)
2.  # 301 天代码
3.  #momscancode(这是一个战胜妈妈们面临的冒名顶替综合症的伟大倡议和创业。如果你是一个忙碌的家庭主妇，想学习编码，那么这是和有趣的妈妈们一起学习的最好机会)
4.  #100DaysOfDataScience
5.  #数据科学 3 个月

#### 学习平台

我也遇到了很多学习平台，对初学者和专家来说都很棒。我在那里学习，在那里构建项目，解决问题，也在他们的知识库中贡献了一些东西。其中一些平台是

1.  FreeCodeCamp(最活跃的，初学者友好的，用户友好的，完全免费的平台，在这里任何人都可以学习全栈 Web 开发，甚至从字面上的编程经验 0。我也在取得 Api 和微服务证书的路上)
2.  TheOdinProject(非常适合学习 ROR)
3.  Exercism(对于学习语言和让导师审查你的代码非常有用)

### **结论**

我开始向 Github 投稿，我的十月几乎都是绿色的，这只是开始。从现在开始，我将在开源上做很多事情，学习和成长自己，并回报社会。如果你正在寻找类似的充满乐趣的活动，我会推荐你检查以下内容

*   [24 个拉取请求](https://24pullrequests.com/)(类似于 12 月的 Hacktoberfest)
*   [GSOC](https://summerofcode.withgoogle.com/) (如果你是一名本科生，那么这是一个为大公司做开源工作的好机会，同时也能赚些钱)
*   谷歌代码(如果你是大学预科生，那么这是扩展你的编码技能的好方法)
*   [TheOdinProject](https://www.theodinproject.com/) (如果你想学习 ROR 平台上的 web 开发，从 0 到 job ready，这个很棒)
*   MomsCan:Code (如前所述，如果你是一位母亲，无论你在哪里，你都可以和其他母亲一起学习)
*   参加在线编码平台，如 [TopCoder](https://topcoder.com) 、 [CodeChef](https://www.codechef.com) 、 [HackerRank](https://www.hackerrank.com) 等，欢迎各种技能水平和各种编程语言，还可以获得现金奖励、t 恤和其他纪念品。

如果你喜欢这篇文章，学到了一些东西，或者发现自己受到了激励，那么你可以用掌声来欣赏。
