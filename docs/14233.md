# 你的开发者“圣经”是什么？

> 原文:[https://dev.to/dittusch/whats-your-developer-bible-3 PBC](https://dev.to/dittusch/whats-your-developer-bible--3pbc)

当我还是一个大三学生的时候，在 Java“火”的时候，我买了一本书，这本书对我作为一个开发人员的影响很大，直到现在——它是《实用程序员——从熟练工到大师》。即使在我认为自己是“资深”开发人员的今天，它仍然给了我很好的建议，我向所有新手和初级开发人员推荐这本书。

作为一名开发人员，你有没有这样一本影响你的书？出示你的推荐！
