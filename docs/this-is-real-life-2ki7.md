# 这是真实的生活

> 原文：<https://dev.to/faradayyg/this-is-real-life-2ki7>

你可能已经掌握了 React，我打赌你也已经掌握了 CSS grid 和 Flexbox，而且你对 VueJS 和 Angular 的了解已经足够了。

我打赌你为自己感到骄傲。

这是现实生活，兄弟。

在现实生活中，人们不更新他们的浏览器，其他人仍然使用 Opera Mini，你猜对了，CSS grid 不会工作。

这是现实生活，你将被迫使用 Jquery 而不是 React。

这是现实生活，在现实生活中，你不能只是转储你的数据库表，并重新开始。你可以，但我不能保证你走后会发生什么。

这是现实生活，即使你的内部 API 是 RESTful 的，我打赌你将不得不使用一些外部 SOAP APIs，运气不好。

我打赌你不喜欢看千行函数，我也是。我敢打赌，你也渴望重构。但是你真的能吗？我的意思是，这是真实的生活，许多其他事情都依赖于这一个功能，有些东西可能会打破你知道。但是，嘿，如果你愿意的话，试一试。

我们都喜欢开发新产品，而不是调试一些你不认识的人写的旧代码，他们从 API 端点返回普通的 HTML。但这是真实的世界，你必须保持这棵摇钱树。

最后，这与你懂什么语言无关，也与你能用这个闪亮的新框架做多少“忍术”无关。最后，最基本的才是最重要的。
