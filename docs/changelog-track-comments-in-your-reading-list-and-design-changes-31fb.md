# 变更日志:跟踪阅读列表中的评论(和设计变更)

> 原文：<https://dev.to/ben/changelog-track-comments-in-your-reading-list-and-design-changes-31fb>

今晚推送最大的功能变化是你现在可以在你的阅读列表部分看到评论了。这意味着添加到你的阅读列表(通过大的保存按钮或通过书签反应)可以让人们更好地跟踪他们关心的对话。

侧边栏上的# Discuss 部分也进行了修改，因此最近发布的帖子的评论会将其推到列表的顶部，有点像传统的论坛。我们认为这将刺激更好地进行对话。

以及一些新图标和其他设计修改。

希望你喜欢这些变化。我们将尽快尝试修复我们创建的任何新错误。快乐编码。❤️
