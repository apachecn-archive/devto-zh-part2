# 然而，凯莉·沃恩编码了

> 原文：<https://dev.to/kelly/nevertheless-kelly-vaughn-coded--m7o>

## 我开始/继续编码是因为...

我是一名开发人员、企业主、经理、企业家，也是计算机科学世界里的全能女强人。(允许我这么称呼自己吗？我刚刚做了，所以我想现在回头已经太晚了。)

大约 11 岁的时候，我发现我可以在 Neopets 上创建自己的公会，从那时起，我就把建立网站作为一种爱好。(我仍然相信这是互联网最美好的几年。)虽然我现在靠写代码赚钱，但我仍然把它当成一种爱好。我喜欢为独特的设计或复杂的功能编码所带来的挑战。越复杂，我越开心。

虽然随着时间的推移，我已经涉足了各种语言，但我最喜欢的地方是在前端，为 Shopify 进行定制主题开发。这是一个特别有趣的领域，因为我的代码直接帮助其他人启动和发展他们自己的业务。在过去的三年和变化中，我一直在做 Shopify 商店，我非常喜欢它。

## 我想吹嘘一下...

到目前为止，我最大的成就是创办了一家女性拥有并经营的公司，名为“酒吧代理”。最初的“只有我”自由职业者现在是一个机构，利用 7 位才华横溢的女性的技能来帮助企业家创办自己的企业。学习如何经营企业和管理他人一直是一个挑战，但我绝对欢迎。

我从来没有想过要有一个只由女性组成的团队，但我很高兴事情是这样的。在一个拥挤的代理机构的世界里，这个事实(除了我们所做的伟大工作)真的帮助我们脱颖而出。额外的收获是，我不仅有机会与 Taproom Agency 的女性员工一起工作，还能与许多其他女性(和男性)一起努力实现自己的创业梦想。

## 我对支持女性和非二进制代码的盟友的建议是....

如果我能给另一位女性开发者和企业家一些建议的话，那就是坚持到底。熬过艰难的时刻。他们会来，他们会过去，然后更多的人会来。人们会告诉你你做不了什么，你不够有天赋，你没有足够的经验。证明他们是错的。

最重要的是，支持其他和你有相同目标的女性。少关注竞争，多关注提升彼此。我们的最终目标应该是让更多的女性对计算机科学职业感兴趣。从事计算机科学工作的女性越多，我们就越富裕。
