# 然而，布雷亚·安妮编码

## [](#i-begancontinue-to-code-because)我开始/继续编码是因为...

我有点爱上它了...并坚持了下来。我在大学主修商科，不知道毕业后要做什么。我让一个朋友帮我进入了当地的一家科技公司。我开始做数据录入，因为我认为这是一家福利很好的大公司。我有了新的抵押贷款，更重要的是，我必须照顾一个刚出生的儿子。随着我开始在公司获得新的机会，我开始问很多关于自动化过程的问题。有一天，我的团队被要求对我们正在使用的产品中的一个新引擎进行质量检查。我甚至没有在 QA 上工作，但我开始问事情是如何工作的。我一直在烦开发团队的经理，以至于有一天她问我是否愿意加入开发团队。更不可思议的是，她问我是否想远程工作(他们在南加州，我在北内华达州)。在这家公司里，像那样的机会不常出现。嘿！我要去一个新的地方旅行。

当我加入这个新的开发团队时，它是我的第一个，真正的编码和开发团队。这是一种公司专有的语言，非常像 pascal。我不会说这是真正的工程，但肯定是我的第一份开发工作。随着时间的推移，有传言称一种基于 Java 的新服务将取而代之。我们团队中的所有人都一脸呆若木鸡的表情，因为没有人懂 Java。我决定回学校学习。那时我想我真的决定了我想走工程这条路。

我已经当了大约十年的工程师了。我喜欢工程学的原因是因为每天都是不同的，不管你知道多少...还有更多要学的。我非常幸运地拥有导师和经理，他们给了我学习、成长和领导的绝佳机会。现在，就像我喜欢学习新事物一样，我也喜欢通过指导即将到来的初级工程师来回馈社会。只要我能学习，只要我能回报，我就做工程师。
