## [](#i-begancontinue-to-code-because)我开始/继续编码是因为...

我开始写代码是因为我发现建立网站是一种令人惊奇、令人着迷的经历，我想成为这个过程的一部分，即使是小规模的。我仍然记得我第一次打开 Squarespace 主题，摆弄 HTML & CSS 的时候。要是我当时知道我在做什么就好了。

我继续写代码，因为随着网络的成熟，我们需要让我们的工具&网站也随之成熟。出于各种原因，网站(以及整个科技领域)为用户提供了非常狭隘的视野和体验。有无数这样的例子，人们不能简单地注册一个帐户，因为一个服务不承认他们的名字是一个“合法的名字”，或者出于其他原因，不允许他们使用别名。

这一点，加上许多网站在过去没有考虑可访问性的事实，意味着我们隔离了每天浏览网页的很大一部分用户。既然互联网几乎是每个人的“必备”工具，那么我们为所有用户创造令人惊叹的、可访问的和包容性的前端体验就更加重要了。

了解网站设计/开发中的辅助功能。还有 CSS Grid，因为我对 CSS 布局的所有东西都很兴奋。

## [](#my-advice-for-allies-to-support-women-and-nonbinary-folks-who-code-is)我对支持女性和非二进制代码的盟友的建议是....

积极主动地提供指导机会。很多时候，我们都面临着成为“精英中的精英”的压力——尤其是如果你是公司里第一个担任技术职位的女性/非二元性别人士。在这种情况下，提供一个你可以从中获得灵感和另一种支持的导师会是一个很大的解脱。
