## 我开始/继续编码是因为...

我开始写代码是因为我想成为团队的资产，我想在开始职业生涯时增加我的价值。我一直在学习如何编码，因为我可以看到技术有可能有效地改变我们教育后代的方式，以及我们如何改善每个人的生活质量。

## 我最近克服了...

在大学里，我被编程的想法吓倒了。我不觉得自己有能力学习计算机或软件。毕业两年后，我终于有足够的信心学习 Python 作为我的第一门编程语言。我克服了每当我试图写一个简单的函数时会失败或出错的恐惧。我已经成为一名战士，我不打算让我的负面经历阻止我犯错，我计划从中吸取教训。

## 我想吹嘘一下...

最近，我开始研究一个很酷的概念，通过一个有趣的游戏来提高青少年的金融素养。这是一个每天都在增长的大项目，我想吹嘘一下我是独自一人的先锋(在一个盟友的大力支持下！)从创造到充满希望的完成。

## 我对支持女性和非二进制代码的盟友的建议是....

对于任何盟友来说，在不妨碍工作的情况下，对人们参与的项目表现出真正的兴趣是很重要的。不要主动提供建议，只是倾听！如果有人向你寻求帮助，把这看作是对某人相信你能提供他们需要的答案的赞美，而不是认为这是他们软弱或能力不足的表现，你必须弥补。
