# Vulkan 和 DX12 中的内存管理:幻灯片在线

> 原文：<https://dev.to/reg__/memory-management-in-vulkan-and-dx12-slides-are-online-21fb>

我在游戏开发者大会(GDC) 2018 上演讲的幻灯片:“Vulkan 和 DX12 中的内存管理”现已在线提供，作为来自 **[高级图形技术教程](http://www.gdcvault.com/play/1025458/Advanced-Graphics-Techniques-Tutorial-New)** 的材料的一部分。任何人都可以访问这个 PDF，而不是在 GDC Vault 付费墙后面。我在最后的“备份”幻灯片中添加了一些我在演示时没有展示的附加信息。幻灯片的设计方式让你即使没有看到演讲也能从中学习。
