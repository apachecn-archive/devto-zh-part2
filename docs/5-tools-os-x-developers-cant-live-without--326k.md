# 5 个 OS X 开发者离不开的工具

> 原文：<https://dev.to/_robynedgar/5-tools-os-x-developers-cant-live-without--326k>

*供稿[埃里克·沃克](https://www.linkedin.com/in/eric-walker-18119569/)*

### 1.景象

在 MacOS 的所有应用中，[奇观](https://www.spectacleapp.com/)让生活变得简单多了。眼镜帮助你管理你打开的窗口，让你只使用热键来调整它们的大小。这听起来可能不多，但是当你开发一个网站，并且在多个显示器上有无数个窗口打开时，这是很方便的。使用热键，您可以快速调整大小和移动窗口，以重新组织您的屏幕。

[![alt text](../Images/0e6b2a849154be8ff04bd2cb25bd008a.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ZSvUswmg--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AawJLW2NUDSkwbFZzCbftEA.jpeg)

这个应用程序提供了各种各样的命令，但最重要的命令允许你在顶部|底部或左侧|右侧分割半个屏幕，当然也可以全屏显示，而不会丢失菜单栏。

### 2.小 Ipsum

当你建立一个网站的时候，在设计阶段没有最终的副本来填充网站是很常见的。为了解决这个问题，大多数开发人员使用称为 Lorem Ipsum 的占位符文本。Lorem Ipsum 虚拟文本是一个重要的工具，用来可视化网站的最终版本。

[![alt text](../Images/a6fd2ed34f73c9433576b2ce315e07ba.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--fLieym4B--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AVz9B7U2yOGHBBvDu9w5xww.png)

虽然有很多网站有不同类型的 Lorem Ipsum 文本，但在浏览器中使用它而不打开新标签是很方便的。 [LittleIpsum](http://dustinsenos.com/littleIpsum) 让您选择 Lorem Ipsum 的最佳数量，并自动将其复制到您的剪贴板以供快速使用。
T3】

### 3.iTerm2

作为开发人员，除了 macOS 终端所能提供的，我们还需要更多的命令行界面。当你寻找更多时，iTerm2 是一个很好的选择。我们最喜欢的一些功能包括:自动完成，窗口分割和搜索功能。您甚至可以使用 imgcat 功能预览图像。

[![alt text](../Images/bfc68d9c8697995d71b644a4e9fc85f3.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--SfRIG_mS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/0%2A5MYl-7XOiaDnG0O9.)

### 4.防毒面具

当你在浏览器中输入网站名称时，域名系统(DNS)会查找网站的 IP 地址，然后将你发送到正确的位置。但是，如果您想键入一个网站的名称，并转到 DNS 会让您导航到的地方以外的地方，您必须在计算机上的 Hosts 文件中覆盖 DNS。 [Gasmask](http://clockwise.ee/) 让访问和编辑您的主机文件变得简单明了。通常要编辑主机文件，您必须使用基于终端的文件编辑器和 sudo 命令来编辑文件。防毒面具让你跳过头痛，直接跳到编辑。

[![alt text](../Images/da18a03758b96855c30f8b726e4fa7a3.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--TlVA60QS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/0%2AYy9toEFKDPDeOc8Z.)
T4】

### 5.颤抖

[颤](http://happenapps.com/#quiver)是程序员用的笔记本工具。该笔记本将改变您记录工作的方式，并允许您快速回忆您的开发过程。它处理内联代码，点击和拖动文件存储，网址，以及任何你能想到的东西。如果你花时间适当地记录你的项目，这将使回忆变得即时，并快速简单地教别人。与竞争对手相比，标签、搜索、LaTeX 支持和 ACE 代码编辑器单元等功能确实让颤颤巍巍。

[![alt text](../Images/f27312e6fdcb14623f88d35dd4d299a1.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--5Pnw3imJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AfuU1-aR8qnZvwRdXc4pgwg.png)

使用这五个工具，管理您的开发项目将是无缝的。
