# 解决问题:直觉

* * *

我称之为直觉。

有时候我感觉有些不对劲。

它没有可见的指示器。开发人员说进展顺利，产品人员说它按时完成并在控制之下，专家说一切都协调一致并达成一致。

但是我感觉到了什么。

然后我问一个问题。我总是不好意思问，因为每个人都说没问题，他们都已经同意了，为什么我还要再检查一遍呢？

但是当他们听到我的问题时，发现他们没有想到一个问题。或者他们心里有不同的“开启时间”。或者他们对协议的理解不同。或者他们做了非常不同的假设。

每周都会发生。我称之为直觉。
