# 人力资源软件公司使用 Nylas 日历 API 来加快上市时间

自动化就在我们身边，从无人驾驶汽车到自动支付，从 Alexa 和 Echo 这样的虚拟助手到机场的登机亭。自动化从根本上改变了企业的运营方式:它通过处理乏味重复的任务并将更多的人工任务留给人类，提高了效率、透明度和工作与生活的平衡。这就是为什么人力资源软件的自动化如此普遍，如此重要。

像 [Beamery](https://beamery.com/) 、 [Lever](https://www.lever.co/) 和 [CATS](https://www.catsone.com/) (一个申请人跟踪系统)这样的公司都为他们的用户提供了简化的招聘和人力资源流程——这是有充分理由的。根据 [WorkMarket 的 2020 In(Sight)](https://content.workmarket.com/2020-insight-report-what-ai-automation-mean-for-work/?utm_source=forbes&utm_medium=blog&utm_campaign=2017_automation_report&utm_content=home-page) 报告，53%的员工表示自动化可以帮助他们每天节省多达 2 个工作小时，相当于每年节省 6 周(240 小时)的时间，而 78%的企业领导者可以每天节省多达 3 个工作小时，每年总共节省多达 9 周(360 小时)的时间。

您可以使用 [Nylas 日历 API](https://www.nylas.com/calendar-sync) 在您的人力资源软件应用程序和您的用户日历之间建立日程安排同步。通过自动为最终用户安排任务，为他们节省数百个工作时间。

### **让用户能够通过日历同步简化招聘流程**

招聘是一个复杂的、多方面的过程，有多个变量，几乎没有出错的余地:从创建职位公告到简历筛选、进行面试、审查候选人、收集反馈，可能需要几周甚至几个月才能确定下来。自动化有助于使招聘成为一个无缝流程。

使用 Nylas calendar API，开发人员可以构建完整的日历 CRUD(创建、读取、更新和删除功能)，以及双向日历同步，以加快和改善招聘人员的招聘流程。招聘过程越快，招聘到最佳候选人的机会就越大。

用户可以在您的人力资源软件界面中安排面试和电话。只需一次点击，就可以安排多个采访，显示忙/闲时间表，并将活动添加到所有与会者的日历中。

### **让用户能够通过日历同步来组织会议**

会议是让团队达成共识、讨论策略和规划未来的好方法。如果会议管理不当，每月会占用 [31 个小时](https://www.atlassian.com/time-wasting-at-work-infographic?utm_source=zapier.com&utm_medium=referral&utm_campaign=zapier&utm_source=zapier.com&utm_medium=referral&utm_campaign=zapier)的工作时间。

尽管组织一次会议可能很简单，但随着与会者人数的增加，难度也会增加——发送电子邮件、获取每个人的日程安排、打电话、找房间、安排资源。

花在组织会议上的时间越多，花在更高优先级任务上的时间就越少。

将日历同步添加到会议室预订软件中，消除了单独预订一切的需要。会议组织者节省了时间，让他们有更多的时间来准备会议。

[Fountain](https://get.fountain.com/) 是一个申请人跟踪和招聘平台，帮助世界各地的公司使用自助面试预约、自动化、机器学习和可定制的工作流程大规模雇用小时工。

当 Fountain 用户预订会议时，他们会选择一段被其他工作会议或个人计划包围的时间。在 Nylas Calendar API 的帮助下，Fountain 的日程安排系统与用户的整个日程安排协调一致。

简而言之，它将在一个地方保存每次会议的记录。每当你创建一个新的存储数据的地方，你给自己更多的维护和更多的机会让细节从缝隙中溜走。

在整个招聘过程中，应聘者会经历不同的阶段。Nylas 日历 API 将跟踪您的申请人的旅程，还将跟踪组织或候选人的任何日程变化。
