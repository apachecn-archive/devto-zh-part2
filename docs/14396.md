# 一种反直觉的增加科技多样性的方式

你想增加科技的多样性。你是一个有特权的人，可能是白人，男性，甚至可能很富有。你已经知道拥有一个多元化和包容性团队的好处:多种多样的眼睛有助于获得更多见解，减少惊喜。听到代表不足的同龄人的悲伤故事，你会很痛苦。你想帮忙但不知道怎么帮。

这听起来像你吗？太好了，你是一个了不起的人，我很高兴你在这里！

但是，在你跑去贡献你的时间和金钱，让更多的人——不仅仅是像你一样的人——在技术领域担任初级职位之前，请听我说完。

我正处于网页开发和设计行业的职业生涯中期。现在，我从我的处于职业生涯中期的朋友那里听到了很多同样的故事，他们来自代表性不足的群体(女性、少数民族、LGBTQ 等等):你在职业生涯中到达了某个点，你不能赢。

我还从许多善意的、享有特权的同事那里听说，他们希望看到情况有所改善，管理层中有更多女性，有色人种有更多加薪，各方面都有更多动力。但这些人中的大多数是个人贡献者，而不是经理或董事——没有资格指导人们担任高级职位或赞助他们晋升。简而言之，尽管他们的心是好的，但他们没有能力做任何实质性的事情来改变现状。这就出现了一个问题。

不管我们从代表性不足的群体中培训了多少人来获得初级职位，如果他们没有向上的流动性或者由于偏见而不能被录用，这就意味着他们是无用的。

以下是来自代表性不足群体的许多人在踏上辉煌职业道路时面临的一些障碍:

*   他们没有被提升和聘用到高级职位来推进他们的职业发展。
*   他们在目前的岗位上没有得到充分的利用。
*   在管理链的上游，没有人能像他们一样他们可以学习并渴望成为。
*   他们 *[缺乏导师](https://www.forbes.com/sites/margiewarrell/2017/06/24/women-mentoring/#29992df922db)* 来获得超越上述所有人所必需的技能。
*   他们缺乏赞助来确保他们的想法不偏不倚地被倾听和重视。
*   他们*面临着来自一个无私的管理层的下行压力*，这个管理层“看不到颜色”或者“只雇佣/提拔最好的人。”

对于一个具有良好性格和健全头脑的人来说，无论他们在自己的条件下有多优秀，要想在这些挑战面前取得长期的成功并实现他们的全部潜力是非常困难的，甚至是不可能的。

## 好的，那么你能做些什么来帮助提高科技的多样性呢？

很高兴你问了！很简单:进入管理层。

我知道，这似乎违反直觉。一个已经拥有特权的人进入管理层会如何改变 T2？难道我们不想让*少一点*吗？嗯，这完全取决于你在那个位置上做什么。

你可以将多元化作为公司的目标，指出其他经理作为同事的偏见，创建一个吸引人的团队，让人们可以尽最大努力工作，并从上述经历中开始成功的职业生涯。当员工抱怨时，你可以支持他们。你可以利用你的地位。

管理并不意味着更高的薪水或成为“老板”

做经理更像做牧羊人。一个好的经理是一个力量倍增器，能让他们的团队发挥出最佳水平。他们帮助做出重要决定，并指导和建议他们的员工。但归根结底，这不是他们的事。是关于他们的团队。一个好的经理的成败取决于他们团队的表现。

把自己放在一个可以雇佣人的位置上。将你的职业生涯与他们的成功联系起来，然后投入精力帮助他们成功。

## 但是如果……

有时我会遇到一些很棒的人，他们希望创造一个多元化的工作场所，但他们所处的结构阻碍了多元化，而不是有助于多元化。也许管理层不愿意接受既定的黑暗模式的挑战。也许有一种“老男孩俱乐部”的态度，打破这种态度无异于自杀。也许你为人们挺身而出，却被击落。也许你，你自己，在这里感觉不到安全或被养育。

我问这些善良却沮丧的人们:这是你们“最好的生活”吗？你是如何在这里实现你的全部潜力的？有多少人没有实现他们的梦想，因为你没有？

如果你所在的地方没有让你帮助你想要成功的人，去找一个你能帮助他们的地方，在那里你能产生有意义的影响并为之创造更好的东西。

让公司明白*和*是你做出这个决定的原因。你处在一个被倾听、被怀念、被迷失的位置。这是一个比其他人在梯子底部等待更强的位置。

使用它。

## 不是关于你:是关于你能为别人做什么。

在美国，我们喜欢抱怨很多。我们在上次总统选举中抱怨过。我们现在还在抱怨。抱怨所有的政治家、经理和其他权威人士都有偏见不会改变他们的政策。现状喜欢抱怨者，因为他们从来不*做任何事情。他们是抱怨者，不是实干家。抱怨者什么也改变不了。*

你是盟友吗？你是实干家吗？变革者？然后你游行抗议，给你的代表写信，竞选公职。

你进入领导阶层，然后分水。

有特权的人可以享受闲坐的奢侈，享受个人贡献的低风险生活，选择不往上爬；进来，做你的工作，回家，训练某人，希望他们凭自己的能力成功。但是对于受训者来说，这个系统并不像它为你所做的那样工作。他们需要担任高级职位的导师、管理层的支持者、职业道路上的支持者，而不仅仅是初级职位。

如果你真的想改变这个系统，现在是时候开始考虑更大的事情了。是时候站起来走出你的舒适区了。

不要抱怨和感到无力。开始从比你自己更大的角度考虑你的职业。

## 这是你的行动号召。

为了让这篇文章少一点抱怨，多一点做事，这里有一些资源和建议可以帮助你:

*   学习如何维护你的团队，你会经常这么做。
*   [领导者不擅长判断他们有多包容](https://hbr.org/2017/10/leaders-arent-great-at-judging-how-inclusive-they-are)，所以你想*让你周围的人在你没有做出他们需要的改变时召唤你*。
*   此外，你需要营造一种环境，让人们觉得他们可以和你进行这样的对话。这通常意味着不知疲倦地改进你的沟通方式，[积极倾听](https://en.wikipedia.org/wiki/Active_listening)——而不是试图得到所有的答案(你不知道，这没关系)。)我发现 *[至关重要的对话](http://amzn.to/2F4TlkN)* 在风险很高的时候很有帮助，作者提供了[培训](https://www.vitalsmarts.com/crucial-conversations-training/)。
*   拥有多样化的员工并不等同于拥有包容性的工作场所。不要停留在数字上:努力[创造一个让每个人都有归属感的地方](https://www.entrepreneur.com/article/296635)这样他们就能做到最好！
*   不要只做导师，[做发起人](http://larahogan.me/blog/what-sponsorship-looks-like/)。(事实上，[劳拉·霍根](http://larahogan.me/blog/)的任何作品都值得一读！)
*   一直都是。是。学习。关注《哈佛商业评论》,阅读并分享黄金帖子，如[11 位首席执行官从倡导多元化中学到了什么](https://hbr.org/2017/08/what-11-ceos-have-learned-about-championing-diversity)。管理和其他任何角色一样要求严格，优秀的管理者是[无限的学习者](https://mastersofscale.com/barry-diller-infinite-learner/)。

我的一位同事热衷于包容性领导，他最近开始了自己的关于这个主题的[时事通讯](https://tinyletter.com/theinclusiveleader)，我对他的项目寄予厚望。

## 不要犯这一个大错！

记得我说过这不关你的事吗？你不应该努力成为一个穿着闪亮盔甲、值得赞美的白衣骑士。这不是那种路。(事实上，如果人们过分称赞你，你也要担心——很难对每个人都喜欢的人吹毛求疵。你可能会错过重要的对话。)

当有人的声音被忽视时，你应该成为传声筒；当人们无法得到代表时，你应该成为倡导者；当人们受到攻击时，你应该成为护盾持有者；当人们被困时，你应该成为电梯；当人们不被注意时，你应该成为信号助推器。只要有可能，就让他们去做，让他们被倾听和理解，让他们代表自己。你是后援，是开路者，是负面影响的承受者，总是问:“我能帮什么忙？”总是抬高别人，甚至抬高自己。

努力成为仆人，而不是超级英雄。

这并不迷人。没有奖品。回报是你将帮助人们发挥他们的全部潜力，并在死时感到你已经帮助人类踢了更多的屁股。

最后，如果你在想，“但是，我没有任何特权——我不是白人，”想想总有一些人没有特权:白人女性和非裔美国男性比有色人种女性享有更多特权。利用你所拥有的一点点特权，考虑如何利用它来提升你周围的好人。

总有一些事情是你可以做的，从雇用一名学员并给他们写推荐信，到为你的二手办公设备提供按比例折扣。

如果你有一盎司特权，在你的一生中，明智地使用它。
