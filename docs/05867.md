初学者的错误:JavaScript 的变量、函数和对象。。。我的天啊。

编码 bootcamp 的 JavaScript 后四课，我发现自己的脑袋有点‘懵懵懂懂’。。。与 HTML 的元素、CSS 的选择器和属性，是的，甚至 CSS grid 相去甚远。相反，变量、函数、对象、属性、方法、事件、条件和循环在我的大脑中游动。所以，我做了任何正常人都会做的事，寻找痛苦。。。我是说。。。公司，用 JavaScript 在谷歌上搜索“初学者常见错误”。。。我发现自己。叹气。好的。常见的错误是普遍的，除了随机编码专家。

在被引用最多的常见错误中，语法问题似乎排名最高。这包括:缺少或不匹配的括号、圆括号、花括号、逗号和分号；命名变量时无法区分大小写；以及用 JavaScript 保留字命名变量。。。仅举几个例子。

JavaScript 新手的另一个最爱是混淆赋值和比较。我可以亲自证明这一点。赋值“=”和比较“==”(宽容相等)和“===”(严格相等)之间是有区别的，区别是有功能的代码和无功能的代码。

如果你能理解我所描述的错误，那就不用担心了！我在下面提供了一些例子和图表来帮助理解变量、函数和对象是如何在一个基本的层次上工作的，并且突出了一些语法。希望有帮助！保持冷静，继续编码。
