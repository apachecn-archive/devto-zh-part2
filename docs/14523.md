# 学习、工作和家庭

> 原文:[https://dev . to/Adam friedl/learning-and-working-and-family-ing-30c 3](https://dev.to/adamfriedl/learning-and-working-and-family-ing--30c3)

如今有成千上万的资源可以帮助你学习如何编码。你有愿望和勇气吗？你可以成为一名软件开发人员。就其本身而言，这无疑是正确的，但它通常默默地假设你拥有无限的其他同等重要的资源——即时间。如果对你来说是真的，马泽尔！好好利用。但是，对我们中的许多人来说，这是不正确的——尤其是那些抚养小孩和全职工作的人。

所有这些都是想说，我真的很感激听到 CodeNewbie 播客最近的一集,它花了一分钟向那些人大声呼喊。才华横溢的[莎拉·德拉斯纳](https://twitter.com/sarah_edo)和永远令人敬畏的主持人[萨隆·伊特巴雷克](https://twitter.com/saronyitbarek)花了一分钟说“恭喜你们！我们不知道你是怎么做到的，”很难表达那种认同的感觉。

当我试图向人们传达我对编程的热情时，我解释说，当我女儿还是婴儿时，我参加了一个在线训练营，当时我正在做一份全职律师工作。我试图利用每一秒钟的空闲时间来学习或练习。我仍然经常在凌晨 4:30 起床，熬夜到深夜来挤出时间。我不是一个人。我们中有很多人，很多人都面临着比我自己更具挑战性的时间考验。他们牺牲了睡眠和家庭时间，因为他们热爱这项工作，并且想做得更好。

我经常感到沮丧的是，在线开发社区中没有更多关于这个话题的讨论。非常感谢 Saron 和 Sarah 的赞美之词！
