# 2017 年 Web 组件统计

> 原文:[https://dev . to/this isbinh/web-components-statistics-in 2017-2 ECF](https://dev.to/thisisbinh/web-components-statistics-in2017-2ecf)

本文是“测试 696 个 Web 组件”系列的最后一篇文章。本文中的统计数据涵盖了许多主题:webcomponents.org 上发布的组件数量、基于其库的组件分布、工作和非工作组件，以及一些更有趣的统计数据。

注:这些统计数据大约是一年前得出的。其中一些在今天可能变得无关紧要。

### 3 个月 300 个元素。这是一个令人印象深刻的数字。

[![](../Images/606c4d082f0e2befb7e2f8862d90550c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--7_Qcz4Bn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/rzvu8nmg4kovi609kaw1.png) 
让我们把这些数字分解一下:

*   聚合物始终占出版元素的 90%以上。很明显，Polymer 在 web 组件库领域占据主导地位。
*   每月大约有 100 个定制元素发布，数量惊人。
*   这里最显著的变化是聚合物 1 和 2 组分的分布。聚合物 2 有 278 个元素的激增，而聚合物 1 大致保持不变。
