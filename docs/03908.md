# 贴纸——你的文字如何变成贴纸

> 原文:[https://dev.to/gaserd/stickeroid-如何把你的话变成贴纸-5981](https://dev.to/gaserd/stickeroid---how-your-words-turn-into-stickers-5981)

我周围都是开发者，商人，创业公司，所以我从他们那里画项目做分析。

今天我选择了 Stickeroid。

下面是链接:[https://stickeroid.com/](https://stickeroid.com/)

[![Demo UI](../Images/b8cfe61014ea08288e91f190e7e453f5.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--iPnHFSsQ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/c9duw55bxg46c2odjo46.png)

这个项目是关于我引以为豪的人，维克多·科赫。

**项目是什么样的？**

该项目是一个人工智能系统，允许你为你的话选择贴纸。

**项目受众？**

项目的受众是开发者，尤其是信使。

**下载呢？**

没有这样的下载，但他们声称在他们的帮助下发送了 692120430 张贴纸。

**项目的优势是什么？**

对开发者来说简单易用，全部免费。所以如果你有一个需要贴纸的 app，为什么不用一个很酷的东西呢？

我想在设计中提到我最喜欢的维提风格，我不知道他是自己做的还是伴随着每个像素，但他们的设计非常酷:[https://stickeroid.com/intro](https://stickeroid.com/intro)

他们立即覆盖了最高信使和一切可能的东西。

这个项目的缺点？

对于我这样的项目来说，现在还不清楚，但是卡尔的钱在哪里？

但与此同时，我明白项目背后的人清楚地知道他们在做什么，钱在哪里。

**项目里的钱呢？**

据我所知，有些人已经投入了 20 万美元，但我有一种感觉，最重要的是资金。

**结果呢？**

这个项目有着明确的目的和强硬的团队，但对我来说，有着奇怪的兑现时间。
