戴夫·托太酷了，我忍不住想他。
所以今天我要向大家介绍新的功能建议:社交网络地图！
我认为编程是关于相互交流的，知道你的网格覆盖的区域总是很有趣，所以我有以下建议:

向带有全球地图的设置页面添加新部分，其中可以查看 3 种不同类型的连接:

*   谁在跟踪你
*   你在跟踪谁
*   和(可能)标签网络

假设可以很容易地将这些数字(关注者)转换成美丽的地图。

有不同的可能适合的地图类型，仅举几个例子:

*   要显示连接:人造三维弧:[http://bl.ocks.org/dwtkns/4973620](http://bl.ocks.org/dwtkns/4973620)
*   显示密度:[https://beta.observablehq.com/@mbostock/d3-choropleth](https://beta.observablehq.com/@mbostock/d3-choropleth)

线条的颜色可以取自用户的颜色偏好。
