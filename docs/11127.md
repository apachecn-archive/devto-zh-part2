# 开发人员和设计师沟通

> 原文:[https://dev.to/cat/dev-and-designer-communication-5dh5](https://dev.to/cat/dev-and-designer-communication-5dh5)

我最近目睹了一场工程师和设计师的大战，他们说，“这是在浪费我们的时间！”由工程师大量表达。

我对你的问题是:

*   你希望你的设计师如何解决一个问题？
*   你和设计师之间的沟通如何改善？
*   你和设计师有过什么好的和/或不好的经历/合作/对话？

请在评论中添加任何其他见解或建议！
