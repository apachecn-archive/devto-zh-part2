# 我应该用什么语言写我的编程博文？

> 原文：<https://dev.to/espoir/what-language-should-i-write-my-programming-blog-post-in-1g68>

我的问题很奇怪，但事实是:

我的背景

我的母语是法语，是一名软件工程师，我在一个说法语的国家学习，在那里说法语长大。

我的问题是我应该用哪种语言写我的技术博客文章？

***一方面*** :

法语对我来说可能更容易写，它可以让我通过将英语中的知识传递给说法语的人来发挥作用。反对意见:观众少得多，我一直认为计算机科学或技术的美只能用英语来欣赏。

***另一方面英文*** :

写博客对我来说有点困难(我在 6 个月内写了 2 篇博客)，我担心没有掌握这门语言会让我很多次止步不前。但是有了英语，我可以面向更多的读者，并提高我的写作技巧。
你怎么看？
