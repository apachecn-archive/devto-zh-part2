# 为什么软件不应该免费

> 首次发布于 [Dev Letters](https://devletters.com/letters/software-shouldnt-be-free/)

人们真的很难支付软件、在线课程、数字订阅和其他类似的在线东西。这真让我难过。

在我们的文明中，软件(或任何代码支持的产品)已经像任何实物商品一样司空见惯。那么为什么会有歧视呢？

我会告诉你为什么。

人们知道软件、歌曲、课程等可以很容易地大量分发。也就是说，如果你做了一个软件或一首歌，你几乎可以以同样的成本把它分发给数十人或数百万人。开销很小。

这在一定程度上是对的。也许这就是为什么人们不愿意为软件付费，因为他们认为一个歌曲/软件开发者有不公平的优势。他们本质上只是复制代码(不像物理拷贝那样是免费的),并从中赚取 100%的利润。

或者可能还有别的原因。我们已经变得如此习惯于消费来自“硅谷”支持的巨头的免费服务，以至于我们觉得为一个软件付费是愚蠢的。

想想吧。你使用的所有流行的在线工具，脸书、Gmail、OneNote、Youtube、谷歌搜索等等，一切都是免费的。这让我们所有人都养成了自由软件的习惯。

但所有这一切都是有代价的，这些代价对我们来说是隐藏的。首先，这些公司利用你的数据为你提供广告。他们给你做侧写，这样他们可以帮助广告商更好地销售你。

我知道这对我们大多数人来说都没问题。好吧，在 80 亿人口中我们是谁。

让谷歌给我建档，让他们知道我的每一个小细节，从我暗恋的对象到我的信用卡号码，这都没关系，因为他们可能不会对我做任何坏事。

这是正确的。但是通过允许他们这样做，你正在强迫自己养成使用自由软件的习惯。这对独立软件创作者来说是不好的，他们没有技术来“描述”你，只是想提供纯粹的价值作为回报。

现在，这就像是支持全球成熟的产业，抑制本地生产。或者以另一种方式，允许已经成功的公司继续成功，并阻止想要成为公司的人甚至开始起步(所有这些都是因为他们向你收取他们可以增加的 2-1000 倍的价值)。

现在我不能在这里做你的道德指导，你应该做你觉得对的事情。

但是下一次你决定不购买这个软件，仅仅因为它对两杯咖啡收费，请三思。

也许产品给你增加的价值与其收取的费用相比不值一提。
