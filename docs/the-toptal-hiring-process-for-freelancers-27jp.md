### 第 0 轮:正在应用

要想获得 Toptal 的自由职业者职位，你必须向他们提交你的个人资料。你可以去他们的网站，使用“申请自由职业者”选项。

如果你能使用我的推荐链接，我将不胜感激。

### 第一轮:筛选电话

这是筛选过程的第一步。在这一轮中，你将接受招聘人员面试。

这或多或少会像一次普通的面试，只是尽你所能回答问题。

不过有一点，如果你的母语不是英语，建议你先提高自己的口语技能。也就是说，一定要说清楚正确的英语。在日常生活中练习几天英语口语应该就可以了。

### 第四轮:项目

这可能是所有回合中最容易的。基本上，你需要在你的主要技术领域做一个项目。这里唯一的问题是，你做的最终产品应该是可以生产的。

这是什么意思呢？我的意思是它应该有...

*   正确的输入验证(例如，仅在电子邮件字段中输入电子邮件)
*   没有 UX 故障(例如，点击用户照片应该打开他们的个人资料)
*   优化的代码(例如，发送电子邮件不应该发生在主线程上)
*   整洁的设计(例如，文本不应该溢出，应该在任何地方都可读)

你有固定的时间来完成这项任务。好好利用这段时间，尽你最大的努力完成这个项目。

很容易错过我刚才提到的事情。多次测试你的项目，确保它是完美的。

这是最后一轮。熬过去了，就能通过 Toptal 做自由职业者了。

### 如果我一轮失败会怎么样？

作为申请人，你应该知道这一点。

一轮失败会暂时搁置你的申请过程。持有期限可以是几个月到几年，所以在申请之前要记住这一点。
