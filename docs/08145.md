像我一样的许多 web 开发人员利用 **Chrome 开发工具**作为 web 开发的通用工具。这对于实时修改 CSS 或者 HTML 或者分析网页来说非常方便。Chrome release 65 给了我们一个期待已久的漂亮工具，可以覆盖网页使用的资源，并用我们自己的本地资源替换它们。例如，您可以覆盖网页使用的 **CSS** 文件，并对 **CSS** 文件进行本地更改。您对 **CSS** 属性所做的任何更改都会实时反映在 web 浏览器中。

一个鲜为人知的事实是，这个本地覆盖特性可以用来创建一个 web 服务器来服务静态 web 页面...
