# 如何删除 Prestashop 主题目录菜单

> 原文：<https://dev.to/franken/how-to-remove-prestashop-theme-catalog-menu-4elk>

[![](img/40530d374ac82e65964ba6147c083995.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--_bneysBc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.prestasoo.cimg/stories/remove-prestashop-theme-catalog-menu.jpg)

如果你是一名开发人员/设计师/自由职业者，你真的不想在 PrestaShop 1.7 后台保留“主题目录”菜单项。它可以在网站后台的“设计”部分找到。在那里，你的客户可以找到官方插件市场主题的目录。它在那里**仅仅是为了广告目的**，它在那里完全没有用。

本 [Prestashop 教程](https://www.prestasoo.com/blog/how-to-remove-prestashop-theme-catalog-menu.html)帮助您轻松删除菜单，没有任何麻烦。
