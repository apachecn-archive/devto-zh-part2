# 铁笔

所以我为我的网络浏览器(Chrome)找到了这个宏伟的扩展，叫做 Stylus。
用手写笔你可以搜索社区编码的 css 主题并上传到他们的网站。所以在我发现 [GitHub 黑暗](https://userstyles.org/styles/37035/github-dark)之后，我的眼睛又一切正常了。

## 安装

首先，我们需要为您的网络浏览器安装手写笔:

*   [铬合金](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)

*   [火狐](https://addons.mozilla.org/firefox/addon/styl-us/)

*   [歌剧](https://addons.opera.com/extensions/details/stylus/)

它们的安装非常简单，就像浏览器的其他扩展/插件一样。

# 奖金

## 全局主题

我喜欢很多这些主题，但我不想为每个网站下载一个主题！Stylus 有一个解决方案，他们有“全球主题”,这是一个通用的主题，适用于所有网站。

我安装了[暴风](https://userstyles.org/users/43566)的[全球黑暗风格](https://userstyles.org/styles/31267/global-dark-style-changes-everything-to-dark)。

## 添加异常

可能有些网站你不想应用这个全局主题，或者你已经为那些看起来更好的网站准备了其他主题。您可以通过单击浏览器上的手写笔图标并单击“管理/管理”按钮(左侧按钮)来添加例外。选择您的全球主题，并前往页面底部添加例外网站。

[![Exceptions](../Images/978745ab7552a7062171a97e13c79e93.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--etxkMesH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/9pcwy5wxx6s8mxhuw4ml.PNG)

正如你所看到的，我为已经有另一个主题的网站停用了全局主题，因为有时多个主题之间会有冲突。
