# MarkdownHelper#include

> 原文:[https://dev.to/burdettelamar/markdownhelperinclude-16bk](https://dev.to/burdettelamar/markdownhelperinclude--16bk)

GitHub 用户多年来一直要求在 GitHub markdown 页面中包含文件。

我刚刚放了一个新的 Ruby gem， [markdown_helper](https://rubygems.org/gems/markdown_helper) ，它实现了包含文件。

文件可以包括为:

*   突出显示的代码块。
*   明码块。
*   逐字文本。

gem 有 API 和 CLI。(对于非常年轻的用户，CLI <=>命令行界面。)
