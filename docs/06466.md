# 用 python 显示日历

> 原文:[https://dev . to/Hasan sajedi/display-calendar-with-python-56pd](https://dev.to/hasansajedi/display-calendar-with-python-56pd)

> Python 有一个内置函数“日历”来处理与日期相关的任务。通过下面的例子，你将学会显示给定日期的日历。

```
 import calendar

yy = 2018
mm = 8

print(calendar.month(yy, mm)) 
```
