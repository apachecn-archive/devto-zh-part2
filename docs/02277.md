# Magento 平台迁移

通常，有五类重要的数据必须从你现在的商店转移到 Magento。

它们是:
·目录数据(产品描述、类别)
·一般和个人数据(客户账户、订单、订阅、博客文章)
·后勤部门
·网址
·促销和销售
迁移到 Magento 2 平台时需要考虑的事项
