# 最终的 CSS3 属性列表

> 原文：<https://dev.to/flippedcoding/the-ultimate-css3-property-list-1170>

CSS 属性比你可能想记住的要多得多。没关系，因为没有真正的 web 开发人员会记住所有这些东西。这里有一个参考，当你需要记住一个 CSS 属性或者你需要找到一个新的属性时，你可以随时使用。

| 属性名称 | 描述 |
| --- | --- |
| **对齐内容** | 指定当项目没有使用所有可用空间时，灵活容器内各行之间的对齐方式 |
| **对齐项目** | 指定灵活容器内项目的对齐方式 |
| **自我对齐** | 指定灵活容器内选定项的对齐方式 |
| **全部** | 重置所有属性(unicode-bidi 和 direction 除外) |
| **动画** | 所有*动画-** 属性的简写属性 |
| **动画-延迟** | 指定动画开始的延迟时间 |
| **动画方向** | 指定动画是向前、向后播放还是交替播放 |
| **动画时长** | 指定动画完成一个循环需要多长时间 |
| **动画填充模式** | 指定动画不播放时(开始前、结束后或两者)元素的样式 |
| **动画-迭代-计数** | 指定动画应该播放的次数 |
| **动画名称** | 指定@关键帧动画的名称 |
| **动画播放状态** | 指定动画是正在运行还是暂停 |
| **动画-计时-功能** | 指定动画的速度曲线 |
| **背面可见度** | 定义面向用户时元素的背面是否可见 |
| **背景** | 所有*背景-** 属性的简写属性 |
| **背景-附件** | 设置背景图像是与页面的其余部分一起滚动，还是固定不变 |
| **背景混合模式** | 指定每个背景层(颜色/图像)的混合模式 |
| **背景剪辑** | 定义背景(颜色或图像)在元素中应该延伸多远 |
| **背景色** | 指定元素的背景色 |
| **背景图像** | 为元素指定了一个或多个背景图像 |
| **背景-起源** | 指定背景图像的原点位置 |
| **背景位置** | 指定背景图像的位置 |
| **背景——重复** | 设置是否/如何重复背景图像 |
| **背景尺寸** | 指定背景图像的大小 |
| **边框** | 关于*边框宽度、边框样式*和*边框颜色*的简写属性 |
| **边框-底部** | *边框底部宽度、边框底部样式*和*边框底部颜色*的简写属性 |
| **边框-底部-颜色** | 设置下边框的颜色 |
| **边框-左下角-半径** | 定义左下角边框的半径 |
| **边框-底部-右侧-半径** | 定义右下角边框的半径 |
| **边框底部样式** | 设置下边框的样式 |
| **边框底部宽度** | 设置下边框的宽度 |
| **边框折叠** | 设置表格边框是应该折叠成单个边框还是分开 |
| **边框颜色** | 设置四个边框的颜色 |
| **边框图像** | 所有 *border-image-** 属性的简写属性 |
| **边框-图像-开头** | 指定边框图像区域超出边框的量 |
| **边框-图像-重复** | 指定边框图像是否应该重复、变圆或拉伸 |
