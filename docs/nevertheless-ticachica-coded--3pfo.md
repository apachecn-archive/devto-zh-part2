## 我开始/继续编码是因为...

我开始编码是因为我小时候总是弄坏我所有的玩具。我妈妈从 RadioShack 给了我一个小键盘和漫画书，我终于找到了一个我不会弄坏的玩具。

我继续编码，因为这是我的禅的地方。当我打开编辑器、控制台和技术设计时，我大脑的一部分会被激活。不管世界上发生了什么？当我失去某人时，当我沮丧时，或者当世界上发生了悲剧时，我知道我正在磨练我的技能，这将使我能够建立一些可以帮助别人的东西。它给了我生命。

## 我想吹嘘一下...

我想吹嘘自己是第一台 Xbox Kinect 设备的一部分。我要创造吉尼斯世界纪录。作为 Xbox Kinect 用户研究和用户体验团队的语音项目经理，我比大多数人更早与电视对话，并领导设计语音用户体验的团队。我和世界上一些最不可思议的人一起研究一项尖端技术。

我亲身体会到技术是如何开始带有种族主义和性别歧视的。换句话说，Kinect 既不擅长识别深色皮肤，也不擅长识别女声。通过一个追求卓越的多元化团队，我们能够解决工程问题，并为更多人提供更公平的技术。

## 我目前正在入侵...

去年 11 月我失业了。从那以后，我一直在更新我的编码技能。自由代码营和其他在线课程一起成为了我新的学习社区。通过学习 NodeJS 和如何构建微服务，我重新找到了我对后端的热爱。然而，将数据收集到一个整洁的 API 中是不够的，所以我目前正在通过 React 在线课程来构建信息丰富的前端。接下来，我将关注我可以贡献的开源社会影响项目。

## 我对其他女性和非二进制程序员的建议是...

*   即使你不觉得自己是专家，也要教导和指导。你不必成为专家来教别人一些东西
*   找到你的部落。它可以是一个 meetup 小组，一个开源项目，或者如果你周围没有这样的小组，或者你有特别的兴趣，可以创建一个。
*   **吸引孩子**孩子是有魔力的。他们比我更有创造力，用一些编码技巧帮助他们解决自己的问题非常有影响力
