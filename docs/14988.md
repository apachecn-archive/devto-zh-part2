# 66 天学习挑战

> 原文:[https://dev.to/sandordargo/66-day-learning-challenge-18bk](https://dev.to/sandordargo/66-day-learning-challenge--18bk)

*本文最初发表在[我的博客](http://sandordargo.com/blog/2018/02/21/66-day-learning-challange)上。*

活到老学到老，还是活着就是为了学习？！重点是一样的:每一天都要学习！如果你想领先一步，如果你想过上高质量的生活，你必须不断鞭策自己，你必须不断学习。加入我的 66 天学习挑战！

坚持学习，你的努力会有丰厚的回报。为什么学习如此伟大和重要？就拿[海明的](https://en.wikipedia.org/wiki/Richard_Hamming)思想来说:

“我总是每周花一天时间学习新东西。这意味着我比我的同事多花 20%的时间学习新东西。现在 20%的复利意味着四年半后我知道的将是他们的两倍。而且因为复利，这 20%的额外收益，每周一天，五年后我会知道三倍多...[或无论数字是什么...]"

这是如此强大，以至于 Erlang T1 的发明者乔·阿姆斯特朗在《工作中的 T2 编码员 T3》一书中引用了他的话。

在阿姆斯特朗做研究的时候，他花了大约 40%的时间在学习上，他已经这样做了几十年。当他被邀请去解决一个问题时，他真的会觉得自己知道得更多。我们都应该以他为榜样，投资于自己，每天学习，这样我们就能在我们的领域处于领先地位。

学习不一定意味着在学校接受正规教育。学习可以表现为许多不同的形式。你可能想参加在线课程。或者就拿一本书，读一读你不知道或者不够了解的话题。教书。写吧。和你的孩子一起玩，注意他们做什么，想想他们为什么这样做。重点是要注意。如果你密切关注你所做的任何事情，你就会学到东西！

每天至少吃一颗番茄来提高自己，但是我相信如果你把大部分时间都浪费在活动上，你会有时间吃更多的。

为了让自己保持动力，加入我的 66 天挑战，每天用标签[# 66 天学习](https://twitter.com/search?f=tweets&q=%2366DaysLearning&src=typd)和#D 来标记你当天学到的东西，所以第一天是#D1，#第二天是#D2，等等。

为什么是 66 天？不是因为我想沿着 66 号公路开车。那就太傻了。我花了 66 天，因为平均来说，一个新的行为变成自动的需要 2 个多月——确切地说是 66 天。

在中间和最后，随时分享你的想法，无论你想知道挑战如何改变了你的生活，它如何让你成为更好的自己。

快乐学习！
