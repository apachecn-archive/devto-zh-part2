> 原文:# t0]https://dev . to/yurfwa/gdiff-gblame-phead-5e 5g

在 Vim 上使用 Gdiff 或 Gblame，因为总是会忘记，所以备注
在 Gblame 上发布提交日志后，在~，p 上追溯过去和未来，之后想发布现行的 index 和 diff 的时候

```
:Gdiff : 
```
