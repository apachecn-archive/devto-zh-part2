# 多元视觉

> 原文：<https://dev.to/funkysi1701/pluralsight-epf>

今年早些时候，我注册了 pluralsight。如果你也想注册 pluralsight，也可以点击此链接[http://referral.pluralsight.com/mQd8BJ4](http://referral.pluralsight.com/mQd8BJ4)赚钱。

Pluralsight 是一个销售各种技术主题培训视频的网站。你可以按月或按年订阅，你可以随时随地观看超过 6000 门课程。

我不是一个视频的大粉丝，如果我坐在屏幕前，我宁愿建造一些东西，但是我确实花了很多时间听播客。所以我决定把这段时间转换成听 pluralsight 视频。

如果我们假设没有看到视频的视觉部分，你就失去了 75%的学习机会(这只是一个猜测，我不知道确切的数字是多少，也不知道你会如何计算)。然而，这仍然比我没有注册的时候多学了 25%。

自从我注册以来，我已经听了超过 27 小时的技术视频。这些涵盖了 C#、安全性、Xamarin、MVVM、接口、参与和干净架构等主题。我学到的第一件事是有很多东西我仍然不明白。

[![pluralsight](img/c695c1bc76aac67061d9ce74fa234ff2.png)](http://referral.pluralsight.com/mQd8BJ4) 也就是说，我愿意认为自从开始听复数观概念以来，它已经从未知的未知数(我不知道它们，也不理解它们)变成了已知的未知数(我知道它们，但还没有完全理解它们)

为了记录你的学习情况，pluralsight 有一些你可以完成的测试。我特别自豪的是我在 Azure 测试中的分数，它把我列为专家。
