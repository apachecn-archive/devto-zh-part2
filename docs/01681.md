# 谢谢，黑客啤酒节！

> 原文:[https://dev.to/cjbrooks12/thanks-hacktoberfest-13g2](https://dev.to/cjbrooks12/thanks-hacktoberfest-13g2)

今年是我第一次听说 Hacktoberfest，这是一次令人惊奇的经历。

作为[兰花](https://orchid.netlify.com/)的创作者和维护者，我完全惊讶和谦卑地在这个月从其他贡献者那里获得了它的第一份 PRs，它在 Github 上也达到了 100 颗星！所以我只想说谢谢，并向这些人吹嘘一下:

*   我一直在将 Orchid 的所有代码从 Java 迁移到 Kotlin，Andrey Mochalov 对这项工作帮助很大，转换了几十个文件。
*   Yaroslav Matveychuk 增加了一个新的发布机制，在网站发布的同时创建一个 Github 版本。这真的很好，因为兰花自己的发行说明现在正在更新每一个新版本。
*   Bastien Jansen 创建了一个 Maven 插件，这样兰花站点可以很容易地集成到 Maven 项目中，并随着项目不断地部署。以前只支持 Gradle。
