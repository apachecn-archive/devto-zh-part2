# 终于找到了一个社区

> 原文：<https://dev.to/jotasprout/finally-finding-a-community--1odi>

大约两年前，我开始自学编程。是的，你会在 **Reddit** 等网站上找到成功的故事，人们会说，“我六个月前开始学习编程，刚刚得到了我的第一份开发工作！”但那不是我的故事。

所以，也许我的故事也是你的。

我有一份全职工作，妻子和两个孩子，所以我没有太多时间学习，也没有能力休假去参加训练营。照顾和喂养这三个人包括三份学费，所以我没有钱参加训练营。

他们说是网络。他们说，找一个导师。

我没有的另一件事——我不会为此向任何人道歉——是想放弃晚上有限的家庭时间，去参加我所在地区的无数编程聚会。我甚至不认识一个可以提问的人。

我一个人。或者曾经是，直到最近。

去年年底，当我在 Reddit 上阅读其他人的成功故事，感觉就像 Coderella 希望自己能参加舞会一样，我了解到了**与谷歌一起成长挑战**并申请了。如果被录取，根据学生的兴趣和现有知识，他们将被安排到几门 **Udacity** 课程中的一门。每门课程的尖子生(仍在进行中)将获得 Udacity 奖学金，以完成他们从第一门课程开始的整个纳米学位课程。你可以免费参加这些课程，但付费/奖学金学生也可以获得我现在知道的最有价值的资源——导师和同学社区，他们几乎生活在论坛中，放松下来，全力以赴地互相帮助和鼓励。

有，有 subreddits，**栈溢出**等。但是…我可以说这门课没有他们的任何缺点…但是实际上滔滔不绝地谈论所有的优点要容易得多。每个人都很积极，你可以问什么或多久问一次都没有限制，每个人都很酷，很有帮助，而且…我是作为一个长期的教学设计师说这些的…大多数课程材料(一些视频，一些阅读，一些作业)都远高于平均水平。

我在协作和社区中发现了以前做梦也想不到的宝藏。我找到我的同伴了。

套用一句话，这就是我一直在寻找的文化。每个极客(是的，这是一个词)都非常热爱他们所做的事情，他们情不自禁地通过互相帮助来分享他们的热情。一个没有人想独自站在顶峰的地方，所以没有人会被落下。极客越多越好。
