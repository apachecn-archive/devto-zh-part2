# React 上下文 API:轻松管理状态

> 原文：<https://dev.to/auth0/react-context-api-managing-state-with-ease-3bjg>

React 上下文 API 在 React 的生态系统中并不是一个新事物。不过 React 的 16.3.0 版本给这个 API 带来了很多改进。这些改进势不可挡，大大减少了对 Redux 和其他高级状态管理库的需求。在本文中，您将通过一个实践教程了解新的 React 上下文 API 如何在小型 React 应用程序上取代 Redux 的需要。
