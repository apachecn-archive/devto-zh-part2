你可能知道我是那种每周定期来一次，在每个语言知识库中推销我的示例程序的人。尽管那很有帮助，我仍然需要更多的帮助。

## 每种语言的示例程序

在我开始之前，让我们先谈一谈存储库。如果这是你第一次听说它，它本质上是尽可能多的语言的代码片段的集合。

老实说，这个集合与你可能看到的 [Rosetta 代码](http://rosettacode.org/)没有太大的不同。也就是说，我认为为这个库做贡献仍然是值得的。毕竟，随着每个代码片段的添加，我写了一篇描述解决方案的补充文章。

多酷啊。！

## 有需要的技术作者

问题是存储库在不断增长。似乎每次有人帮忙，都会带来一个新的好主意。例如，有人认为将 Docker 文件添加到他们的[新语言文件夹](https://github.com/jrg94/sample-programs/tree/master/archive/o/opa)是个好主意，这样任何人都可以启动 Docker 镜像并测试代码。当然，我想把 docker 文件添加到所有的语言文件夹中。

也就是说，这不是我写这篇简短文章的原因。我在这里是因为我需要技术作者。对于存储库中的每一个代码片段，[我喜欢写一篇相应的文章](https://therenegadecoder.com/code/sample-programs-in-every-language/),该文章谈了一点语言，解释了代码片段，并分享了如何执行该代码片段。

虽然有代码贡献很棒，但我不可能跟上这些文章。最近，我已经将我的网站向客座博文开放，希望能跟上源源不断的代码贡献。不幸的是，我没有这么幸运。

这也是我向开发社区伸出援手的原因。如果你或你认识的人可能对写一些编码文章感兴趣，**让我知道**。

与此同时，我在我的[投稿文档](https://github.com/jrg94/sample-programs/blob/master/CONTRIBUTING.md)中有一些关于写作指南的笔记，下周我会发表一篇文章，提供更多细节。这里有几篇文章样本，可以让你了解我喜欢写的内容类型:

*   [你好，红色世界](https://therenegadecoder.com/code/hello-world-in-red/)
*   [你好朱莉娅的世界](https://therenegadecoder.com/code/hello-world-in-julia/)
*   [在 Python 中反转一个字符串](https://therenegadecoder.com/code/reverse-a-string-in-python/)

有时我甚至喜欢写文章，讨论我迄今为止对这个系列的了解。例如，我写了一篇关于一些最酷的编程语言特性的文章。

你可以为这个系列贡献自己的力量，看着它成长。
