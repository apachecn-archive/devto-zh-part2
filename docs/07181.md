当我在 General Assembly 做我的编码训练营时，我们做的第一个项目是用 HTML、CSS 和 JavaScript 创建一个游戏。我喜欢写下故事的想法，寻找和创造小的艺术作品，然后写出实际的代码。这只花了一个星期就完成了，非常简单，但是非常有趣！

今年，我的朋友向我介绍了每周一次的游戏聚会，从那时起，我就想学习一些关于游戏开发的知识，即使是 HTML5 和 JavaScript。(我也想学团结。)有大量的框架可以帮助您入门，所以在接下来的几周内，我将关注 pixi.js 和 Construct，因为它们是基于 JavaScript 的。

由于我在 Github 页面上重新制作了我的作品集，我需要一个新的地方来存放我为 GA 制作的游戏。我从游戏 jam discord 和我几周前参加的一个编写游戏 dev meetup 的女性那里了解到了 [itch.io](https://itch.io/) ，所以在那里举办它是有意义的，因为它可以接受 HTML 游戏。这是我的第一个游戏——咖啡因僵尸。看看吧，我希望今年能上传一些新游戏！
