React 是脸书用于构建用户界面的 JavaScript 库。

根据 React 的官方文档，以下是 React 的显著特点:

> *   **Declarative:** React makes creating interactive ui less painful. Design a simple view for each state in the application. When the data changes, React will effectively update and present the correct components. Declarative view makes your code easier to predict, understand and debug.
> *   **Component-based:** Build encapsulated components that manage their own state, and then combine them into a complex ui. Since the component logic is written in JavaScript instead of templates, you can easily pass rich data through your application and keep the state outside the DOM.
> *   **Learn once, write everywhere:** We don't make any assumptions about the rest of your technology stack, so you can develop new functions in React without rewriting existing code. React can also use nodes to render on the server, and use React Native to provide support for mobile applications.

## 28)远程浏览器

[![](../Images/cf12e633d4cd0004c6cf59404bcb0932.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--oI2K5L3L--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://github.com/intoli/remote-browser/raw/master/media/tour-screenshot.png)

Remote Browser 是一个使用 JavaScript 以编程方式控制 Chrome 和 Firefox 等 web 浏览器的库。

你可能以前听说过类似的浏览器自动化框架，比如[木偶师](https://github.com/GoogleChrome/puppeteer)和[硒](https://github.com/SeleniumHQ/selenium)。

与这些其他项目非常相似，Remote Browser 可以用来完成与 UI 测试、服务器端呈现(SSR)和 web 抓取相关的各种任务。

Remote Browser 与其他库的不同之处在于，它是使用标准的跨浏览器兼容技术构建的，其主要目标是促进与现有 API 的交互，而不是创建自己的新 API。
