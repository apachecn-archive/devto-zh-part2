# 作为一名新兵训练营的毕业生。

> 原文:[https://dev.to/braydon322/hitting-the-ground-running 26ko](https://dev.to/braydon322/hitting-the-ground-running---as-a-bootcamp-grad--26ko)

两周前我毕业了，从周一开始，我面临着一生中最大的项目。就复杂程度而言，这确实不是一个大项目，但接近我的人都是一些大人物，我不得不说，我有点吓坏了。

我现在在硅谷，任何了解硅谷的人都知道它的竞争有多激烈。申请工作看起来好像你淹没在一群申请者中，而一个幸运的幸存者能够活着出来。

本周，我有机会测试我的新技能。

一家大公司找到了我——一家以我们都非常熟悉的极其著名的网络浏览器而闻名的公司。我被要求将他们的 API 与另一个第三方 API 集成，合并一些数据，然后呈现在前端。

我选择用 Rails 做后端，React/Redux 做前端。我设置了我的前端，向我自己构建的后端 API 发出异步获取请求。一切都运转得非常好。周三早上，它被展示在公司的两位副总裁面前。

后来我发现了这个项目的价值——以及其他一些竞争对手对我作为一名训练营毕业生在 3 天内所能做的事情的评价。我与提供数千甚至数百美元(一个报价是 12.5 万美元)的大机构竞争。我能够将我的技能应用到真实的职业环境中——老实说，这并不难做到。

很多人可能是来这里寻找接近学习曲线的建议，但我只能说一件事。

当你学习的时候，确保你理解正在发生的事情。我现在作为一名毕业生坐在这里，对我作为一名开发人员的技能和能力非常有信心。

我可以诚实地说,“陡峭的学习曲线”真的没有那么陡峭。要勤奋，不断学习，对自己要有耐心。

好事来了。
