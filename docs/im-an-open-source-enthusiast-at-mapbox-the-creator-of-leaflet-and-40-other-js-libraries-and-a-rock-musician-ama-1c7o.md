# 我是 Mapbox 的开源爱好者，传单和 40+其他 JS 库的创作者，也是一个摇滚音乐人。AMA！

> 原文：<https://dev.to/mapbox/im-an-open-source-enthusiast-at-mapbox-the-creator-of-leaflet-and-40-other-js-libraries-and-a-rock-musician-ama-1c7o>

嘿！我是来自乌克兰基辅的弗拉基米尔·阿加丰金(又名[@哀悼者](https://twitter.com/mourner))。

2008 年，我开始秘密地从我的老板那里开发一个小型网络地图库，他坚持要我停止“重新发明轮子”，只使用“成熟的、已建立的解决方案”。当我联系地图开发者社区时，我的想法被嘲笑为“浪费时间，而不是为现有项目做出贡献”。

这个小图书馆，[传单](https://leafletjs.com/)，最终成为测绘行业最有影响力的项目之一。它被脸书、GitHub、Foursquare、Evernote、Flickr、Pinterest、Etsy、Craigslist、欧盟委员会、NPR、Washington Post 等使用，并在 GitHub 上拥有近 22k 名明星和 600 名贡献者[。](https://github.com/Leaflet/Leaflet)

在过去的 4.5 年里，我一直在 [Mapbox](https://www.mapbox.com/) 构建未来的地图工具，专注于下一代地图技术、[矢量地图](https://www.mapbox.com/maps/)，特别是[性能](https://twitter.com/mbostock/status/976847549503074305)，数据可视化和计算几何算法。以下是我工作的几个例子:

*   [探究空间搜索算法](https://blog.mapbox.com/a-dive-into-spatial-search-algorithms-ebd0c5e39d2a)
*   [我如何用 WebGL 建立风地图](https://blog.mapbox.com/how-i-built-a-wind-map-with-webgl-b63022b5537f)
*   [在 Mapbox GL 中引入热图](https://blog.mapbox.com/introducing-heatmaps-in-mapbox-gl-js-71355ada9e6c)
*   [使用 GeoJSON-VT 动态渲染大型地理数据](https://blog.mapbox.com/rendering-big-geodata-on-the-fly-with-geojson-vt-4e4d2a5dd1f2)

为了跟踪我参与的所有开源项目，我创建了这个列表。感谢 Mapbox，有很多——我非常幸运，也非常感激能有机会全职从事开源软件的工作，并在基辅远程完成这项工作。

在工程之外，我写歌、弹吉他，并在进步摇滚乐队中演唱。我们最近发行了一张已经制作了 8 年的专辑。歌词是乌克兰语，但是[给这张专辑一个机会](http://rain.in.ua/son/en.html)——如果你喜欢优美的、令人回味的、多层次的摇滚乐，它还是会和你说话。

我也是一个快乐的 4 岁双胞胎女儿的父亲。这是我和我的家人去年庆祝乌克兰独立日的照片。

问我什么都行！
