# 什么是 Web 栈！！！？？？

> 原文：<https://dev.to/frugencefidel/what-is-the-web-stack-2njm>

这篇文章最初发布在[我的博客](http://frugencefidel.com/blog/what-is-the-web-stack)。

有人会问你:“你的筹码是多少？”/“您的项目中使用了哪个堆栈？”/贵公司使用哪种堆栈？
在这篇文章中，我分享了我对 Web Stack 的了解。

**Web stack** 是用于 Web 开发的技术/工具的集合。

Web 栈有三种类型，即前端、后端和全栈。

1.  前端栈 -这个栈与浏览器渲染语言一起工作。也叫客户端。使用这个堆栈的开发人员被称为前端开发人员，通常他们使用 HTML、CSS 和 JavaScript。它被称为前端堆栈，因为它提供了你在网站上看到并与之交互的内容。HTML 用于定义网页的结构，CSS 用于样式 HTML 文档，JavaScript 用于向网页添加逻辑和交互性。

2.  后端栈(back end Stack)-这个栈与构建应用程序背后的实际逻辑的语言一起工作。也叫服务器端。使用这个堆栈的开发人员被称为后端开发人员。通常，前端开发工作更多的是关于创建用户界面，而后端开发实现使应用程序工作的实际业务逻辑。在后端堆栈中，大多数操作不是发生在用户面前，而是发生在服务器和后台。后端开发人员使用服务器端脚本语言和数据库语言。服务器端脚本语言是运行在服务器上的语言，例如 PHP、NodeJs、Python 和 ASP.NET。它们用来创建用户、服务器和数据库之间的通信。数据库语言用于读取、更新、删除和创建数据库中的数据。数据库语言有两大类，即 SQL(关系数据库)和 NoSQL(非关系数据库)。

3.  全栈(Full Stack)-这种栈可以与前端和后端栈一起工作。作为一名全栈开发者，你需要了解所有的前端和后端技术。
