# Quora 查询

> 原文：<https://dev.to/yechielk/quora-queries-op3>

我爱我的博客。它让我能够与更多的观众分享我学到的所有很酷的东西和我的想法。

我也喜欢我的博客，因为它让我可以教书。作为一名拉比，我仍然认为自己是一名教师。我发现向别人解释一些事情是我学习和巩固我难以理解的概念的最好方法。这也是我喜欢在 Quora 上写作的原因。

Quora，对于那些不知道的人来说，是一个问答平台，来自世界各地的人聚集在一起分享知识。我在那里保持着相当活跃的形象。虽然我在那里写的大部分内容都与犹太教有关，但我也有一些与编程相关的回答，我想我会在这里与我的观众分享。
