## 云交付模式

在云服务的基础上，我们可以依靠以下云交付模式:

**1)。软件即服务(SaaS)**

它是云的最顶层。最终用户通过互联网按需体验应用程序。云提供了对许多应用程序的访问，用于开发、部署和执行其他有用的任务，如使用 excel、word online、电子邮件、ERP 工具等。

**2)。网络即服务(NaaS)**

云为您的应用程序提供配置、管理和开发网络环境。在这里，您可以在虚拟私有或公共云中部署您的应用程序。您可以设置您的子网组，执行屏蔽，生成静态 IP，管理域名系统，以及添加或删除您需要的所有网络层(如 HTTP、HTTPs、FTP、sFTP 等)的安全组。

**3)。平台即服务(PaaS)**

云提供了一个直接编写、配置和部署代码的平台。你不用担心机器。云会自动管理你的设备，不管你的代码是用哪种语言写的，比如 node.js，PHP 或者 Python。你必须选择正确的，例如，AWS 提供 lightsail 服务，你可以部署它来托管任何语言的网站。

**4)。基础设施即服务(IaaS)**

最先进的特性 cloud 提供 IaaS，便于使用云中的机器资源。您可以根据自己的需求从广阔的市场中配置您的机器。您可以从 Ubuntu、Amazon、Microsoft 等选择机器映像类型。，并且可以批量或并行添加 CPU 以满足您的利用率，根据您的数据输入输出速度等选择固态设备内存(SSD)或 HDD。因此，请始终选择正确的架构，因为这种架构将是您组织的效率支柱。

**5)。存储即服务**

云为你的应用提供存储服务。任何人都可以从服务提供商那里购买从 5GB 到 1tb 的少量云存储。它还将提供您想要购买的存储类型，如磁盘、固态硬盘、IOPS 固态硬盘和硬盘。

**6)。测试即服务**

许多软件、基础设施和其他[测试工具](https://apiumhub.com/tech-blog-barcelona/top-software-testing-techniques/)现在都可以在云中使用，比如 Blazemeter、LoadStorm、Jenkins Dev 等。

## 云提供了什么？

云提供由提供商管理的强大服务。它有像 Alexa，mail 这样的工具来提高业务效率，可以帮助你设置任务的优先级，发展你的业务。它为开发、身份和访问管理、CRM、安全等提供工具。对于数据库，云提供了难以置信的服务，如备份和恢复、快照服务和复制服务。如果您想将数据库迁移到另一个服务提供商，它也有一个迁移服务机制。您可以随时监控云中的资源，甚至设置警报和观察日志文件。Cloudwatch 就是这样一个监控机器的工具。人们可以扩展产品并限制资源的使用，同时确保最短的停机时间。
