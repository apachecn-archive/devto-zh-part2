# 蜂鸟页面的基础知识

> 原文：<https://dev.to/enolalabs/the-basics-of-hummingbird-pages-2b6f>

数字营销人员的一个普遍观点是内容为王。长格式、有意义的内容会吸引注意力、链接，进而提高搜索引擎的优先级。Web 开发人员和营销团队应该记住，在他们的网站中构建产品或服务内页时，内容是关键，但以有意义的方式构建内容变得越来越重要。进入蜂鸟页面。

Hummingbird 页面允许开发人员通过利用页面顶部的内容类型概览表，在给定页面上更好地构建长格式内容。这为产品或服务页面提供了结构和意义，这些页面利用关键字丰富的内容，专注于为用户提供对任何给定主题的深刻见解。

我们来分解一个例子。本页重点介绍 [AWS 咨询服务](https://www.enolalabs.com/solutions/aws-consulting-and-services)。如果你快速滚动到底部，你会注意到这个页面塞满了与亚马逊网络服务相关的有用信息。如果没有目录，用户可能会被信息淹没，很快就会跳出页面，这就为页面在搜索引擎中变得不重要创造了条件。

通过添加一个可点击的目录，将用户引导到他们可能会发现最有用的特定部分，您可以随时提供对正确信息的快速访问。由于长格式内容仍然是许多销售产品或服务的企业的关注领域，一个关键的区别可能不在于内容，而在于它为最终用户组织得有多好。为此，蜂鸟网页是一个很好的解决方案。
