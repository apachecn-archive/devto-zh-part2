## [](#configure-the-software-for-your-convenience)为您的方便配置软件

下载整个播放列表时，最好在软件中设置**自动递增**选项。

playlist-index 将相应地为视频添加索引

```
 %(playlist_index)s-%(title)s.%(ext)s 
```

结合视频的索引

`---playlist-start NUMBER`

从您停止的地方继续
