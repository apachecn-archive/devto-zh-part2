# 从不完美的设计中学习

> 原文:[https://dev . to/gauravchaddha 1996/从不完美的设计中学习-4np0](https://dev.to/gauravchaddha1996/learning-from-an-imperfect-design-4np0)

在实现我的项目的一个高级功能时，我发现我不能支持它，因为它会破坏我的设计和其他功能。这是由于疏忽和跳过研究工作造成的。当我在设计我的项目时，我认为这个功能太先进了，并且没有正确地侦察它的实现，这导致了这次失败。幸运的是，我能够以其他方式包含它，但我完美的初始(或不那么完美的)设计仍然让我失望。

**我所学到的** : *不要留下任何设计方面的事情，并且确保它的实现符合你的设计。此外，确保它的实现不会破坏时间和内存需求。*

而且，这个项目现在终于结束了。作为一名开发者，我感到很满足。所以，无论你在做什么项目，只要完成它。发布 1.0 版，其他的也会随之而来。
