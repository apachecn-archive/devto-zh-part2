# 第四天

1.  关闭闹铃。⏰
    是的，我又做了一次。我按下手机上的闹钟，爬回床上。这个胃的问题真的太不受欢迎了。

    今天早上，我有动力早起一点。今天是星期五。是的，一定是这样。

    我看了看时间，是早上 6 点 55 分。我仍然感觉很糟糕，眼睛疲劳，全身疼痛。这是关键时刻:我是想过另一个糟糕的一天还是美好的一天？

2.  让我们跑吧！🏃‍♂️:我决定选一个非常好的。违背我的意愿，抱怨每一步，我一边听着跑步前的音乐一边穿上跑步装备。

    [https://www.youtube.com/embed/grlPEkzJbU4](https://www.youtube.com/embed/grlPEkzJbU4)

    [https://www.youtube.com/embed/n1t6I7wE5AE](https://www.youtube.com/embed/n1t6I7wE5AE)

    一旦齿轮打开，音乐响起，我的身体会自动知道是时候认真对待了！

3.  **臂章中的电话**📱
    我和[跑步管理员](https://runkeeper.com/user/JoseGonzalez321/)一起记录我的跑步。手机进入我的腕带，打开应用程序，继续收听我的播放列表。

    在跑步过程中喝了一些橙汁来补充能量后，我和我的孩子们说再见，向外走去。

    今天很凉爽，但是太阳出来了。跑步的好天气。我会让结果来说话:

4.  **开车上班**🚗
    冲了个澡，迅速做好准备后，我抓起东西，把手机放进了口袋。我和妻子聊了一会儿天，喝了我的奶昔和茶。🍵

    出于某种原因，内啡肽让一切都变得更好。我感觉很好，并期待着我的通勤。

    我最后听了经典的[如何赢得朋友和影响他人](https://amzn.to/2r55xc9)。这本书我已经听了很多遍了，但是每次我听的时候都会得到一些有趣的东西。

    不批评别人和倾听的第一部分放大了我在[昨天的第三天条目](https://dev.to/josegonz321/no-social-media-in-the-mornings---day-3-5569)中所说的内容。

5.  **工作，工作**👨‍💻
    早上并不疯狂，但下午变得有点紧张，因为我发现我正在写的故事开始出现意想不到的需求。

    当我结束一天的工作时，我的经理看到了我，并问我过得怎么样。

    “老实说，我有点紧张”——我说。

    “今天是星期五，回家享受周末吧。它会在星期一到这里。”

    我没有多余的时间，接受了她的建议，并感谢她如此有益的提醒。有时候，听听脑袋外面的声音会有帮助，你不觉得吗？
