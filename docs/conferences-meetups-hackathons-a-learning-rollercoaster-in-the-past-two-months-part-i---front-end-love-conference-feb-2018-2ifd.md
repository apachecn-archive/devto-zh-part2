# 会议、聚会、黑客马拉松:过去两个月的学习过山车。第一部分-前端爱情大会

> 原文：<https://dev.to/guergana/conferences-meetups-hackathons-a-learning-rollercoaster-in-the-past-two-months-part-i---front-end-love-conference-feb-2018-2ifd>

唷，这两个月是我一生中最无聊的时候。我已经有一段时间想在我的脑海里写一篇文章，但是我没有机会。

[![alt text](../Images/f0327b5211c1b3460bd14861953a4474.png "Front End Love Logo")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--b5jYcrmX--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://yt3.ggpht.com/a-/AJLlDp1jbXqpbHrOD2ljBC2E57rsg2AaJgdjDBKJVQ%3Ds900-mo-c-c0xffffffff-rj-k-no)

这一切都始于 2 月 15 日至 16 日在阿姆斯特丹举行的前端爱情会议。我没有沉迷于经典的“我是单身，今天是 2 月 14 日”，而是决定第一次去阿姆斯特丹旅行，庆祝我此刻的最爱:我的工作(一段充满危机和起伏的关系，就像任何人与人之间的关系一样)。我从未参加过这种规模的会议，当我试图从 Medium 或 CSSTricks 阅读文章来学习新知识或解决问题时，看到我通常会阅读其文章的人，感觉怪怪的。它们确实存在。前端摇滚明星，耶！

会议的中心是赞扬 VueJS 的伟大，尽管有关于 React 和 React Native、Angular 和 Nativescript 的讨论。第二天甚至只用于 VueJS。

发布会的亮点是莎拉·德拉斯纳在那里(请热烈鼓掌，*鼓掌鼓掌*)。她谈到了如何用 VueJS 和 D3 制作很酷的动画。她似乎在很多会议上重复使用相同的幻灯片，
YouTube 频道上正在进行一些讨论。

[![](../Images/d0333e184cf14c14bb0dc9722c7ec2b2.png)T2】](http://www.youtube.com/watch?v=LLnVLjpY6gE)

伟大的肖邦兄弟的现场编码会议:如何在 15 分钟内将一个项目从 jquery 迁移到 VueJS，尽管我刚刚在 youtube 上看到会议名称被更改了。(我的意思是，不仅仅是和你的兄弟一起编写代码，而是和你的亲骨肉一起在会议上发表演讲，这难道不甜蜜吗？

[![](../Images/3fcf36e92c9b957c10cb197d0146ef8a.png)T2】](http://www.youtube.com/watch?v=NcV9U3s2ETQ)

第三个演讲我很喜欢，叫做“Gouda，Edam 或 Maasdammer”。Jen Looper 的《比较 Nativescript、Ionic 和 React Native for Mobile Apps》。这让我特别感兴趣，因为我正在选择一个框架来开发一个工作应用程序，她通过在每个框架中构建相同的应用程序，很好地解释了三者之间的差异。她在 Nativescript 工作，所以猜猜哪个最终成为了她选择的武器！
