# 首都一开发者学院(CODA)

[![CODA](../Images/39173c34a808b901089b2e034b94647e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0zVzzvAc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/i2ji0hkdfbifo4uik0sr.jpeg)

### 简单介绍一下我的工作...

我坚信我拥有世界上最好的工作之一。大约一年前，第一资本创立了一个名为“第一资本开发者学院”的项目。它实际上是一个针对非计算机专业学生的为期 6 个月的沉浸式全栈网络开发训练营(更多信息可在此处找到:[https://medium . com/capital-one-tech/untapped-tech-talent-why-its-value-looking-beyond-CS-majors-6e 242 e 672 cf 3](https://medium.com/capital-one-tech/untapped-tech-talent-why-its-worth-looking-beyond-cs-majors-6e242e672cf3))。

有机会将我的职业生涯转向一个高需求的技术领域，同时还能拿到全职薪水？绝对闻所未闻。每天当我走进工作岗位时，我不禁觉得自己置身于某种幻想世界中，拿着“报酬去学习”。

### 简单介绍一下我所学到的...

像大多数编码训练营一样，理论往往会被抛弃。这就是 CODA 的情况。只有五到六个月的实际训练营，你真正能适应的也就这么多了。然而，我获得的 web 开发知识的数量绝对是相当可观的。分成四个不同的单元，我们的第一个单元相当简单(HTML/CSS 和 Javascript 专用于前端)。从那以后，这个班真的有了起色。从 Node.js、快递、护照、邮差等等的一切都被涵盖了！第二单元以一个合作项目结束(本文第二个大声喊出来的是莎拉)。这个项目可能是迄今为止我在这个项目中所做的最激动人心的事情。这个项目本质上是 CODA 成员的“LinkedIn”。我们的目标是帮助用户连接当前和以前的 CODA 群组。由于我们是一个快速发展的项目(2019 年计划招聘 150 名新员工)，我们认为这是一个好主意。能够在一个有可能被使用的项目上合作，甚至是内部使用，是这个项目最有收获的感觉之一。

### 简单介绍一下我正在学习的东西...

截至目前，CODA 目前正在学习 React！它进行得很好，我可以肯定地看到它在简化 web 应用程序方面有多么强大(肯定是我们正在做的 EJS 布局的一个进步)。学习一些在科技世界中非常有用的东西肯定会更有收获！最重要的是，我们将学习 Angular 和 MongoDB 这样的工具！

# 向前看

正如我之前提到的，这个博客的直接希望是记录我在整个 CODA 中的想法和成长。一旦项目结束，我将立即被安排到公司的软件工程助理职位。我祈祷我会被安排到一个全栈的网络开发角色。我很担心未来会发生什么，但我真的希望这个项目能成为成为一名成熟的软件工程师的第一步。

如果你已经做到这一步，为这篇文章的“咆哮”道歉。我对这个博客的另一个希望是作为一名作家继续成长。毕业才半年左右，但我已经能感觉到我写作的简洁失去了棱角。希望你能在我的旅途中一直陪着我！
