# 远程工作一年

> 原文：<https://dev.to/smakosh/one-year-working-remotely-1k4m>

> 这篇文章最初发表在我的个人博客 [Smakosh](https://smakosh.com/blog) 上。

还记得上次我和你分享我的故事吗[当我拒绝了前端开发人员的工作邀请](https://smakosh.com/getting-my-first-job-offer)然后[在一次黑客马拉松](https://smakosh.com/first-time-first-prize)中赢得了第一名，这很好笑，因为我在 36 个小时而不是 30 天&每天工作 8 小时后得到了他们给我的同样多的钱？

活动结束后，我回家了，第二天早上醒来时，我收到了一封来自一位伟大的首席执行官的电子邮件，他给了我一份在 [Obytes](https://obytes.com/?ref=smakosh.com) 担任前端工程师的工作，我接受了这份工作，原因有很多:(记住，这些都是没有工作经验的方面)

*   我想体验远程工作。
*   他们制造了一些令人惊叹的产品，如获得 220 万美元投资的 Paribus(最近被 Capital one 收购)。
*   使用尖端技术工具。
*   体验分散的团队工作。

当我开始我的远程工作时，我只是用普通的 html、sass & vanilla js 编写用户界面，我们称之为“切割 UI ”,因为我还在学习 React/react-router-dom & redux，而且我还在业余时间从事其他项目，例如:

*   [未命名](https://unnamed.smakosh.com) (css 框架)|开源
*   免费视频下载(PWA 和本地应用程序)
*   [明日 Todolist](https://firebaseapp.smakosh.com) (PWA) |开源

在我从伊斯坦布尔旅行回来后，我收到了一封电子邮件，我也应该在 react 应用程序中做出贡献，否则他们可能会采取严重的行动，如果不是我的同事 [Ismail Mechbal](https://twitter.com/ismailmechbal) 教我如何在那种情况下行动，我的整个职业生涯就不会走这么远。所以我承担了全部责任，并让团队相信我将开始做出贡献。

真的要感谢我的两个队友[萨阿德](https://github.com/saadaouad)和[阿布马利克](https://twitter.com/abumalick0)，他们在为 react 应用做贡献的 3 个月时间里支持了我，因为有时我会接到与使用静态站点生成器(如 Jekyll)构建的登录页面相关的任务，我花了一些时间查找是否有基于 node js 而不是 ruby 的 SSG，然后我发现了 Gatsby js。

当我还在学习《盖茨比》时，我发现它比《哲基尔与鹈鹕》要好，因为我试了所有的书。我用它建立了一些静态站点。

*   [我的个人博客](https://smakosh.com) |开源
*   [鲨鱼云](https://staging.sharkcdn.net)静态
*   [Couscous.bot](https://couscous.bot) 登陆页面

我也在重新学习微积分、线性代数&深入了解单个人工神经元是如何工作的，因为我对机器学习非常感兴趣。除了学习用 Express js & Mongoose 制作 rest API 的基础知识，还学习了路由、模型、http 请求(Post、Get、Patch & Delete)、中间件等等。我学会了在 Heroku 上部署它，并使用 Mlab 在那里托管我的数据库。后来，我构建了自己的 rest api 样板文件。

当我第一次尝试 React native 时，我开始接触本机应用程序，我真的很喜欢它，尽管它并不完美，但更好。我为我的朋友做了一个简单的视频下载/转换应用程序。

忘了分享我当时正在学习 Angular 5，当时公司正在前端使用它，但他们后来改变了反应。

我学了林挺&用 Eslint &更漂亮地格式化我的代码，我一开始不喜欢用超光速粒子，但后来，我真的喜欢上了它。

以下事实确实鼓励我拒绝了其他工作邀请

*   这是一个非常棒的团队，他们真正热爱互相帮助&永不停止学习。
*   了不起的项目业主，他们是技术人员，重视我们的任务和工作。
*   学习和使用尖端技术工具。
*   分享知识的爱。

我知道你们中的大多数人都在等着我给出如何获得远程工作的建议，但是说真的，只要开始做兼职项目，特别是开源项目，这样你的雇主就可以看到你的代码质量，应用程序性能，git 工作流程等..所以一定要写干净的代码，成为一个快速的学习者。

至于这个问题，“我喜欢现在的工作吗？”是的，我做，我有时甚至在周末工作。
