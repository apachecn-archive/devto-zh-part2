> "不要拥有太多的杂物，这样你就可以放心地看到你的房子着火了。"-温德尔·贝里

最近，我想在我的 Mac 上安装 Mojave 的最新测试版，但我没有。原因只有一个:我的电脑有太多的混乱，我的磁盘几乎满了，我的备份驱动器也是。

听起来很熟悉？

在现实生活中，我妈妈多年来一直想搬家，但她没有。她房子里堆积的杂物压垮了她。

你拥有的东西，占有你。

# 数字和现实生活中的杂波是相似的

奇怪的是，我们的数字世界似乎遵循着和家里一样的模式:

1.  我们随着时间积累东西
2.  它挡了我们的路

全新安装就像搬家:令人生畏。

在格式化磁盘和从头开始安装之前，我们需要备份。因此，我们需要在旧房子里精心挑选一些最终会搬到新房子里的东西。

我们害怕:

*   有一天，扔掉我们可能需要的东西，就好像我们的生活依赖于它一样
*   无法再次正确设置我们的机器(如此多的内置首选项、书签、设置文件、到处隐藏的依赖项)
*   在这个可怕的过程中浪费时间

# 格式化是不是太极端了？

其他选项:

*   手动卸载应用程序，清空垃圾箱。
*   尝试使用一次性虚拟机来避免搞乱你的主操作系统。
*   拥有不同的分区和操作系统实例(一个用于开发，一个用于非实验性任务)

所有这些选项实际上都是兼容的，有助于在两次磁盘格式化操作之间尽可能保持干净。

磁盘格式化的优点:

*   它删除所有隐藏/不可见/垃圾文件，你不能手动删除。
*   它删除了所有在你背后启动的降低你电脑速度的后台进程。
*   它删除所有混乱的安装，或配置设置
*   它迫使你在清除数据之前对数据进行整理。
*   它提高了性能
*   你摆脱了“机器依赖”

# 总结一下

定期格式化磁盘，就像经常搬家一样。或者经历一场台风，横扫一切。
你成为一名敏捷的数字漫游者。你轻装旅行，只带重要的东西。

从空虚中，你可以重建。
下载您喜爱的软件的最新版本，更轻松地体验新软件。
从备份和云中恢复您的敏感数据。
为意外做好准备(机器故障、机器抢劫)，习惯于在任何电脑上工作，并接受培训以立即恢复您最喜欢的环境(每个开发人员都应该为此保留一本手册)。

无畏地工作，不受破坏，尝试新事物，拥抱数字自由。
