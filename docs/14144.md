# 如何在线框化上不再是一个 00b

> 原文:[https://dev . to/Charlie develops/how-to-stop-be-a-n00b-at-wire frameing-28fb](https://dev.to/charliedevelops/how-to-stop-being-a-n00b-at-wireframing--28fb)

你的线框很奇怪吗？在公共场合被人看到和他们在一起你会尴尬吗？它们会因为你无法正视它们而变得毫无用处吗？如果你想变得更好，但不知道如何着手，这篇文章就是为你准备的！

首先，你真的试图创建线框吗？当开始一个项目时，十有八九感觉一头扎进代码会更容易。当你意识到你需要一个线框来指导你的时候，感觉已经太晚了。但好消息是，线框永远不会太晚。许多项目已经用一个过期的线框拯救了。在尝试任何事情之前，想想有一个计划的好处——在生活中或在编码中。如果不先看地图或设置卫星导航，你是不会向一个你从未开车去过的目的地出发的。给自己最好的成功机会，并在开始之前创建一个线框图。

无论你是新开发人员还是有更多经验的人，你可能会发现学习线框的想法有点吓人。事实是，在开发中学习新东西没有什么不好，所以要么私下练习，要么在合作项目中练习。通常，当被要求建设性的批评时，其他人会提供积极的帮助。

那么，你实际上如何变得更好呢？这里有一个方便的待办事项列表:

**1。保持低调 T2 你所需要的只是纸和笔。这是将你的想法转换成线框的最快、最简单的方法，而且你不必习惯使用线框软件。另外，如果你不喜欢这个结果，你可以放弃它，再试一次。**

**2。去掉不必要的细节**
确保你的线框符合目的。如果你只是使用线框来指导你的结构或布局，它不必是一个所有的歌唱，所有的舞蹈设计文件，每一个元素都非常详细。线框是页面上指示内容走向的方框的集合，这是完全可以接受的。

**3。贴上标签**
更具体地说，贴上标签来帮助你完成工作，节省你的时间。这意味着识别 HTML 标签，如 div。使用 HTML5 语义元素的额外标记，如页眉、页脚、节和主要部分，以更好地组织您的文档。这会让你在打开笔记本电脑之前就开始思考你的 HTML 结构会是什么样子。

**4。构建你的线框**
将餐巾纸背面的草图变成一个辉煌的网站的过程将帮助你了解更多，你将更深刻地理解你未来需要从你的线框中得到什么。

**5。随着你的发展记录变化**
正如一个企业家明白最好的商业计划是那些随时更新并全年用作指南的计划一样，你也必须保持你的线框图随时更新，以便在你的整个构建过程中充分利用它。把那个`<div>`改成了`<aside>`？将该元素移到新的一行？请务必记录下这些变化。好消息是你用铅笔写的。那块橡皮在哪里...？

7 .**。继续寻找灵感**
出去逛逛 [Codepen](https://codepen.io/) ，查看设计博客，寻找开发者作品集——这些都是找到令人难以置信的创新作品的好地方，这些作品可能会让你的创意源源不断，并为你下一个令人兴奋的线框布局提供灵感。

这些只是一些快速的胜利，让你的线框游戏更上一层楼，增加你的能力，这是一个非常有用的技能。只要坚持练习，你很快就会有信心像冠军一样画线框了！
