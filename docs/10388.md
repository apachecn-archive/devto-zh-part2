# 如何在工作中成为一个叛逆者

> 原文:[https://dev.to/angelariggs/how-to-be-a-rebel-at-work-5ffk](https://dev.to/angelariggs/how-to-be-a-rebel-at-work-5ffk)

最近，我在 Twitter 上看到了这幅插图，它立刻引起了我的共鸣。这幅插图完美地捕捉到了我的感受和工作方式，以及我在角色中努力追求的目标。这是我希望我工作的每一家公司、每一位经理了解我的原因。

我想为那些不仅理解我这样，而且积极支持和鼓励我努力的公司工作。我想为那些希望员工足够关心并提出改进建议和实施变革的公司工作。我总是希望我的团队、我的项目、我的领导尽可能做到最好，我真心喜欢投入工作来帮助他们取得成功。

当你对权力说真话——或者要求权力对你说真话——有很多方法可以有效地做到这一点；带着同情、尊重和成熟。下面的列表并不详尽，但它反映了我在工作中作为一名反叛者的经历，以及我所学到的关于如何开展这些倡导和改进的反叛。

倡导是积极的倾听和积极的交谈。花时间倾听同事和领导的意见，这样你就可以正确地进行双向沟通和倡导。当你倾听相关人员的意见时，你会增加对情况的了解和认识，从而有助于做出正确的决策。

**努力从不同层面理解观点。**了解不同角色和不同层级的人并不总是能获得相同的观点。你所进行的对话的一部分应该是解释你的观点，提供对情况的见解，以及能够引出其他人从他们的角度所看到的。

**询问，不要强求。**持久的变革需要认同和协作。要求有时可以采取强烈要求的形式，但如果你不得不要求参与，你成功的机会会低得多，因为人们没有投入。

提供解决方案，而不仅仅是问题。如果你把时间花在提出问题上，而没有为解决问题出一份力，你就开始从一个有用的工作叛逆角色变成一个只会抱怨的人。仔细考虑如何提出你想要改变的情况，并确保你来自一个合作的地方。

保持开放的心态，为改变主意的可能性做好准备。如果你在进行有效而开放的交流，你会学到更多关于人和事物背后的道理。一旦你接触到一个更广阔的视角，你可能会意识到它需要不同的迭代——甚至根本不需要迭代。把你的叛逆视为学习的机会——改变事物并不是成功的唯一途径。

要有耐心——有意义的改变是通过行动来证明的，而不是言语。不同级别的重大有效变革可能会很复杂，开展对话只是第一步。变化可能会以较小的增量出现，但即使是这些也需要时间才能真正付诸行动。坚持不懈，但是要用耐心来平衡，坚持到底。
