# 让我的投资组合更好

> 原文：<https://dev.to/itsjzt/making-my-portfolio-better-bm3>

不久前，这就像一个投资组合的季节，每个人都在制作新的投资组合。这是我重新设计我自己的投资组合的时候了，这个投资组合一直挂在那里，用你能想到的最简单的设计。

## 想着重新设计。

在重新设计之前，我需要考虑一个看起来不错的设计。我看了@alispitel 的作品集，它有一点帮助，实际上它是一个很棒的设计，但我没有任何方法来制作这个设计的个性化版本。我还看了@benharper 的作品集《独一无二》。在看了很多投资组合后，要注意的是大多数投资组合至少有 4 个部分[关于，联系，项目，社交]。他们也使用 GIFS、动画和画布。

## 获得灵感的设计。

之后看了看擅长设计的网站的登陆页。他们大量使用插图，我也喜欢他们。柔和的颜色、吸引人的标题字体和精美的插图/图像是他们的特色。我现在对未来有了一个模糊的印象，但仍有很多不清楚的地方。

## 编码&寻找真正有用的东西。

此时，我克隆了我的回购协议，并开始尝试新的设计，以找出什么工作。用不同的方法做了之后，但是我没有得到一个吸引人的设计。我用 HTML 制作了 4 个部分，但设计仍然很难看。终于来了 [https://cssgrid.io](https://cssgrid.io) 。如果你不知道，这是神话般的 [@wesbos](https://dev.to/wesbos) 制作的免费 CSS 网格课程的登陆页面。我想到了用 SVG +颜色来表现英雄。我甚至为那张图片使用了多边形。

我使用[英雄模式](http://www.heropatterns.com/)为多边形找到正确的 SVG 和[剪辑](https://bennettfeely.com/clippy/)。

## 调整并使之更好。

我已经做出了设计，现在只需要一些小的调整和一些动画。我在我的名字上使用了一个动画，并使用了谷歌字体中的自定义字体。我对社交图标使用了 awesome 字体。然后我使用了自定义列表样式类型(😫只有 firefox 支持它。得到一个旧的编号列表。

## 在手机和不同浏览器上测试

完成所有这些工作后，我现在需要离开我的主浏览器，在不同的主机和浏览器上运行这个页面，看看它是如何跨平台工作的。我在 chrome Android、Edge、Firefox 上试了一下，除了 Edge 不支持的多边形之外，它似乎工作得很好。我减少了智能手机的衬垫。
