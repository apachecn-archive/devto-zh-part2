> 原文:[https://dev . to/daviducolo/http-status-code-symbols-for-rails-31o 4](https://dev.to/daviducolo/http-status-code-symbols-for-rails-31o4)

一个非常有用的小命令，用于获取 HTTP 响应代码列表及其 Ruby on Rails 符号映射。

```
 Rack::Utils::HTTP_STATUS_CODES
```
