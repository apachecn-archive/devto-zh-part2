# 营利性社交媒体的结构性问题

> 原文：<https://dev.to/djmoch/structural-problems-with-for-profit-social-media-2hg>

很多了解我的人都知道，在过去的几年里，我越来越关注网络隐私。这是一个我非常关心的话题，因为我认为隐私，包括网络隐私，对健康的公民生活非常重要。我希望今后能写更多关于这方面的东西，但现在我想把你介绍给一个人，他的作品我才刚刚开始深入了解，但他在网络隐私这个话题上有很多非常聪明的话要说(除了别的以外):Zeynep Tufekci。

我不想给你写小传，主要是因为我是世界上最没有资格写小传的人。相反，我会带你去看她去年 10 月做的这个 TED 演讲，她谈到了像脸书和 YouTube 这样的公司控制的基础设施，以及他们能够利用他们收集的我们的数据做多少事情。

下面是我在听的时候记下的几段话:

*   这并不是说运营脸书或谷歌的人恶意地、故意地试图让世界变得更加两极化，鼓励极端主义。...但是[这些人]的意图和声明并不重要；这是他们正在建立的结构和商业模式。
*   要么脸书是一个 5000 亿美元的骗局，广告在网站上不起作用...或者说它的影响力令人担忧。

她的处方似乎切中要害:“我们需要一个数字经济，在这个经济中，我们的数据和注意力不会被卖给出价最高的专制者或煽动者。”这实际上只是她更微妙的呼吁的摘要，呼吁企业、消费者和政府共同努力，确定我们对这些系统背后的所有算法魔法(理解为:人工智能)的集体愿景是什么，它们应该如何运行，以及最终必须编码到其中的道德体系。

在过去几年里，我所在的圈子里的福音派基督徒开始认真面对这样一个事实:文化结构的存在往往是为了让权贵掌权。我认为我们中的许多人将不得不看看自由市场和互联网的结合产生了什么，然后就可能需要改变什么进行非常严肃的讨论。
