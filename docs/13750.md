# Android 开发选项

> 原文:[https://dev.to/funkysi1701/android-development-options-55m2](https://dev.to/funkysi1701/android-development-options-55m2)

一个朋友问我如何开始 Android 开发，我想我可以在这里回答这个问题。

我绝不是 Android 开发方面的专家，我在 play store 上有一个应用程序，所以我知道一些东西。

#### [](#manifest-file)清单文件

这可能是最简单的选择，实际上也不会创建一个 android 应用程序，所以我不确定它是否应该包含在这个列表中。

如果你有一个网站，你想创建一个应用程序，你只需创建一个清单文件，并将其添加到你的网站。

一旦你的网站有了一个清单文件，如果你使用手机或平板电脑访问你的网站，你将可以选择在主屏幕上添加一个快捷方式。然后你就有了一个类似应用的体验，你可以点击一个图标来启动你的网站。

清单文件是一个简单的文本文件，它指定了一些设置，如图标大小、文件名、点击时加载的页面以及“应用程序”的名称

[更多信息](https://developers.google.com/web/fundamentals/web-app-manifest/)

#### [](#visual-studio)Visual Studio

这是我最了解的选项，也是我用过的。

如果您熟悉 Visual Studio，您可以使用 Xamarin Forms 软件在 C#中创建您的应用程序。Xamarin Forms 允许您轻松创建在 Android、IOS 和 windows phone 上运行的跨平台应用程序。到目前为止，我只在 Android 上做过实验，但是扩展我的代码在其他平台上运行应该相对容易。

Xamarin Forms 允许您编写一个单一的代码库，可以编译运行在不同的平台上。Xamarin 要求使用 XAML(一种类似 XML 的标记语言)来设计 UI 元素。

关于 [Visual Studio](https://www.visualstudio.com/) 、 [Xamarin Forms](https://www.xamarin.com/forms) 的更多信息

#### [](#android-studio)安卓工作室

我不太了解这个选项，所以如果我没有得到正确的细节，请纠正我。

Android Studio 可以从 Google 上下载，它允许你创建 java 代码直接在 Android 设备上运行。据我所知，这与 Visual Studio 非常相似，但你不用 C#编写代码，而是使用 Android Studio，直接用 java 编写。

[更多信息](https://developer.android.com/studio/index.html)

#### [](#cordova)科尔多瓦

Cordova 允许你使用 HTML，CSS 和 javascript 来创建跨平台的应用程序。我不知道为什么直到今天我才听说这项技术，因为它听起来非常灵活，尤其是如果你懂一点 javascript 的话。

[更多信息](https://cordova.apache.org/docs/en/latest/guide/overview/)

总而言之，创建 android 应用程序有很多不同的选择。你选择什么取决于你想要构建什么，你有什么语言和经验，以及你的应用是否需要跨平台。
