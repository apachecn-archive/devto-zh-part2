# 找份工作，他们说。他们会教你的，他们说。

> 原文:[https://dev . to/k1 pash/find-a-job-they-say-they-teach-you-they-say-5525](https://dev.to/k1pash/find-a-job-they-said-they-will-teach-you-they-said--5525)

[![The job you and I not getting](../Images/0d5e0c3571e2940d6c8cb200d56b4c50.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--QpMBIV-E--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/upbmtn9ig1xuil6wdjvr.jpg)

所以，说清楚一点，我没有被雇佣。我想，但我没有，这就是我故事的重点。

我试图在我的领域找到一份工作。我用 Unity3D 制作游戏，并用 C# .NET 编写代码。令人震惊的是，游戏产业是脆弱的。所以很多小公司都很难生存。问题是，我的目标正是那些苦苦挣扎的人。他们只是无法承受不确定性。

我试着和小的交谈..我失败了...或者他们，很快。所以我问了这里的大人物，他们说了这样的话:“嗯，我们只雇佣大学生或毕业生……”“好吧，那你能在我们办公室呆几个小时？”诸如此类。太年轻，太没有经验，不能做那么多工作。总是有些问题。然后我的校长来了，给了我一份一次性的工作，他意识到我有能力完成项目。

从那以后，一切变得疯狂。首先是小工作，然后是新人。从那以后，我开始通过上课来教学。最重要的是...我们开了一家公司。

找份工作，他们说。他们会教你的，他们说。

这就是问题所在，没有经验的人有自己的公司，并找到出路。我 18 岁了，我不知道我是否真的做错了。

但老实说，我没有太多的选择。这就是让我难过的地方。年轻开发者的市场只是独立的...如果你想在家工作，独立是很好的选择。我不知道。

一位年长的朋友说过，“年轻的自己成长的关键点不在于有多快，而在于是否能成长”。

然后我意识到我活在别人的期望中。事实是..我热爱我的工作，我热爱建立我的事业。我们可能会失败，但这很有趣。我玩得很开心。最棒的是，我仍然可以把它添加到我的简历中。
