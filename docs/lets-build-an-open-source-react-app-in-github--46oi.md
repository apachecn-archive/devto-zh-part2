# 让我们在 Github 中构建一个开源的 React 应用程序

> 原文：<https://dev.to/srebalaji/lets-build-an-open-source-react-app-in-github--46oi>

嘿伙计们，

我一直想开始一个开源项目，帮助人们学习新的东西，并帮助他们与其他人合作

所以最近，我为那些对 React 感兴趣的人启动了一个小的 Github 项目。

下面是链接:
[https://github.com/srebalaji/Todo-er](https://github.com/srebalaji/Todo-er)

该项目的主要目的是将人们聚集在一起，在 React 中构建一个简单的 todo 应用程序。

我正在确保这个项目是(React)初学者友好的，以便正在学习 React 或处于初学者阶段的人可以一起做出贡献

然而，我们欢迎每个人对这个项目做出贡献。
