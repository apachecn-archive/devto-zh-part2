# 十七个白人

> 原文:[https://dev.to/mark_nicol/seventeen-white-dudes-3kgc](https://dev.to/mark_nicol/seventeen-white-dudes-3kgc)

这是一个相当引人注目的视频。这是我的偶像 Sarah Mei [@sarahmei](https://dev.to/sarahmei) 写的，她写了各种关于敏捷、文化、软件以及其他技术主题的好东西。

[https://www.youtube.com/watch?v=YL-6RCTywbc](https://www.youtube.com/watch?v=YL-6RCTywbc)

这是她做的一个主题演讲，其中她提出了一个挑战，即许多敏捷实践至少是无意排斥的，如果我们不解决问题，我们可以看到这变成了有意排斥。她列出了一些关于办公室里的动机、接触和权力动态的问题。

我建议看完整部电影。然而，我今天思考的是大约 18:16 的关键时刻，她放了一张敏捷宣言签名者的幻灯片。这实际上是 17 个白人，在年龄和外貌上没有太大的不同。

[![Seventeen white guys in a grid](../Images/816196a8d4e8d25e53aeb709b4f71f91.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--n1D8vYIh--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/zwtfxkbcxoho4gvhflwf.png)

我想知道，如果一个不同的，也许更多样化的团队最初聚在一起，敏捷会有什么不同？

这个问题我还没有任何答案。我所在的团队是一个由来自不同文化的完全不同的人组成的小团队，他们被地理位置所分割，但仍然知道如何做事。我们正在一起学习敏捷，可能有很多地方我们可以做得更好。

其他人对不同的、更加多样化的敏捷会是什么样的想法？

不一定要基于现实。创意很有趣。但是听到一些人们要么做对了要么做错了的故事是令人高兴的。
