# 在 React Native 中设置 TextInput 组件的边框颜色

> 原文：<https://dev.to/skptricks/setting-border-color-of-textinput-component-in-react-native-54gl>

帖子链接:[https://www . skptricks . com/2018/08/setting-border-color-of-textinput-component . html](https://www.skptricks.com/2018/08/setting-border-color-of-textinput-component.html)

本教程解释了如何在 react 本地应用程序中更改 TextInput 组件的边框颜色。在这个演示中，我们在 TextInput 布局组件中添加了一些样式表设计属性，这有助于改变文本输入字段的边框颜色。这里我们指定了 TextInput 组件中 CSS 样式表的 borderWidth，borderColor 属性。

[![](img/dbc47ba8944ca090bef709daf49f03b8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Lo_YsOAu--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://4.bp.blogspot.com/-VLMhXopVvfs/W3AfDAA0O-I/AAAAAAAABx8/0rwJummWWxYF529vNXFSDMe_kNhPTAz4QCLcBGAs/s400/bc.jpg)

更改 TextInput 布局的边框颜色的代码段:
style=
//使用占位符选项在 TextInput 中添加提示。
placeholder= "输入你的名字"
//使底线透明。
underlineColorAndroid = " transparent "
/>
