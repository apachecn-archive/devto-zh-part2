# 宣布链接:一个用于分组、保存和发送链接的工具

> 原文：<https://dev.to/ognjengt/announcing-capsulelink-a-tool-for-grouping-saving-and-sending-links-as-one-1l6f>

今年年初，我制定了关于发布/部署更多产品的新年计划。即使我对他们的结局不太满意，我也会发布最终结果，看看我从用户那里得到的参与度。所以直接进入主题。

一个多星期前，我出版了《http://capsulelink.com/》。这是一个简单的工具，让您保存多个链接到一个，并减轻你的痛苦发送许多链接给许多其他人。相反，你只需给他们发送一个链接，它将包含所有其他的链接。

[![First version of capsulelink.com](../Images/f268bbebfaf5b4f54cbedd90811e4d7e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--MAh9rMbv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/jgjddspz88hguyf53yiq.png)

起初，它是一个准系统产品，功能有限。但是我一直在添加更多有用的东西。

# 特性

匿名用户(未登录的用户)创建的链接在 48 小时后过期，而如果你有帐户，你的链接永远不会过期。因此，这是一个很好的和快速的方法来保存你目前正在研究的一些网页，并在任何时候访问它们。

[![Group link](../Images/3fa3d95ea56a8b1af568d000cc6a2951.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--IcMbEf-6--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/tb9sxjo9wcvtlj6eemxl.png)

此外，还有注册用户的私人链接。私有链接仅对创建它们的用户可见。

[![Private links](../Images/7e40842e2c0249acd51034403da66858.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--qM3m0Xzb--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/esdv1bsl00k144qkrho8.png)

最近我看到了这个应用的发展方向，所以我增加了向上投票功能和趋势页面。通过这种方式，其他人可以发现有用的链接，这是合理的有用。

说实话，我自己也发现了一些很好的工具，其他人发布的，包括网站、工具、文章等等。

然后我发现有时候总是不得不切换到 capsulelink.com 来粘贴你想要保存的 URL 是一件痛苦的事情。所以我创建了 chrome 扩展，事实证明它甚至比网站本身更容易使用。

[![Chrome extension in action](../Images/5804f3c74d3886eea0ce35d5f4495c11.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Ju2vUUC6--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ecxtgrzd5zyk5smn0krp.png)

可以在这里找到:[https://chrome . Google . com/web store/detail/capsule link/khgappllikejideeefljdafjbnnclbno](https://chrome.google.com/webstore/detail/capsulelink/khgappllikejideefljdafjbbnnclbno)

只需一次点击，您就可以添加网站，然后当您添加了您想要的页面时，只需一次点击，您就可以创建群链接，并将其保存在服务器上。

虽然，目前 chrome 扩展只创建匿名链接，不支持登录功能，但我正在努力，并尽快部署。然后，您将能够直接保存到您的个人资料。
