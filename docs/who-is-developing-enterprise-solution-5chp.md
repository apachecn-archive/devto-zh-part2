# 谁在开发企业解决方案？

> 原文：<https://dev.to/imben1109/who-is-developing-enterprise-solution-5chp>

我认为我正在构建企业解决方案。

企业解决方案是一个复杂的系统，它嵌入在企业中，帮助企业解决问题。

由于它很复杂，开发商可能会面临很多困难。

让我们分享你面临的问题。我想我可以帮助别人。

# 我的困难

*   不明确的要求
*   用户认为它可以变魔术(我们可以变魔术)
*   排得紧紧的时间表
*   没有领导，只有老板
*   糟糕的文档
*   肮脏代码/技术债务
*   没有计划
