# 引爆点-第一部分

> 原文：<https://dev.to/muddybootscode/the-tipping-point---part-1-3cho>

从 2016 年开始，我开始朝着成为一名软件开发人员的目标努力。当我找到时间和灵感的时候，我会断断续续地写博客。我最受欢迎的努力是我的第一次，[当单干辞职时(或者我如何学会热爱编程)](https://realworldcoding.io/when-going-it-alone-quit-working-or-how-i-learned-to-love-programming-d6e011d9b3bd)写完那篇文章一年半后，我很高兴能够说一些事情:

**一个**，我从事软件开发工作已经快一年了。

**两个**，除了我主要编写 Python web 抓取代码的日常工作之外，我已经能够使用 React、Node.js 和 NoSQL 数据库(如 MongoDB 和 Neo4j)自学全栈 web 开发。

**三**，我已经准备好把这一切带到下一个层次，建造一些非常诚实的东西，我不确定我是否准备好了，它看起来就像这样。

现在你可能正在看这个，挠头，想知道这到底是什么。长话短说，这是土地和矿产所有者与石油公司之间的关系图，石油公司租用他们的土地来钻探油气井。这是这些关系的一个子集。追踪这些关系所涉及的问题数不胜数，但我想我知道怎么做。

[![Subset](../Images/d676e34cce4be67718ea8252ca40676c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--avWJD89u--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4drh8873evf8pmzkl0s2.JPG)

如果你没有读过我以前写的任何东西，只需知道我生活在西德克萨斯州的大规模石油繁荣中。虽然我已经成功地让自己摆脱了泥泞的靴子，进入了一个写代码的办公室，但我一直没有成功地让自己进入一个不同的地理位置(这很快就会发生变化)，所以我找到的许多工作仍然与石油有关。简单来说就是钱在哪里。我为一家家族企业做兼职，自动化一些更艰巨的任务，在这项工作中，我发现了一个问题，我认为我可以用我学到的技能为他们解决这个问题。

过去的一年是令人惊奇的学习之旅。我加入了一个导师项目，学习 JavaScript 及其最佳实践。我从他们的 React/Redux 项目中获得了 Udacity Nanodegree 学位，并且每当我有机会时，我都会参加我所能得到的每一个教程来学习 GraphQL 以及如何构建数据库，我最喜欢的是提供 Neo4j 的 NOSQL。我努力完善自己，成为一名真正的全栈开发者。我已经达到了临界点，我觉得我可以利用所有这些技能创造一些真实的东西。我希望在这一系列文章中详细介绍我试图解决的问题、我选择的技术以及原因。
