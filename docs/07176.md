# 为任何 macOS 应用程序自定键盘快捷键

> 原文:[https://dev . to/real _ ishan/customizing-keyboard-shortcut s-for-any-MAC OS-app-2l9f](https://dev.to/real_ishan/customizing-keyboard-shortcuts-for-any-macos-app-2l9f)

如果你正在读这篇文章，那么你很有可能在日常工作中使用键盘快捷键。从剪切、复制、粘贴到“粘贴和匹配风格”，再到 IDE 的代码折叠和展开，有些快捷方式您甚至没有意识到。大多数应用程序允许你自定义快捷键，但你可能会发现一些不支持这一点。

几周前我就遇到过这种情况，当时我需要一个快捷方式来存档邮件应用程序中的邮件。默认的快捷键不是很直观，我需要一些快捷的东西，比如`⌘ + A`。

经过一番搜索，我发现 macOS 有一个内置设置，可以让你为任何应用程序定制快捷方式。

你只需要进入系统首选项→键盘→快捷方式→应用快捷方式，添加一个新的即可。您可以为特定应用程序或整个系统添加快捷方式。

我在邮件应用程序中使用的一些其他快捷方式:

1.  `⌘ + 1`对于编号列表
2.  `⌘ + 8`对于未编号的列表
