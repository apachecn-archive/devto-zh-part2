# [](#whats-in-an-icon)图标里有什么？

把图标想象成你的应用程序的品牌形象。应用程序图标是应用程序的可视化表示。它出现在用户移动设备的主屏幕上和 app store 中。

## [](#a-few-elements-of-a-successful-app-icon)一个成功应用图标的几个要素

*独特的*
你的应用图标需要与众不同。它必须有一个元素让它与众不同。

用户友好的
你的应用程序的图标需要易于学习和理解。图标应该是干净和简单的。

用户需要能够从以前的接触中或从对图标的了解中认出或识别出你的图标。

*传达含义*
用户应该通过查看图标来了解应用程序的功能。

*难忘的*
人们应该记住你的应用图标。使你的图标独特和用户友好有助于使它令人难忘。

# [](#icon-vs-logo)图标 vs 标志

理想情况下，图标应该代表用户可以在应用程序上执行的操作。如前所述，它应该*传达含义*。另一方面，标志可以被认为是代表一个公司或一个组织的形象。
