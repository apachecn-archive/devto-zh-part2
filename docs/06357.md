GitHub 处于 Rails 的风口浪尖是件大事。Shopify 最近一直处于领先地位，我觉得这种新的参与将进一步推动事情的发展。Rails 在保持概念一致和避免特性蔓延和复杂性蔓延方面做得很好。

不管你喜不喜欢 Rails API，这个生态系统依然强大。

对于庞大的 web 框架来说，Ruby 是一个奇怪的选择，它看起来确实像是一个玩具脚本语言，但事情已经在 Rails 上运行了。参与其中的人类推动了事情的发展。

Rails 仍然是最佳选择之一，而且只会越来越强大。在这个问题上，你可以随意和我辩论！

欢迎其他人代表我参加这场辩论。这是一场社区辩论。我不确定我是否真的想参与这场对话。我们会看到的。😋
