有几个非常简单的原因可以解释为什么新手可能处于现代编码领域的最佳位置。鉴于现在学习编程可能比历史上任何时候都要容易。

虽然，这个假设纯粹是基于这样一个事实，你没有固定的方式，你没有接受大量不同的意识形态。简而言之，你不是一个粉丝...然而...

令人悲伤的事实(以及最终的现实)是，很快你就会！所有的陈词滥调都完全正确。我认为这不会发生在我身上，但它不可避免地发生了。所以，这是我的尝试，在对你来说太晚之前引导你走向光。

你可能要做的第一件事就是寻找一个代码编辑器或 IDE(集成开发环境)。有些人更喜欢重型 IDE，它能够处理你扔给它的任何东西。

其他人实际上可能会被更简单的工具所吸引，wysiwyg 编辑器允许你编写 HTML5 / Markdown，其中许多都可以在网上找到。

就像所有的工具一样，如果你使用锤子，它就是你的锤子，你最终会喜欢上它。并不是说你的锤子比以往任何一把锤子都好。只是你的锤子帮着成就了这么多。它就像你武器库中的一件武器...值得信赖的工具，我们都知道赢得信任有多难。所以你和你的锤子有着独特的关系。它对你和你一起工作。事实上，它甚至会让你觉得自己做得更好。

这对任何工具都是正确的，甚至对大多数日常物品也是如此。那么这如何应用于编码呢？

你的 IDE 就是你的锤子！

(所有这些锤参考，而不是一个单一的北欧神双关语，我必须滑这些天)。别担心...

实际上，你的 IDE 最终会感觉更像蝙蝠侠的工具带而不是雷神之锤。里面装满了东西，大部分你都用不到。但是一些令人敬畏的技巧、工具，甚至可能到处都有一些魔术，因为程序员似乎对戏剧感兴趣，有点像蝙蝠侠。

许多开发人员会非常乐意告诉您他们使用的 IDE，以及为什么它比其他任何 IDE 都好。就像任何一个带着锤子参加战争的人，一次又一次地取得胜利，他们甚至会写下关于锤子的传奇，这取决于你实际上用它做得有多好。

还有，你得有多疯狂才能拿着锤子去打仗？就像那些北欧传说一样，让我们把它留给后人去写吧。

此时此地，你的 IDE 就是你的锤子/工具带，没有它你不会想离开家。

所以选择你的第一个 IDE 是一个重要的话题，大多数人都没有真正探索他们的选择。你不认为托尔曾经不得不用剑吗？就连他最近也被迫用锤子换了一把大镐。

所以，是的，虽然这与工具无关，但关键是正确的工具会让整个过程变得更愉快，更有希望...富有成效。

选择合适的工具就是尝试一些事情，直到你觉得合适为止。所以找点乐子，做些酷的东西，然后锤打一会儿，直到感觉合适为止。

这里列出了市场上最好的 IDE。没有一个像另一个。他们可能都有共同的特征，其他人可能有大量的特点。有些是专有的，有些是开源的。简而言之，它们各不相同，就像编程世界中的其他事物一样。
