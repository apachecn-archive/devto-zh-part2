# i3 是什么？

来自[官方网站](https://i3wm.org/):

> i3 是一个平铺窗口管理器，完全从头开始编写。目标平台是 GNU/Linux 和 BSD 操作系统，我们的代码是 BSD 许可下的自由开源软件(FOSS)。i3 主要面向高级用户和开发人员。

# 怎么拿？

## Ubuntu

你只需要:

```
sudo apt install i3 
```

Enter fullscreen mode Exit fullscreen mode

然后重启你的机器，选择 **i3** 然后登录你的用户。

## Manjaro Linux i3 社区版

或者你可以尝试预装了 **i3** 的 Linux 发行版。

在这里看看:[曼哈罗社区版](https://manjaro.org/community-editions/)。
