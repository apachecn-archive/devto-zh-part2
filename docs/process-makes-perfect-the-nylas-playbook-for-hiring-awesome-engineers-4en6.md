# 过程造就完美:雇佣优秀工程师的 Nylas 行动手册

> 原文：<https://dev.to/nylas/process-makes-perfect-the-nylas-playbook-for-hiring-awesome-engineers-4en6>

我们在招人！如今，这是任何聚会或会议上的常见说法。但是你怎么能确定一个工程师拥有你的团队现在和将来都需要的技能呢？你如何评估文化契合度？你如何让面试过程对你的团队和每个候选人都变得精彩？这些都是我们在 Nylas 一直在考虑的问题。

作为一家快速成长的初创公司，我们的团队没有找到反映我们价值观的全面而多样化的候选人。

作为提出解决方案的一种方式，我们制作了自己的招聘手册。这是一个迭代的过程，但我们希望分享我们的一些见解，希望它们能帮助其他技术领导者雇佣优秀的人才。

## **确定是什么让你的公司与众不同**

**剧本策略#1:** 找到你的公司对候选人的独特价值，并广而告之，让他们在申请前了解你公司的文化。

作为一个团队，我们知道我们需要找到一种方法，将自己与其他寻找世界级工程师的创业公司区分开来。

但是是什么让我们与众不同？

我们集思广益，想出一个计划来宣传我们公司的故事和特点。我们发现了一个名为 [<u>Keyvalues</u>](https://www.keyvalues.com/nylas) 的很棒的网站，它将传统的公司招聘页面从*我们做什么*转变为*我们是谁*。我们思考了定义我们的价值观，并提出了一系列理想:

*   工程驱动:我们的工程目标驱动其他公司目标
*   工作/生活平衡:我们尊重你工作之外的生活
*   客户至上:我们的开发成功工程师在客户和产品之间架起了桥梁
*   开放式沟通:我们每周召开全体会议，以保持整个公司的一致
*   积极实践包容性:我们提供工具和培训，以确保我们的流程是协作性的
*   自始至终的所有权:我们有一个开放、自治的结构，每个人都是全栈工程师
*   开源贡献者:我们的团队为 Debian、mypy 等开源项目做出贡献。我们也是活跃的成员，并出席会议，如 Pycon 和 Pybay，以及其他当地聚会
*   社区:我们喜欢在每天的午餐和下班后的晚餐上一起进餐

今年早些时候，我们还<u>开源了我们的员工手册，这样任何人都可以看到我们的公司政策。这有助于我们回答许多关于休假、福利、远程工作等常见问题。我们在福利待遇方面非常努力，并强调了我们引以为豪的因素，例如我们的看护假政策，该政策给予父母双方带薪休假，并包括照顾老人的时间。</u>

 <u>经过几个月的努力，我们从许多候选人那里听说，他们能够在进门之前就了解我们的文化。这太棒了，因为它让我们把注意力集中在更深入的讨论上。我们知道随着这些变化，我们正在做一些事情，这些透明度的努力成为我们剧本的第一部分。

## **让你的候选人多样化**

**剧本战略#2:** 实施将多样性和包容性融入公司生态系统的方法，并与你支持的代表性不足的组织建立合作伙伴关系。

我们在透明度方面的努力是实现我们目标的非常重要的基础步骤，但这只是建立我们想要成为的公司的一部分。纵观整个软件工程世界，我们已经看到了关于工程团队缺乏多样性的讨论。我们希望找到一种方法，确保我们尽自己的一份力量来帮助弥合这一鸿沟。

我们的工程副总裁 Tomasz Finc 是去年夏天 [<u>Code2040</u> Code2040 在支持更多工程专业的黑人和拉丁裔学生方面处于领先地位，我们很高兴今年夏天迎来了我们的第一位实习生。](http://www.code2040.org/)

我们还与 HackBright 学院合作，支持更多女性进入工程领域。HackBright 是旧金山湾区领先的女子工程学校，我们很高兴在他们的支持下发展我们的公司。为了更好地平衡工程世界，还有很多工作要做，但我们非常自豪能够与志同道合的组织直接合作，推动工程世界向前发展。

我们现在积极讨论如何支持更多代表性不足的候选人。作为我们行动手册的一部分，我们实施了一个重复的跨公司招聘偏见会议，我们每个人都参与其中，以确保我们尽一切努力将这些群体融入我们的招聘工作。

## **钉钉面试经历**

**剧本策略# 3:** 从每一个候选人那里收集关于你面试过程的反馈，在评估一个潜在的新员工时，创建一个标准化的过程，以尽量减少偏见。

一旦我们邀请申请人与我们进行现场面试，我们创造一种让他们感到舒适并允许他们最好地展示自己技能的体验是非常重要的。我们的每一次采访都使用一套标准的问题，这些问题是用一个标准来判断的，以尽量减少偏差。我们关注候选人如何符合我们公司的价值观。我们努力使我们的面试循环尽可能多样化，并吸收工程以外的部门参与。我们的流程还包括每位面试官通过[杠杆](https://www.lever.co/demo-signup?utm_campaign=2018-Google-paid&utm_source=ppc&utm_medium=demo)以个性化的方式提供反馈，然后评估其他面试官对候选人的其他看法。在这个过程之后，所有的反馈都会被集中讨论。

在每次面试结束时，我们一定会发出一份调查问卷，以便更好地了解如何改进我们的面试流程。剧本的这一部分产生了大量的观察结果，供我们在下一轮面试中实施，让我们每次经历这个过程时都能做得更好。我们希望继续成长，并从所有这些努力中学习。
