# 用 VS 代码进行结对编程

[https://www.youtube.com/embed/vq5FzSPmu-I](https://www.youtube.com/embed/vq5FzSPmu-I)

今天我要和你分享一个很棒的插件，它可以让你和其他人一起编写代码。叫做 VS 直播分享。

我们今天要做的是设置 VS 实时共享。我还将向您展示如何开始协作会话和加入协作会话。

## 安装 VS 实时共享

你要做的是安装 [VS Live Share](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare) 扩展。VS Live share 扩展可以在 Visual Studio Code 的扩展市场中找到。

在搜索框中输入 VS Live Share，你就可以找到它了。

然后，单击 install 安装扩展。

然后，点击重新加载。

## 登录 VS 直播分享

您将在状态栏上的 Visual Studio 代码的左下角看到一个“登录”按钮。点击“登录”，会弹出一个窗口要求你登录。

您可以使用 Microsoft 帐户或 Github 帐户登录。就我而言，我使用 Github。

现在，你已经准备好合作了。

接下来会发生什么取决于您是开始会话还是加入会话。

## 首发

如果您正在启动会话，您可以单击左下角的您的姓名，然后选择启动协作会话。或者，您只需点击“共享”即可开始会话。

一旦开始会话，VS 代码会在右下角显示一个通知。它会说一个链接已经被复制到你的剪贴板。

你可以点击“再次复制”来确保链接被复制到你的剪贴板上。

这个链接看起来是这样的:

```
https://insiders.liveshare.vsengsaas.visualstudio.com/join?0EUI09T34IISNETHUI09435 
```

Enter fullscreen mode Exit fullscreen mode

您会将此链接传递给您的合作伙伴，让他们加入您的会话。

## 加入会话

要加入一个会话，你点击左下角的你的名字，然后点击“加入协作会话”并粘贴你的伙伴给你的链接。

就是这样。

## 变得更好

你可能想使用视频聊天室应用程序，如 [Zoom](https://zoom.us) 或 [appear.in](https://appear.in) 在聊天的同时输入代码。
