# 市场上集成了 Pentaho 数据的自适应大数据层

> 原文：<https://dev.to/dhrumitshukla/the-adaptive-big-data-layer-with-pentaho-data-integration-in-the-market--1f62>

自过去几十年以来，业务发展突飞猛进，导致许多系统在个人和组织层面产生大量数据。这就迫切需要将所有数据整合到一个平台中，并进行深入分析，以帮助组织做出正确的决策。

Pentaho BI 套件是世界上最受欢迎的商业智能套件，用于报告、分析、仪表板、数据挖掘、工作流和 ETL 或导出转换加载功能。有许多软件开发服务提供商是 [Pentaho 业务分析和集成开发](https://www.tatvasoft.com/software-development-technology/pentaho-development)方面的专家，并为所有向尊贵客户提供最先进解决方案的堆栈提供服务。

**自适应大数据层**
[自适应大数据层(ABDL)为 Pentaho 用户](http://www.pentaho.com/pentaho-data-integration-adaptive-big-data-layer)提供了使用任何大数据源的能力，提供了一个完全正常运行的 bugger，将 IT 开发人员和分析师与复杂的数据隔离开来。作为 Pentaho 数据集成的核心组件，ABDL 将数据开发人员与数据分析的流沙隔离开来，并允许“创建一次，在任何地方运行”的转换流程与任何大数据商店对抗。

当前 Pentaho 数据集成的改进有助于大数据项目快速实现价值。与 Spark 的更多集成、Hadoop 安全兼容性的新[级别](https://hadoop.apache.org/docs/stable/hadoop-project-dist/hadoop-common/SecureMode.html)以及扩展的元数据注入功能相结合，使公司能够以更有效的方式管理企业大数据供应，同时加速和实施创新以提高投资回报。

**PENTAHO 数据整合平台概述**
PENTAHO 的整合平台允许公司整合、混合、转换和变换来自整个企业的任何数据源的数据。该平台提供提取、转换和加载必要的功能，以集成广泛的数据源，包括企业应用程序、关系数据库、文件和大数据。

当前的 PDI 版本 6.1 提供了以下功能:

提供图形 ETL 设计器，使数据集成团队能够设计、测试和部署集成流程、通知、工作流和警报。

提供广泛的预建数据集成转换库，支持复杂的流程工作流。

允许连接到广泛的大数据存储、关系数据库、文件和企业应用程序，作为集成项目中的源或目标。

提供基于存储库的开发工具，管理支持元数据和集成流程的设计、创建、测试、部署和操作。

允许用户在准备数据和向分析工具发布元数据模型的过程中可视化数据。

此外，该版本为用户提供了将数据转换为数据服务的能力，这使得来自服务的查询结果能够作为虚拟数据表进行分析。此外，最新版本还通过支持 Hadoop 的 Cloudera 分发以及使用 Spoon 连接到 Hadoop 集群来提供增强的大数据功能。

**世卫组织受益于 PENTAHO 集成平台？**
小型、中型以及大型企业都使用该平台来提供一个有凝聚力的、全面的数据集成和业务分析平台。除了直接销售， [Pentaho 还嵌入了 OEM 网络](https://www.pentaho.com/product/embedded-analytics/software-saas-products)，允许供应商利用数据集成和分析能力扩展其产品。除了商业版本，Pentaho 还提供了一个名为 Kettle 的数据集成产品的开源版本。许多公司最初开始使用开源工具 Kettle 来探索集成能力或有限的集成工作负载。

**享受这些优势**
Pentaho 是一个强大的数据分析平台，为希望从其数据中获取更多信息的企业提供了一系列优势，例如完整而强大的可视化功能，让用户可以清晰地看到数据，放大信息以及其他远远超出统计数字的相关细节。通过内存中的数据缓存获得信息的实时分析。利用基于 web 的可定制交互式拖放式仪表盘和充满过滤功能的库，实现完全控制。数据集成系统允许用户融合来自其他信息库的信息，包括 NoSQL、Hadoop、关系数据库和分析数据库。

**受益于全面的分析工具**
Pentaho 的仪表盘和报告提供了深入的数据分析。此外，由于极端的内存数据缓存，即使是大量的数据也可以以闪电般的速度进行分析。人们可以使用 Pentaho 数据整合工具以及其他工具来访问、混合和管理来自各种来源的数据。此外，人们可以将商业分析与其他软件应用程序相结合，如谷歌地图。分析工具可用于从数据中获取可操作的见解，并做出信息驱动的决策。

Pentaho 的大数据解决方案非常适合金融、政府、零售、服务和医疗保健行业的企业。它们允许用户访问、合并和管理来自不同来源的数据。
