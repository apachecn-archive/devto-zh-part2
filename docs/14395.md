# 然而，卡罗琳娜·希门尼斯编码

## [](#i-began-to-code-because)我开始编码是因为...

我迷路了...我刚刚获得商业学士学位，正在全职工作，论文也写了一半。后来有一天，我意识到我正在做的事情没有一件让我满意，没有一件事情令人兴奋，没有挑战性，没有目标。什么也没有把我带到阶梯的下一步。就在这个时候，在我身边有一个声音多年来一直告诉我要学习代码，这像中风一样打击了我。

第一次认真对待，没有在学习时间睡着(可能就那么几次)。能够从零开始构建一些东西，并且知道我写的那些行可以在未来帮助某人，这让我恢复了呼吸。

我仍在探索这条道路，但随着时间的推移，我越来越确信这是正确的道路。

## [](#i-recently-overcame)我最近克服了...

我不断的消极，这是通过学习更有弹性。

## [](#i-want-to-brag-about)我想吹嘘一下...

事实上，我每天都能理解更多。

## 我对支持女性和想编码的非二进制人的盟友的建议是....

去吧！任何时候都不晚，如果你喜欢，你会找到时间的。不要害怕，我们比你想象的要复杂得多。
