# VSCode 主题可突出显示的缩进参考线💯

> 原文：<https://dev.to/vscode/vscode-theme-highlight-able-indent-guides-ftw--2hmi>

*这只是 VSCode 的一个小技巧。*

自从 VSCode `v1.23`发布以来，你可以设置活动缩进指南的样式。😲只有当你的 VSCode 主题支持时，VS Code 现在才可以突出显示活动的缩进参考线。我发布了我的新版本[🦄紫色的阴影](https://marketplace.visualstudio.com/items?itemName=ahmadawais.shades-of-purple)支持这个特性的 VSCode 主题。

> 所以，当你在不同的源代码块之间移动光标时。色名为`editorIndentGuide.activeBackground`。

我将此作为 [VSCode 提示](https://vscode.pro/tip/7) →
