# TypeScript 3.0:探索元组和未知类型

> 原文：<https://dev.to/auth0/typescript-30-exploring-tuples-and-the-unknown-type-4a1>

TypeScript 3.0 出来了！它附带了对类型系统、编译器和语言服务的增强。此版本附带以下内容:

*   项目引用通过允许 tsconfig.json 文件引用其他 tsconfig.json 文件，使 TypeScript 项目依赖于其他 TypeScript 项目。

*   支持 JSX 的 defaultProps。

*   rest 参数中的元组和带有可选元素的展开表达式。

*   新的未知顶型！它是任何。

在这篇博文中，我们将把重点放在对元组和未知类型的增强上！
