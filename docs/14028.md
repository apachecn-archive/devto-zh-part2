#### **一旦安装了 Atom Runner，运行 Perl 脚本有两种选择。**

 **#### 第一个选项:使用 shebang 行** 
[![](../Images/ca7b78646b51c16032d95c1314edff8c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--HtPqovr9--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4stoi9mueq6g4j8w9r2f.png)
 **#### 第二个选项:编辑 Atom 配置文件** 
打开 config.cson 文件
[![](../Images/23958ba8beb4ca47ed89935004521b89.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--JqESOHpa--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/m6sqfayxzex35z1v1mrm.png) 
并插入
[![](../Images/7fb34b013f08bcb0cb2f87e537aed90a.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--8vCucVeJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/yq9gmgx0kewe6jiz80t8.png)
