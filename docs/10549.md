我是一名全职开源开发人员，作为 WordPress 的核心开发人员，在过去的几年里，我为每一个主要版本做出了贡献(不算在 Twitter 上开玩笑和制造谣言的时候 [@MrAhmadAwais](https://twitter.com/mrahmadawais/) )。

我还精通 JavaScript/Node.js 生态系统，对 DevOps、云以及新的无服务器技术有浓厚的兴趣。我非常关心开发人员的体验，我的大部分开源工作都与开发工具相关。[更多在此→](https://AhmadAwais.com/)

既然你已经了解了我，你就更容易理解我为什么写这篇文章，以及为什么我认为我的观点很重要。下面是我在 Twitter 上创建的关于这个话题的帖子的复制粘贴:
