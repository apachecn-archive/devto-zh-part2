浏览我最喜欢的应用替代资源之一，我发现了一个非常有趣的应用:单手浏览器...其想要解决在不使用另一只手的情况下使用不那么小的设备的问题。

而且，它很好地解决了问题！

用户界面非常简单，不需要花太多时间去学习如何使用。

[![](../Images/adf20a191bc280a009cfb3783864c4f1.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--1dV4eG5g--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/gfictftu6g7aatdd4dev.jpg)

点击一个按钮(“手”按钮),你的拇指就可以到达页面的顶部(这对有顶栏的网站非常有用)。当页面滑动超过顶部限制时，也会发生同样的情况。

[![](../Images/a669a199e435a6eeca951c2040ab84b8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--bvjH4ILE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4re5dhugjnttdkwo0wne.jpg)

它也有夜间模式(我的眼睛非常感谢！).

另一件我发现非常有用的事情是，当你长按“搜索”图标时(它实际上也是地址栏激活器)，你可以选择复制和粘贴网址。在 Chrome 中，为了选择一个 url，我总是必须把光标放在那里，长按，然后按下“全选”选项，因为 Android 选择工具把选择放在 url 的末尾之前。
