# WooCommerce 直接结账插件

> 原文:[https://dev . to/bobhert _ bob/woo commerce-direct-check out-plugin-2 ap4](https://dev.to/bobhert_bob/woocommerce-direct-checkout-plugin-2ap4)

[![](../Images/a5aa998bfabadde7dd9ca4b0db66a697.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--VP4v7gee--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/wy4ko4k7epo2f6zbggcq.png) 
WooCommerce 直接结账插件让电子商务网站所有者通过从 3 个选项中进行选择来为客户快速结账。选项有粘性结帐，一个页面，一次点击，你可以每次应用任何一个结帐选项。**[woo commerce direct check out](https://codecanyon.net/item/woocommerce-direct-checkout-skip-cart-or-checkout-on-same-page/21456470)**插件启用粘性结账，当顾客点击购物车按钮时，该插件在滑块中显示结账表单。一旦用户点击“立即购买”按钮，一键结账通过跳过购物车页面将用户重定向到结账页面。“单页结帐”会在客户单击“添加到购物车”按钮后，在同一页面上显示结帐字段。
