## C#和 JAVA，两种最流行的编程语言

Sun 是引入 Java 的公司。可以理解的是，当谈到 Java 开发服务时，它想要控制设计和发展。然而，微软也想分一杯羹，特别是想以某种方式改变 Java 以适应它的需求。这成了一场法律噩梦，最终微软决定不再试图影响 Java，而是简单地设计自己的语言，即 C#。它在某些方面比 Java 更好，但在大多数方面也非常相似，一旦开发人员学会了如何用 C#编程，将知识转移到 Java 将是微不足道的。

### C#与 JAVA，主要区别

1.  Java 编程语言被设计成在 JRE 或 [Java 运行时环境](https://docs.oracle.com/goldengate/1212/gg-winux/GDRAD/java.htm)的帮助下在 Java 平台上运行。一方面，C#语言被设计为在 CLR 或公共语言运行库上运行。
2.  在 Java 中，通过值传递的内置数据类型称为基本类型。在 C#语言中，通过值传递的内置数据类型称为简单类型。
3.  Java 类型安全是安全的，但 C#安全不是。
4.  Java 不支持条件编译。另一方面，C#支持带有预处理器指令的条件编译。
5.  Java 中的数组是对象的直接特化。但是 C#中的数组是系统专门化的。
6.  Java 不支持结构和联合，但是 C#支持联合和结构。
7.  Java 不支持 go to 语句，但 C#支持。
8.  Java 支持检查异常和未检查异常。C#支持未检查的异常。

#### 主要 JAVA 项目

1.  **脸书**。最流行的社交媒体也是由 Java 驱动的。脸书使用 HBase，这是一个基于 Java 的非 SQL 数据库作为消息平台，并在其他领域使用 Java。
2.  **易贝**。这是一家总部位于加利福尼亚州圣何塞的跨国电子商务公司，通过其网站促进消费者对消费者和企业对消费者的销售。易贝的架构是基于 Java 的。
3.  **谷歌**。它在 Google Plus、高可扩展性、GTalk 等方面大量使用 Java。
4.  **亚马逊**。除了其他语言，亚马逊也使用 Java。
5.  **LinkedIn** 。LinkedIn 是一个面向商业和就业的服务，通过移动应用程序和网站运营。它主要是用 Java 编写的。

#### 重大 C#项目

1.  **女妖**。它是一个跨平台的开源媒体播放器。在 Ubuntu 和 Linux Mint 的一段时间里，它是默认的音乐播放器，但后来在两个发行版中都被 Rhythmbox 取代了。
2.  **Visual Studio** 。微软的 [Visual Studio 是微软的一个集成开发环境。Visual Studio 包括一个支持智能感知和代码重构的代码编辑器。](https://visualstudio.microsoft.com/)
3.  **Paint.net**。这是一个免费的微软视窗光栅图形编辑器程序。它是在[上发展起来的。NET 框架](https://www.c-sharpcorner.com/article/what-is-new-in-net-framework-4-7-1/)。
4.  **科勒狄加**。一套使用开放标准管理统计调查和官方统计的程序，使档案管理员、研究人员。目前，各种大学调查研究小组，如纵向研究、国家统计局、商业调查研究组织和数据档案馆正在使用 Colectica。

有了这么多的共同点，最终使用的语言在很大程度上取决于为项目选择的平台。现在，C#主要用于。NET 框架，CLI 和 Mono 的 Portable.NET 实现。如果软件或 web 应用程序是为 Windows 构建的，那么 C#最适合与。NET 技术套件。也就是说，如果想为 Linux、Unix 或微软之外的其他平台开发，Java 这个大的开源生态系统是更好的选择。社区不断创造新的工具和库。

Java 编程语言和 C#编程语言都不会很快消失。这两种编程语言都已经存在了一段时间，事实上，没有什么是不能在一种语言中构建的，而在另一种语言中不能实现的。底线是，所有的选择都归结为最适合项目平台选择的语言。
