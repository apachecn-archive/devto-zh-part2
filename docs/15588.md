# 大公司的黑客马拉松项目:Foursquare，Paypal，Banco Sabadell

> 原文:[https://dev . to/Apium _ hub/hackathon-projects-by-big-companies-four square-paypal-Banco-Saba Dell-4k4k4k](https://dev.to/apium_hub/hackathon-projects-by-big-companies-foursquare-paypal-banco-sabadell-4k4k)

你听说过黑客马拉松吗？在[软件开发世界](/)中，这是一个时髦词。

[黑客马拉松](https://apiumhub.com/tech-blog-barcelona/what-is-a-hackathon/)是一种类似短跑的活动，开发者和其他参与软件开发的人，包括设计师、项目经理和其他人，在几天内，通常在一天到一周之间，就创新项目进行密集协作。在大多数情况下，组织黑客马拉松是为了创造新的东西，或者解决特定的问题。今天，我想向大家展示一些由全球领先公司创建的有趣的黑客马拉松项目。

## [](#hackathons-gaining-popularity-as-innovation-vehicle)黑客马拉松作为创新工具越来越受欢迎

老实说，黑客马拉松如今已经成为一种必须，公司从创新鼓励和人才发展的角度考虑。

黑客马拉松也作为正常的内部业务活动被采用，让自己的员工参与进来，以促进超越传统业务和 IT 转型的创新和创造性问题解决。

黑客马拉松项目可以在短时间内激发对一个重点项目的协同开发努力，并且可以增加对关键问题的关注。对于在业务中需要动力和希望的人来说，黑客马拉松项目可以创造这种可能性。它还可以帮助找到意想不到的内部资源，以促进数字创新。

近年来，创新的本质已经发生了变化，今天的特点是速度、灵活性和开放性。当我们谈到大公司时，与不同部门的合作，尤其是与 IT 部门的合作，可能是一个问题。一个快速的解决方案就像一个梦想，永远不会实现。但是，根据[快速公司](https://www.fastcompany.com/3030628/why-do-big-companies-do-hackathons)的说法，随着企业开始接受软件优先的未来理念，他们开始在从研发和产品路线图到人才保留的所有事情上采用黑客马拉松。对于财富 500 强来说，这不仅仅是一个事件，而是一项变革举措。

例如，在过去几年中， [**Capital One**](https://www.capitalone.com/) 已经为各种内部创新举办了内部黑客马拉松，例如构建原型、向高级业务和技术领导者推销想法、创造新产品、引入新功能以及采用新解决方案。这是一种很好的方式，有助于营造现代公司所需的创客文化。

再比如 [**孩之宝**](https://www.hasbro.com/es-es) ，几年前举办了一次黑客马拉松，150 名开发者来了，开发了 45 款产品，相当于传统 R & D ( *来源: [Fast company](https://www.fastcompany.com/3030628/why-do-big-companies-do-hackathons)* )的数十亿美元。

至于传统汽车行业，据黑客马拉松组织者称，他们正在使用黑客马拉松作为创建车载应用和安全功能的一种手段。 [**雪佛兰**](https://www.chevrolet.es/) 提供了与 AT & T 的裸眼 4G LTE 集成数据，允许黑客为车辆安全创建应用程序，而福特已经试验了社交驾驶应用的黑客马拉松。Edmunds.com 的[](https://www.edmunds.com/)**已经与[财富 500 强公司](http://fortune.com/fortune500/)合作举办了名为 Hackamotive 的受邀才能参加的黑客马拉松，许多人可以利用其庞大的汽车数据来破解购车体验，**

 **我应该说，这不仅仅是技术的问题。它是关于你如何实施它，你如何作为一个团队工作，你如何解决问题，你如何在不断变化的环境中工作，等等。

## [](#why-do-big-companies-invest-in-hackathon-projects)为什么大公司会投资黑客马拉松项目

**1。从不同的角度看问题**

你实际上是边做边学，通过观察别人如何解决问题来学习。一个问题总是有不止一个解决方案，这是一个很好的机会来看看解决同一个问题的不同方法。

**2。成就感**

团队有多经常开始一个项目却从未完成？或者想了一些东西，却从来没有尝试过？一个团队，尤其是大公司的团队，会对那些“永无止境”的任务感到疲惫。有时候，人们需要做一些能立竿见影的事情。它有助于提高积极性和主动性。

**3。新的人才，新的团队**

对于外向和内向的人来说，黑客马拉松是一个培养关系和提高软技能的绝佳场所，同时还可以从事他们共同感兴趣的事情。

**4。创新**

黑客马拉松项目是新想法的温床。他们创造了一种独特的生产氛围，迫使参与者将他们有远见的概念提炼为可行的解决方案。

**5。协作环境**

黑客马拉松在开发者和营销人员之间架起了一座桥梁。黑客马拉松项目有助于在技术开发者和使用者之间建立双向对话。这种反馈循环提高了效率和生产力，增加了动力，并使两个团队更好地相互理解。

**6。软技能提升**

一个好的职业，公认的不是他的硬技能，而是硬技能和[软技能的结合](https://dev.to/apium_hub/importance-of-soft-skills-for-software-architects-b26)。黑客马拉松环境绝对是培养这些技能的最佳场所。

## [](#hackathon-projects-by-big-international-companies)国际大公司的黑客马拉松项目

Paypal

[Fast company](https://www.fastcompany.com/3029885/why-you-should-probably-host-a-hackathon)称，Paypal[在全球举办黑客马拉松。该公司在全球 14 个城市举办了 Battle Hacks，号召最有才华的开发人员 24 小时工作，创造出一些神奇的东西，这些东西融入了 PayPal API，并解决了当地的一个问题。例如，这些黑客马拉松的顶级项目之一是一个帮助收容所小狗的应用程序，将用户与可供收养的狗联系起来，并允许捐赠者使用 PayPal 为疫苗接种和兽医治疗捐款。](https://www.paypal.com/es/home)

four square

[four square](https://foursquare.com/)特别举办了大型的全球黑客马拉松，来自世界各地 30 多个站点的 500 多名开发者竞相使用 Foursquare API 创建应用程序。据 Techcrunch 报道，其中一个顶级项目是“计划你的下一次旅行”，它使用 Foursquare API 来帮助你计划你即将到来的为期 2 天的世界任何城市之旅，使用 Foursquare 的“探索”功能的建议。

**Banco Sabadell**

几年前，Banco Sabadell 开始组织即时银行黑客日、 [Banco Sabadell 的黑客马拉松](http://prensa.bancsabadell.com/en/News/2014/09/banco-sabadell-is-organising-the-first-hackathon-to-boost-digital-innovation-in-the-financial-sector-with-the-collaboration-of-mobile-world-capital-barcelona)，与 Mobile World Capital Barcelona 合作，推动金融领域的数字创新。他们聚集了大量的程序员来产生新的想法和应用。即时银行黑客日的参与者致力于创建新的网络和移动应用程序，以在四个方面增强用户的直接体验:支付手段、客户服务转型、物联网以及上下文和预测服务。在 30 多个小时里，23 个团队在 Banco Sabadell 的黑客马拉松上开发了金融科技应用和软件。它发生在巴塞罗那，而 Sabadell 银行已经成为西班牙对科技最友好的银行之一。它不仅为开发人员提供了访问 API 的机会，以便在此基础上构建应用程序，而且还为金融科技项目运行 BStartup 10 启动程序和黑客马拉松。即时银行黑客日的获胜者是:

*   Wallabe :这是一个社会银行的概念，允许 Sabadell 银行成为共享经济的一部分。该应用程序允许用户使用智能手机随时随地取款。
*   [Quipu](https://getquipu.com/en) :面向创业公司和自由职业者的在线开票软件。该公司在活动之前已经存在，但该团队在那里开发了 Quipu Ripae，这是一个与 Sabadell 的 API 集成的扩展。
*   [Investcity](http://www.investcity.com/) :该项目试图在金融产品和游戏之间建立联系，让用户更好地了解金融行业的运作方式，并在过程中应用游戏化来买卖股票。

另一个黑客马拉松是与 Vitalik Buterin 一起举办的——以太坊区块链背后的年轻天才是由 Banco Sabadell 筹备的区块链黑客马拉松的明星，来自全球各地的开发人员都参加了这次活动。

作为 4YFN 的一部分，Banco Sabadell 举办了黑客马拉松，致力于将智能合约中的商业规则转化为[区块链](https://apiumhub.com/tech-blog-barcelona/blockchain-technology/)，这是一种革命性的新技术，允许数据以更简单、更安全的方式传输。来自世界各地的黑客和开发人员在以太坊的年轻创始人、科技界最有影响力的人物之一维塔利克·布特林(Vitalik Buterin)的指导下，在团队中工作。

其背后的想法很简单，Banco Sabadell 寻求在金融科技、保险科技、法律科技和清洁科技领域的科技初创企业和企业之间建立共同创造的协同效应。参加这次黑客马拉松的公司包括微软、苏黎世、Roca Junyent、Endesa 和合资企业 Banco Sabadell/Zurich。
