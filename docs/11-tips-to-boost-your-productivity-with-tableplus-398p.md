# 使用 TablePlus 提高工作效率的 11 个技巧

> 原文：<https://dev.to/tungtono/11-tips-to-boost-your-productivity-with-tableplus-398p>

TablePlus 是一个现代化的本地客户端，具有直观的 GUI 工具来创建、访问、查询和编辑多个关系数据库:MySQL、PostgreSQL、SQLite、Microsoft SQL Server、Amazon Redshift、MariaDB、CockroachDB、Vertica 和 Redis。

从[官网](https://tableplus.com)下载 app。

#### 1。导入 URL

在创建到数据库的连接时，如果记住用户名和密码对你来说很麻烦，那么试试 TablePlus 的这个特性。选择驱动程序，然后复制并粘贴要导入的数据库 URL。搞定了。

[![Import URL](../Images/147e09ed0a31fc6d4fec719e82afbecc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--lgMUuuip--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vgqxhj5gyacdccpi68ci.gif)

#### 2。打开任何东西

你可以期待这个功能在 Mac 上作为 Search Spotlight 运行。TablePlus 将搜索数据库、表、视图、快照和函数。要从结果列表中打开一个项目，请双击该项目或使用向上箭头和向下箭头键选择该项目，然后按 enter 键。

[![Open Anything](../Images/c147603a5827b61155c8681c3c5c11c6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--OyP84DPF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://tableplus.io/assets/images/11-tips/open-anything.gif)

#### 3。最喜爱的

您可以通过将经常使用的查询添加为收藏夹来快速查看它们，这样您只需单击一下就可以获得它们。您可以在左侧边栏的收藏夹选项卡中看到所有您喜欢的查询。
要将查询添加到收藏夹，从编辑器中选择查询，右键单击然后添加到收藏夹。

[![Favorite](../Images/28a7b737dd373f0f4ab1ed7b2922b8a8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ZubkP9H0--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://tableplus.io/assets/images/11-tips/favorite.png)

#### 4。内嵌编辑

借助类似电子表格的视图，您可以使用内嵌编辑功能轻松修改数据。最常见的方法是双击单元格，编辑完成后按回车键。不要忘记点击 cmd + S 将更改提交给服务器。

[![Inline Edit](../Images/01ebcbf1ee1c9b0ae617f0d5d09f3629.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--qelu3GnQ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://tableplus.io/assets/images/11-tips/edit-inline.gif)

#### 5。过滤器

过滤器可用于缩小表中数据的范围。它将只显示满足您指定的标准的特定行，并在当前视图中隐藏不需要的部分。您可以同时使用多个过滤器。

[![Filter](../Images/6c51c3fdd6c63a828b46bb612468fb1f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s---d2fmS7Z--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://tableplus.io/assets/images/11-tips/filter.gif)

#### 6。显示选定的列

如果数据筛选功能要显示一定数量的行，则该功能要显示表中一定数量的列，而隐藏其余的列。

[![Show selected columns](../Images/5b179269953309cedead3cfbac0f676b.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--JlZ32R5e--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://tableplus.io/assets/images/11-tips/show-column.gif)

#### 7。多选项卡-多窗口

很有可能，您将同时处理大量的连接/数据库/表。因此，这个功能将帮助您更快地导航，而不必在它们之间来回切换。

[![Multi tabs](../Images/a1448f0c7f62710926af8565960daa9c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--pmvPX10c--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://tableplus.io/assets/images/11-tips/multi-tabs.gif)

#### 8。代码审查

怕自己不小心犯错？代码审查会处理好的。Code review 将记录您所做的所有更改，并向您展示这些更改，然后您可以决定是将更改提交到服务器还是将其丢弃。

[![Code Review](../Images/ddd740ce7f8c33b3a428e8e7f9837e7e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--5CT3Lm3H--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://tableplus.io/assets/images/11-tips/code-review.png)

#### 9。拆分标签

对此有两种方法。您可以将查询编辑器分成两个屏幕，分别编写和运行查询，也可以在同一个屏幕上编写多个查询，然后将结果分成不同的选项卡。这可以很容易地比较相邻的结果。

[![Split Tab](../Images/580a98d9ccb55cf32659387fc6422553.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--gO3O0EiF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://tableplus.io/assets/images/11-tips/split-tab.png)

#### 10。快捷键

快捷键节省大量时间。句号。TablePlus 中的每个功能都被分配到一个快捷键，这样你就可以更快地完成任务。点击了解快捷键[的更多信息](https://tableplus.io/blog/2018/02/shortcut-keys.html)

#### 11。黑暗主题

如果你像大多数开发人员一样，使用深色主题将有助于你更加专注，并提高你的工作效率。你可以从应用程序的偏好设置中启用黑暗主题。此功能仅适用于注册帐户。

[![Dark Theme](../Images/3d34ef68294994b959bb581b8c93d070.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--2ZWdskhz--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://tableplus.io/assets/images/11-tips/dark-theme.png)
