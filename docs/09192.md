# 介绍我们 2018 年 7 月的赞助商

> 原文:[https://dev.to/ben/introducing-our-july-2018-sponsors-p9f](https://dev.to/ben/introducing-our-july-2018-sponsors-p9f)

我们自豪地宣布七月份的持续社区赞助商。我们出色的赞助商对 [dev.to](https://dev.to) 的健康发展至关重要，能与对生态系统做出如此大贡献的公司合作真是太棒了。

七月份，我们有了两个新的赞助商和一个持续赞助商。请检查他们的精彩服务。

## 数字海洋

[DigitalOcean](https://try.digitalocean.com/performance/?utm_source=devto&utm_medium=display&utm_campaign=Devto_2018_Brand) 是一个备受喜爱的云计算平台。我总是对数字海洋核心产品的良好构建*和良好记录*印象深刻。说起来容易做起来难，他们已经能够保持领先。

## 流

[Stream](https://getstream.io/try-the-api?utm_source=dev.to&utm_medium=banner&utm_campaign=july) 的 API 是为创建新闻提要和活动流量身定制的。我们应该知道，当我们选择 Stream 来构建 dev.to 通知提要的组件时，说起来容易做起来难。如果您正在实现一个提要，那么让 Stream 来完成繁重的工作。

## 落水狗

[Underdog](https://underdog.io/?utm_medium=homepage&utm_source=devto&utm_campaign=homepage2018-jun-30&utm_content=engineers-tech-companies) 在旧金山和纽约这两个美国主要市场，为优秀的开发者和优秀的科技公司牵线搭桥。如果你正在这些中心之一寻找一个职位，跳过这条线，让超狗成为你的伙伴。
