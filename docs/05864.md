# 天气 App

天气预报 app，纯 javascript 内置，将学习中所学付诸实践。

## 建有

*   是六个
*   java 描述语言
*   网络包。
*   厚颜无耻
*   [openweather](https://openweathermap.org)

### 安装

克隆此存储库

```
git clone https://github.com/mariorodeghiero/weather-fetch-api 
```
