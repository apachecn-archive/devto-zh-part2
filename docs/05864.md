# [](#weather-app)天气 App

天气预报 app，纯 javascript 内置，将学习中所学付诸实践。

## [](#built-with)建有

*   是六个
*   java 描述语言
*   网络包。
*   厚颜无耻
*   [openweather](https://openweathermap.org)

### [](#prerequisites)先决条件

安装软件需要哪些东西以及如何安装

*   节点. js
*   NPM 包装经理
*   在 [openweather](https://openweathermap.org) 中创建令牌，并包含在文件“/src/js/token.js”中
