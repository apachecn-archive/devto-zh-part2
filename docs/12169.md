# 面向学习者的 Github repos

> 原文:[https://dev.to/oathkeeper/github-repos-for-learners-634](https://dev.to/oathkeeper/github-repos-for-learners-634)

我想以一些破冰会议开始，为那些想选择计算机科学的人，但他们采取了其他路线，或者他们想进入这个领域，但不知道从哪里开始，我想分享我的发现，而我开始准备。任何一个印度人都不太可能自学，所以我希望这也能影响到其他国家的人，我认为那里的许多人都在自学，他们只需要一个大纲来完成所有的检查。
有这样一个 GitHub 回购

*   开源社会大学 OSSU 分校/计算机科学([https://github.com/ossu/computer-science](https://github.com/ossu/computer-science))，
    *   他们有明确的课程，预计完成时间就像大学一样，但你可以在:D 凌晨 2 点学习
*   p1Xt -他们只是成为软件开发人员的另一个指南，或者让我们说你想从后端开发开始，他们有一个精选的事情清单，肯定会帮助你获得许多免费资源 p1Xt/p1Xt-guides([【https://github.com/P1xt/p1xt-guides】](https://github.com/P1xt/p1xt-guides)
    *   只需在链接上查看他们的指南，它已经统计了许多路径。
*   自学计算机科学-是的，我知道，这不是 GitHub repo，你可能会说，它在这里做什么。我必须说这是值得分享的。自学计算机科学([)https://teachyourselfcs.com/](https://teachyourselfcs.com/)
    *   很少有人开始这个项目来自学计算机科学，他们有献身精神、毅力、意志去工作，但是却灰心丧气，或者我应该说脱轨了，因为他们不知道大纲，这个资源为我们解决了这个问题
    *   这是整个计算机科学的主要课题
        *   算法和 DS
        *   操作系统
        *   数据库管理系统
        *   计算机网络
        *   密码系统
        *   语言和编译器(在印度，每个大学都教得很差，或者人们甚至不知道这可能是一门学科)
    *   如果你找不到列出的书，不要担心，只要谷歌搜索或在 reddit 上搜索就行了，肯定有人已经把书上传到他们的谷歌驱动器上了。:p
*   学习任何东西([)https://learn-anything.xyz/programming](https://learn-anything.xyz/programming))这个网站很棒，它展示了学习的图解路径，里面有链接的资源，我分享了关于编程的链接，但你也可以搜索任何其他主题。
    *   你会再次说，Divyesh(未定义)，我现在要取消关注你，你已经无法分享 GitHub repos 了。但是沉住气，去那个网站打开所有链接，那些是 GitHub repos 上的精选资源列表。你以后一定要感谢我。哈哈哈
*   任何人都想从贡献开源项目开始，但到目前为止，你只练习了你的数据结构和算法技能在线法官，如 HackerRank，CodeChef，CodeForces，LeetCode。别担心，独角兽来了，它能让你在战场上跑得飞快，还能让你想飞多高就飞多高。
    *   看看这个 Karan 的大型项目列表——Karan/Projects([https://github.com/karan/Projects](https://github.com/karan/Projects))bambozed 当你可以实现 DS &算法和许多其他基本算法时，你也可以练习，以使你的 GitHub 简介丰富多彩
*   计算机科学基础的基于项目的学习包括许多语言——TUV tran/基于项目的学习([【https://github.com/tuvtran/project-based-learning】](https://github.com/tuvtran/project-based-learning))虽然这已经在本答案的 Learn-anything.xyz 部分中有所涉及，但我想在`this` `scope`中再次明确地定义它(我知道我在学习 EcmaScript 上有点吃力)
*   BMO relli 25/Become-A-Full-Stack-Web-Developer([)https://github . com/BMO relli 25/Become-A-Full-Stack-Web-Developer](https://github.com/bmorelli25/Become-A-Full-Stack-Web-Developer))对于 Web 开发和开始使用 JavaScript，已经有大量的资源，但这是一个经过整理的教程和项目列表，可以帮助你通过 codeburst([http://codeburst . io](http://CodeBurst.io))学习完整的 JS
*   牛逼 Github sindresorhus/牛逼([https://github.com/sindresorhus/awesome](https://github.com/sindresorhus/awesome))——没话说就是牛逼，自己看看吧。包含关于所有平台、语言、框架等的精彩内容。
*   coding Interview University-jwasham/coding-Interview-University([https://github.com/jwasham/coding-interview-university](https://github.com/jwasham/coding-interview-university))确实如其所说，这是一条为高端谷歌、亚马逊等做准备的精心策划的路径。公司面试，主要关注数据结构和算法
*   我必须分享一些与 React 相关的东西——因为这是某人英语播放列表中唯一的 Despacito 西班牙语歌曲——enaqx/awesome-React([https://github.com/enaqx/awesome-react](https://github.com/enaqx/awesome-react))
