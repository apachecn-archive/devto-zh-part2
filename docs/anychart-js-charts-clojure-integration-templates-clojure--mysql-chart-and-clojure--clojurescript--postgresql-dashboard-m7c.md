# AnyChart JS 图表 Clojure 集成模板:Clojure + MySQL(图表)，和 clo jure+ClojureScript+PostgreSQL(仪表板)

> 原文：<https://dev.to/andreykh1985/anychart-js-charts-clojure-integration-templates-clojure--mysql-chart-and-clojure--clojurescript--postgresql-dashboard-m7c>

交互式数据可视化的全球领导者之一 AnyChart 正式展示了更新的 [Clojure 集成模板](https://www.anychart.com/technical-integrations/samples/?filter=clojure)，展示了如何使用他们的[获奖](https://www.anychart.com/blog/2018/02/06/best-javascript-technology-anychart-js-charts-library-devies-award/) [JavaScript 图表库](https://www.anychart.com)和这种编程语言以及 MySQL 或 PostgreSQL 数据库来创建酷的交互式 dataviz。

演示了两个集成:

*   Clojure + MySQL(图表)，以及
*   clo jure+ClojureScript+PostgreSQL(仪表板)。

在 [AnyChart Integrations](https://www.anychart.com/technical-integrations/samples/?filter=clojure) 页面上查找它们。
