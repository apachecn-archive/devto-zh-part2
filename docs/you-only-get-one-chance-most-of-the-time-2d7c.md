# (大多数情况下)你只有一次机会

> 原文：<https://dev.to/threedeeprinter/you-only-get-one-chance-most-of-the-time-2d7c>

[![Dog Struggling on Leash](img/37a9e026cacc95d691fce63639cab118.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--AoJoaF2J--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/zug8rsuxos3h5exo3h81.png)

你渴望去面试那份新工作。你偶然发现了一个梦寐以求的职位，在你一直想为之工作的团队中起步。除此之外，其中一个要求是对框架的经验，你可能只是一知半解，他们需要一个自信的人。

你也许能找到出路。你也许能够掌握足够框架知识来进行面对面的面试。

停下来。转过去。给我回来。

我知道想要一个梦寐以求的机会有多难，以至于害怕失去它会驱使你去参加一个你还没有准备好的面试。

即使你可能会错过你梦寐以求的好职位，你也不想让雇主知道你是谁，直到你完全准备好接受面试。你永远不知道那家公司什么时候会有另一个职位空缺，你也不希望他们记得你是那个试图在面试中蒙混过关的人。

我提出这个问题是因为我最近不得不放弃独角兽。那是一家从事分布式云存储的公司，这是我非常感兴趣的东西。他们想要一个非常了解 VueJS 的人，这样我就可以用它来设计一个解决方案。我只把它作为 UI 插件专业使用过。我仍然在学习课程和建立实践项目。我还在用例子填充我的 Github。我还是有点嫩。所以我把它传了过去，关掉了我链接的就绪灯。也许几个月后，我会有一些值得骄傲的东西给未来的雇主看。同时，我会享受学习的乐趣。我也知道我没有搞砸一家公司，也许有一天我会得到第二次机会。
