# 网络发展的现状

> 原文：<https://dev.to/flippedcoding/the-state-of-web-development-52gk>

网络开发的压力水平低于平均水平，工作生活的平衡程度远远高于平均水平。更不用说预计的就业增长了。[【1】](https://money.usnews.com/careers/best-jobs/web-developer)

如果你甚至短暂地考虑过进入科技行业，网络开发是一个很好的方式，让你的脚进入大门，并开始攀登你想去的高度。

我知道这听起来好得不像是真的，所以我要给你一个关于网络发展状况的快速分析。

*   乔布斯

    *   这通常会吸引人们。web 开发人员的就业市场正在蓬勃发展，预计至少在未来十年内会继续繁荣。在这个低迷的经济中，这是少数几个增长高于平均水平的领域之一。这是因为人们每天越来越多地转向移动设备和电子商务。
    *   我敢肯定，你正试图找出作为一名网络开发人员你能期望赚多少钱，所以我会告诉你。首先，这真的取决于你的位置和你的经验水平。你可以从 47，000 美元到 91，000 美元不等，但平均工资在 67，000 美元左右，这还不错！[【3】](https://money.usnews.com/careers/best-jobs/web-developer/salary)
    *   尽管网络开发处于科技行业工资范围的低端，但它在未来 10 年仍有 15%的增长率。现在工作机会比开发人员多，这导致了大量的外包。
    *   外包网络开发工作在美国已经成为热门话题有一段时间了。有些公司喜欢它，有些讨厌它。最近，由于跨时区和不同文化期望的沟通困难，有一种推动在国家招聘方面做得更多的趋势。(对不起海外的伙计们…)
    *   有一些州对程序员来说机会更好。加州当然是第一名。然后是纽约、德克萨斯、佛罗里达和华盛顿。现在，美国没有一个州不雇佣网络开发人员。你几乎可以选择任何你想去的地方。看看上面列出的所有工作，提醒自己这还不是全部。
    *   有一件事一直让人感到困惑，那就是 web 开发人员和软件开发人员之间的区别。网络开发人员是为网站写代码的人。他们可能关心网站加载的速度、网站的安全性以及网站的外观。
    *   另一方面，软件开发人员编写不同种类的代码。他们通常关心内存分配，有效的算法，在某些情况下，软件如何与硬件一起工作。这是一门完全不同的学科，但是人们总是把它们混淆在一起。了解这种差异很重要，这样你才能知道你的工作期望是什么。
*   技术

    *   Web 框架正在占据主导地位。有角，反应，Vue，电子，还有很多其他的。您会发现成千上万的开发人员使用这些框架开发 web 应用程序。现在，Angular 和 React 是最受欢迎的两个，但总有机会一个新来的人可以改变一些事情。
    *   您需要尽可能深入理解的一种语言是 JavaScript。它最初是一种前端语言，但作为服务器端语言也越来越受欢迎。Node.js 之类的东西已经成为开发的首选工具，因为它可以让您轻松启动本地服务器。
    *   CSS 也在经历一些变化。如果你不熟悉 CSS Grid，花点时间学习它是如何工作的是个好主意。人们也比以往更多地使用动画来使他们的网站更具互动性。很好地理解 CSS 动画是如何工作的很重要。如果你能用 CSS 代替 JavaScript 制作动画，你可以节省一些加载时间。[【6】](https://www.infoq.com/articles/web-development-trends)
    *   聊天机器人也在向前发展。人们已经厌倦了糟糕的客户服务，所以公司正试图实施聊天机器人来帮助解决这一问题。(使用真人可能会更有效，但我知道什么……)学习如何让他们以实际帮助客户的方式工作是一项有价值的技能，现在是学习它的时候了。[【7】](https://insanelab.com/blog/web-development/web-development-trends-2018/)
    *   不要忘记加速移动页面！移动设备产生了超过一半的在线流量，这意味着互联网上的所有网站都需要对移动设备友好。不是所有的网站都会如此，但大多数新网站都会涉及到移动优先设计的某些方面。还有一股力量推动编写适用于所有设备的代码。拥有一个单独的移动样式表不再是不寻常的事情。
*   社会的

    *   这是有趣的部分。web 开发者世界在社交方面仍然落后。女性在这一领域获得了很大的发展，占新初级开发人员的 53%。尽管你很难在这些职位上找到少数族裔。截至 2017 年，只有 4.5%的网页开发者是非裔，5.8%是拉美裔，9.8%是亚裔。[【10】](https://www.bls.gov/cps/cpsaat11.htm)你也很难找到年长的开发人员。行业内的开发者大多在 20 - 29 之间。30 岁以后，分布急剧减少。[【11】](https://www.businessinsider.com/silicon-valley-age-programmer-2015-4)
    *   此外，还有开发者对技术堆栈的问题。没有人能比 Reddit 线程更好地解释或演示这一点:[【12】](https://www.reddit.com/r/webdev/comments/8orzqx/does_anyone_feel_like_the_state_of_web/)随着所有的变化、新发布、版本和其他一切，很难跟踪什么是使用的最佳实践。当一个库或框架的更新使得它与你写的所有代码不兼容时，这可能是每个 web 开发人员最大的挫折。

这只是 web 开发中正在发生的事情以及您可以期待的事情的一个简要概述。如果这看起来很难理解，请记住这只是表面现象。web 开发世界的底层发生了如此多的事情，以至于我们这些身在其中的人都跟不上。
