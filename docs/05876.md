# 我在 90 年代早期在海军准将 Amiga 上用汇编语言编码，问我任何问题！

准将阿米加可以被视为传奇准将 64 (C64)的继任者。这是一台家用电脑，图像和声音非常好，当时普通的个人电脑无法与之竞争。这使得它成为了一个令人敬畏的游戏机。

但除此之外，它很容易理解系统架构，所以很多孩子用汇编语言在上面编程。我是其中之一，问我任何事情！
