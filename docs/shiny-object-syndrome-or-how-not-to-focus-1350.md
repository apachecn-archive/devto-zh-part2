当我考虑我作为一个开发者的身份时，我喜欢首先把自己看作一个业余爱好者，然后才是专业人士。

这种心态的根源是我编写应用程序是因为我喜欢这样做的过程，而不仅仅是因为这是我的工作。我喜欢看到我的解决方案走到一起；漂亮、对称的风格和漂亮的模块化代码。我喜欢以高效的方式将一箱箱的东西装进稍大的箱子里。我喜欢这样做，也喜欢开发应用程序。

当我发现自己被一些新技术、编程语言或库所吸引时，问题就出现了。如果你看了我的 [Twitter](https://twitter.com/iwilsonq) ，你会发现我正在参加#100DaysOfCode 挑战。如果你浏览了我过去的推文(到目前为止已经 57 天了！)你会发现我已经使用了相当多的不同技术。

我花了一些时间与 Golang，React Native，TDD/BDD 与 Jest，GatsbyJS，Rust，VueJS，以太坊区块链应用程序，虚幻游戏开发，UI 设计与草图，以及最近的 GraphQL。我处理过一些不同的项目，但最终都失败了，有利于吸引我眼球的其他项目。我在下班后或周末探索这些其他途径，也许是为了摆脱我每天都在使用的更加例行公事的 JavaScript。大量编码！

## 兔子洞

我从来不喜欢递归。当面试官告诉我计算斐波那契数列或者某个数字的阶乘时，我吐出了我已经记住的答案，并说让我们开始吧。我对递归的问题是，当调用栈变得太大时，当递归函数调用自己太多次，程序放弃时会发生什么。想知道[栈溢出](https://stackoverflow.com/)是如何得名的吗？也许这太明显了。

我大部分的空闲时间都花在了研究一个递归上，这个递归和我描述的没有太大的不同。考虑下面的代码。

```
let golangLearned = false

function learnGolang() {
  if (!golangLearned) {
    learnGolang()
  }
  // that is all... nothing else happens in this function
}

learnGolang() 
```

你能看到这个程序的结果吗？请注意，没有一个该死的东西可以翻转布尔运算，将我从这种折磨中释放出来。时光飞逝，星巴克的打烊时间到了。我收拾好自己的东西，困惑而不知所措。在过去的几个小时里，我什么也没做。

我开车回家，泡些茶，吃块饼干，上网消费内容。我看到有人发了一条关于玩虚幻游戏引擎，用 C++写游戏脚本的推文，我就像 *wow* 一样，这听起来比我的夜晚好多了！我上 Udemy，选了一门课程跟着学，这实际上是一股新鲜空气。我试了几天，我们建造的东西真的很棒！

但是蜜月结束了——这个新爱好变成了一场斗争。现在，它并没有因为它天生无聊或困难而结束，而是因为我了解到我现在可以在以太坊区块链上构建分布式应用程序！我是说，拜托，我什么时候才能再有这样的机会？

快进一点:反应自然。

再快进一些:GraphQL。

最初是一种编程语言专有的递归兔子洞已经暴露出它是一个更大的问题，我不知道我想做什么。

艾伦·沃茨说，你不知道自己想要什么有两个原因。

1.  你拿到了。
2.  你不认识你自己，这实际上是不可能的，因为神性从来不是它自己知识的对象。

## 不是陌生的敌人

当我在中学的时候，我暑假的大部分时间都在玩魔兽世界。我很投入，浪漫地想象着和我的公会一起挑战突袭 bosses 我的虚拟朋友。我想得到带着漂浮水晶的闪亮装备，坐在伟大城市暴风城和 AFK 的拍卖行附近，和贸易聊天频道上的所有戏谑一起，这个频道覆盖了世界上所有主要城市的特定派别...魔兽争霸。

但我从未去过那里。尽管我花了大量的时间去磨野猪和运行死亡矿井，尽管我知道几乎每一点点的知识，我只玩了一小会儿残局内容。因此，我必须坦白:我的名字叫伊恩，我是一个*嗜酒者*。

> alto holic——用于指 MMORPG 的(大型多人在线角色扮演游戏),如《魔兽世界》或
> Everquest，指的是创造许多不同角色的人，而不是一两个。也可以用来指
> 在同一个游戏的许多不同服务器上创建角色的玩家，在那里他们看到的体验通常会互不相同
> 。

当我玩够了的时候，魔兽世界已经有 10 个职业了。十！我能做什么，扮演一个角色作为一个不虔诚的时间，然后意识到我应该是一个德鲁伊？说真的，如果我更喜欢那堂课，我会更开心吗？不要让我开始谈论派别、种族和职业。

让一个角色达到等级上限，或者至少非常接近，然后创建一个新的，再做一次。虽然这从来不是我的目标，但事情总是这样发展。

## 两颗星星向右

你还记得[彼得潘](https://en.wikipedia.org/wiki/Peter_Pan)吗？他就是那个长得有点像《塞尔达传说》里的林克的男孩；他带着一把匕首，还有一个烦人的仙女。然而，彼得潘的故事并非始于迪斯尼。他的名字部分来源于希腊的潘神，他代表了人类最自然的状态；一个不受成年禁忌束缚的孩子。

潘代表了一个潜力无限的孩子。他或她可以成长为任何人或做任何事。成年人会说，“哇看看那个孩子！不知道他会做什么大事？”对彼得潘来说，这就是故事的结尾。

永远，他花他的天打击海盗，与小精灵飞来飞去，和印第安人一起抽和平烟斗。

他永远不会长大去朝九晚五地工作，结束伦敦的无家可归，或者在 DMV 排队等候(他会飞)。

## 直走到天亮

有时候觉得自己像小飞侠；不断地满足我的幻想，花费我大部分的时间在现代 web 开发的各个方面，想知道我的周末去了哪里，而我在追查我到底是如何用 React Native 设置 Enzyme 的。如果你对此有任何建议，请。

虽然我一生都被这种注意力不集中所困扰，但它以不同的方式表现出来。在魔兽世界里，我确实玩得很开心，我一直都在和 noobs 一起练级。当谈到学习不同的技术时，我似乎找不到一条可以停留的船。一切都太整洁了。

我什么都想做。我想玩所有的玩具。我想建立一个帝国来屠龙和怪兽。我还想成为一名专业的滑雪板运动员，赢得超级马拉松比赛，成为一名现代游牧民族。

我的一些梦想是相互排斥的。沃伦·巴菲特说[你应该在一个特定的目标上全力以赴](https://jamesclear.com/buffett-slots)。看似合理，但为什么这么难？

通过选择一条道路，我摧毁了其他道路提供的可能性。由于根本没有选择，我停滞不前，并想知道为什么我没有在生活中做更多的事情。这就是选择的悖论，尽管选项很多，我还是忍不住有一种失落感。

我想尝试所有这些不同的途径，这有问题吗？也许问题是我没有一个真实的项目来构建，也没有真正选择使用什么。在这一点上，也许我只是需要一个更宏伟的目标和更系统的规划。

但是我跑题了；我可以逛一整夜，这将只是另一次兔子洞之旅，在那里我会醒来，想知道，我昨晚到底做了什么？如果你患有闪亮物体综合症，[这篇文章可能会帮助你](https://www.entrepreneur.com/article/296203)。

巩固你的焦点。
