# 我是如何开始蔻驰和导师的

> 原文：<https://dev.to/recursivefaults/how-i-coach-and-mentor-5e7j>

在某些时候，会有人来寻求建议或帮助。在 Dev.to，他们实际上以几种方式实现了这一功能。最近，他们开始了一项试验，让人们结成指导关系。

我自己也报名当导师了。我选择报名有几个原因。首先，我最近结束了与一个人为期两年的指导关系，这个人从看到他们的第一行代码发展到获得了一份中级开发人员的工作。我想知道更多关于怎样才能帮助人们取得如此大的成就。我也真的很喜欢这些经历，人们朝着有意义的目标努力，我可以和他们在一起。最后，随着我努力成为一名完全独立的顾问，这让我对如何与他人合作以及我能为他人提供什么并获得报酬有了宝贵的见解。我的第一笔佣金是一瓶波旁酒，虽然很棒，但这并不意味着健康的职业生涯。

所以，有人求助了。采取什么步骤来确保健康和富有成效的指导关系？

## 1。设计关系

没错，清单上的第一步是有意设计双方的关系。跳过这一步会让彼此都感受到他们能说或不能说的东西。这会让彼此怀疑他们是如何解释情况的。因为怀疑和不确定性，它可以拖延强有力的对话。

通过有意识地设计关系，我们把我们想要的品质放在我们的关系中。现在，我们对彼此的身份有了明确的期望，这些不确定性会很快消失。

我是这样做的。我很简单地说，我想花几分钟时间有意识地决定我们在我们的关系中想要什么样的品质，以便我们能够最大限度地利用我们在一起的时间。然后，我提供了两个品质来开始行动:诚实和保密。诚实，因为没有好的理由让我们互相误导。此外，诚实为我们中的一个人达不到另一个人的标准，我们需要谈论它的那一天奠定了基础。我提供的保密性是，我不会与任何人讨论我们谈论的内容，除非他们设定了条款。这种安全提供打开了他们能够更自由探索的领域。如果他们担心他们讨论的内容会出现在某个博客上(咳咳)，他们很可能会更严格地审查自己。在这里，我邀请他们谈谈这些品质对他们意味着什么，并根据他们的喜好加入、改变或删除他们自己的品质。

在确认这些品质足以让我们开始我们的关系后，我们进入第二步。

## 2。目标

目标就像冰山。我们陈述的目标通常是一个更大的、更有成就感的愿景的一部分。所以我和他们一起探索揭示冰山的既定目标。拥有 X 会给你什么？这和你的生活有什么关系？会有什么不同？这些大问题有助于揭示他们对生活的想法到底是什么。这样做再次打开了对话，看看他们的具体目标如何融入他们的生活，以及我们如何将这一部分或其他部分添加到生活的混乱中。关键是这些目标对他们来说很重要，但为什么或如何实现可能并不明显。此外，考虑一下我们一直面临的现实，我们构建了一个没人想要的特性。有时候，人们认为一个目标对他们来说很重要，但这是对他们真正想要什么的一种假设。弄清楚这一点，我们就可以越过代理目标，进入一个令人满意的目标。

## 3。条款

有些物流是最后来的。这是我们谈论谈话的节奏，多长时间，何时以及如何谈话的地方。我们将讨论每次对话的进展情况，并确保我们对进展感到满意。

从这里开始，真正的工作可以开始以一种有意义的方式帮助别人实现他们的目标。建立信任、支持和清晰。
