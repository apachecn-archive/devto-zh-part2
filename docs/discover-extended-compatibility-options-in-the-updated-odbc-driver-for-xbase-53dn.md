# 在更新的 xBase ODBC 驱动程序中发现扩展的兼容性选项

> 原文：<https://dev.to/devartsoftware/discover-extended-compatibility-options-in-the-updated-odbc-driver-for-xbase-53dn>

用于数据库管理和开发的数据库连接解决方案和工具的捷克软件提供商 Devart 发布了用于 Windows、macOS、Linux 的 xBase 的更新 ODBC 驱动程序，包括 32 位和 64 位。采用增强的 ODBC 驱动程序，用户可以在许多流行的 ide、第三方工具(如 MS Access、Tableau、Omnis Studio 等)中顺利地访问所需的数据。，以及各种编程语言。

下面是关于 xBase 的 Devart ODBC 驱动程序的新特性的独家新闻:

除了已经支持的 FoxPro 2、Visual FoxPro 和 dBaseIII-dBase10 之外，该产品的更新版本现在允许用户查看和编辑 Codebase 和 Clipper DBF 文件格式的数据库。

此外，用户现在可以在连接对话框中看到并复制一个使用过的连接字符串，以简化与 xBase 的连接。

仔细看看现在在 xBase 的 ODBC 驱动中可用的一些当前和以前的特性:[https://www.devart.com/odbc/xbase/](https://www.devart.com/odbc/xbase/)。
