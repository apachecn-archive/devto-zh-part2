如今，许多开发人员在终端上花费了大量的时间，以至于它成了开发工作流程中的一个重要工具。

因为我整天都在使用终端，所以我喜欢根据自己的喜好来改变它。

最近，我很高兴地看到有一个叫做 [Hyper](https://hyper.is/) 的[开源项目](https://github.com/zeit/hyper)正在制作一个可黑客攻击的终端。他们仅仅用 HTML、CSS 和 JavaScript 来构建它。

你可以根据自己的喜好改变风格，用主题和插件来增加趣味。对于 mac 终端上 **oh-my-zsh** 的大粉丝来说，你可以在 Hyper 中继续使用这个😎

### 平台

超级终端仅适用于所有平台:

*   [Mac](https://releases.hyper.is/download/mac)
*   [窗户](https://releases.hyper.is/download/win)
*   Debian
*   [软呢帽](https://releases.hyper.is/download/rpm)
*   其他 Linux 发行版(。AppImage) 。

### 设置

您可以在配置文件中更改许多内容。

你想知道如何改变设置？

1.  用您喜欢的编辑器打开~/.hyperterm.js 配置文件。
2.  将主题或插件名称添加到插件数组中。像这样:插件:['插件或主题名称']

### 主题

就像我们使用的所有编辑器一样，有很多主题，Hyper 也有很多主题。在这个列表中，你会发现一些很酷的！

*   **我目前的** [钴 2 主题](https://www.npmjs.com/package/hyperterm-cobalt2-theme):这个主题是由[@维斯博斯](https://twitter.com/wesbos)创作的，也可以用于很多其他应用
*   星球大战主题:不是说我会用它，但是我们是书呆子，所以星球大战是其中的一部分！
*   日晒深色主题:这个主题漂亮干净，也被许多其他编辑使用。
*   Monokai 豪华主题:这个主题就像许多其他编辑器中使用的日光化一样。在我看来，经典中的经典。
*   MacOS 主题:如果你使用的是 Windows 或 Linux，并且喜欢 MacOS 风格的终端，这就是你的主题！
*   超级扁平化 2 :还需要我多说吗？这完全是平面风格
*   干净:这个主题非常干净，颜色也很好看！
*   Atom One Dark :这是 Atom 编辑器中使用的主题之一，如果你喜欢 Atom，这就是你的主题！

Github 上有一个更大的主题集合，在[牛逼的超级知识库](https://github.com/bnb/awesome-hyper#themes)中

### 插件

你也可以在牛逼的 Hyper repo 上找到大量的[插件。但是这里有几个是我最喜欢的！](https://github.com/bnb/awesome-hyper#packages)

*   [Hyperterm-1 密码](https://www.npmjs.com/package/hyperterm-1password):如果您使用 1 密码管理器来存储您的所有凭据，此插件会将其集成到您的终端中。
*   超级终端-历史记录:想要根据你的命令显示历史记录，添加这个令人敬畏的插件！
*   超级终端管理器(Hyperterm-visor):有了这个插件，你可以通过快捷方式从任何 finder 打开你的终端。
