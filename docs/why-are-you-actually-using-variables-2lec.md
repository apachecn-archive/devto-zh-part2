# 你为什么要使用变量呢？

> 原文：<https://dev.to/codevault/why-are-you-actually-using-variables-2lec>

何时使用变量？需要的时候实例化一个变量感觉很自然，但是我们通常不会想到背后的原因。我提出了 4 个最普遍的原因:

*   可读性
*   抽象
*   表演
*   可维护性

我知道这个主题有点敏感，但是，为了让我们能够创建良好的编码标准，我们需要理解其背后的原因。
