# 学习新编程语言的有趣方法

> 原文：<https://dev.to/damcosset/fun-ways-to-learn-a-new-programming-language--14j1>

嘿人们！

最近有一些关于一个人应该学习什么编程语言，对于一个特定的用例什么是最好的，等等的讨论...

我开始学习围棋，我想摆脱“阅读官方文档，做一个待办事项应用”的想法。所以，我想知道你是否用不同的方法学了一门语言。到目前为止，我想到了这些:

*   阅读官方文件和教程

最经典的一个。我们都经历过。不总是有趣的，但大多数时候，一个坚实的选择。

*   开放源码

我看过几篇推荐这个的文章。只需选择一个开源项目，开始阅读代码，选择一个问题，并从那里学习。听起来很有趣，很吸引人，但是对新手来说很复杂。

*   比赛

嗯，我就是这样开始下围棋的。到目前为止，我学习这门语言的目的就是解谜和编码。我有一个 Go 文档来检查我的语法是否正确。我不关心工具或编译或其他任何东西，只关心代码和谜题。目前为止很有趣。

你学习一门新语言或工具的其他有趣和非传统的方法是什么？
