# Linux 时间跟踪软件:团队和个人的解决方案

> 原文:[https://dev . to/acti time/time-tracking-software-for-Linux-solutions-for-teams-and-personal-3ob 3](https://dev.to/actitime/time-tracking-software-for-linux-solutions-for-teams-and-individuals-3ob3)

由于其安全性和可靠性，Linux 发行版被广泛用作服务器操作系统。总的来说，Linux 在桌面上并不流行，自由职业者和办公室职员仍然在他们的个人电脑上使用各种版本的 Linux。

为了控制个人或团队的时间支出和测量生产率，有各种 Linux 兼容的计时解决方案。其中许多是免费和开源的，这是一个优势。基于 Linux 的时间追踪器的不好的一面是它们大多都有问题，并且在安装和配置时需要额外的努力。但我们做了一些研究，发现 Linux 操作系统的时间跟踪软件易于安装和使用。

[**1。actiTIME**](https://www.actitime.com/)
**适用于:团队、个人**
actiTIME 是一个直观且健壮的时间追踪和工作管理系统。它作为一个 web 应用程序运行在云中或公司的内部服务器上。可以从任何能够连接到互联网的设备访问云解决方案。至于自托管选项，它兼容 Windows 和 Linux 平台。

该工具的功能包括记录在特定任务上花费的时间(带注释)、根据收集的数据运行报告、计算可计费金额、开具发票、管理项目以及跟踪员工的缺勤情况。

actiTIME 对于项目经理、会计、人事以及所有需要记录时间的办公室职员和自由职业者来说是一个有用的工具。它帮助任何规模的公司和个体经营者组织他们的工作流程，并更好地控制他们个人的生产力、团队绩效和特定项目的盈利能力。

[**2。仓鼠项目**](https://projecthamster.wordpress.com/about/)
**针对:个人**
仓鼠是一个流行的 Linux 特有的时间跟踪器。它的功能非常简单:它包括跟踪一天中花费在任务上的时间，计算总数，并将时间跟踪数据导出到 HTML 报告中。

为了跟踪仓鼠的时间开销，当从一个任务切换到另一个任务时，改变你在程序中的活动。该计划计数的时间总数，并显示在界面上。

为了简化搜索历史数据和对时间跟踪结果进行分类，请标记您的活动。结果将按标签分割，并显示在统计部分。向活动添加描述有助于跟踪整体进度中工作的特定部分。

[**3。Kimai**](https://www.kimai.org/download/)
**适用于:团队、个人**
Kimai 是一个开源的时间跟踪应用程序，作为一个 web 服务工作。该计划允许配置不同的数据访问级别的普通工人，管理员和客户帐户。

该应用程序提供了基本的时间表和发票功能。用户可以用计时器跟踪他们的时间开销，然后，如果有必要，编辑他们的条目。至于成本和计费功能，Kimai 支持用户费率、项目预算和费用的输入，以及发票导出。

[**4。Rachota**](http://rachota.sourceforge.net/en/index.html)
**适合:个人**
Rachota 是一款适用于 Linux、Windows 和 Mac OS X 的便携式时间追踪软件，它可以在任何地方运行，因为它允许将时间追踪数据保存在 USB 驱动器上。

除了收集时间轨迹数据，该应用程序还会对其进行分析，并就如何更有效地利用时间提出建议。此外，它还具有报告和发票功能。

Rachota 是一个开源解决方案，所以完全免费。与许多其他应用程序不同，它被本地化为 10 种语言，这为那些不精通英语的人简化了时间跟踪。

[**5。KTimeTracker**](https://userbase.kde.org/KTimeTracker)
**针对:个人**
这个应用是 Kontact 的时间追踪部分，是 KDE 桌面环境的个人信息管理器。它提供了基本的待办事项管理和计时功能，允许用户创建任务列表并记录在任务上花费的时间。数据被汇总并存储在日志中，用户可以在日志中查看结果。

时间费用是自动计算的；用户只需要启动和停止定时器。如有必要，该应用程序还允许编辑已经记录的时间。

KTimeTracker 的配置选项非常简单:它们包括空闲时间检测、时间节省频率和显示细节。因此，对于那些不需要记录日常时间开销的人来说，这是一个不错的选择。

[**6。GnoTime**](http://gttr.sourceforge.net/)
**For:个人**
这个 Gnome 时间追踪器是在 GNU GLP 许可下发布的，所以是免费工具。它是为 Unix 系统设计的(所以它不仅可以在 Linux 上运行，也可以在 Mac OS X 上运行)。它提供简单的界面和基本的时间跟踪和计费功能。

像许多其他用于 Linux 的时间跟踪器一样，提供了一个计时器来记录时间开销。这个应用程序的具体功能是自动合并为一个任务记录的短时间间隔。要合并的间隔长度由用户定义。

每个条目都有一个“计费状态”参数:它可以标记为可计费或不可计费，并且可以设置计费率。还提供了紧急和完成状态。

记录的信息收集在应用程序中，然后可以显示在各种 HTML 报告中。它们可以显示一天中执行的任务、可计费金额和项目状态。

[**7。Fanurio**](https://www.fanuriotimetracking.com/)
**针对:个人**
Fanurio 是一款时间追踪和计费 app，大多是自由职业者使用。它支持 Windows 和 Unix 系统，可以从任何计算机上的 USB 驱动器运行，这对于那些在不同地点工作的人来说很方便。

该应用程序有一个计时器和一个手动时间输入表单。它的具体功能是智能定时，结合了空闲时间检测和提醒——以防你忘记启动定时器。Fanurio 的计费功能包括发票，其中包含已完成工程的详细数据和已计费金额。

Fanurio 是一款商业软件。它是按用户许可的，这意味着一个许可允许您在多台计算机上运行该应用程序。Fanurio 用户也可以使用付费维护计划。

[**8。arbtt**](http://arbtt.nomeata.de/#what)
**针对:个人**
ARBTT 的意思是“自动的基于规则的时间追踪器”，所以用户不需要手动输入任何东西。该程序记录哪些窗口已经打开并处于活动状态以及持续了多长时间，然后计算时间。

该产品支持 Windows 和 Linux，并在 GPL 许可下发布。自 2014 年初以来，似乎没有发布任何更新，但 arbtt 仍可在开发者网站上下载。

[**9。jTimeSched**](https://kbase.dominik-geyer.de/apps%3ajtimesched)
**针对:个人**
jTimeSched 是一款用于 Linux 的轻量级时间追踪软件。它允许创建任务和项目，使用计时器跟踪时间，添加注释到时间跟踪，并将收集的数据保存到日志文件中。

该应用程序基本上只包含一个。jar 文件，可以从 u 盘上运行。它的配置数据存储在每个目录中，这意味着您可以使用多种配置。该解决方案的作者强调，该应用程序将适合那些需要一个简单的时间跟踪器和极简界面的人，但当需要高级功能、花哨的界面和详细的任务层次时，可能不适合你的需求。

[**10。GTimeLog**](https://gtimelog.org//)
**针对:个人**
GTimeLog 是一个计时的小 GTK+应用。开发者表示，它的设计尽可能不具侵入性，因此它提供了简单的时间跟踪功能。当你到达工作地点并开始工作日时，启动应用程序并输入“到达”。完成工作日的每项活动后，在 GTimeLog 提示符下输入该活动的名称。

该程序计算你的时间支出，并在一天结束时创建一个活动报告，显示你花了多少时间工作和“偷懒”。简单的配置选项是可用的:有一些命令可以将特定的活动标记为与工作无关，或者在每日报告中完全忽略它们。

[**11。TimeSlotTracker**](https://sourceforge.net/projects/timeslottracker/)
**用于:个人**
一个简单的应用程序，用于跟踪分层树中任务的时间段内的工作时间。该工具适用于 Linux、Mac OS X 和 Windows，并在 GNU GPL 下免费分发。

该应用程序提供了与吉拉和 iCalendar 的集成。用户可以根据收集的时间跟踪数据运行报告，并将它们导出为 CSV、HTML 和 TXT 格式。该工具有七种本地化版本。
