# 你在哪里学的？

> 原文：<https://dev.to/rommik/where-do-you-learn--2342>

排名不分先后，这里列出了我最喜欢的学习新事物的地方。

1.  以前是 LearnVisualStudio.net，课程包罗万象。Net，VisualStudio，C#，微软工具。我获得了终身会员资格。我大三时买的最好的东西之一。

2.  多见——爱死了！优秀的 IT、DevOps、管理等主题！

3.  [EDX](https://www.edx.org/) -免费在线大学。我喜欢他们的微硕士和认证。

当然还有其他的，但是上面三个是我最喜欢的。

Highscalability.com 不是一个你可以找到课程的网站，但我喜欢它的“互联网上关于可扩展性的东西……”每周五发布。如果你对高流量、大数据、快速和边缘技术感兴趣，我强烈推荐这个网站。即使你不是，它仍然对你有用:)
