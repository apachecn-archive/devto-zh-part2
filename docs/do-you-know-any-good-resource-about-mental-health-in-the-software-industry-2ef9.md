嗨伙计们！

这是 Github 库:[https://github.com/dreamingechoes/awesome-mental-health](https://github.com/dreamingechoes/awesome-mental-health)。你知道什么好的资源吗？如果你能在这里留下评论，或者如果你想的话，你可以在知识库中提出请求，我将非常感激。
