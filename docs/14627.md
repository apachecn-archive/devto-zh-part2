# 然而，弗劳克施编码

> 原文:[https://dev.to/fraukesch/nevertheless-fraukesch-coded-5c CCC](https://dev.to/fraukesch/nevertheless-fraukesch-coded--5ccc)

## [](#i-begancontinue-to-code-because)我开始/继续编码是因为...

...我希望能够改变一些事情。找到做事的方法，让你能够真正去做事。当然，这不仅对编码是正确的:)，而且对我来说，它肯定是开始编码的关键助推器。我继续，因为我喜欢这样做。一方面，通过简单地知道如何让计算机/编译器/解释器理解我想要的东西，另一方面，自由地为复杂的问题找到创造性的解决方案，创造东西是非常令人满意的。

## [](#i-recently-overcame)我最近克服了...

...一个阶段，我告诉自己，我太愚蠢，太没有经验，无法与我的超级熟练的同事竞争。幸运的是，我周围的同事们帮助我说服了自己。每个人都曾经是一个学习者。事实上，你永远不会停止学习。一旦你承认了这一点，你也将永远不会停止好奇。只是探索！

## [](#i-want-to-brag-about)我想吹嘘一下...

...没什么，真的！我不喜欢吹牛，也不认为有必要让人们理解和重视你的贡献:)

## [](#my-advice-for-allies-to-support-women-who-code-is)我给盟友的建议是支持女性主义者的守则....

...帮助他们不被低估，帮助他们被认真对待。帮助他们不要低估自己。
