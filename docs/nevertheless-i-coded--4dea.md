# 然而，我编码了

> 原文：<https://dev.to/rosejcday/nevertheless-i-coded--4dea>

#### 我开始编码是因为...

回顾这些年，我可以说我从小就知道我会从事工程，以一种或另一种形式编码，但我没有想象它会发生。

2012 年 6 月我走过高中毕业的舞台，兴奋地成为康涅狄格大学的英语专业学生。几个月后，通勤到校园后，我开始了漫长而艰辛的大学之旅。大约一个学期后，我意识到英语不适合我，我可以很肯定地说，我尝试了许多其他选择，但在我的内心深处，我为自己做好了工程准备，在兼职编写代码的同时学习数学和科学。

连续两年来，每次去我的导师办公室，都是为了让我的课程得到批准。他想让我的课程迎合我仍然宣称的英语专业，但我知道这是行不通的。随着最后一个学期的临近，我申请了我唯一想去的另一所学校，哈特福德大学，主修电子工程。

在签署了离开康涅狄格大学的文件，经历了房屋选择和搬家计划之后，我第三次把我的专业改为了计算机工程。它的数学课比电气课少，但提供了编程课和更多选修课的好处。当我踏上校园时，我一点也不知道会发生多大的变化。

2014 年秋天，我开始了在哈特福德大学的第一个学期，我不认识任何人，我的导师对我来说也是一个陌生人。直到 2015 年秋季第三学期，也就是申报专业一年后，我才认识了一群体现工科精神的朋友。坐在电路分析 I，我们四个人开始了毕业的奋斗。从那时起直到 2017 年春天，我们开始了我们需要完成的电路和电子课程的四部分系列，但作为一个小组，我们学习，做实验室，做项目。在实验室的漫漫长夜，外卖和无数的测试证明是一场斗争，但我们互相支持。

到了仲春，我意识到我想编码，但不再是同样意义上的编码。我走近我的导师，说出了 5 年前不会发生的话，“我将攻读数据科学硕士学位。”

在完成了我的高级顶点设计，在我的大学里获得了 [CETA 设计博览会](http://unotes.hartford.edu/announcements/2017/05/2017-05-09-ceta-design-expo---spring-2017.aspx)的第一名，并走过毕业舞台后，我开始在 WPI 兼职学习数据科学，同时在一家家庭自动化公司的 DevOps 中全职工作。

到了 2018 年春天，我辞掉了工作，搬到了马萨诸塞州，全职完成硕士学位。

#### 我继续编码是因为...

在我高中毕业走上讲台的六年后，我主要通过 shell 脚本、Python 和 R 为学校编写代码，同时志愿作为一名英特尔软件创新者支持全世界的创新和独立开发者。通过这些，我已经能够使用英特尔 Edison、英特尔 NUC 和各种框架和库进行编码，以探索新的想法和项目。

通过志愿参加这个项目，我能够回到我毕业的技术高中，给追求工程和软件开发的年轻学生讲授工作坊。我们见面讨论了研讨会，更多地讨论了工程和编码的整体情况，并对未来可能发生的事情进行了一瞥，如果你在那所学校开设电子技术商店的话。

在我继续作为一名计算机工程师和数据科学家的旅程的过程中，像为英特尔做志愿者或访问我的母校进行演讲和研讨会这样的经历和机会都非常宝贵。

#### 我对其他编码女性的建议是…

别停下来。作为一名女性，在 STEM 领域你可能总会被另眼相看，有时你可能会被遗忘，但不要停止。找到一个合适的位置并坚持下去，但最重要的是，确保你热爱你所做的事情。
