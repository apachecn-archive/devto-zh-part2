# 点击刷新

> 原文:[https://dev.to/pavsaund/hitting-refresh-3lig](https://dev.to/pavsaund/hitting-refresh-3lig)

[![](../Images/03ab508eb018abb04f8dcd2e0f2f406f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--RsddxI7B--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2A8sQWAJE-NnO6TMMGgxk4HQ.png)

忘却是任何转变中最难的部分之一。你带来了多年的经验、期望和偏见。随着时间的推移，你已经适应了不同的工作环境。过去的经历能让你在工作和交流时迅速进入状态。它们也可能是沉重的包袱，拖你的后腿。在一家初创公司，这个包袱可能正是拖你后腿的东西。你正在用与你的情况不相符的观点看待全新的问题。

在杜立德，我们正在刷新我们以前的许多假设。用新鲜的眼光看待老问题。我们如何对待我们的编码实践。我们如何与客户合作。我们如何处理合同。我们如何制造产品。我们在重新学习，让我们的文化成长。

> “如果你的头脑是空的，它总是为任何事情做好准备，它对任何事情都是开放的。在初学者的头脑中有许多可能性，但在专家的头脑中几乎没有。”- [铃木顺溜](https://www.goodreads.com/quotes/17022-if-your-mind-is-empty-it-is-always-ready-for)

通信就是一个例子。来自一个非常分散的团队，我们中的一些人想要一个更正式的过程。更多的文档，更多的书面交流。随着我们的成长，这一点很重要，但在开始时，当优先级几乎每天都在变化时，这一点就不那么重要了。

点击刷新是开始任何新事物的重要部分。明确表示你需要后退一步，重新考虑你的期望。你会遇到新的问题，如果你专注于解决过去的问题，你就无法解决它们。同时，利用普遍的知识和原则也很重要。同样重要的是要认识到这些是有效的。

它们现在对你真的有效吗？
