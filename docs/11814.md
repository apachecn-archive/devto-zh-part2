### [](#new-features-of-angular-6)棱角分明的新特性 6

Angular 6 刚刚发布，具有令人惊叹的新功能和大量变化。

### [](#ng-update)ng 更新

运行 ng update<package>CLI 将分析您的 package.json 并更新您的包。更新 Angular 应用程序只需一个命令。</package>

### [](#ng-add)ng 加

它与 ng update 相同，但是它允许您添加和配置库。

例如，如果你想安装有棱角的材料，你必须运行:

```
ng add @angular/material 
```

### [](#other-changes)其他变化

。angular-cli.json 现更名为 angular.json，新结构

你可以在[官方公告](https://blog.angular.io/version-6-of-angular-now-available-cc56b0efa7a4)这里查看其他变化

### [](#upgrading)升级

Angular 团队创造了一个伟大的工具来升级你旧的 angular app

[Angular 更新指南](https://update.angular.io/)。使用它来迁移您的 angular 应用程序。

一旦你选择了你当前的 app 版本和你 app 的复杂程度。它将向您展示升级应用程序所需的步骤。我不会重复他们的步骤，只是按照他们的步骤，并在更新前观察命名的步骤，因为它必须正确。
