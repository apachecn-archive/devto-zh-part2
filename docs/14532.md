# 移民、外籍人士和移居国外者

> 原文:[https://dev.to/horia141/immigrant-expat-and-migr-2jhe](https://dev.to/horia141/immigrant-expat-and-migr-2jhe)

罗马尼亚的一项全国性运动是过去 30 年中向“西方”的大规模移民。关于是什么组成了散居群体，还存在一些争议。但自 2007 年加入欧盟以来，已有约 340 万人离开。有些是学生，但大多数是外籍工人。这大约是该国人口的 20%。因此，这种情况已经持续了一段时间，而且很可能还会持续一段时间。

有一个特定的网络文学分支致力于这个问题。它的目标是白领和专业人士，也就是那些经常在博客上闲逛的人。从如何在一个新的国家定居的建议，到生活在国外的其他人的印象，以及最受欢迎的关于 T2 糟糕经历的故事，以及另一边的草不那么绿的故事。我不愿意称之为移民-色情-但只是一点点。

所以这是我对这股文学潮流的贡献。我是从一个在国外生活了相当长时间并已回国的人的角度来写这篇文章的。我不会催促人们离开或留下。这是个人的决定。无论你有什么样的背景，都会比网上随便哪个 J Random Dude 说的话重要得多。

但是如果你决定离开，你最好做好你的功课。我说的不是学习你需要从当地税务机关得到什么样的文件，或者医疗系统如何运作，或者生活费用是多少。你应该*绝对*也这么做。但这是谷歌快速搜索可以解决的简单问题。我在想的是如何给自己定位，让自己作为一名外籍员工有一个*成功的好机会*。

首先，有一些定义。他们是模糊的，但是你可能听说过这些术语*外国人*和*移民*。它们都是外国工人的名字。前者有很好的内涵，并引发高层管理人员或从事高附加值工作的专业人员的思考。后者的内涵更差——非技术工人、建筑工人等。不过，这主要是一个认知问题。但据我所知，并不存在一种与反移民情绪相同的凝固的反外国人情绪。有一大堆讨厌的[-主义](https://en.wikipedia.org/wiki/-ism)来解释事实，但这不是这篇文章的重点。第三个定义是移民。这意味着人们在艺术界或科学界流动。通常是某种高知名度的创造者。所以你会说纳布可夫、海明威或哥德尔是移民圈的一部分。但不是跨国公司当地分公司的营销副总裁。非正式地说，如果我们把它们叠起来，它会变成`immigrant < expat < émigré`。

因此，建议是，无论你想去哪里，都要试着在外籍人士或移民群体中着陆。说起来容易做起来难。这是一个外界强加的标签——你不能*真的*为自己选择。但是你*可以*选择在一个给定的国家你搬到哪里。最重要的是搬到一个已经有很多外国人和强大的外籍人士社区的地方。这将增加你的机会，当地大多数人口有一个良好的看法外国工人。拥有一个强大的外籍人士社区也意味着会有更多专门的社交活动，而且通常会有更多和自己处境相同的人*可以联系。*

到目前为止，我们一直在抽象地谈论的地方是大城市和工业或行政中心。当你是学生或学者时，大学城也是不错的目的地。除此之外，我会感到厌倦。就像罗马尼亚的城市之间存在巨大差异一样，比如像布库列斯蒂这样的大城市和亚历山大这样的小城市之间，伦敦和伯恩茅斯之间，或者慕尼黑和不伦瑞克之间也存在巨大差异。情况也差不多。所有国家都面临着从小社区向城市的大规模内部移民。这具有你能想象的社会经济影响。作为一名外籍工人，为什么要前往这些地区呢？有趣的是，许多糟糕的外派经历都源自这些地方。

无论如何，这不是任何火箭科学或“T0”秘密“T1”级别的“T2”传说“T3”。如果有的话，这是最有可能的情况。大都市本身就很吸引人。作为工业和商业中心，它们是你更容易找到工作的地方。但这是一个值得强调的点，也是在计划任何这样的冒险时要牢记的。
