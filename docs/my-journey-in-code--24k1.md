# 我的代码之旅

> 原文：<https://dev.to/laraneedscoffee/my-journey-in-code--24k1>

这是我多年来学习如何编码的历程。

### 1。高中和代码

我的代码之旅始于我上高中的时候，我们最初被介绍到 FrontPage，我当时就想*哇，我做网站，我是天才*。后来，QuickBASIC 出现了蓝屏，我以为我正式成为黑客了！然后我们进入了 Visual Basic，在那里我写了我的第一个 if 语句，我的第一个 for 循环，并制作了我的第一个计算器——喜极而泣，我对机器说话并告诉它做什么的想法让我兴奋不已。然而，我花了一段时间来掌握和理解所有这些编程概念。

### 2。大学，代码和眼泪

然后考上了大学，深入编程。我基本上学习了 C++和 Java 作为编程语言，然后向我介绍了 OOP 的概念、算法、数据结构和数据库。这是非常艰难的，我在大学期间所做的就是哭泣，因为我没有取得好成绩，每学期我的平均绩点都会下降，我的眼泪和我的自尊。

### 3。工作和代码

在那之后，我用 Objective-C.
做了一份 iOS 开发的实习，在过去的 2 年里，我一直是一名使用 MVC #的. NET 开发人员。

我知道我仍然处于职业生涯的开始，然而事情对我来说并不容易，我正在尽我最大的努力变得越来越好。我试着参加一些在线课程，并阅读一些关于编码的书籍，但我仍然没有达到目标。
