# 重新定义技术

> 原文:[https://dev.to/dipisha03/redefining-tech-4208](https://dev.to/dipisha03/redefining-tech-4208)

#### 一瞥我的过去:

我在印度东北部出生并长大，直到十岁。我在西孟加拉邦一个名叫加尔各答的古老小城长大，这里因知识、正宗美食、音乐和高质量的友谊而繁荣。在漫长的十年后，我和我的家人获得了美国的永久居留权，并决定为了更好的机会重新定居。

在一个中产阶级家庭中长大，我和姐姐得到了生活中所有的必需品，并被教育要永远追逐自己的梦想。我们的父母教导我们要努力成为最好的自己，追求充满激情的职业，在这些职业中，我们可以真正改变社会，我想不出比技术更好的职业道路了。

#### 我为什么报名参加罗格斯大学软件开发项目，以及它对我生活的影响:

我决定加入该计划的一个主要原因是，技术触及到了我们今天所做的一切，而可用的机会是无穷无尽的。但我更大的目标一直是成为一个社区的一部分，这个社区日复一日地简化人们的生活，并庆祝多样性。

尽管我们可能与其他人有相似之处，但每个人都是独一无二的，都能带来真实的观点。这个项目让我有机会与不同性格和职业生涯不同阶段的人合作，这促进了多样性和创造性。

通过这个项目，我得以与一群独特的人见面并合作，他们与我一起创建了 ReDefiNingTech，这是一个致力于重新定义技术空间的有抱负的开发人员社区。我们的目标是帮助人们认识到，你不一定需要有计算机科学学位才能从事技术工作，但你需要有创造力，有激情，热爱学习，解决问题，并与来自不同背景的人交往才能取得成功。我很高兴成为这个充满激情的社区的一员，并被认可为科技界的一名女性。

#### 对我决策的重大影响:

我的主要影响之一是年轻的女企业家 Adda Birnir，她用自学成才的技术技能为《纽约时报》和 MTV 建立了网站。她的作品在 BBC、Fast Company、NBC 和 Mashable 上都有报道。虽然 Adda 毕业时获得了美术和非裔美国人研究的学士学位，但她在技术领域找到了自己真正的激情。

在参加这个项目之前，我通过 Adda 创建的 Skillcrush 学习编程的基本概念。这是一个互动的技术学习平台，有友好的教师，活跃的学生社区，并致力于帮助人们实现他们在技术领域的职业目标。它为女性提供了在家工作、照顾孩子以及在技术领域获得成功的机会。我真的很钦佩 Adda 追求她的激情，并为人们创造了一个支持性的平台，让他们拥有灵活的职业生涯。

#### 为什么其他女性应该考虑这个沉浸式项目:

自婴儿潮时代以来，女性的机会已经扩大，现在是我们成为领导者并有所作为的时候了。技术赋予我们以难以置信的方式改善社会的力量，每个女性都有机会成为这一重大运动的一部分，如果她愿意的话。女性在技术领域是多元化的，她们将在这个项目中学到宝贵的经验，成为她们想要的任何公司的资产。在这个项目中，你将学会与各种背景的人一起工作，并有机会领导自己的项目。所以，自豪地抓住这个机会，准备好一次改变人生的经历吧！
