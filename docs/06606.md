# [](#ltgt-gist-interview)< / >要诀面试

Gist Interview 是一个用 ReactJS 构建的应用程序，任何人或公司都可以使用它来模拟编程问题，只需创建“db.json”文件并填充其 Gist-id，最后生成一个仪表板。

