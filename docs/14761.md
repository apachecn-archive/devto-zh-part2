# 介绍我们 2018 年 3 月的赞助商

> 原文:[https://dev.to/ben/introducing-our-march-2018-sponsors-3bj 2](https://dev.to/ben/introducing-our-march-2018-sponsors--3bj2)

感谢这些出色的开发驱动型公司以各种方式支持我们。我们很高兴地介绍 Manifold 成为本月新的持续三月赞助商，非常感谢 TripleByte 和 Algolia 连续第二个月保持其金牌赞助级别。

## [](#manifold)歧管

[Manifold](https://www.manifold.co/?utm_campaign=dev.to_march&utm_source=dev.to&utm_medium=sponsorship) 是一个独立开发者服务的市场。这意味着他们提供了一种*更少混乱的*方式，通过一个单一的仪表板管理不同软件服务提供商的计划。在发现 Manifold 之前，我已经通过其他平台体验过这种便利，但总是受到大量供应商的限制，这使得它无法成为长期的解决方案。流形是一个出色的执行想法。我们碰巧也是 [Torus](https://www.torus.sh/) 的用户，用于在我们的开发过程中管理密钥，在我的一个帖子上，Manifold 的创始人之一发表了[这个评论](https://dev.to/ianlivingstone/comment/7if)。

## [](#triplebyte)三重交换

Triplebyte 是一个非常高效的就业安置平台。关于进入一家伟大的工程公司，最重要的是在这个过程中有人在你身边。Triplebyte 通过一系列他们自己的面试使你有资格为你争取冠军。直到我亲自参加了[的测验](https://triplebyte.com/?ref=devto_mar18)(为了好玩，我很高兴参与[开发到](https://dev.to/)😋).我建议你自己看一下。这是一次奇妙的暴露盲点的经历。

## [阿尔及利亚](#algolia)

[Algolia](http://bit.ly/2F3SjlV) 的核心产品是极低延迟的搜索，我们以各种方式在 [dev.to](https://dev.to/) 上充分利用了这一点。它支持我们的核心搜索，以及根据您关注的标签定制您的主页订阅源。我们也计划将它整合到其他地方。它已经成为我们整个产品的关键基础架构之一，正如您可能知道的那样，这一切都与性能有关。[加上免费试用](http://bit.ly/2F3SjlV)。

在与 Algolia 的合作中，我们还在搜索页面的侧边栏上添加了一条由 Algolia 支持的*消息。我们认为这是一种自然的合作关系，因为我们经常会收到“您的堆栈是什么”之类的问题。我们希望始终与赞助商合作周到，但您可以随时关闭任何现场赞助信息，可以在您的设置中禁用[。](https://dev.to/settings/misc)*

随着我们建立可持续发展的业务，我们正在努力整合其他赞助和融资渠道。我们努力使这些对所有各方都尽可能有价值，并将在具体合作伙伴启动时让您了解最新情况。
