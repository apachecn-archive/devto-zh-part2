在我开始之前，如果你不知道什么是 Hyperledger，这里有一个简单的介绍:

Hyperledger 是由 Linux 基金会维护的区块链框架。它旨在使企业能够真正快速地开发基于区块链的应用程序。它是目前最稳定的许可、通用和企业就绪的区块链开发平台之一。

以下是您现在可以利用的 Hyperledger Fabric 的五大优势:

### 1.模块结构

Hyperledger Fabric 鼓励开发人员在其架构中创建可插拔组件。例如，您可以在需要时以最快的方式插入 condenses。

这种模块化是由其健壮的架构实现的，该架构展望了这种全新区块链技术的未来。当你想把一些东西放入系统时，比如一个定制的身份管理系统，让用户使用建立在 Hyperledger Fabric 之上的区块链平台，这是非常有价值的。

### 2.按需数据检索

Hyperledger Fabric 的频道可能是迄今为止最被低估的功能之一。信道允许所谓的数据分区。这使我们能够保护我们需要保护的数据。

当愿意采用区块链的金融公司对其竞争对手偷看他们的数据深表担忧时，这尤其有用。我们谈论的是公司和银行，考虑到黑客可以在十年内破解加密技术，即使是非常好的加密技术也是不够的。

借助 Hyperledger Fabric 上的通道，您可以仅公开必要的数据，并将敏感数据存储在数据分区中。

### 3.为许可的区块链建造

不是每个区块链都必须匿名曝光。Hyperledger Fabric 允许所有实体具有已知的身份。金融公司需要得到许可的区块链，尤其是考虑到数据保护和医疗保健。

例如，以一家使用区块链的抵押贷款公司为例。房贷不是公开曝光的东西。这要求各方在网络中识别他们自己以验证真实性。

### 4.信任级别

Hyperledger Fabric 处理交易的方式与其他方式完全不同。这里的重点是减少信任层和交易需要的验证次数。这使得交易发生得更快更好，没有任何后果。
