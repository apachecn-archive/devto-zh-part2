# 如何用 5 个词表达 AWS 的价值主张

> 原文:[https://dev . to/Kyle Galbraith/how-to-articulate-the-value-propose-of-AWS-in-5-words-cc5](https://dev.to/kylegalbraith/how-to-articulate-the-value-proposition-of-aws-in-5-words-cc5)

最近，我写了一篇有趣的文章，讲述了开发人员如何说服 IT 经理开始使用 Amazon Web Services。由于所涉及的角色，这很快在互联网上遭到了很多鄙视。我在邮件中试图传达的信息显然没有被理解。

所以这次我会非常明确。

在上一篇文章中，我强调了像 AWS 这样的云提供商不仅可以为开发者，也可以为整个企业提供的价值主张。

这在我之前由一个长胡子的角色和一个时髦的云第一开发者组成的帖子中是没有听说过的。所以，吸取教训，不要再在我的帖子里搞笑了。这次让我们直接切入正题。

我们将从整体上看一下 AWS 给开发者和企业带来的 5 个好处。

### [](#value-1-security)值 1:安全

大型企业中的早期怀疑论者花了一段时间才真正相信这一点。有些人可能至今都不知道。但是，现实情况是，安全性确实是 AWS 的头等大事。

作为一名开发人员，您可以从 AWS 已经获得的安全性和合规性认证中受益。需要支持 HIPAA 还是 SOC II？使用被认证为兼容的 AWS 服务，并且您至少可以选中该复选框。这使您能够专注于对您的应用程序至关重要的安全性和法规遵从性方面。

当您考虑 AWS 时，记住共同责任模式非常重要。

该模型指出，亚马逊将提供云的安全性，而您提供云中的安全性。这完全说得通。作为开发人员，您需要负责保护您的应用程序。然而，通过利用 AWS，您可以获得 Amazon 的物理安全性和专业知识。对企业来说是全面的胜利，尤其是如果他们目前正在维护自己的数据中心。

### [](#value-2-flexibility)值 2:灵活性

撰写本文时，AWS 包含 100 多种服务。它提供从图像识别到数据库迁移的各种支持服务。您可以在 AWS 上运行的工作负载有很大的灵活性。

作为一名开发人员，这意味着很有可能你可以在 AWS 上运行任何栈。当然，在这样做之前，你应该问自己一些问题。例如，工作负载是否针对在 AWS 上运行进行了优化，如果不是，会有什么后果？

这就是灵活性对于开发者和企业的另一种意义。有大量的服务可以运行各种不同的工作负载。但是，有了 AWS，您还可以灵活地适应适合自己的变化步伐。直接投入并运行许多不同的服务，或者从一个工作负载和一个服务开始。两种方式都可行。

### [](#value-3-elasticity)值 3:弹性

不再需要过度调配工作负载来支持每年某一天的流量激增。以高昂的价格预先租赁硬件的日子已经一去不复返了。小型和大型企业的巨大胜利。

利用像 AWS 这样的云提供商，您可以按需扩展。再也不会有浪费容量或资源调配不足的情况，让您在周六凌晨 2 点醒来。这对开发人员来说是一个很大的好处，可以在您睡觉的时候响应事件，即时扩展您的工作负载。

### [](#value-4-cost)值 4:成本

这似乎是显而易见的，不再需要预先租赁硬件，不再需要维护数据中心，也不再需要过度配置容量。所有这三个因素都会降低成本。

但成本价值主张实际上比这更微妙。是的，运行自己的数据中心或预先购买/许可硬件成本高昂。但是，让您的几十年前的应用程序无法利用 AWS 也是如此。其实做得差，等着吧，云可以更贵。

这并不是说这个价值主张是错误的。就将资本支出转换为可变支出而言，这是完全正确的。在 AWS 中，您只需为您使用的内容和使用时间付费。请注意，在过度调配的硬件上全天候运行工作负载是非常昂贵的。

### [](#value-5-agility)数值 5:敏捷

这是我最喜欢 AWS 和所有云提供商的一点。它们真正为企业和开发者提供了试验、迭代和以令人难以置信的速度前进的灵活性。

曾经可能需要几个月或几年的想法可以在几个小时或几天内执行和交付。在一个发展速度越来越快的世界里，缩短从假设到数据的时间是非常强大的。通过不再关注基础设施供应、维护和开销，开发人员和企业都可以解放出来，专注于在其系统中创造价值。

### [](#conclusion)结论

这些价值主张是在 Amazon Web Services 的上下文中讨论的，但它们也与其他云提供商相关。关键是云计算正在加速推动技术向前发展。

与十年前相比，从一个想法转向另一个想法的能力变得简单了。现在，根据您的需求进行扩展就像一段代码一样自动且可行。成本正在大规模下降，但即使在利用云提供商时，仍应受到监控。

我的长胡子博客帖子在传达这些价值主张时没有击中要害。但希望现在，用简单的英语，他们是清楚的，值得在饮水机旁讨论。
