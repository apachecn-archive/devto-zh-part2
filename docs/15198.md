# Go 中的日期时间格式

这个*总是*在围棋上把我搞糊涂。你是怎么把一个`time.Time`值转换成一个`string`值的？

文档是[这里](https://golang.org/pkg/time/#Time.Format)和[这里](https://golang.org/pkg/time/#pkg-constants)和[这里](https://golang.org/pkg/time/#Time)。关于从哪里开始，真的没有任何线索。

当你进一步挖掘这个包和它的代码时，你会发现[这个](https://golang.org/src/time/format.go?s=23757:23803#L88)小宝石。它定义了格式中可用的不同占位符的值。

所以让我们从一个简短的例子开始:

```
 t := time.Date(2018, 1, 5, 20, 43, 22, 123456789, time.Local)
    fmt.Println(t.Format("2006/01/02 15:04:05.999 -07"))
    fmt.Println(t.Format("3:04 PM, January 2 06"))

    // Output:
    // 2018/01/05 20:43:22.123 +00
    // 8:43 PM, January 5 18 
```

这就是完整的日期/时间字符串格式…

| 密码 | 意义 | 例子 |
| --- | --- | --- |
| 一月 | 这是完整的月份名称 | “一月”/“二月” |
| 一月 | 短月份名称 | “一月”/“二月” |
| one | 普通数字形式的月份 | “1” / “2” / “12” |
| 01 | 2 位数的月份 | “01” / “02” / “12” |
| 星期一 | 一周中的第几天 | “星期一”/“星期三” |
| 孟人 | 一周中短暂的一天 | "我的" "婚礼" |
| Two | 一个月中的某一天，用普通数字表示 | “5” / “27” |
| 02 | 一个月中的某一天，以两位数表示 | “05” / “27” |
| _2 | 一个月中的某一天，为固定宽度的普通数字 | ” 5” / “27” |
| Fifteen | 24 小时制，2 位数字 | “02” / “15” |
| three | 第 12 小时，普通数字 | “2” / “15” |
| 03 | 第 12 小时，2 位数 | “02” / “15” |
| four | 普通数字形式的分钟 | “9” / “42” |
| 04 | 分钟为 2 位数 | “09” / “42” |
| five | 普通数字形式的秒 | “8” / “25” |
| 05 | 秒为 2 位数 | “08” / “25” |
| Two thousand and six | 至少 4 位数的年份 | “0023”/“1999”/“25643”(*WFT！*) |
| 06 | 两位数的年份 | “23” / “99” / “43” |
| 首相 | 大写的 12 小时周期 | “上午”/“下午” |
| 下午 | 小写 12 小时周期 | “上午”/“下午” |
| 山地时间 | 时区短代码 | MST" / "CET" / "UTC " |
| Z0700 | 以小时和分钟偏移量表示的时区(Z 表示 UTC) | "-0600" / "+0100" / "Z " |
| Z070000 | 以小时、分钟和秒为单位的时区偏移量(Z 代表 UTC) | "-060000" / "+010000" / "Z " |
| Z07 | 以小时为单位的时区偏移量(Z 代表 UTC) | "-06" / "+01" / "Z " |
| Z07 点 | 用冒号分隔的时区的小时和分钟偏移量(Z 表示 UTC) | "-06:00" / "+01:00" / "Z " |
| Z07:00:00 | 用冒号分隔的时区的小时、分钟和秒钟偏移量(Z 表示 UTC) | "-06:00:00" / "+01:00:00" / "Z " |
| -0700 | 时区以小时和分钟表示，始终为数字 | ”-0600” / “+0100” / “+0000” |
| -070000 | 时区以小时、分钟和秒表示，始终为数字 | ”-060000” / “+010000” / “+000000” |
| -0700 | 时区以小时和分钟表示，始终为数字 | ”-0600” / “+0100” / “+0000” |
| -070:0:00 | 用冒号分隔的时区的小时、分钟和秒钟偏移量，始终为数字 | ”-06:00:00” / “+01:00:00” / “+00:00:00” |
| .0/.00… | 秒的分数，包括尾随零 | ”.67” / “.900” |
| .9/.99… | 秒的分数，包括尾随零 | ”.67” / “.9” |
