# 我选择自学！

> 原文：<https://dev.to/brendalimon/i-choose-self-learning-5g01>

嘿伙计们！在我的[上一篇文章](https://dev.to/brendazam/code-camp-or-self-learning-1klc)中，我请求你们的帮助，因为我需要决定我的开发者生涯的走向，正如我所说的，你们中的很多人给了我很棒的建议，因此我做出了选择，走自学之路。

我不知道你来自哪里，但至少在墨西哥这里，(你被邀请参观并与我一起吃墨西哥玉米卷)技术社区很小，只集中在几个城市，如果你没有机会，你只能尝试去上大学，看书(几本)或上网。现在，分析利弊，我决定自学。制定一个计划，我想成为什么样的开发人员，我想做什么工作，根据我的学习方法找到课程(我讨厌无聊的声音)。

但是我也需要压力，所以我不会停止做这件事，所以[我开始了这个](https://www.youtube.com/watch?v=-nw-ANXaJVM&t=)，这是我自己的 vlog，一些新手可以在这里学习，获得关于应该选什么课程的建议，所以很多像你们这样的优秀的人可以帮助我成为一名优秀的开发人员。

我还为我的 vlog 设计了一个图案，我认为展示我的个性很重要，我还没有给它起名字，所以如果你有这个小东西的名字，请告诉我！
