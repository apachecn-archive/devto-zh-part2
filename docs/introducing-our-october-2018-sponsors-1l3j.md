# 介绍我们 2018 年 10 月的赞助商

> 原文：<https://dev.to/ben/introducing-our-october-2018-sponsors-1l3j>

感谢我们出色的赞助商，他们帮助维持了 [dev.to](https://dev.to) 生态系统和普通开发者社区。

这三个开发者平台都是各自领域的领导者，如果你还没有这样做的话，我强烈建议你去看看他们的产品。说真的，现在花几分钟时间浏览一下他们的产品指南和文档。

*也是*，这个月是 [#hacktoberfest](https://dev.to/t/hacktoberfest) 。这是由 DigitalOcean 与 Github 和 Twilio 合作发起的一项倡议，旨在让人们参与到开源项目中来。查看 [dev.to](https://dev.to/) 上的标签，了解你可以在哪些方面做出贡献的大量指导。有关如何兑换 Hacktoberfest 奖品的信息，请查看官方网站。

## 数字海洋

[DigitalOcean](https://try.digitalocean.com/performance/?utm_source=devto&utm_medium=display&utm_campaign=Devto_2018_Brand) 是一个备受喜爱的云计算平台。我总是对数字海洋核心产品的良好构建*和良好记录*印象深刻。说起来容易做起来难，他们已经能够保持领先。

迄今为止，DigitalOcean 一直是我们最长期的支持者，我们非常感谢他们的一贯支持。他们对开发者生态系统的承诺非常明确。

## Twilio

自从[发展到](https://dev.to/)以来，Twilio 一直是我们的好朋友。我们是他们 API 的用户，我们团队的成员总是很乐意参与他们的社区活动。有他们做赞助真是太好了！

我个人可以说，我们选择 Twilio Video 是因为它充分利用了本地对等浏览器技术，同时还为包装在同一 API 中的必要用例提供了 Twilio 的基础架构。文档和示例组织得很好，启动和运行是同步的。

## Qlik 分支

[Qlik Branch](https://developer.qlik.com?utm_medium=direct_buy&utm_source=devto&utm_campaign=allylogo) 是一个编程环境，允许您使用、探索和快速测试使用 Qlik 技术的应用程序。Qlik 是一个数据分析平台，几乎在每个行业都用于摄取、可视化和分析。

如果没有像 Qlik Branch 这样的开发工具，针对这些类型的产品进行开发可能会很麻烦。正是这个工具让我们的团队将 Qlik 作为一个可能的解决方案，因为我们将 [dev .扩展到](https://dev.to/)。
