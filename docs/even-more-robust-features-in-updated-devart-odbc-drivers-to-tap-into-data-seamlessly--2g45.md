# 更新的 Devart ODBC 驱动程序中更强大的功能，可以无缝地利用数据

> 原文：<https://dev.to/devartsoftware/even-more-robust-features-in-updated-devart-odbc-drivers-to-tap-into-data-seamlessly--2g45>

用于数据库管理和开发的数据库连接解决方案和工具的捷克软件提供商 Devart 宣布发布用于 Windows、macOS、Linux 的云和数据库的 ODBC 驱动程序的更新版本，包括 32 位和 64 位。采用增强的 ODBC 驱动程序，用户可以在许多流行的 ide、领先的分析和报告工具以及各种编程语言中顺利地获取所需的数据。

下面是关于 Devart ODBC 驱动程序新特性的独家新闻:

*   所有 ODBC 驱动程序现在都提供了将字符串类型返回为 Ansi 或 Unicode 的可能性

*   从此版本开始，MySQL 的 ODBC 驱动程序支持最新的 MySQL 版本 8.0 以及 sha2_password、caching_sha2_password 身份验证。

*   在 MongoDB 驱动程序中，文档获取速度大大提高。

*   Zoho CRM 的 ODBC 驱动程序现在允许用户选择域——Zoho CRM 的 URL。

*   与 MS Access、Tableau、Omnis Studio、Power Pivot 和 DBeaver 的兼容性显著增强，消除了使用 Devart ODBC 驱动程序时的任何问题。

仔细看看数据库和云的 ODBC 驱动程序中现在提供的一些新特性:[https://www.devart.com/odbc/](https://www.devart.com/odbc/)。
