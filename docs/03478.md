# 帮助网站开发者克服骗子综合症的 5 个工具

你有没有过这样的感觉:每天都是万圣节，而你是一名网络开发人员？

冒名顶替综合症会在最不方便的时候给新的有抱负的网络开发人员带来沉重的打击。它将在以下情况下命中:

*   你刚刚错过了一份很棒的工作。
*   你失去了一份合同，因为你的费率太高或者其他人更有经验。
*   你发现你把一个 bug 推向了生产。
*   一位同事或经理斥责你不具备某项技能。
*   一个客户拒绝付钱给你。

不管你有多了不起(让我们面对现实吧，你可能真的很了不起)，**冒名顶替综合症最终会露出它丑陋的一面**。你需要一些工具来处理它，把你从毁灭的边缘拉回来，让你重新站立起来。这里有**5 个工具，当我觉得自己不够优秀而不能成为一名网站开发人员时，它们可以帮助我恢复**。

1.  与朋友和家人谈论这件事。你的朋友和家人希望看到你成功。当你觉得你做不到的时候，他们会提醒你已经走了多远，你有什么能力。我发现与开发人员的朋友交谈是有好处的，因为他们对你能力的评价比业外人士更有分量。
2.  衡量你的进步，不要和其他人比较，而是和你昨天、上个月或去年在哪里比较。最引人注目的开发人员比我们更有成就。这并不意味着我们什么都不能提供。即使我们不是最好的，甚至不是前 10%，我们也可以变得非常有价值。一个中庸的 web 开发人员仍然比 99.9%的企业主更好，这些企业主可以在他们的服务中找到荒谬的价值。
3.  认识到“专家”并不意味着最好。与最后一个相关但又不同。有人曾经告诉我，要成为专家，你只需要领先一课。如果我们现在就能做一些人需要的事情来推动他们的业务，那么我们不知道周围的一切，甚至我们上周才学会做这些事情都没有关系。尽管我不是大多数人认为的“专家”意义上的“专家”，但我创建了这个网站，但我有一系列的经历，让我在你现在的位置上给你一些价值。(我希望是这样，反正。😉)这使我成为这方面的“专家”,我能够利用我的“专业知识”来帮助你前进，尽管还有很多我不知道的东西。
4.  回忆并庆祝你过去的成就。有时感觉一切都在倒塌，这通常是骗子综合症发作的时候。即使此时此刻你的运气可能会让你失望，你仍然有很多成就可以反思。就在几周前，我最近参与的两个付费项目都停止了工作。我得到了三周零报酬的工作。这真的很可怕，这让我开始怀疑我是否能做到这一点。我帮助自己克服的一个方法是回想一下，就在几年前，我怎么也想不到会横跨整个国家搬到我现在称之为家的西雅图。我想了想我是如何找到一份工作，并谈妥了一份令人难以置信的薪水。我想到了我在 Entrepreneur.com[和 T4【CSS-Tricks】上发表文章的事实。我看着电脑上的项目目录，想起了这些年来我建立的所有东西。当然，在那三个星期里，我做得不是很好，但是历史告诉我，我很快就会再次回到巅峰状态。](https://www.entrepreneur.com/article/242406)
5.  **教人。**即使你刚开始工作，也有人比你级别低。能够帮助别人是一种难以置信的满足感。它迫使你确定你真的明白你在教什么。这里有一些地方你可以找到真正需要你专业知识的人:
    *   **论坛**如[站点](https://www.sitepoint.com/community/)、[自由代码营](https://www.freecodecamp.org/forum/)、[代码学院](https://discuss.codecademy.com/latest)和[奥丁项目](https://forum.theodinproject.com/)
    *   **子编辑**像 [learnprogramming](https://www.reddit.com/r/learnprogramming/) 、 [learnjavascript](https://www.reddit.com/r/learnjavascript/) 和 [webdev](https://www.reddit.com/r/webdev/)
    *   开发者聚会，尤其是那些自称对新手友好的人

教导别人是另一个提醒，提醒你已经走了多远，不管你看到多少人比你知道得多，还是有很多人知道得少。通过一点点努力销售，其中一些人会成为你的客户，这证明你不是骗子。
