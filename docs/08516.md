我做了一个 gem“thas _ it”，有了这个 gem，你就可以用`it`的方法得到块的单个参数。

```
require "thats_it"

p [1, 2, 3].map { it * 2 }
# [2, 4, 6] 
```

## 它是如何工作的？

基本思想是在调用之前重写块的 iseq。

```
# rewrites this block ↓
[1, 2, 3].map { it * 2 }

#          to this ↓
[1, 2, 3].map {|it| it * 2 } 
```
