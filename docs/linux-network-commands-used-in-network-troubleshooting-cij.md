# 网络故障排除中使用的 Linux 网络命令

> 原文：<https://dev.to/macnux/linux-network-commands-used-in-network-troubleshooting-cij>

一旦您确认了物理网络正在工作，下一步就是对您的网络进行故障排除，现在我们进入我们的主题，Linux 网络命令以及如何使用它们对您的网络进行故障排除。

1 使用 ping 命令检查网络连接
2 使用 dig 和 host 命令获取 DNS 记录
3 使用 traceroute 命令诊断网络延迟
4 mtr 命令(实时跟踪)
5 使用 ss 命令检查连接性能
6 安装并使用 iftop 命令进行流量监控
7 arp 命令
8 使用 tcpdump 进行数据包分析
