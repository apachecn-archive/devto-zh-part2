# 写作更多

> 原文:[https://dev.to/gonsie/writing-more-1e4p](https://dev.to/gonsie/writing-more-1e4p)

我的新年决心之一是多写点东西。当我说出我的目标时，似乎每个人都或多或少做过同样的事情。他们还与我分享了许多专业技巧，我在此分享。

## [](#meaning)意为

对我来说，写更多的东西是一项包罗万象的努力。它意味着以各种形式写作:

*   博客
*   为工作中的每月时事通讯写作
*   为软件项目编写发行说明
*   为软件项目编写文档
*   写每日(个人)日志

为了这个目标，几乎所有的写作形式都很重要。我心中唯一的例外是电子邮件。电子邮件是用来交流的，是一种即时的东西，文字通常不会持续很久。

## [](#tip-1-join-a-group)提示 1:加入一个团体

有很多写作小组，他们在新年伊始特别活跃。找到一个适合你写作风格(学术/技术与创造性写作)的小组是关键。如果找不到合适的团，至少找个朋友。有个哥们什么都好！

## [](#tip-2-create-time)提示 2:创建时间

写作需要时间和努力。它包括集中精神，组织你的想法并清晰地表达出来。创建一个有专门写作时间的时间表是必要的。一个典型的建议是早上第一件事就是写作。尤其是如果你是(或者可以让自己成为)一个早起的人，拥有一个安静的早晨会是很有力量的。

我发现早晨的第一件事(实际上第二件事，我在早上 5:30 锻炼)就是写我的个人日记，这是开始我一天的好方式。一开始，打破查收邮件和沉迷网络的习惯可能会很难。但是，我很快发现我会期待我的写作时间，并且不愿意让它结束。

一个同事建议我在工作生活中制造同样的空间。每天早上工作的时候(在检查邮件之前)抽出一些时间来写作。虽然我认为这是一个好主意，但我不确定我到底会写些什么，因为为我的博客写作并不合适。

## [](#tip-3-create-space)技巧三:创造空间

创造写作时间的必然结果就是创造空间。这可以是物理的或数字的。给自己准备一个漂亮的笔记本，一张结实的桌子，和/或一个优雅的写作应用程序。找到一种令人愉快和舒适的写作形式。

## [](#tip-4-writing-vs-editing)技巧 4:写作与编辑

这位同事明确表示，写作时间应该与编辑时间明确区分开来。每一个都是完全独立的过程，使用大脑的不同区域。因此，划出单独的时间来编辑你的文字是很重要的。

我还没有找到实现这一目标的方法。目前，我把写作的日子和编辑的日子交替进行。

## [](#tip-5-track-progress)提示 5:跟踪进度

最后，记录你写了多少也很重要。这可以用来观察你一段时间内的进步，并作为一个激励你继续前进的动力。我可以想象许多写作小组会将这样的写作日志作为问责的一步。

只要你在进步，精确的字数目标并不重要。我可以想象许多写作应用程序都内置了字数统计，我确信有一种方法可以用 Emacs 和 Org 模式来记录。

## [](#motivation)动机

通过这一努力，我希望实现几件事:

1.  用内容填满我的博客。我真心喜欢与他人分享知识，并教人们如何做我所学到的事情。博客是实现这一目标的绝佳工具。
2.  填补工作上的空白。我从事软件开发，通常情况下，文档会留到以后。我确实喜欢创建文档，并且我愿意认真地为此努力。此外，简化文档处理已经在我的任务清单上很长时间了。
3.  在工作中获得更多认可。正如我告诉别人的，没有人知道你做了什么，除非你告诉他们(即使这样他们也可能听不到你)。我想练习分享自己和团队成就的艺术。
4.  最后，还有一长串我今年想要实现的事情。写每日个人日记是我一直想做的事情，但我从未真正做到过。我希望通过明确每周的工作重点，我可以完成更多的工作，无论是写作还是生活。
