## 初始应用配置和设置

要设置**虚拟化**、**谷歌素材图标**和**虚拟化**，您需要将这些行添加到`src/main.js` :

```
import Vuelidate from 'vuelidate'
import Vuetify from 'vuetify'

import 'material-design-icons/iconfont/material-icons.css'
import 'vuetify/dist/vuetify.css'

Vue.use(Vuelidate)
Vue.use(Vuetify) 
```
