本教程解释了如何在 Javascript 中使用 class。像 C++、java、php 等编程语言一样，我们可以用非常简单的步骤用 javascript 创建类。JavaScript 类是一种函数。类是用 class 关键字声明的。类实际上是“特殊函数”，正如你可以定义函数表达式和函数声明一样，类语法有两个组成部分:类表达式和类声明。我们将逐一讨论。
