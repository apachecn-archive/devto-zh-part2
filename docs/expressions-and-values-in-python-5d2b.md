Python 表达式描述了对数据执行的计算或操作。它是所有程序的基本构件。它计算出一个 python 值，并且可能使用几个操作数。

例如，最简单的表达式就是像 5，“h”，1.1 等这样的文字表达式。变量名也是一个表达式。算术表达式(5+10)描述了 5 和 10 的加法运算。

一个表达式可以包含子表达式:表达式(5+10)包含子表达式(5)和(10)。所以总的来说，表达式(5+10)包含了三个不同的表达式(5)、(10)和(5+10)。
值

python 中数据的内部表示称为值。表达式的求值会产生一个 pythonic 值。
Python 如何对一个表达式求值？

对表达式求值会产生一个 python 值。文字表达式 5 不同于 python 值 5。Python 通过首先评估子表达式并使用该值评估更大的表达式来评估表达式。子表达式可以包括更多的子表达式。这个过程一直持续到一个基本表达式。

涉及复合表达式评估的步骤:

如果其中一个操作数本身是一个表达式，那么它被称为复合表达式。比如——(5 * 10+3–1)。

考虑这个例子，来评估(5 * 10+3–1)。

*   Python 首先将 5*10 计算为值 50。
*   然后，它将 3–1 计算为值 2。
*   然后将两者相加得到 52。

注意，为了计算一个表达式，Python 会计算几个更小的表达式(比如 5*10)。此外，为了计算 5*10，Python 将表达式 5 计算为值 5，依此类推。像 2 这样的文字表达式的值就是相应的值，所以这是 Python 停止划分子表达式的地方。

通常，括号用于强制表达式中的求值顺序。因为括号的优先级最高。当一个表达式中出现多个运算符时，求值的顺序取决于优先规则。对于数学运算符，Python 遵循数学惯例。缩写“PEMDAS”是记忆规则的一种有效方式:

*   括号具有最高的优先级，可以用来强制表达式按照我们想要的顺序求值。由于首先计算括号中的表达式，因此 2 *(3–1)是 4，(1+1)* *(5–2)是 8。
*   取幂具有次高的优先级。
*   乘法和除法具有相同的优先级，高于加法和减法，加法和减法也具有相同的优先级。
*   具有相同优先级的运算符从左到右进行计算。所以表达式(5–3–1)将被计算为 1，而不是 3，因为首先计算(5–3 ),然后从 2 中减去 1。
*   取幂是一个例外，它是从右向左计算的。所以(2* *3* *2)是 512，因为它的计算结果是(2* *(3* *2))而不是 64(即(2* *3)* *2)。
