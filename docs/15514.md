## [](#about-us)关于我们

Nylas Cloud API 使公司将电子邮件、日历和联系人集成到他们的应用程序中变得更加容易。作为商务沟通、日程安排和联系的核心，Nylas 正在塑造人们工作方式的未来。

我们的[团队](https://www.nylas.com/team/)在确定的性别方面大致平等，我们积极并定期与整个团队合作，按照我们的授权、透明和善良的理想塑造我们的文化。

我们合作写下了我们的价值观、福利、津贴，并发表在我们的[开源手册](https://github.com/nylas/handbook)中。阅读它以了解更多关于 Nylas 的信息。您还可以在 https://www.keyvalues.com/nylas 的[了解更多详情。](https://www.keyvalues.com/nylas)

## [](#about-the-opportunity)关于机遇

Nylas 正在寻找一个聪明、有抱负和精力充沛的人加入我们的 API 开发者成功团队，在这里你将直接负责确保我们的合作伙伴和开发者的成功。数百家企业依靠 Nylas Cloud APIs 来支持他们的产品，您的角色对于确保他们的持续增长和成功至关重要。您将与我们行业内的初创公司首席执行官和顶级技术人才合作，帮助他们使用 Nylas Cloud APIs 构建出色的应用程序。

这包括与 Nylas 工程师一起工作，帮助诊断影响我们产品可靠性和速度的问题。您将帮助确定客户问题的优先级和趋势，并实施解决这些问题的流程和自动化。您对我们基础设施的深刻技术理解将使您能够将新客户纳入我们的平台，并快速帮助现有客户。

你应该有一个成长的心态，有管理自己项目的记录，在日常工作中有很强的自主性。你也应该对自己的能力保持谦逊，并有一种与生俱来的将知识传授给他人的愿望。

## [](#responsibilities)职责

*   帮助客户解决技术问题并回答有关产品功能的问题
*   以技术专家的身份引导客户对话，您在 Nylas Cloud APIs 方面的知识和专业技能将帮助您指导实施和与客户沟通。
*   管理客户支持票证，并定期更新未结票证
*   开发面向客户的文档和集成指南
*   与我们的开发团队(提供反馈、归档和修复错误)、业务团队(快速响应高优先级问题)和产品团队(提供路线图反馈)密切合作
*   开发描述 Nylas 云 API 的技术消息和平台架构文档

## [](#what-were-looking-for)我们在寻找什么

*   您有通过电子邮件、电话和面对面与用户沟通并提供产品支持的经验
*   你有处理技术问题、排除故障和与工程师沟通的经验
*   你渴望在一个快节奏的环境中工作，并且有很大的自主权
*   你发现隐藏的细节和总结更广泛的趋势一样有效
*   你有很好的口头和书面沟通技巧
*   你对技术用户有很深的同理心，你擅长让客户感到被授权
*   你对新技术感到好奇和舒适，你喜欢学习新工具。
