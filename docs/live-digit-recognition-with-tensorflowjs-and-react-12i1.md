## 设置环境

这个项目是基于 JavaScript 环境，你需要安装与纱线或 NPM 的依赖关系:

```
$ yarn install 
```

## 本地启动

```
$ yarn start
$ Open https://localhost:9000 with your favorite browser 
```

## 为生产而制造

```
$ yarn build 
```
