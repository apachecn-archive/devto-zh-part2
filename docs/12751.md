## 设置项目

设置一个长颈鹿项目非常简单。首先，让我们从添加长颈鹿项目模板开始:
`dotnet new -i "giraffe-template::*"`

前面的命令安装了一些项目模板，以便构建 Giraffe web 应用程序。之后，您可以通过运行`dotnet new giraffe`创建一个新的 Giraffe 应用程序。
