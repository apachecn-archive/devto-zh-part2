# 你理想的编码时间是多少？

> 原文：<https://dev.to/mmarinez20/what-is-your-ideal-coding-hour-1pn5>

作为一个白天做软件质量保证，晚上做软件工程学生的人，我很难在白天提高我的编码技能。有工作，有学习，(漂亮的)女朋友，有家人，有朋友，想抽出一点时间给我做代码的时候，我有时候会很纠结。当我在 Dev.to 阅读一个帖子时，我发现有人建议早上提前一个小时起床去做我想做的事情，这真的很有帮助...直到我有那么多作业要做...嗯，我想睡至少 5 个小时。

出于激励的目的，我一直在考虑在我最近知道的#100DaysOfCode 挑战赛期间为我女朋友的生日建立一个生日网页

所以我想出了这个问题给大家！，你最适合做代码的时刻是什么(可以加上你做准备的任何仪式 '-')>)。
