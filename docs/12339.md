## [T1](#tensorflowjs)[tensor flow . js](https://js.tensorflow.org/)

TensorFlow.js 是 TensorFlow 家族中令人敬畏的新成员之一。TensorFlow.js 支持 TensorFlow Layers API，通过访问低级 Ops API(以前的 Deeplearn.js)进行硬件加速，可以在浏览器中训练模型，并使用导出的 Keras 模型和 TensorFlow SavedModels。

这使得通过 JavaScript 使用机器学习变得更容易，这具有很多含义——因为 JavaScript 几乎可以在任何地方使用:用于移动、桌面甚至 VR 开发。现在我们可以很容易地将机器学习应用到那些领域。此外，几乎每个 web 开发人员都知道 JavaScript，这使得 AI 民主化又进了一步。

[![TensorFlow Hub](../Images/f93e556bc3c1e21f713ec3a585ff2b2e.png)T2】](https://www.tensorflow.org/hub/)

## [](#tensorflow-hub)[TensorFlow Hub](https://www.tensorflow.org/hub/)

类似于 TensorFlow.js 如何导入 Keras 和 TensorFlow 模型，我们现在可以使用 TensorFlow Hub 在一行代码中轻松导入强大的机器学习模型。TensorFlow hub 就像是 Inception 之类的流行 ML 模型的存储库。见鬼，连纳斯特都在。

[![PopMotion POSE](../Images/8d74b97b501aebf44b75c087fcda0591.png)T2】](https://popmotion.io/pose/)

## [](#popmotion-pose)[PopMotion POSE](https://popmotion.io/pose/)

PopMotion POSE 是 PopMotion 的声明版本。声明性基本上意味着你定义某事应该做什么，而不是关心它应该如何以及何时发生。这允许你拥有 PopMotion 的力量，并赋予它简单性。这使得使用 React 非常棒，React 本质上也是声明性的。

## [](#jenkins-x)[詹金斯 X](http://jenkins-x.io/)

Jenkins X 为 CI/CD 带来了 Jenkins 的强大功能和灵活性，并将其带到了 Kubernetes。虽然具有相当高的抽象水平，但它仍然没有失去灵活性，因为您仍然可以深入并根据自己的喜好定制一切。
