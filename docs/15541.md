Ug，谁想看关于 Web 开发的博客？我打赌你们中的一些人。我不记得是谁把我推向这个方向，但前提是:如果你正在寻找一个 web 开发主题的信息，但没有找到你要找的东西，那就写一个博客。即使有 100 篇关于相同或相似主题的其他博客帖子，您对该主题的看法可能会有所不同，足以满足另一个开发人员对其特定情况的答案的不懈搜索。

所以我们在写博客。目前有三个活跃的开发人员在我们的团队中，我们的经验涵盖了从老学校，自学，2400 波特调制解调器风格的开发到新学校，刚从大学毕业，得到所有新的铃铛和口哨开发。

现在，如果我们能想出一些超级史诗般的话题，我们就可以了。
