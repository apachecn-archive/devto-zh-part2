# 坐在电脑前如何避免腰酸背痛

> 原文:[https://dev . to/nathan _ Nico 27/如何避免坐在电脑前背痛-2cg4](https://dev.to/nathan_nico27/how-to-avoid-back-pain-while-sitting-in-front-of-computer-2cg4)

使用电脑是颈部和背部疼痛的首要原因。尝试这些基本的调整来改善你在办公室的姿势，并停止脖子和背部疼痛。

[![](../Images/0c2aedf773a87b68cab415468c5892c3.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ZXPFYzSc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/s5oe7ronlb6u55f03k8l.jpg)

**抬起头来**

专注于将你的头和脖子对准肩膀上方；防止过度紧张。

**选择你的椅子**

选择一个能让你的下背部放松的，而不是背部支撑。然后倾斜椅背，让它真的有点倾斜。

**坐在伸手可及的地方**

你的上半身应该与显示器保持一臂的距离，应该在眼睛上方 2 到 3 英寸。

**站稳脚步**

将它们平放在地板上，与肩同宽，以减轻膝盖和脚踝的压力。

**找到最佳角度**

双膝呈 90 度，位于脚踝正上方；这将使你的脊柱保持直立。

**伸展你的肩膀**

将它们向后和向下卷，并在臀部上方成方形。想象你把一个盘子顶在头上保持平衡，而且你不想让它掉下来。

**不要跷二郎腿**

盘腿而坐会让你很难保持脊柱正确，肩膀也很难保持平坦，而且你可能会过度拉伸臀部周围的肌肉，通过中断血液流动增加血管曲张的风险。所以不要交叉和放松。

**从肚子里吸一口气**

每次吸气时，考虑将肚脐拉向背部；这涉及到核心肌肉群以及支撑上半身。

**休息时带上床垫**

当你在床上休息时，试着选择合适的床垫，如 Lucid 或 Zinus，以保持身体平衡，并仰卧。

**休息一下**

每小时至少起来一次去淋浴室或做一些肩部运动可以减少背部椎间盘的压力并增加流量。回报:你会更加灵活，压力也更小。
