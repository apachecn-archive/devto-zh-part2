## [](#about-us)关于我们

Nylas Cloud API 使公司将电子邮件、日历和联系人集成到他们的应用程序中变得更加容易。作为商务沟通、日程安排和联系的核心，Nylas 正在塑造人们工作方式的未来。

我们的[团队](https://www.nylas.com/team/)在确定的性别方面大致平等，我们积极并定期与整个团队合作，按照我们的授权、透明和善良的理想塑造我们的文化。

我们合作写下了我们的价值观、福利、津贴，并发表在我们的[开源手册](https://github.com/nylas/handbook)中。阅读它以了解更多关于 Nylas 的信息。您还可以在 https://www.keyvalues.com/nylas 的[了解更多详情。](https://www.keyvalues.com/nylas)

## [](#about-the-opportunity)关于机遇

我们想在 Nylas 雇佣我们的第一个合伙人工程师。你将负责我们客户的技术成功，帮助他们集成和挖掘 Nylas API 中的技术问题和错误。你将直接与我们的合作伙伴和 Nylas 的客户成功团队合作，调查这些系统中的错误，然后修复它们。

我们的开源 Python 同步引擎定期在一个大规模 SQL 集群中归档万亿字节的数据，我们的 Flask APIs 每天处理数千万个请求。您的工作是快速识别、修复并防止问题在该架构中发生。通过改进我们的测试基础设施和建立在客户之前快速识别问题的机制，您将对我们系统的可靠性和可伸缩性产生直接影响。您将直接影响我们客户的幸福感，因为您将解决他们最关心的问题。

你应该有一个成长的心态，有管理自己项目的记录，在日常工作中有很强的自主性。你也应该对自己的能力保持谦逊，并有一种与生俱来的将知识传授给他人的愿望。
