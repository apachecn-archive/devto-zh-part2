# 作为社交媒体的道歉

> 原文：<https://dev.to/egryaznov/in-apology-of-social-media-----56m2>

[![](../Images/080a1b8760c0ecaa61fbc4130def4afd.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--m4XQxq1M--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ou9jfwlpr82f705bmf1g.jpg)

互联网上有一个相对较新的趋势，那就是与社交媒体“断开”。这一运动的追随者认为，所有的社交平台都是“邪恶的”、“不好的”，或者至少有损于你的精神健康。考虑到最近脸书的惨败，这对任何人来说都不足为奇。

在这个 21 世纪大规模数字外流的时代，我敢说:

```
 Social media are not evil, immoral or bad; the way we use them is. 
```

在你开始向我扔你的智能手机之前，我想解释一下我自己。社交媒体本身没有任何问题。这是一项极其强大的技术，通过它你可以和数百万不同的人联系。而且，像任何其他技术一样，它有可能极大地影响我们的生活，但这种影响是积极还是消极完全取决于我们。这种“双刃剑”是任何新思想或发明的内在品质。我们的世纪，虽然才刚刚开始，已经有大量的例子支持这种说法。想想*抗生素*，它在几十年前彻底改变了医学，但却产生了所谓的“超级细菌”，一种几乎可以在任何治疗中存活的强大细菌，我们不知道该拿它怎么办。从第一部拍照手机到第一次自拍相关的死亡，仅仅过去了 11 年。不要忘记朝鲜最近进行的核试验，结合其他因素，已经将*末日时钟*调至 23:58，这是目前为止最接近午夜的时间。

它的寓意是，作为人类，我们总会找到滥用任何新技术的方法，直到它的存在受到质疑。这种情况以前在我们身上发生过很多次，现在在社交媒体上也在发生，而且还会继续发生。随着虚拟现实和自动驾驶汽车成为主流，你会听说饥饿的人们把自己娱乐到筋疲力尽，然后开着虚拟现实饿死。但是你猜怎么着？这是自然的。

这就是我们测试和推动事物发展的能力，这样我们就能更好地理解和使用它们。我们是好奇的动物，我们喜欢把东西拆开，只是为了看看它们是由什么组成的。我们喜欢漫游到伟大的未知，荒野的黑暗，海洋的深处，高山的高处，宇宙的浩瀚。它促使我们扩展到整个世界，在最艰苦的条件下居住在地球上最严酷的地方。这不仅仅是一种好奇，而是出于对了解现实本身的基本渴望。这是我们成为人类的原因。

首先你有一项新发明，然后它进入社会，之后它成为一种技术，随着它越来越受欢迎，人们开始越来越频繁地滥用它。最后，就在它被虐待的顶点之前(想想*末日时钟*的*午夜*)，我们意识到我们的困境，感到内疚，抑制我们贪得无厌的野心，并开始实施措施来修复我们无知的行为。最终，一切都稳定在生产率的*平台*。更重要的是，如果你在 y 轴上描绘虐待的程度，在 x 轴上描绘时间，那么这种模式将非常类似于*炒作曲线*，它显示了我们的*期望*，而不是我们对特定技术的误用。

[![hype curve](../Images/3b516b389e0d7b32a5c13bc2adec93af.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--08FI_owQ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://upload.wikimedia.org/wikipedia/commons/9/94/Gartner_Hype_Cycle.svg)

在社交媒体的背景下，这意味着我们刚刚到达*社交末日时钟*的午夜，事情将从这一点开始平静下来。有了社交网络，我们将达到所谓的*稳定期*，那时我们将确切知道*健康使用互联网*这句话的真正含义。界限将被建立，墙将被竖起，所以没有人，即使是大公司也不敢再跨越它们，因为我们已经看到这条路通向哪里。

随着所有这些关于社交媒体内在危险的骚动，我们似乎忘记了它实际上是怎么回事。它应该把我们团结在一起，而不是分成不同的团体，对彼此零容忍。我们应该更好地理解我们的同胞，而不是让自己在别人精心挑选的时刻受到持续的嫉妒和怨恨。

综上所述，我们都同意，我们与社交媒体互动的习惯必须永远改变。但是简单地“选择退出”并不是改善这种状况的好策略。如果我们不使用不适应的应对机制，而是反思我们使用社交媒体的方式，并试图了解到底哪里出错了，会怎么样？

不，我并不是建议你回到脸书或推特，或者继续保持你的网上*角色*而忽略真实的自己。不喜欢脸书？我也没有！但是你猜怎么着，外面有一个完整的互联网，在那里你可以加入各种各样的社区，讨论地球上(甚至地球以外)的几乎所有事情。找到最吸引你的东西，发现新的想法、地方和人。

如果你和我一起乘坐，这里有一些快速提示，让我们的旅程更愉快:

*   设定目标。明确并写下你想对你的新账户做什么。

*   具体点。加入小社区，那里的成员和你有共同的兴趣。它可以是从语言学习到生物收集的任何东西。

*   要更真实。摄影是一种异化的记忆。任何一幅画都已经是一个谎言，因为没有一个肖像能完全真实地代表它所代表的东西。那么，为什么要用这些背信弃义的过滤器更加扭曲现实呢？

*   还有最重要的一条:不要在这个过程中迷失自己。

祝你好运，网上见。
