# 域名查找器

> 原文:[https://dev.to/ahreflink/domain-name-finder-48mn](https://dev.to/ahreflink/domain-name-finder-48mn)

每当我想做一个新的周末项目时，我厌倦了花费数小时试图找到一个好的域名，所以我决定做点什么。

我的最新项目，ahreflink.com，有一个[域名搜索器](https://ahreflink.com/domains/name-finder)，可以在几秒钟内轻松搜索数百个域名。

其中一个关键特性是能够添加域名组合，这使得查找类似 FireFox、WordHippo、MailChimp 等域名变得非常容易。

该网站还可以让您非常轻松地指定您感兴趣的顶级域名，然后同时查找每个 TLD 的多个域名，无论有无组合。

我对它目前的功能感到非常满意(我已经用它进行了所有的域名搜索)，但我一直在寻找新的改进方法。例如，在一些色觉缺陷的人强调区分红色和绿色的困难后，我添加了“隐藏不可用域”按钮。

任何反馈或功能要求将不胜感激。我对实现一切都持开放态度，只要它与领域相关。我确实计划让 ahreflink 也支持其他技术主题的工具，比如 seo、vpn、优惠券等。但是在继续下一件事之前，我想构建更多与领域相关的特性。
