# 《我的世界》你的 AWS

> 原文：<https://dev.to/veermanhas/minecraft-your-aws--4a08>

记得《我的世界》吗？这是由于新的界面和它允许你建立和创造任何可以想象的东西的方式而爆发的游戏之一。

我们对 TotalCloud 有着相同的愿景——让您只需轻点几下鼠标，就能为您的 AWS 云基础架构操作一切可以想象的东西。

TotalCloud 目前仅适用于 AWS。

它是如何工作的？

在《我的世界》，你可以从现有的世界中开采资源，创造一个新世界。同样，在 TotalCloud 上，您只需点击两下鼠标，就可以断开(操纵)您的云资源与 IT 基础设施的连接(就像在《我的世界》采矿一样)。听起来很简单，对吧？确实是！

一旦您的清单中有了资源，您就可以添加、删除或做任何能想到的事情。这就是 TotalCloud 基于游戏引擎的原因之一——为您的 AWS 提供《我的世界》体验。
