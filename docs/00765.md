# 高级 JavaScript 面板

> 原文:[https://dev . to/BG Adrian/advanced-JavaScript-panel-talks-4b5o](https://dev.to/bgadrian/advanced-javascript-panel-talks-4b5o)

> 之前发表在我的博客[https://coder . today/advanced-JavaScript-2017-talks-FCC 3717 a 9 f 3 f](https://coder.today/advanced-javascript-2017-talks-fcc3717a9f3f)。它包含了 2017 年 JS 会议上一些关于内部和事情如何工作的最佳(在我看来)演讲。即使它们是“旧的”,它们也不会过时，希望你能学到很多关于你最喜欢的语言的新东西！

JavaScript 的成功让互联网充满了低入门水平的资源、介绍和教程。我没有抱怨，但结果是很难找到关于 JavaScript 的高级主题。

我看了很多 YouTube 视频，所以我决定与你分享我找到的关于 JavaScript 高级主题的最新视频。让我们开始吧:

## V8，高级 JavaScript，&下一个性能前沿(Google I/O’17)

这个视频将很好地介绍浏览器的 JavaScript 引擎，它是如何工作的，以及有哪些优化技术。在第二部分，他将深入细节并解释新的 V8 引擎，为什么&如何制造，以及它将如何改进我们的节点和客户端应用程序。性能与快速启动、低内存与最大优化。

[https://www.youtube.com/embed/EdFDJANJJLs](https://www.youtube.com/embed/EdFDJANJJLs)

## 使用 WebAssembly 为 Web 编译(Google I/O’17)

有许多关于 WebAssembly 的面板，但这一个现在更相关，我认为它更好地解释了我们将如何&为什么在 JavaScript 应用程序中使用本地库。您还会发现使用 WebAssembly 的局限性。

[https://www.youtube.com/embed/6v4E6oksar0](https://www.youtube.com/embed/6v4E6oksar0)

## Arindam Paul—JavaScript VM internals、EventLoop、Async 和 ScopeChains

如果你跳过前 5 分钟和难以理解的英语口音，你会学到很多关于 JavaScript 运行时间、内存模型、作用域链和闭包的知识。

[https://www.youtube.com/embed/QyUFheng6J0](https://www.youtube.com/embed/QyUFheng6J0)

## 节点的事件循环由内而外由萨姆·罗伯茨、IBM

这是对 NodeJS 事件循环非常技术性的一瞥。找出是什么让 JavaScript 在服务器端“滴答”作响。

[https://www.youtube.com/embed/P9csgxBgaZ8](https://www.youtube.com/embed/P9csgxBgaZ8)

## Denis Radin:将 NASA 编码标准应用于 JavaScript | JSConf EU 2017

你会驾驶由 JavaScript 运行的飞机吗？您将了解一些科学设施在使用 JavaScript 创建一个非常稳定和安全的系统时所学习和应用的规则。他们与 NASA 合作&欣赏干净、快速和稳定的代码。

[https://www.youtube.com/embed/z8hG-3Ak_b4](https://www.youtube.com/embed/z8hG-3Ak_b4)

## JavaScript 中的高级异步和并发模式

这只是并发主题中的一个介绍，整个故事持续了许多小时，但这是一个很好的总结。Kyle Simpson 讲述了他对 JS6 中承诺、生成器、异步和其他很酷的新东西的想法。

[https://www.youtube.com/embed/Qg1SvpIau6U](https://www.youtube.com/embed/Qg1SvpIau6U)

## Anjana Vakil:函数式 JS 的不可变数据结构| JSConf EU 2017

不变性在 JavaScript 中不是一个新话题， [immutable.js](https://facebook.github.io/immutable-js/) 已经存在很长时间了。但是我认为大多数初级到中级的 JavaScript 开发人员不使用这种范式，因为他们不理解它，并且你不理解你害怕的东西。我也认为她是一个很好的演讲者和一个很好的小组。您将学习函数式编程中使用的一些技术，它们将提高您的代码质量！

[https://www.youtube.com/embed/Wo0qiGPSV-s](https://www.youtube.com/embed/Wo0qiGPSV-s)
