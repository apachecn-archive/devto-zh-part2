# 如何使用 Github 上的开源库改进网站的最佳想法

> 原文：<https://dev.to/vitalykuprenko/best-ideas-how-to-improve-a-website-using-open-source-libraries-on-github-2l1i>

JavaScript 库对于开发者和设计者来说是一个有用的工具，因为它们可以通过使设计更加智能来为 web 应用程序或网站带来真正强大的功能。每天都有越来越多的新替代品出现在 GitHub 这样的开源服务中。如何选择值得你关注的图书馆？本文分析了最有效的免费 [JS 库](https://www.cleveroad.com/blog/discover-top-javascript-libraries-to-keep-an-eye-on-in-2017)。来自这些来源的信息和技术将使你的网站更有趣和受欢迎。

## D3。射流研究…

[![D3.js](../Images/921ee069591e8fffe47f653fd8dde150.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--EiyR4TR5--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/6ytl59xr0oennirnda12.png)

[D3.js](https://github.com/d3/d3) 通常被称为最强大的数据可视化开源 JavaScript 库。它与 SVG 合作来呈现图形、图表和动态可视化。不久前，它还被《纽约时报》、《Weather.com》、优步等网站广泛使用。

D3 符合几乎所有网络浏览器都支持的 W3C 网络标准。尽管它不包括一些初步构建的图表和图形，但它提供了大量的示例和诸如“输入”和“输出”等有用的功能。它还处理调试和强大的动画转换。

## 创建。射流研究…

它是一个 JS 库工具包，带有开放源代码和工具，用于为开放 web 技术开发丰富的交互式内容。

[Create.js](https://github.com/CreateJS) 包括:

**Easel.js** 用于处理 HTML5 画布元素。

该库使得在 HTML5 中创建高性能的交互式 2D 内容成为可能。它提供了多功能显示列表来显示图形。它还为感官和鼠标交互提供了可靠的交互模型。除了对游戏有益之外，它也是艺术创作、广告、数据可视化和其他高度发达的图形体验的完美选择。它可以独立工作，也可以作为 Create.js: SoundJS、PreloadJS 和 TweenJS 的一部分。由于没有外部依赖性，它几乎可以与任何框架兼容。

**Tweet.js** 网页动画开发

库的 API 是非常简单但非常强大的工具，允许用户轻松地创建复杂的动画。它支持对象的数字属性和 CSS 样式属性的动画。

**Sound.js** 处理数字媒体

这个 JavaScript 库是为处理音频而开发的。它为在各种浏览器和设备中加载音频和音频播放创建了一个一致的接口。目前，它支持 WebAudio、HTML5 音频、Cordova/PhoneGap и Flash Fallback。

**Preload.js** 协调资产的加载

它代表了加载不同类型文件的统一 API，自动检测 XHR (XML-HTTP-Request)可用性，作为扩展下载的替代选项。它包括插件模型，以帮助在其他库中下载，如 SoundJS。

## 图表。射流研究…

[Chart.js](https://github.com/chartjs/Chart.js) 是一个开源代码的库，用于使用 JavaScript 和 Canvas API 开发图形、条形图、图表和其他线性图形元素。它代表 8 种图表类型，并允许混合使用。此外，它还为动画提供了惊人的现成选项。它可以轻松安装和定制。

## 三。射流研究…

## gsp

[![GreenSock Animation Platform](../Images/48beabacbd7a562d59b6378d86cf477e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--4iwyNBkd--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ull7lm7rpi1e2cuz3x1c.jpg)

GSAP(green sock Animation Platform)是一款先进的动画解决方案，受到福特、耐克、Adobe、微软、三星等全球大型企业的高度评价。

其特点是入门门槛低，文档详细，性能高，灵活性强，兼容老浏览器。GSAP 有一个重要的优势——这个库不仅可以动画显示 CSS 或 Canvas，还可以动画显示任何 JS 对象的任何数值变量。

另一方面，GSAP 是相对沉重的，因此，最好是使用其他模拟创建一个小的 HTML 元素动画，如外观，消失，翻转等。

## Bounce.js

[![Bounce.js](../Images/fb591a5f574eb85809b3e9ff5542ea46.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--3kJvGI0A--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/2ti9xnwoy0p2i4n6h05x.jpg)

JS 库，用于使用 CSS 开发令人印象深刻的动画。它允许应用到监视器的各种类型的运动，旋转，可伸缩性和转换的基础上的 CSS3。该库可以在所有支持 3D 转换和关键帧动画的浏览器中运行。

这个库是免费的，并且拥有麻省理工学院的许可，如果你不需要一长串的动画类型或者只是想尝试视觉艺术，它是其他库的一个很好的替代品。

在 [GitHub](https://github.com/tictail/bounce.js) 上找到库。

## Math.js

[Math.js](https://github.com/josdejong/mathjs/) 是 JavaScript 和 Node.js 的大型数学库，它包括数学表达式的定义，支持符号计算。除了包含大量的集成函数和常数，它还提供了处理不同类型数据的集成解决方案，如大数、复数、分数、单位、矩阵等。
