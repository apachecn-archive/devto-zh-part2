# 变更日志:AMA 更加异步，对所有人开放

> 原文：<https://dev.to/ben/changelog-ama-more-async-and-open-to-all-1icp>

以前，我们将 [#AMA](https://dev.to/t/ama) 标签作为“需要批准”标签。我们也把它当作一个“现场”活动，有趣的人来回答大约一个小时，然后离开。

我们现在正在改变这些，以便向任何人开放，并保持更长时间。所以如果你开始了一个，当你有机会的时候，保持回答一两天。只要你坚持回答，它会在主页上停留几天。

这是 AMAs 现在将有一个家的侧边栏小部件。
