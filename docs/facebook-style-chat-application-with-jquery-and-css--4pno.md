因此，这里我们将聊天滑块布局和聊天弹出框布局设计集成在一个 HTML 文件中。当用户点击聊天滑块框中的用户名时，它会像脸书应用程序一样显示相同的用户弹出框。

此聊天框应用程序的功能:

1.  从聊天滑动框中轻松选择用户。
2.  有了它的帮助，你可以在一个网页上进行多人聊天。
3.  您可以根据需要随时关闭聊天会话。
4.  您可以最小化聊天弹出框。
5.  最新的聊天框，应该显示在右上角。
