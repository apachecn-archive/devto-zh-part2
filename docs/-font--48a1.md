# 第一次买帐的经验

> 原文：<https://dev.to/armariya/-font--48a1>

上周买下了第一个写代码的字体所以我想把 555 年的经验放在这里

首先，我是一个对 Font 非常着迷的开发人员，他写了一个代码，因为他觉得 Font 在我们的生活中，浪费了很多时间。

自从我工作以来，我就换了很多字体(这还不包括 Syntax Highlight)，从 Ubuntu mono，Input，Iosevka，Hack，leke)。

但几天前，我厌倦了使用它的 Font，于是我通过在 Twitter 上的搜索进入了一个新的 Font 编码。它遇到了一个有人做的 font。
