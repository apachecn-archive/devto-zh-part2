# Changelog:对评论框的更新

> 原文：<https://dev.to/theoutlander/changelog-updates-to-the-comment-box-3f04>

你好 DEVs！评论框中现在支持一些热键。所有命令都将接受 **Ctrl** 或 **Cmd** 作为控制键。另外，控制键后面的字符不区分大小写。

[![demo](../Images/c55c7a800f0377ee705d55bcd509c577.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--6knw1tdF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/r9bu3vudrnhxikn40yuo.gif)

# **加粗**

您可以简单地突出显示文本，然后按下 **Ctrl+B** 或 **Cmd+B** 来使文本加粗。

# *斜体*

同样，你可以突出显示一个单词，然后按下 **Ctrl+I** 或 **Cmd+I** 将文本变成斜体。

# [链接](https://dev.to/p/editor_guide)

按下 **Ctrl+K** 或 **Cmd+K** 会在高亮显示的文本周围加上一个链接标签，并将光标放在需要输入 URL 的位置。确保在它前面加上 **http(s)://** ，否则它会把它变成一个相对于 dev.to 的 URL

# 热键提交

您可以按下 **Ctrl+Enter** 或 **Cmd+Enter** 来提交您的评论，而不必点击提交按钮。
