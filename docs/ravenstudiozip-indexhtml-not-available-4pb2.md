# 渡鸦。studio . zip Index.html 不可用

> 原文：<https://dev.to/rkosta/ravenstudiozip-indexhtml-not-available-4pb2>

我在 aspnet.core web api 中嵌入了 RavenDb，当我试图访问 Raven Studio 时，出现了这个错误:以下文件不可用:index.html。请确保乌鸦。Studio.zip 文件存在于主目录中(在 Raven 附近。Server.exe)。

解决方案是复制 Raven。Studio.zip 到 web api 的根目录。
