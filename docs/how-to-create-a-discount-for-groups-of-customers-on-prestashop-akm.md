# 如何在 Prestashop 上为客户群创建折扣

在 Prestashop 上创建客户团体折扣非常简单。例如，您希望为那些在您的网站上注册的用户设置折扣。默认情况下，你可以很容易做到这一点。但是**如何为一群顾客创造和应用限时优惠**？

事实上，这个电子商务平台可以让你给你的客户一定的特权，通过分配他们到组。您可以根据需要创建任意多个客户组，并将客户分配到任意多个组。

但是你的顾客需要的是限时优惠，而不是不间断的折扣！。这就是为什么我们需要改变一些东西来适应我们在这个 Prestashop 教程中的目标。

为什么你应该提供折扣

顾客天生就被打折商品所吸引。如果有两种产品在功能上非常相似，消费者几乎总是会选择价格更低的那一种。这种商业策略作为一种短期和偶尔的策略来促进销售，对小企业来说利大于弊。

电子商务店主不应该害怕提供折扣的原因有 5 个:

增加客流量——在有限的时间内提供折扣将会吸引顾客进入你的商店，很可能会吸引他们立即购买这些商品。销售额增加——更多的流量通常伴随着销售额的增加。顾客很可能会在你的店里逛逛，看到其他可能没有打折的商品，但他们愿意购买，因为他们已经在其他打折商品上省了钱。

实现销售目标——考虑到消费者购买行为的季节性波动，你可以通过提供折扣来抑制淡季。这样，无论你的预测如何，你都能在每个时期内实现你的目标。更强的客户关系——让你的顾客为高质量的产品获得巨大价值，可以极大地提高他们对你的商店的忠诚度。
货币化库存-打折商品可以让你处理掉旧库存，甚至是你不再打算出售的商品。

限时优惠

不要称自己为小企业主。当你称某人为小企业主时，你马上会说，“哇，他是个男孩”。你有你的梦想，你知道限时优惠是一个强大的营销工具。

消费者有一种强烈的愿望，想赶在错过机会之前买到特别的东西。在 Prestashop 网站上提供限时折扣促销是个好主意。

在限定时间内为客户群创建和应用折扣

首先，您需要安装[售前折扣规则模块](https://www.prestasoo.com/prestashop-modules/soo-prestashop-discount.html)

然后，访问管理面板左侧的 Prestashop 折扣规则->管理折扣规则->新折扣规则。
