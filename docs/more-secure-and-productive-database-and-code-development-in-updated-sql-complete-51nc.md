# 在更新的 SQL 中完成更安全、更高效的数据库和代码开发

> 原文：<https://dev.to/devartsoftware/more-secure-and-productive-database-and-code-development-in-updated-sql-complete-51nc>

Devart 是为开发人员和 DBA 提供专业数据库工具的知名供应商，它发布了一个新版本的 dbForge SQL Complete，这是一个用于 SQL Server Management Studio 和 Visual Studio 的插件。现在，SQL Complete 提供了连接到 Azure 多因素身份验证的可能性，从而确保最高的安全标准。在更新的 SQL Complete 中实现的其他改进包括增强的代码完成和代码重构功能，以及许多旨在简化 SSMS 结果网格中的数据管理的生产力功能。

下面是对 dbForge SQL Complete 中一些新特性的详细介绍:

两步身份验证带来额外的安全层
为了在为最终用户提供便利的同时提供更安全的数据和应用访问，SQL Complete 现在提供了 SSMS 17.2 版的新身份验证方法，“活动目录-通用 MFA 支持”，包括电话、短信或移动应用代码。

二。代码完成
代码完成特性现在包括以下新的语句建议:
TEMPORAL _ HISTORY _ RETENTION
HISTORY _ RETENTION _ PERIOD
IDENTITY _ CACHE 和自动数据库调整

三。代码重构
在更新的 SQL Complete 版本中，用户可以使用一个方便的重命名对话框窗口，在眨眼之间重命名 UPDATETEXT、WRITETEXT、READTEXT 和 REVERT 语句中的变量。

四。简化 SSMS 结果网格中数据管理的生产力特性:
结果网格数据可视化工具，通过十六进制、文本、XML、HTML、富文本、PDF、JSON、图像和空间视图实现结果网格的无缝数据可视化

2)能够从 SSMS 网格复制 XML、CSV、HTML 或 JSON 文件格式的数据

3)将任何可用文件格式的数据复制到文件或剪贴板的能力

4)分配到 SSMS 网格弹出菜单的快捷键
