# GitHub 面向学生的促销活动

> 原文：<https://dev.to/delphin10/github-promotions-for-students-29p9>

#### 对于那些不知道 GitHub 教育特别优惠的人来说，我认为这是一个不用花一分钱就可以完成更多事情并获得经验的时候。

这个想法非常简单，出于教育目的和学生面临的经济困难，他们有时很难负担得起他们需要的工具，以获得现实世界开发工具的经验，因此 GitHub 提供了一个名为**“GitHub 学生开发包”**的促销活动，其中包括免费的无限私人存储库(通常为 7 美元/月),当你是学生时，还可以从他们的合作伙伴那里获得更多。

[![alt text](../Images/75e0e94c35549d8b3c9d69fbc6d97687.png "Free GitHub Private repos")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--FXcdsUOR--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ogiy7mjsssgltnxogozr.png)

### 它是如何工作的？

> 你所需要的只是一个 GitHub 账户和一个可验证的学校颁发的 T2 电子邮件地址。
> 点击[此处](https://education.github.com/pack)申请此套餐。

### 该产品包中一些最有趣的产品:

*   **$75-$150** 的信用额度用于 **AWS** 服务。![alt text](../Images/37d7af9bab01eb4eee3d956b50be320e.png "AWS Educate")
*   ****数字海洋**新用户平台积分 5** 0 美元。![alt text](../Images/dd847b1c1523c7c3186bfe9a0a5cc791.png "Digital Ocean credit")
*   在**条**上处理的第一笔 **$1000** 收入免除交易费用。![alt text](../Images/72c5b7405096d9a9731d6332a938ba34.png)

希望这能帮助你开始你的下一个项目！
