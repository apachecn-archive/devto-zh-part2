# Magento 产品评论扩展

> 原文：<https://dev.to/fmeaddons/magento-product-reviews-extension--1kfa>

[![](../Images/6ab9a4baaa5e67a7e4283660405d8386.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--XgE01Vcg--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/0efequxua81ytvwg3zkl.png) 
Magento 产品评论扩展允许您在产品页面上显示客户评价。它启用产品页面上的评论和评级按钮，引导客户进入评级表单。 **[Magento 产品评论](https://codecanyon.net/item/magento-product-reviews-rating-extension-with-review-reminder/21162835)** 扩展首先显示评分最高的评论，同时也显示平均产品评分。顾客可以根据价值、质量和价格对你的产品进行评级，当新访客看到你的产品评级时，他会毫不犹豫地购买你的产品。这个 Magento 插件的价格只有 45 美元。
