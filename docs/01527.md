# 介绍我们 2018 年 11 月的赞助商

> 原文:[https://dev . to/dev team/introducing-our-November-2018-赞助商-355b](https://dev.to/devteam/introducing-our-november-2018-sponsors-355b)

感谢我们出色的赞助商，他们帮助维持了 [dev.to](https://dev.to/) 生态系统和普通开发者社区。

这三个开发者平台都是各自领域的领导者，如果你还没有这样做的话，我强烈建议你去看看他们的产品。说真的，现在花几分钟时间浏览一下他们的产品指南和文档。

## [](#digitalocean)数字海洋

[DigitalOcean](https://try.digitalocean.com/performance/?utm_source=devto&utm_medium=display&utm_campaign=Devto_2018_Brand) 是一个备受喜爱的云计算平台。我总是对数字海洋核心产品的良好构建*和良好记录*印象深刻。说起来容易做起来难，他们已经能够保持领先。

迄今为止，DigitalOcean 一直是我们最长期的支持者，我们非常感谢他们的一贯支持。他们对开发者生态系统的承诺非常明确。

## [](#twilio)Twilio

自从[发展到](https://dev.to/)以来，Twilio 一直是我们的好朋友。我们是他们 API 的用户，我们团队的成员总是很乐意参与他们的社区活动。有他们做赞助真是太好了！

我个人可以说，我们选择 Twilio Video 是因为它充分利用了本地对等浏览器技术，同时还为包装在同一 API 中的必要用例提供了 Twilio 的基础架构。文档和示例组织得很好，启动和运行是同步的。

## [](#raygun)雷枪

[Raygun](https://raygun.com/?utm_source=dev.to) 是一个错误、崩溃、监控软件服务。他们的产品以真正的可用性为中心，包括我们在 web 可观察性实践中经常忽视的前端性能问题。

在这一领域，服务的一致性和易用性至关重要，Raygun 已经赢得了良好的声誉。如果你想改善你的监控，一定要去看看。

* * *

这些组织正在帮助我们尽可能建立最好的社区。如果你想购买他们的服务，我建议你去看看他们的产品。
