# 最难的是坚持下去

> 原文 https://dev.to/carlymho/the-hardest-thing-is-to-keep-going-27ee

或者:献给年轻的自己，带着爱。

听着:事情会变得艰难，然后会变得容易，然后又会变得艰难，因为你已经获得了想象如何变得更好的技能。甚至当你在学习的时候，你会想:以前这不是更容易理解吗？或许是，或许你正在学习越来越难的东西。

有时，冒名顶替综合症来自内心——不是与你的同事、同学和你在 Twitter 上关注的硅谷名人相比，而是与你十岁时的自己相比，与某人发现新世界的热情以及她将自己的意愿强加于世界的方式相比，甚至与你现在认为自己应该成为的样子相比。我们往往对自己最没有耐心。

不断学习，尽量不要把自己锁在你现在能看到的未来里。你认为你永远也不可能成为一名经理，但是那些技能是你通过练习也会更容易掌握的东西，就像代码一样，这些技能同样重要。能够估计，对你的实现想法有信心，能够交流——这些也是展示工艺掌握的事情。但是在你准备好之前不强求也没关系。

你使用的工具可能会变，和你一起工作的人可能会变，但不会变的是，这项工作会召唤你，驱使你成为最好的自己，有一天，它会让你感觉自己像个专家。总有一天你会成为专家。有一天，你会自信地说“我不知道，但我肯定能想出来”，这就会成真。

当你看不到前方有什么，也不知道回报是什么时，最困难的事情就是继续前进。我可以告诉你的是，如果你坚持下去，如果你找到朋友、同龄人和导师来支持你(你会的)并照顾好自己(这很有帮助，我保证)，你最终会发现自己在一个地方，在那里你会意识到你拥有你需要的一切。

总有一天，一切都感觉很好，你会对自己的工作感到满足，但仍然觉得自己更喜欢躺下来，因为(不管是好是坏)你不是编程知识的无形能量云，你的身体刚刚决定反抗。有时工作会很棒，但全国某个地方的某个你甚至从未听说过的家伙会释放出一个关于像你这样的人是否应该成为程序员的热镜头瀑布，你只会觉得*非常累*。给自己一个机会，做你需要做的事情来重新站起来。头脑和身体的需求是你被允许拥有的需求。生产力可以随后到来。你还有很多时间。

你永远不会完成学习或提高，你永远不会得到一个奖杯，上面写着你要完成，从现在开始一切都会完美顺利。但是就像所有困难的事情一样，通过练习，坚持会变得更容易。
