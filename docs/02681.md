# 2018 年如何学习 Vue.js

> 原文:[https://dev.to/kball/how-to-learn-vuejs-in-2018-51ao](https://dev.to/kball/how-to-learn-vuejs-in-2018-51ao)

Vue.js 是 web 前端领域最热门的话题之一。

Vue 已经超越 React 成为 Github 上最受瞩目的 JavaScript 框架。

这也是大多数 JavaScript 开发者[表示他们想在 2018 年](https://stateofjs.com/2017/front-end/results/)学习的框架。

大问题是:**如何**学习 Vue.js？

最好的资源是什么？你应该阅读什么资料，应该上什么课，等等？

## 学习方法

在深入探讨可用资源之前，让我们先简单谈谈我们是如何学习的。

如果你真的想学习 Vue.js，我强烈建议你选择一个项目来应用它。

不要只是做玩具项目，或者跟着教程走——找到你真正想完成的事情，然后开始用 Vue 去做。

这将让你在挑战中前进，并迫使你探索教程作者和课程教师所认为的狭窄界限之外。

它还会帮你找到一些更晦涩的话题的*相关性*。许多更高级的 Vue 概念(比如[多根组件](https://dev.to/kball/creating-multi-root-vuejs-components-23j0))对我来说没有意义，直到我在一个需要它们的项目中遇到一个问题。

## 入门

开始使用 Vue 最简单的地方是官方指南。。事实上，如果你在谷歌上搜索“learn vue”，这是你首先会找到的东西之一。

然而，当我开始学习像 Vue 这样的新技术时，我通常会从免费或便宜的视频课程开始，给我一个概述。尽管官方指南很好，我还是觉得这些课程更容易找到。

## 初学课程不错

幸运的是，有许多适合初学者的好课程。

1.  **[【Vue JS 2】-完整指南(含 Vue 路由器&【Vuex】](https://click.linksynergy.com/deeplink?id=hIdOL5Z4eK4&mid=39197&murl=https%3A%2F%2Fwww.udemy.com%2Fvuejs-2-the-complete-guide%2F)**)。这是我开始学的课程——这是一门付费课程，但经常会打折到 10 美元左右。我强烈推荐。
2.  逐步学习 Vue 2(Laracasts)。另一个很棒的入门课程。我认为它在排序方面不如上面的课程做得好，但另一方面，它是免费的。
3.  **[Vue 简介。](https://www.vuemastery.com/courses/intro-to-vue-js/)JS(VueMastery)**。这是一个比上面两个都短的课程，我认为视频不是很好，但 VueMastery 做得很好，给你的不仅仅是一个视频。你也可以获得文本形式的课程，下载任何使用的资源，并参与基于 CodePen 的练习。哦，而且是免费的！
4.  **[终极 Vue JS 2 开发者课程](https://click.linksynergy.com/deeplink?id=hIdOL5Z4eK4&mid=39197&murl=https%3A%2F%2Fwww.udemy.com%2Fvuejs-2-essentials%2F)** 。或许被过分宣传为“终极”课程，但这是一门很棒的“边做边学”课程，它侧重于 3 个项目来介绍 Vue 的概念。如果你自己想不出一个项目，这可能是一个很好的课程，因为你可以跟着老师的项目走。

**[奖励初学学习机会:LearnVueJS.com](https://www.learnvuejs.com/)**

这是我刚刚推出的一个项目专注于更多 *live* 学习机会。大多数 Vue 学习机会都缺乏与导师进行现场问答的能力，所以我推出了这个项目。发布的第一篇文章是关于 Vue.js 基础知识的免费网络研讨会，更多内容即将推出。

## 高级课程

也开始有更高级的视频课程。几个好的选项包括:

1.  **[【VueMastery】](https://www.vuemastery.com/courses/advanced-components/the-introduction/)**。深入了解 Vue 的工作原理，并与 Vue 的创造者尤雨溪一起完成多门课程。一旦你开始感到舒适，想要更深入，这就是我要去的地方。
2.  **[【vue js 大师班(VueSchool.io)](https://vueschool.io/courses/the-vuejs-master-class)** 。这是一门以项目为中心的课程，从头开始构建一个完整的应用程序，涵盖了 Vue.js 的方方面面，由 Vue 核心团队成员、《Vue.js 的威严》一书的作者 Alex Kyriakidis 教授

## 继续学习

当你超越了“这东西到底是怎么工作的”的开始，你会想要扩大你用来学习的资源范围。

有许多资源可以用来获得一系列更高级的文章、教程和工具。

1.  **[Vue News](https://news.vuejs.org/)** ，一个播客和每周电子邮件简讯，重点介绍最新的新闻和教程。Vue 核心团队加持的官方消息来源。
2.  **[【Vuejsdevelopers.com】](https://vuejsdevelopers.com/)**，既是博客又是每周时事通讯，突出深度文章。
3.  **[Vue Feed](https://vuejsfeed.com/)** ，一个网站、时事通讯和 twitter feed，突出显示策划的 Vue 新闻、教程、插件等。
4.  **[CSS 招数](https://css-tricks.com/tag/vue/)** 可能已经开始关注 CSS，但他们已经积累了数量可观的高质量 Vue 文章。
5.  **[对 Vue 的看法](https://devchat.tv/views-on-vue/)** 是 devchat.tv 网络中的一个每周播客，主要关注 Vue，以许多核心团队成员为特色。
6.  **[Vue Land Discord](https://vue-land.js.org/)** 是一个专注于 Vue 的聊天室，在这里你可以提问，建立联系，学习。

## 包装完毕

如您所见，有大量资源可用于学习 Vue.js。

唯一的问题是-什么样的学习方式适合你，你会投入时间和精力去学习吗？

如果你擅长自驾学习，从官方指南开始。。

如果你喜欢视频课程，有很多选择，所以浏览一下初级课程，挑选一个看起来最好的。

或者，如果你喜欢更多的互动学习和向专家提问的机会，去看看 LearnVueJS.com 吧。
