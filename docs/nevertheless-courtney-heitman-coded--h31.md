# 然而，考特尼·海特曼编码

> 原文：<https://dev.to/courtneyheitman/nevertheless-courtney-heitman-coded--h31>

## 我开始/继续编码是因为...

我喜欢它。我喜欢那种进入心流/深度聚焦/不管我们这些天怎么称呼它的感觉，以及灵活运用我的大脑解决问题的感觉。

我 11 岁的时候自学了 HTML，那时还没有出现“开发是男人的事业”这种刻板印象。所以，当我去参加高中的一个 IT 俱乐部比赛时，我遇到了第一个因为我是女性而认为我不属于这里的人(他真的告诉我了)，我吃了一惊，然后我就想，“你知道吗:看我主宰吧！”

## 我最近克服了...

另一种程度的冒名顶替综合症。有时你必须停下来环顾四周，然后想“哦，我想我确实对[主题]了解很多。”我最近的话题是易访问性遵从和 scrum。

## 我想吹嘘一下...

被选中在 Magento Imagine 演讲！我生活在这个怪异的技术世界里，我既是 UX 的开发人员，也是 Scrum 大师，因此能够在来自世界各地的数百人面前分享作为 Scrum 大师我是如何在更大的范围内帮助促进 UX 对话的，真是令人兴奋不已。

## 我给盟友的建议是支持女性主义者的守则....

确保女性在会议中有发言权！在社会中，女性被社会化为不那么自信。当女性大声说话时，她们经常会被打断，所以要注意视觉暗示。如果有疑问，请询问他们的意见！
