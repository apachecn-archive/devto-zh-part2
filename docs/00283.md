# Filezilla 恼人的转移声

> 原文:[https://dev . to/vjnvisakh/filezillas-烦人-传输-声音-2d4a](https://dev.to/vjnvisakh/filezillas-annoying-transfer-sound-2d4a)

我知道现在没有人使用 FTP 传输，但我仍然这样做。这是我在转会期间非常恼火的事情。

因此，如果你是那些想要摆脱这种情况的人之一，请遵循以下建议。

**在 Windows 8 及以上版本上**

1.  按下 windows 键打开**通知和设置**。
2.  向下滚动到 Filezilla。
3.  点击它。不要从那里直接关机。
4.  在里面你会发现一个关闭声音的选项。
