Magento 2 规范 URL 扩展通过添加规范 Meta 标签，帮助您消除电子商务网站中的重复内容问题。您可以向产品、类别和 CMS 页面添加标签，以定义其与其他页面的相关性。向您的店面、商店视图或自定义 URL 添加规范标签，以声明您的网上商店的权威版本。将标准应用于分页和分层导航页面进一步阐述了产品和类别与搜索引擎的关系。以下是该产品的功能和链接。
**主要特性**
【易于在网站管理员想要的任何页面上启用或禁用规范化。
允许/不允许规范元标签对分层导航进行分类
排除某些页面以否决规范
为规范元标签添加尾随斜线
通过添加 rel="next "和 rel="prev "属性来显示分类和分类页面之间的关系

**更多详情和演示:** [Magento canonical url](https://www.fmeextensions.com/canonical-url-extension-magento-2.html)
