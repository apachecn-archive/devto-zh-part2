这些天出现的新语言令人惊叹，尤其是来自 C 和 Java 的语言。

由于它的静态类型，Go 速度非常快，编写起来也很愉快。

Rust 有一个我们都知道并且喜欢的语法，但是它有 c 语言的速度。

Dart 也很棒，多亏了 Flutter，我们可以用它制作漂亮的原生移动应用。跨平台也是如此。

但这并不是我刚刚提到的这些语言最令人兴奋的地方。还有一件事-他们更像是没有专家的区域。

没有人对这些语言了解很多(至少从程序员的角度来看)，所以如果你学习了它们，你就能脱颖而出。

事实上，我的第二份工作就是这么做的。那是 2016 年末，当时知道 Golang 的人很少。所以把 Golang 放在我的个人资料里是一件大事。一家公司联系我仅仅是基于这一点，当时可能有很多比我更好的个人资料。

也许 Golang 现在并不罕见，但你会明白的。

例如，Flutter 对于技术世界来说是非常新的。如果你能让自己成为 Flutter 领域的专家，比如开发几个优秀的 Flutter 应用，写一些博客文章等等，那么你就有很大的机会在这个广阔的开发者世界中获得一席之地，这要感谢 Flutter。

这也适用于库和工具包。比如谷歌最近推出的 ML 套件，以及不断壮大的亚马逊 Alexa。在这两种技术中的任何一种中做些工作，分享你的经验，你就会自动被认为是专家。

这太明显了，我不知道为什么大多数人不这么做。去找一个你感兴趣的新技术，然后从中学习。

这样做只会对你的事业有帮助。
