# JWT.io 和 JWT 调试器扩展的又一轮更新

> 原文：<https://dev.to/auth0/another-round-of-jwtio-and-jwt-debugger-extension-updates-4mgn>

我们已经发布了 JWT.io 网站和 JWT 调试器扩展(Chrome，Firefox)的新一轮更新。新特性包括 HMAC 秘密长度提示、常见声明描述、网站的简化库更新、网站的共享按钮、对普通 RSA 公钥的支持，以及可以在 URL 中传递更多令牌类型！

在本帖中，我们将回顾这些新特性。
