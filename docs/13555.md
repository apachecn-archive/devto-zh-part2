# 你的大脑和你的生活

> 原文:[https://dev.to/michaelgv/your-brain-and-your-life-5acb](https://dev.to/michaelgv/your-brain-and-your-life-5acb)

你有没有想过是什么驱使着你？你们中的大多数人会说不，你已经知道是什么驱使你成功了，但是你错了。你不知道是什么潜在的力量驱使你成功；事实上，这与你的想法无关。

从很小的时候起，我们就被告知要做一份待办事项清单，并在清单上勾掉我们能看到的任务。我们大多数人都不会，毕竟我们很懒。

让我问你这个问题，你最后一次做待办事项清单并在当天划掉事情是什么时候。我每天都做。我的同事问我是什么驱使我如此成功，所以我最后告诉他们:

多巴胺驱使我走向成功

是的，一种高度上瘾的化学物质驱使我成功。当你早上醒来时，你做的第一件事是什么？(对自己诚实！)

我是这样做的:

1.  我淋浴，穿好衣服
2.  我写我的待办事项
3.  我整理好床铺，把它从我的待办事项清单上划掉。)
4.  我吃了东西，准备去工作

起床不到一个小时，我就已经有了一些多巴胺来完成我每天待办事项清单上的一项。

你的清单是什么？这是我的一个朋友说的:

1.  醒来，查看我的邮件和短信
2.  玩几个游戏
3.  匆忙起床，淋浴，然后在车里吃东西
4.  工作

注意到这里有些东西看起来和你的程序相似吗？电话，是的，这是一个基本问题，也是为什么你有时会努力争取成功。

多巴胺是自然产生的，以下是四个主要原因:

1.  喝酒
2.  药
3.  投机
4.  手机

你认为这是个笑话，对吗？你的手机怎么可能是个问题并且会上瘾呢？*我没有上瘾，我也不认为自己有上瘾*

如果你做的第一件事就是查看手机，那你就上瘾了。如果你没有电话就不能出门，那你就上瘾了。如果你感到无聊，立即伸手去拿电话，你就上瘾了。手机、平板电脑、电脑、社交媒体在一定条件下都会释放多巴胺。有没有想过为什么脸书、推特、Instagram 和其他社交媒体巨头赚了几十亿？**多巴胺**——你收到了好友请求？多巴胺！你的照片在 2 分钟内获得 10 个赞？**多巴胺！**

多巴胺没有警告信号，它只是感觉很好，你渴望更多。所以你在手机或设备上花更多的时间去获取。

让我告诉你多巴胺对你也有什么作用，如果它不平衡，它可以:

1.  破坏关系
2.  与世隔绝
3.  限制创造性思维
4.  没有成就感

普通人在学校变得抑郁的最常见原因是...**多巴胺和社交媒体**——甚至不一定是欺凌。这种不平衡导致你情绪低落，因为你没有多巴胺。

## [](#how-do-i-escape)我怎么逃？

很简单，放下电话——去散步(多巴胺！)，和朋友在一个公园聊天(多巴胺！)，和老板喝咖啡(多巴胺！)，做一个待办事项清单，当你马上做项目的时候把它划掉(多巴胺！)

### [](#how-i-succeeded)我是如何成功的

我把我的手机放在一个锁着的盒子里，把钥匙给了我的妻子，一个星期没让她还给我，一个可怕的令人沮丧的星期后，我感觉很有成就感(是的，多巴胺来自于拿走我的手机！)，现在我出门不带电话，我花大量时间和儿子在一起，工作时我完全专注，能够跳出框框思考。

所以，让我们重新思考是什么驱使我们，用多巴胺的适当平衡来驱使我们自己。
