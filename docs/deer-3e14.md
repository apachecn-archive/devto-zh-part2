**Deer** 是一款现代、快捷、美观的笔记应用。在 Linux、Mac 和 Windows 上快速方便地写下你的想法和创意。

**鹿**建立在[电子](https://electronjs.org)的基础上，目标是

*   自由的
*   开放源码
*   方便用户的
*   安全的
*   快的
*   简单的
*   可定制的
*   桌面和移动的跨平台
*   多语言
