软件工程师在社会中享有崇高的地位，他们为我们的智能手机开发游戏，为我们愚蠢的工作开发专业工具，同时为他们的程序赚取可观的利润。他们把自己定位为梦想家、天才和摇滚明星，他们表现得好像他们的产品是从天上传下来的。硅谷认为它有知识和技能让世界变得更美好。

但是看看他们给产品取的名字你就不会知道了——有些名字很烂( [Zune](https://en.wikipedia.org/wiki/Zune) 、 [Chumby](https://en.wikipedia.org/wiki/Chumby) 、 [iMuffs](https://www.cnet.com/reviews/wi-gear-imuffs-mb210-wireless-headphones-review/) )。

玫瑰无论叫什么名字，闻起来都一样香(除了杂草)。或者会吗？硅谷是充满了无所不知的技术神，他们将解决我们所有的问题，还是他们是需要阅读一本书的浮夸的书呆子？让我们来看看著名技术和科技公司的名字，看看它们的起源。

谷歌最初被称为 BackRub(源自用于确定搜索结果的反向链接技术)，创始人谢尔盖·布林和拉里·佩奇意识到他们需要为他们新生的搜索引擎取一个新名字。他们举行了一次会议，有人推荐了“googol”作为名字，他们喜欢这个名字，有人拼错了“googol”，他们来到了(Google)[[https://graphics . Stanford . edu/~ dk/Google _ name _ origin . html]。](https://graphics.stanford.edu/%7Edk/google_name_origin.html%5D.%C2%A0)

“谷歌”现在是牛津英语词典中的一个动词——你能想象说“Siri，我附近的足底按摩医生？”

**Java** 最初以创始人办公室外的一棵橡树命名为“Oak”，后来以办公室厨房里的一些 Java 咖啡命名为“Green”，再后来命名为“Java”。其他命名创意包括“订书机”、“迈克”和“迈克桌上的希瑞玩偶”

Javascript 最初是一种名为 Livescript 的浏览器编程语言，Netscape 希望抓住 Java 的一些市场热点(Java 本身是以创始人看到的第三个对象命名的)。

Python 的创造者吉多·范·罗苏姆以 70 年代的一个小品节目 Monty Python's Flying Circus 命名。他的隔壁邻居编写了一个名为“Brian”的编程语言，引起了一些混乱。

Linux 的创造者 Linus Torvalds 以他自己的名字命名了这个流行的开源操作系统。
