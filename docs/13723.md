# 你觉得大括号和分号怎么样？

> 原文:[https://dev . to/17 cupsofcoffee/how-do-how-do-you-feel-about-brackets and-分号-2bpc](https://dev.to/17cupsofcoffee/how-do-you-feel-about-braces-and-semicolons-2bpc)

我正在(慢慢地)通过[制作解释器](http://craftinginterpreters.com/)来工作，并试图为我那小小的解释语言拼凑出一个设计。思考语法让我很好奇——你更喜欢一门语言有:

1.  大括号和分号 *(C，JavaScript，Rust)*
2.  大括号和无分号 *(Go，Wren)*
3.  分号和无括号*(我不确定是否有人这样做？)*
4.  没有大括号和分号，有明显的空格 *(Python)*
5.  没有大括号和分号，没有明显的空格 *(Lua)*
6.  一些我没有想到的古怪设计...

就个人而言，我倾向于选择 1 的明确性，但同时我也不喜欢`{}`有双重含义(它是作用域，还是对象文字？).其他人呢？
