# Svn Eclips 在 Eclipse 中安装 Svn 插件

> 原文：<https://dev.to/skptricks/install-svn-plugin-in-eclipse-542g>

帖子链接:[https://www . skptricks . com/2018/08/install-SVN-plugin-in-eclipse . html](https://www.skptricks.com/2018/08/install-svn-plugin-in-eclipse.html)

本教程解释了如何在 Eclipse IDE 中安装 SVN 插件。默认情况下，Eclipse 不支持 SVN 存储库浏览。为此你需要下载几个插件。

[在 Eclipse 中安装 SVN 插件](https://www.skptricks.com/2018/08/install-svn-plugin-in-eclipse.html)

按照以下步骤在 eclipse IDE 中配置 SVN 插件:

第一步:
首先打开 Eclipse IDE。

第 2 步:
进入帮助- >安装新软件

步骤 3 :
在“使用”字段中输入 Eclipse 更新站点的 URL。如果你使用 Eclipse 开普勒 1.10.x 版本，对于以前的 Eclipse 版本，我建议你使用 1.6.x 版本。它与大多数 SVN 服务器兼容。

1.10.x 版本的链接:[http://subclipse.tigris.org/update_1.10.x](http://subclipse.tigris.org/update_1.10.x)

1.8.x 版本的链接:[http://subclipse.tigris.org/update_1.8.x](http://subclipse.tigris.org/update_1.8.x)

1.6.x 版本的链接:[http://subclipse.tigris.org/update_1.6.x](http://subclipse.tigris.org/update_1.6.x)

1.4.x 版本的链接:[http://subclipse.tigris.org/update_1.4.x](http://subclipse.tigris.org/update_1.4.x)

第 4 步:
你现在会看到要安装的组件列表。选择所有选项，然后单击下一步按钮。

第 5 步:
在下一个屏幕上再次点击“下一步”，同意许可协议并点击“完成”按钮

步骤 6 :
等待一段时间，直到软件包下载并安装完毕。您将需要重启 Eclipse 以使更改生效。

步骤 7 :
点击 Eclipse 右上角的“打开透视图”按钮

在此窗口中选择“SVN 知识库探索”

步骤 8 :
在“SVN 存储库”视图中右键单击，选择新建- >存储库位置
