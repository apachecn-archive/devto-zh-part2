# 女演员学习编码

> 原文：<https://dev.to/thadevelyouknow/an-actress-learns-to-code--oci>

[![](img/64979230530f53dd0f9fbdc3f5852aab.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0_9-lMTq--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/3ogrx6js8j4fuavpcqyg.png)

*“你绝对应该考虑一下你可能适合的混合职位”*一天，一个技术小组的“社区构建者”对我说，因为我举手问了一个关于开发人员职位的问题，这个职位是为我这样的非技术背景的人准备的。就在刚才，坐在我旁边的那个人，一个经常向我寻求调试建议的同学，也问到了*的*工作前景。他说，他的背景是金融。他没有任何专业的编码经验。

他得到的答案和我的一点都不像，我忍不住纠结于其中的差异。技术小组欢迎这位先生的方式是，谈论“利用他过去的技能”以及他应该如何“绝对以金融机构为目标”，因为“钱在那里”——这种偏袒——在我身上没有消失。但不太清楚的是为什么。是因为金融胜过戏剧，还是因为他是个男人？

* * *

我感到自己下意识地变得愤怒和焦虑，这是我经常做的事情，但我没有说出来:它保持在背心附近，但我看到了我生活的灾难性结局——灰色花园式的贫困老太太的结局，还有几个“不公平！”咆哮到了我的舌尖，在它们再次消失之前，没有说出来，直到“正确”的时刻过去。

我飞快的思维开始转向政治不正确的地方，试图想象我作为一个中年妇女站在长幼有序的位置:我是在这个口音很重的外国男人之上还是之下？你能想象会有这样的想法吗？这就是争夺资源的动物思维。作为一名女权主义者，我总是小心翼翼，从不低估男人对我们的潜在恐惧。许多人把我们视为稀缺资源的竞争者。我认为女性记住这一点非常重要。

我是土耳其移民的孩子，我的父母也带着浓重的口音来到这里，他们带来了伊斯坦布尔大学的化学和化学工程学位。仅凭这些优点，他们就能够在美国为自己开创美好的生活，正是因为他们，以及我的前海军飞行员姐姐，她负责管理航母和核潜艇的建造，我最终进入了科技领域，尽管我在剧院有着不同的背景，也有着无数的日常工作。

* * *

我需要严格的编码训练营，因为我一直在编辑和叙述零工经济中的有声读物，将土耳其语翻译成英语，进行商业角色扮演，通过标准化的患者工作协助医学院学生的教育，并为期刊出版物撰写一些医学案例。“管道收入”，没有工作或没有工资的时期，以及追逐客户，让我筋疲力尽。

我开始考虑真正从事职业所需的证书，并开始申请研究生院。我参加了 GRE 预科课程，并在线申请了南加州大学，进入了他们的教学硕士项目，并准备在 2016 年 1 月开始。我将失去 50，000 美元，但我将在 18 个月内获得硕士学位并从事教学工作。罗格斯编码训练营于 2016 年 4 月开始，费用为 10，000 美元，并提供兼职 Fullstack Web 开发课程，该课程将在 6 个月后给我一个罗格斯大学的证书。

我不能仅凭数字来做决定。我必须真正决定，怎样才能让我有勇气放弃艺术生涯。我非常想成为一名成功的艺术家，但持久的成功总是与我无缘。当我评估我的生活，思考我能做什么，我能对什么保持热情时，我想到了我最喜欢的一句话，这句话帮助我足够诚实地面对自己，把表演放在第一位:

> “你就是你每天做的事情。”

我意识到我总是把时间浪费在编辑网站、photoshopping 图片、录制音乐、教孩子科学和阅读科技文章上。我从不看电视或电影，是一个顽固的社交媒体迷，我上网是为了娱乐。

在我年轻的时候，我对音乐的热爱似乎总是与我对所有书呆子的热爱平行增长，我最美好的高中记忆是我的“计算机”课，在那里我学会了编码——在这里我将与自己约会，因为当你谈论突破障碍时，你不会谎报年龄——这是基于 Commodore 2000。我最喜欢的游戏是 Atari Adventure。

在我最近的自由职业者生活中，我对技术非常固执己见。我注意到我对我正在使用的不同软件套件有强烈的感觉，包括声音工程 Daw 和翻译 CAT 工具(在翻译软件中可以看到有趣的机器学习结果)，我经常在论坛上从聊天室中专门研究音频的人那里学习解决方案。我是一个完全的电子产品迷，这要感谢我的工程师父亲收集了每一个电子产品的“第一”，我从不厌倦每天在屏幕上编辑数英里数英里的声波。

* * *

但是，尽管所有这些技术上的自命不凡，教会了我很多，让我很忙，事实是，我的学位是戏剧艺术。无论我多么希望技术小组的那些家伙能感觉到我曾经把检查电路板作为一份兼职工作，或者我是那种在早期型号的个人电脑中安装自己的内存芯片的女人...事实仍然是，我真的是一个前纽约市女演员和前“家庭主妇”。

冒名顶替综合症是我的中间名。夹在我父母给我起的奇怪的名字和我前夫的姓之间，我似乎无法摆脱这个名字，因为从法律上来说，我属于我的孩子，推而广之，属于他...石头上写着“冒名顶替综合症”。

这就是我，一个发现自己徘徊在研究生学位或全栈网络开发证书边缘的前演员。无论我做了什么决定，我都要获得任何能赢得我渴望的尊重的硬技能，无论是作为一名女性还是作为一名创意者，我想要更多地掌控自己的生活。

对我来说，数学很清楚，从任何角度来看，编码训练营都是更好的交易。但是我真的喜欢编码本身吗？我刚刚把我的女儿们送到编码夏令营，并意识到如果我向她们宣讲代码，我应该亲自实践。所以我开始在一个叫 Mimo 的手机应用程序上自学代码。然后我参加了为期一周的 Javascript 课程。我发现我有这方面的诀窍，更重要的是，我被迷住了。当我坐在我的机器前时，我注意到没有人告诉你你的代码不够简洁或者不够“相关”,所以我不断回来寻找更多的代码。我很抱歉 USC 感谢你录取我，但我必须推迟...

我作出了承诺，愿意在 6 个月里吃喝玩乐，就像多年前我在音乐学院沉浸式 BFA 美术课程中被告知要做的那样。只是这次我学到了无懈可击的严谨的编码技术。我知道这只是一个漫长旅程的开始，但我总是对新的开始感到满意。

技术的无限感比我在《艺术的生意》中感受到的任何东西都更加自由。开发人员慷慨而自由地分享信息、指导、知识‘财产’和经验，从这个意义上说，他们就像是最时髦的嬉皮士，他们的价值观近乎乌托邦。我很自豪能够成为技术革命的后来者，也很高兴和荣幸地欢迎其他人加入这场革命，首先是在我完成 6 个月的课程后担任导师和助教，现在是罗格斯大学编码和数据科学训练营的学生成功经理。

是混合职位吗？你打赌。但我和公司里许多职位比我高的女性导师一起，可以帮助选择谁来给那些技术小组的男女学生演讲。
