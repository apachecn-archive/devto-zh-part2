所以，让我们开始吧。如何破解 GSoC？

这是一个非常简单的过程。让我解释一下。我将从 0 级开始写这封信，意思是把观众看作是以前没有做过编程的人。请根据您的水平随意跳过。

### 0:学习基础编程

显然，接近 GSoC 的第一步是学习如何编码。

如果你是刚入门，我推荐你学习 Python 来写你的《Hello World》。

Python 简单而流行，许多 GSoC 项目都使用它。所以去学 Python 吧。

如果你喜欢通过阅读来学习，我会推荐你使用 w3schools 资源。但是如果你更喜欢一个合适的视频课程，参加 Udemy 上的这个[课程。它应该只花你 10 美元左右。](https://www.udemy.com/complete-python-bootcamp/)

### 1:学习实用程序编程又名开发

我说的“发展”是什么意思？

这意味着你应该能够使用代码来开发一个有用的软件。它可以是任何东西，从网页到服务器，到机器学习模型。

在这里，挑选一个你感兴趣的领域是非常重要的。

例如，如果你特别喜欢像 Twitter、Quora 这样的 web 应用程序的工作方式，那么前端开发应该是你的专长。

如果您对代码独立运行并在远程计算机上做事情的想法感兴趣，也许服务器端开发适合您。

ML，AI，GUI 开发等领域也是如此。

你也可以选择你感兴趣的多个领域。

例如，你可能喜欢网络应用的一切。在这种情况下，您可以学习前端开发和服务器端(后端)开发的一些部分。

你也可能不确定你喜欢计算机科学中的什么。在这种情况下，请随意尝试不同的领域，看看有什么坚持。

如何学习某种开发，比如前端开发？

你用谷歌搜索一下。你会在媒体和其他网站上找到好的指南。大部分是免费的。
利用它。

### 2:创建并参与开源项目

你可以看所有关于后端开发的教程，但是除非你真的用它做了一些东西，否则你什么都不会知道。

因此，一旦你浏览了一个主题的大量内容，考虑用你所学的和你将会学到的做一个项目。

您可以使用 [GitHub 搜索](https://github.com/search?utf8=%E2%9C%93&q=javascript+label%3Agood-first-issue&type=Issues&ref=advsearch&l=&l=)来查找您可以贡献的存储库(项目)。

或者您可以创建自己的项目。

现在的目标是获得一些实践经验，这样您就可以对各种事情有一个基本的概念，比如编码约定、目录结构、最佳实践等等。

放心上网咨询吧。在你做项目的技术中找到好的项目，仔细检查他们的代码。

理解它们的代码约定，承认它们的重要性，然后尝试使用您所学到的知识来构建您自己的项目。

在一个特定的专业领域做 2-3 个项目(比如前端)会让你对它很有信心。然后你就能毕业进入下一步。

### 3:查找目标 GSoC 项目

您可以使用“GSoC 组织”页面查找您可能感兴趣的组织。

使用搜索功能过滤编程语言和领域。

例如，下面是查找所有基于 Python 的组织的链接。

类似地，这里是所有在他们的项目中使用 ReactJS 的[组织的链接。](https://summerofcode.withgoogle.com/organizations/?sp-search=react)

一旦你确定了一些组织和他们的项目，浏览它们，找到你最感兴趣和最喜欢的。

如果能找到 3-5 个这样的项目就太好了。

### 4:为目标项目做出贡献，在组织内建立声誉

如果你只是在 GSoC 申请的最后一天出现，并要求他们选择你，组织将不会接受你。

你必须在组织内部建立融洽的关系，而你只能通过为他们做出贡献来做到这一点。

向他们发送 pr，参与讨论并提出新的想法。

既然在上一步中你已经有了一些“目标项目”,那就把它们作为在组织中建立你的关系网和可信度的媒介。

在你开始为某些组织做贡献后，你可能会意识到你不喜欢这些组织，这没关系。

这就是为什么你选择了几个(而不是一个)项目开始。继续关注你感兴趣的组织。

### 5:适用于组织

到目前为止，GSoC 的申请期应该是开放的，您应该已经有了关于您可以向组织提出什么建议的想法。

与组织管理员保持联系，并向他们提交您的建议。

在最终提交之前获得他们的反馈是个好主意，这样你就不会错过任何重要的东西。他们会很乐意帮助你，因为你已经在项目上帮助他们一段时间了。

因此，利用这个机会，确保你的建议是最好的。

### 6:继续贡献

申请截止和结果公布之间通常有 30 天的间隔。这期间不要懈怠。

组织管理员想看看你是否能胜任这项任务，以及你是否是真正的*。所以继续为他们的项目做贡献。*

 *在这个区间表现好，你就有巨大的机会被选中。

### 7:做得好

你现在被 GSoC 选中了。祝贺🎉！
