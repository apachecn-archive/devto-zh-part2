# 在 dbForge Tools for MySQL 中引入全新的连接特性

> 原文:[https://dev . to/devart software/introducing-brand-new-connectivity-features-in-dbforge-tools-for-MySQL-42f 0](https://dev.to/devartsoftware/introducing-brand-new-connectivity-features-in-dbforge-tools-for-mysql-42f0)

数据库工具和数据连接解决方案的捷克软件供应商 Devart 发布了用于 MySQL 的 dbForge Schema Compare、用于 MySQL 的 dbForge Data Compare、用于 MySQL 的 dbForge Query Builder、用于 MySQL 的 dbForge Data Generator 和用于 MySQL 的 dbForge Documenter 的升级版本。增强的工具支持最新的 MySQL Server 8.0 版和 Maria db 10.3 版。此外，它们现在还提供了连接到关系数据库云服务腾讯云的能力。

下面我们来仔细看看 dbForge tools for MySQL 中最值得注意的改进:
1)利用 MySQL Server 8.0 的所有特性
借助 dbForge Tools for MySQL，用户现在可以生成数据、比较模式和数据差异、在最新的服务器版本上构建查询。
2)享受使用最新的 MariaDB v10.3 的乐趣
所有用于 MySQL 的 dbForge 工具的新版本提供了连接到 MariaDB v10.3 的能力。
3)毫不费力地与腾讯云挂钩
使用升级后的用于 MySQL 的 dbForge 工具，用户可以连接到腾讯云(一种关系数据库云服务)，并在云中顺畅地管理 MySQL 数据库。
