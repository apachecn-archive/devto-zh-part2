# 输入账户信息

*   用户名
*   电子邮件地址
*   密码

键入，然后按" Create an account "按钮

[![アカウント情報の入力画面](../Images/6c85ff1730d06069b302af1ce52596ca.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--4Gw4oIhz--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/0goj6w6ho59szty7my5q.PNG)

※请不要忘记在这里输入的密码。

# 方案的选择

在输入帐户信息的下一个画面中进行计划的选择等。

初始状态下会选择免费计划。
这次不制作付费用户，所以用免费计划注册。

[![プラン選択](../Images/0c83223644ecd099e9454622631290e0.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--MoeEl7oE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/8pc28pz90j1kxqq3wbg3.PNG)

因为免费/收费计划的差异会偏离本投稿的宗旨，所以这次不进行说明，
免费用户用于一些小验证和 github 的认证就足够了。

# 3 .问卷调查

下一个画面有简单的问卷调查。

*   你作为开发者有多本领
*   你的 github 用途是？
*   你接近以下哪一个(学生/社会人/游戏者)
*   你感兴趣的话题是？

等，请根据自己的判断回答。

[![アンケート](../Images/eacdc9bc44e9eb87e2dd0d8bde4e8b0a.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--cYWB1au3--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vkhd7u320swvs0n06uef.PNG)

因为我觉得这里只是在做问卷调查，所以即使弄错了，也应该不会发生严重的问题。
不要太沉思，坦率地回答吧。
