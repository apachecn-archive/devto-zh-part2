## [](#why-you-need-a-cloud-presence-for-b2b)为什么 B2B 需要云

作为当前 webMethods Trading Networks 的客户，您可能会对您的内部 B2B 集成非常满意。然而，合作伙伴越来越期待来自你和你的竞争对手的基于 SaaS 的 B2B 流量和合作。

此外，首席信息官们正在评估云解决方案带来的潜在成本节约，该解决方案可确保及时、安全地翻译和交付任务关键型文档，如提前发货通知(ASN)、发票、采购订单和报价等。因此，现在可能是时候评估一个灵活、集中、经济高效的基于云的 B2B 平台了，该平台可以通过避免与内部解决方案相关的硬件支出和基础架构成本来显著降低总拥有成本。

## [](#about-webmethods-b2b-cloud)关于 webMethods B2B 云

Software AG 的 B2B 云建立在 Software AG 行业领先的贸易网络平台(包括通过 webMethods Integration Cloud 提供的后端集成支持)之上，可随时提供合作伙伴集成即服务，支持客户、供应商、分销商和其他贸易合作伙伴所需的现成文档格式和处理规则，包括通过 HTTP 和 AS2 进行实时 EDI (ANSI X12)电子文档交换。

## [](#partner-profile-management)合作伙伴档案管理

全新的增强用户界面有助于提高易用性，加快合作伙伴档案的设置。与 webMethods Trading Networks 一样，B2B Cloud 拥有业内最快的入职能力，允许用户在几分钟内定义合作伙伴身份和联系方式。

此外，该系统使用户能够维护合作伙伴对渠道(EDI 传输格式)和标准的偏好。

## [](#transaction-monitoring-and-processing)交易监控和处理

所有文档流量的端到端可见性是支持客户满意度和 SLA 合规性的关键。B2B Cloud 的事务分析提供了对任务执行和处理的实时跟踪，使您能够深入到事务数据并代表贸易伙伴采取纠正措施，如重新提交失败的文档。

## [](#backend-system-and-integration)后端系统与集成

B2B 云可以快速地将你的 B2B 流程和后端系统捆绑在一起。通过使用 webMethods Integration Cloud，您可以部署和编排基本的 B2B 流程，以增强合作伙伴的连通性。此外，集成云使公司能够利用云连接器的丰富 webMethods 库来实现 Salesforce、Marketo、ServiceMax、ServiceNow 和 Zuora 等解决方案。

## 2019 年来临！合作伙伴自助服务门户

通过自己的 B2B 云为合作伙伴提供支持是一个时机已经成熟的想法。

在我们的下一个发布周期中，webMethods B2B cloud 将为您的合作伙伴提供文档交换和可见性的单一界面。合作伙伴将能够上传和提交文档，通过可操作的警报体验移动支持，并编辑他们自己的个人资料，始终保持准确的信息。
