# 变更日志:宣布开发连接！(测试版)

> 原文:[https://dev . to/Ben/changelog-announding-dev-connect-beta-1i5f](https://dev.to/ben/changelog-announcing-dev-connect-beta-1i5f)

我们很高兴地宣布网站的一个新部分叫做 [DEV Connect](https://dev.to/connect) 。受到社区对此类事情的兴趣的推动，我们增加了一个与其他社区成员联系和聊天的地方。

这是一个消息传递部分，许多人会立即发现它很有用，但我们计划用一些非常好的面向开发的工具来逐步增强它。这里有一个(早期测试版)例子来说明你能做什么:

[![](../Images/c9101ecce481d6c3bff6074ff0bb8f7d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--qQW6AR0Z--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/brcv6dnr50btby7ivglh.gif)

Connect 已经上线，但在很大程度上仍是一项进行中的工作。我们计划将这些功能与围绕导师、办公时间和各种好处的倡议结合起来。你会从这一切中得到很多乐趣。

我们还预计，Connect 将成为与更广泛平台的内容进行交互的枢纽。消费，创建和分享开发职位将非常原生。下一个功能在我写这篇文章的时候还没有上线(但当你读到这篇文章的时候可能已经上线了)，但这里有一个我在本地环境中谈论的内容的快速截图。

[![](../Images/78b1945b154e597cf12ba3fc177af2f6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--oMdTFrmU--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/hgbt3e9l5pxp634rf131.gif)

现在，你可以和任何你共同关注的人打开一个频道。随着 DEV Connect 的发展，我们将提供更详细的关于你可以和谁聊天的配置，但是 mutual-follow 模型是一个非常成熟的协议，可以在网络上实现这种功能。

我们也有一些团体频道，目前可供一些成员测试。随着我们制定细节，我们将使这些更容易理解。

这在很大程度上是一个测试功能，所以有些东西可能会被随机破坏，但我们邀请您来检查它。
