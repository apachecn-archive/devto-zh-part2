# [](#learning-programming)学习编程

几个月前，我开始尝试完成一个编码训练营。总的来说，我一直对编码、软件和技术感兴趣。我觉得在过去的几个月里，我在前端和后端都学到了很多东西。我真的没有太多机会来充分巩固所学的技能，因为这是一种“凭感觉飞行”的冒险。我只剩下不到一个月的时间了，总的来说(尽管我的头很痛)，我会说这是一次积极的经历。

我意识到这里可能有人在他们如何学习和进入开发方面有一系列的经验。每一种学习方法都有它的优点和缺点。我意识到，一直学习是这个领域的标志之一，也是非常酷的一面。我的问题:你如何在保持学习的同时不被信息超载？显然，我听说那些刚开始开发的人试图通过学习几乎太多的东西来弥补他们知识的不足。

我真的很想听听每个人的故事，关于他们创业时的感受，包括他们的成就和失败。作为一个认为自己学习速度很慢的人，此刻一切都令人望而生畏。看起来确实是这样的，个人和职业上的巅峰成就让这一切都变得值得。
