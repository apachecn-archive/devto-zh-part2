> # Animation makes the project look better

从路线过渡到小细节，如点击按钮或显示工具提示时的反馈，动画让您的项目看起来很漂亮。精心制作的动画传达出你或你的组织足够重视细节，为用户创造最佳体验。

Angular 使用其内置的 DSL(领域特定语言)非常方便地创建简单和复杂的动画。更重要的是，Angular Material 组件库充满了许多好看的动画…

让我们看看下面这个从[Angular NgRx Material Starter](https://github.com/tomastrajan/angular-ngrx-material-starter)项目中录制的路线过渡动画来了解一下。
