# 自由职业前的轻松指南

> 原文:[https://dev . to/Darryl dexter/a-fluff-free-guide-to-pre-freelancing-43i 0](https://dev.to/darryldexter/a-fluff-free-guide-to-pre-freelancing--43i0)

我是一名开发人员，我的同行也是开发人员。我有一份白天的工作，我的同龄人也有一份白天的工作。我是自由职业者，我的同行不是。组合断路器对吗？

他们经常问我为什么做自由职业者。我总是回答“你为什么不呢？”。答案几乎总是一样的，总的来说，答案是

> “我只听恐怖故事。”
> 
> “我调查了一下，发现工作量太大了…”

然后他们告诉我他们读了些什么，总是有一些松散的事情要做，这些事情似乎是由一个一生中从来没有自由职业过一天或花钱写文章的人写的。总的来说只是错误的信息。

我打算用一种实用的、没有绒毛的方法来纠正这一点。我假设你是一名开发人员，白天从事自由职业。

此外，考虑一下这种自由职业前的情况，因为这些都是在你开始获得一些好客户之前需要做的事情。一旦你有了工作，你就不再是自由职业者了。那是一个词吗？？)

这是一个 6 分钟的阅读，所以拿起咖啡，让我们开始吧！

## [](#use-your-portfolio)使用你的投资组合

没有作品集？我有好消息和坏消息。

好消息:你是一名开发者。是“简单的工作”！
**坏消息**:你是个开发者，什么叫“简单工作”？？

帮自己一个忙，找一个简单的主题，花不超过一天的时间。我们不需要从头再来，只需要给人们一些东西，告诉他们“我知道我在说什么”。以下是我过去做过的事情。”把它当成一份更冗长更有创意的简历吧。

没有投资组合的开发材料？没问题，开始做一些个人项目。做一个微 app，推出去。做一个小库放在 GitHub 上。在 Dribbble 上找到一个设计，并把它变成现实。你不需要有带薪工作来证明你能胜任这份工作。

没有时间做这些类型的项目？好吧，写几个你以前工作项目的故事，就当是面试自己吧。你所面临的问题，你是如何解决的，下次你会有什么不同的做法。不要忽略细节。我们想在这里变得非常技术化。

总体来说，捶胸一点点！向他们展示你知道自己在说什么，你能向潜在客户提供的关于你能力的信息越多越好。

## [](#you-need-a-proposal-formula)你需要一个建议公式

现在你有了投资组合，你需要开始接触人们。没有人会知道你的存在，除非你告诉他们。

当人们寻找自由职业者时，他们会收到大量的建议。这些提议中的大部分都非常模糊，充斥着关于手头项目的无用信息。更不用说像 Upwork 这样的平台上不需要的手续了。
他们通常是这样的:

> 亲爱的公司:
> 
> 我对废话连篇非常感兴趣…如果你花时间考虑我，这将是废话连篇{在这里插入更多复制粘贴绒毛}

不要那样做。没人喜欢那样。在处理几十份提案时，没人会在意形式。很费时间。

让你的提议简洁明了，切中要点，集中在你的目标项目上。

比方说，你找到了一个项目，正在寻找一名开发人员来开发社交媒体分析仪表板。如何实现这一目标的示例:

> 嘿，
> 
> 看了看你的项目，有几个问题:
> 
> *   你瞄准了哪些社交网络？
> *   有多少用户需要使用这个仪表板
> *   你对仪表板有什么设计吗
> 
> 我参与的最后一个项目仪表板项目使用了这种技术堆栈，您对此满意吗？
> 
> *   与 redux 反应
> *   实时火灾数据库
> *   面向基础设施的 AWS
> 
> 除此之外，我在使用各种类型的仪表板方面有相当丰富的经验{在此插入更多与项目相关的话题}
> 
> 如果你想看更多我的作品，这是我的作品集。YourSite.com

我做了什么？

*   询问您需要了解的关于项目的问题
*   提到了我处理上一个仪表板项目的方式
*   讲述了我在这个项目中的经历
*   给了他们更多了解我的方法

起初，这需要一点时间，在十几个提议之后，你会像制作脸书状态一样快地发出提议。

## [](#get-on-upwork)继续工作！

现在你有了投资组合和你的提案公式，我们应该准备好了！是时候找些项目了。

你问哪里？向上工作。不是 craigslist，不是 googling，不是陌生来电，不是社交活动，不是朋友和家人。向上工作。

为什么这么问？找自由职业者的人最容易找到。

如果它倒下了呢？永远都是另一个。先是 Elance，然后是 oDesk，现在是 Upwork。如果升级失败了，我很确定下一个会出现，到那个时候，你已经不需要它了。

创建您的个人资料并购买高级层。它很便宜，你会很快赚回来的。实际上，非常快。想都别想，因为你一个月花在咖啡上的钱会比工作上的钱还多

## 这是一个数字游戏

文件夹，检查完毕。提案公式，检查。向上工作配置文件，检查。客户端，未检查。让我们改变这一切！

这个过程的这一部分可能是最难的部分，因为它需要一个新的习惯。总部基地“总是提议”朗朗上口吧？

**我注意到自由职业者会:**

*   发送一个提议，等待并期待对方的回应，最终进入下一个。
*   一周或一个月发几个提议
*   不要发送任何，而是希望得到联系

相反，根据我上面提到的公式，建立一个每天最少 5 个质量建议。把它当成例行公事，就像检查你的电子邮件一样。瞄准新项目，因为你会发现自己在旧帖子上浪费时间。
申请这些项目的时候，不要结缘！阅读描述，提出建议，回答问题，提交，然后忘记。重复一遍。

如果得到回复，就开始认真对待一个项目。在那之前，这只是个数字。

当你开始爱上一个项目时，你开始感到失败和没有动力。开始时发出 20 个提议，得到一个回应，这并不罕见。

好消息是，最终你会花 30 分钟发出 20 个提议。如果你问我，半个小时的工作对一个感兴趣的客户来说是非常好的。

## [](#start-small)从小做起

在寻找项目时，要留意那些想雇佣自由职业者的人和其他试图得到同样工作的自由职业者。刚开始时，你需要以低工资提供高质量的工作。是的，这很糟糕，但是，你需要建立一个声誉。

从招聘的角度来看，很难证明选择一个没有评论或工作经历的人是合理的。历史悠久，评价很高的人通常是理想的。问题是，这些人更善于表达(理应如此)。你需要确保查看你个人资料的人不会认为你是一个巨大的风险。你是，但是，我们在这里证明事实并非如此！

以 20 美元左右的价格开始，以引起人们的注意。对于每一个评论，把你的价格提高 5-10 美元，直到你满意为止。只要记得保持竞争力。

## [](#whats-next)接下来是什么？

我想假设你不想浪费时间读这篇文章，也不想申请，所以让我们做一些家庭作业吧！

**0** -找到一个冗长的 Spotify 播放列表！(或任何你用来放音乐的东西)
**1** -创建或清理你的作品集(~5 小时)
**2** -创建你的提案公式(~2 小时)
**3** -设置你的 Upwork 帐户(~1 小时)
**4** -今天每天最少给你发 5 个提案(~1 小时)
**5** -为出色完成的工作自我表扬(~ 20 分钟)

试试吧，让我知道效果如何！
