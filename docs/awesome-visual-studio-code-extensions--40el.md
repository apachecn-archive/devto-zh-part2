## 什么是 Visual Studio 代码？

Visual Studio Code 是微软为 windows、linux 和 MacOS 开发的源代码编辑器，它包括对调试、嵌入式 git 控件、语法突出显示、智能代码完成、代码片段和代码耐火的支持。我喜欢使用 visual studio 代码来编写代码，下面列出了我最喜欢的扩展，排名不分先后。

### 分机

#### [T1】1](#1-open-in-browser)[在浏览器中打开](https://github.com/SudoKillMe/vscode-extensions-open-in-browser)

这个扩展是由 *TechER* 开发的，它允许你在默认浏览器或应用程序中打开当前文件，最后你可以用你指定的浏览器打开 HTML 页面，安装后右击并选择‘在默认浏览器中打开’,很棒吧？。

#### 2 [缩进彩虹](https://github.com/oderwat/vscode-indent-rainbow)

这个扩展使得缩进更容易阅读，它给你的文本前面的缩进着色，每一步交替四种不同的颜色，oderwat 开发了这个扩展。

#### 3 [支架对上色器](https://github.com/CoenraadS/BracketPair)

毫无疑问是我最喜欢的扩展，这个扩展允许匹配括号用颜色来标识，用户可以定义匹配哪些字符，使用哪些颜色，这个可爱的扩展是由*coeen reads*开发的。

#### 4 [Javascript(ES6)代码片段](https://github.com/xabikos/vscode-javascript)

这个扩展包含 ES6 语法的 Javascript 代码片段，它是由 charalampos karypidis 开发的

#### 5 [反应原生工具](https://github.com/Microsoft/vscode-react-native)

这个扩展为 React 本地项目提供了一个开发环境。使用这个扩展，你可以调试你的代码，从命令面板快速运行 react-native 命令，并使用智能感知浏览 React Native APIs 的对象、函数和参数，它是由 *visual studio mobile tools* 开发的。

#### [T1】6](#6-html5-boilerplate)[html 5 样板文件](https://github.com/sidthesloth92/vsc_html5_boilerplate)

这个扩展提供了所有 web 应用程序中使用的标准 HTML 样板代码， *sidthesloth* 开发了这个。

#### 7 [TS(打字稿)皮棉](https://github.com/Microsoft/vscode-tslint)

将 TypeScript 语言的 tslint linter 集成到 VS 代码中，由 *egaama* 开发。

#### 8 [棱角分明的 v6 片段](https://github.com/johnpapa/vscode-angular-snippets)

Visual Studio 代码的这一扩展为 TypeScript 和 HTML 添加了 Angular 代码段，它是由 *John Papa* 开发的。

### 如何安装扩展

1 启动 visual studio 代码
2 启动命令 *Ctrl+Shift+p* 。
3 输入安装扩展并选择“扩展:安装扩展”。
输入你最喜欢的扩展，从下拉列表中选择，安装它。
5 启动命令 *Ctrl+Shift+p* 并键入 reload window，选中它重新加载 visual studio 代码。
