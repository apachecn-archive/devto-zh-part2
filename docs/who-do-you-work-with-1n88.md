# 你和谁一起工作？

> 原文：<https://dev.to/craignicol/who-do-you-work-with-1n88>

在每个公司，以及更大的公司，每个部门和地点，都有让公司运转的关键人物。他们可能是设定清晰愿景的老板，他们可能是超越自我的管理者，他们可能是认识每个人的交际花。不管他们是谁，他们是保持团队工作的人。当他们中的任何一个离开时，团队的动力就会改变。不总是最坏的情况，但总会有一段调整期。

然而，有时它们中的几个会迅速相继离开，就像采煤工作面的金丝雀一样。或者也许他们只是停止做那些使他们有效的事情，来自上面的新指示，在收费时间太忙，或者其他什么。突然团队开始不和谐，你不一定知道为什么，因为你没有看到他们在幕后做了什么。但是如果你感觉到了这股潮流，你就会知道每个人都在准备他们的简历，要么跳起来，要么跳出去。

如果你不知道谁是你的金丝雀，保持警惕，因为除了领导团队，可能还有一些你没有注意到的。到底是谁在幕后操纵？谁超越了？谁会出现在意想不到的地方？当事情出错或需要建议时，人们会向谁求助？不要去寻找那些把自己包围在灯光下的人，他们有一颗金子般的心，并希望你知道。寻找那些关注隐形人的人。谁对猫好？

但同样，要警惕那些金丝雀没有机会被发现的球队，因为有别的东西挡在路上。这些团队由毕业生或男性组成，超出了行业标准。如果文化中滋生了毒性，致命的警告就很难被发现。有毒文化保护自我，保护现状，清除异议，即使这些异议对团队的生存至关重要。

知道你在和谁一起工作。
