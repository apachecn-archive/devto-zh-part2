## 场景

你正在面试一份开发人员的工作。给你一个编码练习来展示你如何逻辑地解决一个问题。你可以选择使用任何语言。

你去学什么语言？

## 背景

我一直对会说多种语言的人很感兴趣。我喜欢问的一个问题是你用什么语言思考？我现在意识到，虽然我只会说一种语言，但我可以用多种语言写作...也就是编程语言。

我对以上的回答？ **Java。**我的第一门语言是 PHP，后来我在 Javascript/jQuery 方面做了很多工作，直到我上大学学习 Java。

奇怪的是，我仍然用 Java 思考。
