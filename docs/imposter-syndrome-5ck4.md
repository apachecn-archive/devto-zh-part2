# 冒名顶替综合症

> 原文：<https://dev.to/jrking365/imposter-syndrome-5ck4>

[图片来源](https://www.chanty.com/blog/stop-feeling-like-imposter-work/)

我已经学习软件工程将近 5 年了，我一直有一个问题，我一直认为(现在有时仍然如此)我不知道如何编码，我甚至不是普通人，或者他们是许多天才程序员，而我只是一个努力写代码的人。
但是，两周前，我的一个朋友告诉了我一件让我彻底颠覆的事情:

> “你是一个榜样，我夜以继日地工作，以达到你所拥有的编码技能，我想成为像你一样的人”。

她这么一说，我惊呆了，好多事情涌上心头:
*当我把时间花在 stack-overflow 上，当我连一些基本的 CSS 规则都记不住的时候，她怎么会想和我一样呢？*
但我意识到一件事，编码是一个旅程，你永远不会是最好的，但你必须努力做到最好。
有时你可能会觉得自己一无所知，但请记住，那些在 Stack-overflow 上回答你问题的人有一天也会像你一样。记住，有人正在努力工作，希望有一天能像你一样，达到你现在的水平。因此，无论何时你患上了冒名顶替综合症，只要记住你已经做了什么，并对自己说你才刚刚开始你的旅程。永远对自己说，你是最棒的，你会成为最棒的，自信是关键。
