# 看看 Android ML 工具包——机器学习 SDK

> 原文：<https://dev.to/auth0/a-look-at-android-ml-kit---the-machine-learning-sdk-131c>

在当今快速发展、信息丰富的世界中，构建智能处理数据的应用程序变得越来越有必要。人工智能正迅速成为软件开发的重要工具。

在本文中，我们将研究 ML Kit mobile SDK，它将 Google 在机器学习技术方面的所有专业知识以一个易于使用的包的形式提供给移动开发人员。我们将看看 SDK 提供的各种 API，然后我们将通过创建一个使用它的 Android 应用程序来测试其中一个 API。
