# Java 简介:修饰符备忘单

> 原文：<https://dev.to/mxl/introduction-to-java-modifiers-cheatsheet-58op>

Java 中的修饰语对我来说仍然很棘手，所以我会尽量把它说得简短些。

修饰符是改变类/方法/变量行为的词。两种主要类型是访问修饰符和非访问修饰符。

1.  **访问修饰符**改变可见性:

    *   *无修饰符*——类对它的包是可见的
    *   *私人*——只对班级开放
    *   *public*——到任何地方的每一个阶层
    *   *受保护的* -到包及其子类
2.  **非访问修饰符**改变其他功能:

    *   *static* -使声明特定于类，而不是特定于实例。它允许在没有实际实例的情况下使用方法
    *   *最终* -不允许参考值改变
    *   *抽象* -对于一个方法，声明该方法应该在子类中被重写，对于一个类，声明它不能被实例化，只能被子类化
