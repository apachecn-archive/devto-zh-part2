# LetsEncrypt:通配符支持

> 原文：<https://dev.to/defman/letsencrypt-wildcard-support--1ojl>

从今天开始，您可以使用支持 ACME v2 的客户端来发布通配符证书(例如 [acme.sh](////acme.sh) )。

一些有用的链接:

*   [宣布者](https://community.letsencrypt.org/t/acme-v2-and-wildcard-certificate-support-is-live/55579)
*   [关于 ACMEv2 的技术信息](https://community.letsencrypt.org/t/acme-v2-production-environment-wildcards/55578)

LetsEncrypt 是一个非营利组织，所以他们需要社区的大量支持。考虑[捐赠给他们](https://letsencrypt.org/donate/)或者[成为赞助商](https://letsencrypt.org/become-a-sponsor/)如果你在生产中使用乐。
