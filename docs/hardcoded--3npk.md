# 硬核。

> 原文：<https://dev.to/devtouser432/hardcoded--3npk>

我生活中反复出现的一个方面总是束缚着我。我对外部活动的依赖。通知。

通知本身没有任何意义，也没有内在价值。我曾在一个非常特殊的环境中如此不可控制地迷恋于一个通知的界面美，以至于我对它在我身上的所有力量感到荣耀和虚弱。是字体颜色吗？它说那个词的方式？不，这是我们赋予屏幕上特定形状的多巴胺。

我不必详细说明是哪种性质的通知使我个人受到奴役。这是任意的。但是我们都知道那种。有些日子，我完全生活在一个又一个通知中。我比一个一天抽三包烟的瘾君子更习惯性地抽它们。介于两者之间的，是那种挠心焦虑。我什么时候会有下一个？怎么样..*现在！*

当然，这都是荣耀和自我射精，直到你翻转你的手机和…什么都没有。脸上酸酸的一拳。*真的吗？*

任何一个理智的人观察到自己的这种行为，都会立即对其不雅之处感到震惊，并立即纠正它。但是，我从来没有注意到它。我从来没有注意到它的发生，直到所有的伤害都已经造成。我被击倒了，变了形，重重地倒在地上，我开始觉得一定出了什么问题。

一旦你注意到它，它就会令人痛苦地刺眼。

今天早些时候，我采取了一些措施来筛选那些让我关注的通知。我不需要在事情发生的第一秒就知道一切。它可以等待。我已经用每日报告取代了我的实时商业活动报告，每天早上给我发一封电子邮件，告诉我我需要知道的一切。我觉得这样，你只能被侮辱一次。

我甚至硬编码了积极的信息，如“您昨天注册了# { num _ signups }，这太棒了”和“您的经常性总收入是#{total}，这太棒了，这意味着您可以在未来很长一段时间内继续这样做。”我甚至做了一份附有附言的周报告，提醒我一些我很容易忘记的事情:“继续做你正在做的事情。这就是了。你已经来了。每天尽你所能做好工作。”

我想，如果我们如此容易受骗，以至于被屏幕上的一个泡泡疯狂地迷住了，我们不妨编一个故事，围绕它编一个漂亮的小故事。我的小脑袋瓜会在一周的任何一天都像狗屎一样倒下。
