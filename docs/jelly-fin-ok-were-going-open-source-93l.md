财政困难。这是每个人不得不面对的首要问题之一。所以，让我们把它变得简单和自动化。几年来，我和妻子一直用预测方法跟踪我们的财务状况，而且都是在电子表格中完成的。我想采用这个概念，并使用无服务器架构和简洁的设计使其可移动的时候到了。

1.  克隆回购。
2.  运行`npm install`以覆盖所有依赖项。
