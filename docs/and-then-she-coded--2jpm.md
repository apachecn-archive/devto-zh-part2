# 代表权问题

> 原文：<https://dev.to/nadirahdurr/and-then-she-coded--2jpm>

我最近看了电影《黑豹》,我钦佩瓦坎达的公主首里。她是技术天才，创造了瓦坎达的大部分现代技术。首里是真正的黑人女孩魅力的缩影。她是一个超级技术，聪明和时尚的黑人女孩。在黑豹看到首里对我来说非常重要，因为作为一个在郊区白人空间长大的黑人女孩，我一直觉得自己与同龄人格格不入。当我看到首里时，我看到了自己。

我来自新泽西州的一个郊区小镇，很多人都没听说过。在成长过程中，我通常是班上最高的，也是唯一的黑人女孩。我是一个超级有创造力的优等生，学习有天赋的艺术课程，喜欢嘻哈音乐和电脑，周围的老师和学生看起来或行为都不像我。我脸上还有一个独特的胎记，这让我从同龄人中脱颖而出。我与众不同，我讨厌这样。我只想合群，看起来像其他人一样。

13 岁时，我开始打篮球。当时，我并不是最擅长运动的人，但我坚持了下来，因为我告诉自己，如果我足够努力，我会像我在《WNBA》中看到的那些了不起的黑人女性一样伟大。后来我获得了体育奖学金。#代表性问题

现在你可能想知道为什么我要喋喋不休地谈论黑豹和我的童年。原因是因为我注意到，在成长过程中，如果我只看到和我有相同追求的人，我会对自己的皮肤更有信心。在我的成长过程中，我从来没有感觉到学术界有我的一席之地，这就是为什么我会走向体育。我一直知道我很聪明，但我总是觉得我不够聪明，因为我的黑和媒体的无知。

这就是为什么代表性很重要，这也是为什么它对年轻的黑人女孩在科技界很重要。追求科技职业是我从未想象过的。当我想到科技时，我想到超级聪明的白人男性，但从来没有人看起来像我。

我是在看了几本编码书，利用 Code Academy 之后开始我的旅程的。大学毕业后，我参加了一次实习，有机会为一家全球制药公司领导网站的重新设计。通过深入的研究和采访，我能够向我的经理和主任展示模拟网站的多个线框。此时此刻，我觉得自己足够聪明，我觉得自己的能力是有效的。我以前从未领导过这样的大型项目，就是在那时，我决定这就是我想去的地方。我决定要领导类似我实习时的项目。我想成为一个设计框架的人，并利用编程语言来创造一些动态和功能。这是我决定成为一名全栈开发者的时候。#shecoded

这也是我报名参加罗格斯编码训练营的原因。我想加入一个空间，在那里我可以挑战自己，也可以激励像我一样的年轻女孩。我想成为历史上这一重要时刻的面孔之一。我想证明与众不同并不意味着你不伟大。我想促成改变，拥抱成为一个有着一串代码和梦想的黑人女孩的可能性。# womenintech # womenwhocode
