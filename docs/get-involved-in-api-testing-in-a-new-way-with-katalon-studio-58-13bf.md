# 使用 Katalon Studio 5.8 以新的方式参与 API 测试

> 原文：<https://dev.to/testingnews1/get-involved-in-api-testing-in-a-new-way-with-katalon-studio-58-13bf>

[![Katalon Studio 5.8](../Images/cda1f85a7a7bea700d70a859131548fd.png)T2】](https://www.katalon.com/download/)

大家好，

Katalon 团队激动地宣布 [**Katalon Studio 5.8**](https://www.katalon.com/download/) 的发布，为您的 API/Web 服务测试项目带来全新的体验:

[![Katalon 5.8 for API testing](../Images/d88e51b6ce6001c4b5ea7d31cbb5d696.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--4eZx78VG--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d1h3p5fzmizjvp.cloudfront.net/wp-content/uploads/2018/10/09142125/Katalon-API-1.png)

## **使用快速启动向导立即导入 Web 服务请求**

快速启动向导允许您快速导入 Swagger 2.0/WSDL 定义，创建草稿 REST/SOAP 请求，或者尝试不同的测试需求。当导入 Web 服务定义时，Katalon Studio 将加载指定的信息并填充到您的项目中，这样您就不必花费时间来手动转换它们。该向导可以在**帮助>快速启动中随时启动和重新启动。**T3】

## ![Quick start in KS 5.8](../Images/a4f4784852fa5b22e99058d77c4ddde5.png) **随时检索请求历史**

在 Katalon Studio 中发送的所有 Web 服务请求都存储在“请求历史”面板中，您可以随时检索。

## ![Request History in Katalon Studio 5.8](../Images/98bd2ed7b49d558527740a264ab02776.png) **向新的/现有的测试用例添加 Web 服务请求**

Web 服务请求对象可以从“对象细节”视图直接添加到测试用例中，有两个可用选项:“添加到新测试用例”或“添加到现有测试用例”

[![Add Web Service request example](../Images/cd70c427205a021f8d29567ebad5fb14.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--LURIygHx--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d1h3p5fzmizjvp.cloudfront.net/wp-content/uploads/2018/10/09142132/Katalon-API-4.png)

## **支持补丁 HTTP 方法**

Katalon Studio 现在支持补丁 HTTP 方法，帮助您对资源进行部分修改。

[![PATCH HTTP example](../Images/47d269d2d5f272e2deaa272eaa245a04.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--mQNcD8P_--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d1h3p5fzmizjvp.cloudfront.net/wp-content/uploads/2018/10/09142133/Katalon-API-5.png)

这个版本还包括了新的错误修复，以提高测试自动化的便利性和质量:

*   修复本地 URL 不能用于 SOAP 请求的问题。
*   删除用户创建新的示例项目时显示的错误消息。

我们希望您喜欢使用 [Katalon Studio v5.8](https://www.katalon.com/resources-center/blog/api-testing-experience-katalon-5-8/) 的 API 测试体验，并且不要忘记提供您的反馈，以便我们能够使 Katalon Studio 更适合您的需求。

谢谢您，祝测试愉快！
