首先，在您的 Netlify 项目中创建一个构建钩子 URL

1.  转到—[https://app.netlify.com/sites/](https://app.netlify.com/sites/)/设置/部署
2.  点击**添加构建钩子**按钮，进入详细信息
3.  生成**构建挂钩 url**
4.  复制**构建钩子 URL**-【https://api.netlify.com/build_hooks/ T2】
