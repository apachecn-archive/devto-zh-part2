## [](#1-the-most-straightforward-solution)1 最直接的解决方法

JavaScript:

```
function isPalindrome(str) {
    const reversed = str.split('').reverse().join('');
    return str === reversed;
} 
```

# [](#2-elegant-but-redundant)2。优雅(但多余)

JavaScript:

```
function isPalindrome(str) {
    return str.split('').every((char, i) => {
        return char === str[str.length - i - 1];
    });
} 
```

# [](#3-an-alternative-with-a-for-loop)3。具有 For 循环的替代方法

JavaScript:

```
function isPalindrome(str) {
    const n = str.length;
    for (let i = 0; i < Math.floor(n/2); i++) {
        if (str[i] != str[n - i - 1]) {
            return false;
        }
    }
    return true;
} 
```
