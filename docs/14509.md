## [](#i-begancontinue-to-code-because)我开始/继续编码是因为...

我开始编程是因为这是我平面设计专业的必修课。我不得不去上网页设计课，这包括学习 HTML 和 CSS 的基础知识。第一次接触后，我对它印象深刻，并看到最终对印刷材料的需求会变得越来越少。我决定追求数字频谱，而不是我第一个想到的品牌。

我继续写代码，因为我喜欢不断学习新东西的想法。出来的技术通常更好，或者以不同的方式做事，这非常有趣。从做简单的邮件，到做不好的网站，再到做的更好更进步的网站和邮件，我慢慢进步了。旅程还没有结束。要学的东西太多了，而且不会很快停止。

我很乐意继续写代码，看看还能创造出什么。也许有一天我自己也能创造点什么。

## [](#i-recently-overcame)我最近克服了...

我正在过渡到一个普通的栈开发角色，最近我克服了学习一些新语言的最初障碍，特别是一些 javascript 框架。我喜欢 Vue，但我也在学习 Angular。我在 Node.js 上进展得很好，希望能继续进步。

## [](#i-want-to-brag-about)我想吹嘘一下...

去年，当我创建了自己的[工作流程来加速我的电子邮件创建](https://dev.to/teekatwo/creating-my-own-gulp-based-email-workflow-system-1h22)时，我真的很自豪。我从未使用过任务运行器、节点或 javascript 模板语言。我很高兴能做一些让我的生活更简单的东西，并让它发挥作用。然后从 json 文件中得到重要的数据，我欣喜若狂。现在让它成为一个 GUI，并且能够对这些 json 文件进行修改。

我认为一个复杂的计划不仅走到了一起，而且正在发挥作用，而且我可以概述它的扩张计划，这种想法令人难以置信。我为自己感到骄傲，为我独自面对未知并取得成功的动力和毅力感到骄傲。

## [](#im-currently-hacking-on)我目前正在入侵...

我在摆弄 Vue，学习它的工作原理。我正在建立一个可编辑的导航，你可以添加，删除，编辑和移动链接。这是一个很大的教训，它只是我的一个随机想法。

## [](#my-advice-for-allies-to-support-women-and-nonbinary-folks-who-code-is)我对支持女性和非二进制代码的盟友的建议是....

像对待其他编程人员一样对待他们，他们不想也不需要额外的特权，他们只想成为一名程序员。不是一个“插入标题”编码器，只是一个编码器。他们想做一份工作，并对它充满热情。像其他人一样建立他们。

如果你看到有人以消极的方式对待这些人，试着成为理性的声音，用你的声音去尝试改变。不是为了这些人，而是为了这些人。

最后，如果你在一个男性主导的领域，无论是黑客马拉松，会议，还是其他形式的聚会，请记住，这是一个女人和非二进制人很难去的地方。这可能会令人生畏和害怕，不要放弃他们，只要继续努力表现出一种中立和有趣的气氛，他们最终会出现的。这些改变不会在一夜之间发生，但是通过有意识的重复，最终会成功的。

## 我对其他女性和非二进制程序员的建议是...

我只能从我的经验来说，不要害怕对某件事表现出兴趣，问很多问题。我让很多人敞开心扉。如果你亲自或在网上看到某人做了某事，你会想“哇，他们是怎么做到的？”。问问他们就知道了。当然，他们可能不会回答你这是常有的事。如果他们这样做了，那么你就获得了一些你以前没有的知识。

此外，关注/与一群强大的人互动。不管性别如何，这应该是一个希望提升每个人并帮助他们成功的团体。如果你需要帮助，你可以去找他们，他们肯定是你可以学习的地方。

在我的职业生涯中，我很幸运没有因为我的性别而与消极或有害的人格发生任何公开的冲突。这并不意味着它不会发生或你不会。意识到不管那些消极情绪是什么，不要让它们让你沮丧或者改变你想要实现的目标。了解你内心的力量和渴望，不要让它影响你。*(旁注:如果你需要喘口气自救，然后继续进行。不要让负面情绪停留和恶化。快把那东西扔掉。)*
