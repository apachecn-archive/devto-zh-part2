## 介绍 MFT 代理商

拥有数十个、数百个甚至数千个位置的公司有一个独特的要求，即以精确或标准化的方式传播单个文件或文档。这一要求可以很容易地满足 MFT 文件机器人或代理。

ActiveTransfer Agents 是轻量级应用程序，允许管理员在集中式服务器或台式机与任意数量的远程位置之间远程交换自动文件传输。

这方面的新特点是代理将自动(并且非常高效地)全天候工作，根据需要或作为批处理作业进行文件传输。此外，中央 MFT 服务器可以配置事件驱动的触发器，按照相似特征的定义单独或按组发送文件。

对于零售，这可能意味着在各个商店同步传播定价和促销信息。对于一家主要银行及其个别分行，代理将自动发送日终对账报告；或者，分销商可以使用代理来同步文件传输，以便在任意数量的仓库之间共享产品目录更改。
