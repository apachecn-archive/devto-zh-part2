# 完全远程化:六个月过去了

自从 Mobile Jazz 离开它的最后一个办公室以来，已经快接近六个月了。对公司来说，这是一个分水岭:那天我们迈出了最后一步，完全远离了公司。现在回想起来，这感觉像是一个伟大的决定——这个决定让我们的公司文化更加紧密，让我们更加团结，而不是更加疏远。

前几天，我和我们的 iOS 开发人员之一 Paolo 讨论了这个问题。他是 Mobile Jazz 最早的员工之一，从一开始他就完全远离公司，在意大利工作。所以他对办公室的体验和我完全不同。对他来说，我们离开办公室的决定没有任何改变。但对我们大多数人来说，这感觉像是公司历史上的一个转折点。

位于巴塞罗那市中心的办公室是移动爵士乐早期历史的重要组成部分。我们的联合创始人 Stefan 和 Jordi 在 2012 年公司成立几个月后租下了它，这是他们成功的背景，因为 Mobile Jazz 发展到了 20 多名员工，并接纳了大量新客户。它给了我们一个基础，一个身份。巴塞罗那成了公司的精神家园；今天依然如此。

该建筑位于 Carrer de Nàpols，位置绝佳，步行 30 秒即可到达巴塞罗那最著名的主干道 Avinguda Diagonal。向北步行五分钟就可以到达高迪的圣家堂，向南我们可以看到美丽的 Ciutadella 公园和海滩。

[![](../Images/55611ec26b30845ed19c498bf8097b90.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--wUZwQYdu--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://mobilejazz.com/blog/wp-content/uploads/2018/09/going-remote-terrace-view.jpg)

*从我们巴塞罗那办公室的露台上看到的景色*

巴塞罗那的每个人都喜欢在办公室闲逛。烧烤和下班后的饮料成为传奇，它提供了一个与来自不同团队的人交往的好地方，特别是在早期，我们中的许多人都住在本地。这有助于形成我们今天的团队精神。

但是，随着时间的推移，感觉公司已经超越了物质基础。从一开始，我们就接受了远程工作。人们被鼓励在他们觉得最有效率的地方工作。因此，随着人们越来越熟悉他们的客户和他们的工作量，许多人选择在家工作或周游世界，利用他们以前在通勤上消耗的时间来完成更重要的任务。

乔迪实际上几乎每天都来，因为他住在本地，喜欢办公室的日常工作，但斯特凡正在欧洲旅行，只有在路过巴塞罗那时才会突然出现。其他人过去可以来去自由。有时办公室会爆满——特别是在周三的团队会议上——而其他时候几乎是空的。

就我个人而言，当我开始使用 Mobile Jazz 时，我每天都去，但到了第二年，我每周去一次或两次。我有 30 分钟的通勤时间，在某些日子可能会更长，在家工作更有意义。办公室关闭的时候，我一个月只去一两次，对于很多在巴塞罗那的人来说，这种情况是一样的。我们仍然会在晚上聚在一起喝酒或吃饭，但我们享受白天在家工作的自由和责任。

*在巴塞罗那办公室和我们的一个会议室工作*

最后，有时候清洁工来了，却没有人让他们进来，或者有人不得不从家里坐地铁来为他们开门，这完全是浪费时间。办公室租金相当高，如果没人用，花这笔钱是愚蠢的。

所以 Stefan 提出了在三月份离开办公室的话题，然后我们在之后不久的每周例会上讨论了这个问题。这没什么好惊讶的；我们以前谈过，MJ 团队的许多人现在都在国际上工作。关闭办公室似乎是合乎逻辑的一步。

但这并不是说我们这些还在巴塞罗那的人没有感情。我记得有几个晚上，乔迪会去打扫卫生，然后坐在办公室里，回忆着自从他和斯特凡租下那栋大楼以来，公司取得了多大的进步。他本人来自巴塞罗那，所以城市中心的物理空间对他来说非常重要。

[![](../Images/2c5999935cb5f343af45e92e2003cf08.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--oOmUrt4K--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://mobilejazz.com/blog/wp-content/uploads/2018/09/going-remote-office-team.jpg)

*我们清理办公室的最后一天*

对我个人来说，这也很有意义。我完全支持关闭办公室，但同时也很难过。这是我在巴塞罗那时间的很大一部分，因为我很快就要回到英国，这感觉就像是我在加泰罗尼亚一次惊人的冒险后的第一个“再见时刻”。

但是正如保罗前几天对我说的，MJ 就是这样工作的。一些公司希望在基础设施到位之前就开始远程工作，但 MJ 在我们还有办公室的时候还是一家远程公司。我们所做的一切都写在某个文档或电子表格中，所以我们已经为这一步做好了准备。即使当人们肩并肩工作时，我们也会通过 Slack 或 Asana 进行团队交流。我们甚至有一本公司手册和一个内部维基，记录了你需要知道的关于为 Mobile Jazz 工作的一切。

## 初期问题

这并不是说没有初期问题。无论你选择在哪里工作，你都需要稳定的互联网连接、合适的设备和有利的环境。我记得一个团队成员去拜访他的家人，希望他父亲的笔记本电脑能够工作，却发现它已经坏了，他不得不紧急寻找替代品！

每个人都必须找到自己的避难所，在那里他们可以不受干扰地有效工作，这并不总是容易的。Paolo 回忆起他刚开始时试图在咖啡店工作，但很快就放弃了这个实验，因为他无法集中注意力。我们中有一两个人遇到过类似的情况。

感情上有时也很艰难。如果你一天工作不顺利，当你在远处时情况会更糟，因为你不能和任何人谈论这件事。我认为，在最初的几个月里，我们许多在办公室工作的人都错过了这一选择——你可以经常去某个地方与团队的其他成员联系并叙旧。

但是 Stefan 和 Jordi 做了一些很大的改变，取代了我们以前的物理集线器。他们带来了一年一度的公司务虚会，这是全薪的，让我们有机会每年聚在一起。这个远程周完全专注于公司，充满了演讲和演示、黑客马拉松、讨论和增长计划。第一个是在大加那利岛，随着夏天接近尾声，我们都急切地期待着秋天的太阳！

*在撒丁岛远程工作的移动爵士乐成员*

我们还开始每周进行一对一的通话，不同团队的人配对在一起。这些电话让我们可以找到不在日常工作中的人，问他们问题并建立联系。它旨在取代办公室的社交方面，在那里你可以与团队以外的人聊天，并与他们共进午餐。

## 利益

现在，我们感觉已经完全达到了 100%远程工作的速度，并且已经看到了优势。财务节省是巨大的，因为它允许我们投资于业务的其他部分，如大加那利岛之旅。但更大的好处是我们拥有的自由。

感觉业务真的在为我们工作，而不是相反。我们都可以自由地探索世界，而不必被束缚在某个特定的位置。我们很多人都热衷于旅行，所以 MJ 的方法是完美的。对于 Stefan 和 Jordi 来说，完全远离意味着他们可以在世界任何地方雇佣到最好的人。人才库无穷无尽。

如果我们建议其他公司如何从基于办公室的环境过渡到远程工作，建议会很简单:确保你做好准备。确保基础设施到位，一切都记录在案，并使用视频会议定期进行团队沟通。此外，不要害怕在试验的基础上进行尝试，在其他人加入他们之前，一小部分员工会远离他们。

但我们也要说，如果你做对了，这是一个真正授权、解放的决定。我们将永远记得巴塞罗那市中心的那间办公室，但在离开它六个月后，感觉我们正在完成我们的自然使命。
