# 慢就是顺，顺就是快

> 原文：<https://dev.to/wiredferret/slow-is-smooth-smooth-is-fast-57j1>

如果你认识这个咒语，你可能知道它来自枪法训练。这个想法是，最好是慢慢来，不要有任何障碍或意外的颠簸，而不是匆匆忙忙，有一个不太可预测的结果。

另一种说法是

> 你没时间做对，却有时间做两次？

这种哲学在技术上有很多明显的应用，但我一直在一个更具触觉的领域处理它——手写。我是在学校学过草书和打字的尖端一代。很明显，我打字比手写快得多——大多数人都是这样，否则我们就不会发明打字机。我的孩子接受了一些非常简单的草书教育，但主要是为了让他们能够阅读，而不是书写。没有人给他们的书法打分，他们的大部分作业都是在谷歌文档上上交的。当我在学习草书的时候，我的老师告诉我，我相信，这是因为它比印刷术更快。

大约在过去的一个月里，我放弃了兔子爱好，拿起了自来水笔。温哥华的 DevOpsDays 人赠送了一套可爱的文具作为演讲者的礼物——合适的自来水笔、墨水、高质量的笔记本。我发现网上的书呆子们说的是真的——用自来水笔写字和用圆珠笔甚至滚珠笔写字是完全不同的体验。自来水笔终于让我明白了草书的意义。

事实证明，有些交流方式是根据特定的工具来调整的。

谁知道呢，对吧？所以我在小学草书的时候都很沮丧，因为草书对我来说并没有感觉更快，我会在一个字母或一个单词中间迷失，然后啊啊啊。这使得学习“钢笔是如何工作的”和“斯宾塞书法”变得非常有趣(纯粹主义者会告诉你，这是*而不是*书法)。原来 10 岁的我有一些复杂的问题:

1.  没有足够的时间/练习来掌握。众所周知，我学体育技能很慢。我花了 3 年时间学会骑自行车。所以对我的同龄人来说足够的学习时间对我来说是不够的。
2.  注意力问题意味着我会在一个单词中间失去注意力，或者忘记如何组成字母，或者试图比我的肌肉准备好的速度移动得更快。
3.  我不觉得这有什么好处。

现在我是一个成年人了，我不仅能买得起合适的工具(相对便宜)，还能买得起让我感到兴奋和有趣的工具(不太便宜)，我觉得更有收获了。我不是要交家庭作业，我只是在学习一项技能，所以时间和准确性的处罚不适用。不出所料，当我慢下来的时候，我有更好的书法。而且就爱好来说，这个真的很好拿，也很容易放下，甚至比编织还容易。

我也学到了多年的技能，如何自学，如何自我纠正和做有意识的改进。因为我花了这么多年做独唱作家，我不得不学会审视自己的作品，迭代，改进。这个基本技能现在为我生活中的各种事情服务。因此，我现在明白了操练和实践的价值。即使不好玩。

要做的第一件事是画线

我曾经对自己的爱好感到很难过——有时我会非常投入地做某件事，并准备好所有的设备去做，参加手工艺班，然后……然后几个月后，我会放弃它。下一次激情来临时，我会惩罚自己。“还记得刺绣吗？你有所有的设备，但你只完成了 2.5 个项目。不，你不能做有趣的事！”。我已经放松了那种态度。我的意思是，我确实试图从我想做的事情的最小可行工具包开始，但是如果我喜欢它，我会一头扎进去。为什么不呢？我容忍轻浮，我没有伤害任何人，这让我乐于学习。

一遍又一遍地练习一个字母来提炼我想要的并学习动作

当你开始审视时，所有的爱好都是分形的。我不确定工作也是如此，或者可能只是深厚的专业知识不太容易分享。所以对于顶级爱好钢笔来说，分形可能是这样的:

*   书法
    *   书法
    *   书法
    *   手写字体
*   墨水
    *   购买
    *   手工制作的
    *   混合
*   钢笔/硬件
    *   珍贵收藏
    *   黑客攻击/修复
    *   恢复
    *   设计

其中的每一个都可以被进一步深入到特殊兴趣的小角落。那就是*惊艳*。说真的，谢谢你，互联网。爱好就是 fandoms，我们都能在某个地方找到适合自己的地方。我发现我喜欢公路自行车，但我讨厌自行车保养。我可以付钱给商店做那件事。还有其他人喜欢修补，调整和升级他们的自行车。我喜欢拼接被子，但认为手工绗缝乏味。没关系，我可以成为一名机器缝纫师。

一旦我认为爱好是分形的，我意识到我们不仅可以深入到子爱好，我们还可以回到为什么我们想做爱好的更大画面，它让我们深入了解为什么我们想做任何事情。

我喜欢学习东西。它给了我一种满足感和对生活的掌控感。我对新想法的感觉就像喜鹊对闪亮珠子的感觉一样。这种基本倾向真的占据了我职业生涯的大部分时间——我曾经开玩笑说，技术写作是一辈子写研究论文，但这并不遥远。这更像是新闻业，但记者仍然会在离开时知道比报纸上报道的更多的事情。

我的爱好是我培养激情的一种方式，这种方式对我和雇主都有好处。当我回到另一个城市的酒店房间时，有时我想做一件有触觉的事情，因为人们而筋疲力尽。有时候我需要提醒自己，艺术和手艺、工作和爱好的区别，在于你得到多少报酬，而不是它有多值钱。即使是喜鹊，在它们真的只是想要休息一下和一些锡纸之前，也只能捡起这么多闪亮的珠子。

这和枪法有什么关系？一切。因为慢就是顺，有时候我们需要慢慢的去欣赏，去学习我们需要的东西。因为流畅是快速的——在我们使用墨水之前，仔细考虑我们想说和写的东西是值得的。因为我们为学习爱好所做的一切本身就是学习自学技能的一种方式。
