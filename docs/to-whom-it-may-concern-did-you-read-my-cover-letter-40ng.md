# 致相关人员:你看了我的求职信了吗？

> 原文：<https://dev.to/perrydbucs/to-whom-it-may-concern-did-you-read-my-cover-letter-40ng>

作为一名大学教授，我从学生那里收到很多关于求职和实习的问题。通常他们会在获得一个闪亮的新 BSCS 学位后试图找到他们的第一份工作。

上周一个学生下课后过来问我是否可以评论一下他们的求职信。它有整整一页长，以一个笑话开始，我看不到第一段。我的建议是撕掉它，写一个三句话的便条:

*   承认招聘人员/经理很忙
*   提到学生对公司感兴趣的一件具体事情
*   感谢招聘人员/经理抽出时间

我曾多次参与科技行业的招聘过程，浏览过成千上万份简历。我从来不看求职信。我对学生的观点是:

*   招聘人员只想通过堆栈；他们不会看你的求职信
*   你不了解招聘人员，所以任何与他们“联系”的尝试都是徒劳的，而且会给人留下不真诚的印象
*   把你本该花在信上的时间花在让你的简历看起来更有效果上；你只有 3 到 4 秒的时间用你最重要的观点来抓住读者的注意力

我很想听听其他人对求职信的看法，尤其是那些积极参与招聘过程的人。你读吗？它们会影响你的决定吗？
