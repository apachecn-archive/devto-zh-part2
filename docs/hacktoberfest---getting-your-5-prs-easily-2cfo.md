# Hacktoberfest，如何轻松获得你的 5 PRs？

我特别为 Hacktoberfest 制作了一个[回购](https://github.com/Webd01/Hello-World)。这个回购的目标是创建多种语言的 Hello World 程序。现在回购上大约有 45 种语言，所以这里有很多事情要做。

## 为什么要创建另一个 Hello World 存储库？

我知道已经有很多这样的仓库了。还有一个官方回购，那我为什么又创造了一个？事实上，当我想为官方 Hacktoberfest 的知识库做贡献的时候。我发现自己陷入了困境，因为已经有大量的语言了。我找不到还没有添加的。这就是为什么我创建了这个回购，它不包含很多语言，所以你可以添加一个，并得到你的 5 拉请求很容易。

## 贡献的

1.  [分叉仓库](https://github.com/Webd01/Hello-World)
2.  创建自己的分支，例如: **feat/my-awesome-language**
3.  做个公关！
4.  不要忘记在自述文件中添加您的 Github 用户名。更多信息见本[期](https://github.com/Webd01/Hello-World/issues/54)！

感谢你阅读这篇文章，我希望你能得到 5 分！
