# WWDC 2018 年奥运会首字母缩略词

> 原文：<https://dev.to/michael/wwdc-2018-acronyms-spelt-out-1e13>

昨天，苹果公司以[主题演讲](https://developer.apple.com/videos/play/wwdc2018/101/)拉开了 2018 年全球开发者大会( [WWDC](https://www.spellitout.xyz/wwdc-apple-worldwide-developers-conference/) )的序幕。在主题演讲期间，苹果宣布了许多令人兴奋的 iOS、macOS 和更多产品的新功能。在观看演示时，我注意到使用了许多缩略语。因此，我添加了一些我从未听说过的新词，比如 [USDZ](https://www.spellitout.xyz/usdz-universal-scene-description-zip/) 和 [ZSO](https://www.spellitout.xyz/zso-zero-sign-on/) ，并编制了[一份在“拼出来”演讲中使用的所有缩略语](https://www.spellitout.xyz/wwdc-2018/)的列表。

特别是一位演讲者，Adobe 的首席技术官阿沛·帕拉尼斯，我认为他在拼写经常缩写的单词和短语方面做得很好。在我看来，这使得他的主题演讲更受欢迎，也更容易被预期的开发者和技术观众之外的观众所理解。

如果你第一次观看或正在观看主题演讲，并对演讲者使用的一些缩写感到困惑，请查看拼写出的 WWDC 2018 年缩写列表。
