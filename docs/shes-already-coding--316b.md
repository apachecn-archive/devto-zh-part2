# 她已经开始心跳了

> 原文：<https://dev.to/catriname/shes-already-coding--316b>

**它？21 ..我准备好了吗？**
当我进入 web 开发领域时，我的公司只经历过一个比我早的女性尝试进入这个领域。由于没有经验，她大部分时间都在学习计算机科学的课程，但她自己并没有接触过很多计算机。结果是她出现了“书教”。在现实生活中，她完全没有准备。因此，该公司有点厌倦在其中试用女性。

另一方面，我曾生活在一个声名显赫的社区，那里的人们整夜整夜地来到 T4 向你出售或交换东西。只需很少的钱！我有无穷无尽的“坏”电脑来建立我的经验。

不要误会我，我去了一所很棒的大学，追求所有我认为我想做的事情。我不知何故偷偷学了基本的 HTML、C++、Java 课程，作为我“科学”的先决条件。我是说，科学！

当我回到我的小城镇时，工作很少，我只需要一份工作，任何工作。我申请的最后一个职位，未来的 CEO 承认我上过计算机课。他特别询问了那些课程，我给出了一个非常没有挑战性的回答，“是的，我参加了一些计算机课程……”。我不想提我在计算机实验室地板上学习 C++的那一年。谁不会做呢？(解释或更正刚说过的话)我是说..科学！

我低估了自己。我还是被录用了。

找到我的位置
我当了整整三个月的秘书。最具挑战性的部分是与其他秘书打交道。我会在上午 10 点前完成，接听随机的电话，一天中的大部分时间都会数我认为在我面前的广告牌上有多少瓷砖，并试图在不搜索的情况下计算它的实际宽度。老板们明白了。

嗯，那个，还有一天下午网络管理员抓到我在拼凑一个 Depeche Mode 网站。他问我下周下班后要不要留下来，帮他换几台电脑等等。我同意了，一个月后，我加入了。

**17 年后**
我是专家..适应？当我加入 HTML/CSS 部门时，一点点 JS 是热门的东西。在我从事 IT 职业的几年里，我们必须做出一个决定:跳到这个新的领域。或者使用 PHP(IBM 大力支持)。多年来，我们一直使用 PHP。更糟糕的是，我们必须保持特定的版本(最大 5.3)才能跨系统兼容。我如何让自己变得如此停滞不前困扰着我，我发誓永远不会让它再次发生。

**什么变了？**
我放开了限制。如果我不必使用 PHP，或者从来不知道它，会怎么样？如果我现在是一名新的开发人员，答案会是什么？API！死胡同是:我们的遗留系统及其开发人员都不熟悉这些东西。

***所以，我自己编码了！***
我用 PHP 在他们的系统上编写自己的 API，然后跳到我这边，换了帽子，心想:好了，现在我有自己的工作了。

现在，我已经打开了大门——我可以使用任何语言和环境。

**我的决定**
。网芯。我喜欢这个主意。NET 标准以及 Xamarin(移动)是如何跨平台迁移的。可视化代码刚刚推出(开源和跨平台)，令人兴奋。这一切都是全新的，但与过去的我不同，这次我跳了。

现在-无云的 Azure 开发者，VS，VSTS 是我的主要工具..我非常高兴。

那些时刻他们永远不会让我忘记

*硬件上不大*

*   在帮助将 25 台以上的计算机安装在一个位置时失去了焦点，没有注意“槽口”这只公羊为什么不进去！！？？
*   哎哟欧啊啊啊！！那次我把回形针放进软盘驱动器取出磁盘，结果电死了自己。
*   我用仓库胶带分割器割伤了另一家分店的拳击设备，血流了一箱子。还是发了。

*网络管理员可能没有切断*

*   发现了如何通过内部网络以编程方式发送文件，并通过发送文件夹而不检查其内容来进行测试。把树枝弄倒了。

*不是一个放弃者*

*   总是资历较浅(比我高 2 岁的人比我多 10 到 20 岁)，我是永远的试验品，尤其是对网络管理员来说。有一天，他决定通过 MSN(安静开发办公室)告诉我，他将接管我的电脑，然后他开始尝试这样做——只是想看看我会尝试什么。我尝试了基本的端口拦截，防火墙，RDP，但这是一场失败的战斗。和另一个开发者在一个安静的房间里，我钻到桌子底下，拿出我的 cat5，坐了下来。什么？不，一切都很好。
