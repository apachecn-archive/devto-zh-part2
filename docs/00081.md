## [](#release-highlights)发布亮点

测试版是一个重要的里程碑，我们很高兴推出一些关键功能，包括:

*   ClassicPress 自动更新现已实现
*   从 ClassicPress 基础设施提供翻译服务的工作已经完成
*   删除了兼容 PHP 5.5 和更低版本的旧代码
*   我们已经删除/修改了更多 WordPress 特有的措辞和管理面板内容
*   使用 renew 保持所有构建依赖项最新
*   其他各种修复和开发流程改进
*   添加了新的 ClassicPress 徽标和“以业务为中心的 CMS”品牌
