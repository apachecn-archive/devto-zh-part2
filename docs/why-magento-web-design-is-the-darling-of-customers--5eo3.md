# 为什么 Magento 网页设计是客户的宠儿

> 原文：<https://dev.to/shirawinget/why-magento-web-design-is-the-darling-of-customers--5eo3>

那些用 Magento 设计师的 Magento 设计来开发或改造网站的客户的一个普遍评论是，他们无法相信成品取得了如此大的成就。Magento [的网页设计](http://www.bindapple.com/web-design-and-web-development/)经常超出所有人的预期。人们通常对成品的成功抱有很高的期望，但是 Magento web 设计客户经常被最终的结果所震惊。

“最终产品在形式和功能上都超出了我最初的预期(它很漂亮，工作起来也很漂亮！)."Hal Leonard 数字解决方案管理员 Elyse Coward

一个设计良好的网站有很多方面，其中至少有两个是外观和功能。当一个网站引人注目，当它看起来很漂亮，本身很吸引人，那么网页设计师的工作就完成了一半。但是网站本身的美是不够的，它必须功能强大，功能美观。

这是因为 Magento 网站的设计者在准备过程中倾注了如此多的智慧和努力，以至于访问者的体验是如此的愉快和流畅。所有的活动都在表面之下。所有的客户需要做的是使用 glide 毫不费力地通过这个令人惊叹的网站的许多页面。

当然，行动自由是一回事，能够进行商业交易是另一回事。这就是 Magentoweb 设计发挥作用的地方。是的，它本身就是一种营销工具。是的，被赞美和欣赏是一件美好的事情。但最肯定的是，它是一个功能性的电子商店，顾客可以在这里轻松、安全地找到他们想要的东西。这是一个吸引回头客的网站的完美例子。
