# DenverCoder9 和发展的未来

> 原文：<https://dev.to/devteam/denvercoder9-and-the-future-of-dev-44k3>

随着 DEV 越来越受欢迎，看到人们找到社区、鼓励和指导真的很令人兴奋。同样令人兴奋的是，人们正在寻找解决方案。DEV 是一种社会/功利的混合体。随着时间的推移，我们写的每一件事都会增加一个有用的知识库。

[![](img/6e6928128e6e10b20a643cb7a4d49536.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ZXR6UWOh--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://imgs.xkcd.com/comics/wisdom_of_the_ancients.png)

随着我们平台的成熟，我们将继续增加功能，扩大博客的形式因素。更多的上下文线索，指出一篇文章过时或半过时的能力，通过注释更有帮助的协作，更智能的书签，等等。
