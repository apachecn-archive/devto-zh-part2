# 儿童与规划

> 原文：<https://dev.to/brightdevs/children-and-programming--2378>

最近在学校等侄子的时候，我在看周围的广告，不想太无聊。令我惊讶的是，关于编程课的广告和关于体育课的广告一样多。他们都瞄准了最小的孩子——小学一年级。我坐在那里，想知道当孩子们还不会用波兰语书写时，怎么可能教他们用任何编程语言编写代码。嘿，他们几乎不识字！

[![Image](../Images/1063fc9bb48f5356a631cccf86681c05.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s---MVhCozR--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/i4o0fjlb7n3cuf5nta3u.jpg)

当我更仔细地阅读通知时，发现<cite>编程</cite>对于最小的孩子来说，主要是关于逻辑思维、因果概念和问题解决。这些可能被认为是编程的基础，实际上也是数学的基础。那一刻，我意识到我在小学的时候就已经开始学习<cite>编程</cite>！

首先我想提到的是一个你可能知道的世界范围的比赛。如果你是波兰人，我敢肯定你曾经参加过，或者至少听说过。你们中的一些人可能赢得了一些奖项:) <cite>Kangourou</cite> 竞赛由 <cite>Kangourou sans Frontières</cite> 协会举办，旨在促进年轻人的数学。**it 考验的关键能力是逻辑组合，而不仅仅是单纯的公式知识**。编程的基础，对吧？

参加这样的比赛对孩子来说是一个很好的机会，在获得所有提到的技能之前进行练习。你可能会在网上和书店里找到前几年的练习。我只希望这个比赛仍然像几年前我参加的时候一样有趣。

我想告诉你的另一件事是 Szczepan Jeleński 写的一本波兰书。这是**一套各种类型的数学问题、练习、轶事和奇闻来鼓励儿童和青少年学习数学**，最终甚至让他们着迷。不幸的是，它可能不像 Kangourou 那样广为人知，因为没有它的翻译(至少我没有找到)。

如果你对这本书那个奇怪的书名很好奇，在前言里有解释:
<cite>Lilavati——这是十一世纪印度著名数学家的女儿的名字[...].
Lilavati 的意思是:可爱，迷人！这可能是一个有着不同寻常的数学天赋的印度女孩；但这主要是数学。</cite>

我的父亲是一名开发者，也是一名数学爱好者，他不读普通的童话故事，而是每天晚上给我们读一个《Lilavati》中的故事。我们和姐姐一起试图解决这个问题。信不信由你，这是我最喜欢的回忆之一——那些小的头脑风暴和讨论。

《莱拉瓦蒂》一书于 1926 年首次出版，我的这本书是 1964 年出版的。它很古老，你可以在报纸上感觉到，也可以在有点过时的语言中感觉到。我有点担心现在的孩子们可能不会欣赏这本书的内在，因为它的外表并不吸引人。尽管如此，我还是迫不及待地想把它读给我的女儿听(她现在只有一岁，不幸的是还有很长的一段时间)，并且真的希望她会像我一样喜欢它。毕竟，书只是一个工具，真正重要的是一起度过的时光。
