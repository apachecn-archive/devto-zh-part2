# 德瓦特 SSIS Marketo 数据流组件让营销自动化变得简单

> 原文:[https://dev . to/devart software/marketing-automation-made-easy-with-devart-ssis-data-flow-components-for-marketo-gi8](https://dev.to/devartsoftware/marketing-automation-made-easy-with-devart-ssis-data-flow-components-for-marketo-gi8)

Devart 是一家公认的为开发人员和数据库管理员提供专业数据库管理软件的供应商，它发布了支持 Marketo 的新版 SSIS 数据流组件。它们允许用户通过 SQL Server Integration Services(SSIS)将 Marketo 模板、订户、活动和许多其他对象与云应用程序和数据库集成在一起。组件列表包括优化的 Marketo 源、Marketo 目的地和 Marketo 查找组件，这些组件提供了自己的 Marketo 连接管理器。

为了实现快速、完美的营销自动化，SSIS Marketo 数据流组件提供了以下功能:

```
• Building SSIS-based integration solutions for Marketo in the blink of an eye
• No-brainer automating Marketo integration via SSIS Data Flow tasks
• Handy importing and exporting CSV files to Marketo
• Synchronizing Marketo with SQL Server or other data sources
• And many more 
```
