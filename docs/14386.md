# 然而，梅根编码

> 原文:[https://dev.to/meganmarie610/nevertheless-megan-coded-46pi](https://dev.to/meganmarie610/nevertheless-megan-coded--46pi)

## [](#i-began-to-code-because)我开始编码是因为...

...我哥哥有。高中的时候，他做的一切，我都做了。他吹萨克斯，我吹萨克斯。他上了计算机科学课，我上了计算机科学课。我们现在都是工程经理了。

## [](#i-continued-to-code-because)我继续编码是因为...

...我无法摆脱它。大学开始是化学专业，上了计算机图形学课，转到了计算机科学专业。在 ILM 工作，目标是生物开发艺术家，然后是效果艺术家，而是为艺术家编写 python 脚本和工具，喜欢为有才华的艺术家解决棘手的问题。从事视觉特效 6 年后，我去了神奇的独角兽创业公司 land。我找到了 Bloc，并深入到代码中，构建了一个平台，让这么多人可以追求自己的开发和设计生涯。

## [](#i-want-to-brag-about)我想吹嘘一下...

...我们工程团队里的优秀女工程师。我管理着 Bloc 的工程团队，这是一家为所有想成为网页开发者或设计师的人提供服务的在线教育公司。

Courtney 在工程团队的一年工作即将结束。她拿着全额奖学金参加了激励训练营，而且是一个不断学习的人。仅仅一年后，我们真的可以信任她为我们的学生和导师构建任何新功能。当你在这个问题上需要另一个大脑时，她是一个很好的搭档。

Meghana 在完成了 web 开发人员跟踪项目后，于上周作为实习生加入了我们的团队。你可以称她为 bug 粉碎者，她已经在头两周修复了这么多 bug。她的第一份职业是硬件，她休息了一段时间来抚养她的两个孩子，现在她加入了我们，开始了她的第一份软件开发工作。

今天，我们进行了一个下午的学习，讨论了块范围、let 和 const。庆祝国际妇女节的好方法。

## [](#my-advice-for-allies-to-support-women-and-nonbinary-folks-who-code-is)我对支持女性和非二进制代码的盟友的建议是....

...假设我们在那里参加会议，如果我们在会议上。假设我们在会议中有有趣且有影响力的观点。给我们反馈，用挑战你所有同事的方式挑战我们。我们可以为您做同样的事情！
