> 原文:[https://dev . to/dvidsilva/front-end-challenges-to-be-better-developer-codecorgi-50nh](https://dev.to/dvidsilva/front-end-challenges-to-become-a-better-developer---codecorgi--50nh)

嗨！我最终把它放在一个分享起来并不尴尬的状态，codecorgi 是我一年多来的一个想法，我们给用户带来的挑战不是琐碎的代码库，没有教程，没有手握，只是代码和功能请求。就像正式工作一样。

派生并克隆 repos，在本地运行，添加所需的功能或修复，推送到 github，并将您的版本提交给 codecorgi，以便其他社区成员可以给你反馈，比如工作中的代码审查。现在有 7 个挑战，从超级简单的 HTML 和 CSS 到获取 React 应用程序的数据，我计划根据时间，大约每周增加新的挑战。

还有一个个人资料页面，你可以将你的项目添加到[http://codecorgi.co/profile/dvidsilva](http://codecorgi.co/profile/dvidsilva)并展示你的技能，仍然有点粗糙，但正如一切一样，它会不断改进。

此外，这一次，所有代码都是开源的，[https://github.com/corgicode/api](https://github.com/corgicode/api)[https://github.com/corgicode/frontend-react](https://github.com/corgicode/frontend-react)如果你想查看代码库，看看应用程序的进展；如果你正在寻找一些项目来做贡献，我很乐意带你走一遍，并有一些配对会议。我们的路线图也是公开的，你可以在华夫上看到，【https://waffle.io/corgicode/frontend-react】[https://waffle.io/corgicode/api](https://waffle.io/corgicode/api)的
