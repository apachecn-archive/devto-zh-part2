对于从 REST APIs 导入数据并将其发送到存储中，您会推荐什么服务或产品(最好是云中的)?如果可能的话，这个服务或产品应该是用户友好的，可以用最少的代码进行配置？

在一个完美的世界中，我设想了一个管理控制台，您可以在其中提供 API 的 URL、认证凭证、数据模式、目的地信息(即 MS SQL、MongoDB 或另一个 REST API)。您可以安排数据导入，或者设置一个事件来触发导入。
