Go 有一个 http 包，里面有简单的 API 来完成 HTTP 操作。例如，如果你想下载一个页面，你需要做的就是这个。

```
resp, err := http.Get("http://example.com/")
if err != nil {
    // handle error
}
defer resp.Body.Close()
body, err := ioutil.ReadAll(resp.Body) 
```
