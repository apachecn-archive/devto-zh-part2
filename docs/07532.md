## [](#2-extract-to-the-system-theme-folder)#2:解压到系统主题文件夹:

你可以将下载的主题文件解压到系统的主题文件夹中(即 **/usr/share/themes** )。您可以使用[终端](https://fossnaija.com/the-anatomy-of-the-linux-command-line/)(命令行界面，CLI)这样做:

```
unzip /path/to/downloaded/theme/zip -d /usr/share/themes 
```

## [](#3-set-the-desire-theme-using-tweak-tool)**#3:使用调整工具设置欲望主题。**

ubuntu 有两个主要的调整工具；统一调整工具和 T2 Gnome 调整工具。它们都执行相同的功能，管理和调整一些深层的系统设置，这些设置在默认的 ubuntu 设置管理器中是不可用的。无论你选择哪个，我们都服务得很好(这里我使用了 unity tweak 工具)。
