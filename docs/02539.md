# 和我的鼠标分手(Mac 版)

> 原文:[https://dev . to/GK sander/breaking-with-my-mouse-MAC-edition-49md](https://dev.to/gksander/breaking-up-with-my-mouse-mac-edition-49md)

作为一名开发人员，我完全是为了提高工作效率，所以我正试图强迫自己花更少的时间将右手从键盘移到鼠标上。我一直在思考如何舒适地变得更加以键盘为中心。我不是这方面的专家，但我认为值得分享我迄今为止的旅程。

我在 Mac 上工作(MBP 在工作，iMac 在家里)，因此这篇文章将讲述我发现的一些让我的生活变得更简单的技巧(并且让我的手保持在键盘上)。

## 第一步:改变大写锁定键的用途

我不太需要我的**大写锁定**键。我真的只在我用愤怒的语气写评论的时候用它(提醒自己有机会的时候去修正/添加一些东西)。但是，**大写锁定**键是*，所以*很方便地位于我的左手小指旁边(顺便说一下，这不是超级灵活)。如果我能用那个**大写锁定**键做些有用的事情就好了...事实证明，macOS 实际上允许您将 **Caps Lock** 键重新绑定到任何其他修饰键(Control、Option、Command)。方法如下:

1.  打开*系统偏好设置*。
2.  点击“键盘”。
3.  找到“修饰键...”按钮并点击它。
4.  选择你的键盘(如果需要的话)，然后从“大写锁定键:”下拉菜单中，把它改成你喜欢的。我使用 Option/Alt ，因为似乎没有很多系统级热键使用**选项**。
5.  *瞧吧*！现在按下**大写锁定**并不能真正将你锁定成大写字母，它具有**选项**键的功能。

## 第二步:设置任务控制

MacOS 的任务控制相当不错(我认为，无论如何)。拥有多个桌面并在它们之间轻松导航是件好事。默认情况下，我相信 macOS 会绑定 **Ctrl + →** 来“右移一格”， **Ctrl + ←** 来“左移一格”，等等。这是一个不错的默认设置，但我想用我的新**大写锁定**键，因为它比**控制键**更容易让我笨拙的左手指尖按到那个键。所以，我继续前进，改变了一些任务控制快捷键。方法如下:

1.  打开*系统偏好设置*，然后点击“键盘”。
2.  点击“快捷键”标签，这允许你修改默认可用的键盘快捷键。
3.  从左边选择“任务控制”,从那里你可以开始改变快捷键。我重新绑定了以下内容:
    *   将“任务控制”设置为**选项+ I** 。
    *   将“应用程序窗口”设置为**选项+ K** 。
    *   将“左移一格”设置为**选项+ J** 。
    *   将“右移一格”设置为**选项+ L** 。
4.  我选择了这些绑定，这样我就可以用我的左手去触发**选项**，然后把我的右手放在 home-row 上，控制任务控制。这允许我在桌面之间快速来回导航。

## 第三步:为窗口管理设置眼镜

向阿里·斯皮特勒大喊(我能@人吗？)和她最近的帖子，让我进入了奇观。我以前使用过 BetterSnapTool，但这不是一个免费的应用程序，它的功能比我需要的要多。[在这里欣赏奇观](https://www.spectacleapp.com/)。眼镜基本上允许您轻松地将一个窗口与屏幕的某个部分对齐。因此，您可以快速将窗口捕捉到左半部分、右半部分或全屏等..一旦你习惯了使用这样的工具，就没有回头路了。尝试使用它一段时间，然后使用没有安装它的苹果电脑...很痛苦。

一旦安装了眼镜，你必须在你的 Mac 上给应用程序权限。一旦完成，我打开它，清除所有的快捷方式。我想继续使用我的**大写锁定**键！所以，(除了别的以外)我设置了以下内容:

1.  “居中”到**选项+ ↓** 。
2.  “全屏”到**选项+ ↑** 。
3.  “左半部”到**选项+ ←** 。
4.  “右半部”到**选项+ →** 。

## 第四步:使用阿尔弗雷德获得更大的力量！

[Alfred](https://www.alfredapp.com/) 是聚光灯的替代工具，它比*做得更多*。免费版可以让你做类似聚光灯的事情(和其他一些事情)，而且相当棒。我强烈建议你现在就用它取代聚光灯。

Alfred 还有一个名为“Powerpack”的付费扩展包，价格约为 19 欧元(约 22 美元)。我对我的自由软件很感兴趣(在预算上)，但是如果我要在软件上花钱的话，我会在 Alfred Powerpack 上花钱。你可以用它做很多很酷的事情，但是我只展示一件和这篇文章相关的很酷的事情。

一旦你安装阿尔弗雷德，你可以选择什么热键，你想打开阿尔弗雷德搜索栏。我使用**选项+ A** ，这样我就可以做**大写锁定+ A** 来快速打开它，而不用从主行移动我的手指。

现在，在本节的剩余部分，我将讨论一项功能-“工作流”，它仅在 Powerpack 中可用。如果您有 Powerpack，您可以使用 Alfred 创建自定义的“工作流”。这些可以包括以各种方式触发的各种动作。我们可以使用这个功能来添加额外的热键！我发现我喜欢将热键绑定到某些应用程序，这样我就可以使用快速热键打开一个应用程序。Mac 的应用程序切换器(通过**命令+ Tab** )很不错，但当我管理 4 或 5 个应用程序时，我觉得它并不太快。

对于 Alfred，我喜欢以一种我很容易记住的方式将热键绑定到应用程序。以下是我举的一些例子:

1.  让**选项+ C** (也就是现在的**大写锁定+ C** )打开 VSCode。在我心目中，“C”代表“代码”。
2.  让**选项+ W** 打开 Chrome。在我心目中，“W”代表“网络浏览器”。
3.  有**选项+ T** 开端子。再说一遍，“T”代表“终端”。
4.  让 **Option + M** 打开 Spotify。“M”代表“音乐”。

为此，打开 Alfred 的设置/偏好设置窗口(您可以从 Alfred spotlight 搜索中搜索“Alfred ”),然后转到“工作流程”选项卡(同样，假设您有 Powerpack)。下面是我们如何设置**选项+ C** 来打开 VSCode:

1.  在左侧面板的底部，有一个小小的 **+** 按钮。点击创建一个新的工作流程，并选择“空白工作流程”。填写工作流程的名称。这将会给你一个空白的画布。
2.  右键单击空白画布，您会看到一个下拉菜单。选择“触发器>热键”。![](../Images/43af63b688fa137b6218ffd299f8ab52.png)
3.  一旦你选择“热键”，它会问你想要选择哪个热键。我选择了**选项+ C** ，你其实可以在这里按**大写锁定+ C** 。选择热键后，单击保存。
4.  现在，再次右键单击画布并选择“操作>启动应用程序/文件”。这将打开一个对话框，供您“放入一个程序”。为此，我触发了 Alfred，搜索 vs code——一旦找到，就把它从 Alfred 拖到对话框中。完成后按“保存”。![](../Images/b2ff1e4511a5e0abd3c3950021e69143.png)
5.  现在，我们只需要将我们的*触发器*链接到我们的*动作*。将鼠标悬停在画布上的热键触发器元素上。它应该突出显示元素，并在其一侧显示一个小“旋钮”。单击并按住它，开始向动作方向拖动它。你应该能把这两件事联系起来。![](../Images/8eaf9c153597fa15b9e89ce169ca4833.png)

这就对了。现在点击 **Caps Lock + C** 将打开 VSCode！一旦你设置好了其中的一些，你就可以开始轻松地在应用程序之间切换，而不需要把你的手从键盘上拿开！

当我想在笔记本电脑上工作的时候，我发现这非常有用，而且我的应用程序还是全屏的。我可以在 Chrome、VSCode、我的终端、我的 MongoDB GUI 等之间切换。真的很快！
