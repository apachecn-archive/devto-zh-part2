## [](#the-problem)问题:

使用下面的代码片段，对于一个 **`font-size`** 等于 *12px* 的段落元素，您希望该段落的`line-height`等于其`font-size` ( *24px* )的两倍。如果将下面的哪个`line-height`值应用于`body`元素，它将实现这一点？
(注意浏览器默认字体大小为 *16px* )

```
<p style="font-size: 12px">Having fun solving CSS Quiz</p>

<style>
body {
  line-height: <value>;
}
</style> 
```
