## 一个最小的基于网络的 iTunes Windows 应用程序的遥控器

为了让生活变得简单一点，下面是我的小应用程序

nodeJS 服务器在本地网络上寻找请求，并生成一个 python 脚本来执行远程操作

谷歌助手不和你 PC 上的 iTunes 对话？让我们来解决这个问题-*噗* **pyTunes**

只需前往 [Dialogflow](https://dialogflow.com/) ，将您的 Google Actions 和 Fulfillment URL 设置为您的服务器地址(使用 ngrok 将 localhost 公开)

回购的名称是 pyTunes，少数代码是在 python 中🙃
