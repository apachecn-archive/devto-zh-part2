## 我开始/继续编码是因为...

我喜欢能够从零开始构建东西，并释放我的创造力。

我参加了普通中等教育证书和高级证书的 ICT 考试，但决定在大学攻读经济学学位。3 年后，我毕业了，在实习、志愿工作和通过指导与年轻人一起工作中，我有些飘忽不定。通过在 Macromedia Flash 上创建动画，我重新发现了我对构建事物的热爱，然后转向了 web 开发。我做过的最好的决定。

## 我最近克服了...

有效使用 Git 和 Github！在通过多种资源来提高我的理解和使用两者之后，我终于找到了一个改变一切的资源。对于任何正在学习或想知道学习 Git 和 Github 的最佳地点的人来说，请前往 Udacity 的 Git 和 Github 课程。你不会后悔的！

## 我想吹嘘一下...

我写的关于 dev.to 和 Medium 的技术文章。我觉得通过写作，我学到了很多东西。理解与 web-dev/frontend 相关的主题是一回事，但在不到处跑的情况下向其他人解释是另一回事！我想吹嘘的另一件事是我在 Codepen 上的笔。我收集了比以前多得多的工作，我觉得我(终于)取得了进步。
