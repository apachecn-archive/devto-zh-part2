# 然而，Sadukie 代码

> 原文：<https://dev.to/sadukie/nevertheless-sadukie-codes--1876>

在我的记忆中，我是在八年级开始接触编程的，并且真的很喜欢。我叔叔记得我两岁的时候让我坐在他腿上看他写代码，所以这可能也有帮助。但我从未想过我会以此为生。我生长在一个几乎买不起电脑的家庭。尽管经济上处于劣势，我最终还是编码了。

高中的时候，我们有一门计算机编程课，是我大三的时候上的。我和我最好的朋友是班上唯一的女生，这并没有让我们放慢脚步。其实那节课我们更无聊，因为我们知道自己在做什么。尽管我是一个被困在 Mac 实验室的 Windows 女孩，我还是继续写代码。

在高中，我遇到了一个家伙，他看到了我作为程序员的潜力。我们一起参加了高中的网站团队。他建议我学习 HTML，我不知道他怎么知道我会编程，但我相信他的推荐。因为他，我爱上了他...不管是用 HTML 还是用他。很久以前，我生命中的爱在高中结束时真正开花结果。它们一直延续到今天。我嫁给了一个很棒的男人，即使 20 多年过去了，他仍然看到了我的潜力。

在高中和大学之间，我的实习经历让我看到了雄心勃勃、充满好奇心的程序员。我遇到了这个领域的人，他们希望看到我成功，把我介绍给人们，向我展示技术。有伟大的导师和欣赏年轻开发人员的高级开发人员真的鼓励我继续走这条路。

当我从事 IT 工作时，我仍然为了乐趣而编码。Visual Basic，SQL，C#，HTML，JavaScript，CSS，Perl，Python，Ruby，PHP，还有现在的 R...我曾在业余时间学习过许多语言，即使这并不是我的职业。当我在 it 岗位上碰壁时，感觉自己在职业发展中碰了一堵砖墙，我丈夫鼓励我去找一个开发人员的职位，去我喜欢的地方。我的禅在编码。

如今，我更多的是教别人编程——无论是让新人从事第二/第三/第四职业，还是帮助现有的程序员学习一门新语言，我都是在教别人如何应用各种语言。当我厌倦了看我的课程表，知道我在做什么吗？我正在学习 Python、R 和数据科学的在线课程。为什么？**因为...Sadukie 代码。**

虽然可能有怀疑者和反对者...虽然有些人可能想说“祝你好运”而不提供帮助...我不会让他们拖累我。当我写代码的时候，我就进入了这个奇妙的领域。如果你喜欢编码，就去接触社区，因为这里有和你一样的人在这个领域中享受乐趣。不要让消极拖你的后腿。跟随你的好奇心！
