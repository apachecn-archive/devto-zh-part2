过去的几个月对我来说是无代码区。我真的没有用 Java 做过任何事情。这主要归功于我忙于为学校写论文，我的学位将在我目前的课程结束后完成。它主要关注文档而不是实际的代码。我有点不知所措，当我有时间编码的时候，我就是没有。我累坏了。我的蜡烛一直点着。

我得到了一个急需的休息，再次充满了动力，但是有什么建议可以防止这种倦怠再次发生吗？我不想再浪费 3 个月了。
