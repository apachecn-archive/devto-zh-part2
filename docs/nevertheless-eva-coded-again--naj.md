# 然而，伊娃编码...又来了！

> 原文：<https://dev.to/lavigi/nevertheless-eva-coded-again--naj>

去年，我分享了我作为青少年开发者的第一步的故事...在上世纪 80 年代！今年，我想吹嘘一种不同的成就。

在 20 多年的中层管理岗位上，我谈了很多关于软件生命周期的问题，但我并没有亲自开发它，我发现了一种方法，可以让我的大脑专注足够长的时间，以便能够再次编码。

在过去的两年里，在业余时间，我已经成为一名快乐的初级 JavaScript / node.js 开发人员。我做的第一件事是我 16 岁时制作的一个简单游戏的 21 世纪版本(当然，是用 JavaScript /jQuery 替换 Turbo Pascal)。我为自己掌握了一些 Alexa 技能而自豪。

所以，在那里。如果我失去了中层管理工作，我有可以转移的技能。我觉得这很棒。
