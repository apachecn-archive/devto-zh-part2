## 它还活着！

一段时间以来，我一直想把我现在的从 Wordpress 上移走，但是时间总是一个因素。作为一名开发人员，我发现自己生活在 *Visual Studio 代码*中，所以我决定建立一个博客，它可以:

*   使用我的 Visual Studio 代码
*   使用我的 git 技能

我知道 github 有 [Github 页面](https://pages.github.com/)，它用了 [Jekyll](https://jekyllrb.com/) 。所以我创建了一个 Github 页面。这是第一天。我为网站选择了一个最小的主题，网站的每一个变化我都会以博客的形式记录下来。把它们当成发行说明吧！

随着我继续前进，我改变了风格和布局等，但现在(和后代)这里是第一天的主页:
