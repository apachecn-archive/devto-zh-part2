# [视距地图](https://stadiamaps.com)

## 为什么

地图不应该很难也不应该很贵。

我们创建了 [Stadia Maps](https://stadiamaps.com) 来解决这两个核心问题。大多数在线地图都很贵，而且当你有钱的时候，很难实现和整合。

我们从基本的静态地图和路线开始。从简单开始，我们可以提供公平透明的定价。随着我们继续扩展我们的服务(地理编码和卫星地图在我们的短名单上)，我们将保持一个简单的定价模式。

我们认为地图应该服务于客户，而不是地图提供商。一些地图提供商在您的数据旁边显示您的业务竞争对手、评论(可能是欺诈性的)或他们自己的徽标的地图。我们相信，一个设计良好的底图是让您的数据熠熠生辉所需要的一切。随着我们的扩张，我们将继续专注于将您的公司和数据放在首位。

那只是开始；我们正在开发几个简单的工具来帮助企业主和开发者在他们的网络和移动应用中使用地图。最终，我们希望让其他人也能创造奇妙的地图体验。

## 如何

*   我们努力提供公平定价和透明的计划。
*   我们为各种各样的环境提供美丽的主题。
*   我们从不追踪*或出售用户数据。
*   我们从不在您的数据旁边显示评论或潜在竞争对手(“相关”业务)。
*   我们从不在您的地图上显示广告。
*   我们正致力于 JS 库，让交互式地图设计变得不那么糟糕。

*在必要的细节之外实施速率限制和类似的技术措施，以确保每个人都有良好的体验
