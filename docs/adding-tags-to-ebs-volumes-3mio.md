# 向 EBS 卷添加标记

> 原文：<https://dev.to/ajardin/adding-tags-to-ebs-volumes-3mio>

如果你想有一个高效的账单跟进，标记你所有的亚马逊资源是必须的。在大多数情况下，这很容易，但并不总是像自动缩放组中的 EBS 卷那样。即使天生不可能，AWS 仍然有一个解决方案。

目前，自动缩放组不会自动将其标记应用于附加到其 EC2 实例的 EBS 卷。只有一个待定的功能请求，没有提供 ETA。也许在接下来的几个月有可能，也许没有。

在那之前，我们必须自己实施一些事情。

在与 AWS 支持讨论之后，我们得出结论，最佳解决方案是通过启动配置在 EC2 实例启动时实现一个定制逻辑。该脚本构建在 AWS CLI 之上，使用 EC2 实例元数据。让我们看一看。
