# 如何在 Android 上开始游戏开发

> 原文：<https://dev.to/bauripalash/how-to-start-game-development-on-android-5fe>

[![.](../Images/8a405e6c4a589f23a36923cdffb6390c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Gz34udCC--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://palash.tk/assets/images/gamedev-android.jpg)

游戏开发是有趣和创造性的工作。大多数游戏玩家，无论是 Android 还是 PC，都想开发自己的游戏并与世界分享。因此，今天我们将寻找一些方法，我们可以用来建立自己的游戏！

游戏开发的过程可能看起来很简单，但它需要大量的辛勤工作，奉献精神，创造力，编程知识，图形设计技能等等...

[![cat](../Images/ddfad3fed1f532abc9dd45bd6dd69871.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--DQL_WjRS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://media1.tenor.com/images/ff787c5e41b7c47a5f4b1d4dcbc76074/tenor.gif%3Fitemid%3D5947150)

但是我知道，像我和你这样的人总是想开发他们自己的游戏，所以今天我们将寻找一些方法来创建我们自己的游戏，并玩它们。

> 如果你想开发 android 应用，请阅读[如何在 Android 上开始 Android 应用开发](https://palash.tk/Android-AppDev-On-Android)

那么，不再多说，让我们开始吧。

## TIC-80

这是一个全新的环境，一台你自己的幻想电脑，在你的手机里。

在这里，您可以创建自己的游戏并与他人分享。开发，如代码编辑器，精灵编辑器，声音生成器等等。

这是一个非常有趣的方式来创造和建立复古风格的游戏。

[![TIC 80](../Images/ee7e6d8f406b2f7ced374659ea5bab74.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--tvX_JOFM--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://user-images.githubusercontent.com/1101448/29687467-3ddc432e-8925-11e7-8156-5cec3700cc04.gif)

> 它不仅可以在 Android 上运行，还可以在 Windows、Mac 和 Linux 上运行，甚至可以在浏览器上运行:[访问他们的网站了解更多信息](https://tic.computer/)

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.nesbox.tic)

## ·Sketchware

我在之前的文章[如何在 Android 上开始 Android 应用程序开发](https://palash.tk/Android-AppDev-On-Android)中也提到了这一点

它既适合应用程序开发，也适合游戏开发。但是我注意到一件事，Sketchware 上的游戏开发不像应用程序开发那样简单。

[![Sketchware](../Images/b10a94cc855ca126dc3e31561c45b501.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Ei8UfvlI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/http://sketchware.io/images/merged1.5x.gif)

不过，如果你有足够的创造力，在 Android 上开始游戏开发并不困难。

你可以用简单易学的程序设计创造出令人惊奇的游戏。它也有其他伟大的功能，这将是有益的游戏开发。

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.besome.sketch)

## 援助

这是 Android 最古老的 IDE 之一。使用 LibGDX 库，它可以为 Android 开发应用程序，也可以开发游戏。

可以写 Java，C++开发 app 和游戏。

[![AIDE](../Images/f87b7878152518fca3fb54142b3d5ccd.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--fUNPtSBi--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://www.android-ide.com/img/aide-devices.png)

它有一个强大的 ide，语法突出和许多更强大的功能。

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.aide.ui)

## ·游戏厂商份额

几周前，当我在 Play Store 上寻找印度时报应用程序时，我发现了它，我不知道它是如何出现在搜索结果中的，但我安装了它，并进行了尝试。

这是一个非常好的应用程序，但只要稍加改进，它就可以成为游戏开发的热门应用程序。

[![Game Maker Share](../Images/b4c2a4ad63db42cd25db1f5c00c1f84b.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--AbuSA5ex--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://lh3.googleusercontent.com/Bqzo3fhe6zeTyU3qM1fcvjQRG5eHTrvVBtvsHey-9H4_JlOHeKqm2sG0gsliG0oIZEg%3Dw774-h440-rw)

它有很好的功能，你可以在各种精灵中选择添加功能，完成后你可以提交到那里，这样你的朋友就可以玩你的手工游戏了。

[从游戏商店安装](https://play.google.com/store/apps/details?id=com.vejaqui.gamemakersharefree)
