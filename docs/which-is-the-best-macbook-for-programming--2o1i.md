# 编程用的 macbook 哪个好？

> 原文：<https://dev.to/aurelkurtula/which-is-the-best-macbook-for-programming--2o1i>

我的 macbook pro(2011 年初的)刚刚坏了。我以为是内存问题，但不是。我有一种感觉，它可能不值得修理。简而言之，我想要一台新笔记本电脑。

这一次我真的很想买最便宜的 macbook(计划几年后全力以赴)。但是，我不知道不同的规格意味着什么。

macbook air 是编程的好笔记本吗？目前我只关心节点开发，但是不管我探索什么编程语言(比如移动开发)，它是一台好的笔记本电脑吗？
