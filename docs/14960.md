# gearman 简短摘要

> [https://dev . to/alexribeiro dev/gearman-um-简短摘要-3kp3](https://dev.to/alexribeirodev/gearman-um-breve-resumo-3kp3)

[![](../Images/05cd65b4dd13cde3433748f26b41c886.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--WSktFCgR--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/480/1%2AVwt9QsSurFa0BF-Gw0GgjQ.png)

gearman 提供了一个通用框架/应用程序，用于浏览其他机器或进程的工作，这些工作更适合于特定的工作。

允许您在不同语言之间并行工作、平衡处理和调用函数。

[![](../Images/33ec7cde5c91efb9e88d6ab7f1453b77.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--SMzLXiDZ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/479/1%2AAnWo6lxBowRxe5fbDljlGg.png)

它可用于多种应用程序，从高可用性站点到数据库复制事件传输。

[![](../Images/751bd2aed1bbadadd31f3e4400cbdbf8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--0vKrDyzH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/478/1%2A5rFs1UZt95PLS3l_HVYhlQ.png)

换句话说，是分布式处理如何沟通的神经系统。

[![](../Images/b90dacb9f460cf8ef3dddb296a0e97bd.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CpRB63fs--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/390/1%2AGXfGLe0JikBX85G66tAZ9g.png)

gearman 的一些优点:

*   开放源码
*   多种语言
*   灵活的
*   快车
*   可嵌入
*   无单点故障
*   不受数据大小限制
*   可扩展

```

$ add-apt-repository ppa:gearman-developers/ppa
$ apt-get update
```

### [](#exemplo)实例

我们将创建两个 PHP 脚本，一个用于客户端，另一个用于 worker。

客户:

```
\<?php
$client = new GearmanClient();
$client-\>addServer();
print $client-\>do("reverse\_string", "Hello World!"); 
```

<svg width="20px" height="20px" viewBox="0 0 24 24" class="highlight-action crayons-icon highlight-action--fullscreen-on"><title>Enter fullscreen mode</title></svg> <svg width="20px" height="20px" viewBox="0 0 24 24" class="highlight-action crayons-icon highlight-action--fullscreen-off"><title>Exit fullscreen mode</title></svg>

工人:

```
\<?php
$worker = new GearmanWorker();
$worker-\>addServer();
$worker-\>addFunction("reverse\_string", function ($job) {
 return strrev($job-\>workload()) . "\n";
});
while ($worker-\>work()); 
```
