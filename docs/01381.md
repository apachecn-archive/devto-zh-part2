# 海洋下一个 Bimbo 主题

## 装置

1.  打开边栏上的扩展
2.  搜索“海洋下一个荡妇”
3.  安装它
4.  重新加载编辑器
5.  按下`⌘K + ⌘T`
6.  从列表中选择“Oceanic Next Bimbo ”,然后点击回车
7.  完成的
