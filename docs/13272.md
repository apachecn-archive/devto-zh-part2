# 加密很难

> 原文:[https://dev.to/apotheon/crypto-is-hard-3fbe](https://dev.to/apotheon/crypto-is-hard-3fbe)

软件容易出错，安全软件需要对。这使得好的安全软件变得很难。

这对初学者来说尤其困难，因为(根据我的经验)很少有好的文档、例子和通用的“hello world”级别的安全软件开发的教程，它们实际上是正确的。试着找一个例子，说明如何编写一个使用 ChaCha20 甚至 AES 加密和解密文本的工具，这对于编写 web 应用程序、测试框架和 Unix 命令行工具的人来说是容易理解的。

这是一个大问题，但似乎没有人注意到，或关心。
