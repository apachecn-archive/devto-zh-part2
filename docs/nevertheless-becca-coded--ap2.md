# 尽管如此，贝卡编码

> 原文：<https://dev.to/beccaanderton18/nevertheless-becca-coded--ap2>

## 我开始/继续编码是因为...

我不得不！在我攻读摄影学位期间，我们的任务是为我们的工作创建一个作品集网站，我从那里迷上了它。毕业后，我继续在当地一家初创公司实习，以提高自己的技能，直到几个月后，我在一家小机构找到了第一份设计师的工作。

从那以后，我担任过各种不同的角色，包括在一家大型保险公司的内部网络团队工作，现在我在英格兰北部的一家快速发展的机构担任前端开发人员。

我继续写代码仅仅是因为我喜欢它，而且我在解决你已经困扰了一段时间的问题中找到了巨大的满足感。

## 我是这方面的专家...

没什么！但这并没有阻止我作为一名开发人员学习和努力成长和提高。我主要关注 JavaScript 开发人员，从更小、更简单的项目到更大、更复杂的系统，比如 SPA。最近，我一直在涉猎 PHP，试图提高我的后端系统知识。

## 我目前正在入侵...

在内部，我一直致力于开发前端 CLI 工具和 linters，以改善我们的工作流程，并帮助前端团队的其他人一起成长。

## 我对其他女性和非二进制程序员的建议是...

坚持下去！作为这个行业的少数派是令人畏惧的，我花了几年时间去习惯和适应它，然后才觉得自己可以开始茁壮成长。因此，更重要的是花时间鼓励和赞美行业内的每个人，让每个人都成为最好的自己。
