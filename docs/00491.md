# 什么是对象？

> 原文:[https://dev.to/yoonsl/what-is-object-3gf0](https://dev.to/yoonsl/what-is-object-3gf0)

在学习 JavaScript 的时候，对象是你最先接触到的东西。当我们第一次学习 object 时，它似乎真的很简单和容易。毫无疑问，object 在开始时很容易使用，但是随着你对 object 了解的越来越多，事情就没那么简单了。简单地说，对象是具有键和值对的数据。

在本例中，我们使用 Object.keys()和 Object.values()指出每个键和值。这些方法将每个键和值分隔成一个数组。在 forloop 的帮助下，我们可以分别显示每个键和值。

但是随着我们对对象了解的越来越多，我们可以看到它不仅仅是一个简单的键值对数据。对象可以用于 OOP(面向对象编程)。我们知道，世界上的一切(大多数，可能不是全部，不确定)都是对象。物体有一些我们不会同时想到的属性，但是我们会意识到。例如，我们有一个罐子，对我们来说这只是一个为了许多目的而包含其他物体的物体，但是如果我们深入地思考它，它可以有许多描述这个物体的属性。罐子有多结实？这个罐子是用什么材料制成的？罐头是什么颜色的？它代表了什么？这些只是一个罐子拥有的众多特性中的一小部分！有了这些属性，我们就能够虚拟地创建一个真实的对象。在下一篇博客中，我将会谈论更多关于 OOP 的内容，并且会有更多的例子！
