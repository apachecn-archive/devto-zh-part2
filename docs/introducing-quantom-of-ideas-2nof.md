# 介绍“量子思想”

> 原文：<https://dev.to/dmerejkowsky/introducing-quantom-of-ideas-2nof>

*最初发表于[我的博客](https://dmerej.info/blog/post/introducing-quantom-of-ideas/)T3。*

今年我决定尝试一些不同的东西。

“思想的量子”是一系列新文章的标题。你可以在标签为“想法”的文章列表[中找到它们](https://dmerej.info/blog/tags/ideas)

在每篇文章中，我会提出我对某个话题的想法。

大多数时候，这些想法仍然处于“草案”状态:仍然有实际问题需要解决，它们甚至可能不值得实施。

所以为什么要分享这些呢？

我希望他们能对某些问题提供不同的观点，并引发有趣的讨论。

所以，请随意分享它们，如果你有足够的勇气开始研究它们，请让我知道。

此外，更重要的是，不要犹豫给我[反馈](https://dmerej.info/blog/pages/feedback/)。如果你确实喜欢我的想法，一个小小的正面评论将会受到赞赏，如果你不喜欢，你可以吼我是一个对现实世界如何运作毫无头绪的白痴。
