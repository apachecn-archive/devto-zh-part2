# [](#the-chuck-norris-library)查克·诺里斯图书馆

Chuck Norris 库是用 C++编写的，它包含一个类，当调用`getFact()`方法时，该类能够给你随机的 Chuck Norris 事实:

下面是使用方法:

```
#include <ChuckNorris.hpp> 
ChuckNorris chuckNorris;
std::string fact = chuckNorris.getFact(); 
```
