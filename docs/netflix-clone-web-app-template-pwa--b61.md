# zuix-web-flix

一个进步的 Web 应用程序模板，灵感来自网飞移动应用程序。

这个模板仅使用 HTML、JavaScript 和 CSS 构建，因此它最终可以与您喜欢的开发环境和构建工具集成。

## 特征

*   网飞手机应用的设计灵感
*   使用 zUIx.js 的模块化和基于组件的结构
*   浏览器内捆绑器:可以将所有资源打包在一个文件中，提高加载速度
*   PWA 灯塔评分 100/100(！)
