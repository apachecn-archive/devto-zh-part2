# 回到静态

> 原文：<https://dev.to/hybrid_alex/back-to-static-3l0i>

今年年初，我将我的个人网站从 Jekyll 迁移到 Craft CMS。

在那段时间，我对提高我的 Craft CMS 开发技能很感兴趣，所以把我的网站作为学习的平台是有意义的。

从那时起，我学到了很多东西，并在此过程中制作了一些[截屏图](https://www.youtube.com/watch?v=VSXglzqJ2Z4&list=PLmv4wIIvTDgGX6bpR3cdVddTDTR8DbDmO)。花在学习和实验上的时间对我最近如何处理新的 CMS 构建产生了很大的影响，成功！

最近我一直渴望回到更简单、更轻松的生活。我看到 Zach Leatherman 最近发布了 11 eventy T1，它是基于 Javascript 的 Jekyll 的替代版本。

经过大约一周的设置、设计/开发和内容迁移，我有了一个新的网站设置。与我以前的 Jekyll 版本相比，我真的很喜欢它提供的灵活性。 <abbr title="Eleventy">11ty</abbr> 管理我的内容和模板，我有自己的风格和脚本层。

在[源](https://github.com/alexcarpenter/alexcarpenter.github.io)随意取一个峰值。
