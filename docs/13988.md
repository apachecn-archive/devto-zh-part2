# [](#preparation)准备

你唯一需要的是最新的 NodeJS 和你最喜欢的编辑器，我的是 Visual Studio 代码。现在为我们的项目创建一个文件夹，打开一个命令行，用 npm init 初始化一个 npm repo，并安装一些我们需要的库:

```
npm i --save discovery-swarm dat-swarm-defaults portfinder get-port 
```
