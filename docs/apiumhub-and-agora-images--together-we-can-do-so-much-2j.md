# Apiumhub 和 AGORA Images——我们可以一起做很多事情！

> 原文：<https://dev.to/apium_hub/apiumhub-and-agora-images--together-we-can-do-so-much-2j>

AGORA images 是第一个全球图像社区，不同的品牌和媒体无需中介或佣金即可获得任何人的图像。这是一个基于巴塞罗那的摄影应用和网络平台，已经成为摄影行业、公司和品牌的重要在线营销渠道。今年夏天，该公司成功完成了第二轮融资，融资额超过了 2M。而 [Apiumhub](https://dev.to/) 在这个过程中扮演了至关重要的角色，几乎从诞生之日起就成为了技术合作伙伴。我们一直与 Agora 的软件开发团队携手合作，并实现了进入市场所需的所有功能。此外，我们还构建了强大的软件架构，随时准备快速扩展。我们负责开发 Android 应用程序和网络平台，我们应该承认和 Agora 团队一起工作是一种真正的乐趣！

主要焦点始终是产品的质量，因此 [AGORA Images 从一开始就与 Apiumhub](https://apiumhub.com/software-projects-barcelona/agora-product-development-barcelona/) 并肩工作。我们的团队不仅专注于数字产品的开发，还专注于技术战略、选项“购买/出售图像”的实施、搜索(弹性搜索)和竞赛。

这种关系始于 2016 年，当时 AGORA Images 的主要目标之一是开发其数字 MVP(最低可行产品)以获得融资。Apiumhub 之前也做过类似的项目，有着快速持续的交付和宏大的目标。几个星期后，很明显，两家公司处于同一浪潮中，拥有相同的价值观。此外，Apiumhub 发现该产品和团队非常有前途，因此决定与杰拉德·皮克、Mnext Venture Capital (Javier Faus)、WidInvest AG(国际基金)、Soller Invest SL(Rullan 家族)、Kabche Real State (José Kabche)以及 IESE 和 ESADE BAN 投资集团的成员一起成为 Agora Images 的官方投资者。

在完成价值超过 200 万欧元的第二轮投资后，Agora Images 设定了新的目标，重点是对平台进行改进，以进入摄影社交网络的新阶段，以及赞助竞赛的新阶段。作为技术合作伙伴，Apiumhub 将帮助您实现这一目标！

联合创始人兼首席执行官 Octavi Royo 表示:“这笔新资本将使我们能够继续进行国际扩张。在不到两年的时间里，我们已经拥有了 200 万用户，我们相信这一数字将在未来几个月内迅速增长。”

**AGORA Images 人物:**

*   已经出版了 600 万张照片
*   200 万摄影师
*   190 个国家
*   每 4 秒钟，就有一张照片从世界的某个地方发布出来

AGORA Images 将所有这些对摄影的热情投入到项目和 B2B-B2C 摄影比赛中，以便巴塞罗那基金会、Wallapop 或雅马哈等品牌能够与他们的粉丝和客户产生互动。

Agora Images 是由用户生成的内容驱动的，它允许世界上任何人使用智能手机或数码相机来分享、销售和展示他们的图像，此外还允许用户、公司和品牌访问数百万人的照片作品。

通过由 AGORA Images 团队和品牌组织的竞赛，用户发送符合需求的独特的高质量图像。这些比赛增强了用户、摄影师和赞助他们的品牌之间的关系。此外，为图片投票的其他参与者也在比赛中发挥了重要作用，并感觉更接近品牌。根据投票，拥有“最佳”图片的用户将获得奖品，不仅是经济上的，还有体验或旅行。这些照片成为一种独特的营销行为，对于任何想要吸引观众的公司来说都具有巨大的价值。

2018 年似乎是两家公司都在成长的一年，有很多想法和举措。当前的首要任务是发展其技术设备，以便能够实现其新的目标。

让我们看看 2019 年会发生什么！
