# 作为一名软件工程师的职业经历和奋斗

> 原文:[https://dev . to/imad/career-first-experience-and-fight-as-a-a-software-engineer-27ig](https://dev.to/imad/career-first-experience-and-struggle-as-a-software-engineer-27ig)

在一家大型上市公司呆了 18 个月后，我刚刚在一家小创业公司开始了一份新工作(这是我有生以来的第一次经历)。我和一个不错的团队一起工作，但是没有高级开发人员，没有有趣的项目，有时根本没有项目，所以我只是维护一个我和另一个初级开发人员制作的电子支付网站。我在那里真的很沮丧，因为我看不到任何通过与真正的开发团队一起工作来提高自己作为一名开发人员并有一天成为一名高级开发人员的可能性。所以我搬到了这家新的创业公司(薪水更高)，我发现 web 应用已经做好了，一些开发人员在维护后端，一些人在迁移前端以做出反应，我仍然不知道我将做什么，但在我看来，大工作已经完成了，所以我们将只做维护和一些小的添加。

我很沮丧，因为我找不到合适的公司和团队来学习和提高自己，我被困在只能自己开发小应用程序的地方，但我肯定无法以正确的方式开发更大的应用程序。我觉得我成为软件架构师的目标受到了威胁。

我有一个信息系统的工程学位，所以我喜欢从头开始构建系统，从分析、设计开始，然后开发，我不想只是一个 java 专家或 react 大师。

那么你会给我什么建议来充分利用我的经历来实现我的目标呢？是我多虑了吗？
