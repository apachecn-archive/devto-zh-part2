## 1 -你必须首先掌握一些其他的 JavaScript 概念

在掌握`this`之前，你必须了解某些 JavaScript 概念。尤其是当事情变得很奇怪的时候-因为事情确实变得很奇怪。这就是为什么您将首先学习 JavaScript 对象的基础知识。

随着你的进步，你会发现很多很酷的东西，它们会帮助你对`this`有更深的理解。然后你会成为一个更好、更自信的开发者。你将拥有比使用`this`更有用的技能。

## 2 -你不能仅仅通过查看创建它的代码就知道`this`是什么

这是一个大的。当你在代码中遇到`this`时，你只能猜出`this`应该是什么的一部分。为什么？因为`this`在哪里创建与代码运行时它代表什么几乎没有关系。所以，不要自责！
