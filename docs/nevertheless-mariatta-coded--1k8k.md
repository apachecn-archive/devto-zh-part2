# 然而，玛丽雅塔编码

> 原文：<https://dev.to/mariatta/nevertheless-mariatta-coded--1k8k>

## 我开始编码是因为...

我小时候玩过电子游戏，随着年龄的增长，我对学习如何为电子游戏编写程序产生了好奇和兴趣。我是从初中开始学基础编程的。然后就自学了 HTML。然后我攻读了计算机科学学位，学习了 Java 和 C#等几种语言。

## 我继续编码是因为...

感觉编程和编码是我唯一知道的生活。我不断发现新的问题需要解决，我不断学习新的东西。我学得越多，就越意识到自己有多少不知道。但是没关系。永远不要停止学习。

## 我最近克服了...

令人沮丧的一段时间。不知何故，我熬过来了，并开始再次看到事情积极的一面。我知道这不会是最后一次..

## 我想吹嘘一下...

我的名字出现在蜘蛛侠返校节电影的片尾字幕中！...同样在表情符号电影的最后。

在世界各地的各种会议上发言后，我现在可以合法地称自己为国际公众演说家。

朋友 Dan 写了一本 [Python 的书](https://dbader.org/products/python-tricks-book/)，邀请我写前言。我觉得这很酷！

## 我目前正在入侵...

我为了工作在学 GraphQL。它很强大，但我还没有完全理解它。

我还想建立更多的 GitHub 机器人。我从为 CPython 学习和构建机器人中获得了很多乐趣。我已经开始和另一个开源维护者谈论为他的开源项目构建机器人。

我正试着给新的 PyPI 网站[仓库](https://pypi.org)投稿。我昨晚刚做了一个新的拉取请求，但是我破坏了测试。所以我需要回头再谈。

## 我仰望...

我在 Python 和 PyLadies 社区中遇到的朋友和导师。社区非常支持。通过他们的行动，他们向我展示了积极 Python 的真正含义。如果没有他们的持续支持，我想我不会有现在的成就。
