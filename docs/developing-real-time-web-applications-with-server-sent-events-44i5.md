# 使用服务器发送的事件开发实时 Web 应用程序

> 原文：<https://dev.to/auth0/developing-real-time-web-applications-with-server-sent-events-44i5>

服务器发送事件(SSE)是一种标准，使 Web 服务器能够将数据实时推送到客户端。在本文中，我们将通过使用 React 和 Node.js 构建一个航班时刻表演示应用程序来学习如何使用该标准。但是，您在本教程中学习的概念适用于任何编程语言和技术。
