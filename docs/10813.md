# 数学:从恨到爱

程序员通常不喜欢承认自己数学不好。此外，大多数非程序员认为所有程序员都是某种数学天才，这也于事无补。不幸的是，我没有任何数学超能力。一点也不。事实上，我曾经最讨厌数学。一想到初中要去上数学课，就让我想起了平克·弗洛伊德的视频。

到我上高中的时候，我确信我完全不懂数学，我缺乏那种神秘的数学基因，而且只有天生有数字头脑的人才能接近并擅长这门学科。我想，我只是不够聪明。

大约在同一时间，我的高中开始教我们计算机科学。很快，我发现我喜欢它，甚至很擅长它。当其他同学在努力解决算法问题时，我相对容易地解决了这些问题，而我似乎有这方面的诀窍。

高中毕业后，我应征入伍，在那里我设法找到了一份编写计算机数据库应用程序的工作。随着我越来越擅长，我想知道更多。我想真正了解计算机是如何工作到原子的，学习这些我一直听说的更高级的算法，了解加密技术是如何工作的，计算机动画是如何生成的等等。我被各种可能性迷住了。

所以我报名参加了一个远程大学课程，攻读计算机科学学位。在整整 4 节数学入门课的学习之后，我的脑袋因为集合论而旋转得如此厉害，以至于我放弃了，就像我的生活依赖于它一样，并且再也没有回头。我觉得自己是个彻底的失败者，对自己在数学上如此无能感到羞愧。

退伍后，我设法找到了一份电脑程序员的工作。通过勤奋自学大部分书籍，我能够提高我对编程和软件的理解，并为自己建立了一个以平均标准衡量相当成功的职业生涯。

虽然这很好，而且我的工作是支付账单，但我对这个主题的理解是表面的。我缺乏真正掌握这门学科所需的深度理解。我就知道。但是每次我试图深入主题时，我都不可避免地碰壁。数学似乎无处不在。

一天，当我在做自己的事时，我偶然听到了一个名叫萨尔·汗的人做的 TED 演讲。萨尔之前是一名对冲基金分析师，他在 2009 年辞去了工作，在自己创建的一个网站上工作，免费向任何希望学习数学的人教授数学。他谈到了公立学校系统的问题:孩子们批量地学习课程，由于一刀切的方法，他们的知识积累了差距。所以当他们进入更高级的科目时，他们的基础已经不稳固了，以至于他们撞上了一堵过不去的墙。他说他的网站正在解决这个问题。

他的演讲确实引起了我的共鸣。但我仍然持怀疑态度。这在理论上听起来很棒。但是我，我个人，能从中受益吗？我当时还不知道。

我报了名，从一开始就开始按顺序看视频和做练习。我真的在做算术。这些概念解释得非常清楚，我可以一遍又一遍地看。我也可以使用内置的练习来练习这些科目，并真正将概念带回家。

接下来，你知道，我开始处理越来越多的高级课题，这些课题我以前完全不了解。而且做得很成功，甚至很轻松，因为我的数学基础已经坚如磐石。

今天，根据可汗学院的网站，我已经掌握了 700 多种不同的技能。它真正恢复了我的数学教育，我的自尊，甚至点燃了对数学之美的真正欣赏。但也许更重要的是，它打开了许多了解我的职业和我生活中真正激情的大门:计算机编程。

附言

你在这篇文章顶部看到的图像是所谓的分形。分形是在不同尺度上自相似的图案(注意当你放大时，花的图案是如何重复的)。

它们是通过一遍又一遍地重复一个简单的数学过程而产生的，而不是描述和编码每一个细节。分形图案非常常见，因为自然界充满了分形。例如:树木、河流、海岸线、山脉、云彩、贝壳、飓风等。
