活动文本为管理`RichText`带来了一个多态模型

### [](#add-a-field-to-your-form)向表单添加字段

**<% = form . rich _ text _ area:content %>**:rich _ text _ area 是显示矩阵
的新内容类型

```
<%#app/views/posts/_form.html.erb %>
<%= form_with(model: post) do |form| %>
  …
  <div class="field">
    <%= form.label :content %>
    <%= form.rich_text_area :content %>
  </div>
  …
<% end %> 
```
