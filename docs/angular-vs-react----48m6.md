我使用 react 和 Angular 已经有一段时间了。这两个框架都是健壮、可靠和超快的。这两个框架都是由科技巨头建立的(脸书的 react 和谷歌的 Angular)。然而，在学习过程中还是有一些不同的。

我在一周内学会了反应。如果你已经知道如何编写 Javascript，学习 react 并不需要太多的努力。另一方面，我花了 6 周时间学习 Angular。你可以把 Angular 看作一种全新的编程语言。我对 Angular 的用途做了一些研究，许多专家说 Angular 是一个叫做 Google Web Toolkit (GWT)的框架的替代品。根据许多文章，有角比 GWT 容易。此外，谷歌将其部分产品从 GWT 迁移到 Angular，如 Google Adwards。然而，一些谷歌产品仍然是在 GWT 写的(比如 Blogger.com)。

简而言之，如果你有足够的时间学习新的语言，那就去学 Angular 吧。另一方面，如果你没有时间并且你熟悉 Javascript，那么就去找 React 吧。
