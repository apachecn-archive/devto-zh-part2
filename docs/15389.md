# 使用双视 OLL 和锁相环解决 2x2 扭曲难题

> 原文:[https://dev . to/Cr 0 wst/solving-2 x2-twisty-puzzles-using-two-look-oll-and-PLL-d78](https://dev.to/cr0wst/solving-2x2-twisty-puzzles-using-two-look-oll-and-pll-d78)

这是我在玩 2×2 的魔方时注意到的。我不是一个速度立方，我也不太擅长记忆算法，但是，解决扭曲的难题已经是我几年来的一个爱好。

## [](#threebythree-two-look-oll-and-pll)三乘三两看 OLL 和 PLL

我主要使用 [CFOP 或 Fridrich](https://ruwix.com/the-rubiks-cube/advanced-cfop-fridrich/) 方法来解决 3×3 的扭曲谜题。然而，因为我不能记住很多算法，我补充了最后一层与[双看 OLL 和 PLL](http://www.cubewhiz.com/2lookoll.php) 。典型的过程如下:

1.  解决十字架。这是你标准的初学者入门方法。
2.  解决角落和中间边缘件。这是初学者方法的第二步和第三步的结合。这是通过找到 F2L 对并将它们插入适当的空间来完成的。
3.  解决顶部中心件(至少形成一个十字)。
4.  调整最后一层的方向，使十字对面的颜色(传统上是黄色)朝上。
5.  放置最后一层，使碎片处于正确的位置。

## [](#twobytwo-twisty-puzzle)两两缠绕的拼图

二乘二扭曲的难题以类似于三乘三的方式解决。它本质上是一个三乘三的扭曲拼图，没有任何边缘部分或中心。这与处理三乘三的角是一样的。典型的过程如下:

1.  解决底层。这是通过定位和定向这些片来实现的，这样单一的颜色在底部，匹配的颜色在第一层的侧面。
2.  调整顶层的方向，使十字对面的颜色(传统上为黄色)朝上。
3.  放置最后一层，使碎片处于正确的位置。

处理角点的巧妙之处在于，2 乘 2 和 3 乘 3 的角点之间存在直接的一对一关系，尤其是在求解最后一层时。

## [](#last-layer-relationship)最后一层关系

当解决三乘三的最后一层时，重要的是首先解决中心部分。这形成了七种不同的条件。这些相同的条件可以在 2 乘 2 中找到，但是，你可以跳过解决顶部中心部分，因为它们不存在。

### [](#twolook-pll)双视 PLL

当处理 3 乘 3 时，你可以做一个两眼 PLL 来确定角和边的方向。然而，对于 2 乘 2，没有边，所以这成为可以使用的单一算法。我首先找到一条已经匹配的边。我旋转顶层，直到它的边缘与正面相对，然后执行算法:`l'UR'D2RU'R'D2l2`。

如果没有匹配的边，可以先执行算法，它会对齐其中一条边。然后你只需要确定正确的边的方向，让它背对着你，然后重复这个算法。

算法完全一样。但是，当您看到`f`、`r`或`l`指示要移动两个前层或两个右层时，您只需移动单个前层、右层或左层。

## [](#conclusion)结论

我认为解决扭曲谜题的很大一部分来自于能够与 3×3 的扭曲谜题进行比较。无论你是在用镜子立方体，不同形状的模型，图片立方体，等等..进行这种比较是有益和令人兴奋的。

玩得开心！
