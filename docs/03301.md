### 2018 年 web 开发初学者最好的 7 本书

你是喜欢从书上学习编程，还是只是一般的爱书？以下是我认为对 web 开发初学者最有帮助的几本书。即使你正在学习在线课程，它们也很有用。

[![](../Images/faf2bc235e56a1e364f3ce54c12216f0.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--vV_DbDkh--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2ADen2wP9_wGqixNJsEGfPaA.png)

### 1.HTML&CSS 和 JavaScript & JQuery (2 本书集)作者乔恩·达克特

[![](../Images/9dac0a2a0fd727333b14a981273661df.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--C7qlByIG--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/250/0%2AU2Utr5K-kw0GZixH.jpg)

乔恩·达克特的书非常受欢迎，这是有原因的——它们很好地解释了手头的主题，设计得非常好。他们绝对不是你通常的沉闷的编程教科书！这两本书都用图形来说明概念，老实说，它们放在你的书架上会很酷🙂

在亚马逊上看看乔恩·达克特的书。

### [](#2you-dont-know-js-by-kyle-simpson)[2。你不知道凯尔·辛普森的 JS](https://amzn.to/2RxfNGu)

[![](../Images/3fa5cf00c36b914e8b79bdc541d1bb28.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--cpM940u_--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/250/0%2AdbgaoBZ_Ishr0wXW.png)

现在有很多 JavaScript 书籍，而且很多都很大。你不知道 JS 不是一本很大的教科书——它是六本小书的系列，都致力于 JavaScript 中的一个特定主题。没有废话，只有必要的核心信息。

[在 GitHub 上查看你不知道的 JS。](https://github.com/getify/You-Dont-Know-JS)

或者在亚马逊上买纸质书。

### [](#3dont-make-me-think-by-steve-krug)3。史蒂夫·克鲁格的《不要让我思考》

[![](../Images/1edce233681716e7e035ea0548c42ee8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--n6OmZvOu--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/250/0%2A6y0MXiXXsLhDj9R4.jpg)

听说过 UX(用户体验)或用户设计这样的术语，但不知道它们是什么意思？这本书很好地介绍了制作人们喜欢使用的网站的世界。它解释了访问者如何访问你的网站的基本行为，以及网站的哪些方面会让他们感到沮丧。

底线:不要设计你的访问者会讨厌的网站。

在亚马逊上查看不要让我思考。

### [](#4frontend-developer-handbook-by-cory-lindley)4。Cory Lindley 的前端开发人员手册

[![](../Images/4d734094fb95438285ae5a52ab904e49.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--jrXhNArJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/250/0%2AaYav7ROVjnnpeEm0.jpg)

这种免费的在线书籍每年都会发布一个新版本，并作为“前端开发的状态”这是一本非常全面的指南，涵盖了前端 web 开发、学习资源和工具的概述。

如果你想快速了解这个快速发展的行业的最新动态，你会发现这本书是必不可少的。

在 GitBook 上查看前端开发人员手册。

### [](#5the-clean-coder-by-robert-martin)[5。罗伯特·马丁的《干净的编码者》](https://amzn.to/2RDJ56e)

[![](../Images/2914eeed052599796d8a039c1a1d43df.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--gAUbk5df--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/250/0%2A9fBkpZzI0Xru53TV.jpg)

这本书有很多关于编码和事业成功的建议。这是从软件开发的角度写的，但即使作为一名 web 开发人员，我也从 Robert Martin 的见解中获得了很多价值。

马丁有很多有价值的建议——从如何作为一名专业人士，到学会如何说“不”，再到拥有良好的职业道德。无论你现在是否是一名网站开发人员，你都会学到一些如何在工作中脱颖而出的真知灼见。

在亚马逊上查看干净的编码器。

### [](#6deep-work-by-cal-newport)[6。加州纽波特](https://amzn.to/2Rvm2e5)的深度工作

[![](../Images/670ad4f4c549d768c13701de123c97ce.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--A1rhN6kf--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/250/0%2AAGgX-NyZoiMP4gVl.jpg)

这本书真正改变了我对待工作的方式。当大多数人过着极其忙碌的生活，并不断尝试做得更多，取得更大成就时，新港却采取了完全相反的方式“少即是多”

正如这本书的主题所暗示的，他谈了很多关于他个人如何在这些年里发展了一种超专注深度工作的实践(他的成就相当令人印象深刻)。纽波特还提倡从你的生活中剔除那些不会引导你实现最重要目标的因素。

这本书适用于网络开发人员和任何人，他们的工作将受益于长时间的完全精神集中。我非常非常推荐给任何领域的人阅读。

查看亚马逊上的深度作品。

### [](#7the-power-of-habit-by-charles-duhigg)[7。查尔斯·杜希格的《习惯的力量》](https://amzn.to/2C1EPs9)

[![](../Images/fdd01f65ad94b6cadae1ec754d6100ab.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--iqaplHhq--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/250/0%2AOGESOXGaxwe_BvjK.jpg)

习惯的力量是另一本书，可以用来更有效地工作，诚实地改善你的生活习惯。杜希格雄辩地描述了习惯(好的和坏的)是如何形成和打破的，而我们往往没有意识到。

他用真实世界的科学例子以一种非常有说服力的方式来阐述这些概念。如果你对改掉坏习惯、培养好习惯、增强意志力感兴趣(他认为意志力可以像其他技能一样培养！)那么你可能会从这本书里受益。

在亚马逊上看看习惯的力量。
