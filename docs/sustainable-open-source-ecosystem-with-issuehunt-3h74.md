作为维护者，我们一直在开发一个名为 [Boostnote](https://github.com/BoostIO/Boostnote) 的开源项目，我们在两年内已经有了超过 8400 个 Github stars。

# 什么是 IssueHunt？

[IssueHunt](https://issuehunt.io/) 是 OSS 的资金平台。

开源项目的任何问题都可以由任何人资助，资金将分配给维护者和贡献者。

我与 Boostnote 团队反复讨论开源生态系统和发布 Hunt 概念。事实上，我们也和 Ruby 的父亲 Matz 讨论过。

为了证明这个概念，起初我们把它作为一个奖励计划提供给 Boostnote 社区。

结果，通过这个项目，许多资金问题在几天内就解决了，我们的社区得到了进一步的推动。所有的贡献都非常惊人。

所以我们决定推出 IssueHunt！

# 使命

我们的使命是维护一个环境，在这个环境中，维护者和贡献者可以自由地发展他们的优势，发挥他们的潜力，并受到开源社区的激励。

我们正致力于实现这一目标！

正如 Boostnote 已经在 IssueHunt 上得到资助一样，您的开源项目也可以从现在开始得到资助。
