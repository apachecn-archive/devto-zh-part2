# 我的第一个 Chrome 扩展:Stackoverflow Starred Viewer

> 原文：<https://dev.to/alahmadiq8/my-first-chrome-extension-stackoverflow-starred-viewer-1nm6>

> **更新 2021-05-21:** 我计划回去改进这个项目。由于其他责任，我过去忽略了它。

你是不是和我一样[平庸的开发者](https://dev.to/sobolevn/i-am-a-mediocre-developer--30hn)？你坚信 [StackOverflow 驱动开发](https://meta.stackoverflow.com/questions/361904/what-is-stack-overflow-driven-development)吗？你忘了把你经常复制粘贴的答案加入书签了吗？

这是促使我做这个副业主要原因。

**TL；浏览 StackOverflow 问题的简单方法。当你访问 stackoverflow.com 时，它只是在你的浏览器上注入一个侧边栏。该侧边栏显示了您所有标有星号的问题的可过滤列表。**

*   [Chrome 扩展链接](https://chrome.google.com/webstore/detail/stackoverflow-starred-vie/cnamfgnggohmnbilcjdnjjahlgkndhmp?hl=en)
*   [项目 Github 链接](https://github.com/AlahmadiQ8/StackFave)

我[明星](https://stackoverflow.com/search?q=infavorites%3A5431968)T2T4【stack overflow】我经常提到的问题。然而，浏览它们并不容易。我试着在网上搜索，但是我找到的下一个最佳答案是[这个](https://meta.stackexchange.com/questions/3912/better-favourites-organisation)。

这促使我创建自己的扩展，以更简单的方式浏览我标有星号的问题。我最喜欢的扩展八叉树给了我巨大的灵感。
