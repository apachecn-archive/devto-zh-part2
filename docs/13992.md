# 学习中的禅

> 原文:[https://dev.to/henautumn/zen-from-learning-4dn3](https://dev.to/henautumn/zen-from-learning-4dn3)

我一直想学习，但学习并不总是轻松的。

我上过学，但老实说，我一直讨厌它。结果太慢了，老师太慢了，目标太远了，我半途而废了。记得有一次，三年级开学第一天，我们在讨论外套挂哪里，午饭去哪里吃之类的事情。我举起手，尽可能礼貌地问道。

"我们今天会学到任何东西吗？"

好吧，这可能有点粗鲁，但我真的控制不住自己！我想学习，我觉得必须尽可能多地吸收。我知道老师跟上了每个人的步伐，我感到有点内疚，因为我无情的学习速度让其他孩子受苦。但我总是觉得停滞不前，窒息，缓慢。

我在大学里学的是建筑学，但从未拿到学位。我离开了学校，真的试着去寻找一些能满足我学习和应用需求的东西。我在呼叫中心工作(讨厌这份工作)。我是一个全职妈妈(很喜欢)。我没有真正的道路，没有可言的“事业”。缺乏目标和新的东西让我和我的家人疯狂。

我注意到我把动力用在了不需要我全部精力的事情上。你曾经试图让一个蹒跚学步的孩子说出字母表吗？

我试了 3 个小时(老实说，更像 20 分钟，但仍然！).我们俩都筋疲力尽了。

那时我参加了微软免费的 Java 在线课程。我喜欢它。我的结果被记录在终端里。如果我写错了什么，几乎总会有解决办法。我在创造惊人的东西，我可以看到我的进步。我感觉棒极了。

但我也感受到了别的东西。“需要知道”这种挥之不去的感觉正在慢慢消失。我意识到有太多关于编程的东西我不知道，而且我可能永远也不会知道。没有必要急着去吸收，它们都还在那里。我可以随时访问它，寻找它。编程帮助我安抚了我那一直很匆忙的一部分。我发现它令人沉思。

我目前正在训练营学习 JavaScript。我再开心不过了。我有一个目标，我有一个目的，我有一个事业要去实现。
