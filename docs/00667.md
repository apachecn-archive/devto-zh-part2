# 数据科学资源列表#2

> 原文:[https://dev . to/nicologiso/a-short-list-of-data-science-resources-2-457n](https://dev.to/nicologiso/a-short-list-of-data-science-resources-2-457n)

上周，我开始为 dev.to 写一篇关于一些有用资源的[帖子](https://dev.to/nicologiso/a-short-list-of-data-science-resources-12de)，这些资源是我作为一名数据科学家在日常工作中偶然发现的。我决定重复这个“实验”，收集一些我在过去七天里找到的其他工具和文章，希望其他人也会感兴趣:

*   Data Viz 项目提供了你能想到的所有可能的图表，有详细的描述和一堆例子[https://datavizproject.com](https://datavizproject.com)

*   在 Python tuple 中，解包使得代码更具可读性，例如避免硬编码索引。不管名字是什么，它适用于任何 iterable。来个好听的介绍:[https://trey hunner . com/2018/03/tuple-unpacking-improves-python-code-readability/](https://treyhunner.com/2018/03/tuple-unpacking-improves-python-code-readability/)

*   有时候，在 Python 中的命令行程序上有一个基本的 GUI，而不需要花费几个小时，这是很酷的。看看这个黏糊糊的包裹[https://github.com/chriskiehl/Gooey](https://github.com/chriskiehl/Gooey)

*   DeepL Translator 使用构建在 Linguee 数据库上的卷积神经网络来提供高质量的翻译[https://www.deepl.com/en/translator](https://www.deepl.com/en/translator)

如果你有什么反馈或者很酷的资源，请在评论里分享！
