# [](#introduction)简介

当我第一次开始作为 Django 开发人员的旅程时。我是一个完全失去联系或者已经放弃(幸运的是我没有)在大学毕业后找到一份软件工程师工作的人，认为我可能不适合这份工作。

# [](#talentless-as-a-developer)没有开发人才

幸运的是，在期末考试前，我在一家教育科技初创公司找到了一份业务拓展的工作。在这家初创公司呆了大约 5 或 6 个月，经历了一些人生的曲折之后。我辞去了工作，开始涉足创业，但没有取得多大成功

失败和我日渐空虚的存钱罐。在导师的帮助下，我去找了份工作，肯尼帮助我找到了自己的职业，并摆脱了我所处的困境。我设法找到了一些工作，但这并没有持续多久。

在经历和申请了无数来自亚洲科技公司的创业工作后，Linkedin 或 e27 有很大比例的工作是某种形式的网络开发。

# [](#eureka-moment)灵机一动

然后我突然灵光一现，就像一把大锤砸破了一堵脆弱的墙。突然想起来在 **PyCon 2016 SG** 期间其实也有人在做 web 开发，名字很时髦，叫**“Django”**。

作为一个 python 爱好者，我一直想找一份使用 Python 的工作。所以我开始狼吞虎咽我能为 Django 得到的任何东西，比如看 youtube 频道的教程，比如克里斯·霍克斯、T2 为企业家编程。我甚至从我的朋友 Joie 那里借了一本叫做**【两勺姜戈】**的书。

我非常渴望找到一份工作，为了提升自己的技能，我最终报名参加了一个名为**“React”**的 Udacity 纳米学位项目，该项目教授 **React ** 、**Redux** 和 **React Native**

# [](#focus-in-a-niche)专注于一个小生

有时候，通过知道你能做些什么来选择你的战斗而变得超级专注可能会给你带来意想不到的结果。由于我一直在寻找使用 Django 的工作。

我得以在一家名为**“Microsec”**的本地物联网安全初创公司实习，直到今天，我一直作为**“全栈开发者”**与他们一起工作
