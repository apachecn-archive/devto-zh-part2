# CodeOne 主题演讲集锦 Java 的未来就在眼前

> 原文:[https://dev.to/ethanmillar/keynote-highlights-of-codeone-未来-爪哇-现在-513o](https://dev.to/ethanmillar/keynote-highlights-of-codeone---the-future-of-java-is-at-present-513o)

甲骨文 Code One 2018 的开幕主题演讲让观众有机会聆听甲骨文 Java 平台集团和业界的知名人士。Oracle 还宣布了该语言和平台的最新进展，并展望了未来的发展。

**关于 Oracle Code One**
Oracle Code One 是业界领先的开发者大会，汇集了国际和本地的专家和演讲者，探讨最新的开发者技术、语言等。

负责 Java 平台开发的副总裁乔治·萨博(Georges Saab)重申了甲骨文对 Java 平台的承诺，从而拉开了活动的序幕。他谈到了 Java 的性能、可靠性、安全性和生态系统，使其成为现代应用程序开发的理想选择。GitHub 现场服务副总裁 Matthew McCullough 强调了 Java 平台的开源参考实现 OpenJDK 的重要性，并讨论了 Skara 项目。这个项目的主要目标是调查 JDK 源代码的替代源代码管理和代码审查选项。他还讨论了最“重要的软件”是通过全球协作开发的，并鼓励观众参与开源项目。

后来，当萨博重返舞台时，他谈到了保留 Java 的品质。Java 一直在发展，是免费的，社区致力于完善当前的平台，并投资于开发者和开发者的兼容性。

这一事件的核心信息是“Java 仍然是免费的”, Oracle JDK 与 OpenJDK 版本非常相似。Oracle Java 平台组的首席架构师 Mark Reinhold 阐述了关于 Java 新发布模型的五大误解。

**这里是甲骨文未来项目的主要亮点:**

1.  Amber 项目:旨在探索面向生产力的 Java 语言特性，比如带表达式的原始字符串、模式匹配和记录
2.  **Project Panama:** 该项目旨在改善和丰富 Java 程序与定义良好但外来的代码和数据之间的联系。
3.  项目 Valhalla: 该项目引入了值类型和通用专门化，以实现更高性能、缓存友好的数据结构。
4.  Project Loom: 旨在引入轻量级用户模式线程，称为纤程，以实现更简单但更高效的并发代码。

Reinhold 展示了一系列使用最新 Java 12 构建的现场编码演示，在相关项目网站上可以找到这样的例子。

活动最后， [Java 开发团队](https://www.aegissofttech.com/java-application-development-services.html)公布了包括自动驾驶汽车 JPoint、社区获奖者 BgJUG 和 Sharat Chander 在内的 2018 年杜克选择奖获奖者。宣布的获奖 Java 工具包括 ClassGraph、Twitter4J、Apache NetBeans 和 Jelastic vertical memory scaling，以及 Project Helidon 等开源项目。

## **关于 Java 平台最近有哪些更新？**

9 月 25 日，Java SE11 发布了重大升级和改进，如下所述:

**飞行记录器:**一个低开销的数据收集框架，用于排除 Java 应用程序和 Hotspot JVM 的故障。

**HTTP 客户端:**一个更新的新 HTTP 客户端 API，在 JDK 9 中首次引入，作为实现 HTTP/2 和 Web 套接字的孵化特性

**启动单文件源代码程序:**对 Java Launcher 的改进，可以运行作为 Java 源代码的单个文件提供的程序，使处于学习 Java 初级阶段的开发人员更加容易。

**ZGC:** 一个低延迟垃圾收集器，可以轻松处理万亿字节大小的堆，无需暂停超过 10 毫秒。

这些是最新 Java 活动的主要亮点。社区对 Java 主题演讲的反应总体上是积极的。一些评论是趋势性的，如 Paul Bakker 在#CodeOne 上发表了精彩的主题演讲，这是有充分理由的，Java 生态系统看起来更好了。
