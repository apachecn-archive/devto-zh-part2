Byteconf React 是一个在 Twitch 上举办的为期一天的免费 React 会议，任何人都可以参加。它发生在 2018 年 8 月 31 日。

我们的演讲者包括肯特·c·多兹、李美玲、凯尔·谢夫林等等！请务必在我们的网站查看完整的日程安排。

还有一件事我认为可能会吸引 dev.to 人群:我们正在围绕 Byteconf 建立一个开发者社区——Byteconf React 只是计划中的一系列免费会议的第一个。如果你挖项目，想支持我们，可以考虑加入不和。这是一个实时聊天，我们在这里讨论 JavaScript、React 等等。

加入我们的纷争吧！
