但现在我还是走了一步，将完全搬到贝尔，正是因为我想用这么少的负担(各种应用程序)来学习计算机科学。

当我指 iA Writer 的`.md`bear 文件时，我复制了很多给我看，我的表自然完全乱了套。但是，因为知道那些格式很好，所以我只是想把它们打包成‘t1’。

> 代码块是用三个[【维基百科链接】](https://de.wikipedia.org/wiki/Gravis_(Typografie))生成的，如上图所示。

我认为它们很有用，而且读起来很好，即使如果你真的想建立长表，这种可能性很可能会达到极限。

但是，希望 Bear 的开发者现在会利用标准的表格 Markdown 实现方法，我只要一出现这些表格，我就会把代码块从表格中删除，然后希望它们有漂亮的表格。
