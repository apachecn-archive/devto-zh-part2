# 你愿意做一个开发演讲/研讨会吗？

> 原文：<https://dev.to/jess/would-you-like-to-give-a-dev-talkworkshop--31c6>

自从我们本月早些时候宣布开发研讨会以来，我们已经上线五次了。弄清楚流媒体技术并与我们优秀的演讲者合作是一件非常有趣的事情。每个活动都像一个小型会议，我期待着有一天组织一些面对面的开发活动。

下面是 Anna 对使用 React & Jest 进行测试的介绍。如果你放大，你会看到她在用一个狗狗兼容性应用程序做演示😊 🐶

反正！让我来说一下这篇文章的要点。你们中的许多人都想成为一名演说家，所以下面是一些要点:

*   谈话时间从 30 - 90 分钟不等，由你决定。
*   我们在*非常灵活，你什么时候*可以出席。因为人们正在从全球各地收看，所以对于*某人*来说，总是合适的时间。
*   没有一个主题是太一般的，小众的，初学者的，或者高级的。
