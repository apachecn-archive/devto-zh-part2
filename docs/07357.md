# 教学的关键

> 原文:[https://dev.to/zellwk/the-key-to-teaching-58ce](https://dev.to/zellwk/the-key-to-teaching-58ce)

一个好老师有许多技能。

他们知道如何很好地沟通，他们知道如何解释事情，他们知道如何判断学生是否困惑。他们也知道如何引出学生的问题。

如果你想教好，有一个永无止境的清单列出你需要擅长的事情。

但只有一件事把他们联系在一起。如果你做对了这一件事，一切都会水到渠成。

## 你的姿势

一个好老师会关心。

他们把学生放在第一位。

他们尽一切可能帮助学生学习。

这包括:

1.  耐心倾听他们的问题。
2.  一遍又一遍地解释事情，每次都用不同的方式，直到他们明白为止。
3.  提供鼓励和希望。
4.  提出引导性问题，让学生找到自己问题的答案。
5.  做练习
6.  对作业要求严格。
7.  将课程分段，以便学生能够消化信息。

他们设身处地为学生着想，以便能够做到以上几点。

## 教师的责任

老师有责任帮助学生学习。

学生不学，老师有错。

问题是，你是否足够在意下一次做得更好？现在教不好也没关系。连自己都教不好也没关系。

如果你在乎，你会找到改善的方法。

## 好老师不多

很少有人想教书。他们教书不是因为他们想帮助学生。他们教书是因为他们想帮助自己。

有时他们教书是因为教书是他们的“工作”。但是教书不是职业。这是一种态度。

问题不是你能不能教。

你可以。

任何人都可以。

真正的问题是:你是否足够关心教学？

如果你走出去开始教书，你会有很大的不同。你现在可能不知道，但 5 年后你会看到回报。
