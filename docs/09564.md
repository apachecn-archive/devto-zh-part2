我们都经历过我们不能和一个非技术人员相处所以我们尽量避免和他们相处。但是营销、销售或客户服务等业务领域的人也避免了我们什么？

我问了与开发商合作的人两个问题。这些问题与人格有关:

*   ？这是你最感兴趣的开发商个性吗？
*   你会改变吗？

不幸的是，这两个问题在四点上是一致的:我们大多是内向的、合乎逻辑的、方的和高度自负的。

> **我们是内向的和平方的**

很常见的情况是，公司的开发者不参与团队的其他成员，被排除在外，或者没有任何共同生活的愿望。我想它可能是我们简介的一部分，但是□？

我们很少看到一个提案中的企图或可能的东西，我们认为只有 0 或 1。依我看，这就是我们在大学里接受教育的原因，因为我们不得不假设所有的情景，不再相信任何事情都是随机的，而只限于不去想‘t0’走出盒子。

> 我们认为自己优越

这是令我最困惑的回答，我们有很高的自尊心吗？

正如我在上一篇文章中提到的那样，在职业生涯中，我们总是(或几乎总是)被告知要让用户的生活变得轻松一些，实际上我有老师说用户是个白痴(这是非常糟糕的)。从我的角度来看，在我转为商业角色之前，我认为其他人(用户)在从来没有这样做过的时候，并没有我同样的智力能力，我认为几个开发商也有同样的信念。

他们可能没有同样水平的数学智力，因为他在职业生涯中没有发展出来，但他们有更多种类的智力，在那里他们很容易超过我们。
