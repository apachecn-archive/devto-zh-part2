# 全新的 VSCode 设置编辑器 UI

> 原文：<https://dev.to/vscode/all-new-vscode-settings-editor-ui-----3j48>

伙计们，你们好！🙌

在过去的几个月里，VSCode 团队一直在开发用于编辑设置的 GUI。在`v1.27`中，新的 UI 设置编辑器已经成为默认编辑器。

[![new VSCode Editor UI](../Images/50baa6fbf2e829b9dcdec603cf5cf4dd.png)T2】](https://marketplace.visualstudio.com/items?itemName=ahmadawais.shades-of-purple)

### 设置修改/验证

已经修改的设置现在用彩色线表示，就像编辑器中修改的线一样。齿轮图标会打开一个上下文菜单，其中包含将设置重置为默认值的选项。现在，设置值已经过验证，并且会显示任何验证错误。

[![new VSCode Editor UI](../Images/45d41741e31fccf78e0fa4fc601a8f55.png)T2】](https://marketplace.visualstudio.com/items?itemName=ahmadawais.shades-of-purple)

### 显示匹配的扩展名

当你在“设置”中搜索时，我们的必应搜索服务仍会提供它认为相关的结果。现在，该服务还将向您显示市场中具有相关设置的扩展。您将在设置搜索结果的底部看到一个按钮，显示匹配的扩展名。单击时，Extensions 视图将打开，并显示匹配的扩展。

[![new VSCode Editor UI](../Images/a1746ded411081bccc5b82f5042e4357.png)T2】](https://marketplace.visualstudio.com/items?itemName=ahmadawais.shades-of-purple)

> 您仍然可以访问 JSON 设置编辑器，或者使用命令`Open Settings (JSON)`或者使用`"workbench.settings.editor"`设置更改您的默认设置编辑器。但是我建议你创建一个键盘快捷键，按下`COMMAND (⌘) + k then COMMAND (⌘) + S`并搜索`Settings JSON`，然后将`COMMAND (⌘) + PERIOD (.)`设置为旧 JSON 编辑器的快捷键。
