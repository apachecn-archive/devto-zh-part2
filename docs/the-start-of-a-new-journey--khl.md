# 一个开始

> 原文：<https://dev.to/kyam/the-start-of-a-new-journey--khl>

在过去的 10 年里，我花了很长时间后悔我以最老套的方式浪费了我的大学教育机会——我加入了一个乐队，放弃了学业，花了两年的学费，除了在英国昏暗的酒吧外，什么地方也没去。从那以后，我就听天由命，投身于不需要技能的行业——服务行业、零售管理——我喜欢但从未完全开心过的工作。

最近，我一直在一家全国性超市管理一个团队，这个职位让我能够发现自己的优势并加以发挥。正是在这段时间里，我意识到我喜欢利用信息技术和科技让生活变得更轻松。虽然这主要包含在电子表格中，但我能够帮助简化和自动化本地流程，否则这些流程将需要手动完成。

尽管我显然很喜欢这份工作，但我排除了从事科技行业的可能性，因为我没有学位，就是这样——对吗？

我以前在酒吧工作过，我以前的一个同事最近找到了一份开发人员的工作。但是他没有学位！他是怎么做到的？我突然想到——如果他能做到，为什么我不能？既然我们都曾是酒吧里的猴子，我当然可以追随他的脚步？

事实证明，我可以。他参加了当地科技公司 [Mayden](https://mayden.co.uk/) 举办的[课程](https://maydenacademy.co.uk/)——他们着手培训当地的编码爱好者，帮助解决当地的劳动力供应问题。此外，事实证明，我就是那种能够通过他们的需求测试，并被他们的全栈 web 开发课程录取的人。

我在超市工作的最后几周非常担心会是什么样:必须超级紧张，一天 8 小时都盯着屏幕。我没有机会继续下去，我将要犯一个大错误——让我的家庭陷入经济困境，浪费了过去两年在工作中获得体面职位的机会。因此，我花了很多在家的时间，尽可能多地学习我们将要使用的语言。

两周前我去了那里，感觉很放松。节奏很现实，尽管我发现提前做所有额外的工作非常有帮助，但我可能(差不多)没有它也能活下来。

现在，两周过去了，我可以开始看到我作为一名开发人员的职业前景了。当然，这可能是不可能的——只有在接下来的 14 周内努力工作并集中注意力，我才能改变这种状况，但至少我发现自己走在了正确的道路上。
