# 团队工作，但不是团队思考

> 原文：<https://dev.to/brightdevs/group-work-but-not-group-think-39k6>

你听说过群体思维吗？当一群人由于相互的压力而做出错误的决定时，就会发生这种情况。Irving Janis 在 1972 年首次提出了这个术语，他认为群体压力会导致心理效率、道德判断以及现实验证的退化。受群体思维现象影响的群体在寻找解决方案时经常忽略其他选择。易受群体思维影响的人大多有相似的背景，在缺乏既定决策过程的组织中工作，不接受任何外界意见。

当团队成员面临做出高质量决策的压力时，集体思维经常出现。人们会觉得没有动力去寻找替代方案或公开表达自己的意见，而不去认同群体的共识。寻求一致意见的巨大压力常常导致非理性主义和粗心大意，结果可能导致不成功的结果。

集体思维对每个公司的成功都是一个巨大的风险。这不仅可能导致每个人都认为已经做出了正确的决定，因为整个小组似乎都支持它，而且更糟糕的是，这还可能导致一致意见往往只是一种错觉。很多时候，一些决定的做出仅仅是因为没有给出真实的意见。

那么，我们能做些什么来避免团队中的群体思维效应呢？

1.首先，我们不应该让压力代替我们做决定。我们经常以类似的方式应对压力。如果我们觉得一个选择比另一个风险更大，我们倾向于选择前者，只是为了避免一些精神和情绪上的紧张，即使我们可能会错过这样的好机会。建立清晰的决策过程可能有助于提高我们的绩效，避免不必要的紧张。

2.我们倾向于关注我们所知道的。我们经常害怕未被探索的事物，而不是问问题，我们急于回答。我们应该把注意力集中在关键信息上，问问自己我们实际上在试图解决什么样的问题。提出适当透明的问题不仅有助于找到最佳解决方案，而且对个人和组织的发展也是必要的。无论职位高低，每个人都有责任进行真实的对话并获得不同的观点。为了避免群体思维，组织中的每个成员都应该被允许在与公司内外的人进行建设性对话时表达他们的疑虑和恐惧。提问阶段不应该被忽视。

3.仅仅因为我们多年来一直以某种方式做一些事情，并不意味着我们现在或将来应该以这种方式做。遵循一些最佳实践是好的，但是我们必须记住不要盲目地遵循它们，尽管事实上似乎每个人都同意它们是好的。盲目的承诺应该通过对市场的变化和预期持开放态度来避免。如今重要的是新的视角和开放的思想。

4.你听说过魔鬼代言人吗？为了避免集体思维的危险，我们应该在会议中至少有一个魔鬼代言人。这不是对每个人和每件事提出质疑，而是通过提出适当的问题，对成员的观点进行批判性评估和挑战。那么领导者的角色是什么？领导要听。他们不应该一开始就陈述偏好或期望。他们应该确保有足够的时间来分析和讨论替代方案。

我们需要观点的多样性，而不仅仅是文化的多样性，我们应该为我们的组织感到骄傲。拥有总是有相同观点并且不断达成一致的人可能是一个错误的警告信号。对我来说，创建一个公司结构，让我们有一些反馈和表达个人意见的空间，是创建一个成功的工作场所的必备条件。我们永远不应该阻止团队中的辩论和创新思维。我们应该缓和它们，但绝不关闭它们。有了共识，并不意味着也有了成功。
