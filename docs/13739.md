今天早些时候，我发现了一种替代屏幕录制的方法，asciinema。

通过抓取 python3-pip 包并使用它来获取应用程序，在 Debian 下安装非常容易。

```
$ sudo apt-get install python3-pip
...
$ pip3 install asciinema 
```
