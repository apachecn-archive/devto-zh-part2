### 自我介绍！你是谁？你在哪里工作？

你好世界！Cyris 是南非人，2008 年移民到新西兰。六年前，我在 Skinny Marketing(与 Skinny Mobile 无关)找到了我的第一份开发工作。我们主要关注数字营销，主要是电子邮件营销。

我最初是一名电子邮件开发人员，使用基本的 HTML 和 CSS 在很短的期限内构建模板和定制活动。只要有空闲时间，我就会练习编码，从免费的 Youtube 播放列表中学习，通过反复试验来构建自己的应用和程序。我没有大学学位或任何官方资格。我是自学成才的，建立在热情之上。

随着时间的推移，随着我技能的进步，我们公司也在进步。现在，我用 NodeJS 构建定制项目、微型网站和应用，以及基于 Wordpress 的完整网站。

### 是谁或什么让你开始编程的？

从小到大，我一直对电脑感兴趣。我想知道它们是如何工作的，是什么让它们“滴答”作响，以及我如何成为其中的一员。

我知道他们是未来，我有热情去追逐他们。从我的第一台 386 台式电脑开始，我就被捕获了。浏览 Encarta 95 百科全书让我大吃一惊，这么多信息唾手可得。我觉得势不可挡。

但在南非，当万维网最终出现时，它非常有限，所以我从来不知道它有多大。我一次花了几个小时用手机上的发电机来建立老学校的 WAP 网站。我读过(并完成)的第一本书是《傻瓜 HTML》。我可以使用记事本，写一些代码，让它在浏览器中显示惊人的东西。震惊了。我觉得自己就像一个在糖果店的孩子，这时候我知道我想从事开发人员的职业。

### 我是[网飞超级浏览](https://www.producthunt.com/posts/netflix-super-browse)的粉丝，祝贺你的产品搜索成功。你喜欢制作 Chrome 扩展的什么地方？

谢谢大家！我喜欢开发能帮助人们的应用或扩展。让事情对用户来说更简单一点，这些功能经常被大公司忽略，但却是消费者所需要的。Chrome extensions 是一个很好的修改服务的方法。作为一名前端开发人员，我爱上了 Javascript，并认为我很自然地会涉足 Chrome 扩展，看看我能否挑战自己，并为消费者带来这些功能。

如果我在浏览网页时看到用户要求某些东西，我的第一反应是看看我是否能做出来。我不在乎付费，我从看到快乐的用户和获得良好的反馈中获得足够的满足感。这是我在这个人人互相挑战的世界里的小小自我提升。我学到了新技能，他们也获得了更好的体验。

### 当你对某人做哑剧编程时，你是用霸王龙的手臂还是摆动的手指？

每当有人递给我一支笔时，我发现自己经常扭动手指，我不得不道歉并说我是一名开发人员，我使用键盘。如果他们不听，我会开始到处挥舞我的霸王龙手臂。

### 你可能是 NZ Tech Twitter 上最受关注的人之一，你认为你是如何做到的？对于围绕你的兴趣发展一个社区，你有什么建议？

我从来不知道我是，但我会接受的！我的兴趣范围很广，所以我的反馈可能没有他们应该有的那样集中，但我建议有一个描述性的简历。你的兴趣，你来自哪里，所有这些事情都会让你和其他人产生个人关系。

我喜欢技术和科学，所以我经常在推特上谈论硬件、基准测试、游戏、科学发现、编程，当然还有任何我觉得有趣的事情。*咳咳爸爸的笑话*

如果你看到其他用户在推特上谈论你的兴趣，不要害怕参与对话。这里或那里的一些好的回复不仅意味着一个关注，而且意味着一个新朋友。

我经常直言不讳，说出我的想法，这让一些人感到不安，但也得到其他人的赞赏。做你自己，选择你的战斗。

### 迄今为止，在你的科技生涯中，你学到的最难的一课是什么？

跟上。毫无疑问，作为一名开发人员是艰难的。一个开发人员不断地在一个每天都在加速变化的世界中工作。今天的新框架，明天的全新语言。这比我想象的要困难，但我喜欢挑战，被扔进深水区。

我个人认为这是一种很好的学习方式，承担一些我不太理解的任务让我保持警觉。

### 对于成功的科技职业生涯，你的第一条建议是什么？

我不能代表其他技术职业，但对于那些想成为开发者的人来说，毅力和激情。没有人能瞬间理解甚至记住所有的事情。你会经历自我怀疑的时期，但没关系，这比你想象的要正常得多。

我经常发现自己相信我遇到的每个开发人员都比我知道得多，这让我感到很自卑，但同时他们也可能在想同样的事情。这就是毅力的来源，不要让那些自我怀疑的想法阻止你达到你的目标，因为我可以向你保证，你会克服它，你会回头看它，意识到你已经走了多远。

你的激情是你坚持不懈的动力。如果你喜欢编程，那就从零开始构建。坚持做下去。你没有什么可失去的，你还可以获得一个全新的世界。

### 工作之外，你有什么爱好吗？你认为它们对你的科技事业有任何帮助吗？

我确实喜欢时不时地骑山地车，和家人在一起或者帮助别人。我也支持我 15 岁的侄女成为一名开发人员。

我认为保持健康的工作/生活平衡很重要。时不时地离开电脑。虽然我确实发现自己有点“渴望”再次编码，如果我有一段时间没有编码的话。

所以我想作为一个爱好，我也编码，这是我学习和提高技能的空闲时间。

### 你会推荐什么书/资源？

我订阅了[代码学校](https://www.codeschool.com/)，经常用[代码学院](https://www.codecademy.com/)。我是一个视觉型的人，所以我更喜欢基于视频的学习和教你的互动课程。

但是最好的学习是你自己的项目。在你的生活中找到一些你可以自动化的东西，或者一个你一直想建立并实现的应用想法。

我通过尝试构建自己的程序自学了一切，因为我知道我想要什么，我只需要知道如何实现它。询问周围的人，和其他开发者交流。谷歌是开发者最好的朋友。

但是当然值得一提的是 [@CodingPoet](https://twitter.com/CodingPoet) 。一个南非人，致力于在学校教授和激励孩子们关于技术、机器人和编码的知识。我强烈推荐一个追随者。
