# 通过升级 SecureBridge 这一可靠的网络安全解决方案确保数据安全

> 原文：<https://dev.to/devartsoftware/ensure-data-safety-with-upgraded-securebridge-a-reliable-network-security-solution-1hln>

Devart 是一家公认的为开发人员和 DBA 提供专业数据库工具的供应商，它发布了一个新版本的网络安全解决方案 SecureBridge。现在 SecureBridge 支持 FTP 和 FTPS 协议，允许用户以通用的方式安全地交换文件。

另一个值得一提的特性是可以通过代理服务器连接到远程主机，从而间接连接到各种网络服务。除了上述功能之外，该版本还包括以下功能:

使用 FTP 协议时支持无缝访问远程文件的 TScFTPClient 组件。线程安全的 TScStorage 为来自不同线程的多个 SSH 和 SSL 客户端使用一个存储。
TScSSHConnectionInfo。LocalSockAddr 属性来检索本地 IP 地址。
TScSSHServerOptions。MaxConnections 属性来限制 SSH 服务器上的活动连接数。
TScSSHServer。BeforeClientConnect 事件在客户端进行身份验证之前根据其 IP 拒绝或接受连接，这大大提高了服务器性能。

在[https://www.devart.com/sbridge/](https://www.devart.com/sbridge/)深入了解 SecureBridge 的其他有用功能。
