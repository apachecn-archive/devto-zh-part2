# [](#what-is-service-workers)什么是服务工作者？

服务人员可以在我们的浏览器中运行后台，无需用户交互。他们在自己的线程上工作，甚至在浏览器关闭/网站关闭时也能工作。

服务工作者只在安全的站点上运行，比如(https ),在开发时它将在本地主机上工作。

今天，我们使用服务人员来构建我们的离线网页。
