# 前进之战——第二部分:渥太华

> 原文：<https://dev.to/qlikbranch/the-forwardjs-battle--part-2-ottawa-5h8m>

随着旧金山的结束，我连续第二年把目光转向了和我一起回到 T2 的家。

ForwardJS 年的渥太华是对加拿大首都的一个冷静、冷静的介绍。但是今年，我们想做得更大…更好。

唯一的办法就是在大会上安排大型研讨会和为期两天的明星演讲，我们做到了。

第一天看到前锋全力冲出大门。随着[安迪·莫克勒](https://medium.com/u/fadbcd01c7a3)和[克里斯汀·斯潘塞](https://twitter.com/kristencodes)开始向我们传授软技能，这一天充满了令人敬畏的演讲，如[亚当·道](https://medium.com/u/624aec3174db)吹捧[透明的力量](https://medium.com/coventure/js-minus-js-the-future-of-the-javascript-community-is-better-through-transpilation-b980d59eaa93)， [Jan C. Liz-Fonts](https://medium.com/u/6d5313b8bef9) 展示如何在 Javascript 中驾驭[区块链](https://forwardjs.com/ottawa/schedule#lecture-403)和[玛丽·斯诺](https://twitter.com/ThisIsMaryCodes)解码 [NodeJS 事件循环](https://forwardjs.com/ottawa/schedule#lecture-387)。

第一天带来了如此多的知识——所以第二天的演讲者真的有他们的工作要做。他们完全胜任这项任务。这一天见证了 [Jenn Voss](https://twitter.com/VossJenn) 带给我们[来自 QA Crypt](https://forwardjs.com/ottawa/schedule#lecture-391) 的故事、 [Eric Adamski](https://medium.com/u/cf9894fccad7) 的惊人能量以及 [Rx 和 Async](https://forwardjs.com/ottawa/schedule#lecture-402) 、 [Ksenia Coulter](http://kscoult) 教导我们[从代码评审中获得最大收益](https://forwardjs.com/ottawa/schedule#lecture-399)和 [Brian Tavares](https://twitter.com/_briantavares) 以一个精彩的 [React Native](https://forwardjs.com/ottawa/schedule#lecture-388) 演讲结束。

[![](../Images/69a918a89e2029f1fea8a7119a17a69c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ee2sr1Jf--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2AtycsPdCz-1xigrlA5WwMCw.jpeg)

我很愿意承认偏见，也不会否认这可能部分是因为我参与了组织活动，或者因为这是在我的家乡，但在旧金山对渥太华的战斗中，我想说“边境以北”的竞争者今年获得了 ForwardJS 的冠军。
