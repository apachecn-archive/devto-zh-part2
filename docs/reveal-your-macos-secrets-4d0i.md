# 透露您的 macOS 秘密

> 原文：<https://dev.to/nickytonline/reveal-your-macos-secrets-4d0i>

不久前，我写了关于我的 Mac 设置的文章😄).为了让自己更有效率，我做了很多调整，但是有一件事一直困扰着我。

每次本地对话框弹出时，我都无法切换按钮。不管出于什么原因，我只是假设这在 macOS 中是这样的，但同时我又想为什么在 OS 中没有这种功能呢？今晚，我的击键流程又一次被一个本地对话打断，我再也受不了了。

一个非常快速的谷歌(为什么我以前没有这样做)，揭示了这确实是可能的，通过设置深埋在键盘设置。非常感谢这篇文章，[在 Mac OS X](http://osxdaily.com/2010/02/26/use-the-tab-key-to-switch-between-dialog-buttons-in-mac-os-x) 中使用 Tab 键在对话框按钮之间切换。我现在是一个快乐的露营者。
