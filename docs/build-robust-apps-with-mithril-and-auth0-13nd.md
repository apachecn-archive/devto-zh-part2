# 使用 Mithril 和 Auth0 构建强大的应用程序

Mithril 是一个用于构建单页面应用程序的小型快速 JavaScript 框架，只需 10 分钟就能学会。

在本教程中，我们将使用 Mithril 构建一个相当复杂的 web 界面来存储有关会议的数据，并用 Auth0 对其进行认证。
