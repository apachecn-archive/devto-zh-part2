## Laravel

[![1*OWWt18PDcDrWtBJUqGZHZw.png](../Images/e3dcfb68ab165c5c40e7a910314eaeb6.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--hOPH39iR--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/560/1%2AOWWt18PDcDrWtBJUqGZHZw.png) 
**描述:**
是具有高效语法的 web 应用框架。Laravel 试图通过简化大多数 web 项目中使用的常见任务(如身份验证、路由、会话和缓存)来减轻开发的痛苦。Laravel 易于访问，但功能强大，为大型健壮的应用程序提供了所需的强大工具。一个极好的反转控制容器，富于表现力的迁移系统，以及紧密集成的单元测试支持，为您提供了构建任何应用程序所需的工具。

GitHub:[https://github.com/laravel/laravel](https://github.com/laravel/laravel)

## HTML5DOMDocument:

[![HTML5DOMDocument.jpg](../Images/30f140aeeba773e0428098c464c85f5f.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--APtCCepK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.portalzine.de/wp-content/uploads/sites/3/2017/02/HTML5DOMDocument.jpg) 
**描述:**
HTML5DOMDocument 扩展了原生 DOM 文档库。它修复了一些错误并增加了一些新功能。保留 html 实体(DOMDocument 不保留)。保留 void 标记(DOMDocument 不保留)。允许插入 HTML 代码，将正确的部分移动到正确的位置(head 元素插入 head，body 元素插入 body)。

GitHub:[https://github.com/ivopetkov/html5-dom-document-php](https://github.com/ivopetkov/html5-dom-document-php)

## Guzzle，PHP HTTP 客户端:

[![guzzle.jpg](../Images/7be583e536c5b8ecf6f56a7e28a4a1e7.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--7qG1F_2W--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.bram.us/wordpress/wp-content/uploads/2012/12/guzzle.jpg) 
**描述:**
Guzzle 是一个 PHP HTTP 客户端，它可以轻松地发送 HTTP 请求，并轻松地与 web 服务集成。
用于构建查询字符串、发布请求、流式传输大量上传、流式传输大量下载、使用 HTTP cookies、上传 JSON 数据等的简单接口...
可以使用同一个接口发送同步和异步请求。
对请求、响应和流使用 PSR-7 接口。这允许你用 Guzzle 利用其他 PSR-7 兼容库。

GitHub:[https://github.com/guzzle/guzzle](https://github.com/guzzle/guzzle)

## PHP 解析器

**描述:**
这是一个用 PHP 编写的 PHP 5.2 到 PHP 7.2 的解析器。其目的是简化静态代码分析和操作。将 PHP 5 和 PHP 7 代码解析成抽象语法树(AST)。以人类可读的形式转储 AST。将 AST 转换回 PHP 代码。

GitHub:[https://github.com/nikic/PHP-Parser](https://github.com/nikic/PHP-Parser)
