# 你记日记吗？

> 原文:[https://dev.to/rommik/do-you-keep-a-journal-28ln](https://dev.to/rommik/do-you-keep-a-journal-28ln)

你是否写日记或日记来记录你每天的职业活动？你是怎么做到的？纸笔？word 文档？专用软件或工具？

自从我开始作为一名开发人员工作以来，我一直在写下我一天的总结..已经 13 年了。哎哟！然而，我从来没有保存或存档过它们。实际上，我要么把它们交给了我的雇主，要么把它们销毁了。现在，我后悔了。它们应该是很好的自我反省和衡量进步的读物。

不把它们放在身边的原因之一是空间，它们占据了很多空间。我在想是否有更好的方法。钢笔和记事本是我的必需品，我喜欢画画和涂鸦。在平板电脑或笔记本电脑上做不是一回事。

当我听说这款非凡的平板电脑时，我非常激动。这是你能得到的最接近纸和笔的东西，但是价格标签和电池太不合理了。800 加元的平板电脑只是另一个带笔的 Kindle，充电后可以运行 2-3 个小时，不用谢！也许如果价格降到 200-300 美元，我会再考虑一下。

在大学期间，我使用 Livescribe pen。这是一个数字笔，记录你写的东西，你可以用数字形式(PDF 或图像)查看。这真的很棒，但是你必须有特殊的笔记本。那是令人沮丧的！

现在有了智能手机和图像识别，我不介意把我的笔记拍下来数字化。然而，当谈到日记笔记时，只有当你可以搜索它们时，它们才是好的。所以，我不确定这是不是该走的路。
