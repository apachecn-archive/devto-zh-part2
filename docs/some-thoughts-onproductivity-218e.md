# 关于“生产力”的几点思考

> 原文：<https://dev.to/captainsafia/some-thoughts-onproductivity-218e>

你好。所以我真的很喜欢写这些博客，在那里我写下了我经常在会议上、直接信息中、通过电子邮件等从人们那里得到的问题的答案。今天，我想我会回答一些我经常听到的关于…生产力的问题。

出于这样或那样的原因，人们认为我是一个很有生产力的人。我在其他博客文章中提到过，我认为生产率是一个主观的基准。对不同的人来说,“高效”看起来是不同的。我不认为自己是有成效的，但是我想看到我的工作成果的人也会这样看待我。

好吧！是时候回答一些问题了。

> 你怎么做这么多？

这是我经常听到的。大多数时候，我会粉饰太平，谈论我的激情和热情以及我的目标。这都是真的。但我能对你说实话吗？我最大的动力之一是我是不可能满足的。我永远也不会觉得自己做了足够多的会议讨论，写了足够多的博客，提交了足够多的拉取请求。诚然，这是一种危险的意识形态，我不建议其他人遵循。我怎么做这么多？我对“多”的定义每天都在往前推。昨天对我来说足够的东西今天对我来说不够？有些人可能会对此持否定态度；其他人可能会羡慕地看待这种世界观。我不太在乎那些意见。这是我作为移民的女儿(我自己也是移民)，一个低收入家庭中最大的孩子，一个被期望获得满分的女孩的经历中得出的观点。

这就是那个问题的答案，完全没有废话的答案。不管是好是坏，我沉迷于做更多的事情。

> 你睡觉吗？

哦天哪！这个问题。我承认。我有很长一段时间睡眠不足(并为此感到非常自豪)。从那以后，我变得聪明了，也吸取了教训。我承认，在学校和其他一些事情之间，我必须平衡，我的日子必须早开始晚结束。我没有试图反抗。相反，我欣然接受了午睡的美妙。回想起来，这是我母亲经常做的事情。在抚养孩子、工作和获得第二个大学学位之间，她也有许多早出晚归的日子。不管怎样，她通常会满足于饭前或饭后小睡一会儿。我也开始这样做了。我发现它增加了我每天的睡眠时间，增加了我深度睡眠和快速眼动睡眠的时间，并帮助我在中午昏昏欲睡的时候理清思绪。谢谢妈妈。

> 你每天都检查你待办事项清单上的所有事情吗？

哈！不。我会说在一天结束的时候，通常会有一两件事我没有做。它们通常是我最不想做的事情。我试着不要为此太自责，只是在第二天优先考虑这些事情。不过，老实说，我也犯了同样的错误。

> 你如何保持动力？

根据不同的日子，不同的事情激励着我。有时候，我会被那些讨厌的人所激励。其他时候，我被家人做出的牺牲所激励。另一些日子，我被我昨天做出的牺牲所激励，才走到今天。其他时候，我被一个女孩必须支付账单的事实所激励。我的动机通常是由当天的事件、我经历的事情以及我所处环境的其他微妙方面决定的。

我不认为动机必须是一贯的利他或积极的。如果你的努力有时变得有点自私也没关系(只要不总是这样，并且你意识到你目前动机的自私本质)。有时候你会觉得有点自私。有时候你会觉得有点无私。只要你知道自己想要什么，为什么想要它(并且你的动机和其他一些注意事项没有伤害到任何人)，你就很好。

> 你什么时候睡觉？

大约午夜。

> 你什么时候醒来？

早上 6:45 到 7:00 之间。

> 你如何克服下午的低迷？

一杯热红茶。

> 你为什么把你的待办事项写在纸上？

它让我更容易写我的日常任务，而不必上网络或移动应用程序，也不会被互联网上的所有好东西分散注意力。它也是我所完成的事情的一个很好的记录。大多数待办事项应用程序都没有给你一种方法来查看你在哪一天完成了什么任务。在纸上写下我的待办事项清单让我能够了解我工作的历史和进展，并从中获得一些动力。

> 为什么你在谈论生产力的时候这么讽刺？

我对一个日常工作日益自动化的世界有某种感觉，作为资本主义社会中的人，我们获取价值的唯一方式是从清单上检查事情。我试着内省，审视自己想做更多事情的欲望，并意识到有多少是因为我不想觉得自己是无用的空间浪费。“生产力”在科技行业是一个非常沉重的术语，我觉得它带有很多阶级歧视和性别歧视。通常，被认为是有成效的事情是那些让你赚更多钱的事情。抚养一个孩子(传统上是母亲的任务)不算多产，尽管这是一项巨大的事业。创作公共艺术(一项纯粹利他的努力)通常不被视为富有成效。我偏离了使用“生产性”这个词，以及由此衍生的概念体系，因为我认为这是一个对社会有害且有局限性的概念。有时，我对这个词、这个概念及其对社会的影响的厌恶以讽刺的形式表现出来。

> 作为一个以任务为导向、雄心勃勃的人，什么能帮你释放压力？

这种变化。在夏季的几个月里，我喜欢去我公寓附近的海滩，在海浪中袖手旁观，眺望密歇根湖，或者清理冲上海岸的啤酒瓶和啤酒罐。在秋天，我喜欢在晚上散步。最近，我开始进行高强度的锻炼。我在说被汗水湿透，不能说话，不能呼吸的运动。谁知道我接下来会做什么来缓解压力。
