# Aspose 中的新内容。CAD 18.6

> 原文:[https://dev.to/nnevod/whats-new-in-asposecad-186-op8](https://dev.to/nnevod/whats-new-in-asposecad-186-op8)

Aspose 的新版本。CAD，指定为 18.6，即将发布。以下是最重要的变化:

## 新增功能

### 基本支持 IGES 文件格式

现在做个手势。CAD 可以加载和渲染 2D IGES 文件。这种支持还处于早期阶段，所以很有可能，许多文件将无法正确呈现。如果是这样，请通过[支持论坛](https://forum.aspose.com/)将文件发送到 Aspose。

## 改进

### 代理实体支持 DWG/DXF 文件

代理实体支持以前部分存在，但不完整，实体属性没有正确解析，现在它是正确的。

### 改进 DWG/DXF 文件的影线渲染。

与代理实体支持相关，影线渲染现在得到了改进。

## Bug 修复

### 修正了光栅格式的虚线渲染。

导出为光栅格式时粗虚线的渲染已修复。

### 渲染选项现在在导出为光栅格式时有效。

现在，通过设置用于导出图像的 [CadRasterizationOptions](https://apireference.aspose.com/net/cad/aspose.cad.imageoptions/cadrasterizationoptions) 的 [GraphicsOptions](https://apireference.aspose.com/net/cad/aspose.cad.imageoptions/vectorrasterizationoptions/properties/graphicsoptions) 的相应属性，可以在导出为光栅格式时使用抗锯齿、插值和文本渲染选项。它们是预先可用的，但不适用于光栅图像。
