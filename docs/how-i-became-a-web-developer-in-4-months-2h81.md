# 我如何在 4 个月内成为一名网络开发人员

> 原文：<https://dev.to/smithmanny/how-i-became-a-web-developer-in-4-months-2h81>

就像标题说的那样，我只用了 4 个月的学习时间就获得了我的第一个 web 开发职位。老实说，可能 4 个月多一点。这一切都要追溯到我高三的时候。我的第一堂计算机课是“HTML/CSS 入门”。那是在 2010 年，所以我不确定响应式设计是否已经过时了，但我知道我们没有学过。这堂课只是在复习网页开发的基础知识，我们从来没有建立过什么“惊人”的东西，只是一些简单的带有一点风格的网站。我喜欢这门课，但不幸的是，我上完这门课后就不再编程了。

快进到 2016 年，我在 Publix warehouse 工作，担任订单选择器。生活是美好的，我挣着体面的钱，有全额福利(这是我的第一份正经工作)。直到他们宣布搬到阿拉巴马州。无论如何，我不认为自己会以仓库工人的身份退休，所以我让我的经理们知道我不会和他们一起搬迁。在地点转移之前我们有 12 月底的时间。对我来说幸运的是，我在 11 月份离开，并注册了 Treehouse 团队。这实际上很尴尬，因为当我离职时，我只是随机看到了一个前端 Web 开发的广告，所以我注册了 7 天免费试用。

完成 7 天的试用后，我又迷上了 web 开发。所以从试用升级到实际程序。就好像我上瘾了，我不能停止学习。每天我花 3 个多小时学习。有时我会通宵完成课程，睡上几个小时，醒来后再做一遍。

所以从 2016 年 11 月到 2017 年 2 月，我能够完成 Team Treehouse 前端开发者栈。一旦我完成了，我就有足够的信心开始着手一些项目和我的作品集。我希望我仍然有我的第一个投资组合设计来提醒自己我走了多远，但我当时不知道 git🤦🏾‍.不过这没什么特别的，这是一个用 HTML5、CSS3、Bootstrap 和一行 jQuery 构建的基本组合页面。老实说，我为我的成就感到骄傲，我开始申请工作。通过向多家公司发送了多份申请后，我幸运地获得了一家小公司的面试机会。

我仍然记得我接到第一次开发人员面试电话的那一天。我简直不敢相信。就在三个月前，我还是一名仓库工人，试图弄清楚我这辈子要做什么。走进面试时，我很紧张。我无法停止出汗，我开始全身发冷，我出汗太多，我觉得我开始闻到。我终于和办公室经理坐下来，开始了关于我的谈话。她想更好地了解我，了解我知道什么和不知道什么。所以我对她很诚实，我的意思是她已经知道了，因为她看过我的简历。在告诉她我是自学的，刚开始工作后，她最后告诉了我更多关于这家公司的信息，以及我的角色。当我们聊完之后，她把我介绍给了 IT 经理和首席执行官……我是不是得到了这份工作？

嗯，我发现他们已经决定雇佣我了，因为我的作品。他们知道我刚刚起步，想要一个懂 PHP 的人，但是因为我的决心和我学东西的速度，他们当场就给了我这个职位！

因此，我对那些感到迷茫并认为自己在浪费时间的新开发人员的建议是，你没有。继续学习和完善你的手艺，建立一些项目，建立一个投资组合和网络。不要放弃，因为最终一切都是值得的。
