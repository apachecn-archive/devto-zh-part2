# 即将到来的 WordPress 编辑器有什么新内容？

> 原文:[https://dev . to/alexanderbiscajin/WordPress-gutten Berg-review-whats-new-in-the-coming-WordPress-editor-i67](https://dev.to/alexanderbiscajin/wordpress-guttenberg-review---whats-new-in-the-upcoming-wordpress-editor--i67)

Gutenberg WordPress editor 是一个新的编辑器，允许初学者和高级用户为你的网站编辑和制作内容。目前它处于测试阶段，并不完全适合发布，但可以作为测试用的插件。

[![](../Images/b7c3f279eaff902436cbbcad477f5b8c.png)T2】](https://youtu.be/IKwGmqLM8l8)

**如何安装古腾堡编辑器**

安装 Gutenberg WordPress 编辑器简单而容易。

转到插件>添加新的，然后输入古腾堡。在安装 Gutenberg WordPress 编辑器之前，也要确保你的 WordPress 站点升级到了最新的 4.9 版本。如果已经升级，您会在右下角看到一个复选标记。之后，按右上角的“立即安装”按钮。

[![](../Images/cb334ed11eec404be77116e66492a25c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Kgl8lhs1--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/pjxgcsnbxl6ufubnri3v.png)

最后，现在按下蓝色的激活按钮。

[![](../Images/f5e2a7c3e283fd3e9f8e466e33ed8a16.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--HysK4RzJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/0y919mqv6rdd0cgpb0ki.png)

**如何使用新的古腾堡编辑器**

成功安装插件后，您可以通过两种方式访问它:

1.  选择文章>所有文章，点击古腾堡而不是编辑链接。另请注意，旧的编辑链接仍然可用。

[![](../Images/41a40c362bc8e4b1f7479511b1018e48.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--D5eRLMME--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/svkwq2fxw7pr2o2p0ygh.png)

1.  在仪表板上选择 Gutenberg >新帖子。

[![](../Images/9364b940a7ac0ad1c9fa38ab0c468e9f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ctS0lNRA--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ezlnjys6bq9wkhcrlw70.png)

它看起来像 medium.com 风格的设计。此外，发布元数据配置可在右栏的选项卡中访问。你可以通过点击文章设置来隐藏/显示。

[![](../Images/c313c7683a6868f66e852cc7e6a17c74.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--rmQcvnAH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4bvbtowz4ik6qq28retj.png)

您可以有效地添加、格式化和移动文本。你可以开始写标题和正文。如果你想添加另一个段落，然后只需点击+>文本。

[![](../Images/85d2d020c6d8354275e1ac160f8f9989.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--hbwZwTxK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dai4eoy0kyox7e5tvmrl.png)

这个新 Gutenberg 编辑器使用块来组织文本。写完段落后，您可以使用旁边的箭头键轻松更改它在文本中的顺序。如果你想删除，你可以使用右边的垃圾桶图标。

[![](../Images/54238849efcc4b8085ae3597a1c0a417.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--J5PaYSKT--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/khkljb22jcld7ajy8nyr.png)

如果你想在两个段落之间添加一个标题，你需要通过+按钮，然后使用箭头图标移动它。

Gutenberg WordPress 编辑器最重要的亮点之一是它对多媒体和丰富内容的支持。嵌入过程真的很简单容易。要嵌入，只需点击+>嵌入>搜索 meetup >点击 Meetup.com

[![](../Images/c7099cf9c9fc17c8592f089934752f68.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--b-idbnNT--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/q45rtv7q9mcnhdkbwncp.PNG)

[![](../Images/006ceb25a9b35be5103a52ba599cde0a.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--qLZtDcl4--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/yw6qmksk67cfu0b1kg2p.png)

可用的嵌入式服务可以在 Twitter、脸书、Instagram、DailyMotion、YouTube、Kickstarter 和 Spotify、TED、Slideshare 和 Tumblr 上运行。你只需选择源，输入相应的网址，然后完成！

Gutenberg 编辑器提供了更多包括按钮和分隔符等布局块。

[![](../Images/4c6c74192d55d8e0855c05ada0aaedb2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--HHgvMR6n--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/rlr2891wy40qgtdk0wtb.png)

**引用语**
古腾堡还引入了被称为引用语的引用形式。

[![](../Images/872062cae7171f25ee726752a2e1f19e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--VcDTd5F3--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/luwe9ny0k4928os0xybh.png)

**表块**
这是对这个古腾堡编辑器最有用的补充。目前只支持 2×2 的表格，不需要对表格进行任何配置。您可以简单地添加列或行。这是古腾堡编辑器中的一个很好的选项，因为目前你不能在不使用第三方插件的情况下使用表格。

[![](../Images/9598f108ab7a6d9d529ae03bb2179860.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--sfoxcC_5--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/2k8rlsn2jxux7ul2otb3.png)

**自定义 HTML**
你也可以添加 HTML 块，方便预览。现在，您可以在文本和可视模式之间快速切换。
