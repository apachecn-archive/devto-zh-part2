# 支持 Lazarus 1.8.4，并在更新的 DAC 产品中增加了许多有用的新特性

> 原文:[https://dev . to/devart software/support-for-Lazarus-184-and-bunch-of-new-help-features-in-updated-DAC-products-5fj 4](https://dev.to/devartsoftware/support-for-lazarus-184-and-a-bunch-of-new-helpful-features-in-updated-dac-products-5fj4)

数据库工具和数据连接解决方案的捷克软件供应商 Devart 推出了新版本的 Delphi 数据访问组件，这是适用于任何数据库的快速可靠的数据连接解决方案。新版本包括对 Lazarus 1.8.4 的支持，UniDAC 中云与数据库提供商的全新功能，以及许多其他功能，以帮助程序员开发快速的原生应用程序。

下面是对该版本中实现的最值得注意的改进的详细介绍:

MySQL 数据访问组件(MyDAC)现在支持 MySQL 8.0，以及 sha2_password、caching_sha2_password 身份验证。SQL Server 数据访问组件(SDAC)的新版本包括 MARS、非阻塞模式和 TDS 中的查询通知。
InterBase 和 Firebird 数据访问组件(IBDAC)现在提供系统加密密码(SEP)。此外，更新的 IBDAC 提供了使用 IBCSecurityService 组件授予/撤销 Firebird 用户管理员角色的能力。
Oracle 数据访问组件(ODAC)现在允许在直接模式下显著增强数据提取。
新版 PostgreSQL 数据访问组件(PgDAC)支持 HTTP/HTTPS 隧道。
SQLite 数据访问组件(LiteDAC)现在在非 Windows 平台的直接模式下支持 WAL。
升级的通用数据访问组件(UniDAC)包括更新的云提供商，支持 Salesforce MC、FreshBooks 新版本、Magento 2.x、NetSuite Sandbox 和 ZohoCRM 域的 App center 客户端身份验证。
此外，在所有 Delphi 数据访问组件中，批处理操作的性能大大增强。
