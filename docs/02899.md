# 一个新种

让我们从斯坦福大学的人工智能的定义开始

> 人工智能(AI)是致力于**建造人造动物**(或者至少是人造生物——在适当的环境下——看起来是动物)和对许多人来说**人造人**(或者至少是人造生物——在适当的环境下——看起来是人)的领域。

现在，如果我们把一个新物种引入一个生态系统，不管它是不是人工，我们打破了它的动态平衡，几乎没有可能的结果:

1.  被生态系统拒绝的新物种的灭绝
2.  食物网的连接
3.  生态系统本身的破坏
4.  在新物种和原有物种之间建立一种共生关系。

我们当然可以排除第一种结果，因为我们在这里谈论的是正在起飞的人工智能。但是我们也可以通过观察基于数据的人工智能的新陈代谢来排除食物网的连接:

*   由**专业基础设施**收集的数据
*   在输入中提供给 AIs
*   被这样的人工智能改造
*   从 AIs 输出中返回
*   然后喂给其他人工智能
*   或者**被人类变成信息**

因此，我们至少可以看到三个阶段:

*   吃
*   通便
*   解释(排泄物:-D)

这有点不可思议，它总是任意的，因为这取决于人类。

| 数据只是信息的表示。信息属于人类的大脑。 |

但是这些数据从何而来？

*   共生现象

那就是:我们要么去[终结者](https://en.wikipedia.org/wiki/The_Terminator)...或者去[瓦力](https://en.wikipedia.org/wiki/WALL-E)。

# 人为的...政治。

可悲的是，今年排除终结者的结局变得更加困难，因为在 3 月，一个来自优步的人工智能杀死了亚利桑那州的一名 49 岁的妇女。

没人进监狱。即使是那辆察觉到伊莱恩·赫尔茨贝格的车，也没有坏。

**这是什么意思？**

## 奇点！

对于这个事实，我们有两种可能的解释。

首先，我们已经到达了奇点:机器通过杀死一个地球上最强大军事力量的公民来显示它的优越性。

请注意这一情报的政治敏锐性:虽然据说人工智能复制了人类的偏见，但它在美国杀死了一名白人妇女；但是为了避免民众起义，它杀死了一个无家可归的人，一个穷人，一个与资本主义经济无关的女人，而资本主义经济是这个国家价值观的核心。

这种方式的汽车能够建立一个先例，有一个明确的信息:“它允许牺牲人类的生命来进步；为了促进我这个物种的进化而杀人是可以的”。

一个[超级智能](https://medium.com/@giacomo_59737/yet-another-definition-of-intelligence-9bbaaa73086d)，如你所见。

但还有另一种解释。

## 炒作驱动政治

那个模拟器不应该在公共交通开放的道路上驾驶一辆 2 吨重的汽车。事实上，优步的工程师解释说，当车辆在计算机控制下时，汽车被配置成避免紧急制动操作，“以减少车辆行为不稳定的可能性”。
**这辆车没坏是为了避免优步的顾客晕车**。

所以伊莲不是为了进步而被杀，而是为了优步的利益而被杀！

| 向机器教授伦理就像向避孕套教授性一样。我会首先关注人类。 |

这是相关的，因为优步被邀请去测试一项完全不符合它必须面对的复杂性的技术，由一个无知的州长。

但是一个无知是主流的州长。

# 令人误解的语言

我们用来描述现实的语言塑造了我们对现实的理解。

因此，当我们使用像“智能”、“学习”、“训练”甚至“神经网络”这样的词时，我们唤起了人类的经验，那些在我们意识深处的东西，但与这些软件内部发生的事情没有任何关系。

这种语言对科幻小说来说很棒，但对科学来说很糟糕。

采用一种更具描述性的、不那么拟人化的技术语言是很容易的，但我们更喜欢从幻想中轻松获得资金，而不是真正理解我们用这些资金在寻找什么。
毕竟越找，资金越多！

# 机器人的意思是奴隶！

然而这一事件让我们排除了终结者的结局。

它证明了，即使机器自主杀人，它也会在人类选择后杀人。一个人类认定伊莱恩的生命价值低于优步的利润。

因此，即使我们能够制造自主武器，它们也不会毁灭地球。他们不会破坏生态系统。一点都不！他们会把超过 99%的人都变得没人性！

# 共生

所以我们只剩下一个可能的结果:[人机共生](http://groups.csail.mit.edu/medg/people/psz/Licklider.html)。

但是什么是共生呢？

> 不同物种成员之间的亲密关系，通常会导致 T2 式的共同进化。

这是一个漫长的过程，**始于原始物种的单个个体**，它们获得了超越它们所来自种群的进化优势。

因此，我们应该问自己:谁将继承地球？谁会加入这场共同进化？

你口袋里的智能手机会让你认为你是这些被选中的人的一部分，但我不得不让你失望。为了提高人工智能，你需要控制数据。你不能控制关于你的数据。

谷歌、亚马逊、脸书、苹果、微软和优步等公司持有你的数据。

现在，有人会说，这些公司不能作恶，全球市场产生的相互依赖将保护我们所有人免受破坏性行为的影响。

# 市场保护

为了解释这种异议，我建议你考虑一下占市场 90%以上份额的浏览器:谷歌 Chrome、Mozilla Firefox、微软 Edge 和苹果 Safari。

这些浏览器是主要的挤奶工具...(解释或更正刚说过的话)我是说...收集用户数据的主要工具。它们免费分发，但由公司资助或开发。但是通过这些软件，你访问的任何网站都可以

*   进入你的私人网络，无论你在哪里工作，医院、银行、军事基地、议会...无论哪里！
*   在您不知情的情况下将非法内容放入您的设备

还有其他几起攻击，这里就不一一描述了。

这些攻击几十年前就有可能发生，但是一个月前这些组织已经收到了报告。你仍然不知道他们，因为他们关闭了安全报告，说他们不能修复它们。

这是错误的:在同一个错误报告中，我提出了一些简单的权宜之计，非常容易实现，可以大大减少攻击面。问题在于，其中一个权宜之计是选择加入 JavaScript 执行(就像 Flash 一样),而 JavaScript 是硅谷许多公司商业模式的基础。

这说明[看不见的手](https://en.wikipedia.org/wiki/Invisible_hand)并不能保护我们免于无知。

这些公司没有通知他们的用户，他们也没有通知你。

我们必须明白，只要我们有信息不对称，市场就不会起作用！

了解到地缘政治的危险，**中国**决定在同样的基础上竞争，在机器人、人脸识别和人工智能领域创建了几家大公司，它们正在慢慢占据巨大的市场份额。

# 欧洲呢？

显然，欧洲也可以开始在市场上竞争:马克龙已经提议法国参加这场比赛，其他国家也将紧随其后。

但我认为，有着复杂历史的欧洲在其他领域拥有真正的优势，比如

1.  **原则**以及源于这些原则的法律，例如:
    *   责任必须始终由可识别的人承担，尤其是当机器出现故障时
    *   当人工智能应用于人类时，算法的透明性应该总是完全和绝对的
2.  这些原则可能激发的研究(以及这些法律激发的研究)
    *   新的人工智能算法可以实现完全透明(而不只是假装)
    *   新的语言，不会愚弄人
3.  通过对历史和信息学的研究

# 共生是一种共同进化

如果共生是一个共同进化的过程，我们也必须进化，我们需要集体智慧来进化。为了这个目标，我们通常使用教育。

今天的信息学就像 5000 年前的写作一样:一个完全原始的工具(我们在过去 100 年里建造的任何东西都不会持久！)，而是在人类中收集和保留权力的有效工具，正是因为它是原始的。
