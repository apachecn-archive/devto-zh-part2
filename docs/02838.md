# Web 开发的 5 个最佳机器学习框架

> 原文:[https://dev . to/Apium _ hub/5-best-machine-learning-frameworks-for-web-development-97h](https://dev.to/apium_hub/5-best-machine-learning-frameworks-for-web-development-97h)

[机器学习](https://dev.to/apium_hub/applications-of-machine-learning-in-cyber-security-you-need-to-know-about-26ie)是计算机科学的一个分支，它使用统计方法赋予计算机在没有人类直接监督的情况下自我改进的能力。机器学习框架改变了网络开发公司利用数据的方式。机器学习算法可以处理大量的非结构化信息，并将它们转化为可操作的见解和预测。

亚马逊、网飞、YouTube 和谷歌等公司使用机器学习来预测你可能感兴趣的内容，为你提供个性化的搜索结果，或确定你点击特定数字广告的可能性。现在有许多工具、库和机器学习框架用于构建机器学习算法——有些适合技术门外汉，只有基本的选项，而另一些则使你能够从头开始开发自己的算法。

为你的 web 开发公司选择合适的机器学习框架不是一件容易的事情。谈到机器学习，不同的公司有不同的侧重点。有些人可能更喜欢使用易于学习和利用的框架，而其他人可能会优先考虑参数优化或生产部署。如果你不确定使用哪种 ML 框架，看看我们列出的市场上 5 大机器学习框架。

## Web 开发的 5 个最佳机器学习框架

*   [**张量流**](https://www.tensorflow.org/)

TensorFlow 是一个开源软件库，用于一系列任务的数据流编程。它最初由谷歌大脑团队开发供内部使用，并于 2015 年在开源许可下首次向公众发布。TensorFlow 主要是一个符号数学库，但它也用于机器学习应用程序，如神经网络。它为创建解决各种任务的算法提供了一个环境，包括自然语言处理、语音/图像/手写识别、文本分类、预测和标记。它在桌面和移动平台上都可用，支持包括 Python、C/C++、Java、Go、Julia 和 R 在内的语言，通过包装器库沿着创建[深度学习模型。

tensor flow 灵活的架构使得开发者可以通过使用单个 API 在一个或多个 GPU 或 CPU 上轻松实现计算。Google 将 TensorFlow 用于其翻译服务，其他主要行业参与者，如空中客车、Twitter 和 IBM 也广泛用于研究和生产。](https://medium.com/cracking-the-data-science-interview/the-10-deep-learning-methods-ai-practitioners-need-to-apply-885259f402c1)

*   [**微软认知工具包**](https://www.microsoft.com/en-us/cognitive-toolkit/)

微软认知工具包是微软研究院开发的深度学习框架。它最初于 2016 年在麻省理工学院的许可下发布，并且是开源的。微软认知工具包是用 C++写的，接口同时支持 C++和 Python。

微软认知工具包使用有向图来描述神经网络可以执行的一系列计算。它支持 Nvidia 的 CUDA API，并行执行，自动微分，并有许多预训练的模型。Microsoft Cognitive Toolkit 经常用于在图像、语音和基于文本的数据上训练神经网络。微软认知工具包支持 RNN 和 CNN 类型的神经网络，因此能够解决图像、手写和语音识别问题。由于易于训练，并且支持跨不同服务器的各种模型类型，它仍然很受欢迎。目前，微软认知工具包缺乏对基于 ARM 的 CPU 的支持，因此它在移动平台上的使用仍然相当有限。

*   [**阿帕奇辛加**T3】](https://singa.incubator.apache.org/en/index.html)

SINGA 是一个 Apache 项目，用于创建开源机器学习库。它是由新加坡国立大学开发的，并于 2015 年在开源许可下首次发布。SINGA 是用 C++编写的，也可以和 Python、Java 一起使用。它为可扩展的分布式培训提供了灵活的体系结构，并且具有足够的可扩展性，可以在各种硬件配置上运行。

SINGA 的开发是为了支持直观的、基于层抽象的编程模型，它支持一系列不同的深度学习模型。由于其灵活的架构，它可以运行同步、异步和混合的训练方法。SINGA 的深度学习使用模型分区和[并行化](https://gist.github.com/peterhurford/3e2f88f549afc390c204)进行训练过程。它提供了一个直观且健壮的编程模型，可以与节点集群一起工作。SINGA 的主要应用是图像识别和自然语言处理(NLP ),它也用于医疗保健行业。

*   [**PyTorch**](https://pytorch.org/)

PyTorch 是基于另一个机器学习框架 Torch 的 Python 开源机器学习库。它最初是在 2016 年开发的，主要由脸书的人工智能研究小组维护。PyTorch 自发布以来，在机器学习社区中的采用率很高，被认为是 TensorFlow 的主要竞争对手。PyTorch 提供了两个高级功能:具有强大 GPU 加速的张量计算，以及基于磁带的深度神经网络 [autodiff 系统](https://alexey.radul.name/ideas/2013/introduction-to-automatic-differentiation/)。

PyTorch 的机器学习算法是按照正常的 Python 控制流程构建的，方便 Python 开发者学习。它主要用于自然语言处理，优步用于概率编程的 Pyro 应用程序就是建立在它的基础上。使用 PyTorch 的主要优势是高水平的速度和 GPU 利用效率，以及用于数据训练的预制模型。

*   [](https://chainer.org/)

 **Chainer 是一个开源的深度学习框架，建立在 Python 的 Numpy 和 CuPy 库之上。因此，它只支持基于 Python 的接口。这项开发由日本风险投资公司 Preferred Networks 与 IBM、英特尔、微软和英伟达合作领导。它于 2015 年首次发布，此后在日本和海外获得了大量粉丝。Chainer 因其早期采用由运行定义的机器学习方法而闻名，以及其在大规模系统上令人印象深刻的性能，这就是为什么许多顶级数字机构开始广泛使用它的原因。

Chainer 允许开发人员在运行时修改他们的神经网络，允许他们轻松地执行定制的控制流语句。Chainer 支持 Nvidia 的 CUDA API，允许它在多 GPU 设置上运行。它主要用于情感分析、机器翻译和语音识别。
