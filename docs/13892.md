# 如何用英语写日语技术博客

> 原文:[https://dev . to/suin/how-to-write-technical-blog-in-English-for-Japanese-1df 1](https://dev.to/suin/how-to-write-technical-blog-in-english-for-japanese--1df1)

1.  打开 hackmd.io 并创建一个新条目，将窗口放在屏幕的左侧。
2.  在 Google Chrome 的另一个窗口中打开该条目，并将其放在屏幕的右侧。
3.  在右侧窗口中启用英语到日语的翻译。
4.  在左侧窗口中，用英语书写内容。在右侧窗口中，检查翻译成日语的内容。

你的英语会被谷歌翻译翻译成日语。你可以通过看到翻译来确认你的写作是好是坏。如果你的英语翻译得好，你的写作就可以，否则就不行。

## [](#how-to-rename-git-local-branch)如何重命名 Git 本地分支

