# 去哪里找人教

> 原文:[https://dev.to/zellwk/where-to-find-people-to-teach-404h](https://dev.to/zellwk/where-to-find-people-to-teach-404h)

你可以教任何想学你已经知道的东西的人。

你可以在很多地方找到这些学生，无论是线上还是线下。这里有一些你可以尝试的策略。

## [](#finding-students-offline)离线查找学生

如果你是教学新手，我建议你先从线下教学开始。我说的线下，是指面对面的教人。

先离线教学总是更好，因为你可以从学生那里获得即时反馈。如果他们说“啊”，你就知道出事了。

刚开始的时候尽量免费教。不收费的时候找学生容易多了。这里有三种方法:

### [](#1-give-a-talk-at-a-meetup)1。在集会上讲话

Meetup 组织者总是在寻找志愿者来做演讲或举办研讨会。你可以自告奋勇。大多数聚会都会很高兴有你。

例如，如果你想在新加坡做一个关于 CSS 或 JavaScript 的演讲，可以去找[新加坡 CSS](https://singaporecss.github.io) 或[新加坡 JavaScript](https://www.meetup.com/Singapore-JS/) 。他们会很高兴欢迎你的。

不要害怕站在人群面前教书。不要让你的冒名顶替综合症阻止你教书。

你可以教任何想学你已经学过的东西的人。如果你刚刚学了这个科目，那就更好了，因为你知道经历这个过程是什么感觉。你可以看到摇摇欲坠的立足点。

例如，Shirlaine 和 Grace 在学习了如何使用 CSS Grid 之后，马上在新加坡 CSS 做了一个演讲。他们在新加坡青年开发者大会上又做了同样的演讲。这是他们谈话的视频。它很受欢迎。

你也可以做个演讲。

### [](#2-create-or-join-a-study-group)2。创建或加入学习小组

学习小组很棒，因为你可以向同龄人教学和学习。全世界有很多学习小组。例如，世界各地都有 freeCodeCamp 的学习小组。如果你想，你可以加入一个。你也可以创建一个。

如果你想学习另一个主题，你也可以创建自己的学习小组。如果你这样做了，请继续邀请人们加入你的聚会。如果你的学习小组是免费的，他们会很乐意分享的。

### [](#3-volunteer-to-be-a-mentor)3。自愿成为导师

你可以教任何想学你所学的人。如果你有朋友或同事，试着教他们。

如果你没有朋友或同事，你也可以尝试在帮助人们学习编码的组织中做志愿者。这种组织的一个例子是新加坡的科技女性。

## [](#finding-students-online)在线查找学生

在线教学有两种方式:

1.  通过文章
2.  通过视频

你可以从任何一种媒介开始。当我开始学习编码时，我开始通过文章进行教学。

请记住，**由于以下三个原因，在线教学**更加困难:

1.  你得不到即时的反馈，因为你看不到你的学生
2.  你需要把人们引向你的文章或视频，这可能相当困难
3.  当没人问你问题的时候，你很难找到可以教的话题

这就是为什么我建议你开始离线教学。如果你以前教过几个人，你就知道怎么教了。

但是如果你愿意，你可以从网上教学开始。我自己也是这样开始的。请记住，你会慢慢看到结果。

## [](#wrapping-up)包装完毕

你可以教任何想学你已经知道的东西的人。线上和线下都可以教。

去教书吧。
