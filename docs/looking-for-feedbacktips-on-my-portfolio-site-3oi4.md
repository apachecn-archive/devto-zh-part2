大家好！

我是一名有抱负的开发人员，试图在行业中取得突破。按字面意思(差不多)(！)互联网上的每一个人，当你需要展示技能的时候，作品集是最重要的。所以，我花了一些时间来创建我的第一个作品集网站。

我想请你在这里看一看。

我不是设计师，但我喜欢漂亮的东西。而且简单。但是我不确定我做的一些设计/内容决定。我脑子里有很多问题。

例如，我可能在关于和**项目**页面的**上添加了太多的文本。你通常读这么多文字吗？此外，我对自己在**关于**页面的语气感到困惑。我相信我是一个谦逊的人，我希望这能在我的话语中得到体现。这是在这里发生的吗？你喜欢这些颜色吗？它们是太有活力还是太沉闷？动画带来的坏处比好处多吗？不胜枚举...**

我期待任何和每一种反馈/建议/提示/评论。我怎样才能做得更好？你会改变什么？此外，我很想看看你的投资组合，这样我就可以偷一些想法，把它放在我的投资组合。我开玩笑的，说真的！；)

我用 React 和 SVGs 制作了这个网站的动画元素。我根本不是专家，所以可能会有一些错误。如果你想看代码，这个网站在 GitHub 上。但是你要自己负责，因为我的代码很糟糕。:(

现在我要一边等你的回应，一边去抓点零食。祝你有美好的一天！:)
