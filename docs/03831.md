# #Hacktoberfest —通过学习创建 Github Pull-Request # OneDevMinute，为开源做出贡献

> 原文:[https://dev.to/ahmadawais/hacktoberfest-通过学习创建 github-pull-request-one dev minute-31f 0](https://dev.to/ahmadawais/hacktoberfest--contribute-to-open-source-by-learning-to-create-a-github-pull-request-onedevminute-31f0)

🙌作为一名全职# OpenSourcerer 十月是我最喜欢的月份之一。现在已经十月了。今天的#OneDevMinute 是关于#Hacktoberfest 的——这是一个为期一个月的在线活动，在 10 月 1 日至 31 日期间，你在 GitHub.com 和 [Hacktoberfest](https://hacktoberfest.digitalocean.com/) 注册至少五次，以庆祝开源。

来自 DigitalOcean、GitHub 和 Twilio 的✅好人将跟踪你的开源贡献，在五次成功的 GitHub 拉请求后，你将收到赠品，如限量版 t 恤、贴纸，也许还有一些折扣代码。

😇开源是面向所有人的，不仅仅是代码——开发者、设计师、甚至营销人员和内容作者都可以参与。改进开源项目的文档，重构代码，给项目一个新的 logo 设计——只要在十月参与并庆祝开源。

```
# Step #0: Create an account on GitHub.com (Also signup at https://hacktoberfest.digitalocean.com/).
# Step #1: Fork any open source GitHub repo.
# Step #2: Clone that repo in your local PC.
git clone https://github.com/AhmadAwais/Hacktoberfest

# Step #3: Make changes and commit/push these changes.
git add .
git commit -m "📦 NEW: Resource for learning Git"
git push

# Step #4: Go to your fork, click open `Pull Request` link and create a PR. 
```
