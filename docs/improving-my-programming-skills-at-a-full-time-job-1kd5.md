# 在全职工作中提高我的编程技能

大学一毕业，我就开始全职工作，已经两年了。最初，我认为这会限制我提高编程技能的时间。它肯定没有，以下是我从那以后学到的东西。

首先，**我换了 Linux** 。我开始在装有 Windows 的笔记本电脑上编程。我非常反对使用 Mac OSX 作为我的主要操作系统(现在仍然如此)，所以我从未买过 Macbook。但是，如果你是一名程序员，使用 Windows 简直太可怕了。这就是为什么我换了一个基于 Linux 的操作系统，对我来说是 Ubuntu。这是我能做的最好的事情之一。使用 Ubuntu 作为我的主要系统让我学会了使用终端来完成大部分日常任务。这反过来又教会了我在基于 Ubuntu 的服务器上安装软件的宝贵经验。

其次，**我学会了使用 PHP 和 JavaScript** 之外的其他语言。我对 JavaScript 相当熟悉，所以我的知识主要是 PHP 和 JavaScript。对我来说，这是你开始建立网站的最好基础之一。当然，您可以将 PHP 改为 Python 或 Ruby，但是任何这些组合都是构建的良好基础。从 JavaScript 和 PHP，我去学习 Python 和 Java 的点点滴滴。我通过与 Raspberry Pi 的交互学习了一些基本的 Python 原理，并通过 Solr 学习了 Java。Solr 是一个基于 Java 的搜索引擎，很像弹性搜索。

第三，**我开始使用 Docker** 。在切换到 Ubuntu 作为我的主要系统后，我对 Unix 环境非常熟悉。这使得切换到 Docker 容易了 10 倍。一旦我理解了不同类型的命令，比如 RUN、CMD 和 ADD，编写 Dockerfiles 就变得轻而易举了。这使得与其他人一起开发变得更加容易，但也使得部署您的应用程序变得非常容易。

我知道我会继续学习新的东西，因为我一直在工作，我很高兴看到未来会带来什么。
