#### CSS&SCS

##### [用 CSS 网格简洁的媒体查询](https://robots.thoughtbot.com/concise-media-queries-with-css-grid)

简洁明了，清楚地表明 CSS Grid 对于创建响应式布局是多么大的进步。创建模板区域并在一个简单的媒体查询中对它们进行重新排序的能力使我们在响应度方面超越了以前的解决方案。

##### [用 CSS 动画从平凡到非凡](https://medium.marqeta.com/from-ordinary-to-extraordinary-with-css-animations-2aafd1bd9a6d)

我不知道这是否名副其实，但它确实有一些我以前没有见过的有用的技术和一些非常酷的例子。

##### [渐变形状](https://codepen.io/yuanchuan/full/vVRKRQ/)

这让我想起了最近另一个关于用 CSS 制作形状的帖子，但是这个例子是所有可以在背景上创建渐变的形状。超级爽！

##### [位于 TPAC 的 CSS 工作组:CSS 有什么新内容？](https://www.smashingmagazine.com/2018/10/tpac-css-working-group-new/)

对 CSS 的未来感兴趣吗？这篇文章不仅介绍了 CSS 工作组如何工作的幕后情况，还强调了一组即将推出的特性。

#### JavaScript

##### [React 钩子入门](https://scotch.io/tutorials/getting-started-with-react-hooks)

React 挂钩是最近宣布的一个新特性(仍然只在 React 的预发布版本中可用)，它在 JavaScript 世界掀起了一场风暴。推特上有很多传言，尤雨溪已经为 Vue 实现了一个相当于 T1 的概念验证。这篇文章很好地介绍了这些东西是什么以及如何使用它们。

##### [如何在 JavaScript 中处理货币值](https://frontstuff.io/how-to-handle-monetary-values-in-javascript)

一个令人惊讶的棘手的话题；金钱不会像我们任何一种原始类型那样天真地行事。幸运的是，已经有了一个很好的模式来处理它，本文介绍了这个模式和一个实现它的特定库。

##### [理解 JavaScript 中的高阶函数](https://blog.bitsrc.io/understanding-higher-order-functions-in-javascript-75461803bad)

这不仅是一篇关于 JavaScript 的好文章，也是一篇关于函数式编程和高阶函数概念的好文章。

#### 其他牛气

##### [快讯:TLDR](https://www.tldrnewsletter.com/)

这份时事通讯很快成为我每天早上必读的内容，以确保我不会错过任何重大的科技行业新闻。它每天早上很早就发行，所以当我醒来的时候，它总是在那里，并且提供前一天的行业新闻。有足够多的上下文，实际上我通常都懒得去点击下面的文章；只要浏览一下，我就能对重大事件有所了解。

##### [与 UX 一起预防自杀:谷歌搜索案例研究](https://css-tricks.com/preventing-suicide-with-ux-a-case-study-on-google-search/)

我喜欢其中的一些东西。首先，它强调了设计和技术决策永远不应该脱离它们对人类的影响。如果我们更容易找到如何自杀的信息，我们就有责任提供护栏，帮助人们在黑暗时刻抓住他们，并引导他们提供帮助。第二，我喜欢人们更多地考虑具体的精神健康，以及我们在技术上可以做些什么来帮助他们。第三，这里和引用的灵感帖子中的思维过程非常吸引人。
