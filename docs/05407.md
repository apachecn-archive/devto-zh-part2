# 初级 iOS 开发人员可能比他们想象的更有效率

当我们教初学者和初级 iOS 开发人员时，我们经常会观察到他们的低生产率和缺乏成就感的错觉。我们注意到许多新的、积极的程序员坐几个小时，练习、参加课程、看视频和在线搜索他们问题的潜在解决方案。然而，当我们定期要求他们向我们描述他们的感受时，他们通常会做出否定或怀疑的回答，因为他们无法以预期的速度推进项目。这是一种错误的看法，因为我们认为初级 iOS 开发人员比他们想象的更有效率。

不幸的是，很多年轻程序员不明白的是，他们用错误的标准来衡量他们的生产力，比如每天产生的代码行数。例如，假设一个新的开发人员渴望能够构建由成千上万行代码组成的成功产品，他们很自然地会觉得“没有接近他们的目标”对于初学者来说，构建如此复杂的系统可能是一项令人难以置信的艰巨任务。实际上，他们只是在为自己目前的经历设立几乎无法实现的目标。需要多年的实践才能达到精心打造一个干净且可持续的复杂系统的熟练程度。

在 Essential Developer，我们教导学生，成为一名熟练的软件开发人员是一个漫长而复杂的过程。它不仅需要掌握创建系统的技术层面，还需要掌握其社会层面。学会理解人和企业是如何运作的，以及如何有效地合作是至关重要的。有两种技能可以帮助初学者忍耐，即在逆境中坚持和不断学习的能力。因此，我们建议我们的学生，如果他们在一天结束时比开始时更有知识和能力，他们就在正确的道路上，应该停止担心。

我们一直在帮助敬业的开发人员从低薪职位晋升到**高层职位——有时只需几周时间**！为此，我们不断开展并分享关于如何提高你的技能的自由市场研究，同时牢记**同理心、诚信和经济学**。如果你想在事业上更上一层楼，[现在就免费获取我们的最新研究](https://www.essentialdeveloper.com/courses/career-and-market-strategy-for-professional-ios-developers)。
