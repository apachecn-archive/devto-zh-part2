# 阻碍你的电子商务网站成功的 7 个 UX 设计陷阱

> 原文：<https://dev.to/violetdsmith/7-ux-design-pitfalls-that-hinders-your-e-commerce-website-success-4oec>

网站设计已经从仅仅是网站的一个表面元素，发展成为更具功能性的东西，在用户体验中发挥着巨大的作用。这就是为什么网站所有者现在为 UX 设计。然而，并不是所有的网站所有者和他们各自的设计师都能够成功。如果不能设计出一个能够提供良好用户体验的网站，通常会导致网站无法实现其目标。
因此，了解这些错误是什么，以及如何应对它们非常重要。关于这一点，这里有 7 个你应该避免的 UX 设计陷阱，以便[提高你网站的转化率](https://redstagfulfillment.com/ecommerce-conversion-rate/)并帮助你成功:

# 1.未能使用响应式设计

UX 应该牢记的第一个要素是响应性。你的站点应该能够响应任何设备的访问。尤其是现在更大的流量来自手持设备而不是台式机。尽管这是一个重要的 UX 元素，但是很多网站都没有优先考虑提高他们网站的设计响应性——甚至是他们的桌面网站！很多网站仍然很沉重，仍然充斥着不必要的网站元素。所有这些都导致了响应能力的限制。此外，尽管移动设备优化非常重要，但大量网站仍然没有注意到这一呼吁。
话虽如此，请确保您优先考虑提高您网站的响应能力，无论是对您的桌面网站还是移动网站。

# 2.可读性和易读性差

对 UX 来说，另一个重要的因素是你网站的可读性。你的网站会充满文本内容，所以你的文本可读性是最基本的。不管这听起来有多简单，很多网站都没有为他们的网站选择易读的字体。大多数时候，这是选择看似美观的字体的结果。但是，为了美化网站而做的事情最终会破坏网站传达信息的主要目的。考虑到这一点，一定要练习排版的[基本规则](https://creativemarket.com/blog/typography-rules):确保文本可读性。否则，你的内容是无用的，因为它不会被阅读。

# 3.没有保持网站更新

你可能犯的最大错误之一就是没有更新你的网站。这样做会使你的网站暴露于一系列的漏洞和风险。这不仅会影响您的业务，还会影响您的客户。随着所有网络安全漏洞的出现，确保你的网站得到更新，以确保你不会成为受害者。这是有助于提高安全性的基本步骤。更新您的网站将有助于配备最新的安全措施，使网络罪犯更难渗透到您的网站。当然，网站安全不仅仅是更新你的网站。然而，这是一个非常大的帮助，所以请确保您定期检查您的网站是否运行在最新版本。

# 4.禁用移动网站缩放

使用移动设备真的很有挑战性。有很多问题需要解决，例如如何在手机硬件有限的情况下让你的网站快速加载，以及如何为移动用户提供完整的用户体验，因为只有很少的数字空间可以做到这一点。还有一个挑战是，有些人可能很难看到屏幕。能够变焦应该是答案。然而，有些网站并没有为他们的移动网站提供缩放功能。这给用户带来了真正令人失望的体验，尤其是当面对需要近距离观看的物品时。这样，确保你的网站没有禁用缩放功能。确保您的用户能够根据需要进行缩放。

# 5.处理破损和优化不佳的页面

一些网站的一个令人沮丧的错误是提供链接，将用户导向错误页面。这导致了用户的大量负面反应。另一个缺点是无法优化网站，无法将流量导向有价值的页面。这些错误会降低你的网站转换率，从而影响销售和失去潜在客户。确保您的网站没有断开的链接，并且您的页面得到优化。

# 6.不友好的屏幕分辨率

也有一些设计师没有考虑到他们的屏幕分辨率会如何影响他们网站的整体吸引力，以及它与各种设备的兼容性。虽然这是一个基本知识，世界上大多数计算机使用 1024 x 768 作为分辨率，不是所有的网站都设计了相同的分辨率，但请确保你这样做。这使得你可以兼容几乎所有的电脑。

# 7.不关注分析

许多网站所有者犯的最后一个错误是没有注意分析。分析帮助企业做出了比以往更好的决策，而投资于能够帮助您更好地理解数据的工具是您可以做出的最佳决策之一。不这样做，或者不听你的工具所说的，仅仅是因为它违背了你自己的信念，这是你可能犯的最大的错误之一。

因此，一定要在分析上投资，留意数字所说的。为了帮助你做到这一点，这里有一个关于电子商务分析的[指南](https://www.shopify.com/guides/ecommerce-analytics)来帮助你找出你需要关注哪些指标来建立和发展你的在线业务。

# 结论

作为人类，犯错是家常便饭。然而，如果我们不从自己的错误和别人的错误中吸取教训，我们将永远无法做正确的事情。以上 7 个 UX 设计陷阱是网站所有者和设计者最常犯的错误，这些错误导致他们无法成功。所以一定要从这些陷阱中吸取教训，这样你的网站才能不像他们的网站那样成功。
