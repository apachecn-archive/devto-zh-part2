## [](#background)背景

直到我进入高中，我对编程的兴趣才开始发展，在那之前。我讨厌它。大约一年前，我的朋友 Oskar 向我介绍了 web 开发，我开始喜欢上了它。我从未成为职业选手，但我设法创建了一些网站，即我自己的网站。所以，我第一次接触代码是在一年前。

## [](#related-experience)相关经历

在我进入开发行业之前，我是一名网站开发人员，但是过了一段时间后，我厌倦了，想换个工作。所以我从 App store 下载了 Xcode 并启动了游戏。过了一段时间，我开始适应斯威夫特了。我相信你永远不知道你什么时候准备好了，你只需要出手，所以我在夏天开始申请实习，尽管我已经在 Swift 编码了大约 1.5 个月(我知道这很疯狂)。

## [](#i-took-the-shot)我拍的

我申请了很多公司，多到我都不记得了。我的收件箱里塞满了电子邮件，内容包括:“抱歉，我们现在无法接收你”、“这里没有空位”、“你需要更多经验”等等。但我从未放弃，我继续寻找和申请，直到有一家初创公司对我感兴趣并想与我见面。我的第一个想法是:“我他妈的做了什么？!"，但我表现得很酷，继续他们的会议。

他们喜欢我(我想)，并决定给我发一个测试。测试与他们正在做的事情有关，如果我通过了测试，我将有很好的先决条件获得这份工作。我不会说太多关于测试的细节，但我想撕掉我的头发，因为我不知道我在做什么。但是，我没有放弃，我继续工作和学习，因为我太想要这份工作了。

本杰明·富兰克林:“只要你下定决心，你就能做任何事情”。

我通过了考试，得到了这份工作。我拿到了协议，签了字，然后“噗”的一声，我被聘为开发人员。我从看视频和阅读有关软件开发的书籍开始，利用自学的知识找到了一份开发人员的工作。我非常自豪，这种感觉太棒了。

## [](#how-i-did-it)我是怎么做到的

#### [](#step-1%E2%80%8A%E2%80%8Ateaching-myself-software-development)第一步——自学软件开发

第一个也是最容易忘记的部分是自学软件开发。我实际上是从购买 Udemy 的课程开始的。我看了一些视频，但我想要一种不同的学习资源，所以我开始用 Swift 阅读《黑客》。这是一本非常棒的书，里面有很多小项目，为你提供了软件开发的基础。但是过了一段时间后，我意识到最有效的学习方法是实践。尝试不同的东西，制作小应用程序，然后继续做下去！

#### [](#step-2-showcasing-myself)第二步——展示自己

我开始将我的项目上传到我的 Github 个人资料中，这样我就有地方展示它们，并表明我是一名积极的开发人员。我也开始建立我的 LinkedIn 个人资料，这样我会看起来更专业。

#### [](#step-3-introducing-myself-to-the-company)第三步——向公司介绍自己

当我开始对软件开发感到满意时，我开始接触我所在领域的公司。起初，一切都感到绝望，没有人回复我的电子邮件，那些抛弃我的人。但是过了一段时间，有一家创业公司回复了，想和我见面。

#### [](#step-4-the-interview)第四步——面试

这个过程中最后的，最可怕的，也是最重要的部分。我看了无数个小时如何搞定面试，如何表现得很酷，但都没用。在这里，你可以谈论你开发的应用程序，你拥有的技能，以及为什么想成为一名软件开发人员。如果面试进展顺利，他们可能会给你发一份他们希望你通过的测试，如果你通过了，你就安全到家了。

## [](#last-remarks)最后备注

我希望你对如何让你的编码生涯更上一层楼有所了解，并且你至少发现了一件有价值的事情！
