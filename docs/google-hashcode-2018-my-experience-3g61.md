# Google HashCode 2018:我的经历

> 原文：<https://dev.to/sara/google-hashcode-2018-my-experience-3g61>

我想在这篇文章中分享我第一次参与 Google HashCode 的经历。我向每一个热爱挑战的人推荐这种体验。虽然很难赢，但参加比赛真的很刺激！

这是我的第一次大型黑客马拉松，我真的很兴奋。我大学里的谷歌开发小组组织了一个中心，我和我的队友决定加入。我们人太多了，网络连接有些问题，但不知何故我们还是设法得到了问题文本。

我的团队由 4 个人组成，包括我自己，我们将使用一个在线平台一起工作。然而，由于这个地方的网络连接不好，我们不得不离线工作。所以首先我们讨论了如何解决这个问题。当我们有了算法的设计，我们决定将工作分成 2 组，每组解决问题的一部分。此外，我们决定使用 Java 作为解决问题的语言，因为我们都有很好的技能。

不过，用 Java 解决这个问题可能不是最好的主意。当我们试图混合我们一直在工作的代码时，我们遇到了一些意想不到的问题。糟糕的网络连接一点帮助都没有。

解决完问题，我们就没时间了，所以没时间改进算法。黑客马拉松持续了 4 个小时，但看起来没那么长。

当然，我们没有赢，但是我度过了一段非常美好的时光，现在我对 2019 年的谷歌哈希码有了更多的准备！
