# 反应本机 PinView

> 原文：<https://dev.to/talut/react-native-pinview-482d>

简单，方便，快速形成的组件。IOS 和 Android 都运行流畅，只有键盘和输入。

使用 *v2.0.1* 时，react-native-pin-view 不会验证用户输入的 pin。PinView 组件将只返回输入的值。
