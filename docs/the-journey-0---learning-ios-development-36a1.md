动机——为什么开发 iOS

每天都有新的应用程序进入应用商店。这款应用以自己的方式影响着用户的生活。无论这些应用程序是让你和家人一起欢笑，还是提高你的工作效率，它们都会让你以不同的方式生活。开发人员有机会真正影响其他人的生活。

用技术改善他人的生活应该是开发新工具供每个人使用的目的。应用程序是给人们提供这些工具的最好方式之一。

[https://www.youtube.com/embed/ckjkz8zuMMs](https://www.youtube.com/embed/ckjkz8zuMMs)

此外，应用程序开发是我很久以来一直感兴趣的事情。如果你停下来想一想，在应用程序开发中，你有机会以两种完全不同的方式进行创新。您获得了设计用户界面和编写使其变得真实的代码的机会。如果最终一切如你所愿，你可以发布你的应用程序，让世界各地的人使用它。

做你爱做的事，全身心的投入，然后就是开心。蒂姆·库克

这都是关于旅程的…

从零开始很难，并且会想:“天哪…我在做什么？”“我怎么能学会这些？”“我永远也不会像……”总是出现。

但问题是。每个人都从零开始。有些人比其他人得到更多的帮助。有的更幸运！当然…生活就是这样。生活不是一场赛跑，也不是一场棋盘游戏，我们都要从零开始。对不起，我们有责任创造自己的幸运时刻，改善我们的现状。
