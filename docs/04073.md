# 用 Javascript 实现队列数据结构

> 原文:[https://dev . to/the outlander/implementing-the-queue-data-structure-in-JavaScript-21pa](https://dev.to/theoutlander/implementing-the-queue-data-structure-in-javascript-21pa)

在这个视频中，我们用 Javascript 实现了队列数据结构。虽然 JavaScript 数组可以用作队列，但这个视频是出于教育目的，因为我们将在未来学习更复杂的数据结构。

队列是一种简单的线性数据结构，它是根据先进先出原则插入和移除的对象的容器。元素可以从后面入队，只能从前面出队。
