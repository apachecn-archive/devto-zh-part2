工作室是我在极客世界中最喜欢的活动。你坐在那里学习如何使用一项技术，通常会有免费的东西，如果你遇到困难，会有一位导师来帮助你，他们没有黑客马拉松固有的竞争性质，因为我们只是来学习的，伙计，放松！

# 车间 0。去柏林的女人-比尔.肯尼迪的终极围棋工作坊。

我实际上不打算提到这个，因为我从这个事件中逃离。前端可能会非常令人沮丧:调整像素以取悦设计师，不得不每两周学习一个新的框架，因为你正在使用的框架已经不再热门，不得不为大量设备进行调整，等等。所以我想，让我们在这里做一些真正的工程，让我们深入到后端开发，我发现了一个由 Hello Fresh 和 Bill Kennedy 的出色团队组织的 [Go 语言研讨会](https://www.eventbrite.com/e/women-who-go-berlin-bill-kennedys-ultimate-go-workshop-tickets-42419698460#)。这是一个整个周末的活动...我在那里呆了 4 个小时，很快意识到周末我需要睡觉和恢复。我在那里的时候，我可以看到这是一个非常好和详细的研讨会，但对于那些已经熟悉这门语言的人来说。

(嗯，你明白我为什么从 0 开始数了吧)。

另外两个工作坊我一直参加到最后。我甚至可以一辈子呆在那里，再也不回去做别的事情。这些工作坊棒极了！

# 车间 1。物联网和 MicroPython 研讨会

这是我参加过的许多活动中的一个，这些活动是由神奇、令人敬畏、令人惊奇、令人惊叹的柏林女性科技创客(T1)和 T2 皮拉迪斯柏林(T3)组织的。

我不是 Python 开发人员，但我非常兴奋他们提供了一个关于如何对一个叫做 Micropython 的(有点)新的板进行编程的研讨会。我提前买了我的冲浪板，它有点贵，我总共付了 32 英镑，但是你可以申请免费冲浪板，声称你没有那么多钱。我工作很多，所以我不想在柏林过得很痛苦，而且我自己付了钱，工作室已经是免费的了。

众所周知，与 Arduino 相比，由于 Python 是一种高级编程语言，所以该板不会那么快。但是该板的优势在于 Python 对初学者的友好性，所以这对于教育和原型制作来说是一个很好的选择。我的意思是，超高速被高估了，无论如何，我们都应该更加耐心和冷静。

当我们到达那里时，我们收到了一包赠品，包括一个触摸液晶显示屏，两个带可连接轮子的连续微型服务器，一个湿度-温度传感器，一个 LED 和一些我不记得的东西。

我们研究了一些包含在官方 Micropython 文档中的例子。

github 页面上有专门为本次研讨会制作的教程，包括在不同设置中使用 LCD 屏幕，包括显示从湿度/温度传感器接收的信息。

# 车间二。从电子垃圾到声音

这个研讨会有点不同，前一个研讨会有点性别焦点，有很多赞助商，我们被宠坏了，得到了很多东西，另一方面，这里的意图是用最便宜的，甚至不是最便宜的，但免费的电子产品来建造一个电子设备:电子垃圾！我们身边都有很多这样的东西。

这是在 Common Ground，一个位于 Neukoelln 时尚中心的小商店/小合成器制造商(我说的是你，Schillerkiez)。

我们的任务是用从垃圾中回收的零件制作一个带有放大器的接触式麦克风。这就是我们所做的！！！！

看一看。

[![alt text](../Images/11c579eaedd138dd3328475fa828f6fd.png "My contact mic")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--swJh_53F--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ejvk2y590vam0vhzy7jk.jpg)

压电是接触面，如果你把白色陶瓷面放在一个表面上，然后让它振动，你可以听到通过扬声器放大的声音。这个扬声器是坐在我旁边的一个非常好的人好心给我的，他做了作业，带了一整袋的电子老歌。

我们从一些已经在车间的电路中取出的电容器。我们不得不将它们解冻，并确保它们不会在加工过程中受损。伙计，这太难了。我们还必须提高焊接技能，因为电路板非常小。我对我的焊接技术非常自豪，看看这种美:
