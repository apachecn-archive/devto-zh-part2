# 如何创建 dev.to 帐户

> 原文:[https://dev.to/programmingmonky/devto-25lm](https://dev.to/programmingmonky/devto--25lm)

如果您有 twitter 或 github 帐户，可以很容易地创建 dev.to 帐户。
本文解说如何利用 Github 的 OAuth 创建 dev.to 的账户。

# 创建 github 的账户

在 dev.to 上，可以通过 twitter 和 github 认证创建帐户。
本文用 github 的账户来说明制作，
还没有 github 账户的人请参考这篇文章

[https://dev.to/programmingmonky/github-309h](https://dev.to/programmingmonky/github-309h)

请先创建 github 帐户。

# 访问 dev.to 的账户制作页面

在登录了 github 账户的状态下，在
dev.to 的账户制作页面

[https://dev.to/enter](https://dev.to/enter)

访问，

[![アカウント作成ページ](../Images/0966592b8fc1bb1f4559d185d7b0ac3e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--YQ2uv0kH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ensschmfwyjr0kex920w.PNG)

请按“Auth With Github”按钮

然后转移到 github 的网站，显示用 github 的账户进行认证的确认画面。

[![githubの認証ページ](../Images/ba997ba89488a726fb13135bca5bafbd.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--jQFvleXH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/owxz75tp8pwut5a41m88.PNG)

确认画面后按绿色的" Authorize thepractivaldev "按钮吧

# 账号创建完成

在 github 上按下绿色的“Authorize thepractivaldev”按钮后，画面转移，显示
dev.to 的你的个人页面。

[![あなたの個人ページ](../Images/3163d6a381577c14d92e8d8d84bb90c6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--XsxBfB9U--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/mcp89ce9otxjzifet26c.PNG)

这样就完成了帐户创建。
