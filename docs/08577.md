# 什么时候给开源库一颗星？

> 原文:[https://dev . to/mustafaturan/when-to-give-a-star-a-open-source library-11 a4](https://dev.to/mustafaturan/when-to-give-a-star-to-an-open-sourcelibrary-11a4)

给开源库一颗星对库的开发者和消费者来说意义重大。因为这将给开发者带来积极的影响，使库变得更好。这绝对是一种委婉地告诉开发者，有人在使用这个库的方式。

### 作为图书馆的消费者

如果您使用的是开源库，请找到它的 Github/Gitlab(或任何版本控制源),并给它打个星号。如果你需要使用一个高质量的泛型库，你应该帮助把新人带到这个库中来。主演将给图书馆带来质量和活力。你带来的人中的一个，可能是图书馆的下一个贡献者。

### 支持工作或想法

您会发现开源库有以下好处:

*   喜欢这个主意
*   计划在你当前或下一个项目中使用它
*   比如图书馆的软件设计
*   感兴趣的主题，如分布式系统、数据结构、有用的链接

别忘了，任何正面的主演活动，都会以社会影响带来更多人喜欢图书馆。这种积极的影响也会给图书馆带来更多的贡献者。并且将防止库过时。

这些只是我在主演图书馆时的想法，请不要犹豫分享你的。我很乐意在故事中包括你的名字。
