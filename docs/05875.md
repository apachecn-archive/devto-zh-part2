在 CS 学校，我们通过用 C 语言重新发明轮子来做低级的东西(必须重写 malloc，printf...)甚至有点组装。然后我们用 C++和 Java 进入 OOP。最后，我为期末学校项目做了一些 Swift(必须创建一个 iOS 应用程序)。

今天，我主要用 Ruby 编程(我以 Ruby on Rails 开发人员为生)，但我对 Elixir 有着浓厚的兴趣(真是令人惊讶)。
