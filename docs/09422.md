## [Magento 2.2.5](https://www.tigren.com/magento-2-2-5/) 的重要更新

*   关闭存储的 XSS、SQL 注入和跨站点请求伪造(CSRF)漏洞。
*   在你的网站升级到 Magento 2.2.4 后，通过在默认商店视图中应用相同的配置来解决多网站的部署问题。
*   将数据索引过程的性能提高了近 15%。
*   提高核心捆绑扩展的性能。
*   增强 Magento 的核心代码。
*   通过允许客户在订单确认页面上注册新帐户来改进结账流程
*   更新产品分类功能，根据销售前的实际价格对所有商品进行分类。
*   添加与 Magento shipping 相关的新功能:为退货选择承运人、批量发货、收集点
*   将目录搜索结果页面的性能提高 30%以上

## Magento 2.2.5 的显著错误修复

*   不可能多次运行 cron 作业
*   轻易放弃捆绑产品的任何选项
*   应用优惠券时摆脱捆绑产品
*   同时运行目录搜索全文索引器和目录产品索引器
*   应用新的数据格式(dd/mm/yyyy)使用区域设置保存产品
*   显示心愿单上可配置产品的选定选项的图像
*   根据“我的帐户”页面上的商店正确过滤最近的订单
*   在多家商店获取准确的 PayPal Payflow Pro 凭证
*   在搜索结果中隐藏可配置产品的脱销选项
*   刷新常用搜索结果的缓存以缩短响应时间
*   通过使用 JavaScript mixins 改进样本功能，与浏览器无关
*   轻松从默认商店的目录页面导航到不同商店的目录页面
*   在订单汇总部分添加准确的税额(在订单汇总上
*   在结账页面显示含税/不含税价格
*   一旦订单取消，允许客户创建 Vertex 发票退款
*   处理一些与 Amazon Pay、Vertex 和 dotmailer 相关的小问题
