# 然而，卡瑞娜编码了

> 原文:[https://dev.to/karinaislam/nevertheless-karina-coded-kfk](https://dev.to/karinaislam/nevertheless-karina-coded--kfk)

## [](#i-begancontinue-to-code-because)我开始/继续编码是因为...

我已经完成了工业工程的本科学业。然后我在 2016 年搬到了美国。搬到这里后，我想开始一个全新的人生旅程，但不知道如何开始。有一天，我在 YouTube 上看到一个补充，一些人分享他们的故事，他们如何加入一个编码训练营，并在完成后找到了一份工作。这让我很兴奋，但是我没有足够的钱去参加新兵训练营。于是，我搜索了他们的*课程——课程表*并开始按照它学习。我作为程序员的故事就是这样开始的。

## [](#i-continue-to-code-because)我继续编码是因为...

我喜欢学习、创造和解决问题的能力。但这对我来说并不容易。从头开始自学代码真的很难。一直保持动力并坚持学习是很难的。我被困住了，几次失去动力。为了克服这些困难，我参加了许多会议、聚会和研习班，以保持我的状态和动力。这需要极大的耐心和努力。

## [](#i-recently-overcame)我最近克服了...

我对公开演讲的恐惧。我的母语不是英语，所以在公共场合说话总是犹豫不决。但是我总是试着和别人分享我的学习。所以，最近我在 React meetup Chicago 上做了第一次关于 React 的演讲。此外，我被选为将于四月中旬举行的 JavaScript 会议的发言人。

## [](#im-currently-hacking-on)我目前正在入侵...

使用后端的 **Django** 和前端的 **ReactJs** 构建一个 web 应用程序。还试图使用 **Python** 制作一款用于机器学习和增强现实的安卓应用。并试图找到一份 Jr.web 开发人员的工作(我非常兴奋地[与那些有 Jr .开发人员职位空缺的人开始对话](https://www.linkedin.com/in/karinaislam/))

## [](#my-request-to-all-seasoned-developer-programmer-both-male-and-female)我对所有经验丰富的开发人员/程序员(男性和女性)的要求.....

请帮助我们成长和学习更多。我们想向你们学习，因为我们知道你们比我们更了解。这就是我们向你求助的原因。

## [](#my-advice-for-other-women-who-code-is)我对其他代码女性的建议是...

在你的舒适区之外做事，每天保持学习和成长。

## [](#finally)终于...

我想对所有为我们制作教程、博客并让它们免费的人说声谢谢。然后感谢那些回答了所有关于 slack、堆栈溢出和其他地方的愚蠢和复杂问题的人。感谢会议、聚会和研讨会的主办者和组织者，他们不断帮助我们学习和不断成长。最后，我想对我们社区中一直支持我们的所有人说声谢谢。没有这些支持，这是不可能的。
