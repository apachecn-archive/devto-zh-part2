# Fullstack developer 是一个骗局术语

> 原文：<https://dev.to/entrptaher/fullstack-developer-is-a-scam-term-3bl2>

[![Fullstack developer is a scam term](../Images/0ad568a71cde07848b618c47e214dd17.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--5OsLLYbR--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/3geyemhqaupfk4tlt2h7.png)

你一个人，被聘为后端开发人员。你专攻 PHP、Python、C#、C++等等。没有一个是和前端相关的。你在你的位置上摇摆不定。但是有一天你也被要求做前台。所以你学了一点 jQuery。以最糟糕的方式做前台，因为那不是你的领域。几年后，您已经是一名全栈开发人员了。

你们两个被聘为前端开发人员。你专攻你的东西。将 PSD 和设计概念转化为令人敬畏的 html 和 css，使一切都具有交互性，所有这些都是你的工作，你在那里发光。但是最近后端开发人员辞职了，他们突然需要有人来修补一些东西。因为你是他们唯一一个被认为是专家的开发人员，所以你最终会学习一些后端技术并修补一些东西。几年后，您已经是一名全栈开发人员了。

不管是第一个人还是第二个人，故事的结局都是一样的。公司希望你能补上缺失的部分。你不再专注于你的创造性部分，你在做两个独立的任务。我们甚至没有提到管理服务器、测试和部署的可怕之处。

回到正题。全栈开发者是一个骗局术语。如果你比较薪水和你要处理的任务，你最终会得到和现在一样甚至更少的收入。

[![](../Images/68ce5e5e0ec5d6da73d5a80a6ec868b8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--mxSQxyVr--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://i.imgur.com/XkhuZDq.png)

根据 paysa 的数据，后端开发者挣 9 万美元，前端开发者挣 8 万 8 千美元，全栈开发者挣 7 万 6 千美元。这是平均值，但是 75%的数据更加可怕。后端、前端和全栈分别为 104，000 美元、106，000 美元和 90，000 美元。

这在像 glassdoor 这样的薪水数据库网站上会有所不同，但最终结果是一样的。

太可怕了。你现在是个万事通。您需要处理 react、express、redux、docker、ci/cl、aws 以及成千上万其他东西。你知道的比别人多，但得到的报酬却比别人少。

这是骗局，这是神话，这是每个人都同意的。最大的骗局是你做得多，得到的报酬却少。即使是在你工作的同一时间，即使是同样的 8 小时工作。你被期望解决他们所有的问题，而薪水却越来越少。

如果他们想让你做前端和后端，那么他们至少应该支付当前金额的 1.5 倍。如果他们想让你做前端，后端，服务器和测试，那么你可以期望至少两倍于你现在的工资，因为你同时处理 4 个部分。但是不，你的报酬是 0.75 倍。

他们使用类似“更灵活”的暗语，这意味着你必须做得更多，熬夜，ofc 将获得更少的报酬，因为你本来就应该灵活。

他们还使用在[开发者招聘上看到的暗语:如何阅读招聘广告](https://dev.to/redfred7/developer-recruitment-how-to-read-job-adverts-1h2c)帖子，“需要全栈开发人员”，“努力工作，尽情玩乐的精神”，“工资有竞争力”，“一家以结果为导向的公司”，“多任务能力”。说真的，这世界怎么了？

他们会说这样的话，

*   好吧，错误日志没什么关系。
*   我们可以事后测试。
*   顾客会替我们测试。
*   这只是一个 mvp。...诸如此类。

你看到最终产品的样子了吗？它没有任何单元测试，是坏的、坏的、易变的。代码非常混乱，因为你要同时关注多件事情，而你只是一个人。

如果你是一名后端开发人员，并且不想一头扎进前端，最好还是保持清醒，而不是变成这样的奴隶。前端也一样。你可以看到这些家伙是如何为了他们自己的利益从每个人身上骗到最好的。他们通过收取巨额资金和提供不完善的服务来欺骗客户，他们通过支付低于他们应得的价格来欺骗开发者，他们是有毒的，他们正在摧毁网络产业。

回答我这个，你开始编程是因为爱好所以你可以成为这种人的奴隶吗？

# 然而...

全栈开发者有他们的存在和需求。他们可以成为团队前端和后端开发人员之间的桥梁。这意味着团队必须有后端和前端开发人员，并有更好的理解，他们应该聘请全栈开发人员。他们以很多名字存在，更好的名字，其中一个叫做**项目经理**。

一个典型的 Web 开发项目经理在 avg 上挣 6 万美元，高级 Web 开发项目经理挣 13.7 万美元，普通项目经理挣 10.5 万美元。要说的是，初级 web 开发人员项目经理的待遇比全栈开发人员差。但是，高级的比高级前端和后端开发人员挣得多也是事实。因此，高年级学生艰难地学习，努力达到顶峰，而低年级学生仍然像垃圾一样被善待，直到他们意识到自己的真正价值。

每个硬币都有三面。头，尾巴和几乎不存在的薄边。全栈时代已经完成了它的工作。是时候做点别的了，更闪亮的东西...也许是机器学习？

编辑注:我自己是一个全栈开发者，我热爱编程，我喜欢前端和后端，我也喜欢 devops。考虑到所有这些，这篇文章对许多人来说似乎是矛盾的。它缺乏很多细节，但我很肯定有很多人会同意我的说法，为了一些三流的人，这些术语是如何被滥用的。最高层可能状态很好，薪水也很高，但是其他层才是被欺骗的。
