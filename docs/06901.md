# 手里剑，放出忍者之星的 slack bot(又名手里剑！)

> 原文:[https://dev . to/ahmedmusallam/shuriken-the-slack-bot-the-give-out-ninja-stars-aka-shurikens-2 kik](https://dev.to/ahmedmusallam/shuriken-the-slack-bot-that-gives-out-ninja-stars-aka-shurikens-2kik)

你可能在 slack 上用过 [plusPlus](https://plusplus.chat) 。你可能和我一样喜欢它！这非常简单，它帮助我们的团队变得更加互动，并促进了健康的竞争。它非常容易使用，如果有人帮了你或者你喜欢他们分享的东西，你输入`@thatCoolSlacker ++`他们就会得到一分，如果你没有，你输入`@thatNotSoCoolSlacker --`他们就会失去一分。你也可以这样做:`@Beer ++`或`@Work --`。

在过去的几个月里，plusPlus 有时变得没有反应。它有时有效，但大多数时候无效。我联系了 plusPlus 团队，询问发生了什么事情，我得到的回复是他们正在解决这个问题，希望很快就能修复。所以我想，为什么不实际构建一个克隆，并在这个过程中了解 slack 机器人！

[**我就是这样做的！**](https://github.com/ahmed-musallam/shuriken)
**介绍 [**手里剑**](https://github.com/ahmed-musallam/shuriken) 给你忍者星辰的 bot！**

现在，这是我的一个早期项目，目前支持每个工作区一个机器人实例。而且还得自己主持。但是我学到了很多，想和社区分享一下。

去看看！阅读文档，如果你愿意，可以投稿！也许在某个时候我会找到时间让它成为真正的产品！

此外，我需要机器人在看到`++`或`--`时应该发送的响应建议。
