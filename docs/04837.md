# 简单明了

> 原文:[https://dev.to/jwhart175/plainsight-4k1d](https://dev.to/jwhart175/plainsight-4k1d)

plain_sight 是一个工具，用于创建简单的密码和掩码来隐藏文本数据。使用可定制的规则文件，用户可以将任何文本文件(包括 html)转换成看起来像是数字数据、日志文件，甚至是有效的 html 文件，或者他们能想到的任何东西。
