## [](#before-you-commit)提交前

在提交代码之前，请确保:

*   README.md 已更新
*   一个成功的构建和 jar 部署在 nexus 上
*   所有新实现的 API 都有文档记录
*   所有新添加的属性都记录在 additional-spring-configuration-metadata . JSON 中
