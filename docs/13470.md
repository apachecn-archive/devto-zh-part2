# 用 Golang 写一个 GIF 播放器

> 原文:[https://dev.to/ntoooop/write-a-gif-player-with-golang-1350](https://dev.to/ntoooop/write-a-gif-player-with-golang-1350)

我刚刚用 golang 写了一个 gif 播放器(不到 200 行代码)。我经常使用 MAC，但它不能像 windows 一样播放 gif，所以这个项目应运而生。

这是一个 Golang 项目，也是一个 korok 项目。核心代码很少:

```
func main() {
    gfile := flag.Arg(0)
    // decode !
    gp, err := Load(gfile)
    if err != nil {
        log.Fatal(err)
    }
    w, h := gp.Size()
    options := korok.Options{
        Title: "Gif Player",
        Width:w,
        Height:h,
    }
    korok.Run(&options, &MainScene{p: gp})
} 
```
