# 用全新的 dbForge Documenter for MySQL 解决 MySQL 和 MariaDB 文档的麻烦

> 原文：<https://dev.to/devartsoftware/take-the-hassle-out-of-mysql-and-mariadb-documenting-with-all-new-dbforge-documenter-for-mysql-2ld2>

数据库工具和数据连接解决方案的捷克软件供应商 Devart 发布了 dbForge for MySQL 产品线的全新产品——db forge Documenter for MySQL。这个方便的工具自动为多个 MySQL 和 MariaDB 数据库生成 HTML、PDF 和 MARKDOWN 文件格式的文档。此外，dbForge Documenter for MySQL 允许在一组丰富的选项和设置的帮助下定制生成的文档。该工具的主要功能包括以下特性:

1)从 A 到 Z 的数据库结构

db forge Documenter for MySQL 提取关于所有 MySQL 对象的大量数据库信息，包括它们的详细信息、属性、SQL 脚本和对象间依赖关系。

2)丰富的定制选项
Documenter 支持选择要记录的单个数据库对象及其属性。该工具还包含一组丰富的样式模板和主题，用于微调文档布局。

3)支持多种文件格式
dbForge Documenter for MySQL 可以生成 HTML、PDF、MARKDOWN 文件格式的文档。此外，以任何格式生成的文档文件都是可搜索的。这使得在大文档中导航变得更快更容易。

4)对象注释
在注释属性的支持下，用户可以给每个 MySQL 对象添加注释。

5)浏览文档文件
Documenter for MySQL 允许用户在生成的文档中快速搜索数据库对象。例如，该工具突出显示与用户正在寻找的文本相匹配的所有项目。此外，Documenter 通过内置的面包屑在整个文档中提供了方便的导航。

6)自动数据库文档
在命令行界面的支持下，dbForge Documenter for MySQL 允许设置自动数据库文档。此外，该工具还包括创建命令行执行文件的选项，只需单击一下就可以运行常规的数据库文档任务。
