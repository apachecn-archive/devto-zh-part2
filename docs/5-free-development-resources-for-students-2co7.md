## 对学生免费

如果你是大学生，你可以免费获得一些非常好的工具，让你开发和探索 CS 世界。

通常，要获得这些工具，你需要一个官方的机构电子邮件地址。不过你也可以提供你的学生证或其他证明。

这里有几个！！！

### 1。JetBrains IDEs

在我看来，这些是目前市场上最好的想法。他们有令人难以置信的用户界面和功能，他们实际上让我更喜欢我的学校作业。

诸如用于基于 **Java** 的应用程序的 [IntelliJ IDEA Ultimate](https://www.jetbrains.com/idea/) 、用于 **PHP** 的 [PhpStorm](https://www.jetbrains.com/phpstorm/) 、 **Python** 的 [PyCharm](https://www.jetbrains.com/pycharm/) 以及我个人最喜欢的工具，我最近开始使用这些工具，并且无法获得足够的 [CLion](https://www.jetbrains.com/clion/) 用于 **C/C++** 。

[![JetBrains IDEs](../Images/4162ee44272852f17f682dc9df5969a2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--r28Ykjft--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/tyjsu8upqo26uw9o72zk.png)

他们还有一个非常有趣的 ide 插件(仍在测试阶段),可以帮助你通过 IDE 内部的练习来学习你选择的语言。

### 2。教育 GitLab

GitLab 正在成为伟大项目的协作和开发的伟大工具。他们目前的重点是简化 DevOps 生命周期。一旦你进入就业市场，这是开始为你的未来获得 DevOps 心态的好方法。

> GitLab 是整个 DevOps 生命周期的第一个单一应用程序。只有 GitLab 支持并发开发，将组织从工具链的约束中解放出来。

[GitLab for education](https://about.gitlab.com/solutions/education/) 允许您的教育机构免费使用 [GitLab Ultimate 和 Gold](https://about.gitlab.com/pricing) 。

不幸的是，只有机构可以申请(学生不能直接申请)，但是如果你将项目开源或为私人项目提供免费功能，你仍然可以将项目托管在 GitLab 上，并拥有完整的功能。

### 3。GitHub 教育

[GitHub Education](https://education.github.com/students) 为学生免费提供了一套很棒的开发工具。从像 [Atom](https://atom.io/) 这样的文本编辑器到像 [GitKraken](https://www.gitkraken.com/) 这样的 Git 图形工具，再到提供 CI、电子邮件服务、主机托管等等的云服务。

你需要申请的只是填写一张表格，解释你打算如何使用 GitHub。请注意，如果你使用的是机构邮箱，你被录取的几率会更高。

如果您对移动开发感兴趣，fastlane 一定会引起您的兴趣。浪子让你可以自动化许多与移动应用程序的开发和发布过程相关的繁琐任务。

这是一个与谷歌 Firebase 团队一起开发的开源平台。

> 构建和发布移动应用的最简单方式。fastlane 处理繁琐的任务，所以你不必去做
