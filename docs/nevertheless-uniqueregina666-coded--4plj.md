## 我开始/继续编码是因为...

我参加了罗格斯大学的编码训练营，因为我觉得这是开始我转行之旅的最好方式。我是一名幼儿教育工作者，拥有幼儿教育本科学位和初等教育硕士学位。我在教室里呆了 10 多年，非常喜欢它！不幸的是，2011 年我被诊断患有残疾，随着时间的推移，我的教育生涯受到了极大的影响。医生和我决定，是改变的时候了。我需要重新开始，这一次我必须确保它不仅不会影响我的身体，而且会对我的精神产生积极的影响。从高中开始，我就“注定”要成为一名教育工作者，从来没有真正考虑过其他职业道路。所以，我想到了过去我喜欢做的所有事情。我真的很喜欢和技术打交道，尤其是网络开发。在“AOL 和 Myspace 热潮”期间，我花了很多空闲时间为我的个人资料和其他资料编写 HTML 和 CSS 代码。这是我引以为豪的事情，我做了一些东西。当然，这需要大量的复制和粘贴，但我以此为生。当我编写代码时，我感到了挑战，当我完成一些代码并点击刷新按钮时，我感到了一股幸福感。哇！成功了！快进到现在，Myspace 早已不复存在，但 HTML 和 CSS 仍然存在，每当我点击刷新按钮时，那种幸福感仍然会出现。编程不仅给了我做自己喜欢的事情的机会，还让我可以克服自己的残疾。我可以支配自己的时间，从事我喜欢的项目，或者在技术社区产生影响。最后，也是最重要的一点，编程让我成为社会的一员，并为社会做出贡献，尽管我有残疾。我成为一名教师，因为我想在我的社区里有所作为。我想回报给我力量让我有今天的系统。那个梦想还活着，我还会有所作为，做别人的榜样。通过学习编码时获得的知识和技能，我可以将其应用到一些新的教育应用中。可能会为一家为如何向学生传授信息设立新标准的公司工作。当你获得正确的技能和知识来处理任何事情时，可能性是无穷无尽的。我在这个领域没有一个榜样，但我保证，在将来，我会成为别人的榜样。
如果你正在寻找改变、挑战或新的开始，无论出于什么原因，这可能就是你要找的！你在这里获得的技能在工作和生活中都是无价的。毅力、纪律、成长心态和友爱是你在学习编码期间将会培养的一些性格特征。你只需要有一个开放的心态，并愿意做出改变。
