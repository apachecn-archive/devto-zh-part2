# 因为你是女人

> 原文:[https://dev.to/melisaim/because-you-are-a-woman-2393](https://dev.to/melisaim/because-you-are-a-woman--2393)

### “[...]，这就是你得到那份工作的原因。”

他告诉我，我得到软件工程实习生的工作是因为我是女性，我的同学(他和我参加了同一个训练营)得到他的工作是因为他很聪明。我非常生气。不是因为他如此不合逻辑和偏执，而是因为这不是真的。我敢打赌，任何读到这篇文章的女人都不会相信。

### “él ya no se puede cuidar solo[他再也不能照顾自己]”

我编码是因为我是某人的女儿。因为我的父亲患有痴呆症，没有资产或很少外部援助。因为当我成功时，当我挣扎时，他的挣扎会减少。他为我们所做的牺牲，不是报偿，而是荣耀。

### [](#there-is-curiosity-that-you-cant-fake)“有装不出来的好奇心”

我编码是因为我是学生。我喜欢学习、探索和觉醒，以新的方式看待这个世界和其中的问题。

### [](#we-are-happy-to-have-you-thanks-for-returning-to-aid-our-tech-divas)“我们很高兴有你！感谢您回来帮助我们的技术天后”

我编码是因为我是一个帮助者。我喜欢和别人分享我学到的东西，提出并鼓励他们。就像我的导师瓦内萨对我说的，“永远不要关上你身后的门。”

### [](#proud-of-you-always-good-to-build-a-thing-to-know-how-it-works-)“为你骄傲！做一个东西知道它如何工作总是好的(= "

我编码是因为我被爱着。每一个鼓励我的朋友，每一个给我力量的导师，帮助我成为一个女人，尽管如此，她编码。

### 我对其他女性和非二进制程序员的建议是...

永远不要让你的奋斗定义你。你不是哭泣的女人，不是受伤或饥饿的女人，不是痛苦的女人...因为这些奋斗，你变得强大，变得不可战胜。
