# 讨论:我们需要“高级”软件工程师这个头衔吗？

> 原文：<https://dev.to/scottshipp/discuss-do-we-need-the-senior-software-engineer-title--4l31>

从我记事起，我就对高级软件工程师这个头衔感到不舒服。在我成为学长之前就开始了，之后继续。我的主要问题是，这个头衔在不同的公司有不同的用法，实际上毫无意义。A 公司的高级软件工程师可能比 b 公司的软件工程师强不了多少，事实上，在我的职业生涯中，我为高级候选人进行过一些面试，结果发现初级候选人在同样的问题上做得更好。

最后，当有人告诉我他们是“高级软件工程师”时，我不确定这意味着什么我当然有一些关于它应该意味着什么的想法，但是在现实生活中它们通常不会被执行。

别人对高级职称怎么看？你觉得有用吗，用在什么方面？你觉得应该是什么意思？你认为我们的整个行业会对“高级”软件工程师的含义达成一致吗？
