# 作为日本人，我是如何自学英语的

> 原文：<https://dev.to/inductor/how-i-learnt-english-by-myself-being-japanese-10h4>

# 关于本帖

我相信有一些人想学习另一种语言，它可以是多种多样的。英语、西班牙语、法语、韩语、中文...是的，多得数不过来。

今天，我想分享我是如何从连“我 19 岁了”这个简单的句子都不会说的水平开始学习英语的。以至于我可以作为一个解释者贡献一个 OSS 社区，虽然不是同时。

我说日语只是因为日语和英语非常不同，我主要想说的是，如果你年轻时不能很好地掌握一门语言，现在尝试还为时不晚。

# 你是谁？

我不是一个伟大的人。只是一个喜欢吃东西和摆弄电脑的日本人。

# 你一定去过好的学院/大学

我甚至没有上过大学。我去了一所科学学院，它就像是一所技术高中和大学的结合。即使在我的国家，这也是一种特殊类型的学校，所以我真的不知道如何描述这一点，但我只是一个懒鬼，因为我辍学了:)

# 那么你是怎么做到的呢？

我不认为方法真的很重要。但是你需要一些这样的心态。

## 有短期目标和长期目标。

当你试图瞄准某物时，你需要一个目标。不劳而获不是很有效率，你应该有一些现实可行的东西。想想你自己将来想用目标语言做什么，如果你想做，你需要不断尝试。

短期目标很重要，让自己不断努力。长期目标有时真的很无聊，因为它需要太多时间。设定可以实现的短期目标会让你在取得成就时感觉更好，这是一个好的迹象，表明你确实在进步。短期目标是“背 100 个单词”、“通过某种程度的考试”和“能用英语说一些愚蠢的笑话”。

长期目标很重要，让自己不要轻易对自己“满意”。如果你想真正擅长某件事，不管别人怎么说你擅长，你都应该继续努力。在接下来的几年里，用你的努力给自己一个惊喜:)

## 知道举行对话和阅读/撰写技术/商务句子的区别

老实说，我最擅长说英语，但我的写作糟糕透顶。我可以用来表达我在写作方面有多糟糕，但不是现在；我可以在 IRC 上写愚蠢的东西，欣赏没有字幕的情景喜剧，对我的澳大利亚女友说蹩脚的笑话，甚至和她大吵一架。即使如此，我也不能像说母语的人一样写出完全“自然”的英语，因为我缺乏词汇和练习。你需要每天练习才能真正精通语言。不管怎样。如果从小不知道这些语言，要成为一个完全的双语者真的很难。

我认识许多能理解英语文章的日本人。但大多数情况下，他们在大脑中翻译成日语，然后慢慢分解，以便理解。他们可以写足够好的英语来表达他们想在电子邮件中说的话，但是不能说很多，包括日常会话，因为他们没有练习过。

在我看来，所有的语言技能；阅读、写作、听力、口语是分开的，但同时又紧密相关。如果你想在总体上做得好，你必须均衡地练习它们。

## 不要太在意你的错误，但要在乎一点。

很多人说“我害怕犯错误。”但是当你想提高的时候，它会拖你的后腿。假自信有时会是一个非常强大的武器，因为你可以假装你对自己说的话非常确定。想象一个正在演讲的人，如果没有自信，他的声音会很低，而且他的喃喃自语你听不到。

但是不要太依赖“假”的自信。这并不意味着你不必努力。你必须知道，当你看起来不像这样的时候，你需要继续努力。

## 享受你正在做的事情

语言只是一种工具。你用它来表达你自己，享受来自其他文化的内容，理解文化和与世界分享想法。这不仅仅是你的目标，更重要的是你想用这个工具做什么。

这就是我这七年来一直在做的学习英语的事情。我没有任何人纠正这一点，所以这纯粹是我的“英语”。如果我写错了什么，我很抱歉。但是我希望每个人都明白这一点，并从这个帖子中学到一些东西。
