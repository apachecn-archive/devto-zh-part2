# 一些 C 游戏(2006 年至 2007 年)

> 原文：<https://dev.to/jalbam/some-c-games-2006-to-2007-2j9e>

以下是我在戴(Ciclo Formativo de Grado Superior en Desarrollo de applicaciones informáticas)的“Programa cióestructurada I modular/Programa ción estructurada y modular”(结构化和模块化编程)科目学习期间制作的一些 C 语言游戏，该科目是一个软件开发副学士学位(类似于英国的基础学位、苏格兰的高等教育文凭或爱尔兰共和国的高级证书)，在西班牙 Reus city 的 IES Baix Camp school(西班牙)。

[![Screenshot](../Images/a63f38ec79338d8bd6a8ca1ee9c858e2.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--p5vjatXj--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://raw.githubusercontent.com/jalbam/c_games/master/screenshot.gif)

都是文字游戏。它包括一个二十一点(又名二十一)纸牌游戏，一个轮盘游戏(有四种不同的玩法)，一个约翰·康威的生命游戏(可以手动或随机插入细胞)，一个智多星游戏，Buscacaminas(带有一些扫雷功能的寻路程序)，Pongetet(一种 pong 游戏)和 Pilotes(只是一个弹跳文本生成器)。使用的 IDE 是 blooding Dev-c++。

他们可以在 GitHub 上找到:[https://github.com/jalbam/c_games](https://github.com/jalbam/c_games)
