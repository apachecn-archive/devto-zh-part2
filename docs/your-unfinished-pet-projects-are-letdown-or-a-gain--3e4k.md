# 你未完成的宠物项目是失望还是收获？

> 原文：<https://dev.to/djviolin/your-unfinished-pet-projects-are-letdown-or-a-gain--3e4k>

你是如何看待他们的？他们给你什么了吗？还是只是浪费时间？

每当我看我的回购，我看到未完成的项目:

我想在 Node.js + Express 中建立我的下一个摄影师作品集，但是我又用了 Wordpress。但作为回报，我学会了 Node.js。

我想创建一个网站，用公共数据列出我的国家的公司。在这个阶段，我是一个未经验证的身份，以获得政府官方数据库的访问权限，所以我在 Go 中建立了一个 web scraper，以便能够开始开发并实际推出一个产品。该项目仍处于这种状态。我从未完成它，但现在我知道如何在 Go 中实现它，如何处理磁盘上的文件和使用标准库。

有哪些让你失望的事，没有给你任何回报，但你学到了新的东西？
