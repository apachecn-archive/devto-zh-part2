# 成为更好的程序员

> 原文:[https://dev . to/erikpischel/being-a-better-programmer-2 hji](https://dev.to/erikpischel/becoming-a-better-programmer-2hji)

亨利·斯坦利写了一篇博客文章“[成为一名出色的程序员](https://recurse.henrystanley.com/post/better/)”。他概述了希望变得更好的程序员可以关注的领域。也就是一边直接学习技能，另一边学习元技能。元技能:学习“刻意练习”——做挑战你的事情并获得反馈——识别你犯的常见错误并学习做深入的工作——专注于工作，不要分心。

我对刻意练习的建议是像 [CodeWars](http://codewars.com) 这样的网站，在那里你可以用几种编程语言解决练习(“katas”)。这很有挑战性(CodeWars 有武术，比如精通程度)，你可以通过查看其他用户的解决方案来获得反馈，从而可以将他们与你的解决方案进行比较。
