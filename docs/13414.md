# MongoDB 世界的包容性

> 原文:[https://dev.to/mongodb/inclusion-at-mongodb-world-bcp](https://dev.to/mongodb/inclusion-at-mongodb-world-bcp)

6 月，我们将启动第五届 [MongoDB World](https://www.mongodb.com/world18?jmp=blog) ，这是我们的全球用户大会。我们将继续努力使这次会议不仅仅是一次有趣和互动的学习经历，而且是一次欢迎所有人参加的活动。方法如下:

#### [](#diversity-scholarship)多样性奖学金

MongoDB 的奖学金计划旨在为技术行业中代表性不足的群体成员提供支持。这包括但不限于黑人、拉丁裔、女性、低收入和 LGBTQ。

**[多元化奖学金获得者领取*](https://docs.google.com/forms/d/e/1FAIpQLSf_u0u_gdvsUDHwj5VyR2fhOYAxUrbYOoEtnpchsdbJeN7aPg/viewform) :**

*   免费进入 MongoDB World
*   免费参加会前研讨会
*   邀请与其他学者共进午餐
*   在活动中与 MongoDB 演讲者进行快速指导
*   适用于开发人员和 DBA 认证考试的 MongoDB 认证凭证
*   六个月的点播 MongoDB 大学课程
*   在线 MongoDB 多样性学者社区的终身会员
*   博客文章中的一个特征

如果你是上述任何一个团体的成员，你就有资格申请奖学金。这里的申请是[这里的](https://docs.google.com/forms/d/e/1FAIpQLSf_u0u_gdvsUDHwj5VyR2fhOYAxUrbYOoEtnpchsdbJeN7aPg/viewform)，申请的截止日期是 5 月 4 日。

<center>[<button>Apply for a Diversity Scholarship</button>](https://docs.google.com/forms/d/e/1FAIpQLSf_u0u_gdvsUDHwj5VyR2fhOYAxUrbYOoEtnpchsdbJeN7aPg/viewform)</center>

**如果你不能或不想申请，但仍想为其他人提供奖学金**，请务必[报名参加 MongoDB World](https://www.eventbrite.com/e/mongodb-world-2018-tickets-41087292197) 。你可以选择向多元化奖学金捐款，并且仍然可以通过给某人参加的机会来产生影响。

#### [](#female-innovators)女性创新者

认识一个认定自己是女性的技术专家，应该参加 MongoDB World？[提名她](https://www.mongodb.com/world18/female-innovators?jmp=blog)！

被提名者将获得一张免费的会议通行证*和一张在女性和跨编码者休息室午餐时进行引导式讨论的邀请。也鼓励自我提名。提名截止日期是 4 月 27 日。

<center>[<button>Nominate a Female Innovator</button>](https://www.mongodb.com/world18/female-innovators?jmp=blog) </center>

#### [](#women-and-trans-coders-lounge)女性和跨码者休息室

女性和跨性别编码员休息室的目的是在我们的工程社区中扩大非二进制人群、女性和跨性别人群的声音。它部分是由 MongoDB 的妇女和跨编码组运行的。在 MongoDB World 期间，停下来建立关系网，在引导式讨论中分享您的想法，或者参加“让它变得重要”会议。

#### [](#affinity-group-happy-hours)亲和群欢乐时光

6 月 26 日，加入我们由 MongoDB 员工资源小组举办的鸡尾酒会和模仿秀。你将能够参加由黑人网络、MongoDB 妇女团体和 Queeries (MongoDB 的 LGBTQ 团体)主办的欢乐时光。所有欢乐时光还将提供鸡尾酒和不含酒精的饮料。

#### [](#black-network-happy-hour)黑网欢乐时光

在黑人网络欢乐时光期间会见 MongoDB 黑人网络的成员和其他认为自己是黑人的与会者。

#### [](#lgbtq-happy-hour)LGBTQ 欢乐时光

结束 MongoDB 世界的第一天，加入那些被认为是 LGBTQ 社区成员的 MongoDB 用户，在酒吧喝一杯。

#### [](#women-in-tech-happy-hour)女性在科技欢乐时光

在科技女性欢乐时光，你将受邀与女性创新者、会议发言人和其他科技女性交流。

我们的目标是让所有人都能访问 MongoDB World。如果您需要无障碍设施，请联系我们。

#### [](#code-of-conduct)行为准则

所有与会者都应该同意并遵守行为准则。
