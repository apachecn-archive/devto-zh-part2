# 我现在在哪里

我正处于学习 Angular 的初级阶段，还没有创建我的第一个应用程序。但是，我不想高枕无忧。我想开始把我的想法开发成真正的产品。我定期做了大量的研究，但我觉得还没准备好。多半是因为技术经验。

在工作中，我能学到多少东西是有限制的。有经验的人和我们在一起的时间很少，总是希望我们自己去学习。虽然在某种程度上是好的，但这影响了我的学习进度。解决一个依赖 bug 需要我一两天的时间。

# 我的计划

很明显，我必须拼尽全力才能看到这些项目变成产品。为了不让自己精神上疲惫，我一大早就一直在做个人项目。据我了解，我至少需要 12-15 天才能适应 Angular。与此同时，我将继续我的研究。也许我会找到我丢失的东西。

VS 2017 对我来说是一场噩梦。让平台为 Angular 开发做好准备已经是一项令人兴奋的工作了(我真的很喜欢这方面的一些建议)。
