## 我第一次听说 Web 开发是什么时候..

高中的时候偶然上了一堂多媒体设计课。我试着去上打字课，但我只是想了想“哦，好吧”,就留在了班上。我的老师是一名足球教练，他需要一份工作，但他不知道这门课的主题。所以我自然对这门课不感兴趣。大约一两周以后，这个女孩走进来，坐在我旁边。Sara 向我展示了 HTML 和 CSS 是如何一起工作的，这就是它开始的地方。

## [](#im-currently-learning)我目前正在学习..

当我独自编码时，如何管理我的多动症并保持自我激励。(如果你有任何建议，请告诉我)我大学时的导师也患有多动症。他是我不放弃的唯一原因。当我需要帮助时，我会去他的教室，在他回答我的第一个问题之前，我们会跑题 100 次。但是他看到了我的潜力，知道我需要帮助，而且他已经帮了我很多。

我的网站 Chofia.github.io 因为我编辑了 12 次，花了一段时间才理解 github。在一位老朋友的帮助下，我已经把它建立起来并开始运转了。(大部分情况下)
