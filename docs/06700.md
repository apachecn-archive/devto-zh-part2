# 编程是一项工作和爱好

> 原文:[https://dev . to/jwoertink/programming-as-a-job-a-hobby-1je](https://dev.to/jwoertink/programming-as-a-job-and-a-hobby-1je)

真的很容易累垮。这不仅仅是一个编程的事情，而是任何事情。你可能会在餐馆吃饭、玩游戏、看表演或在任何行业工作时筋疲力尽。

“筋疲力尽”是指你对某件事情感到厌倦，或者厌倦到你曾经热爱的某件事情，而不是开始怨恨和憎恨。

## [](#the-start)开始

当我第一次开始编程时，我被它淹没了。这花了一些时间，但一旦我开始得到它，它是一个爆炸。我终于处于这样一种状态，人们不断向我寻求帮助，因为我可以创造他们想象的产品。

我对编程如此着迷，以至于当我打开浏览器时，我的第一页会是 [Github](https://github.com/) 。我总是很兴奋地看着新的 [PRs](https://help.github.com/articles/about-pull-requests/) 被提交。哪些新提交修复了我在不同库上遇到的错误。看到别人是如何解决问题的，真是令人惊讶。

在做了一整天的网站建设后，我会坐在那里看电视，我会很无聊。我会拿起我的笔记本电脑，开始写一些代码。

## [](#breaking-the-mundane)打破世俗

由于我白天做 web 开发，晚上无事可做，我决定在空闲时间编程完全不同的东西是改变它的好方法。我开始制作一些 2D 和 3D 的视频游戏。这是一个完全不同的游戏(无意双关)。我现在用以前从来不需要的方式写代码。我在应用新的概念，并且知道我数学很差。

通过做一些完全超出我正常工作范围的事情，我能够摆脱日常工作中的刻板印象和狭隘视野。我这样做了一段时间，但最终导致我再次筋疲力尽。我不擅长制作游戏，也厌倦了制作网站，所以现在是时候尝试移动应用了！一种新的写作语言，一些新的经历会非常有帮助。

## [](#the-downfall)败亡

最终，我又开始精疲力竭了。然而，这一次不是从编码开始。这次更多的是来自文化。感觉文化有了新的转变，我注意到许多开发人员开始属于脾气暴躁的开发人员类别。突然之间，不管代码运行得好不好，人们都在根据我写代码的方式给我打分。它并不“时髦”，也没有使用最新的尖端风格和设计模式。我在开源工作中被嘲笑，我在空闲时间为自己的个人享受贡献东西。这个[不是](https://www.quora.com/What-are-some-reasons-programmers-quit-their-high-paying-jobs) [只是](https://www.quora.com/What-happened-to-Ryan-Bates-of-Railscasts) [我](http://www.slate.com/articles/technology/technology/2012/03/ruby_ruby_on_rails_and__why_the_disappearance_of_one_of_the_world_s_most_beloved_computer_programmers_.single.html)。

我现在的处境是，参加编程会议并不有趣，因为我无法与其他开发人员真正相处。我对不同主题(编程相关或无关)的观点与其他人不一致。这几乎就像一个现场 reddit 评论线程。如果你从来没有在 reddit 上花过任何时间阅读评论，它基本上就像一个有机体。在这一点上，reddit 喜欢 Elon Musk，下一分钟它就是一场全面的仇恨盛宴，如果你不赞同 reddit 的观点，你就会被遗忘。(埃隆就是一个例子，但这也适用于很多话题)。

最近我决定开始写这些博客，因为我觉得我可能已经走到了编程之路的尽头。它并没有像以前那样真正带给我快乐，所以我想在这发生之前把我自己的个人经历和知识传递给大家。就在最近，我写了一篇博文，详细介绍了我为一个特定客户所做的一些工作，以及这项工作是如何具有挑战性的。醒来发现有人告诉我我的帖子是垃圾，我承认洗钱和欺诈不是开始一天工作的好方式。我没有精力坐在那里，与 20 种不同媒介中的每个人争论我所做的一切都是合法的，所以我删除了这个帖子。

说真的，在这一点上，经营墨西哥卷卡车听起来更有趣(我一直想这么做)。

## [](#saving-grace)救国恩

所以如果你已经精疲力尽到这种程度，你怎么回来？你如何找回建设和创造的激情和动力？

*   换一份工作？也许在新的环境中工作会有所帮助
*   休假！有时那是不可能的，但是走出房子。
*   尝试一个新的副业。这可能有助于一些额外的收入，和一个新的挑战。除非你已经积压了很多工作。
*   尝试在新的操作系统环境中编程。一直用 Mac？试试 Windows！说真的，一个不同的操作系统会教你很多新东西，让你走出舒适区。
*   有编程以外的爱好？尝试混合一些代码。我是一个音乐人，所以我确实有一些与音乐相关的代码。
*   韩式 Spa！说真的，如果你还没去过，那就去吧！他们太棒了。
*   与编程无关的 Meetup 群组。比如徒步旅行团之类的。他们知道你是一名程序员，然后开始问你一些疯狂的问题。不过，你每次都会听到“我有下一个十亿美元的想法”的演讲。
*   离开一会儿。就像为什么或 Ryan B .做的那样，他们离开一段时间，然后在雷达下回来。

有适合你的想法吗？下面的评论可以帮助其他人摆脱疲惫。
