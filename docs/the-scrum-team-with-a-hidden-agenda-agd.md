# 我们受雇于一个秘密的议程

> 原文：<https://dev.to/jaymeedwards/the-scrum-team-with-a-hidden-agenda-agd>

有一次，我被要求调查两个团队，他们合作开发一个软件产品来处理税务。

第一个团队说他们需要帮助来更好地发布软件。

所以我打算用我多年来想出的一个采访。

我可以问人们一些关于他们如何一起工作的问题。

比如他们如何收集需求，测试软件，以及批准发布。

采访第一队时，**很明显他们不喜欢第二队**。

第一个团队在追随 scrum，而第二个团队没有。

然而，在采访了两个团队的许多人之后，我们意识到第二个团队正在遵循看板。

但他们这么做是有充分理由的——应对税法中不可预测的变化。

所以最终我们展示了我们的发现，尽管很多人很高兴- *我们的客户很失望我们没有发现第二个团队的问题*！

在这段视频中，我分享了一些需要考虑的事情，可能会帮助你避免在公司政治中被利用。

如果你被迫站在一群人一边，迫使另一群人不情愿地改变，我希望这能帮助你！
