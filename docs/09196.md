我可以谈谈东西是如何工作的，以及我们为什么选择某些故事，但我会把这些留给下一次。今天我只想谈谈这周的新闻文章中让我印象深刻的是什么，哪个故事是我最喜欢的，以及我从这些文章中学到了什么。

### [](#1-what-stuck-out-to-me)1。让我印象深刻的是。

本周有许多可靠的消息，这些消息有一些深度，是其他几周有时没有的。一半以上的文章都是长时间阅读的，并且/或者需要多个链接才能完全理解上下文。例如，Ethereurm-Blockchain size 文章是对 8 个多月前写的一篇文章的回应，要获得完整的“热烈”感觉，你基本上需要阅读 Hacker News 和 Reddit 上的帖子。而且，这两篇文章都超级厚。大量的信息。我读过很多关于区块链科技的文章，这些文章是我去年读过的所有文章中最难懂的。也就是说，它们很可靠，非常值得一读。我捡起的东西让我很高兴我阅读并潜入这个宇宙。

### [](#2-which-story-was-my-favorite)2。哪个故事是我最喜欢的。

我最喜欢的新闻故事无疑是功能生成艺术。我不仅不知道 Elm 可以生成 SVG 图像，而且我也不知道有一种方法可以让我的内在艺术家发光。Xavier Ho 在 30 分钟内从上到下讲述了他如何设置一个机器人用榆树制作艺术品。多酷啊。我总是被告知我永远不会在艺术界成功，但是现在我有了尝试的灵感。然后，我可以宣称自己是一个艺术家，这是我一直想做的。

### [](#3-what-i-learned-from-these-articles)3。从这些文章中我学到了什么。

*   以太坊-区块链的大小超过 1TB，直到最近还有很好的理由相信它不会这么快达到这个大小。

*   谷歌云平台仍在解决一些问题。

*   颤振是一个东西，它看起来真的很整洁。

*   我可以创造一种艺术。感谢 Xavier Ho 向我们展示了技术人员，我们也可以制作艺术！

*   我对 CSS 选择器一无所知，我可能应该花些时间摆弄它们。它们看起来真的很整洁/有趣/酷/有效/用光了形容词/但是说真的，是什么让我尝试这些呢？如果这是一个 JavaScript 框架，我已经有一件 t 恤了。
