# 使用 dbForge Studio for Oracle 将数据库开发提升到企业级别

> 原文:[https://dev . to/devart software/take-database-development-to-the-enterprise-level-with-dbforge-studio-for-Oracle-3585](https://dev.to/devartsoftware/take-database-development-to-the-enterprise-level-with-dbforge-studio-for-oracle-3585)

Devart 推出了新版本的 dbForge Studio for Oracle v.4.0，该版本进行了大规模改造。在其他变化中，新工具现在包括企业版，它具有许多独家的强大功能，如数据生成器，允许立即生成数百万条有意义的测试数据记录，以及用于记录和呈现 Oracle 模式的全新文档管理器。

以下是在更新的 dbForge Studio for Oracle 中实现的最显著改进的独家新闻:

I .数据导入/导出特性现在包括对 JSON 数据和 Google Sheets 的支持，使用户能够轻松地在 Oracle 表和 Google 电子表格之间迁移表数据。

二。SQL 文档特性也获得了一打新选项:
——关闭当前标签页的 CTRL+W 快捷键。
-打开当前标签旁边的新标签。
-CTRL+N 快捷键，用于用一个新的继承当前 SQL 窗口的服务器和数据库。
-状态栏中连接颜色的显示。
-查询导航。要在查询中的关键字之间导航，可以按 Ctrl+F12 或使用快捷菜单中的“在语法对之间跳转”选项。
-改进的查询文本解析。
-大大重新设计的查询执行历史
-以及许多其他内容。

三。数据查看器功能得到了增强，可以显示数据类型和大小。

四。模式比较和数据比较功能现在提供了以下选项，这些选项大大加快了 Oracle 数据库的比较、分析和同步过程:

-改进的比较报告
-许多增强的模式/数据比较和同步选项
-附加脚本选项卡允许键入或选择在数据同步之前和/或之后执行的脚本
-以及许多其他选项

仔细看看在[https://www.devart.com/dbforge/oracle/studio/](https://www.devart.com/dbforge/oracle/studio/)的 dbForge Studio for Oracle 中现有的和以前的特性。
