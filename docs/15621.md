# “英雄”随想

> 原文:[https://dev.to/captainsafia/random-thoughts-onheroes-3195](https://dev.to/captainsafia/random-thoughts-onheroes-3195)

当我第一次接触科技时，我有很多英雄。他们中的大多数完全不像我。我崇拜像比尔·盖茨和谢尔盖·布林这样的人物。在他们身上，我看到了我想看很久的东西:一群书呆子，尽管受到欺凌、戏弄和孤立感，但他们“成功了”当我第一次了解这些人(他们都是男人)时，我感到安慰的是，他们太聪明、太古怪，但却获得了成功。那是 13 岁的我想看的东西。他们是我的英雄。

但即使是你的英雄也会搞砸。

时间过得越久，我就越了解他们所犯的错误，不管是可容忍的还是不可容忍的。

我发现了新的英雄，有时，他们也倒下了。

有一个英雄是一种安慰。把一个人超乎寻常的个性发挥到极致。

这在科技界经常发生，有时会适得其反。

在过去的几年里，有很多人来找我，说是我启发了他们。我不会夸大其词，声称我是他们的英雄，但很明显，他们非常尊重我。

我来告诉你；坐在桌子的对面感觉真的很奇怪。

主要是觉得自己不配。我没有发现我每天做的任何事情特别鼓舞人心。我想对于我这个年龄的人来说，我可能已经完成了很多事情(不管那意味着什么),并且承担了大胆的风险，得到了回报，我认为对一些人来说，这是令人钦佩的，鼓舞人心的，也许是英雄式的。

旁注:那句话完全是连篇累牍的，绝对不是英雄主义。

无论如何，在过去的几个月里，我越来越担心有那么多人声称我激励了他们，或者激励他们去做一些事情等等。

我知道，我知道。谁会对此感到焦虑呢？我很高兴我的行为激励了人们。但我的一部分情不自禁地觉得，我必须成为这些人的完美“英雄”，永远不要搞砸任何事情。这是不合理和不现实的，但我仍然持有这种信念。我想成为人们心目中比我心目中的英雄更好的英雄。

这很奇怪，因为谁不想成为励志和激励，对不对？但是还有另外一个人们不常谈论的方面，那就是让人们尊敬你的责任有时会让人麻痹。

我可能选择让责任成为我生活的一个重要方面，但我就是那种人。感觉什么都搞砸不了。

让人觉得你已经找到了所有问题的答案，这也很奇怪。这里有一个有趣的故事。当我在找工作的时候，当我因为害怕永远找不到在科技行业谋生的方法而在大多数晚上哭泣的时候，当我对整个情况感到极度恐慌的时候，我收到了一些人的电子邮件，他们向我推荐他们自己的工作。见鬼，我自己什么也没想出来，但人们仍在向我寻求答案。对我来说，那是一次非常奇怪的经历。当别人期望你知道答案时，你如何回答一个你不知道答案的问题？

这篇博文又长又乱，可能在某些地方没有意义。但这就是我们在萨菲亚博客上提供的内容！

我想这篇文章主要是想说，我对什么是英雄的看法已经改变了，因为我看到了我的堕落，也注意到有些人把我视为他们的英雄。

正如我所有的博文一样，这一篇带来的问题多于答案。

谢谢你，网络空间，感谢你吸收了我的感受。

再见。
