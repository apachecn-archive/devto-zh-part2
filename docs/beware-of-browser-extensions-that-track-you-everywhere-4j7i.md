# 当心到处跟踪你的浏览器扩展

> 原文：<https://dev.to/ben/beware-of-browser-extensions-that-track-you-everywhere-4j7i>

我最近注意到我安装的一个 Chrome 扩展——一个简单的工具——正在将 Mixpanel 加载到我访问的每个页面中——不管我是否在使用这个扩展。过去，我发布了一个流行的 Chrome 扩展，并收到一些人的随机请求，他们愿意付费来跟踪我们在网络上的流量，我知道这是一个行业。

这是对隐私和安全的明显威胁，但也是对令人愉快的浏览体验的威胁，因为网页停滞不前。这两者对开放网络来说都很糟糕。

如果你还没有注意的话，花点时间去意识到这些问题。
