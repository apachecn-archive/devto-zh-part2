# 破解你的职业生涯:WeRise

> 原文:[https://dev.to/jeremylikness/hack-your-career-werise-2haf](https://dev.to/jeremylikness/hack-your-career-werise-2haf)

#### 一个拥有数十年代码付费经验的家伙给专业开发人员的指南。

2017 年 12 月，我收到了一份邀请，邀请我回到由在亚特兰大编码的[女性主持的](https://www.womenwhocode.com/atl) [WeRise 技术大会](https://werise.tech/)上发言。我非常兴奋和高兴，因为我去年在 T4 有一次非常积极的经历。我想为这次会议做一个不同风格的演讲，并决定不关注某项技术，而是分享一些基于我的专业开发经验的职业建议。

[![](../Images/0f8c9fea76c5bdba1081bcaa5a1ffd77.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--J3CazkAc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2AgoU65-ncH-F_GnxYoHIHaw.png)

我很感谢 WeRise 的组织者愿意投资录制和出版这些演讲。你可以在这里观看我的演讲:

> [点击此处观看演示的完整视频](https://www.recallact.com/presentation/hack-your-career-tips-and-tricks-20-year-veteran)

自 1994 年以来，我一直从事专业的编码工作，曾经担任过个人贡献者和管理者的角色。在那段时间里，我亲自审阅了数千份简历，雇佣了数十名开发人员。

> 在 [@jeremylikness](http://twitter.com/jeremylikness) 的演讲中学习如何破解我们的职业生涯。# werisetech # wwcodeatl # pride 2018
> 
> — [@WeRiseConf](https://twitter.com/WeRiseConf/status/1010246975936237568)

与我职业生涯的早期相比，我现在在演讲前应对“冒名顶替综合症”要容易得多，因为我学会了更多地关注观众和我可以分享的东西，而不是谈论我自己。我的职业演讲不同于我的典型会议，因为这是一个软技能演示，我不知道它会被接受。我也非常敏感地意识到，该活动的重点是技术领域的女性，因此对我来说，这一信息不是作为讲座或肥皂箱收到的，而是真诚的分享，希望帮助和推动围绕人们可能不同意或基于个人经历有不同看法的主题的对话。

我的第一次“真正的”编程是在[的 Commodore 64](https://en.wikipedia.org/wiki/Commodore_64) 上使用 [6502 汇编](https://en.wikibooks.org/wiki/6502_Assembly)，我永远不会忘记这种语言和我在 64 千字节机器上编写代码的乐趣。我仍然认为作为一名开发人员，最有趣的事情是当 [Silverlight](https://en.wikipedia.org/wiki/Microsoft_Silverlight) 发布的时候。虽然 Silverlight 现在也是一个历史书籍的平台，但我不会忘记我在从里到外学习该平台以提供前沿业务解决方案时所面临的令人兴奋的挑战。毫不奇怪，我总能在每次谈话中运用它。
