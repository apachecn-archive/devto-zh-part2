# 总是添加大括号

> 原文:[https://dev.to/timabell/always-add-braces-1c68](https://dev.to/timabell/always-add-braces-1c68)

我一直遇到的代码风格的小问题。

是否写

```
if (thing) Action(); 
```

或者

```
if (thing)
{
    Action();
} 
```

我们应该总是使用后者，以避免在手动或通过合并工具修改代码时引入错误。
