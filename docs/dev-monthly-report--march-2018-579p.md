# 开发月报—2018 年 3 月

> 原文：<https://dev.to/jess/dev-monthly-report--march-2018-579p>

三月是历史新高的一个月。我们很高兴看到越来越多的开发人员不仅加入进来，还将他们的想法贡献给开发社区。

**流量和社区增长**:我们在 3 月份提供了 142 万次会话，现在拥有超过 6.5 万名注册会员。我们还实现了分布的多元化——一年前，dev.to 的大部分流量来自 Twitter。现在，有机已经成为最突出的渠道。这对每一个做出贡献的人来说都是一个令人兴奋的进步。

**入职增强功能**:新成员现在会收到一份推荐名单。我们对这一改变最初引起的混乱表示歉意，并在这里写了更多关于功能[的内容。我们将跨平台工作，使“关注用户”功能更加突出和有用。](https://dev.to/ben/changelog-suggested-follows-on-onboarding-3ifb)

**摘要邮件**:根据标签和你关注的人，我们[开始发送](https://dev.to/ben/changelog-customized-emails-to-catch-folks-up-on-posts-they-might-not-want-to-miss-4ec)摘要邮件，推荐阅读。这是为那些主要通过电子邮件与我们互动并希望了解特定主题的人而创建的。

国际妇女节:我们一年中最喜欢的一天，来自社区的人们分享他们的故事和建议。我们通过突出来自整个发展和社会的妇女的文章来庆祝。感谢所有参与或分享的人。查看 [#SheCoded](https://dev.to/t/shecoded) 标签，看看一些令人惊讶的提交。

**审核工具**:这不是面向用户的，所以我们没有写变更日志，但我们已经对我们的审核工具和流程进行了坚实的改进。我们还没有遇到任何大问题，但我们总是试图提前考虑。三月份，我们在检测潜在垃圾邮件账户方面也有所进步。

开源:我们正在缓慢而稳定地进行，但是我们已经邀请了一些人来帮助我们检查所有的东西。大声欢呼[尼克·泰勒](https://dev.to/nickytonline)成为第一个提交 PR 的社区成员！如果你有兴趣投稿，请填写[这张表格](https://thepracticaldev.typeform.com/to/HkjWky)。我们仍然在逐步进行这个过程，所以你可能不会得到快速的回应。

**[事件](https://dev.to/events)** :我们已经完成了第二个月的直播，正在上传所有视频与大家分享。如果你想做一个讲座或研讨会，这里有[的细节](https://dev.to/jess/would-you-like-to-give-a-dev-talkworkshop--31c6)。
