#### 背景

我开始断断续续地研究 web 开发大约有一年了。我目前正在攻读计算机科学领域的学位，有 C++的工作背景。

在开始自学 web 开发时，我虔诚地查看了免费代码营网站，但我并没有真正专注于做前端项目。我记得我更专注于它的一般算法部分，不知何故，这似乎更容易。“可能是因为我从事的是低级语言的工作”

从那时起，我就一直在自学“网络开发”。因此，我一直在不同的资源和主题之间跳跃，从来没有真正专注于一个。外面有那么多东西！！💀💀

有了它，我找到了大量可以利用的资源。

#### 咆哮

不幸的是，我从免费的代码营算法跳到代码学院的指导课程，再跳到 udacity 上学习如何用 pyhon 建立网站。在此之前，我从未使用过 python？!'

#### 分辨率

我被卡住了，我的作品显示了这一点。我决定关注前端一段时间，我注意到 css 是多么令人沮丧。然而就在这个时候，css grid 越来越受欢迎，我决定追随它。我最终建立了我的第一个作品集网页，我非常乐意分享。
