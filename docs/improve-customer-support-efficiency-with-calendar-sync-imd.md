无论你称之为客户服务、客户支持、客户关怀还是其他，目标都是一样的:提高你的客户满意度得分(CSAT)，同时提高客户支持代表的效率。任何承担客户支持角色的团队都知道，这是说起来容易做起来难。

这就是为什么我们很高兴地宣布 [Nylas](/) 和 Freshdesk 之间的合作伙伴关系，使支持团队能够帮助他们的客户更快地获得解决方案，并提供更直接的支持。T3】

[Freshdesk 日程安排应用](https://apps.freshdesk.com/freshdesk_scheduling_app/)(由 [Nylas 日历 API](https://www.nylas.com/calendar-sync) 提供支持)允许 Freshdesk 代理安排用户和第一个可用代理之间的会议，所有这些都无需离开 Freshdesk 平台。日历同步 API 使您的代理能够无缝集成所有日历服务，包括来自 Freshdesk 的 Gmail 和 Outlook，因此客户可以收到本地活动邀请。T11】

当代理在安排会议等管理工作上花费的时间减少时，他们就有更多的时间进行人与人之间的支持互动。日历同步功能包括:

**与 100%日历无缝集成**

Nylas API 将 100%的电子邮件服务提供商连接到 Freshdesk 应用程序，允许用户从 Gmail、[Outlook.com](/outlook-calendar-api)、微软 Exchange、Outlook.com 等网站同步他们的日历。

**根据上下文安排会议**

上下文对于出色的客户支持至关重要。通过在 Freshdesk 应用程序中安排客户支持代理和用户之间的通话，您可以轻松地将支持票证的所有上下文复制并粘贴到通话议程中，以便两个团队都有完整的上下文。

**更快地将客户转给合适的团队**

背景对于出色的客户支持至关重要。通过在 Freshdesk 应用程序中安排客户支持代理和用户之间的通话，您可以轻松地将支持票证的所有上下文复制并粘贴到通话议程中，以便两个团队都有完整的上下文。

### **节省客户支持代理的时间**

Nylas 应用程序使代理不必跳出 Freshdesk，跳到 CRM 平台，跳到另一个日程安排平台，然后再回到 Freshdesk 来安排客户电话。现在，只需几次点击，代理就可以标记适当的团队来处理问题，并安排会议的日期、时间和媒体。

### **准备好开始了吗？**

任何拥有 Freshdesk 帐户的人都可以下载并使用 Freshdesk 计划程序。只需从 Freshdesk marketplace 下载应用程序。

下次您的客户来信时，您将完全有能力提供神奇的支持。
