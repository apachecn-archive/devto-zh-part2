# 远程工作 3 年来我学到的东西

> 原文:[https://dev . to/bradymholt/things-I-learn-working-for-3-years-4ea 7](https://dev.to/bradymholt/things-i-have-learned-working-remotely-for-3-years-4ea7)

大约 3 年前，我[加入了 YNAB 团队](https://www.geekytidbits.com/joining-you-need-a-budget/)。这是我第一次远程工作的经历，虽然我真的很喜欢远程工作，但也不是没有一些挑战。我早期做错了一些事情，也做对了一些事情，我仍在边做边学。

## 真的很喜欢

首先，我想提一下，我真的很喜欢远程工作。原因是:

1.  **控制我的工作环境** -我能够在家里创造一个安静、舒适的工作环境。对我来说，这对我的生产力和幸福感是一个巨大的提升。
2.  地域灵活性 -这是显而易见的，但我确实喜欢在我工作的地方有灵活性。我妻子和孩子们呆在家里，所以如果我们想去外地或其他州探亲，我们可以在 T2 呆几周，因为我可以在那里设置和工作。

## 吸取教训

当我知道我将远程工作时，我做的第一件事(也是最好的一件事)是联系我一个从大学就认识的朋友，征求他对远程工作的建议。那时他已经远程工作了大约 5 年，我知道他会有很多好的建议。我是对的。他能够给我一些需要注意的事情，以及一些在早期建立一些积极习惯的想法。

我将这些建议付诸实践，并发现了一些对我很有用的东西。我学到的让远程工作可持续发展的最大经验是:

1.  工作和家庭在物理上分开——我的办公室在房子的另一个角落，隔着两道门。当我在家的时候，我看不到我的办公室，有点遥不可及。如果我的办公室是可见的，我失去了工作/家庭的分离，这可能会导致恐惧/倦怠。
2.  在一天结束的时候，我会找一个好的地方停下来，关上笔记本电脑，关上办公室的门。计划“在晚饭后和孩子睡觉前完成一些事情”很有诱惑力，但这是一个危险的先例，因为它会导致精疲力竭，占用家庭时间。
3.  一周在其他地方工作几次 -我花了一段时间才学会这个，但为了防止幽居症，一周几次去咖啡店之类的地方工作几个小时是有好处的，因为这有助于我与真实的人互动，让事情变得更复杂。
