# yasminuroban——JavaScript 中的推箱子游戏(2006 年 3 月)

> 原文：<https://dev.to/jalbam/yasminuroban--sokoban-game-in-javascript-march-2006-2loi>

“Yasminuroban”是一个完全用 DHTML (HTML、CSS 和 JavaScript)编写的开源 [Sokoban(又名仓库保管员)](https://en.wikipedia.org/wiki/Sokoban)版本。带关卡编辑器。

[![Screenshot](../Images/5e6cfe3a84e222bae4415d14a506e102.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--4oV2OadI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/http://dhtmlgames.com/yasminuroban/yasminuroban.gif)

这是我的第四个 DHTML 游戏，制作于 2006 年初。

这个游戏包括一个关卡编辑器，据我所知，这是第一个用 JavaScript 制作的关卡编辑器。它允许你在按下“播放地图”选项后修改当前关卡或创建一个新的关卡来播放它。例如，您还可以复制生成的 JavaScript 代码，用于自己修改游戏代码。

你也可以通过编辑脚本开头的配置变量来改变游戏行为或者增加新的关卡。要改变游戏图像，你只需要用其他图像替换它们。

官方语言是西班牙语和英语。

你可以在互联网上找到这个游戏的其他修改和改编版本(有可配置的皮肤或主题，玩家可以选择许多级别)，包括一个 Opera 小工具。

这款游戏已经在 BeOS、Linux、NetBSD、OpenBSD、FreeBSD、Windows、Mac OS X 等平台下测试过。
