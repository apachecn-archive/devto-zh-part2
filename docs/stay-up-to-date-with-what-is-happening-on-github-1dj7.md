# 关注 Github 上的最新动态

> 原文：<https://dev.to/arojunior/stay-up-to-date-with-what-is-happening-on-github-1dj7>

在几次听到一些不使用社交网络的朋友说“我的社交网络是 Github”之后，我决定制作一个 feed，将你在 Github 中关注的人的活动带来。从对问题的评论，拉请求，星库，提交等…让你跟踪开源社区发生的一切。

类似 twitter 的布局完全是故意的，考虑到它是开源社区的首选社交网络。

它仍然需要改进很多，但由于我一直独自工作，它走得很慢。

你们可以合作:[https://github.com/arojunior/awesome-feed](https://github.com/arojunior/awesome-feed)

使用的工具基本上有:React、Redux、GraphQL、Styled-components 和 Recompose。

一如既往，非常欢迎建议。我将等待拉取请求。
