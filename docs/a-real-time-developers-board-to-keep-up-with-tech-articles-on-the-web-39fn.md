# 我是怎么建的(栈是什么？在这个过程中，我遇到了问题或发现了新的东西吗？)

后端栈我用了 NodeJS/ExpressJS。jQuery 和 materializecss 用于前端代码。客户端和服务器的推送通道 API 也包括在内。我在 Heroku 上托管了这个应用程序。

# 附加资源/信息

该应用程序具有推送通道 API 提供的实时更新功能。这些功能是新增加的渠道，文章，文章喜欢，文章访问。这个黑客提供了一个框架，允许平台的用户实时看到新的和流行的文章。实时交流的美妙之处和开发人员板的创建，对于其他开发人员来说，在他们利用分散在万维网上的公开可用知识成为软件工匠大师的过程中，将是非常重要的。
