# 如何为 2018 年 GDPR 截止日期做准备？

> 原文：<https://dev.to/wenso_smith/how-to-get-prepared-for-the-2018-gdpr-deadline-1fgd>

欧盟 [**通用数据保护条例**](http://www.wenso.co.uk/gdpr-consulting-services/) (GDPR)取代了数据保护指令 95/46/EC，旨在统一整个欧洲的数据隐私法律，以保护和允许所有欧盟公民的数据隐私，并改革整个地区的组织走向数据隐私的方式。这对全球运作的组织有相当大的影响。

借助 **GDPR 评估**计划，我们可以找到做**差距分析和风险评估**的人。为网络安全制定的最佳实践建立了一个强大的框架，以解决与 **GDPR 合规性**相关的所有问题。这种评估对于各种类型的企业，无论其规模大小，都是适用的和有价值的。

[https://www.youtube.com/embed/CKJahUG3FMM](https://www.youtube.com/embed/CKJahUG3FMM)

GDPR 的基本目标是提出当前组织的隐私和安全结构。它还有助于理解为实施 GDPR 而制定的组织计划和行动。这也使组织能够进行战略规划并做出自己的战术决策。

2018 年伊始，GDPR 公司正忙着准备迎接实施该计划的最后期限。这些准备工作是强制性的，因为如果不遵守这些政策，将被处以高额罚款。实施截止日期促使各组织积极地开始在 GDPR 开展工作。他们可以采取的第一步是制定明确的数据分类策略，制定数据使用和保留以及基准安全控制的指导原则。通过做这些后台工作，他们可以自动化 [**GDPR 合规实施**](http://www.wenso.co.uk/gdpr-consulting-services/gdpr-compliance-implementation/) 的整个流程，还可以降低成本。

GDPR 的基本目标是保护其公民的数据权利。GDPR 中列出的这些权利范围很广，肯定会以多种方式影响各种商业模式和流程。
