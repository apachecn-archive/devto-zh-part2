# 征服您的收件箱

> 原文:[https://dev.to/codemouse92/conquering-your-inbox-5h2o](https://dev.to/codemouse92/conquering-your-inbox-5h2o)

**电子邮件收件箱**。噩梦和浪费生产力的地方。信息死亡的黑洞。但是不一定要这样！

这些听起来像你吗？

*   您的收件箱中有 4K 未读邮件。
*   您的收件箱没有未读邮件...你一个都不记得了。
*   你的收件箱是空的，因为你马上删除了所有的东西。
*   你的收件箱管理得很好，因为你每天花四个小时整理它。

作为一名企业主，我有多达 18 个收件箱，其中五个相当活跃。我必须有一个系统来让它们有条不紊，不会占用我一半的时间。

## [](#mailbox-layout)邮箱布局

关键是你的邮箱要有一个有用的结构:

*   收件箱:这应该只包含需要采取行动的消息，或者可能在下周需要的消息。

*   存档:我每个季度都会在存档中创建子文件夹，这样可以更好地查找和组织旧邮件。

*   已发送:您发送给其他人的消息。就我个人而言，我倾向于不去管它，直到我的季度大清洗。

*   垃圾:永远不要低估你的垃圾箱的重要性！大多数电子邮件服务器都有某种形式的自适应垃圾邮件过滤，所以你应该留意这个文件夹。确保没有重要的东西进去。

*   垃圾邮件:这是你不再需要的非垃圾邮件。定期清空！

## [](#daily-management)日常管理

像大多数人一样，当我在工作时，我的电子邮件一直在后台运行。因此，当有消息进来时，我会立即采取以下五种行动之一:

1.  如果我连浏览一条信息的时间都没有，我会**将它标记为未读**。然而，像大多数人一样，看到这个指标让我心烦，所以我会很快浏览一遍。

2.  如果消息**需要一个动作**——重读、回复、参加即将到来的活动或会议、查看附加/链接的材料，无论什么——我将**星**或**标记**该消息。

3.  如果该邮件不需要操作，但我可能需要很快(在下周内)再次查看，我会将它作为已读邮件留在收件箱中。

4.  如果邮件不需要操作，并且我认为我不需要很快再次查看它(或者它已经超过一周了)，我就**存档**它。

5.  如果消息根本不重要，不需要任何操作，我会删除它。

当然，对于您标有星号的邮件，一定要保持在它们的顶部，并且在您采取适当的行动后立即取消它们的星号！

## [](#quarterly-management)季度管理

当然，几周或几个月后，事情自然会变得杂乱无章。因此，每个季度，我都会留出一个小时左右的时间来整理我的收件箱。

*   我确保我已经对所有标有星号的项目采取了行动(除了那些即将到来的项目)。

*   前一季度的所有未加星号的收件箱邮件都将存档。

*   前一季度发送的所有邮件都会存档。

*   垃圾邮件被清空(除非我的邮件服务器有其他规则)。

*   垃圾被清空。

## [](#getting-there)到达目的地

“嗯，哎呀，听起来不错，但是我怎么去那里呢？”你可能会问。

这通常就像存档你所有的旧邮件，然后浏览并删除任何无用的东西一样简单。当你这样做的时候，你将开始发现一些模式——比如主题行中的关键字“Notification”或某个特定的发件人——这些模式可以让你一次找到并删除大量的邮件。

这需要几个小时到几天的时间。如果在一次会议中整理你的收件箱听起来完全不切实际，简单地在你的档案中创建一个`UNSORTED`文件夹。然后，每当你有几分钟空闲的时候，看看这些信息。

我还会留意那些我经常不看就删除的邮件订阅。当你分类的时候，抓住机会取消订阅任何超过一半时间你不感兴趣的时事通讯。

## [](#the-joy-of-multiple-inboxes)多个收件箱的喜悦

我发现保持两个私人电子邮件地址很有帮助，与我的工作邮件完全分开。

一个是我的 Gmail，用于处理任何可能涉及时事通讯、群发邮件、普通通知和垃圾邮件的邮件。如果我想用电子邮件和一个完全陌生的人联系，我就用这个。

相比之下，我在自己的邮件服务器上有一个个人电子邮件地址。这款邮件比 Gmail 更加私密和安全，我用它来处理大部分私人信件。这也是我用于高安全性账户的邮箱。

如果你没有自己的邮件服务器，你可以考虑为这个收件箱使用安全的电子邮件服务。因为电子邮件的绝对数量相当低，你不太可能很快用完更小的邮箱。

如果我*真的*怀疑会有垃圾邮件，并且我知道除了最初的信息之外，我不会在乎任何东西，我会使用一次性电子邮件地址服务，比如[游击邮件](http://guerrillamail.com/)。

## [](#bonus-junk-folder-fun)奖励:垃圾文件夹乐趣

一句忠告:如果你用的是 Windows 电脑，不要打开垃圾短信！这只有在适当安全的 UNIX 机器上才是相当安全的，即使这样，除非你是专家，知道你在做什么，否则你不应该从可疑的垃圾邮件中下载任何东西。

也许使用 Linux 的乐趣之一是我对电子邮件恶意软件的有效免疫。众所周知，我会分解木马，提取病毒下载位置，并向相关部门报告违规的域名。(释放木马是一个很好的编程难题！)

也就是说，我收到了一些可以想象的最有趣的垃圾邮件！我会定期浏览我的垃圾文件夹，寻找一些免费的娱乐。事实上，我收到了五种外语的垃圾邮件(还在增加)，还有一些我见过的最搞笑的电汇欺诈。
