大家好，

每次我们开始学习新技术或工具时，我们都必须安装它，我们开始搜索，
*“如何在那个平台上安装它，使用那个——其他的东西。”*。
我们这样做了很多次，在浏览了多个博客、教程和论坛后，我们终于设法正确安装了一些东西。

为所有工具/软件提供一站式解决方案怎么样？

我在 GitHub 上启动了一个社区驱动的项目，为所有工具/技术/软件创建安装指南。

[https://github.com/moghya/install-it](https://github.com/moghya/install-it)

唯一的目的是让人们能够轻松地完成安装步骤。

请考虑为这个项目做出贡献，叉回购，并通过贡献。MD 添加指南。请与他人分享，让我们创建一个安装指南目录。

谢谢你。
