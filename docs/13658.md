# Changelog:关于入职的建议如下！

> 原文:[https://dev . to/Ben/changelog-suggested-follows-on-on-boarding-3 IFB](https://dev.to/ben/changelog-suggested-follows-on-onboarding-3ifb)

上周，我们为新用户推出了一个 onboarding 功能，其中 [dev.to](https://dev.to/) 根据他们选择的标签，建议他们可能想要关注的人。从那以后，我们就因为人们突然被“一堆新账户”跟踪而感到困惑。

我可以向你保证**这是一个特性，而不是一个 bug** 。那些新账户是真正的新注册用户。许多人还没有填写个人资料，所以很难辨别他们的人性。偶尔会有一些垃圾邮件帐户混入，但这种情况发生的几率非常低。总之，这一特性意味着更多的用户关注更多的固定贡献者。对每个人来说都很棒！

热门标签中的顶级贡献者现在将获得更稳定的新追随者流，如果他们保持伟大的文章。这将有助于人们跟上人们在他们关心的主题上分享的不断发展的思想。算法倾向于最近的，所以如果你还没有写很多，不要觉得你已经错过了机会。

我们试图传播这份爱，这样就不会完全头重脚轻，所以即使你不是*的顶级贡献者*，你也可能会有一些新的追随者。

这个功能是我和[安迪](https://dev.to/andy)共同打造的。他做了一件了不起的工作，使所有的事情都走到了一起。安迪是我们所有成功的关键部分，他在这个功能上杀了它。在这个过程中，令人困惑的推出是我的错。

我希望这能澄清一些事情。**特色，不是 bug！** **功能，不是 bug！** **特性，不是 bug！**自从推出这项功能以来，我们不得不对我们发送的电子邮件进行了一些调整，以确保人们不会受到电子邮件的困扰。因此，任何最初的电子邮件溢出都是一种缺陷。但是欢迎您在这里配置您的[电子邮件设置。因此，任何最初的电子邮件溢出都是一种缺陷。我已经亲自关闭了新的关注者邮件。你可能也想这么做。](https://dev.to/settings/notifications)
