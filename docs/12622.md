# Magento 2 产品缩放扩展

[![](../Images/64b38530e12b576cb6aa618fcbb3954e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--nOXgtHXO--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/xmobhzfbrd5wk1mam47e.png)

通过使用 Magento 2 产品缩放扩展，您可以方便您的客户放大您的电子商店的产品图像。该扩展使您能够选择放大镜和灯箱缩放选项，以允许在您的电子商店上缩放图像。 **[Magento 2 产品变焦](https://codecanyon.net/item/magento-2-product-image-zoom-extension-magnify-on-hover-click/21700914)** 扩展使您可以从后台选择各种放大镜类型。一些类型是镜头、放大或缩小、淡入/淡出等。您还可以启用灯箱缩放，在灯箱中显示图像，客户可以从那里放大和缩小。它只花了你 59 美元。
