## 装置

#### npm

`npm install --save famulus`

#### 故事

`yarn add famulus --save`

## 使用

#### ES6 模块

```
import famulus from 'famulus'
```

```
import helper from 'famulus/helper'
```

#### 节点. js

```
const famulus = require('famulus')
```

```
const helper = require('famulus/helper')
```
