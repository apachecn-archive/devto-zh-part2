# 我想邀请你参加加勒比开发会议

> 原文：<https://dev.to/zerquix18/i-want-to-invite-you-to-the-caribbean-dev-conference-4b1l>

我要去加勒比海开发者大会，我想在那里见到你！加勒比开发大会是加勒比地区的一个全新活动，数百名开发人员将齐聚一堂，结识新朋友，学习新事物，并在加勒比地区最好的地方之一度过美好时光。这是一个欢迎所有人的活动。
