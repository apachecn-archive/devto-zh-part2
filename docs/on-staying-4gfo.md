# 关于留下来

> 原文：<https://dev.to/annarankin/on-staying-4gfo>

自从我加入开发团队以来，我得到的职业建议大致分为两大阵营:

1.  “尽快辞职，去找一个薪水更高的职位——这是在这个行业中获得加薪的最佳途径，你必须充实你的简历”

2.  “找到自己热爱做的事情，然后去做(然后可能自己开公司什么的)。找到有意义的工作”。

我最近一直在思考这些问题，因为我从一个朋友那里得到了一份惊喜的推荐信，推荐我为他的雇主工作。首先我要说，我的背景是，我的职位并不像现在的技术职位那样受欢迎。尽管我已经在这个行业工作了几年，但对我来说，招聘人员、“摇滚明星忍者一个人的军队程序员”、流行语、品牌和无形产品等更广阔的技术世界仍然是一个奇异、神秘、常常令人恐惧的地方。

大多数包含“我正在寻求 X 公司的一个职位”的电子邮件很容易被忽略——开发人员会收到很多这样的邮件，它们通常是垃圾邮件，没有人情味，或者只是被误导了。我的第一反应是在打开这封邮件之前就把它存档，但它来自一位工程主管(而不是招聘人员)的事实引起了我的注意。出于好奇，我打开了它——一位我尊敬的朋友和前同事给了我一封个人推荐信，希望我加入他的团队。

被认可的感觉很好，随着我对这个角色的深入了解，他们招聘的项目与我每天从事的工作非常相似。此外，该公司规模庞大，效益惊人，作为一股社会公益力量享有盛誉，其现任和前任员工对其评价都非常高。我的朋友说服我与他们的工程主管交谈，结果发现他非常讨人喜欢，并分享了许多我对编程、团队构成和个人成长的看法。

经历了这一切，并与其他公司的朋友交谈后，我做出了决定——我要去面试这个职位。随着这个决定而来的是一种出乎我意料的悲伤。我把这归因于对离开我现在的团队，加入一个新团队的焦虑；离开过去几年来与我如此亲近的人让我感到悲伤。我来回犹豫，每隔几天就改变主意。我重复了别人给我的建议，告诉自己这是“正确的职业选择；这是我进入职业生涯下一阶段的方式。然而，那种悲伤挥之不去——一种失落感，甚至在我面试之前就变得越来越强烈。

一个比我在这个领域工作时间长得多的开发人员对我的真实感受进行了验证。我倾吐了我的思想和情感；我告诉他我所感受到的兴奋、不安、焦虑和悲伤，以及我正在考虑的新工作的所有好处和额外津贴。他问了我一个我没有问过自己的问题，这个问题现在回想起来似乎很明显——“嗯，你为什么想离开？”

我回答说，实际上，我真的*没有*——我很高兴我的角色，感到被重视、支持和尊重。在另一家公司联系我之前，我一直有这样一个想法:至少几年内我不会离开我的工作去 T2。他帮我进一步挖掘，并问我提供给我的福利——虽然很慷慨——是否是我想要的，而不是我将要离开的。他问我是否觉得我的工作在这里完成了——不是分配给我的工作，或者对我的期望，而是我*想要*做的工作。我觉得它完整了吗？

从长远来看，我认为我的工作还没有完成。

我努力建立和重视的东西——我的人际关系、同事的尊重、产品本身和我对它的影响、我辅导和被辅导的机会、在一家使命与我的使命如此一致的公司工作的能力——所有这些加在一起，对我来说比一家大公司的知名度和更雄厚的财力更有价值。

TLDR；有些东西值得留下来。
