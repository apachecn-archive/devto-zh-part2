这一周，我们意识到我们上次在选择主持人时搞砸了，我们迅速重述了《T2》上一集中 [Max Antonucci 关于网页可访问性的文章](https://dev.to/maxwell_dev/the-web-accessibility-introduction-i-wish-i-had-4ope)。祝贺马克斯进入前 7 名，祝贺我们*拥有如此好的品味:D* 。

从那以后，我们继续关注用户[【詹姆斯】](https://dev.to/jsrn)的[项目亮点](https://www.notion.so/dgolant/dev-to-review-6-10-29-2018-84686c38092b4551a39d06248622f9c8#d732d6eb02aa434b9b166e0a091ec833)，其中他谈到了无意义的招聘人员电子邮件，以及他使用 Create React 应用程序帮助开发人员突出疯狂的体验需求的项目。

[Jess Unrein](https://dev.to/thejessleigh) 在一个星期里打击了我，我不仅失去了我的`.bash_profile`，还失去了一整台该死的电脑，还有一篇关于[让终端更加用户友好的伟大文章](https://dev.to/thejessleigh/essential-quality-of-life-terminal-improvements-4pa4)。

[Adrián Norte](https://dev.to/anortef) 凭借[“为什么 Git 上的分支是错误的”](https://dev.to/anortef/why-branching-on-git-is-wrong-1pao)赢得了我们的“本周热点帖子”，在我们之间引发了一些有趣的讨论，并在他通往巅峰的路上在评论区引发了一些讨论。

Aly Sivji 为我们带来了关于测试哲学、历史以及测试的“为什么”、“如何”和“什么”的[详尽介绍](https://dev.to/alysivji/testing-101-introduction-to-testing-5hdk)。在*敬畏*这个小伙子的大小。
