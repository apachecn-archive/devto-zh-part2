# 小心你的愿望！

> 原文：<https://dev.to/jicarrozzo/be-careful-what-you-wish-for-3oin>

在我的第一篇文章中，我列出了一个清单，我们称之为“*目标*，我想成为一名独立开发者。在这篇文章中，我列出了开始成为一名真正的自由职业者应该做的事情(我说我的道路是因为它并不适用于每个人)。从那以后，差不多一个月过去了。

出乎意料的是，我工作的公司([阿根廷公司](http://www.sisorg.com.ar))给了我两个新项目；一家俄罗斯公司给了我一个实习机会，让我成为一名本土 iOS 开发者；我的[卡波伊拉老师](https://www.instagram.com/cuecacdo)让我重新开始给孩子们上课(我当时在阿根廷)；我们开始装修公寓！我的父母决定第一次来看我，为期三周！

不，只是开玩笑。我对这一切感到非常兴奋。我是个极度活跃的人，这就像是我的戒指。良好的..也许我会在这些任务之间多留一点空间。但是当宇宙给你柠檬时..你懂的！买卡恰卡，准备一些美味的[caipinhas](https://www.thespruceeats.com/caipirinha-recipe-759290)！

所以，是时候重新组织我的目标了:

**简历:**搞定！但我还是需要在网上找一些很酷的时尚设计来偷。你知道他们怎么说:[“好的艺术家抄袭，伟大的艺术家窃取”](https://quoteinvestigator.com/2013/03/06/artists-steal/)
**Github:** 正在努力！。我上传了一些代码和想法，但仍然需要工作。
**投资组合:**它实际上连接到我的回购和我的个人网站，所以:工作...
**网址:**同上。
**JS 框架:**成功！我设法在我们公司的软件中引入了一些 Vue.js 代码。一段非常小的代码，但这只是开始！
**编程语言:**这里我绕道了。斯威夫特，这是我的新目标。而这包括使用 Mac(我是 windows 用户；第三世界的问题🤷‍♂️)并使用 xCode 作为开发接口(直到现在我只在构建过程中使用它)
**写博客:**我又来了。我有很多关于写什么的想法，但它们总是在街上出现..puf..我忘记了。但是我会试着多写一些(至少每两周一次)好的。够了！让我们开始工作吧！！
