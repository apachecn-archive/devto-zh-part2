# 工具

## 书写工具

*   [海明威 App](http://www.hemingwayapp.com/)
    *   海明威 app 是一个很棒的网站(或者 App！)这有助于把你的文章精简成直接的、没有副词的散文。我喜欢它作为一种编辑工具，尽管我对它有所保留，因为我更喜欢我的作品比海明威更华丽
*   [Swagger RESTful API 文档工具](http://swagger.io/swagger-ui/)
    *   如果你有一个 RESTful API，你应该记录它。既然你在网上记录它，为什么不让文档与你的 API 对话呢？Swagger 使得编写文档变得容易，并且通过自动为读者创建一种与 API 交互的方式，使得文档变得非常有用

## 编辑人员

*   [Atom](https://atom.io/)
    *   Atom 是一个 javascript 文本编辑器，可以在网络和桌面上使用。它有大量的插件，几乎可以作为一个完整的 IDE。我喜欢您如何将命令行集成到 Atom 中，以及它的语法突出显示是多么好
*   [记事本++](https://notepad-plus-plus.org/)
    *   Notepad++已经是我 Windows 开发的老待机了。它有标签！它有多光标编辑！它有基本的语法高亮！它打开大文件相当快！它有宏！你还能要求什么？

## 有用的 Windows 工具

*   [速度](https://www.rapidee.com/en/about)
    *   RapidEE 使得在 Windows 上管理环境变量变得容易多了。它显示了系统和用户环境变量，最重要的是，突出显示了中断的路径。
*   [腻子](https://putty.org/)
    *   目前，如果你想在 Windows 上使用 ssh，putty 是个不错的选择。希望这在未来的 Windows 版本中会有所改变。
*   [自作聪明](https://smartty.sysprogs.com/)
    *   smarTTY 是一个更友好的 Windows ssh 客户端。它允许选项卡式会话，记住你连接过的服务器，并能自动处理密钥交换。它不能做所有的事情——如果你需要用 CAC 卡进行认证，你就被 putty 卡住了。
*   [7-zip](http://www.7-zip.org/)
    *   7-zip 解压几乎任何东西
*   [Cygwin](https://www.cygwin.com/)
    *   如果你在 Windows 上，或者不在允许你使用超级可爱的 Linux 子系统的组织中，Cygwin 是下一个最好的选择。
*   [巧克力糖](https://chocolatey.org/)
    *   Windows 的缺失包管理器。它给 Windows 带来了 apt-get 或 npm 的一些活力。

## 绘图工具

*   [墨水匣](https://inkscape.org/)
    *   矢量艺术工具-有时你想要一个 svg，但你不想支付 Illustrator。Inkscape 支持你
*   [Gimp](https://www.gimp.org/)
    *   Gimp 没有 photoshop 好，但对于编辑图片、调整大小和一起 Photoshop 两张图像来说，它已经足够好了。
*   [图见](http://graphviz.org/)
    *   简单的图表！不是普通的图，而是数学上的相关节点。我的主要不满是，它并不总是按照我想要的方式对节点进行排序，但是它完成了工作。
