# 星期五爆炸#35

> 原文：<https://dev.to/horia141/friday-blast-35-4lg0>

[斯坦福大学一项为期两年的研究显示，在家工作的生产率有了惊人的提高(2018 年)](https://www.inc.com/scott-mautz/a-2-year-stanford-study-shows-astonishing-productivity-boost-of-working-from-home.html) -我可以证实这一点，我很高兴这是一个积极发展的趋势。对于更多的公司和个人来说，在个人、公司、城市和州的层面上有很多改进。

[科技面试的最新趋势——无薪作业日(2018)](https://work.qz.com/1254663/job-interviews-for-programmers-now-often-come-with-days-of-unpaid-homework/)——尽管上一篇文章的消息不错，但这一篇却很糟糕。过去，我不得不礼貌地拒绝这种大范围的采访。我*做*认为意图是好的——那就是消除了*白板面试*的一些偏见，但是实施起来相当糟糕。我真的希望科技行业不要用面试技巧从一种时尚跳到另一种时尚。

[ECMAScript 2016、2017、2018 (2018)](https://medium.freecodecamp.org/here-are-examples-of-everything-new-in-ecmascript-2016-2017-and-2018-d52fa3b5a70e) 中的一切新东西的例子——一篇大文章，功能有大有小。ECMAScript 2018 几乎占据了其中的一半，新功能和 API 改进的趋势呈上升趋势。很高兴看到 JavaScript-land 在过去几年中的巨大进步。

[理解 TypeScript type notation(2018)](http://2ality.com/2018/04/type-notation-typescript.html)——虽然官方文档相当不错，但这份文档也值得一读。它主要关注 TypeScript 中的类型语言以及 JavaScript 和 TypeScript 如何与 wrt 类型交互。鲜为人知的类型构造的好例子，如联合或元组。
