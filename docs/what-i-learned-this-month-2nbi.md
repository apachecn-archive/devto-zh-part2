# 这个月我学到了什么

> 原文：<https://dev.to/gauravchaddha1996/what-i-learned-this-month-2nbi>

前两天， [@jrop](https://dev.to/jrop) 发表了一篇文章**我这周学到的东西**。

我也开始在我的博客上写类似的每周更新。这真的让我明白我的时间都去了哪里。但是我认为在这里分享会更好。下面是我学到的一些东西和我在网上找到的一些神奇的东西:

有一天在编码时，我注意到我的项目中的一个主文件有大约 1000 行，由于类型不同，其中很多都是重复的。所以我引入了**泛型**，我的代码减少到了 300 行左右。现在逻辑清晰了，代码更容易维护，更容易解释，更容易消耗。我建议 java/android 开发人员在适当的地方使用它，在这些地方，不同类型的数据需要重复逻辑。这里有一个很好的[资源](https://www.journaldev.com/1663/java-generics-example-method-class-interface)来学习泛型。

另外，在做一些 UI 平滑工作时，我发现数学方程创造了奇迹。使用它们完成的计算产生平滑的结果，看起来很自然，非常悦目。

内存使用也是一件非常重要的事情。我使用 **android profiler** 来分析 Java 堆转储，然后通过包名过滤类来检查我的应用程序和库的内存占用。*保留大小*显示了如果这个特定对象被垃圾收集，将获得的内存量。
