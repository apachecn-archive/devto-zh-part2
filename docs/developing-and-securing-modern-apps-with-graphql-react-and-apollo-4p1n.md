# 使用 GraphQL、React 和 Apollo 开发和保护现代应用

> 原文：<https://dev.to/auth0/developing-and-securing-modern-apps-with-graphql-react-and-apollo-4p1n>

现代应用程序越来越多地由数据驱动，这导致在获取、过滤和聚合数据上花费大量时间。当使用 Apollo 平台实现时，GraphQL 降低了与数据获取相关的复杂性，并使开发人员能够利用难以置信的工具来制作灵活的 API。在本教程中，我们将探索如何使用 GraphQL、React 和 Apollo 构建一个生产就绪的应用程序。最后，我们将通过学习如何使用 Auth0 添加身份验证来结束本文。
