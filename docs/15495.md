> 原文:[https://dev . to/Apium _ hub/top-16-devo PS-blogs-you-should-be-reading-1je 2](https://dev.to/apium_hub/top-16-devops-blogs-you-should-be-reading-1je2)

在过去的 5 年里，软件开发过程发生了很大的变化。 [DevOps 工程师](https://apiumhub.com/tech-blog-barcelona/devops-technologies-benefits/)已经接管了世界，他们推动着 IT 文化的变革，专注于快速的 IT 服务交付。通过[采用敏捷](/tech-blog-barcelona/benefits-of-agile-project-manage)实现持续部署和基础设施改进。他们在开发、操作和测试之间建立协作。但是因为它是相对较新的东西，所以总有你可以学习和改进的地方！今天的文章是关于我们绝对推荐阅读的 16 个 DevOps 博客。

开发运维团队看到了比传统部署团队高得多的性能数字。在 2017 年[DevOps 状态报告](https://puppet.com/resources/whitepaper/state-of-devops-report)中，Puppet 发现高绩效 devo PS 团队具有:

*   代码部署频率提高了 46 倍
*   从提交到部署的交付时间加快了 440 倍
*   从停机中恢复的平均时间加快了 96 倍
*   变更失败率降低 5 倍

优秀的 DevOps 工程师会不断地阅读、学习和部署新的想法。现在你可以找到很多关于 DevOps 的信息，但是，你永远不能确定它是好是坏。我们今天的列表包括一些我们能找到的最受欢迎、最有用和最有趣的 DevOps 博客。实际上，我们会定期关注他们并突出有趣的信息。不管你是刚进入这个行业还是有经验，这些博客都与每个对 DevOps 感兴趣的人相关。

## 你应该阅读的 16 大 DevOps 博客

**1。 [Atlassian DevOps 博客](https://www.atlassian.com/blog/devops)**

如果你是 Atlassian 产品的粉丝——JIRA、Bitbucket、Confluence、Bamboo、Trello 等。，那么你会爱上这个博客的。他们处理涵盖从 IT 管理到合规标准的整个 DevOps 生态系统的重要主题。在那里你可能会发现这样的主题:如何选择正确的 DevOps 工具，如何建立 DevOps 文化，DevOps 技巧等。

**2。DevOps Cube**

devo PS Cube 是网上最好的 devo PS 博客之一。这是一个博客，里面充满了很棒的教程、技巧、例子和趋势。此外，您还会找到与最佳 DevOps 工具相关的文章，如:Docker、Jenkins、Google Cloud 等等。这些文章既适合初学者，也适合有经验的 DevOps 工程师。

**3。[DevOps.com](https://devops.com/)T3
T5】DevOps.com 就像 DevOps 社区的首页。这完全是关于德沃普斯的。您可以在 DevOps 生态系统上找到各种类型的评论，包括新闻、产品评论、观点文章、策略、最佳实践、案例研究、事件和专业人士的见解。他们还发布了一个名为 DevOps Chat 的每周播客，由传奇 IT 专业人士 Alan Shimel 主持。**

如果你想深入杂草，DevOps.com 有足够的资源让你去做。他们每月举办几次网络研讨会，涵盖具体的开发运维策略和战略，如“企业级业务的自动化部署”和“设置私有云”。他们也有大量的电子书和 pdf 供你下载。

**4。[Docker Blog](https://blog.docker.com/)**

Docker 是一个流行的容器工具，它定期为开发人员更新他们的博客。博客上几乎所有的信息都是特定于产品的，所以它对任何使用 Docker 的人来说都特别有价值。

**5。DZone devo PS**

DZone 是一个技术网站和教育中心，拥有大量的 IT 敏捷信息&。他们每天在多个“区域”发表几篇文章，比如 DevOps、Cloud、Java、Mobile、IOT 等。他们的 DevOps 部分有很多非常有用的资源，你可以在作者中找到著名的 DevOps 工程师。在本节中，您可以找到来自不同行业的工具和实践的理论到实践的实现。如果你在 DevOps 的世界里，它基本上有你需要的一切。

**6。 [Apiumhub 博客](https://apiumhub.com/tech-blog-barcelona/category/agile-web-and-mobile-development/)**

快速发展的科技博客，其中有不同的板块。在那里你可以找到关于自动化技术、DevOps 工具、Jenkins、Docker、持续集成等更多的信息。他们的团队中有 DevOps 专家，他们根据自己的经验定期发布文章。

7 .**。 [IT 革命](https://itrevolution.com/)**

一个非常好的博客，在这里你基本上可以找到任何一种与 DevOps 相关的信息:事件、项目、提示、新闻、社区等。

**8。[Infoworld](https://www.infoworld.com/)**

Network World 是一本涵盖数据、网络、云计算等诸多领域的在线杂志。他们有很多与 DevOps 相关的有趣文章。这个博客将让你了解最新的行业新闻，并带你深入了解技术诀窍和建议。

**9。[Pivotal DevOps](https://content.pivotal.io/devops)**

如果你刚刚开始探索 devo PS 的世界，这是一个不错的目的地。他们有几个高级主题的易读指南，比如 DevOps、敏捷和容器。

10。[The Register DevOps](https://www.theregister.co.uk/devops/)

我注册博客的主要话题之一就是 devo PS，所以每天都有大量有用的资料可供查阅。

**11。这个博客对于想要了解更多的开发者来说是完美的。NET、应用程序性能监控、DevOps 和运行更敏捷的业务。它还涵盖了 DevOps 工程师、软件开发人员、首席信息官和首席技术官的职业建议。**

**12。[敏捷管理员](https://theagileadmin.com/)**

敏捷管理员博客是一个很好的选择。他们所有的博客都是关于 DevOps 的，他们几乎拥有了成为 DevOps 专业人士所需的一切。

13。[DevOs Guys](https://www.devopsguys.com/blog/)

DevOps Guys 博客是关于 devo PS 文化及其周围的一切。这个博客是由史蒂夫·泰尔(负责运营的家伙)和詹姆斯·史密斯(负责开发的家伙)运营的，他们一起创建了非常有趣的帖子。这个博客很有趣，它了解 DevOps 不断变化的文化和行业。有很多关于开发运维机制的重点，无论是工具还是最佳实践。

**14。[XebiaLabs DevOps](https://blog.xebialabs.com/category/solutions/devops/)**

XebiaLabs 博客围绕行业思想领袖和他们关于持续交付和 devo PS 世界正在发生的事情的讨论而构建。每周发布大量内容这是一个博客，在这里您可以找到关于 DevOps 的深入技术讨论和正在进行的博客系列。这些作者讨论了一些棘手的问题，提出了一些有趣的观点，会让你思考好几天。

15。TechTarget Devops

TechTarget 的 Devops 部分有一个与 devo PS 相关的博客帖子的精选列表。在那里，您可以找到由全球 DevOps 专家撰写的文章。

16。 [BMC DevOps 博客](https://www.bmc.com/blogs/categories/devops/)

该博客网站提供各种高质量的文章，从信息性的到前沿性的都有，特别强调企业业务方面。这里涉及的一些关键主题是持续交付&部署、一切如代码、物联网、容器化和最佳实践。
