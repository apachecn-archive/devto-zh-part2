# 部落——寻找归属感

> 原文:[https://dev.to/pavsaund/tribes-寻找归属-4d43](https://dev.to/pavsaund/tribes--a-search-for-belonging-4d43)

[![](../Images/5976afa9e1cea5c0ebbfc7cff628f56f.png)T2】](https://cdn-images-1.medium.com/max/1024/1*gHxQaC66S9c5Oe51laYYKg.jpeg)

找到你的部落并不容易。这是一个充满失误的漫长旅程。你可能一辈子都是一个部落的一部分。为了安全起见，你可能会四处寻找其他部落，然后潜回你原来的部落。其他时候你需要找到另一个属于你自己的。有时候你需要和你身边最亲近的人开始自己的生活。其他时候，你只需要开始，并希望其他人会跟随。

### [](#tribes)部落

但是，我说的部落是什么意思呢？在这种情况下，我把它作为一个让你有归属感的社区。最初，部落是根据邻近程度来定义的，即你所属的土地。所以改变部落是一种物理行为。从一个地方搬到另一个地方。经历各种仪式和牺牲来离开你原来的部落并被你的新部落接受。摩擦很大。

在今天的社会中，对某块土地的归属感不再是我们部落的定义。我们用其他方式定义它，如:种族、国籍、性别、信仰、社区、工作场所、爱好等。有些是层级式的，比如你的团队，它位于你所在组织的部门内，而有些是虚拟的，比如社交媒体团体、论坛、兴趣、编程语言。

在我们的数字时代，部落之间转换的摩擦要少得多。你可以找到一个虚拟部落，在那里转换可以像加入另一个团体一样简单。即使你改变了你的现实世界，你也可以在现实世界中移动，并且仍然留在同一个虚拟部落中。

### [](#a-sense-of-belonging)归属感

随着我们的成长和成熟，我们都会经历自己版本的[《英雄之旅》](https://en.wikipedia.org/wiki/Hero%27s_journey)。在生活中穿梭，寻找归属。

我的家族部落对我来说一直很重要，它是我所属的最坚固的部落。我们共同的价值观定义了我的信仰和价值体系的核心，也是我看待世界的视角。

另一个让我有成就感的是和有爱心的人一起做有意义的工作。努力让这个世界变得更好。这是迄今为止无意识地引导我的东西，现在我开始更多地意识到并采取行动。

### [](#the-search)搜索

寻找和改变部落变得前所未有的容易。也比以往任何时候都更容易在搜索中分心。我们能做的最好的事就是跟随我们的心，而不是满足。随着我们的成长，我们的需求也在增长，我认为找到一个能让你充分发挥潜力的部落是值得寻找的。

随着我越来越清楚我的旅程和对我来说什么是重要的，我也越来越清楚我想从我的部落得到什么。我确信这将随着我的成长而演变和改变，我确信没有一个部落会是完美的匹配。我知道，当我看到更接近我自己价值观的东西时，我需要采取行动。不然我怎么知道这是不是我需要的？

> *“如果你还没找到，那就继续找。不要和解。就像所有关于心灵的事情一样，当你找到它的时候你会知道的。而且，就像任何伟大的关系一样，随着时间的推移，它会变得越来越好。”―史蒂夫·乔布斯*

所以，如果你找到了你的部落——珍惜它，滋养它，并认为自己是幸运的。如果不是，通过摩擦工作，并且**不稳定**。
