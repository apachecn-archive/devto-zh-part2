# 有人比德瓦特快吗？dbForge Studio 已经支持 MySQL v8.0

> 原文:[https://dev . to/devart software/is-any one-fast-than-devart-support-for-MySQL-v80-is-always-in-dbforge-studio-o0m](https://dev.to/devartsoftware/is-anyone-faster-than-devart-support-for-mysql-v80-is-already-in-dbforge-studio-o0m)

数据库工具和数据连接解决方案的捷克软件供应商 Devart 发布了一个显著改进的通用 MySQL 和 MariaDB GUI 工具版本，即 dbForge Studio for MySQL v8.0。继 Oracle 最近更新 MySQL Server 之后，dbForge Studio 现在支持最新版本的 MySQL。

以下是在更新的 dbForge Studio for MySQL 中实现的最突出的改进:

1)由于支持 MySQL 8.0，dbForge Studio 现在为用户提供了在最新服务器版本上使用所有便利功能开发、管理和管理 MySQL 数据库的能力。

2) dbForge Studio for MySQL 继续拓宽兼容性选项，现在提供与最新的 MariaDB v10.3 的连接和工作。

3)对腾讯云用户来说是个好消息——新的 dbForge Studio for MySQL v8.0 允许毫不费力地连接到这个专为在云端部署和使用 MySQL 数据库而设计的云服务。

4)为了更方便地使用代码，dbForge Studio for MySQL v8.0 现在使用快捷菜单或“Ctrl+F12”组合键在 SQL 编辑器中的关键字之间提供无缝导航。

在[https://www.devart.com/dbforge/mysql/studio/](https://www.devart.com/dbforge/mysql/studio/)深入了解 dbForge Studio for MySQL v8.0 的其他有用特性。
