# API 初学者的第一步

> 原文：<https://dev.to/tallmanlaw/beginners-first-steps-with-apis-2p15>

第三方应用程序接口(API)是构建软件应用程序的重要工具。熟悉用于发出请求和接收信息的协议的一个简单方法是选择一个个人感兴趣的 API，并对其进行试验。一个简单的搜索引擎查询产生了一个很好的起点。

在选择了感兴趣的 API 后，我建立了一个个人帐户，并获得了一个 API 密钥。我构建了一个简单的 html，让信息在浏览器中返回给我。
