# 介绍全新的 dbForge Studio for PostgreSQL，这是一个用于 PostgreSQL 数据库的强大 IDE

> 原文：<https://dev.to/devartsoftware/introducing-brand-new-dbforge-studio-for-postgresql-a-powerful-ide-for-working-with-postgresql-databases-38eb>

Devart 宣布了 dbForge Studio for PostgreSQL 的第一个版本，它允许在 PostgreSQL 中管理和开发数据库对象。这个新的 IDE 提供了许多强大的功能，如代码完成、对象资源管理器和数据编辑器，这些功能可以帮助用户创建、开发和执行查询，并在一个方便且用户友好的界面中编辑和调整代码以满足他们的要求。

Devart 是一家为数据库管理和开发提供数据库连接解决方案和工具的捷克软件提供商，它宣布了 dbForge Studio for PostgreSQL 的第一个版本。使用这个工具，用户可以查看他们感兴趣的 PostgreSQL 数据库对象的所有必需信息。在第一个版本中，dbForge Studio for PostgreSQL 提供了以下关键特性:

1)代码完成使用户能够在编写 SQL 查询时节省时间并提高代码质量，同时借助于许多有用的功能，例如:
列表成员
参数信息
快速信息
完整单词

2)具有可扩展对象的对象资源管理器允许用户在对象树中导航，找到任何感兴趣的 PostgreSQL 对象，并执行以下任务:
从表中检索数据以在单个 SQL 文档中呈现
截断表
选择属性以查看关于对象的特定信息

3)数据编辑器简化了编辑 PostgreSQL 表格数据的过程，提供了许多方便的功能:
类似于 Google Sheets 和 MS Excel 或在卡片视图模式下编辑数据
管理表格的设置，如调整列宽
默认设置分页或自动搜索模式
使用丰富的字体和大小设置行和单元格的颜色和格式
提供预定义的数据类型格式，以便用户可以选择最适合他们需求的格式
