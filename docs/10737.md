# 您的组织在 DevOps 成熟度模型中处于什么位置？

> 原文:[https://dev . to/pavanbelagatti/where-do-your-organization-fit-in-this-devo PS-maturity-model-a7b](https://dev.to/pavanbelagatti/where-do-your-organization-fit-in-this-devops-maturity-model-a7b)

DevOps 是一种文化，文化不是你可以在一两天内实施和改变的东西。会有反对，会有延迟，车队之间会有小摩擦，我的重点是，这需要时间。

然而，只有当你的管理不善、协调不力，或者当管理层无法传达采用 DevOps 文化的好处，或者可能因为缺乏预见成功的能力时，上述所有事情才会发生。

除了管理层，团队也应该携手拥抱这种变化。一开始会很困惑，但最终是值得的。你将设定目标、原则和相应的工作人员。
