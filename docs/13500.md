# [](#the-developers-digestive-system)开发者文摘...ive 系统

"我要一份大的 6 号套餐，外加一份奶酪薯条和一大杯苏打水。"...嗯，我的早餐并不是这样的。我通常在早上吃一些燕麦片加少量牛奶，或者早餐吃一些玉米/麦麸片。然后，我会用一些生姜和柠檬蜂蜜茶或绿茶把这些都冲下去。我一直在努力克制自己不喝咖啡。毕竟，咖啡是软件开发人员的燃料。我意识到绿茶实际上有助于我保持清醒。这是我早上的日常饮食。

# [](#whats-for-lunch)午饭吃什么？

我在上午 11:30 吃午饭。我午餐通常带家常菜，平均来说，有一些蔬菜和肉。如果我没有带家常菜，我和我的同事会叫一些外卖。外卖可以从汉堡到比萨饼，或者只是一些馅饼。

# [](#dinner-at-tiffanys)蒂芙尼的晚餐

这就是事情变得奇怪的地方。到了晚饭时间，我通常没有胃口。不，我妈妈厨艺不差。我通常下班回家，洗个热水澡，做些瑜伽，喝杯茶，然后跳上我的床，读本书直到我昏过去。还是没胃口。即使我强迫自己吃东西，我也只能吃两勺，就觉得饱了。

# [](#the-snackdown)大快朵颐

我不喜欢吃零食。也许我白天会喝一杯酸奶，吃一个水果，但这是我吃零食的极限。

# [](#water-water-and-more-water)水，水，更多的水

我们有时会因为写代码而分心，以至于忘记喝水。我尽我最大的努力让我的身体每天至少摄入 1.5 升的水。

# [](#puree-all-the-things)原浆所有的东西

我是最懒的咀嚼者(这个词合适吗？)在那里，我知道关闭。我更喜欢吃温暖柔软的食物。我通常在吃面包前先把它加热。很奇怪，我知道。我不吃松脆的食物，声音很大，吃在嘴里感觉不舒服。如果可以的话，我会把食物做成浓汤，然后一饮而尽。
