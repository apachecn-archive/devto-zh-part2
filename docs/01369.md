Azure 容器实例提供了一种在 Azure 中运行单个容器的快速而简单的方法。没有虚拟机的麻烦，没有要管理的操作系统，也没有流程编排。这项服务最适合小型应用程序。您可以使用 bash (CLI)、Azure 门户或 PowerShell 来部署您的容器。更多信息，请尝试这篇关于创建部署容器的教程。
