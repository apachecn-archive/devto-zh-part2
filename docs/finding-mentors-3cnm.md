# 寻找导师

> 原文：<https://dev.to/zellwk/finding-mentors-3cnm>

[https://www.youtube.com/embed/vn6Qt70wBrs](https://www.youtube.com/embed/vn6Qt70wBrs)

当我们想到导师时，我们通常会想到人。

但这是错误的。

因为导师不一定是人。

## 创业时最好的导师

书籍和课程是你起步时最好的导师。

为什么？因为人们把时间和精力倾注在制作书籍和课程上。你可以从阅读他们的书和课程中获得比与制作这本书或课程的人交谈更多的东西。

你总是想从一本书或一门课程开始。

免费文章可以帮助。你可以使用它们。但是使用免费文章的缺点是你需要知道首先要学什么，接下来要学什么。你需要规划好自己的旅程。

随着你越来越高级，当你有书本、课程甚至网上文章没有回答的问题时，你就想问人。

## 找人帮你解围

有很多方法可以找人回答你的问题。

### 加入一个围绕你购买的书籍或课程的社区

我推荐的第一种方法是，看看你买的书或课程周围有没有社区。

这个社区将是你提问的最好地方，因为社区里的人正在学习和你一样的东西。语境多了，得到优质答案的几率也就大了。

创建课程的人也可能会在社区中闲逛来回答问题。

无耻之徒:当我创建 Learn JavaScript 时，我创建了一个社区来支持这个课程。我在社区里闲逛来回答问题。

如果你有一个关于 JavaScript 的问题，你几乎可以保证这个问题会由一个学生或者我来回答。

如果你想学 JavaScript，可以去[https://learn JavaScript . today](https://learnjavascript.today)了解更多课程内容。

### 前往您附近的活动

找人帮忙的第二种方法是去参加你所在地区的活动。当我说事件时，我指的是聚会和会议。

你可以找到志同道合的人，并和他们成为朋友。这些人比你高一级还是低一级都无所谓。你仍然可以和他们成为朋友。

重要的是**把这些人当作朋友和导师**。我一会儿会谈到为什么它很重要。

### 与你尊敬的人交谈

第三种方法是和你尊敬的人交谈。

通常，你可以在这些人的博客中找到他们。您可以订阅他们的电子邮件简讯，并通过电子邮件与他们交流。

有时你可以通过 twitter 或他们所在的其他社交媒体平台与他们交谈。

再次像朋友和导师一样对待这些人。

### 在社交媒体上提出你的问题

第四种方法是在社交媒体上发布你的问题。

希望有人会回答，但这取决于你有多少粉丝，以及你是否足够幸运。

## 为什么你应该把人当成朋友和导师

如果你把人们当作唯一的导师，你就把他们放在了一个基座上。你很难和他们成为朋友，因为你觉得他们总是比你高一级。你觉得自己没什么可补充的。

如果你把人既当朋友又当导师，你就不用担心专业知识了。反正朋友是不同层次的。

如果你只把别人当朋友，你可能会认为从他们身上学不到什么。结果是，你没有从他们身上学到东西。

但是你确实可以从比你水平高的人和比你水平低的人那里学到东西。可以学到很多东西。由你来决定你能学到什么。

## 不要到处寻求免费的指导

他们可能是你可以获得免费指导的社团，但他们是有限的。你可能进不去。

免费导师制不起作用，因为人们重视自己的时间。人们花时间和你在一起，需要得到回报。

**有时候，这个奖励意味着金钱**。那样的话，你可以花钱请人咨询。如果你付钱给他们，你是在尊重他们的专业知识、时间和精力。

获得咨询几乎总是值得的，因为你学到的东西比你花的钱要多得多。显然，在你打电话咨询之前，确保你先浏览了书籍和课程。

奖励并不总是金钱。有时可能是你和他们建立的关系。

朋友不谈钱。我们只是互相帮助。如果你有更多的朋友，你就有更大的可能性问一个问题，人们会免费帮助你。
