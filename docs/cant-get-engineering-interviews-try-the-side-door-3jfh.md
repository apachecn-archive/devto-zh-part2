# 得不到工程面试？试试侧门。

> 原文：<https://dev.to/lynnetye/cant-get-engineering-interviews-try-the-side-door-3jfh>

如果你正在申请新的工程工作，并且正在努力进入这个领域，考虑寻找其他途径。

想象一下，一大群人在一栋大楼前，推推搡搡地从大门进去。一些人，厌倦了暴民，决定探索的前提，并遇到了侧门。他们敲门，有人开门，然后走进来。也用那个侧门不是很好吗？

## 创业的许多方式。

在一大堆申请中，除了简单地“变得更好”或“做更多令人印象深刻的事情”之外，没有太多突出的方法大多数人努力工作，通过更多的工作经验和更多的项目来巩固他们的简历和投资组合。其他人耐心地等待轮到他们。有些人强行向数百家公司申请，直到其中一家最终同意。

如果你不直接和其他人竞争，而是把你的精力放在寻找一个旁门上，会怎么样？

以下是我个人或看到其他工程师为了插队、获得面试机会和在科技创业公司找到工作而做的几件事:

## 使用产品。

许多 B2C 公司直接从他们的用户群中招聘。拥有消费产品的初创公司希望听取用户的意见，尤其是那些对如何提高 UX、性能时间或修复遇到的错误有想法的用户。积极主动地报告问题。如果你有解决方案，你可以直接和他们的工程师谈。瞧啊。你插队了。

许多 B2B 公司都有你可以使用的产品，尤其是如果你正在做自己的副业或者你的前雇主是你的客户。

## 做个公关。

当然，你可以构建自己的项目并使其开源，但你也可以找到有[开源项目](https://www.keyvalues.com/values/open-source-contributor)的公司并为之做出贡献。制作一个 PR，并与当前的工程师一起维护它。你在展示你的能力和兴趣，甚至不需要面试。

## 推文。

不到一年前，我不情愿地开始真正使用 Twitter，我无法相信我错过了多少。软件工程社区在 Twitter 上非常活跃，所以这是一个任何人都可以轻松获得的巨大优势。在 Twitter 上与你梦想中的公司的现有团队成员交流。回答他们提出的问题。评论他们发布的文章。参与对话。

不知道跟着谁？先从这篇[文章](https://www.keyvalues.com/blog/how-engineers-can-stand-out-from-the-applicant-pool)中引用的技术创始人说起。

有趣的事实:我在 Twitter 上得到了我的第二份网络开发合同。

## 亲自见面。

为什么不当面交谈呢？许多创业公司举办聚会、黑客马拉松和/或研讨会，有时在他们自己的办公室里。参加他们。与每个人交谈，尤其是公司的现任工程师。你不仅可以随便问一些关于 X 公司生活的问题，而且在很多情况下，他们会因为所谓的内部推荐奖金而推荐你。

最坏的情况是:你会遇到各种各样的新人，他们会帮助你拓展人脉。

## 通过[键值](https://www.keyvalues.com)提问。

最后，如果你对任何关于[关键价值](https://www.keyvalues.com)的工程团队感兴趣，我已经在每个公司的简介页面上为你创建了一个小侧门。您可以在页面的左下角为当前的团队成员留下有深度的问题，以及您的电子邮件地址，以便他们能够取得联系。

外面有很多侧门，我喜欢发现它们。所以我问我的开发伙伴:你知道什么我们不知道的？😜
