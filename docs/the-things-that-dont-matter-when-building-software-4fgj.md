# 构建软件时不重要的事情

> 原文：<https://dev.to/pbeekums/the-things-that-dont-matter-when-building-software-4fgj>

我喜欢开玩笑说，有一天我会获得经济学博士学位，并把我的论文作为标签与空间之争的全球 GDP 成本。关于这一点，硅谷有一个很棒的插曲。不幸的是，虽然该剧确实夸大了一些喜剧效果，但该剧的精彩之处在于它切中要害。

或者他们可以缩进他们的代码，如下所示: [![](../Images/21030f7475d401b9e5f8a391145ce0fe.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--9VZJtU4m--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://blog.professorbeekums.com/img/2018/usingspaces.png)

你能看出区别吗？大概不会！我甚至不确定为什么我要在第二张截图中输入空格。

尽管一些开发人员喜欢指出这些细微差别，但制表符和空格之间的差别是如此之小，以至于几乎不值得谈论。然而，我们确实在谈论它。我们谈了好几个小时。每次去新公司或者见新的开发人员，我们都会谈到它。我们把时间花在这上面，而这些时间本可以更好地用于构建给人们带来真正价值的软件。

制表符和空格也不是唯一没有意义的程序员争论。在一行的末尾加上花括号: [![](../Images/b214913dcf4f6188f813d267d9c76116.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--oqXlVetI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://blog.professorbeekums.com/img/2018/curlybraceendofline.png) vs 换行: [![](../Images/3409bd2367ab9e06cc048f2f9dce0352.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Il_2qCuu--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://blog.professorbeekums.com/img/2018/curlybracenewline.png)

还有就是我们是否使用蛇套管: [![](../Images/25a3237b77f7e69725252c199e524015.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--IuXplAqN--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://blog.professorbeekums.com/img/2018/snakecase.png)

或者骆驼肠衣: [![](../Images/8de272fce762fb29c8c0145b98e5ea43.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--znuk-Qbw--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://blog.professorbeekums.com/img/2018/camelcase.png)

如果你试图找出这些事情的原因，你会花很长时间。它们看起来像是微不足道的差异，因为它们是微不足道的差异。

有时候，我们参与这些辩论仅仅是因为我们知道它们很可笑，很有趣。但是很多时候，人们真的会认真对待这些辩论。当这种情况发生时，其影响绝不是无害的。时间被浪费，要么导致项目失败，要么导致工作时间过长。

我也不是无辜的一方。我年轻时的大部分时间都在对所有这些事情保持警惕中度过。我不能代表所有的开发者，但是我认为我的问题是我太专注于为了构建软件而构建软件。那种态度很容易陷入迂腐。在某个时候，我把注意力转移到为了软件被使用而构建软件上。这种观点帮助我摆脱了这些无意义的争论，这也帮助我专注于更重要的软件质量度量:更少的错误，更容易的调试，更容易的阅读，等等。

这也不足为奇。与硅谷的电视节目不同，我发现我所知道的最优秀的开发人员并不关心这些事情。他们有更好的事情可以花精力去做。

**附言**我肯定用制表符。每个人都可以根据自己的意愿将制表符放大或缩小。选项卡符合个人偏好。空间迫使每个人看到相同的东西。标签是自由的特征。

空格代表暴政和压迫。
