# 然而，凯蒂·拉瓦利编码

> 原文：<https://dev.to/katylava/nevertheless-katy-lavallee-coded--4cpp>

## 我是如何开始的

从小到大，我总是被告知我很聪明。我从来不是班上最聪明的孩子，但至少是前 10%，而且很容易达到。尽管我没有为之努力，但我以此为荣。

最终上学确实变得困难了。由于焦虑和抑郁，我逃了太多的课，当我在那里的时候，我趴在桌子上睡着了。我不及格，但不是因为我不聪明。我仍然很聪明，并以此为荣。

这种骄傲，虽然不是自然产生的，却让我“擅长计算机”。我一直以为我会成为某种艺术家——画家、音乐家、摄影师——直到我发现我能比其他人更好地理解软件。用电脑工作是自然而然的，而艺术是艰苦的工作。我不习惯艰苦的工作。而我是“聪明”，不是“有创造力”。

这是一个奇怪的开始方式，不是吗？...陷入我热爱的工作中，做错每一件事。幸运的是，我还热衷于按照自己的意愿改造这台机器，否则当它不可避免地变得具有挑战性时，我永远也不能坚持下去。构建软件让我觉得自己很强大，而且是一种冲劲。

## 我是如何进步的

由于精神疾病，我相当大器晚成。从“我要点击所有的东西，对这台电脑了如指掌”到“我要用代码控制这台电脑”，用了很长时间。我直到 26 岁才真正写代码，这完全救了我。我不知道我会在哪里，如果我没有像“是的，我可以建立你妈妈的牧场网站”和“我在那里写同样的 HTML 必须有一个更好的方法”和“如果她能每年用某种形式添加新的可爱的小马，那会好得多”。

不管怎样，这个故事的其余部分有点无聊。我朋友妈妈的牧场网站给了我一些可以写进简历的东西，这让我得到了一份初级开发人员的工作，我在那里呆了很长时间...等等等等...经营自己的生意失败了...等等等等...“你不是网络开发人员，你是软件工程师，我们会付给你更多的钱”...等等等等...微服务。

好吧，是的，我真的很幸运能一直为那些认可我的能力和潜力的优秀员工工作。我希望每个人都能有那个无聊的故事。

## 我作为女人的故事

在成长的过程中，我也为自己是唯一一个可以和男生在一起的女生而自豪。我喜欢科学和数学，我穿牛仔裤和 t 恤衫，对性别歧视毫不在意。我觉得自己很特别。我“不像其他女孩”，因此更加有趣和与众不同。我有性别歧视。

因此，一旦我找到了我的道路，我就不会有我不能或不应该编码的想法。所以，我能给其他女人的唯一建议就是把那些想法推开——它们不是真的。

我也没什么不同。我明白了有很多像我一样的女人。有很多女性出于同样的原因开始编程，她们被同样的动机所驱使，她们肯定和她们的男性同行一样聪明。我不是异类。
