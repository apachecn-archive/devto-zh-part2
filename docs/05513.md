# 介绍我们 2018 年 9 月的赞助商

> 原文:[https://dev . to/dev team/introducing-our-June-2018-proposers-5bp 8](https://dev.to/devteam/introducing-our-september-2018-sponsors-5bp8)

我们出色的赞助商对 [dev.to](https://dev.to) 的健康发展至关重要，能与对生态系统做出如此大贡献的公司合作真是太棒了。

这三个开发者平台都是各自领域的领导者，如果你还没有这样做的话，我强烈建议你去看看他们的产品。说真的，现在花几分钟时间浏览一下他们的产品指南和文档。

## [](#digital-ocean)数字海洋

[数字海洋](https://try.digitalocean.com/performance/?utm_source=devto&utm_medium=display&utm_campaign=Devto_2018_Brand)是一个备受喜爱的云计算平台。我总是对数字海洋核心产品的良好构建*和良好记录*印象深刻。说起来容易做起来难，他们已经能够保持领先。

迄今为止，数字海洋一直是我们最长期的支持者，我们非常感谢他们的一贯支持。下个月我们将在纽约举办一场 [Hacktoberfest](https://hacktoberfest.digitalocean.com/) 活动——敬请关注更多细节！

## [](#twilio)Twilio

自从[发展到](https://dev.to/)以来，Twilio 一直是我们的好朋友。我们是他们 API 的用户，我们团队的成员总是很乐意参与他们的社区活动。有他们做赞助真是太好了！

我个人可以说，我们选择 Twilio Video 是因为它充分利用了本地对等浏览器技术，同时还为包装在同一 API 中的必要用例提供了 Twilio 的基础架构。文档和示例组织得很好，启动和运行是同步的。

## [](#qlik-playground)Qlik 游乐场

[Qlik Playground](http://playground.qlik.com/?utm_medium=direct_buy&utm_source=devto&utm_campaign=worklogo) 是一个编程环境，允许您使用、探索和快速测试使用 Qlik 技术的应用程序。Qlik 是一个数据分析平台，几乎在每个行业都用于摄取、可视化和分析。

如果没有像 Qlik Playground 这样的开发工具，针对这些类型的产品进行开发可能会很麻烦。正是这个工具让我们的团队将 Qlik 作为一个可能的解决方案，因为我们将 [dev .扩展到](https://dev.to/)。
