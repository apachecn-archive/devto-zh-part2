下面是在 Python 中作为后台线程运行类方法的一小段代码。run()方法永远做一些工作，在这个用例中，您希望它在后台做这些工作。而应用程序的其余部分继续工作。

import threading

    def __init__(self, interval=1):
        self.interval = interval

        thread = threading.Thread(target=self.run, args=())
        thread.daemon = True
        thread.start()
