# 本地覆盖

> 原文：<https://dev.to/sait/local-overrides-with--styles-in-chrome-dev-tools-7of>

你看到了吗，在 chrome 中，我们可以选择覆盖本地样式，如果我们重新加载页面，它仍然会显示出来。

让我们看看如何在 chrome 66 中使用本地覆盖。

我正在使用样本网格文件，默认情况下，我的网格有黑色和蓝色。

我已经通过选择项目目录启用了本地覆盖。

[![](../Images/8888a9cffb286c09a84a6e9ff8e89218.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--z2sjJoUl--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/12cobcbdgwc2cmvdg6do.gif)

你看到上面的图片了吗？当我刷新浏览器时，我改变了蓝色和黑色，我们的改变没有丢失。
