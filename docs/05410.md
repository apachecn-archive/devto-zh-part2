## [](#key-vue-concepts)关键 Vue 概念

本文讨论了以下 Vue 概念:

*   用`:style`(对`v-bind:style`的简写)进行样式绑定
*   `v-model`在表单输入上创建双向数据绑定的指令
*   计算属性
*   受关注的属性
*   `mounted`生命周期挂钩
