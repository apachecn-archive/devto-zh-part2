# 每种语言代表什么？

> 原文：<https://dev.to/cathodion/what-is-each-language-for-21om>

理论上，任何现代编程语言都可以用于几乎任何任务。那么，为什么会有这么多呢？也许有趣的是，在某个时候，一些语言开始接管利基市场。在某种程度上，可能存在一个通用的编程语言翻译器，因此每个人都可以用自己喜欢的语言读写任何代码。不过，现在看来，有这么多编程语言的一个原因是它们各自的社区喜欢关注不同的问题。

下面是一些似乎占据了某些编程语言社区的问题:

*   Ruby :做开发和思考开发的最好方式是什么？
*   Python:我们怎样才能让我们的代码尽可能的具有可读性？
*   Java :我们如何构建我们的代码，以便在企业范围内协同工作？
*   Clojure :我们如何重新思考什么是数据，以及我们的软件如何与数据相关联？
*   JavaScript :我们如何在网络范围内从彼此的工作中获益？

你同意这些问题是这些编程语言社区所关注的吗？其他的是什么？
