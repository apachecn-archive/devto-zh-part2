# 表头-顶层规则

> 我在下面分解出的主题包括:**生产力**、**人际技巧**、**销售&心理学**、**科技领导力**、**生活&职业哲学**以及**心智模型和系统思维**。将这些相互关联的东西分开是非常人为的，所以我将它们中最重要、最常见的线索提取到这个顶级标题部分。

**微:经营你生活的战术规则**

*   微笑！它让你，不仅仅是你周围的人，感觉更好。😄
*   学习比挣钱更重要。
*   你的大脑不是一个存储设备。有一个写东西的系统(比如从 [dev .到](http://dev.to)？)并使其可组织和可搜索。
*   发挥你的长处，根据别人的长处雇佣他们并与他们一起工作。
*   记住讲故事的力量。一个好的故事比一个好的论点更有力量。
*   为你工作中的艺术性感到自豪，并对结果慷慨大方。人们渴望只有你能提供的真实。
*   精通是通过工匠心态实现的，专注于稀有和有价值的技能，以非凡的最终目标下小赌注，尽可能接近。(好到让他们无法忽视你
*   永远寻找你的人生目标，当你找到它时，让一切都与它一致。作为一家公司，也要让员工和客户为你的目的服务。当你发现自己处于[流](https://en.wikipedia.org/wiki/Flow_(psychology))中，这是一个很大的暗示。
*   **流量决定一切**。寻找能让你处于流动状态的工作。围绕优化你的心流时间来安排你的环境、日程和生活。你最好的工作将在这里完成，世界将为此奖励你。忽略那些旨在打断你心流的噪音。摧毁，消除，熄灭，用火杀死那些假装紧急的不重要的东西(比如通知)。
*   了解人们想要什么。关于人们想要什么，有很多很多的理论，但三大理论是对激励、动机和魅力的研究。

宏观:塑造你世界观的大实话

*   技术已经深刻地改变了我们社会和经济的许多规则。
    *   我们时代的主流媒体定义了我们对真理的看法。

# 生产力

*   你的大脑不是一个存储设备。将想法和任务放在一个真实的中心来源中，例如一个清单或一个 Trello 板。现代人类的知识太庞大了，无法在我们的大脑中储存、组织和应用。
*   抛弃那些对他们有用的东西。多说不。
*   不要一心多用。多任务不是生产力。
*   你的意志力有限。
*   积极主动，掌控全局。
*   从最终目标开始:有一个使命、愿景和具体的目标。
*   把重要的事情放在第一位。急！=重要；先做紧急和重要的事情，然后做重要的非紧急的事情。避免紧急的不重要的事情。
*   磨利锯子，充电。确保平衡个人生活和职业生活。
*   腾出空间，定义你的工作空间。定义你的工作时间和空闲时间。定义你的在线时间和离线时间。界限帮助你做深层次的工作。
*   做好项目失败前的设想。和你未来的自己对话，用“哦哦”效应来激励你完成一个强有力的终点。
*   使用简短的清单来防止可避免的错误，并为团队安排交流。

# 人际交往能力

*   微笑！
*   真诚待人。
*   对别人感兴趣，用他们的名字。让他们看起来不错。当他们提出建议时，要说“是，而且”,而不是马上拒绝。
*   立即承认错误。容易被拒绝。
*   想双赢，求正和。
*   先寻求理解，再寻求被理解。感同身受，倾听。询问、定义并框定问题。
*   协同增效；通过积极的团队合作结合人们的力量。
*   **艰难的对话**:留意你想要什么，你想要避免什么。让他们有安全感。关注尊重&共同的目标。将批评与表扬进行对比。向你表示关心。意译以确保理解。把事实搞清楚。确保你有一个达成共识的解决方案或需要决定的人。
*   **常见沟通错误**:过分强调内容，不真实，毫无准备，没有自知之明，重复现状。
*   **沟通策略**:眼神交流、姿势、使用手势、改变声音、停顿、表达观点
*   **推销人员**:缩短你的推销。让他们参与这个过程。问个问题。押韵。调用目的。
*   练习情商的 5 个方面:自我意识、自我控制、动机、同理心和影响力。

# 销售&心理

*   我们为自己的行为辩护，而不是相反。
*   40%的行为是无意识的习惯。习惯坚持是因为你渴望回报。重新引导你的渴望，不要抗拒它们。
*   习惯是通过一个触发->行动->奖励->投资的循环形成的。发现公司对你这样做，并负责任地使用它。
*   建立 Keystone 习惯，帮助你获得一系列的小胜，对你的意志力和习惯养成有积极的溢出效应。
*   当心系统 1 无意识缺陷(启动、光环效应、确认偏差)和系统 2 认知缺陷(启发式、风险评估、记忆)。
*   你的思维受到启动的影响:事实回忆、词汇选择、邀功等。当做出关键决定时，试着脱离给定的背景。
*   如果事物具有社会价值(让人看起来不错)、触发因素(首先想到的)、情感价值、宣传、实用价值或娱乐/教育故事，那么它们就会像病毒一样传播。
*   当以的形式呈现时，想法是有粘性的:简单、意想不到、具体(有例子或图像)、可信(有专家、数据或你的观众的支持)、情绪化、有故事。
*   影响力有 6 种武器:回报、承诺&一致性、社会证明、喜欢、权威和稀缺。
*   人们对外界和内心的期望有四种反应方式:支持者重视自力更生；告诉他们你需要什么。提问者看重目的；向他们证明你想要什么。债务人重视团队合作；追究他们的责任。反叛者看重自由；让他们决定做什么。
*   **有 7 种引发迷恋的因素**:情欲(引发对快乐的期待)、神秘感(填补知识空白)、警报(倾向于相关的威胁)、声望(追求社会地位)、权力(控制欲)、邪恶(禁忌的事物)、信任(熟悉的事物)。
*   当倒楣的事情发生时，你可以用积极的方式重建你的现实。
*   快速判断擅长于从噪音中过滤信号，当有了智慧和经验时。当理性化直觉、将能力与外表联系起来、处于压力之下以及存在无意识偏见时，它们是不好的。
*   人们有三个层次的动机:基本需求、外在动机(如金钱、力量)和内在动机。内在动机最强，由心流状态、对意义的追求、自我决定、挑战性任务和在决策中的发言权驱动。
*   你也可以将此表述为对激励机制的研究:经济、社会和道德。小心意想不到的后果断章取义地使用激励，或者用一个较小但更可衡量的激励(如经济激励)取代一个较大的、不可量化的激励(如道德激励)。
*   **心流**是对新奇和成长的享受，技巧娴熟，身临其境，包含挑战和内在回报。
*   游戏化工作。
*   **做许可营销**而不是打断营销:诉诸私利，把精力集中在合格的受众身上。你的目标是建立关系，而不是销售。传授产品的优点，赢得信任。这建立了长期的忠诚度和病毒式传播。

# 科技领导

*   雇佣基于实力，而不是没有弱点。仅在必要时雇佣。建立灵活的团队。总是问你是否有合适的人在合适的地方。快速公平地开火。
*   拥有合适的人比合适的想法和流程更重要。
*   知道什么时候你需要关注战略而不是执行，做一个和平时期而不是战时的 CEO。
*   找到并阐明一个简单的愿景。确保传达这一信息，并以身作则。与 KPI 或北极星交流。
*   激励人们。人们在需要的时候会更加努力地工作，并被一个目标团结在一起。根据团队结果评估绩效。给他们反馈和他们需要的支持。鼓励良性竞争。
*   评估人们的任务相关成熟度。如果他们非常有经验，给他们空间。如果他们不动手的话。重新评估久而久之。和育儿很像。帮助人们最终管理自己。
*   决策:这个决策值得做吗？定义谁对决策负责，从过去学习，并征求不同的观点。
*   监控、配对和趋势指标，以经营业务并理解杠杆。区分目标的优先次序，找出障碍。主动发现问题。
*   避免保密。在关系和评估中提倡开放的交流/激进的真理；使用流程图和指标来阐明事情是如何运作的。
*   尝试新事物，并期待失败。尽早失败并计划迭代。一系列快速决策比过度计划要好。为了抢先上市，花钱“挥霍”R&D。允许个人项目日。
*   建立势头。
*   技术只是加速器。它不会修复你没有的东西。从问题开始，而不是解决方案。
*   用心设计工作场所。
*   **第 5 级领导者**是优秀的管理者，谦虚、有抱负、做继任计划、面对事实、参与苏格拉底式辩论。
*   如何跨越从空想家到实用主义者的鸿沟:拥有一个完整的产品，建立一个海滩头，做一个强有力的声明，锁定你的分销渠道，像市场领导者一样定价。
*   **评估公司的 7 个问题**:这个 10 倍好吗？这是正确的时间吗？我能获得很大的市场份额吗？我有合适的人吗？我能找到合适的客户吗？这能在竞争中生存吗？这算不算非共识？
*   B2B > B2C。
*   **人性化设计**:帮助用户学会使用产品；避免危险的错误，让用户和技术更紧密地联系在一起。
*   口碑的力量意味着你必须**变得不同凡响或者**死去。冒险比回避风险更安全。发明时要考虑营销。
*   早点推出，坚持某事，让它变得无与伦比。尽早、反复、真实地开始。
*   确定市场的中枢；围绕它隔离和创造需求；创新和预期二阶效应；测试假设；从别人的过去中学习。
*   **如何组建一个部落，围绕你的事业发起一场运动**:从慷慨而真实地给予开始，以一种新的方式连接到一个现有的向往，讲述一个未来的故事，创建横向和纵向连接的渠道，给他们一些事情做。

# 人生&事业哲学

*   避免快乐的跑步机。
*   很多媒体都是噪音。不要听噪音。
*   生命太短暂，没有时间抱怨或闲荡。
*   当倒楣的事情发生时，你可以用积极的方式重建你的现实。
*   承担风险，有明确的偏好，建立企业。避免懒惰和傲慢。
*   日子很长，但岁月很短。你每天做的小事比你偶尔尝试做的大事更重要。
*   **正确是一种错觉**:选择你能控制的价值观，从失败中学习，致力于事情，缩小你的选择范围。
*   确定并写下你的核心原则:让你坚持目标的基本原则。
*   用低水平的目标建立大目标。平衡深思熟虑和紧急策略。
*   选择自己能有所作为、有自主权、能进入心流状态的有趣工作。要知道阻力反对我们所有的创造性努力。把你的梦想当成一份工作，并为之努力。
*   避免我们隐藏的偏见去责备和奖励人才。努力和毅力比天赋更有价值；通过深思熟虑和明智的练习，你可以走得非常非常远。
*   永远处于测试阶段，优先考虑学习。学习比挣钱更重要。不断学习。我说过学习对你有好处吗？到了那一步，不要怕转行。
*   了解你自己。了解你的竞争优势。
*   拥有一个领域:抓住注意力，定义一个利基市场，赢得忠诚。
*   顽强地追求突破机会。
*   你应该追求两种层次的人际关系网:深厚的职业联盟，以及广泛而薄弱的熟人。两者都有价值。关系需要不断的努力，但有长期的回报。
*   满意度=动力+纪律
*   召唤缪斯。在导师手下当学徒，然后成长。
*   “做自己喜欢的事”是错误的。通过掌握、自主和深厚的关系，学会热爱你所做的事情。
*   精通是通过工匠心态实现的，专注于稀有和有价值的技能，以非凡的最终目标下小赌注，尽可能接近。
*   达到精通:挑战你所学的。拓宽你的思维。练习技能，直到它们是自动的。着眼于大局。
*   我们这个时代的主流媒体定义了我们对真理的看法。
*   新技术总是被低估为现有技术的延伸。
*   发挥你的长处；在别人身上寻找力量；利用同事的长处来应对他们的挫折。
*   死亡是生命的一部分；面对死亡，活得有意义。
*   成长心态寻求发展，将失败视为机遇，享受困难。改变你的思维模式:改变榜样，寻求支持，谈论错误。
*   玩无限游戏:看到一系列的瞬间，无限的可能性，没有明确的赢家，把它当成游戏，从过去中解放出来。
*   信息时代已经成为概念时代:对左脑的强调已经让位于“右脑思维”，设计、故事、交响乐、移情、游戏和意义在今天变得更加重要。
*   成为关键人物。关键人物是不可或缺的艺术家，他们找到自己的方式，而不是接受指令，他们将情感投入到工作中，并因此获得声誉，他们选择通过重复曝光而不是退缩来对抗恐惧和不适。
*   我们把运气误认为技能(叙事谬误)，忘记了生活是非线性的，是路径依赖的。走完最后一英里往往会有不对称的回报，也要注意对自己不完全了解的风险过度自信(卢迪克谬误)。
*   为了生存而吃:每天 2 磅以上的蔬菜，从植物中获取 90%的热量。
