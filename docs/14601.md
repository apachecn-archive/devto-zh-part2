# 开始赋予我作为一名网页开发者的角色

> 原文:[https://dev . to/la viku/starting-to-empower-me-of-my-role-as-a-web-developer-1 fpm](https://dev.to/laviku/starting-to-empower-me-of-my-role-as-a-web-developer--1fpm)

大家好！我是来自尼加拉瓜的拉维尼娅。我已经在这个“行业”工作了大约 5 年，在 Boombit 公司做网页开发。我基本上在网站的后端工作，我真的很喜欢在代码后面工作。

我最近做了一个小的内部谈话，讲述了我作为一名开发人员的经历，以及对我来说学习一些对男性来说似乎很容易的东西是多么困难，我不是在谈论性别平等，但有些人强调在某些情况下这是关于性别的。

所有这一切让我觉得我应该更多地推动我的人，一个**女人**，开发者。一些对我来说似乎很正常的事情，然而作为这次闲聊的结果，我明白了这真的不正常。我们有很多女人藏在某个地方，等待合适的时机出现，我想利用那个时刻。

正是这些小事让我明白了为什么我选择了这条道路，大多数女性都接触到了社会的指示，成为妻子和母亲，当我们是女孩时，他们给我们买娃娃，如果你是孩子，他们给你一个视频游戏机。就我而言，我不认为我已经过度接触了女孩们的传统事物，我喜欢去我家的 3 个房间或一些杂货店的拱廊玩视频游戏。我比任何女孩都更喜欢数学，我和一些男孩比赛看谁的成绩最好，我想从那时起我就一直是一个相当专业的人，所以我把问题看作是一个可以一步一步解决的流程图。

我希望和我一样多的女性将决定走出去，让人们知道我们每天都在做什么。
