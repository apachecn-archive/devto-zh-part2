# 我们能做到！

> 原文：<https://dev.to/aboydleslie/we-can-do-it--2bh5>

## 我开始编码是因为...

诚实地...我需要一份新工作。我在一家咖啡馆工作了 4 年，并不喜欢它。我发现了 Codebar，这是一个每周一次的研讨会，旨在帮助技术领域中未被充分代表的人学习如何编码。我立刻爱上了网络，在不到一年的时间里，我得到了我的第一个开发角色，我再也不打算回去了！

## 我最近克服了...

SVG 动画！我让这东西动了！是的。我真的很喜欢动画，并希望在这一领域进一步发展。

## 我想吹嘘一下...

我和 codebar。我知道吹嘘自己有时会被人讨厌，但我为自己感到骄傲！找到我喜欢和擅长的事情，从现在起不到一年的时间里找到一份开发人员的工作，知道任何我非常自豪的事情。Codebar 是我想吹嘘的另一件事。这是一个非常棒的组织，帮助了这么多人找到工作。从学生>教练>组织者到现在一直是一个令人惊奇的经历，看到学生们在这个过程中学到更多和发展是非常值得的。

## 我给盟友的建议是支持女性主义者的守则....

多招！做而不只是说。支持 codebar 等组织并为其做出贡献。
