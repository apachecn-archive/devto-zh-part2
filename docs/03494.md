# 我的域名和网站内容被劫持了！我该怎么办？

> 原文:[https://dev . to/jj mpsp/my-domain-name-and-website-content-has-hacked-what-should-I-do-3pm 9](https://dev.to/jjmpsp/my-domain-name-and-website-content-has-been-hijacked-what-should-i-do-3pm9)

由于个人疾病，我已经离开 it 行业很长时间了。这段时间我的个人域名到期了，没有考虑续费。我现在正慢慢从疾病中恢复，重新开始用电脑工作，甚至考虑让我的网站重新上线，但今天我发现我的域名和网站内容被盗了！

我最初的网站运行在 Wordpress 上，有一个时尚的自定义主题，但内容似乎是抄袭和克隆的。这不是一个直接的克隆，因为看起来无论是谁做的都降低了内容，现在在我最初的博客文章中有他们网站的反向链接。例如“C++移动开发- Snapchat”链接，来自:

[http://joel-murphy.com/mobile-app-development.html](http://joel-murphy.com/mobile-app-development.html)到 https://intetics.com/portfolio/technologies/c->T2

我在我的旧域名上做了一个 whois 查询，似乎是谁买了这个域名并在 Amazon s3 服务器上托管内容。

我有一种强烈的感觉，这是[https://intetics.com/](https://intetics.com/)只是为了 SEO 的好处，但我不确定他们为什么会针对我？我拥有我的域名 http://joel-murphy.com/[已经很多年了，我的简历、名片、LinkedIn 等等都有链接。我的网站确实有一些流量，但不足以成为目标。我想我除了在将来使用不同的域名之外没有太多可以做的，但我更担心我的在线身份被嘲笑或玩弄的可能性。有人遇到过类似的情况吗？](http://joel-murphy.com/)
