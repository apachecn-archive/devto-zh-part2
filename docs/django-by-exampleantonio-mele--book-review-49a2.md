### 想法

这本书试图向已经足够熟悉这个框架的人传授更先进的 Django 技术。我认为它在这方面做得很好。我喜欢其中的项目，尤其是社交网络项目。

### 项目:

包括的项目有

*   创建一个全面的博客(几乎所有你能在博客中找到的预期功能)，与第三方集成(第 1、2、3 章)
*   用图片创建一个基本的(但不是微不足道的)社交网络。(第五、六章)
*   创建一个集成贝宝支付和推荐引擎的网上商店(没有机器学习，纯黑客的东西:)
*   创建一个电子学习平台，重点是 Django 的缓存机制

### 使用的一些技术(当然除了 django)

*   芹菜
*   jQuery
*   创建交互式、快速动态网页应用的网页开发技术
*   [DjangoRestFramework](http://www.django-rest-framework.org/) 帮助你让你的应用 [RESTful](https://searchmicroservices.techtarget.com/definition/RESTful-API)

正如所有大型项目一样，一些缺陷是不可避免的。尽管他们很少。我注意到的最重要的是:

*   代码中的一些错误。好吧，你会说，哪本书没有？
*   这本书的版本是 1.8。当然，这不是作者的错，只是这本书 3 年前出版了，我们现在是第二版。

### 要我读吗？

我觉得这本书很有趣，对我的项目也很有用。所以依我拙见:

*   当然，如果你有兴趣在 Django 上变得更好。它可能是一个稍旧的版本，但是所讨论的概念并没有受到太大的影响
*   完全不是，如果你对成为一名更好的 [Django](https://www.djangoproject.com/) 开发者不感兴趣，或者你不知道 [Django](https://www.djangoproject.com/) :)的基础知识，在后一种情况下，有各种各样的资源可以让你做好准备。比如[网忍者](https://www.youtube.com/playlist?list=PL4cUxeGkcC9ib4HsrXEYpQnTOTZE1x0uc)和[sendex](https://www.youtube.com/playlist?list=PLQVvvaa0QuDeA05ZouE4OzDYLHY-XH-Nd)甚至[官方](https://docs.djangoproject.com/en/2.0/intro/tutorial01/)教程。
