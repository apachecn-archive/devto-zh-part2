# OpenBSD 6.3 上的 PostgreSQL 10.3:安装

> 原文:[https://dev . to/nab bisen/installing-PostgreSQL-server-on-openbsd-1kc 0](https://dev.to/nabbisen/installing-postgresql-server-on-openbsd-1kc0)

这篇文章展示了如何在 [OpenBSD](https://www.openbsd.org/) 上安装 [PostgreSQL](https://www.postgresql.org/) 并设置它。

<center>✿ ✿ ✿</center>

## [](#environment)环境

*   操作系统:OpenBSD 6.3
*   数据库:PostgreSQL 10.3

<center>✿ ✿ ✿</center>

## [](#procedure)程序

* *注意*:在代码区，前导`#`表示由超级用户执行；相当于使用`doas`命令(作为 root)，而前面的`$`是指一般用户。

### [](#install-package)安装包

```
#  pkg_add postgresql-server 
```
