# React 本机选取器微调器下拉菜单列表示例- Android

> 原文：<https://dev.to/skptricks/react-native-picker-spinner-dropdown-menu-list-example---android-2b6i>

帖子链接: [React Native Picker Spinner 下拉菜单列表示例- Android](https://www.skptricks.com/2018/10/react-native-picker-spinner-dropdown-example-android.html)

在本教程中，我们将讨论如何在 react 本地应用程序中创建简单的选择器下拉菜单列表。选取器微调器提供了从集合中选择一个值的快速方法。在默认状态下，微调器显示其当前选定的值。触摸微调器会显示一个包含所有其他可用值的下拉菜单，用户可以从中选择一个新值。
