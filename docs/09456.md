了解 node js 的一个方法是构建东西。很多开发人员说他们知道 nodejs，因为它只是服务器端的 javascript，但他们中的许多人不知道如何使用 Nodejs 标准库。让我们从使用 assert 库开始，构建一个简单的测试库。
