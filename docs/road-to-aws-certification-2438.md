我在 AWS 工作了大约两年。我不宣称自己是专家或类似的东西，但肯定我在这段时间里学到了很多。

几个月前，我决定通过学习获得 AWS 认证——AWS 解决方案架构师。这就是为什么我在 AWS 上创建了一系列帖子，如教程、指南和提示/技巧。

有很多东西我需要学习和深入理解，我认为教他们是最有效的学习方法。

欢迎对该系列帖子的任何反馈、更正和/或建议！此外，我将自己重新发明 2018，如果有人想在那里闲逛，请随时与我交谈！

Thanks a lot, fam! ❤️
