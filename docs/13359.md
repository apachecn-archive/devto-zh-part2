# 为初级开发人员准备课程

> 原文:[https://dev . to/rapasoft/preining-a-course-for-junior-developers-4fnf](https://dev.to/rapasoft/preparing-a-course-for-junior-developers-4fnf)

因此，我们决定为初级开发人员开设一门持续几周的课程。重点是向他们介绍真实项目中使用的最新技术，并向他们展示软件工程中的最佳实践。虽然之前我有过几次演讲和短期研讨会，但我仍然有点纠结如何进行才能给参与者带来价值。

我确定了几件事，比如:

*   准备所有先决条件(环境、材料、Git 存储库)
*   与较小的团队合作(或在克服障碍方面获得额外的帮助)
*   不要给人们过多的信息
*   如果可能，选择参与者(以便每个人都在同一水平上开始)
*   友好，试着激励而不是仅仅陈述事实

无论如何，我想就你认为成功课程的关键点展开讨论。如果你有什么小技巧可以分享，请不要犹豫；).
