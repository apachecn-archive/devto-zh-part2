# 如何在 Div 中将 HTML 元素居中

> 原文：<https://dev.to/flippedcoding/how-to-center-an-html-element-in-a-div-1lgp>

我在 web 开发中遇到的最烦人的问题之一是，元素并不总是在我需要它们的地方。尤其是当他们需要居中的时候。这是 CSS 中简单的事情之一，由于 CSS 样式的工作方式，它可能会比它应该的更复杂。你可以在一个 HTML 元素上设置一种样式，仅仅因为你把代码按不同的顺序排列，就把整个页面打乱了。

当我的元素不在中心时，我总会尝试几件事情，在这五件事情之间，其中一件通常可以解决问题。所以，如果你想找出 HTML 元素不居中的原因，先试试这些东西。

1.  你检查过你的 div 了吗？

    *   您是否确保了您尝试居中的元素在正确的 div 中？我知道这听起来很简单，但是当您有一个大型的 web 应用程序时，很容易迷失在代码中。检查您是否拥有正确的类名或 id，以及您是否以正确的元素为目标。
    *   确保你的 CSS 选择器和属性是针对正确的元素，因为如果你选择了错误的元素，那么它永远不会居中。
2.  你检查过你的利润了吗？

    *   边距很难处理，因为它们会将元素的宽度改变为您不希望的宽度。如果你在 CSS 的其他地方设置了边距，那么它可能会影响到你正在更新的元素。当您设置当前元素的边距时，您会想要使用:`margin: 0 auto;`
    *   请记住，当 margin 属性为这种格式时，第一个值用于上边距和下边距，第二个值用于左边距和右边距。如果您希望上边距和下边距有一个特定的值，如 2rem，则零可以是另一个值。auto 值会将元素左边和右边的边距设置为相等，这将导致元素在页面上居中。
3.  你检查过你的衬垫了吗？

    *   您可能为网页上的某个元素设置了一个填充值，该值会将您试图居中放置的元素移位。检查没有任何填充值会改变当前元素的位置。你可以通过按键盘上的 F12 键或者浏览你的 CSS 代码，在网络浏览器的开发工具中检查这一点。
    *   通常，只要左右填充值相同，在元素上设置填充值不会使元素偏离中心。如果你在开发工具或 CSS 文件中看到任何类似的填充:`padding: 1em 3em 1em 6em;`那么你可能想检查一下。
4.  你量过你的宽度了吗？

    *   元素的宽度会直接影响您将其居中的能力。如果元素没有固定的宽度，这很容易使它偏离中心，因为浏览器会自己决定元素的宽度。确保你的元素有正确的宽度，如果你有任何疑问，通常安全的做法是给它赋值:`width: 100%;`
    *   这样你就不会有任何奇怪的 CSS 问题发生，因为你的浏览器试图决定宽度。即使指定了宽度，也要检查以确保值是正确的。这样你就能确定宽度不会使元素保持在错误的位置。
5.  你检查过你的显示器类型了吗？

    *   这个问题花了我最长的时间才弄明白。当你和一堆 div 一起工作时，很容易忘记你在哪个 div 级别上。有些 div 可能是浮动的，有些可能有绝对位置，有些可能有独特的显示类型。
    *   如果你在 CSS 中检查你的元素的显示属性，但是你没有看到，试试这个:`display: block;`这通常会直接将元素定位。当您没有需要使用的特定显示类型时，这是最有用的。
