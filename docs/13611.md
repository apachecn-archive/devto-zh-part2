# Clojure(/Script)最好的首次运行和工具体验是什么？

我有兴趣尝试 Clojure 和/或 ClojureScript。然而，我承认在工具上有点被宠坏了。

> 奢侈品，一旦享受，就变成了必需品。
> 
> *   C.诺思科特·帕金森

最好的(非 Emacs)第一次运行和持续的工具体验是什么？免费是最好的，但有吸引力的付费选项也可以。我目前习惯的是:

在前端，我使用榆树。有一个很棒的模板项目叫做 Create-Elm-App，它建立了产品构建和开发工具，比如热重装、时间旅行调试器等等。然后我使用 VS 代码来处理普通的事情(高亮显示、git 集成、文件组织、智能感知等)。

在后端，我使用 full VS，它有项目模板、调试，现在甚至还有 docker 集成(将为您设置可调试的容器)。除了正常的东西。
