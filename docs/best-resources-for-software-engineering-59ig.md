人们常说软件工程师的工作就是解决问题。但是仅仅在技术上强是不够的，你不能整天坐在屏幕前按按钮，在这个职业中有更多的东西需要了解。像活动，面试准备，获得灵感，避免倦怠，了解薪水和职业步骤，博客都在某种程度上影响你的生活。

互联网是一个巨大的资源库，以下是我为帮助你入门而列出的清单。

*   [分享会](http://2359media.com/2014/10/29/perspective-what-i-learnt-from-a-serial-entrepreneur/)每周一次的内部分享会，让我们的开发人员、设计师和业务开发团队能够通过相互学习扩展技术和非技术知识

*   实验室已经成为学习制造新东西、测试新兴技术和制造有趣东西的重要地方。
