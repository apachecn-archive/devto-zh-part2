## [](#postgres)Postgres

第一步是运行 Postgres (≥9.5)，因为 HGE 依赖它。最简单的方法是启动 Docker 容器，如果你的 CI/CD 平台支持的话。

```
docker run -d -p 5432:5432 postgres 
```

这将使用名为`postgres`的默认数据库启动 Postgres。数据库 url 将是`postgres://postgres:@localhost:5432/postgres`。
