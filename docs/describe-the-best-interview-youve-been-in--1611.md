# 描述一下你参加过的最好的面试

> 原文：<https://dev.to/kaydacode/describe-the-best-interview-youve-been-in--1611>

我最喜欢的一次面试是，我可以带着自己的笔记本电脑做一个代码练习，大部分是提前交流的。他们还允许我使用我在现实世界中可以使用的任何工具:房间里的其他开发人员、Google、Stackoverflow 等。

我也喜欢清晰地描绘出我的一天将会是什么样子——一个时间表，上面有我将和谁谈话，我将被问什么类型的问题。例如:上午 8 点，你将与软件经理 Bob Bobbs 会面，讨论以前的经验。中午你将和队员们一起吃午饭。下午 3 点，您将与建筑师 Linda Lindz 会面，进行白板面试。好吧，好吧，我会被问到的问题类型是一个梦-但一个女孩可以希望。

但我喜欢这样做的原因是，1)在别人的笔记本电脑上编写代码实际上相当具有挑战性。考虑相反的滚动方向，不同的键盘快捷键和 IDE 颜色。这些都会让你失去兴趣。2)我有焦虑症..这使得面试压力特别大。了解日程安排和一天中要做的事情有助于我为即将到来的一天做好心理准备和应对。我能描述它的最好方式不是我的头脑在赛跑“你将会失败整整 6 或 8 个小时”，而是听起来更像“嗯，早晨将会是艰难的，但是你已经完成了 11-3 部分”..有了这个想法，我真的可以呼吸了。

你最喜欢的采访是什么？为什么？
