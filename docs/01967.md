# 为什么你应该重新发明轮子

> 原文:[https://dev . to/quantum sheep/why-you-should-re vent-the-why-you-the-wheel-4ha2](https://dev.to/quantumsheep/why-you-should-reinvent-the-wheel-4ha2)

有时一些人告诉不要通过创造已经存在的东西来“重新发明轮子”。是时候澄清为什么我们要重新发明轮子，为什么他们是错的了！

# 学习

在这种情况下，你能说的最好的理由就是你想学习。革命性的想法来之不易，如果你想学习，就做你想做的事情，不要特别关注创造新事物。

你需要技能来创造你的想法。通过做任何你想到的事情来获得这些技能！

# 了解事物如何运作

做一些已经存在的事情，可以让自己明白它是如何运作的，如何改进。事情并不完美，但可以完美，让我们看看智能手机行业的例子。

# 你想做什么

你重新发明了轮子？那又怎样？人们不应该把你带入不必要的境地。做东西开心就好。如果那些已经存在也没关系，至少你会快乐，这才是最重要的。

# “车轮”得到了改进

“不要多此一举”这个比喻大错特错！严格地说，轮子已经被重新发明了无数次。在第一次尝试中，我们没有在高科技汽车中使用完整的橡胶轮。事物的发展是因为人们“重新发明轮子”，它应该继续这样。

总之，只是提升自己，不要想着做出一个全新的想法。找到还没有被创造出来的东西并不容易，所以如果你的唯一目标是发现和探索新事物，就不要想太多。

创意来自发现。随着时间的推移，会有新的发现。
