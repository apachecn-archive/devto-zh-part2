是的，咖喱食品非常适合这个。箭头语法有助于这一点。

```
valueChange = (key) => {
  return function (e) {
    var obj= {};
    state[key] = e.target.value; //<-- this is a bug btw.
    this.setState(obj);
  }.bind(this);
} 
```
