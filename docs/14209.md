# 我喜欢犯错的 5 个理由

> 原文:[https://dev.to/lpasqualis/5-reason-why-i-love-being-wrong-4f 58](https://dev.to/lpasqualis/5-reason-why-i-love-being-wrong--4f58)

这篇文章最初全文发表在 [CoderHood](https://coderhood.com) 上，名为[我喜欢犯错的 5 个理由](https://www.coderhood.com/5-reason-why-i-love-being-wrong/)。CoderHood 是一个致力于软件工程人类层面的博客。

当我用 100%的时间编程时，我的工作就是编写健壮且可维护的代码。我的工作需要被很好地记录，用适当的抽象层次设计，尽可能地无错误和高效。我还必须想出如何解决问题和创造可行产品的好主意。

作为一名开发人员，我必须在很多时候都是正确的。写代码时出错会导致错误和其他问题。不管代码审查和测试如何，软件代码中愚蠢的想法总是会增加税收。

在我的管理角色中，事情是非常不同的。这是一份完全不同的工作。我的主要职责是雇佣优秀的员工，为他们创造最佳的成长和工作环境，确保现实得到很好的界定，并确保我们做出正确的决策。

在我的工作中，如果我有好的想法会有所帮助，但这并不意味着我必须提出最好的想法。这只是意味着我必须确保最好的想法浮出水面，不管它们来自哪里或由谁提出。我对我团队的结果完全负责。我不负责在我的团队做的每一件事上都打上指纹。

## 在被冤枉上

我不否认。大多数时候我都有自己的看法。拥有明智的观点是我工作的一部分，但我不期望它们总是正确的。我雇佣比我聪明得多的人来想出比我更好的主意。

当与我的团队讨论时，我不介意出错。其实我挺喜欢的。我甚至可以说，我喜欢犯错，至少事后看来是这样。

我不想犯错；恰恰相反。我尽最大努力做到消息灵通，有好的论据，尽我所能去学习，以确保我尽可能地正确。如果我每次开口都是错的，最终我的团队会对我的能力失去信心，这将使我无法领导。

尽管很努力，我还是错了很多。这就是雇佣聪明人的快乐和负担。他们不会让老板得逞，这太棒了！我不想以任何其他方式得到它。

最重要的是，当我努力去做正确的事，但我意识到我做错了，结果，做出了正确的决定，这时我就赢了。这可能看起来像《爱丽丝梦游仙境》的颠倒逻辑，但让我解释一下我给你五个理由的意思。

### 如果我用一个坏主意挑战一个好主意，好主意会变得更强

如果我的团队中有人有意见，我会测试它，如果它是一个好主意，那么它会成长并变得更强大。如果这是一个坏主意，那么它就会死去或者选择一个不同的方向。不管怎样，我们取得了进展。

为了取得进步，我的挑战是对是错几乎无关紧要。对于重要的事情，我不会让听起来不错的第一个想法未经更深入的审查就被接受，这确实很重要。

我并不主张你应该 100%地质疑任何人所说的一切。这很烦人，有时候你只需要相信别人。走极端从来都不是好事。我的意思是，当事情很重要时，即使听起来不错，也应该仔细检查。

### #2 -当我错了，我团队中的其他人是对的

当我发表意见时，我确信它是正确的，除非我在唱反调，有时我就是这样做的。然而，我的工作不是要正确；我的工作是确保正确的决定浮出水面。得到最好的想法才是最重要的。当一个想法比我的更好时，我确实需要小心识别；但是，只要我听着，睁大眼睛，竖起耳朵，这通常不是问题。

我也不是思想的法官。我团队中的每个人都是想法的评判者。我的工作是听取包括我在内的众多角度的意见，并选择一个通过审查并走在实现公司高层目标的正确道路上的角度。

当有人证明我的观点是错误的，我们都取得了进步，每个人都赢了。无论作者是谁，抛弃不好的、分散注意力的想法，寻找更好的想法是至关重要的。这是导致良好决策和结果的原因。如果我有一个坏主意，帮助别人想出一个更好的，我们取得了进展，每个人都赢了，包括我。

可能发生的最糟糕的事情不是犯错。可能发生的最糟糕的事情是，没有人能想出经得起推敲的东西。如果发生这种情况，那是我的责任，因为我显然没有雇用合适的人，或者我没有为我的员工提供合适的机会或环境来做他们需要做的事情。

### # 3——当我错了，我会学习

我不否认。某件事是对的让我感觉很好，谁不喜欢这种感觉呢？我知道我有。然而，如果我一直是对的，我就什么也学不到。

只要团队最终做出正确的决定，我宁愿犯错并吸取教训，而不是正确并停滞不前。停滞是我对地狱的定义。

### # 4——当我错的时候，我成长

每当我错了，我就学到了一些东西，但我也成长了一英寸。意识到自己错了是让自己变得更成熟、更能适应下一个挑战的一种方式，它让我脸皮更厚。下一次在类似的情况下，我将能够应用我所学到的东西，帮助团队取得进步。

现在，再一次，我不是在谈论极端。大多数时候犯错最终会毁了我。当谈到“因犯错而成长”时，有一个甜蜜点和一个健康的比率。我明白了，生活中的一切都是一条钟形曲线，极端的情况很少是健康的。

### # 5——当我错了的时候，就是在进步

想象一个极权主义政权，在那里独裁者在法律上必须永远是正确的。不允许任何人质疑独裁者说的任何话，每个人都必须捍卫他的决定，不管这些决定有多古怪。

即使独裁者是个天才，你认为那个政权会成功吗？一个人不可能从所有的角度看现实。每个人都有盲点、偏见和对现实的有限理解。不同的想法和对最佳想法的不断追求是进步和创新的动力。

只要有公开的辩论，只要最好的想法存在，就有进步，不管谁的想法好，谁的想法错。

## 一些最终的想法

在软件行业，一切都是团队合作的结果。每隔一段时间，你需要一个天才来播下一颗梦幻般的、革命性的种子，但除非通过开放的对话、合作和“精英思想”的环境来培育，否则这颗植物不会生长。

精英管理是一种决策环境，在这种环境中，最好的想法会胜出。为了创造它，你必须给人们一个机会把他们的想法放在桌子上让每个人都看到，有建设性的分歧，并有一个协议来克服任何仍然存在的冲突。该协议可以是“老板决定”只要做出了最有可能成为正确决定的决定，这都没关系。没有决定是最糟糕的决定。

在精英管理的理念中，谁对谁错并不重要。唯一重要的是，最好的想法能够存活并茁壮成长。
