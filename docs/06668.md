# 女孩修补工作坊

> 原文:[https://dev.to/guergana/tinkering-workshop-for-girls-dc0](https://dev.to/guergana/tinkering-workshop-for-girls-dc0)

几个月前，我加入了柏林一个名为“未来传奇”的组织，该组织的目标是向 6-10 岁的女孩传授技术，无论是编程、肢体语言还是自信。目标是帮助下一代缩小技术上的性别差距。我们希望更多的女性进入科技行业！

昨天我在第一节课上做了辅导，我们教了女孩们一些基本的电子原理:什么是马达，什么是电池，什么是闭合电路，以及我们如何将所有这些原理结合起来制作一个艺术机器人。用标记腿画画的机器人。

他们还可以用眼睛和马克笔来装饰他们的机器人。通过这种方式，我们向他们展示了技术也是有趣的，并且对创造力开放。它也可以用来做艺术的东西。
