我一直在琢磨如何使用 Grav CMS 平台，目前为止我很喜欢它。它有一些学习曲线，以了解一个人如何围绕核心，以及如何把它们放在一起，但总的来说，它工作得很好。我唯一担心的是，通过微小的配置调整很容易破坏系统。虽然 Grav 有很多主题和插件来丰富你的网站，但它仍然是一个非常实用的平台，你需要了解 TWIG、Markdown 和 YAML 以及传统的 HTML 和 CSS。
