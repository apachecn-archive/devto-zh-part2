# 什么是 Aws Lambda？

AWS Lambda 无需供应或管理服务器即可运行您的代码，这意味着您无需购买或管理服务器。您只需在您的代码运行或用户访问您的网站时付费。

#### 创建和部署 Rest API 端点

在您的工作目录中创建一个模板

```
serverless create --template aws-nodejs 
```
