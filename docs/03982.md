# 通过终端#OneDevMinute 转到前一个 Git 分支或前一个目录

> 原文:[https://dev . to/Ahmad awais/go-to-previous-git-branch-or-previous-directory-via-terminal-onedevminute-23ig](https://dev.to/ahmadawais/go-to-previous-git-branch-or-previous-directory-via-terminal-onedevminute-23ig)

在另一个#OneDevMinute 中，您将了解访问前面的 git 分支是多么容易，然后在您的终端中对普通目录进行同样的操作。

说真的，如果您在两个不同的目录或两个不同的 git 分支中工作，那么一次又一次地来回移动会非常麻烦。这个提示为你设置了`cd -`或`git checkout -`。观看视频了解更多信息。

```
git checkout other-branch
git checkout yet-another-branch
git checkout - # Takes you back in the other-branch
git checkout - # Takes you back in the yet-other-branch

cd dir1
cd dir2
cd - # Takes you back in the dir1
cd - # Takes you back in the dir2 
```
