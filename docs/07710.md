# 介绍我们 2018 年 8 月的赞助商

> 原文:[https://dev.to/ben/introducing-our-august-2018-sponsors-a82](https://dev.to/ben/introducing-our-august-2018-sponsors-a82)

我们很自豪地宣布我们八月份的持续社区赞助商。我们出色的赞助商对 [dev.to](https://dev.to) 的健康发展至关重要，能与对生态系统做出如此大贡献的公司合作真是太棒了。

所有这些赞助商过去都曾与我们合作过，所以我强烈建议你去看看他们的产品。他们对整个社区都非常好。

## 数字海洋

[数字海洋](https://try.digitalocean.com/performance/?utm_source=devto&utm_medium=display&utm_campaign=Devto_2018_Brand)是一个备受喜爱的云计算平台。我总是对数字海洋核心产品的良好构建*和良好记录*印象深刻。说起来容易做起来难，他们已经能够保持领先。

## 木材

木材是伐木业中最有趣的产品之一。围绕可读性和可用性构建，Timber 是一个您实际上很乐意使用的记录系统。我们自己从测试期就开始使用木材了。他们正在做伟大的工作。

Timber 目前与 Ruby、Elixir 和 Node 应用程序的集成最为无缝。这是我们团队极力推荐的。

## 落水狗

[Underdog](https://underdog.io/?utm_medium=homepage&utm_source=devto&utm_campaign=homepage2018-aug-30&utm_content=engineers-tech-companies) 在旧金山和纽约这两个美国主要市场，为优秀的开发者和优秀的科技公司牵线搭桥。如果你正在这些中心之一寻找一个职位，跳过这条线，让超狗成为你的伙伴。

* * *

这些组织正在帮助我们尽可能建立最好的社区。如果你想购买他们的服务，我建议你去看看他们的产品。
