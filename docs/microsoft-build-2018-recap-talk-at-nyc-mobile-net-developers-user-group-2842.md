上周，我在 [NYC Mobile 做了一次有趣的演讲(和](https://www.meetup.com/nycmobiledev) [Sara Morsi](https://twitter.com/eowyn327?lang=en) 一起做演讲人)。NET 开发者用户组。我们策划了一系列主题，这些主题来自我们最近在华盛顿州西雅图举行的[微软 Build](https://www.microsoft.com/en-us/build) 的经历。

在构建阶段，我们发现与 Xamarin 开发人员最相关的一些主题包括:

1.  Hyper-V Android 模拟器支持
2.  Xamarin 的 Visual Studio 工具
3.  平台改进
4.  Xamarin 表单 3.0
5.  流畅的 UWP 设计
6.  Visual Studio 实时共享…等等…
