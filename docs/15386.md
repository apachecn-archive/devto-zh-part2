## [](#to-whom-i-recommend-firebase)我向谁推荐 Firebase？

我最推荐 Firebase 给企业家、初创公司和参加[黑客马拉松](https://en.wikipedia.org/wiki/Hackathon)的人，我这么说是因为这些特定市场的需求在开发阶段不断变化。

使用 Firebase 构建 MVP 相对于使用其他工具的人来说有很大的优势，因为使用其他工具会花费很多时间，例如对大型数据库结构的分析。如果你以前参加过黑客马拉松，你已经知道时间是至关重要的；你必须花时间思考一个好的商业想法，然后当你脑海中有了具体的东西，你必须记录下来，然后开发时间减少到大约 40%或更少，以创建一个功能性的 MVP...如果是这样的话，Firebase 将是你的英雄。

如果你想了解更多关于 Firebase 的知识，欢迎访问官方指南:[https://firebase.google.com/docs/guides/](https://firebase.google.com/docs/guides/)
