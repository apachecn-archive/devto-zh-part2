# 关于网络调查现状的最后思考

> 原文:[https://dev . to/Ben/final-thoughts-on-the-state-of-the-web-survey-44nn](https://dev.to/ben/final-thoughts-on-the-state-of-the-web-survey-44nn)

今年夏天，我们开展了一项名为“网络现状”的社区调查。我们询问了开发者，网络和其他方面的，关于网络作为用户和开发者的平台的想法。万维网是有史以来最普遍的平台之一，在可预见的未来，它仍将是计算的关键。这是一项发展迅速的技术，也是一个需要经常探讨的话题。那是保守的说法。

尽管网络很重要，但它远不是一个完美的平台。由于需要平衡兼容性、创新和标准开发，web 经常看起来像一团乱麻。硅谷浏览器巨头每个都有自己的功能开发动机，而世界各地的开发者和用户都有自己的需求。我们设法让这个弹性平台发挥作用。

我们对 2000 名开发人员的调查帮助我们发现，CSS flex-box 确实已经在开发人员中得到重要使用，在 web 上禁用 JavaScript 仍然是一个激烈的争论，并且爱狗的人比爱猫的人更有纪律。最后一个在推论数据中似乎很清楚，试着证明它是错的！
